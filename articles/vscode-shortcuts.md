# קיצורי מקלדת ב-VS Code

| קיצור                     | פעולה                  | קטגוריה |
| ------------------------- | ---------------------- | ------- |
| `Ctrl` + `Shift` + `P`    | לפתוח את תפריט הפקודות | בסיסי   |
| `Ctrl` + `P`              | לפתוח קובץ לפי שם      | בסיסי   |
| `Ctrl` + `Shift` + `N`    | לפתוח חלון חדש         | בסיסי   |
| `Ctrl` + `Shift` + `W`    | לסגור חלון             | בסיסי   |
| `Ctrl` + `,`              | לפתוח הגדרות           | בסיסי   |
| `Ctrl` + `K` `Ctrl` + `S` | לפתוח תצורת מקשי קיצור | בסיסי   |

## עריכה

| קיצור                        | פעולה                      |
| ---------------------------- | -------------------------- |
| `Ctrl` + `X`                 | לגזור שורה (אם אין סימון)  |
| `Ctrl` + `C`                 | להעתיק שורה (אם אין סימון) |
| `Alt` + `↑/↓`                | להזיז שורה למעלה/למטה      |
| `Shift` + `Alt` + `↓/↑`      | לשכפל שורה                 |
| `Ctrl` + `Shift` + `K`       | למחוק שורה                 |
| `Ctrl` + `Enter`             | להוסיף שורה חדשה מתחת      |
| `Ctrl` + `Shift` + `Enter`   | להוסיף שורה חדשה מעל       |
| `Ctrl` + `Shift` + `\`       | לקפוץ לסוגר התואם          |
| `Ctrl` + `]` / `[`           | להזיח שורה ימינה/שמאלה     |
| `Home` / `End`               | לקפוץ להתחלה/סוף של שורה   |
| `Ctrl` + `Home` / `End`      | לקפוץ להתחלה/סוף של קובץ   |
| `Alt` + `Click`              | להוסיף סמן משני            |
| `Ctrl` + `Alt` + `↓/↑`       | להוסיף סמן נוסף למטה/למעלה |
| `Ctrl` + `U`                 | לבטל פעולת סמן אחרונה      |
| `Shift` + `Alt` + `←/→`      | להרחיב סימון בלוק          |
| `Ctrl` + `Shift` + `[` / `]` | לקפל/לפתוח בלוק קוד        |
