
### סוף סוף: תמיכה ב-Markdown ב"פנקס רשימות" של Windows 11 זמינה לכולם 👏

התמיכה בעיצוב טקסט בתחביר Markdown ב"פנקס רשימות" (Notepad) זמינה כעת לכל משתמשי Windows 11.

ב"פנקס רשימות" ניתן לעצב כותרות (באמצעות סימון כמו H1), ליצור רשימות עם תבליטים ורשימות ממוספרות, ולהדגיש טקסט באמצעות כתיב מודגש (בולד) או נטוי (איטליק). לבסוף, הכלי תומך גם בהיפר-קישורים.

קיצור המקשים `Ctrl + K`, המוכר גם מ-Word, מאפשר להוסיף קישור שייפתח בדפדפן ברירת המחדל.

בנוסף, ל"פנקס רשימות" נוסף כפתור "תצוגת עיצוב" (Formatting Preview) בתחתית המסך, המאפשר לעבור למצב תצוגת תחביר Markdown. הדבר מאפשר להפוך סולמיות (#) לכותרות, להשתמש בכוכביות (*) להדגשה, לעטוף קטעי קוד בגרשיים הפוכים (`) ועוד. תצוגה זו דומה לעבודה עם עורך Markdown בצד-שרת (backend).

באתר Windows Latest גילו כי תמיכת ה-Markdown מופעלת כברירת מחדל, אך ניתן לחזור לחוויה הקלאסית. לשם כך, יש לפתוח את תפריט "הגדרות" (Settings), שם נוספה אפשרות חדשה בשם "עיצוב" (Formatting). השבתת אפשרות זו תחזיר את "פנקס רשימות" לגרסתו המוכרת והפשוטה.

מבדיקות שבוצעו עולה כי הטמעת תכונת העיצוב ב"פנקס רשימות" אינה מאטה את ביצועי המחשב.

מיקרוסופט החלה לשלב את תמיכת ה-Markdown ב"פנקס רשימות" עבור Windows 11 בתחילת חודש יוני. עם זאת, חלק מהמשתמשים הביעו חוסר שביעות רצון מהחידוש. לטענתם, הוא סותר את הפשטות המקורית של "פנקס רשימות", וכי התכונות השימושיות הללו כבר קיימות ב-OneNote.
