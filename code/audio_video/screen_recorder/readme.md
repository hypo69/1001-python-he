# סקריפט להקלטת מסך באמצעות Python

סקריפט זה, `screen_recorder.py`, מאפשר להקליט את מסך המחשב שלכם לקובץ וידאו באמצעות הספריות `opencv-python`, `pyautogui`, `numpy` ו-`keyboard`.

## דרישות מוקדמות

לפני הפעלת הסקריפט, ודאו כי הספריות הבאות מותקנות אצלכם:

-   `opencv-python`
-   `pyautogui`
-   `numpy`
-   `keyboard`

ניתן להתקין אותן באמצעות pip:

```bash
pip install opencv-python pyautogui numpy keyboard
```

## תיאור הקוד

הסקריפט מבצע את השלבים הבאים:

1.  **ייבוא ספריות:** מייבא את הספריות הנחוצות.
2.  **הגדרת פרמטרים:**
    -   מקבל את גודל המסך באמצעות `pyautogui.size()`.
    -   מגדיר את כמות הפריימים בשנייה (FPS) עבור הווידאו.
    -   מגדיר את המקודד (codec) עבור הווידאו (XVID).
    -   קובע את שם קובץ הפלט (`screen_recording.mp4`).
3.  **יצירת אובייקט VideoWriter:** יוצר אובייקט `cv2.VideoWriter` עבור הקלטת הווידאו.
4.  **לולאת הקלטה:**
    -   לולאה אינסופית המצלמת צילומי מסך באמצעות `pyautogui.screenshot()`.
    -   ממיר את צילום המסך למערך NumPy.
    -   ממיר את מרחב הצבעים מ-RGB ל-BGR (נדרש על ידי OpenCV).
    -   מקליט את הפרַיים לקובץ הווידאו.
    -   בודק אם המקש 'q' נלחץ באמצעות `keyboard.is_pressed('q')`. אם נלחץ, יוצא מהלולאה.
5.  **סיום ההקלטה:**
    -   משחרר את אובייקט `VideoWriter` (`out.release()`).
    -   מציג הודעה המציינת שהווידאו נשמר.

## שימוש

1.  שמרו את קוד ה-Python בקובץ בשם `screen_recorder.py`.
2.  הפעילו את הסקריפט:

    ```bash
    python screen_recorder.py
    ```

3.  הסקריפט יתחיל להקליט את המסך. כדי לעצור את ההקלטה, לחצו על המקש 'q'.

4.  קובץ הווידאו `screen_recording.mp4` ייווצר באותה ספרייה שבה נמצא הסקריפט.

## הערות

-   ייתכן שיהיה צורך להתקין מקודדים (codecs) נוספים במערכת שלכם עבור המקודד XVID.
-   ביצועי הסקריפט תלויים בעוצמת המחשב שלכם ובגודל המסך.
-   ייתכן שיהיה צורך להתאים פרמטרים, כגון FPS והמקודד (codec), להשגת תוצאה מיטבית.
