# המרת אודיו לטקסט באמצעות Python
```

מודול `audio_to_text.py` זה מדגים כיצד להמיר קובץ אודיו לטקסט באמצעות ספריות `speech_recognition` ו-`pydub` ב-Python.

## דרישות קדם

לפני הפעלת הסקריפט, ודאו שהספריות הבאות מותקנות אצלכם:

-   `speech_recognition`
-   `pydub`

ניתן להתקין אותן באמצעות pip:

```bash
pip install SpeechRecognition pydub
```

## תיאור הקוד

הסקריפט מבצע את השלבים הבאים:

1.  **ייבוא ספריות:** מייבא את הספריות הנדרשות: `speech_recognition` לזיהוי דיבור ו-`pydub` להמרת פורמט אודיו.
2.  **אתחול מזהה:** יוצר מופע של המחלקה `Recognizer` מהספרייה `speech_recognition`.
3.  **ציון קובץ האודיו:** מגדיר את הנתיב לקובץ האודיו (`audiobook.wav`).
4.  **המרת פורמט אודיו (אופציונלי):** כולל קוד מוער (מצוין כהערה) להמרת קבצי MP3 לפורמט WAV באמצעות `pydub`. שלב זה שימושי אם קובץ האודיו שלכם בפורמט MP3, שכן `speech_recognition` עשויה לעבוד טוב יותר עם WAV.
5.  **קריאת נתוני האודיו:** פותח את קובץ האודיו באמצעות `sr.AudioFile` וקורא את נתוני האודיו באמצעות `recognizer.record`.
6.  **ביצוע זיהוי הדיבור:** משתמש במתודה `recognizer.recognize_google` כדי להמיר את נתוני האודיו לטקסט באמצעות ה-Google Web Speech API.
7.  **טיפול בשגיאות:** כולל בלוקי `try...except` לטיפול בשגיאות פוטנציאליות, כגון:
    -   `sr.UnknownValueError`: אם האודיו לא ניתן לזיהוי (קול לא ברור).
    -   `sr.RequestError`: אם אירעה בעיה עם בקשת ה-API.
8.  **הדפסת התוצאה:** מדפיס את הטקסט שחולץ או הודעת שגיאה לקונסולה.

## שימוש

1.  שמרו את קוד ה-Python בקובץ בשם `audio_to_text.py`.
2.  ודאו שקיים קובץ אודיו בשם `audiobook.wav` (או שנו את המשתנה `audio_file` לנתיב לקובץ האודיו שלכם).
3.  הפעילו את הסקריפט:

    ```bash
    python audio_to_text.py
    ```

    הסקריפט ינסה להמיר את האודיו לטקסט ולהדפיס את התוצאה. אם קובץ האודיו שלכם אינו נקרא `audiobook.wav`, עדכנו את המשתנה `audio_file` בסקריפט בהתאם. אם אתם עובדים עם MP3, הסירו את הסימון (הערה) מהשורה `AudioSegment.from_mp3`.

## הערות

-   ל-Google Web Speech API יש מגבלות שימוש. לשימוש אינטנסיבי, שקלו להשתמש במפתח API או בשירותי זיהוי דיבור אחרים.
-   דיוק זיהוי הדיבור תלוי באיכות הסאונד ובבהירות הדיבור.
