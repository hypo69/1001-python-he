## כללי משחק אונו:

**מטרת המשחק:** להיות השחקן הראשון שייפטר מכל הקלפים שלו בכל סיבוב ולאסוף 500 נקודות.

**הכנה למשחק:**

*   ערבבו את חפיסת הקלפים.
*   חלקו 7 קלפים לכל שחקן.
*   הניחו את החפיסה הנותרת במרכז השולחן כשגבה למעלה.
*   חשפו את הקלף העליון מהחפיסה והניחו אותו ליד החפיסה כשפניו כלפי מעלה. קלף זה יהיה קלף ההתחלה.

**מהלך המשחק:**

*   שחקנים משחקים בתורות בכיוון השעון.
*   כל שחקן חייב לשחק קלף התואם בצבע או במספר (נומינל) את הקלף האחרון ששוחק.
*   **קלפים מיוחדים:**
    *   **דילוג:** השחקן הבא מדלג על תורו.
    *   **היפוך כיוון:** כיוון המשחק משתנה לכיוון הנגדי.
    *   **קח שניים:** השחקן הבא לוקח שני קלפים מהקופה ומדלג על תורו.
    *   **החלף צבע:** השחקן בוחר כל צבע, והשחקן הבא חייב לשחק קלף בצבע זה או לקחת שני קלפים.
    *   **החלף צבע וקח ארבעה:** השחקן בוחר כל צבע, השחקן הבא לוקח ארבעה קלפים מהקופה וחייב לשחק קלף בצבע זה.
*   **קלף "אונו":** כאשר נותר בידי שחקן קלף אחד, עליו לצעוק "אונו!". אם מישהו מבחין ששחקן שכח לצעוק "אונו!" ולא אמר זאת לפני שהשחקן הבא החל את תורו, השחקן ששכח חייב לקחת שני קלפים כקנס.

**סיום הסיבוב:**

*   השחקן הראשון שנפטר מכל קלפיו מנצח בסיבוב.
*   שאר השחקנים מקבלים נקודות בהתאם למספרים על הקלפים שנותרו בידיהם.
*   כל הקלפים נאספים לערימה, מעורבבים, וסיבוב חדש מתחיל.

**ניצחון:**

*   השחקן הראשון שאסף 500 נקודות מנצח במשחק.

**כללים נוספים:**

*   **כללי "בית":** בחלק מהקבוצות עשויים להיות כללים נוספים משלהן, ההופכים את המשחק למעניין אף יותר.
*   **קנסות:** אם שחקן שיחק קלף שגוי או שכח לצעוק "אונו!", הוא עלול להיענש בקלפים נוספים.

**עצות:**

*   עקבו אחר קלפי השחקנים האחרים כדי לחזות את מהלכיהם.
*   השתמשו בקלפים המיוחדים כדי להפריע ליריבים או להשיג יתרון.
*   אל תשכחו לצעוק "אונו!" כדי להימנע מקנסות.
