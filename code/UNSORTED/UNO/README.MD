## כללי המשחק אונו:

**מטרת המשחק:** להיות השחקן הראשון להיפטר מכל הקלפים שברשותו בכל סבב ולצבור 500 נקודות.

**הכנה למשחק:**

* טרפו את חבילת הקלפים.
* חלקו לכל שחקן 7 קלפים.
* הניחו את הקופה הנותרת במרכז השולחן עם הפנים כלפי מטה.
* הפכו את הקלף העליון מהקופה והניחו אותו לצד הקופה עם הפנים כלפי מעלה. קלף זה יהיה קלף הפתיחה.

**מהלך המשחק:**

* שחקנים משחקים בתורות בכיוון השעון.
* כל שחקן חייב לשחק קלף התואם בצבע או בערך את הקלף האחרון ששוחק.
* **קלפים מיוחדים:**
    * **דילוג תור:** השחקן הבא מדלג על תורו.
    * **שינוי כיוון:** כיוון המשחק מתחלף לכיוון ההפוך.
    * **קח שניים:** השחקן הבא לוקח שני קלפים מהקופה ומדלג על תורו.
    * **קביעת צבע:** השחקן בוחר צבע כלשהו, והשחקן הבא חייב לשחק קלף בצבע זה או לקחת שני קלפים.
    * **קביעת צבע וקח ארבעה:** השחקן בוחר צבע כלשהו, השחקן הבא לוקח ארבעה קלפים מהקופה וחייב לשחק קלף בצבע זה.
* **קלף "אונו":** כאשר נשאר לשחקן קלף אחד, עליו לקרוא "אונו!". אם מישהו מבחין שהשחקן שכח לקרוא "אונו!" ולא אמר זאת לפני שהשחקן הבא החל את תורו, על השחקן ששכח לקחת שני קלפים כקנס.

**סיום סבב:**

* השחקן הראשון שנפטר מכל קלפיו מנצח בסבב.
* יתר השחקנים צוברים נקודות בהתאם למספרים שעל הקלפים שנותרו ברשותם.
* כל הקלפים נאספים לערימה, נטרפים, וסבב חדש מתחיל.

**ניצחון:**

* השחקן הראשון שצבר 500 נקודות מנצח במשחק.

**כללים נוספים:**

* **"כללי הבית":** בחברות מסוימות עשויים להיות כללים נוספים משלהן, ההופכים את המשחק למעניין עוד יותר.
* **קנסות:** אם שחקן שיחק קלף שגוי או שכח לקרוא "אונו!", הוא עשוי להיענש בקלפים נוספים.

**עצות:**

* עקבו אחר קלפיהם של שחקנים אחרים על מנת לחזות את מהלכיהם.
* השתמשו בקלפים מיוחדים כדי להפריע ליריבים או להשיג יתרון.
* אל תשכחו לקרוא "אונו!" כדי להימנע מקנסות.
