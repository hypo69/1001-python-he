# HOMOPHONES

**הומופונים** – אלו מילים הנשמעות באופן זהה, אך בעלות משמעות שונה, ובדרך כלל גם כתיב שונה. תופעה זו שכיחה למדי בשפה האנגלית ועלולה לגרום לקשיים מסוימים ללומדיה.

**לדוגמה:**

*   **two, too, to:** מילים אלו נהגות באופן זהה, אך בעלות משמעויות שונות: שניים, גם כן, ל- / אל-.
*   **know, no:** לדעת, לא.
*   **here, hear:** כאן, לשמוע.
