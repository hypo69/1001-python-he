# **Amazin** (מחולל מבוכים)

#### תיאור
המשחק הוא מחולל מבוכים שיוצר בכל פעם מבוך ייחודי עם מסלול אחד בלבד. השחקן מגדיר את מימדי המבוך, והתוכנה בונה אותו בהתאם לפרמטרים שסופקו.

- **תכונות:**
  - השחקן מזין את רוחב וגובה המבוך.
  - התוכנה מבטיחה קיומו של מסלול יחיד דרך המבוך.
  - המבוך מוצג על המסך.

---

### הנחיות שלב אחר שלב ליישום:

#### 1. **אתחול המשחק**
   - להציע לשחקן להזין את מימדי המבוך (רוחב וגובה).
   - לוודא שהמימדים שהוזנו תקינים (לדוגמה, גדולים מאחד).

#### 2. **לוגיקת ליבת יצירת המבוך**
   - ליצור מטריצה בגודל שסופק, שתייצג רשת תאים.
   - להשתמש באלגוריתם ליצירת מבוכים, לדוגמה, *אלגוריתם חיפוש רקורסיבי לעומק (DFS)*:
     - להתחיל מתא התחלתי אקראי.
     - להתקדם לתאים סמוכים, תוך הסרת הקירות בין התא הנוכחי לתא הבא.
     - אם כל השכנים כבר בוקרו, לחזור לתא הקודם ולהמשיך.
     - לסיים את התהליך כאשר כל התאים בוקרו.
   - להבטיח קיומו של מסלול יחיד דרך המבוך.

#### 3. **הצגת המבוך**
   - להשתמש בסימנים לצורך הצגה:
     - `+`, `-`, `|` עבור קירות.
     - רווחים עבור מעברים.
   - להדפיס את המבוך המוגמר כטקסט.

#### 4. **פונקציות נוספות**
   - אפשרות להגדיר גודל מוגדר מראש (לדוגמה, 10x10) אם המשתמש הזין נתונים שגויים.
   - התרעה על מימדים גדולים מדי של המבוך כדי למנוע עומס יתר על הזיכרון.

### דוגמת הפעלת התוכנית

1. **תחילת עבודה**:
   ```
   הזן רוחב וגובה למבוך:
   > 10 8
   ```
