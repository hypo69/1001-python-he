# מחשבון קוד צבע של נגד

## תיאור

סקריפט Python זה (`color_code_resistor.py`) מיועד לחישוב התנגדות של נגד על פי הסימון הצבעוני שלו.
המשתמש מזין שלושה צבעים, המתאימים לפסים על הנגד, והסקריפט מציג את ערך ההתנגדות באוהם ובקילו-אוהם.

## שימוש

1.  **שכפל את המאגר** (אם נחוץ).
2.  **הפעל את הסקריפט `color_code_resistor.py`:**
    ```bash
    python color_code_resistor.py
    ```
3.  **עקוב אחר ההוראות במסוף:**
    הסקריפט יבקש ממך להזין בזה אחר זה שלושה צבעים, המתאימים לפסים על הנגד.
4.  **עיין בתוצאה:**
    הסקריפט יציג את ערך ההתנגדות באוהם ובקילו-אוהם.

## דוגמה לשימוש

```
Enter the 1st color: red
Enter the 2nd color: violet
Enter the 3rd color: brown

The Resistor Value Is:
270 and in Kiloohm: 0.27kΩ
```

## אופן הפעולה

הסקריפט משתמש ברשימת צבעים כדי לקבוע את הערכים המספריים המתאימים לכל פס צבע. שני הפסים הראשונים קובעים את הספרות המשמעותיות, והפס השלישי משמש כמכפיל.

## רשימת הצבעים בשימוש

-   `black` - 0
-   `brown` - 1
-   `red` - 2
-   `orange` - 3
-   `yellow` - 4
-   `green` - 5
-   `blue` - 6
-   `violet` - 7
-   `grey` - 8
-   `white` - 9
