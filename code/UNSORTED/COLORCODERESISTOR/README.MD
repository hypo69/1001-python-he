# מחשבון צבעים לנגדים (Resistor Color Code Calculator)

## תיאור

סקריפט פייתון זה (`color_code_resistor.py`) נועד לחשב את התנגדותו של נגד בהתאם לסימון הצבעים שלו.
המשתמש מזין שלושה צבעים, המתאימים לפסים על הנגד, והסקריפט מציג את ערך ההתנגדות באוהם ובקילואוהם.

## שימוש

1.  **שכפול המאגר (repository)** (במידת הצורך).
2.  **הפעלת הסקריפט `color_code_resistor.py`:**
    ```bash
    python color_code_resistor.py
    ```
3.  **מעקב אחר ההנחיות במסוף:**
    הסקריפט יבקש ממך להזין בזה אחר זה שלושה צבעים, המתאימים לפסים על הנגד.
4.  **צפייה בתוצאה:**
    הסקריפט יציג את ערך ההתנגדות באוהם ובקילואוהם.

## דוגמת הפעלה

```
Enter the 1st color: red
Enter the 2nd color: violet
Enter the 3rd color: brown

The Resistor Value Is:
270 and in Kiloohm: 0.27kΩ
```

## אופן הפעולה

הסקריפט משתמש ברשימת צבעים כדי לקבוע את הערכים המספריים המתאימים לכל פס צבע. שני הפסים הראשונים קובעים את הספרות המשמעותיות, והפס השלישי הוא מכפיל.

## רשימת הצבעים המשמשים

-   `black` - 0
-   `brown` - 1
-   `red` - 2
-   `orange` - 3
-   `yellow` - 4
-   `green` - 5
-   `blue` - 6
-   `violet` - 7
-   `grey` - 8
-   `white` - 9
