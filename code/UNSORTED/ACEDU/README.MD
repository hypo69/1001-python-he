# **Acey-Ducey משחק קלפים**

## תיאור
זוהי סימולציה של משחק הקלפים Acey-Ducey. שחקן מבצע הימורים על בסיס ההסתברות שהקלף הבא שיפתח יהיה בעל ערך הנמצא בין ערכיהם של שני קלפים שכבר נפתחו.

- **הון התחלתי:** השחקן מתחיל עם 100 דולר.
- **חוקי המשחק:**
  1. המחשב מניח שני קלפים.
  2. השחקן רשאי להחליט אם לבצע הימור או לא.
  3. אם בוצע הימור, קלף שלישי נפתח.
  4. אם ערכו של הקלף השלישי נמצא בין ערכיהם של שני הקלפים הראשונים, השחקן זוכה בהימור. אחרת, ההימור מפסיד.
- המשחק מסתיים כאשר השחקן מאבד את כל הונו או כאשר הוא מסיים אותו ידנית.

## כיצד הקוד פועל
---

### **1. ייבוא ספרייה**
```python
import random
```
- **random** – ספרייה ליצירת מספרים אקראיים. היא משמשת לערבוב חבילת הקלפים ובחירת קלפים אקראיים.

---

### **2. יצירת חבילת קלפים**
```python
def create_deck():
    ranks = list(range(2, 15))  # קלפים מ-2 עד 14 (אס = 14)
    deck = ranks * 4  # 4 סדרות (מאסט)
    random.shuffle(deck)
    return deck
```
- **ranks** – רשימה של ערכי קלפים מ-2 עד 14 (אס = 14).
- **deck** – נוצר על ידי הכפלת רשימת ה-`ranks` ב-4, על מנת לקבל 4 סדרות (עלים, לבבות, יהלומים, תלתן).
- **random.shuffle(deck)** – מערבב את חבילת הקלפים.
- הפונקציה מחזירה את החבילה המעורבבת.

---

### **3. הצגת קלף בפורמט קריא**
```python
def card_name(value):
    if value == 11:
        return "נסיך"
    elif value == 12:
        return "מלכה"
    elif value == 13:
        return "מלך"
    elif value == 14:
        return "אס"
    else:
        return str(value)
```
- הפונקציה מקבלת ערך מספרי של קלף (מ-2 עד 14) ומחזירה את הייצוג הטקסטואלי שלו.
- לדוגמה, 11 → "נסיך", 14 → "אס".

---

### **4. לולאת המשחק הראשית**
```python
def play_acey_ducey():
    print("ברוכים הבאים למשחק Acey Ducey!")
    print("חוקים: בצעו הימור, נחשו האם הקלף הבא יהיה בין שני הקלפים שנחשפו.")
    print("אם הקלף שווה לאחד מהקלפים שנחשפו או שהוא אס, הפסדתם.")
    print("הקלידו '0' כדי לדלג על תור.\n")

    money = 100  # הון התחלתי של השחקן
    deck = create_deck() # נוצרת חבילת קלפים באמצעות הפונקציה create_deck()
```
- **money** – יתרת המזומנים ההתחלתית של השחקן (100 דולר).
- **deck** – נוצרת חבילת קלפים באמצעות הפונקציה `create_deck()`.

---

### **5. חשיפת שני קלפים**
```python
while money > 0 and len(deck) >= 3:
    print(f"יתרתכם הנוכחית: ${money}")

    # חושפים שני קלפים
    card1 = deck.pop()
    card2 = deck.pop()
    while card1 == card2:  # אם הקלפים זהים, לוקחים חדשים
        deck.insert(0, card1)
        deck.insert(0, card2)
        card1 = deck.pop()
        card2 = deck.pop()

    print(f"קלף ראשון: {card_name(card1)}")
    print(f"קלף שני: {card_name(card2)}")
```
- **deck.pop()** – שולף את הקלף האחרון מהחבילה.
- **while card1 == card2** – בודק האם הקלפים שנחשפו זהים. אם הם זהים, מחזיר אותם לתחילת החבילה ולוקח חדשים.
- **card_name(card1)** – ממיר את הערך המספרי של הקלף לייצוג טקסטואלי.

---

### **6. קביעת הטווח**
```python
low_card = min(card1, card2)
high_card = max(card1, card2)
```
- **low_card** – הערך המינימלי מבין שני הקלפים.
- **high_card** – הערך המקסימלי מבין שני הקלפים.

---

### **7. ביצוע הימור או דילוג על תור**
```python
try:
    bet = int(input(f"בצעו הימור (מ-0 עד {money}) או הקלידו '0' לדילוג: "))
    if bet < 0 or bet > money:
        print("הימור לא תקין. נסו שוב.")
        continue
    if bet == 0:
        print("דילגתם על תור.\n")
        continue  # מדלגים על תור
except ValueError:
    print("אנא הזינו מספר.")
    continue
```
- **input()** – מבקש מהשחקן להזין הימור.
- **try-except** – מטפל בשגיאות קלט אם השחקן הזין קלט שאינו מספר.
- **if bet < 0 or bet > money** – בודק שההימור חוקי (לא שלילי ואינו עולה על היתרה).
- **if bet == 0** – אם ההימור הוא 0, השחקן מדלג על התור.

---

### **8. משיכת הקלף הבא**
```python
next_card = deck.pop()
print(f"הקלף הבא: {card_name(next_card)}")
```
- **next_card** – שולף את הקלף הבא מהחבילה.
- **card_name(next_card)** – ממיר את ערך הקלף לייצוג טקסטואלי.

---

### **9. בדיקת התוצאה**
```python
if next_card == card1 or next_card == card2 or next_card == 14:
    print("הפסדתם!")
    money -= bet
elif low_card < next_card < high_card:
    print("זכיתם!")
    money += bet
else:
    print("הפסדתם!")
    money -= bet
```
- **if next_card == card1 or next_card == card2 or next_card == 14** – אם הקלף הבא שווה לאחד משני הקלפים שנחשפו או שהוא אס (14), השחקן מפסיד.
- **elif low_card < next_card < high_card** – אם הקלף הבא נמצא בטווח שבין שני הקלפים שנחשפו (לא כולל אותם), השחקן זוכה.
- **else** – בכל המקרים האחרים, השחקן מפסיד.

---

### **10. סיום המשחק**
```python
if money <= 0:
    print("נגמר לכם הכסף. המשחק הסתיים.")
else:
    print(f"המשחק הסתיים. יתרתכם הסופית: ${money}")
```
- אם לשחקן נגמר הכסף, המשחק מסתיים.
- אם חבילת הקלפים נגמרה, המשחק גם כן מסתיים.
