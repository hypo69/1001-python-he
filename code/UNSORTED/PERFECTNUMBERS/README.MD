# מודול לחיפוש מספרים מושלמים

מודול זה, `perfect_numbers.py`, מספק פונקציות לקביעה ולחיפוש של מספרים מושלמים.

## מהו מספר מושלם?

מספר מושלם הוא מספר שלם חיובי השווה לסכום מחלקיו החיוביים העצמיים (למעט המספר עצמו).

דוגמאות:

- 6 = 1 + 2 + 3
- 28 = 1 + 2 + 4 + 7 + 14
- 496
- 8128

## פונקציות

### `sum_of_divisors(n)`

מחשבת את סכום כל המחלקים העצמיים של המספר `n`.

**ארגומנטים:**

- `n`: מספר שלם.

**מחזירה:**

- סכום המחלקים העצמיים של המספר `n`.

### `is_perfect_number(n)`

בודקת האם המספר `n` הוא מספר מושלם.

**ארגומנטים:**

- `n`: מספר שלם.

**מחזירה:**

- `True` אם המספר `n` הוא מושלם, אחרת `False`.

### `find_perfect_numbers(limit)`

מוצאת את כל המספרים המושלמים בטווח מ-1 ועד `limit` (כולל).

**ארגומנטים:**

- `limit`: הגבול העליון של הטווח.

**מחזירה:**

- רשימה של כל המספרים המושלמים בטווח הנתון.

## שימוש

לשימוש במודול:

1.  שמרו את קוד ה-Python בקובץ `perfect_numbers.py`.
2.  ייבאו את הפונקציות לפרויקט שלכם:

    ```python
    from perfect_numbers import find_perfect_numbers, is_perfect_number

    limit = 10000
    perfect_numbers = find_perfect_numbers(limit)
    print(f"Совершенные числа до {limit}: {perfect_numbers}")


    number_to_check = 28
    if is_perfect_number(number_to_check):
       print(f"{number_to_check} является совершенным числом")
    else:
        print(f"{number_to_check} не является совершенным числом")
    ```
    דוגמת קריאה תודפס בעת הפעלת הסקריפט `perfect_numbers.py`.
## דוגמת פלט:
