## ארקנואיד

### תיאור

משחק ארקייד קלאסי שבו השחקן שולט בפלטפורמה אופקית בחלק התחתון של המסך, ומקפיץ כדור, שבתורו מקפץ מהקירות ומהפלטפורמה. מטרת המשחק היא למנוע מהכדור ליפול מעבר לקצה התחתון של המסך.

### כללים

1.  הכדור נע על פני המסך, מקפץ מהקירות ומהפלטפורמה.
2.  השחקן מזיז את הפלטפורמה שמאלה וימינה.
3.  אם הכדור נוגע בפלטפורמה, הוא מקפץ בחזרה.
4.  אם הכדור נופל מעבר לקצה התחתון של המסך, המשחק מסתיים.
