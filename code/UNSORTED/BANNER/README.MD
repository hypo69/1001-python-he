# משחק Banner: יצירת באנר טקסטואלי

&nbsp;&nbsp;&nbsp;&nbsp;
התוכנה יוצרת באנר טקסטואלי מהטקסט שהוזן על ידי המשתמש. היא מברכת את המשתמש, מבקשת את הטקסט ומציגה אותו בצורה של באנר מעוצב.
הקוד מדגים עקרונות בסיסיים של עבודה עם פונקציות, קלט-פלט ואופרטורים מותנים ב-Python.

---

## **כיצד פועלת התוכנה**

התוכנה מורכבת משני חלקים עיקריים:
1.  **הפונקציה `create_banner(text)`** – אחראית על יצירת והצגת הבאנר הטקסטואלי.
2.  **החלק הראשי של התוכנה** – אינטראקציה עם המשתמש: ברכה, בקשת טקסט ובדיקת קלט.

---

## **קוד התוכנה**

```python
# משחק Banner: יצירת באנר טקסטואלי

# פונקציה ליצירת באנר
def create_banner(text):
    """
    יוצרת באנר טקסטואלי מהטקסט המוזן.
    :param text: מחרוזת שיש להפוך לבאנר.
    :return: None
    """
    # מגדירים את רוחב הבאנר (אורך הטקסט + 4 תווים למסגרת)
    banner_width = len(text) + 4

    # מציגים את הגבול העליון של הבאנר
    print("*" * banner_width)

    # מציגים את הטקסט עם המסגרת
    print(f"* {text} *")

    # מציגים את הגבול התחתון של הבאנר
    print("*" * banner_width)

# החלק הראשי של התוכנה
if __name__ == "__main__":
    # ברכת המשתמש
    print("ברוכים הבאים למשחק Banner!")
    print("הזינו טקסט, ואני אצור עבורכם באנר טקסטואלי.")

    # מבקשים מהמשתמש טקסט
    user_text = input("הזינו טקסט: ")

    # בודקים שהטקסט אינו ריק
    if user_text.strip() == "":
        print("לא הזנתם טקסט. אנא נסו שוב.")
    else:
        # יוצרים ומציגים את הבאנר
        print("\nהבאנר שלכם מוכן:")
        create_banner(user_text)
```

---

## **תיאור פעולת הקוד**

### **1. הפונקציה `create_banner(text)`**
-   **מקבלת:** מחרוזת `text` – הטקסט שיש להפוך לבאנר.
-   **מבצעת:**
    -   מחשבת את רוחב הבאנר, בתוספת 4 תווים למסגרת (`*` ורווחים).
    -   מציגה את הגבולות העליון והתחתון של הבאנר באמצעות התו `*`.
    -   מציגה את הטקסט מוקף במסגרת.

### **2. החלק הראשי של התוכנה**
-   **מברך** את המשתמש ומסביר מה התוכנה עושה.
-   **מבקש** מהמשתמש טקסט ליצירת הבאנר.
-   **בודק** שהטקסט אינו ריק:
    -   אם המשתמש הזין מחרוזת ריקה, התוכנה מדווחת על שגיאה ומציעה לנסות שוב.
    -   אם הוזן טקסט, התוכנה מציגה את הבאנר שנוצר.
