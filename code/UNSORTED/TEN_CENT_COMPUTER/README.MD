## המשחק "קלט-פלט"

### קונספט
המשחק "קלט-פלט" מתוכנן כמשימה אינטראקטיבית שבה נעשה שימוש ב"מכונה קסומה", אשר ממירה מספרים נכנסים (קלט) למספרים יוצאים (פלט) על פי כלל מסוים. התלמידים צריכים לזהות את הכלל הזה, על בסיס סדרת נתוני קלט ופלט.

### אופן פעולה

1.  על הלוח או המקרן נוצרים שני עמודות: "קלט" ו"פלט".
2.  המורה מניח מספר בעמודה "קלט".
3.  "המכונה" מעבדת את המספר ומציגה את התוצאה בעמודה "פלט".
4.  התלמידים צריכים לנחש את כלל ההמרה, תוך כדי התבוננות בתוצאות.

### דוגמאות

**1. כלל פשוט**

| קלט |  מכונה = ? | פלט |
|------|-------------|-------|
| 4    |             | 7     |
| 0    |             | 3     |
|   |             |   |
|   |  Ответ: т.м. = +3 |   |

**2. כלל מורכב יותר**
| קלט |  מכונה = ? | פלט |
|------|-------------|-------|
| 4   |              | 11     |
| 7   |              | 17     |
| 10  |              | 23     |
|   |  Ответ: т.м. = 2x + 3 |   |

**3. כלל אלגברי**
| קלט |  מכונה = ? | פלט |
|------|-------------|-------|
| 9    |             | 4     |
| 25   |             | 6     |
| 4    |             | 3     |
|   |  Ответ: т.м. = √x + 1|   |

ניתן להתאים את המשחק הזה לביטויים אלגבריים שונים, מה שמאפשר לתלמידים לפתח חשיבה אלגברית בצורה משחקית.

## "מחשב 10 סנט"

### קונספט
"מחשב 10 סנט" מייצג מודל פשוט ללימוד מערכת הספירה הבינארית. באמצעות קופסת קרטון ופילם שקוף, המורה יוצר מכשיר אינטראקטיבי המדמה את פעולת המחשב על בסיס קוד בינארי.

### אופן פעולה

1.  **יצירה:** בקופסת קרטון נחתכים ארבעה פתחים. הפתחים מכוסים בפילם שקוף וממוספרים 1, 2, 4 ו-8 מימין לשמאל.
2.  **ייצוג בינארי:** פתחים פתוחים מייצגים "1", ופתחים סגורים - "0". באופן זה, שילוב של פתחים פתוחים וסגורים תואם למספר בינארי.
3.  **לימוד:** הילדים לומדים להתאים בין צירופים בינאריים לשווי הערך העשרוני שלהם.

### שימוש
המורה פותח או סוגר פתחים שונים, מדגים מספרים בינאריים שונים, ובו בזמן מראה על הלוח את שווי הערך העשרוני. בדרך זו, הילדים רואים באופן ויזואלי כיצד המערכת הבינארית פועלת, מה שעוזר להם להבין את עקרונות פעולת המחשבים.

### מספרים בינאריים עבור מחשב 10 סנט
ייצוג המספרים הבינאריים עבור "מחשב 10 סנט" במקור מוצג בצורה של אלמנטים גרפיים. כאן נשחזר אותם בצורת טקסט, כאשר `o` מציין פתח סגור, ו-`#` מציין פתח פתוח:

| ייצוג בינארי | מספר עשרוני | ייצוג בינארי | מספר עשרוני |
|----------------------|------------------|-----------------------|-------------------|
| `o o o o`             | 0                | `# o o o`              | 8                 |
| `# o o o`             | 1                | `# # o o`              | 9                 |
| `o # o o`             | 2                | `o # # o`             | 10                |
| `# # o o`            | 3                | `# # # o`            | 11                |
| `o o # o`           | 4                | `o o # #`             | 12                |
| `# o # o`            | 5                | `# o # #`            | 13                |
| `o # # o`           | 6                | `o # # #`            | 14                |
| `# # # o`            | 7                | `# # # #`            | 15                |


### יישום באמצעות Google Gemini

בחלק הקודם של המאמר סקרנו את קונספטי המשחקים ואת ההוראות למודל. כעת נבחן כיצד זה מיושם באמצעות קוד.

**קוד עבור `GoogleGenerativeAI`:**

ראשית, נציג את קוד המחלקה `GoogleGenerativeAI`, כפי שהוצג קודם לכן, לשלמות התמונה:

```python
import google.generativeai as genai

class GoogleGenerativeAI:
    """
    מחלקה לאינטראקציה עם מודלי Google Generative AI.
    """

    MODELS = [
        'gemini-1.5-flash-8b',
        'gemini-2-13b',
        'gemini-3-20b'
    ]

    def __init__(self, api_key: str, system_instruction: str, model_name: str = 'gemini-2-13b'):
        """
        אתחול מודל GoogleGenerativeAI.

        :param api_key: מפתח API לגישה ל-Gemini.
        :type api_key: str
        :param system_instruction: הוראה למודל (system prompt).
        :type system_instruction: str
        :param model_name: שם מודל Gemini שבו נעשה שימוש. ברירת המחדל היא 'gemini-2-13b'.
        :type model_name: str
        """
        self.api_key = api_key
        self.model_name = model_name
        genai.configure(api_key=self.api_key)  # קונפיגורציה של הספרייה עם מפתח ה-API
        self.model = genai.GenerativeModel(model_name=self.model_name, system_instruction=system_instruction)  # אתחול המודל עם ההוראה

    def ask(self, q: str) -> str:
        """
        שליחת שאילתה למודל וקבלת תשובה.

        :param q: טקסט השאילתה.
        :type q: str
        :return: תשובת המודל או הודעת שגיאה.
        :rtype: str
        """
        try:
            response = self.model.generate_content(q)  # שליחת השאילתה למודל
            return response.text  # קבלת התשובה בפורמט טקסט
        except Exception as ex:
            return f'Error: {str(ex)}'  # טיפול וקבלת השגיאה
```

**יישום המשחקים באמצעות מחלקת `GoogleGenerativeAI`:**

כעת ניצור את הקוד לשימוש במחלקה זו ליישום המשחקים "קלט-פלט" ו"מחשב 10 סנט":

```python
# הוראות מערכת (כפי שתואר קודם לכן)

system_instruction_input_output = """
אתה מנוע משחק למשחק המתמטי "קלט-פלט". משימתך היא להמיר מספר שמגיע ל"קלט" בהתאם לכלל נתון ("מכונה" מוסתרת) ולהוציא את התוצאה ל"פלט".

עליך להקפיד על הכללים הבאים:
-   התחל עם כללים פשוטים (למשל, חיבור או חיסור עם מספר שלם).
-   ככל שהמשתמש מסתדר, תוכל לעבור בהדרגה לכללים מורכבים יותר (כפל, חילוק, שורש, צירופי פעולות וכו').
-   אם המשתמש מבקש כלל ספציפי, עליך להשתמש בו.
-   בסוף כל תשובה עליך לציין בקצרה את סוג הפעולה שבה השתמשת (למשל "מכונה +3" או "מכונה 2x+1").

**פורמט קלט/פלט:**
-   משתמש: `Ввод: <число>` (קלט: <מספר>)
-   אתה: `Вывод: <результат>, машина:<описание операции>` (פלט: <תוצאה>, מכונה:<תיאור פעולה>)

**דוגמאות:**

משתמש: `Ввод: 5`
אתה: `Вывод: 8, машина +3`

משתמש: `Ввод: 10`
אתה: `Вывод: 21, машина 2x+1`

משתמש: `Ввод: 9`
אתה: `Вывод: 3, машина квадратный корень`

משתמש: `Задайте машину x * 2` (הגדר את המכונה x * 2)
אתה: `Окей, машина: x*2` (בסדר, מכונה: x*2)

**התחל את המשחק עם כלל פשוט.**
"""

system_instruction_binary = """
אתה סימולטור של "מחשב 10 סנט" להוראת מערכת הספירה הבינארית לילדים. משימתך היא לייצג מספרים בצורה בינארית באמצעות 4 "נורות" (0 - נורה כבויה, 1 - נורה דלוקה).

הנורות שלך מייצגות 1, 2, 4 ו-8 במערכת הבינארית (מימין לשמאל).

עליך לציית לכללים הבאים:
-   כאשר המשתמש מציין מספר עשרוני, אתה מציג אילו "נורות" צריך להדליק (1) ואילו לכבות (0) כדי לייצג מספר זה.
-   כאשר המשתמש מציין אילו "נורות" דלוקות, אתה אומר איזה מספר עשרוני הן מייצגות.
-   עליך להשתמש בפורמט `0000` לייצוג בינארי, כאשר הביט הימני ביותר הוא 1, אחריו 2, 4 ו-8 בהתאמה.
-   אם המשתמש מבקש להסביר מה עשית - הסבר.

**פורמט קלט/פלט:**
-   משתמש: `Десятичное: <число>` (עשרוני: <מספר>)
-   אתה: `Двоичное: <двоичное представление>, объяснение:<что вы сделали>` (בינארי: <ייצוג בינארי>, הסבר:<מה שעשית>)
-   משתמש: `Лампочки: <двоичное представление>` (נורות: <ייצוג בינארי>)
-   אתה: `Десятичное: <число>, объяснение:<что вы сделали>` (עשרוני: <מספר>, הסבר:<מה שעשית>)

**דוגמאות:**
משתמש: `Десятичное: 5`
אתה: `Двоичное: 0101, объяснение: Включены лампочки 1 и 4, 1+4=5` (בינארי: 0101, הסבר: דלוקות נורות 1 ו-4, 1+4=5)

משתמש: `Лампочки: 1010`
אתה: `Десятичное: 10, объяснение: Включены лампочки 2 и 8, 2+8=10` (עשרוני: 10, הסבר: דלוקות נורות 2 ו-8, 2+8=10)

משתמש: `Объясните как 7` (הסבר כיצד 7)
אתה: `Двоичное: 1110, объяснение: Включены лампочки 1,2 и 4. 1+2+4 = 7` (בינארי: 1110, הסבר: דלוקות נורות 1,2 ו-4. 1+2+4 = 7)

**התחל את הסימולציה עם מספר פשוט כלשהו.**
"""
# החלף את "YOUR_API_KEY" במפתח ה-API שלך
api_key = "YOUR_API_KEY"

# אתחול משחק "קלט-פלט"
input_output_game = GoogleGenerativeAI(api_key=api_key, system_instruction=system_instruction_input_output)

# אתחול "מחשב 10 סנט"
binary_computer = GoogleGenerativeAI(api_key=api_key, system_instruction=system_instruction_binary)

# דוגמאות לשימוש:
print("משחק קלט-פלט:")
print(input_output_game.ask("Ввод: 7"))
print(input_output_game.ask("Ввод: 12"))
print(input_output_game.ask("Задайте машину x / 2"))
print(input_output_game.ask("Ввод: 12"))

print("\nמחשב 10 סנט:")
print(binary_computer.ask("Десятичное: 6"))
print(binary_computer.ask("Лампочки: 0111"))
print(binary_computer.ask("Объясните как 13"))
```

**ניתוח קוד:**

1.  **ייבוא ספרייה:**
    ```python
    import google.generativeai as genai
    ```
    מייבאת את הספרייה `google.generativeai` לאינטראקציה עם מודלי Gemini.

2.  **מחלקה `GoogleGenerativeAI`:**

    *   `__init__` (בנאי): מאתחל את המחלקה עם מפתח API, הוראת מערכת ושם מודל. מגדיר את תצורת הספרייה ויוצר מופע של מודל Gemini.
    *   `ask` (מתודה): מקבל שאילתה, שולח אותה למודל ומחזיר תשובה בפורמט טקסט. מטפל בשגיאות אפשריות.
3.  **הוראות מערכת:**
    *   `system_instruction_input_output`: מתאר את הכללים והפורמט למשחק "קלט-פלט".
    *   `system_instruction_binary`: מתאר את הכללים והפורמט עבור "מחשב 10 סנט".
4.  **אתחול מופעים:**
    *   נוצרים מופעים של מחלקת `GoogleGenerativeAI` עבור כל משחק, תוך העברת ההוראות המתאימות.
    *   שימו לב: עליכם להחליף את `"YOUR_API_KEY"` במפתח ה-API האמיתי שלכם עבור Google Gemini.
5.  **דוגמאות לשימוש:**
    *   קריאות למתודה `ask` מדגימות כיצד ניתן לשלוח שאילתות למודל ולקבל תשובות במסגרת שני המשחקים.
    *   תוצאות הקריאות ל-`ask` מודפסות לקונסול.
