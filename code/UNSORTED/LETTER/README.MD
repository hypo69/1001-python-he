**משחק ניחוש אותיות (האלפבית הרוסי)**
**תיאור:**
המשחק "LETTER" דומה למשחק "נחש את המספר", אך במקום מספר, המחשב מגריל אות אקראית מהאלפבית הרוסי. משימתך היא לנחש את האות הזו, באמצעות רמזים שתקבל/י לאחר כל ניסיון. בדרך כלל, ניחוש האות דורש לא יותר מחמישה ניסיונות.

**כללי המשחק:**
1. המחשב בוחר אות אקראית מהאלפבית הרוסי (מ-А עד Я).
2. השחקן מזין את ניחושיו לגבי האות שהוגרלה.
3. לאחר כל ניסיון, המחשב מדווח האם האות שהוכנסה מופיעה באלפבית לפני או אחרי האות שהוגרלה.
4. המשחק נמשך עד שהשחקן מנחש נכונה את האות שהוגרלה.

**אלגוריתם:**
1. אתחול מונה הניסיונות ל-0.
2. יצירת אות אקראית מהאלפבית הרוסי (מ-А עד Я).
3. התחלת לולאה "כל עוד האות לא נוחשה":
    3.1. הגדלת מונה הניסיונות ב-1.
    3.2. בקשת קלט אות מהשחקן.
    3.3. אם האות שהוכנסה שווה לאות שהוגרלה, מעבר לשלב 4.
    3.4. אם האות שהוכנסה מופיעה באלפבית לפני האות שהוגרלה, הדפסת ההודעה "קודם באלפבית".
    3.5. אם האות שהוכנסה מופיעה באלפבית אחרי האות שהוגרלה, הדפסת ההודעה "מאוחר יותר באלפבית".
4. הדפסת ההודעה "ניחשת נכון ב-{מספר ניסיונות} ניסיונות!"
5. סיום המשחק.

**דוגמת משחק:**
- המחשב הגריל את האות **М**.
- השחקן מכניס את האות **Д**.
  המחשב: "קודם באלפבית".
- השחקן מכניס את האות **С**.
  המחשב: "מאוחר יותר באלפבית".
- השחקן מכניס את האות **М**.
  המחשב: "ניחשת נכון ב-3 ניסיונות!"

### קוד Python למשחק עם האלפבית הרוסי:

```python
import random

def letter_guessing_game():
    # יצירת אות אקראית מהאלפבית הרוסי
    target_letter = random.choice('АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ')
    attempts = 0

    print("ברוכים הבאים למשחק 'נחש את האות'!")
    print("נסו לנחש את האות שהגרלתי (А-Я).")

    while True:
        # בקשת קלט מהשחקן
        guess = input("אנא הכנס את האות שלך: ").strip().upper()

        # בדיקת תקינות הקלט
        if len(guess) != 1 or guess not in 'АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ':
            print("אנא הכנס אות בודדת ותקנית מהאלפבית הרוסי (А-Я).")
            continue

        attempts += 1

        # בדיקת האות שניחשו
        if guess == target_letter:
            print(f"ניחשת נכון ב-{attempts} ניסיונות!")
            break
        elif guess < target_letter:
            print("קודם באלפבית.")
        else:
            print("מאוחר יותר באלפבית.")

# Запуск игры
letter_guessing_game()
```

### אופן פעולת הקוד:
1. **יצירת אות אקראית:**
   נעשה שימוש ב-`random.choice` לבחירת אות אקראית מתוך המחרוזת המכילה את האותיות `'АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ'`.

2. **לולאת המשחק הראשית:**
   - השחקן מזין אות, אשר מומרת לאות רישית ונבדקת תקינותה.
   - אם האות הוזנה כהלכה, מונה הניסיונות גדל.
   - אם האות נוחשה, המשחק מסתיים בהודעת ניצחון.
   - אם האות לא נוחשה, מוצג רמז: "קודם באלפבית" (אם האות מופיעה לפני האות שהוגרלה באלפבית) או "מאוחר יותר באלפבית" (אם האות מופיעה אחרי האות שהוגרלה).

3. **סיום המשחק:**
   המשחק מסתיים כאשר השחקן מנחש את האות, ומוצג מספר הניסיונות.
