**שאלה 648.** מהי הבנת מילון (dict comprehension) בפייתון וכיצד היא משמשת? הצג דוגמה ליצירת מילון באמצעות הבנת מילון.

-   א. הבנת מילון משמשת ליצירת מספרים אקראיים המשמשים כמפתחות למילון.
-   ב. הבנת מילון משמשת ליצירת מילון שהמפתחות שלו הם ערכי רצף, והערכים הם האינדקסים שלהם.
-   ג. הבנת מילון היא תחביר מקוצר ליצירת מילונים המבוססים על אובייקטים ניתנים לאיטרציה. היא מאפשרת ליצור מילון באמצעות זוגות מפתח-ערך בסוגריים מסולסלים, ועל ידי איטרציה על אובייקט ניתן לאיטרציה קיים.
-   ד. הבנת מילון היא דרך ליצור מחלקה המבוססת על מילון.

**תשובה נכונה: ג**

**הסבר:**

הבנת מילון (dict comprehension) בפייתון היא דרך ליצור מילונים באמצעות תחביר קומפקטי, הדומה להבנת רשימה. היא מאפשרת ליצור מילונים המבוססים על אובייקטים ניתנים לאיטרציה, על ידי יישום ביטויים ליצירת מפתחות וערכים.

*   **מאפיינים עיקריים של הבנת מילון:**
    *   **תחביר מקוצר:** תמציתי יותר מאשר לולאות ליצירת מילונים.
    *   **אובייקט ניתן לאיטרציה:** יוצר מילון חדש המבוסס על כל אובייקט ניתן לאיטרציה.
    *   **זוגות מפתח-ערך:** יוצר מילון המשתמש בזוגות "מפתח-ערך".
    *   **ביטוי:** יכול להשתמש בביטויים ליצירת מפתחות וערכים.
    *   **תנאים (אופציונלי):** יכול להשתמש בביטויים תנאיים (`if`) לסינון אלמנטים.

*   **תחביר הבנת מילון:**
    *   `{key: value for item in iterable}`
        *   `key` - ביטוי ליצירת מפתח.
        *   `value` - ביטוי ליצירת ערך.
        *   `item` - האלמנט הנוכחי ב-`iterable`.
        *   `iterable` - אובייקט ניתן לאיטרציה.
    *   `{key: value for item in iterable if condition}` - עם תנאי.

**דוגמאות:**

```python
# דוגמה 1: יצירת מילון מרשימה באמצעות enumerate
import string

alphabet = list(string.ascii_lowercase) # יצירת רשימת אותיות
d = {val: idx for idx, val in enumerate(alphabet)} # יצירת מילון עם מפתחות - תווים, וערכים - אינדקסים
print(f"המילון שנוצר: {d}") # פלט המילון עם תווים ואינדקסים

# דוגמה 2: יצירת מילון עם ריבועי מספרים
numbers = [1, 2, 3, 4, 5]
squared_dict = {x: x**2 for x in numbers}
print(f"מילון עם ריבועי מספרים: {squared_dict}")   # פלט: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# דוגמה 3: מילון עם תנאי if
numbers = [1, 2, 3, 4, 5, 6]
even_dict = {x: x*2 for x in numbers if x % 2 == 0} # יצירת מילון שבו המפתחות הם מספרים זוגיים, והערכים הם המספרים הזוגיים מוכפלים ב-2
print(f"מילון עם מספרים זוגיים: {even_dict}") # פלט: {2: 4, 4: 8, 6: 12}

# דוגמה 4: מילון עם תנאי if else
numbers = [1, 2, 3, 4, 5, 6]
dict_if_else = {x: "even" if x % 2 == 0 else "odd" for x in numbers }
print(f"מילון עם תנאי if else: {dict_if_else}") # פלט: {1: 'odd', 2: 'even', 3: 'odd', 4: 'even', 5: 'odd', 6: 'even'}

# דוגמה 5: משתי רשימות
keys = ['a','b','c']
values = [1,2,3]
dictionary_from_two_lists = {key: value for key,value in zip(keys,values)} # שימוש ב-zip לאיטרציה על הרשימות בו-זמנית
print(f"מילון מרשימות: {dictionary_from_two_lists}") # פלט: {'a': 1, 'b': 2, 'c': 3}
```

**ניתוח האפשרויות:**
*   **א. הבנת מילון משמשת ליצירת מספרים אקראיים המשמשים כמפתחות למילון.:** לא נכון.
*   **ב. הבנת מילון משמשת ליצירת מילון שהמפתחות שלו הם ערכי רצף, והערכים הם האינדקסים שלהם.:** לא נכון. זו רק אחת מהאפשרויות לשימוש ב-dict comprehension.
*   **ג. הבנת מילון היא תחביר מקוצר ליצירת מילונים המבוססים על אובייקטים ניתנים לאיטרציה. היא מאפשרת ליצור מילון באמצעות זוגות מפתח-ערך בסוגריים מסולסלים, ועל ידי איטרציה על אובייקט ניתן לאיטרציה קיים.:** נכון.
*   **ד. הבנת מילון היא דרך ליצור מחלקה המבוססת על מילון.:** לא נכון.

**לסיכום:**

*   יכולות להשתמש בביטויים ליצירת מפתחות וערכים.
*   תומכות בביטויים תנאיים.

לפיכך, התשובה הנכונה היא **ג. הבנת מילון היא תחביר מקוצר ליצירת מילונים המבוססים על אובייקטים ניתנים לאיטרציה. היא מאפשרת ליצור מילון באמצעות זוגות מפתח-ערך בסוגריים מסולסלים, ועל ידי איטרציה על אובייקט ניתן לאיטרציה קיים.**
