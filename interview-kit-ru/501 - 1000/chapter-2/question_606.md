### `question_606.md`

**שאלה 606.** מהו ההבדל בין העתקה עמוקה (deep copy) להעתקה שטחית (shallow copy) של אובייקטים ב-Python, במיוחד בהקשר לאובייקטים ניתנים לשינוי, כגון רשימות? הבא דוגמאות המדגימות את ההבדלים בהתנהגות.

-   א. העתקה שטחית והעתקה עמוקה הן זהות.
-   ב. העתקה שטחית יוצרת אובייקט חדש, אך מעתיקה רק הפניות לאובייקטים מקוננים. העתקה עמוקה יוצרת עותק בלתי תלוי לחלוטין, הכולל את כל האובייקטים המקוננים.
-   ג. העתקה עמוקה יוצרת עותק, אך אינה שומרת קשרים בין אובייקטים, בעוד שהעתקה שטחית שומרת קשרים אלה.
-  ד. העתקה שטחית אינה ניתנת לשינוי, בעוד שהעתקה עמוקה ניתנת לשינוי.

**תשובה נכונה: ב**

**הסבר:**

ב-Python, בעבודה עם אובייקטים ניתנים לשינוי, כגון רשימות, מילונים וקבוצות, חשוב להבין את ההבדל בין העתקה שטחית (shallow) להעתקה עמוקה (deep). הן יוצרות עותקים של אובייקטים בצורות שונות, וזה חשוב בעבודה עם מבנים מורכבים.

*   **העתקה שטחית (Shallow copy):**
    *   יוצרת *אובייקט חדש*, אך אינה מעתיקה אובייקטים מקוננים. במקום זאת, בעותק החדש נשמרות הפניות לאובייקטים שהיו במקור.
    *   שינויים באובייקטים מקוננים יתבטאו בשני העותקים (המקורי והחדש).
     *  **שימוש:** `list.copy()`, `dict.copy()`,  או `list[:]`.

*   **העתקה עמוקה (Deep copy):**
    *   יוצרת *עותק בלתי תלוי לחלוטין*, הכולל את כל האובייקטים המקוננים.
    *   שינויים באובייקטים מקוננים לא ישפיעו על האובייקט המקורי ולהפך.
    *  **שימוש:** `copy.deepcopy()` מהמודול `copy`.

**דוגמאות:**

```python
import copy

# דוגמה 1: העתקה שטחית של רשימה (shallow copy)
original_list = [1, 2, [3, 4]]
shallow_copy = original_list.copy() # או  shallow_copy = list(original_list) או shallow_copy = original_list[:]
print(f"Исходный список: {original_list}")
print(f"Мелкая копия: {shallow_copy}")

original_list[0] = 100
print(f"Измененный исходный список (только первый элемент): {original_list}")
print(f"Мелкая копия после изменения: {shallow_copy}") # שינוי ברמה הראשונה לא השפיע על ההעתקה השטחית
# Выведет:
# Исходный список: [1, 2, [3, 4]]
# Мелкая копия: [1, 2, [3, 4]]
# Измененный исходный список (только первый элемент): [100, 2, [3, 4]]
# Мелкая копия после изменения: [1, 2, [3, 4]]

original_list[2][0] = 5 # משנים את האלמנט הראשון ברשימה המקוננת
print(f"Измененный исходный список (вложенный список): {original_list}")
print(f"Мелкая копия после изменения вложенного списка: {shallow_copy}") # שינוי ברמה המקוננת השפיע על ההעתקה השטחית
# Выведет:
# Измененный исходный список (вложенный список): [100, 2, [5, 4]]
# Мелкая копия после изменения вложенного списка: [1, 2, [5, 4]]


# דוגמה 2: העתקה עמוקה של רשימה (deep copy)
original_list = [1, 2, [3, 4]]
deep_copy = copy.deepcopy(original_list)
print(f"Исходный список: {original_list}")
print(f"Глубокая копия: {deep_copy}")

original_list[2][0] = 5 # משנים את האלמנט הראשון ברשימה המקוננת
print(f"Измененный исходный список: {original_list}")
print(f"Глубокая копия после изменения: {deep_copy}") # ההעתקה העמוקה אינה משתנה
# Выведет:
# Исходный список: [100, 2, [5, 4]]
# Глубокая копия: [1, 2, [3, 4]]
# Измененный исходный список: [100, 2, [5, 4]]
# Глубокая копия после изменения: [1, 2, [3, 4]]
```
**ניתוח האפשרויות:**
*   **א. העתקה שטחית והעתקה עמוקה הן זהות.:** לא נכון.
*   **ב. העתקה שטחית יוצרת אובייקט חדש, אך מעתיקה רק הפניות לאובייקטים מקוננים. העתקה עמוקה יוצרת עותק בלתי תלוי לחלוטין, הכולל את כל האובייקטים המקוננים.:** נכון.
*  **ג. העתקה עמוקה יוצרת עותק, אך אינה שומרת קשרים בין אובייקטים, בעוד שהעתקה שטחית שומרת קשרים אלה.:** לא נכון.
*  **ד. העתקה שטחית אינה ניתנת לשינוי, בעוד שהעתקה עמוקה ניתנת לשינוי.:** לא נכון.

**לסיכום:**
* העתקה שטחית אינה יוצרת עותקים של אובייקטים מקוננים, אלא רק הפניות אליהם.
*  העתקה עמוקה יוצרת עותקים בלתי תלויים לחלוטין של כל האובייקטים, כולל המקוננים.
* בחירת סוג ההעתקה תלויה בשאלה האם ברצונך לשמור על קשרים בין אובייקטים או ליצור עותקים בלתי תלויים לחלוטין.

לפיכך, התשובה הנכונה היא **ב. העתקה שטחית יוצרת אובייקט חדש, אך מעתיקה רק הפניות לאובייקטים מקוננים. העתקה עמוקה יוצרת עותק בלתי תלוי לחלוטין, הכולל את כל האובייקטים המקוננים.**
