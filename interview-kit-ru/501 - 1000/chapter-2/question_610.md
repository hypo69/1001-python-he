### `question_610.md`

**שאלה 610.** מהי התוצאה של פעולת הכפלה של רשימה במספר שלם ב-Python, וכיצד היא משפיעה על הרשימה המקורית?

-   א. פעולת הכפלה של רשימה במספר שלם משנה את הרשימה המקורית, ומשכפלת את איבריה כמספר הפעמים שצוין.
-   ב. פעולת הכפלה של רשימה במספר שלם יוצרת רשימה חדשה, המשכפלת את הרשימה המקורית כמספר הפעמים שצוין, כאשר הרשימה המקורית נשארת ללא שינוי.
-   ג. פעולת הכפלה של רשימה במספר שלם יוצרת רשימה חדשה, המכפילה כל איבר ברשימה המקורית במספר הנתון.
-   ד. פעולת הכפלה של רשימה במספר שלם אינה נתמכת ותגרום לשגיאת `TypeError`.

**התשובה הנכונה: ב**

**הסבר:**

ב-Python, פעולת הכפלה של רשימה במספר שלם (`list * n`) מובילה ליצירת רשימה חדשה, המכילה את איברי הרשימה המקורית, משוכפלים `n` פעמים. חשוב להדגיש שפעולה זו אינה משנה את הרשימה המקורית, אלא יוצרת עותק *חדש* עם האיברים המשוכפלים.

*   **מאפיינים עיקריים של כפל רשימה:**
    *   **שכפול רצף:** יוצרת רשימה חדשה, המשכפלת את הרשימה המקורית כמספר הפעמים שצוין.
    *   **אי-שינוי הרשימה המקורית:** הפעולה אינה משנה את הרשימה המקורית, אלא יוצרת עותק חדש.
    *   **כופל שלם:** האופרנד השני חייב להיות מספר שלם.
    *   **העתקה שטחית:** איברים של מבנים מקוננים ניתנים לשינוי (לדוגמה, רשימות בתוך רשימה), אינם מועתקים, אלא ברשימה החדשה נשמרות הפניות לאובייקטים הישנים.
    *   **קומוטטיביות:** `list*n == n*list`
*   **חריגים:**
    *   בעת הכפלה בסוגי נתונים אחרים, למעט מספר שלם, תתרחש שגיאת `TypeError`.

**דוגמאות:**

```python
# Пример 1: Умножение списка на положительное число
# דוגמה 1: הכפלת רשימה במספר חיובי
my_list = [1, 2, 3]
repeated_list = my_list * 3
print(f"Исходный список: {my_list}")   # Выведет: Исходный список: [1, 2, 3]
# תוצאה: הרשימה המקורית: [1, 2, 3]
print(f"Новый список с повторением: {repeated_list}") # Выведет: Новый список с повторением: [1, 2, 3, 1, 2, 3, 1, 2, 3]
# תוצאה: רשימה חדשה עם שכפול: [1, 2, 3, 1, 2, 3, 1, 2, 3]

# Пример 2: Умножение списка на ноль
# דוגמה 2: הכפלת רשימה באפס
my_list2 = [1, 2]
empty_list = my_list2 * 0
print(f"Исходный список: {my_list2}")
# תוצאה: הרשימה המקורית: [1, 2]
print(f"Новый список с умножением на 0 : {empty_list}")   # Выведет: Новый список с умножением на 0 : []
# תוצאה: רשימה חדשה עם הכפלה באפס: []

# Пример 3: Умножение списка с вложенными списками
# דוגמה 3: הכפלת רשימה עם רשימות מקוננות

my_list3 = [ [1, 2], [3, 4]]
repeated_nested = my_list3 * 2
print(f"Исходный список (вложенный) : {my_list3}")
# תוצאה: הרשימה המקורית (מקוננת): [[1, 2], [3, 4]]
print(f"Новый список с умножением (вложенный) : {repeated_nested}") # Выведет [[1, 2], [3, 4], [1, 2], [3, 4]]
# תוצאה: רשימה חדשה עם הכפלה (מקוננת): [[1, 2], [3, 4], [1, 2], [3, 4]]

# Пример 4: Коммутативность
# דוגמה 4: קומוטטיביות
my_list4 = [1, 2]
print(f"Результат (list * num):{my_list4 * 3}")  # Выведет: Результат (list * num):[1, 2, 1, 2, 1, 2]
# תוצאה: תוצאה (רשימה * מספר): [1, 2, 1, 2, 1, 2]
print(f"Результат (num * list):{3 * my_list4}") # Выведет: Результат (num * list):[1, 2, 1, 2, 1, 2]
# תוצאה: תוצאה (מספר * רשימה): [1, 2, 1, 2, 1, 2]
print(f"Операция коммутативна: {3* my_list4 == my_list4 *3}") # Выведет: Операция коммутативна: True
# תוצאה: הפעולה קומוטטיבית: True

# Пример 5: TypeError
# דוגמה 5: שגיאת TypeError
try:
  my_list5 = [1,2,3]
  print(my_list5 * "test")  # Вызовет исключение
  # תעורר חריגה
except TypeError as e:
    print(e) # can't multiply sequence by non-int of type 'str'
    # לא ניתן להכפיל רצף בסוג שאינו שלם 'str'
```
**ניתוח הדוגמאות:**

1.  **דוגמה 1 (הכפלה במספר חיובי):**
    *   נוצרת רשימה חדשה `repeated_list`, שהיא שכפול של `my_list` שלוש פעמים. `my_list` נשארת ללא שינוי.
2.  **דוגמה 2 (הכפלה באפס):**
    *   `my_list2 * 0` יוצרת רשימה חדשה ריקה, `my_list2` אינה משתנה.
3.  **דוגמה 3 (רשימות מקוננות):**
    *   `my_list3 * 2` יוצרת רשימה חדשה, כאשר הרשימות המקוננות אינן מועתקות, אלא מועברות על ידי הפניה (reference).
4.  **דוגמה 4 (קומוטטיביות):**
    *   סדר האופרנדים אינו משנה בעת הכפלה של רשימה במספר.
5.  **דוגמה 5 (TypeError):**
    *   הכפלת רשימה במחרוזת מעוררת שגיאת TypeError.

**ניתוח האפשרויות:**
*   **א. פעולת הכפלה של רשימה במספר שלם משנה את הרשימה המקורית, ומשכפלת את איבריה כמספר הפעמים שצוין.:** שגוי. הרשימה המקורית אינה משתנה.
*   **ב. פעולת הכפלה של רשימה במספר שלם יוצרת רשימה חדשה, המשכפלת את הרשימה המקורית כמספר הפעמים שצוין, כאשר הרשימה המקורית נשארת ללא שינוי.:** נכון.
*   **ג. פעולת הכפלה של רשימה במספר שלם יוצרת רשימה חדשה, המכפילה כל איבר ברשימה המקורית במספר הנתון.:** שגוי.
*   **ד. פעולת הכפלה של רשימה במספר שלם אינה נתמכת ותגרום לשגיאת `TypeError`.:** שגוי.
