### `question_699.md`

**שאלה 699.** נתון מערך מחרוזות `words`. פתח/י אלגוריתם שמחזיר רשימה חדשה, המכילה רק את המחרוזות שניתן להקליד באמצעות מקשים משורה אחת בלבד במקלדת QWERTY סטנדרטית.

**דוגמאות:**
```
Ввод: words = ["Hello","Alaska","Dad","Peace"]
Вывод: ["Alaska","Dad"]

Ввод: words = ["omk"]
Вывод: []
```

-   א. כדי לפתור את הבעיה, יש לעבור על כל המילים ולבדוק בלולאות מקוננות שהתווים נמצאים בשורה אחת.
-   ב. כדי לפתור את הבעיה, יש להשתמש בתכנות דינמי.
-   ג. כדי לפתור את הבעיה, יש להשתמש בפונקציות רקורסיביות בלבד.
-   ד. כדי לפתור את הבעיה, יש ליצור שלוש קבוצות עבור שורות מקלדת QWERTY, ועבור כל מילה יש לוודא שכל התווים שלה שייכים לאחת מהקבוצות הללו.

**תשובה נכונה: ג**

**הסבר:**

כדי לפתור את בעיית סינון המילים שניתן להקליד בשורה אחת של המקלדת, הגישה האופטימלית תהיה יצירת קבוצות עבור כל שורה ולאחר מכן בדיקת כל מילה לוודא שתוויה שייכים לאחת הקבוצות.

*   **אלגוריתם עם קבוצות:**
    1.  **יצירת קבוצות:** יוצרים שלוש קבוצות המייצגות את שלוש שורות מקלדת QWERTY:
        *   `row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}`
        *   `row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}`
        *   `row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}`
    2.  **איטרציה על המילים:** עוברים על כל מילה במערך `words`.
    3.  **בדיקת שייכות לשורה:** מוודאים שכל התווים של המילה הנוכחית, שהומרו לאותיות קטנות, שייכים לאחת הקבוצות שנוצרו.
        *   לשם כך, המילה מומרת לקבוצה, ונבדק האם קבוצה זו היא תת-קבוצה של אחת מקבוצות שורות המקלדת (משתמשים במתודת `issubset()`).
        *   אם תנאי זה מתקיים, פירוש הדבר שניתן להקליד את המילה בשורה אחת של המקלדת, והיא מוספת לרשימה התוצאתית.
    4.  **תוצאה:** מחזירים את הרשימה המסוננת.

*   **יתרונות האלגוריתם:**
    *   **יעילות:** קבוצות מאפשרות בדיקת שייכות מהירה של איברים בזמן O(1) על בסיס ערכי ה-hash שלהם.
    *   **קומפקטיות:** הקוד פשוט, ברור וקל לקריאה.

**דוגמאות (פסאודו-קוד):**
```
function find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}
    result = []
    for word in words:
        word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
```
**דוגמאות מימוש ב-Python:**
```python
def find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
words1 = ["Hello","Alaska","Dad","Peace"]
print(f"קלט: words = {words1}")
print(f"פלט: {find_words_from_keyboard_row(words1)}")  # יוצג:  פלט: ['Alaska', 'Dad']


words2 = ["omk"]
print(f"קלט: words = {words2}")
print(f"פלט: {find_words_from_keyboard_row(words2)}")  # יוצג: פלט: []

words3 = ["adsf","qwer","mxcv","asdfghjkl", "qwertyuiop"]
print(f"קלט: words = {words3}")
print(f"פלט: {find_words_from_keyboard_row(words3)}")  # יוצג: פלט: ['adsf', 'qwer', 'mxcv', 'asdfghjkl', 'qwertyuiop']
```
