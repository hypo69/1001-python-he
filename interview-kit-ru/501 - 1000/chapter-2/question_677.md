**שאלה 677.** פתח אלגוריתם לשמירת מחירי מניות ולספירת מספר הימים עד ליום הנוכחי (כולל היום הנוכחי) בהם מחיר המניה היה קטן או שווה למחיר ביום הנוכחי.

**דוגמה:**

```
Ввод:
["StockSpanner", "next", "next", "next", "next", "next", "next", "next"]
[[], [100], [80], [60], [70], [60], [75], [85]]
Вывод:
[null, 1, 1, 1, 2, 1, 4, 6]
Объяснение:
StockSpanner stockSpanner = new StockSpanner();
stockSpanner.next(100); // return 1
stockSpanner.next(80);  // return 1
stockSpanner.next(60);  // return 1
stockSpanner.next(70);  // return 2
stockSpanner.next(60);  // return 1
stockSpanner.next(75);  // return 4, מכיוון שהמחירים בארבעת הימים הקודמים (כולל היום) היו קטנים או שווים;
stockSpanner.next(85);  // return 6
```

-   א. כדי לפתור בעיה זו יש להשתמש בתור.
-   ב. כדי לפתור בעיה זו יש להשתמש באלגוריתם חיפוש לרוחב (BFS).
-   ג. כדי לפתור בעיה זו ניתן להשתמש במחסנית ולשמור זוג (מחיר, מונה), אם המחיר הנוכחי גדול מהמחיר בראש המחסנית, להסיר אלמנטים עד שמגיעים למחיר גדול יותר, ולאחר מכן להגדיר את המונה הנוכחי, או 1 אם לא נמצא מחיר גדול יותר.
-   ד. כדי לפתור בעיה זו יש להשתמש ברקורסיה.

**תשובה נכונה: ג**

**הסבר:**

בעיה זו דורשת אלגוריתם העוקב ביעילות אחר היסטוריית מחירי מניות ומחשב את מספר הימים בהם המחיר היה קטן או שווה למחיר הנוכחי. הפתרון האופטימלי הוא שימוש במחסנית ומעקב אחר ערכים הקטנים מהמחיר הנוכחי.

*   **אלגוריתם (עם שימוש במחסנית):**
    1.  **אתחול:**
          *  יצירת מחסנית ריקה.
         *  המחסנית תאחסן זוגות: (מחיר, מונה).
    2. **עיבוד ציטוט:**
          * עם קבלת ציטוט חדש `price` בצע את הפעולות הבאות:
              *  אתחל את המונה `count` = 1;
            *    כל עוד המחסנית אינה ריקה והמחיר בראש המחסנית קטן או שווה למחיר הנוכחי `price`, הגדל את `count` בערך המונה מהמחסנית, והוצא את הזוג מהמחסנית.
             * הוסף את הזוג הנוכחי `(price, count)` למחסנית.
         *   החזר את `count`.

*   **מאפייני האלגוריתם:**
    *   **מחסנית:** משמשת למעקב אחר היסטוריית המחירים וכמות הימים הקשורה אליהם.
    *   **סיבוכיות לינארית:** האלגוריתם מספק סיבוכיות זמן לינארית O(n), כאשר n הוא מספר מחירי המניות שהתקבלו.

**דוגמאות (פסאודו-קוד):**

```
class StockSpanner:
    def __init__(self):
        self.stack = [] # מחסנית (מחיר, מספר ימים)

    def next(self, price):
        count = 1
        while self.stack and self.stack[-1][0] <= price:
            count += self.stack.pop()[1]
        self.stack.append((price, count))
        return count
```
