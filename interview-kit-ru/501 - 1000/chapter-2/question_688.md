**שאלה 679.** נתון מערך מחרוזות `words`. פתחו אלגוריתם המחזיר רשימה חדשה, המכילה רק את המחרוזות שניתן להקליד באמצעות מקשי שורה אחת בלבד במקלדת QWERTY סטנדרטית.

**דוגמה:**
```
קלט: words = ["Hello","Alaska","Dad","Peace"]
פלט: ["Alaska","Dad"]

קלט: words = ["omk"]
פלט: []
```

-   א. לצורך פתרון הבעיה יש להשתמש בלולאות מקוננות ולבדוק עבור כל מילה מאיזו שורה היא.
-   ב. לצורך פתרון הבעיה יש להשתמש בביטויים רגולריים.
-   ג. לצורך פתרון הבעיה יש ליצור קבוצת תווים עבור כל שורת מקלדת ולבדוק עבור כל מילה שכל התווים שלה נמצאים רק באחת מהקבוצות הללו.
-   ד. לצורך פתרון הבעיה ניתן תחילה למיין את רשימת המילים לפי אורכן ולאחר מכן לבדוק אותן.

**תשובה נכונה: ג**

**הסבר:**

לצורך פתרון בעיית סינון המילים שניתן להקליד בשורת מקלדת אחת, גישה אופטימלית תהיה יצירת קבוצות עבור כל שורה, ולאחר מכן בדיקת כל מילה לגבי השתייכות התווים שלה לקבוצה אחת.

*   **אלגוריתם עם קבוצות:**
    1.  **יצירת קבוצות:** נוצרות קבוצות (sets) של תווים עבור כל שורת מקלדת.
    2.  **איטרציה על המילים:** עוברים על כל מילה במערך `words`.
    3.  **בדיקת השתייכות לשורה:** בודקים שכל התווים של כל מילה, שייכים לאחת הקבוצות.
        *   לשם כך, ממירים את המילה לקבוצה ובודקים שהחיתוך עם כל שורת מקלדת נותן או את קבוצת תוויי המילה, או קבוצה ריקה (set), מכיוון שהיא אמורה להיות מוכלת רק בשורה אחת.
        *   אם תנאי זה מתקיים, המילה מתווספת לרשימה התוצאתית.
    4.  **תוצאה:** מחזירים את רשימת המחרוזות המסוננת.

*   **יתרונות האלגוריתם:**
    *   **חיפוש יעיל:** קבוצות מספקות חיפוש מהיר (סיבוכיות זמן ממוצעת של O(1)) בעת בדיקת קיום אלמנטים.
    *   **פשטות:** הקוד הופך לקריא ותמציתי יותר.

**דוגמאות (פסאודו-קוד):**
```
function find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
```
**דוגמאות למימוש ב-Python:**
```python
def find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result

words = ["Hello","Alaska","Dad","Peace"]
print(f"קלט: words = {words}")
print(f"פלט: {find_words_from_keyboard_row(words)}") # Выведет ['Alaska', 'Dad']

words2 = ["omk"]
print(f"קלט: words = {words2}")
print(f"פלט: {find_words_from_keyboard_row(words2)}")  # Выведет: []

words3 = ["adsf","qwer","mxcv","asdfghjkl", "qwertyuiop"]
print(f"קלט: words = {words3}")
print(f"פלט: {find_words_from_keyboard_row(words3)}") # Выведет: ['adsf', 'qwer', 'mxcv', 'asdfghjkl', 'qwertyuiop']
```
