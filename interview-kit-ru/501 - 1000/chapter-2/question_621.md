**שאלה 621.** כיצד ניתן ב-Python להגדיל ולהקטין ערך של מספר שלם באחד (אינקרמנט ודקרמנט)?

-   א. עבור אינקרמנט ודקרמנט של מספרים שלמים ב-Python, משתמשים באופרטורים `++` ו-`--`, בהתאמה.
-   ב. עבור אינקרמנט משתמשים במתודה `increment()`, ועבור דקרמנט משתמשים במתודה `decrement()`.
-   ג. עבור אינקרמנט ודקרמנט של מספרים שלמים ב-Python, משתמשים באופרטורים `+=` ו-`-=`, בהתאמה.
-   ד. עבור אינקרמנט ודקרמנט של מספרים שלמים ב-Python, משתמשים בפונקציות `inc()` ו-`dec()`.

**תשובה נכונה: ג**

**הסבר:**

ב-Python, להגדלה (אינקרמנט) ולהקטנה (דקרמנט) של ערך מספר שלם, משתמשים באופרטורי ההשמה המורכבים `+=` ו-`-=`, בהתאמה. אופרטורים אלו מאפשרים לשנות את ערך המשתנה על ידי הוספה או חיסור של ערך ישירות מהערך הנוכחי של המשתנה.

*   **אופרטור `+=` (אינקרמנט):**
    *   משלב את פעולת החיבור וההשמה.
    *   מגדיל את ערך המשתנה בגודל שצויין (בדרך כלל ב-1) ומשייך את התוצאה חזרה למשתנה זה.
    *   שקול ל-: `value = value + 1`, אך זהו כתיב מקוצר יותר.

*   **אופרטור `-=` (דקרמנט):**
    *   משלב את פעולת החיסור וההשמה.
    *   מקטין את ערך המשתנה בגודל שצויין (בדרך כלל ב-1) ומשייך את התוצאה חזרה למשתנה זה.
    *   שקול ל-: `value = value - 1`, אך זהו כתיב מקוצר יותר.

**דוגמאות:**

```python
# דוגמה לאינקרמנט
value = 5
value += 1  # שקול ל- value = value + 1
print(f"ערך לאחר אינקרמנט: {value}") # ידפיס: 6

# דוגמה לדקרמנט
value2 = 7
value2 -= 1  # שקול ל- value = value - 1
print(f"ערך לאחר דקרמנט: {value2}") # ידפיס: 6

# דוגמה לדקרמנט מרובה
value3 = 10
value3 -= 1
value3 -= 1
print(f"ערך לאחר דקרמנט מרובה: {value3}")  # ידפיס: 8

# דוגמה עם מספרים אחרים
value4 = 10
value4 +=5
print(f"ערך לאחר +=5: {value4}") # ידפיס 15

value5 = 10
value5 -= 5
print(f"ערך לאחר -=5: {value5}") # ידפיס 5

```

*   **ב. עבור אינקרמנט משתמשים במתודה `increment()`, ועבור דקרמנט משתמשים במתודה `decrement()`.:** לא נכון. למספרים שלמים ב-Python אין מתודות ייעודיות כאלה לאינקרמנט ודקרמנט.
