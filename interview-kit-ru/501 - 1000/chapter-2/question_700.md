**שאלה 683.** מהו "מספר מושלם" (perfect number), וכיצד לפתח פונקציה ב-Python הבודקת האם מספר שלם נתון `num` הוא מספר מושלם.

*   מספר מושלם הוא מספר שלם חיובי השווה לסכום מחלקיו, למעט המספר עצמו.

**דוגמאות:**
```
Ввод: num = 28
Вывод: True
Объяснение: 28 = 1 + 2 + 4 + 7 + 14

Ввод: num = 7
Вывод: False
```

-   A. כדי לפתור את הבעיה יש להשתמש בחיפוש בינארי ולבדוק את המחלקים בצורה ממוינת.
-   B. כדי לפתור את הבעיה יש לעבור על כל המספרים מ-1 עד `num` ולבדוק האם הם מחלקים, ולאחר מכן לחשב את סכומם.
-  C. כדי לפתור את הבעיה יש לעבור על כל המחלקים מ-1 עד שורש המספר תוך שימוש באופטימיזציה מתוך התחשבות בכך שהמחלקים תמיד מופיעים בזוגות.
-   D. כדי לפתור את הבעיה יש ליצור מערך של מספרים ולבדוק את ראשוניותם, ולאחר מכן להשתמש בנוסחה למספרים מושלמים.

**תשובה נכונה: C**

**הסבר:**

כדי לפתור את בעיית הבדיקה האם מספר הוא מושלם, יש לחשב את סכום מחלקיו (למעט המספר עצמו) ולהשוותו למספר המקורי. לצורך כך ניתן להשתמש באלגוריתם איטרטיבי בעל סיבוכיות O(sqrt(n)), המאפשר להימנע ממעבר ממצה על כל המספרים.

*   **אלגוריתם:**
    1.  **בדיקה עבור 1:** מספרים מושלמים חייבים להיות גדולים מ-1. אם `num` שווה ל-1, מחזירים מיד `False`.
    2.  **אתחול:** מאתחלים משתנה בשם `sum_of_divisors` ל-0 לאחסון סכום המחלקים.
    3.  **חיפוש מחלקים:** עוברים על המחלקים מ-1 עד השורש הריבועי של `num`:
        *   אם `i` הוא מחלק, מוסיפים ל-`sum_of_divisors` את `i` ואת `n // i`, תוך וידוא ש-`i` אינו שווה ל-`n/i` (כדי לא להוסיף ערך אחד פעמיים).
    4.  **בדיקת שלמות:** לאחר בדיקת כל המחלקים, משווים את סכום המחלקים למספר המקורי. אם הסכום שווה למספר, מחזירים `True`, אחרת - `False`.

*   **יתרונות האלגוריתם:**
    *   **יעילות:** האלגוריתם משתמש בסיבוכיות O(sqrt(n)), במקום O(n), מה שמאפשר עבודה עם מספרים גדולים יותר בזמן קצר יותר.
    *   **מניעת כפילויות:** נמנעת הוספה חוזרת של אותו מחלק באמצעות בדיקת `i != num // i`.
    *   **פשטות ובהירות:** האלגוריתם קריא וקל ליישום.

**דוגמאות (פסאודו-קוד):**
```
function is_perfect_number(num):
    if num <= 1:
        return False
    sum_of_divisors = 0
    i = 1
    while i * i <= num:
        if num mod i == 0:
           sum_of_divisors = sum_of_divisors + i
           if i != num / i:
              sum_of_divisors = sum_of_divisors + (num/i)
        i = i+1
    return sum_of_divisors == num
```
**דוגמאות ליישום ב-Python:**
```python
import math
def is_perfect_number(num):
    if num <= 1:
        return False
    sum_of_divisors = 0
    i = 1
    while i * i <= num:
        if num % i == 0:
            sum_of_divisors += i
            if i != num // i:
                sum_of_divisors += num // i
        i += 1
    return sum_of_divisors == num

print(f"האם המספר 28 מושלם? {is_perfect_number(28)}") # Выведет: Является ли число 28 совершенным? True -> יציג: האם המספר 28 מושלם? True
```
