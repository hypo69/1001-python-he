**שאלה 644.** אילו שיטות (methods) ב-Python משמשות להמרת מחרוזת לאותיות רישיות (uppercase) ואותיות קטנות (lowercase)?

-   A. השתמשו בשיטות `uppercase()` ו-`lowercase()`.
-   B. השתמשו בשיטות `toUpperCase()` ו-`toLowerCase()`.
-   C. השתמשו בשיטות `upper()` ו-`lower()`.
-   D. השתמשו בפונקציות `toUpper()` ו-`toLower()`.

**תשובה נכונה: C**

**הסבר:**

ב-Python, שיטות מובנות של מחרוזות, `upper()` ו-`lower()`, משמשות להמרת מחרוזות לאותיות רישיות ואותיות קטנות.

*   **השיטה `upper()`:**
    *   **המרת לאותיות רישיות:** ממירה את כל התווים במחרוזת לאותיות רישיות.
    *   **מחרוזת:** מיושמת על אובייקטי מחרוזת.
    *   **מחזירה מחרוזת חדשה:** אינה משנה את המחרוזת המקורית, אלא מחזירה מחרוזת חדשה.
*  **השיטה `lower()`:**
     *   **המרת לאותיות קטנות:** ממירה את כל התווים במחרוזת לאותיות קטנות.
     *   **מחרוזת:** מיושמת על אובייקטי מחרוזת.
     *   **מחזירה מחרוזת חדשה:** אינה משנה את המחרוזת המקורית, אלא מחזירה מחרוזת חדשה.

**דוגמאות:**

```python
# דוגמה 1: המרה לאותיות רישיות
small_word = 'potatocake'
upper_word = small_word.upper()
print(f"מחרוזת מקורית: {small_word}") # תדפיס: potatocake
print(f"מחרוזת באותיות רישיות: {upper_word}")  # תדפיס: POTATOCAKE

# דוגמה 2: המרה לאותיות קטנות
big_word = 'FISHCAKE'
lower_word = big_word.lower()
print(f"מחרוזת מקורית: {big_word}")
print(f"מחרוזת באותיות קטנות: {lower_word}")  # תדפיס: fishcake

# דוגמה 3: שימוש עם רווחים
my_string = "  test String  "
print(f"'{my_string.upper()}'") # תדפיס: '  TEST STRING  '
print(f"'{my_string.lower()}'") # תדפיס: '  test string  '
# דוגמה 4: תווים שונים
my_string2 = "АБВ гДЖ  123  "
print(f"'{my_string2.upper()}'")  # תדפיס: 'АБВ ГДЖ  123  '
print(f"'{my_string2.lower()}'")  # תדפיס: 'абв гдж  123  '
```

**ניתוח האפשרויות:**
*  **A. השתמשו בשיטות `uppercase()` ו-`lowercase()`.**: שגוי.
*   **B. השתמשו בשיטות `toUpperCase()` ו-`toLowerCase()`.**: שגוי.
*  **C. השתמשו בשיטות `upper()` ו-`lower()`.**: נכון.
*   **D. השתמשו בפונקציות `toUpper()` ו-`toLower()`.**: שגוי.

**לסיכום:**
* השיטות `upper()` ו-`lower()` משמשות להמרת מחרוזות לאותיות רישיות ואותיות קטנות, בהתאמה.
* השיטות מחזירות אובייקט מחרוזת חדש; המחרוזת המקורית אינה משתנה.

לפיכך, התשובה הנכונה היא **C. השתמשו בשיטות `upper()` ו-`lower()`.**
