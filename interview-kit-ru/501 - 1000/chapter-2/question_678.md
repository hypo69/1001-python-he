**שאלה 678.** פתח אלגוריתם לשמירת מחירי מניות ולחישוב כמה ימים לפני היום הנוכחי (כולל היום הנוכחי) מחיר המניה היה קטן או שווה למחיר ביום הנוכחי.

**דוגמה:**

```
קלט:
["StockSpanner", "next", "next", "next", "next", "next", "next", "next"]
[[], [100], [80], [60], [70], [60], [75], [85]]
פלט:
[null, 1, 1, 1, 2, 1, 4, 6]
הסבר:
StockSpanner stockSpanner = new StockSpanner();
stockSpanner.next(100); // return 1
stockSpanner.next(80);  // return 1
stockSpanner.next(60);  // return 1
stockSpanner.next(70);  // return 2
stockSpanner.next(60);  // return 1
stockSpanner.next(75);  // return 4, так как цены за четыре предыдущих дня (включая сегодняшний) были меньше или равны;
stockSpanner.next(85);  // return 6
```

-   א. לפתרון בעיה זו יש להשתמש בתור.
-   ב. לפתרון בעיה זו יש להשתמש באלגוריתם חיפוש לרוחב (BFS).
-   ג. לפתרון בעיה זו ניתן להשתמש במחסנית ולשמור זוג (מחיר, מונה). אם המחיר הנוכחי גדול מהמחיר מהמחסנית, יש להסיר איברים עד שמגיעים לגדול יותר, ולאחר מכן להגדיר את המונה הנוכחי, או 1 אם גדול יותר לא נמצא.
-   ד. לפתרון בעיה זו יש להשתמש ברקוּרְסִיָה.

**תשובה נכונה: ג**

**הסבר:**

בעיה זו דורשת אלגוריתם שעוקב ביעילות אחר היסטוריית מחירי המניות ומחשב את מספר הימים עם מחיר קטן או שווה למחיר הנוכחי. הפתרון האופטימלי הוא שימוש במחסנית ומעקב אחר ערכים שקטנים מהמחיר הנוכחי.

*   **האלגוריתם (עם שימוש במחסנית):**
    1.  **אתחול:**
          *  נוצרת מחסנית ריקה.
         *  המחסנית תאחסן זוגות: (מחיר, מונה)
    2. **עיבוד מחיר:**
          * בקבלת מחיר חדש `price` מבצעים את הפעולות הבאות:
              *  מאתחלים מונה  `count` = 1;
            *    כל עוד המחסנית אינה ריקה והמחיר בראש המחסנית קטן או שווה למחיר הנוכחי `price`, אנו מגדילים  את `count` בערך המונה במחסנית, ומוציאים (pop) את הזוג מהמחסנית.
             * מוסיפים את הזוג הנוכחי `(price, count)`  למחסנית.
         *   מחזירים  את `count`.

*   **מאפייני האלגוריתם:**
    *   **מחסנית:**  משמש למעקב אחר היסטוריית מחירים ומספרי הימים הקשורים אליהם.
    *   **סיבוכיות לינארית:** האלגוריתם מבטיח סיבוכיות זמן לינארית O(n), כאשר n הוא מספר מחירי המניות שהתקבלו.

**דוגמאות (פסאודו-קוד):**

```
class StockSpanner:
    def __init__(self):
        self.stack = [] # מחסנית (מחיר, מספר ימים)

    def next(self, price):
        count = 1
        while self.stack and self.stack[-1][0] <= price:
            count += self.stack.pop()[1]
        self.stack.append((price, count))
        return count
```

**דוגמאות מימוש בפייתון:**
