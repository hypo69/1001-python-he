**שאלה 629.** כיצד מיושמת ירושת מחלקות בפייתון ואילו יכולות היא מעניקה?

-   A.  ירושת מחלקות מיושמת באמצעות מילת המפתח `inherit`, ומחלקות צאצא אינן יכולות לדרוס שיטות של מחלקת אב.
-   B. ירושת מחלקות מיושמת באמצעות מילת המפתח `extends`, ומחלקות צאצא יכולות לדרוס שיטות של מחלקת אב רק בתנאי שהן מופשטות.
-  C. ירושת מחלקות מיושמת על ידי ציון שם מחלקת האב בסוגריים עגולים לאחר שם מחלקת הצאצא (`class ChildClass(ParentClass):`). מחלקות צאצא יורשות את כל השיטות והמאפיינים של מחלקת האב ויכולות לדרוס אותם.
-  D. ירושה מיושמת באמצעות מילת המפתח `implements`. מחלקות צאצא אינן יכולות לרשת מכמה מחלקות בו זמנית.

**תשובה נכונה: C**

**הסבר:**

ירושה (inheritance) היא אחד מעקרונות המפתח בתכנות מונחה עצמים (OOP), המאפשר ליצור מחלקות חדשות (מחלקות צאצא או תת-מחלקות) על ידי ירושת תכונות ושיטות ממחלקות קיימות (מחלקות אב או מחלקות-על).

*  **מושגי יסוד בירושה:**
    *   **מחלקת אב (מחלקת-על/מחלקת בסיס):** המחלקה שממנה יורשות מחלקות אחרות.
    *  **מחלקת צאצא (תת-מחלקה):** המחלקה שיורשת תכונות ושיטות ממחלקת האב.
    *   **ירושת מאפיינים ושיטות:** מחלקת הצאצא יורשת את המאפיינים והשיטות של מחלקת האב (אם אינם פרטיים).
    *   **דריסת שיטות:** מחלקת הצאצא יכולה לדרוס (להחליף) שיטות של מחלקת האב.
    *  **הרחבת פונקציונליות:** מחלקת הצאצא יכולה להוסיף שיטות או מאפיינים חדשים, וגם להשתמש בשיטות של מחלקת האב.
    *   **ירושה מרובה:** פייתון תומכת בירושה מרובה, כלומר, מחלקה יכולה לרשת מכמה מחלקות אב בו זמנית.

*   **תחביר ירושה:**
     *  `class ChildClass(ParentClass):` — ציון שם מחלקת האב בסוגריים לאחר שם מחלקת הצאצא.

**דוגמאות:**

```python
# Example of inheritance with overriding
class Car:
    def drive(self):
        print("vroom")

class Audi(Car):
    def drive(self): # The drive method from Car will be overridden
       print("Audi vroom")

car = Car()
car.drive() # Output: vroom
audi = Audi()
audi.drive() # Output: Audi vroom
# Example of inheritance with adding new attributes

class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed

    def bark(self):
        print("Woof!")

class GoldenRetriever(Dog):
  def __init__(self, name):
     super().__init__(name, "GoldenRetriever") # calling the parent class constructor.
     self.has_ball = True # Adding attribute
  def fetch_ball(self):
    if self.has_ball:
        print("Fetching ball")
    else:
       print("No ball")

golden = GoldenRetriever("Buddy")
golden.bark() # Output: Woof! (method is inherited)
golden.fetch_ball()  # Output: Fetching ball

```
