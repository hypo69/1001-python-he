**שאלה 623.** כיצד ניתן ב-Python להסיר כפילויות מרשימה, תוך שמירה על האיברים הייחודיים?

-   א. להסרת כפילויות משתמשים במתודה `remove_duplicates()`
-   ב. להסרת כפילויות מרשימה יש להשתמש בלולאה עם בדיקה לכפילויות
-   ג. להסרת כפילויות מרשימה ניתן להשתמש בהמרת הרשימה למבנה נתונים מסוג קבוצה (set), ולאחר מכן חזרה לרשימה. ייתכן שסדר האיברים לא יישמר במקרה זה.
-   ד. להסרת כפילויות מרשימה ניתן להשתמש רק בפונקציה `filter()`.

**תשובה נכונה: ג**

**הסבר:**

הסרת כפילויות מרשימה ב-Python ניתנת לביצוע יעיל באמצעות המרת הרשימה למבנה נתונים מסוג קבוצה (set), ולאחר מכן חזרה לרשימה. שיטה זו מבוססת על כך שקבוצות ב-Python מאחסנות רק איברים ייחודיים. חשוב לציין שבמקרה זה ייתכן שסדר האיברים ישתנה.

*   **המרת הרשימה לקבוצה:**
    *   יוצרת אובייקט חדש מסוג `set`, המכיל רק את האיברים הייחודיים.
    *   מסירה את כל הכפילויות מהרשימה המקורית.
    *   מבנה נתונים מסוג קבוצה אינו מבטיח שמירה על סדר האיברים.
*  **המרת הקבוצה חזרה לרשימה:**
    *  יוצרת רשימה חדשה מאיברי הקבוצה.
     *  סדר האיברים לאחר ההמרה עשוי להשתנות.

**דוגמאות:**

```python
# דוגמה 1: הסרת כפילויות מרשימה של מספרים
my_list = [1, 1, 1, 2, 3, 3, 4, 5, 5]
unique_list = list(set(my_list)) # ממירים את הרשימה ל-set ואז חזרה לרשימה
print(f"רשימה מקורית: {my_list}")
print(f"רשימה ייחודית: {unique_list}")  # הדפסה: רשימה ייחודית: [1, 2, 3, 4, 5] או סדר אחר

# דוגמה 2: הסרת כפילויות מרשימה של מחרוזות
my_list2 = ["apple", "banana", "apple", "cherry", "banana"]
unique_list2 = list(set(my_list2))
print(f"רשימה מקורית: {my_list2}")
print(f"רשימה ייחודית: {unique_list2}") # הדפסה: רשימה ייחודית: ['apple', 'banana', 'cherry'] (הסדר עשוי להיות שונה)

# דוגמה לשמירת סדר, אם נדרש (ב-Python 3.7 ואילך בלבד)
my_list3 = [1, 1, 1, 2, 3, 3, 4, 5, 5]
unique_list3 = list(dict.fromkeys(my_list3)) # fromkeys מבטיח סדר כמו ב-Python 3.7+
print(f"רשימה מקורית: {my_list3}")
print(f"רשימה ייחודית (שמירת סדר): {unique_list3}") # הדפסה: רשימה ייחודית (שמירת סדר): [1, 2, 3, 4, 5]

# דוגמה עם קבוצה ושמירת סדר באמצעות list comprehension:
my_list4 = [1, 1, 1, 2, 3, 3, 4, 5, 5]
seen = set()
unique_list4 = [x for x in my_list4 if x not in seen and not seen.add(x)] # איטרציה על הרשימה, שומרת על הסדר
print(f"רשימה ייחודית (עם list comprehension): {unique_list4}") # הדפסה: רשימה ייחודית (עם list comprehension): [1, 2, 3, 4, 5]
```

**ניתוח האפשרויות:**
*  **א. להסרת כפילויות משתמשים במתודה `remove_duplicates()`:** שגוי.
*   **ב. להסרת כפילויות מרשימה יש להשתמש בלולאה עם בדיקה לכפילויות:** שגוי, למרות שזו אחת הדרכים האפשריות, היא פחות יעילה.
*   **ג. להסרת כפילויות מרשימה ניתן להשתמש בהמרת הרשימה למבנה נתונים מסוג קבוצה (set), ולאחר מכן חזרה לרשימה. ייתכן שסדר האיברים לא יישמר במקרה זה.:** נכון.
*  **ד. להסרת כפילויות מרשימה ניתן להשתמש רק בפונקציה `filter()`.:** שגוי.

**לסיכום:**
*   המרת הרשימה לקבוצה וחזרה לרשימה היא דרך יעילה להסרת כפילויות.
*  סדר האיברים עשוי להשתנות בעת שימוש ב-`set`.
*  אם נדרשת שמירה על הסדר, ניתן להשתמש ב-`list(dict.fromkeys())` או ב-list comprehension.
