**שאלה 690.** נתון מערך של מחרוזות `words`. פתחו אלגוריתם המחזיר רשימה חדשה המכילה רק את המחרוזות שניתן להקליד באמצעות מקשים משורה אחת בלבד במקלדת QWERTY סטנדרטית.

**דוגמה:**
```
Ввод: words = ["Hello","Alaska","Dad","Peace"]
Вывод: ["Alaska","Dad"]

Ввод: words = ["omk"]
Вывод: []
```

- א. על מנת לפתור את הבעיה יש להשתמש בלולאות מקוננות ולבדוק עבור כל מילה מאיזו שורה היא.
- ב. על מנת לפתור את הבעיה יש להשתמש בביטויים רגולריים.
- ג. על מנת לפתור את הבעיה יש ליצור קבוצת תווים עבור כל שורת מקלדת ולבדוק עבור כל מילה שכל התווים שלה נמצאים רק באחת מקבוצות אלה.
- ד. על מנת לפתור את הבעיה ניתן למיין קודם את רשימת המילים לפי אורכן ולאחר מכן לבדוק אותן.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית סינון המילים שניתן להקליד בשורת מקלדת אחת, הגישה האופטימלית תהיה ליצור קבוצות עבור כל שורה ולאחר מכן לבדוק כל מילה לגבי השתייכות התווים לקבוצה אחת.

*   **אלגוריתם עם קבוצות:**
    1.  **יצירת קבוצות:** נוצרות קבוצות (sets) של תווים עבור כל שורת מקלדת.
    2.  **איטרציה על המילים:** עוברים על כל מילה במערך `words`.
    3.  **בדיקת השתייכות לשורה:** בודקים שכל התווים של כל מילה משתייכים לאחת הקבוצות.
        *   לשם כך, ממירים את המילה לקבוצה ובודקים שהחיתוך עם כל שורת מקלדת נותן או את קבוצת תווים המילה, או קבוצה ריקה, מכיוון שהיא אמורה להיות מוכלת רק בשורה אחת.
        *   אם תנאי זה מתקיים, מוסיפים את המילה לרשימה התוצאתית.
    4.  **תוצאה:** מחזירים את רשימת המחרוזות המסוננת.

*   **יתרונות האלגוריתם:**
    *   **חיפוש יעיל:** קבוצות מבטיחות חיפוש מהיר (O(1) average time complexity) בעת בדיקת קיום אלמנטים.
    *   **פשטות:** הקוד הופך לקריא ותמציתי יותר.

**דוגמאות (פסאודו-קוד):**
```
function find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
```

```python
def find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}

    result = []

         result.append(word)
    return result
words = ["Hello","Alaska","Dad","Peace"]
print(f"Вход: words = {words}") # קלט: words = ["Hello","Alaska","Dad","Peace"]
print(f"Выход: {find_words_from_keyboard_row(words)}") # פלט: ['Alaska', 'Dad']

words2 = ["omk"]
print(f"Вход: words = {words2}") # קלט: words = ["omk"]
print(f"Выход: {find_words_from_keyboard_row(words2)}")  # פלט: []

print(f"Вход: words = {words3}") # קלט: words = ["adsf","qwer","mxcv","asdfghjkl", "qwertyuiop"]
print(f"Выход: {find_words_from_keyboard_row(words3)}") # פלט: ['adsf', 'qwer', 'mxcv', 'asdfghjkl', 'qwertyuiop']
```
