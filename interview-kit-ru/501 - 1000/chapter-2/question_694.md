### `question_694.md`

**שאלה 694.** פתח/י פונקציה ב-Python שמקבלת מספר שלם `columnNumber`, המייצג מספר עמודה בגיליון אלקטרוני (כמו ב-Excel), ומחזירה את הייצוג המחרוזתי של עמודה זו (ייצוג אותיות).

**דוגמאות:**

```
קלט: columnNumber = 1
פלט: "A"

קלט: columnNumber = 28
פלט: "AB"
```

-   א. על מנת לפתור שאלה זו יש פשוט להמיר את המספר למחרוזת.
-   ב. על מנת לפתור שאלה זו יש להשתמש באלגוריתם רקורסיבי, על ידי חלוקת המספר והוספת אות.
-   ג. על מנת לפתור שאלה זו יש להשתמש באלגוריתם איטרטיבי לצורך ייצוג המספר בשיטה בסיס 26, עם התווים A-Z, ולאחר מכן ליצור מחרוזת מהתווים שהתקבלו בסדר הנכון.
-   ד. על מנת לפתור שאלה זו מתאימה רק שיטה המשתמשת בספריית pandas.

**תשובה נכונה: ג**

**הסבר:**

לצורך המרת מספר עמודה לייצוג האותיות שלה, כפי שנהוג ב-Excel, יש להשתמש באלגוריתם הפועל באופן דומה להמרה ממערכת ספירה עשרונית למערכת ספירה בבסיס 26, כאשר הספרות מיוצגות על ידי האותיות מ-A עד Z.

*   **אלגוריתם (איטרטיבי עם המרה למערכת בסיס 26):**
    1.  **אתחול:** נוצרת מחרוזת ריקה `result` לאחסון ייצוג האותיות.
    2.  **לולאה כל עוד `columnNumber` גדול מ-0:**
        *   **חישוב שארית:** לוקחים את השארית מחלוקת `(columnNumber-1)` ב-26, המתאימה לתו הנוכחי (`0` מתאים ל-"A", `1` מתאים ל-"B", וכן הלאה).
        *   **ההפניה לתו:** ממירים את השארית לתו באמצעות `chr(offset + 65)`, כאשר 65 הוא קוד ASCII של התו "A".
        *   **הוספה לתוצאה:** מוסיפים את התו לתחילת התוצאה `result = char + result`.
        *   **חלוקה ב-26:** מחלקים את `columnNumber` ב-26 באמצעות חילוק שלם `columnNumber //= 26`.
    3.  **תוצאה:** לאחר סיום הלולאה מוחזרת המחרוזת `result`.

**דוגמאות (פסאודו-קוד):**

```
function convert_to_excel_column(columnNumber):
    result = ""
    while columnNumber > 0:
      offset = (columnNumber - 1) % 26
      char = chr(offset + 65)
      result = char + result
      columnNumber = columnNumber // 26
    return result
```

**דוגמאות מימוש ב-Python:**

```python
def convert_to_excel_column(columnNumber):
    result = ""
    while columnNumber > 0:
      offset = (columnNumber - 1) % 26
      char = chr(offset + 65)
      result = char + result
      columnNumber //= 26
    return result

print(f"קלט: columnNumber = {1}")
print(f"פלט: '{convert_to_excel_column(1)}'") # יוצג: פלט: 'A'
print(f"קלט: columnNumber = {28}")
print(f"פלט: '{convert_to_excel_column(28)}'") # יוצג: פלט: 'AB'
print(f"קלט: columnNumber = {52}")
print(f"פלט: '{convert_to_excel_column(52)}'")  # יוצג: פלט: 'AZ'
print(f"קלט: columnNumber = {701}")
print(f"פלט: '{convert_to_excel_column(701)}'") # יוצג: פלט: 'ZY'
```
