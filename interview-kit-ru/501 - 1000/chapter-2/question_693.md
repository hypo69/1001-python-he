**שאלה 693.** נתונה מחרוזת `s`, המורכבת מספרות בלבד. פתח אלגוריתם שמחזיר את כל כתובות ה-IP התקפות האפשריות שניתן ליצור על ידי הוספת נקודות למחרוזת `s`.

*   כתובת IP מורכבת בדיוק מארבעה מספרים שלמים, מופרדים בנקודות.
*   כל מספר שלם נמצא בטווח 0 עד 255 (כולל) ואינו יכול להכיל אפסים מובילים (למעט `0`).

**דוגמאות:**

```
קלט: s = "25525511135"
פלט: ["255.255.11.135","255.255.111.35"]

קלט: s = "0000"
פלט: ["0.0.0.0"]
```

-   א. כדי לפתור את הבעיה יש להשתמש באלגוריתם חמדן.
-   ב. כדי לפתור את הבעיה יש להשתמש בביטויים רגולריים ולבדוק כל חלק לגבי תקיפותו.
-   ג. כדי לפתור את הבעיה יש להשתמש ברק ורסיה עם בקטראקינג, לבדיקת כל החלוקות האפשריות.
-   ד. כדי לפתור את הבעיה יש להשתמש בתכנות דינמי, כדי לשמור את כל כתובות ה-IP התקפות האפשריות.

**הסבר:**

כדי לפתור את הבעיה של יצירת כל כתובות ה-IP התקפות ממחרוזת המורכבת מספרות בלבד, הגישה האופטימלית היא שימוש באלגוריתם רק ורסיבי עם שימוש בבקטראקינג (backtracking). גישה זו מאפשרת לחקור את כל האפשרויות האפשריות לחלוקת המחרוזת לארבעה חלקים, תוך התחשבות במגבלות על הערך המספרי והיעדר אפסים מובילים.

*   **אלגוריתם (רק ורסיבי עם בקטראקינג):**
    1.  **פונקציה רק ורסיבית:** יוצרים פונקציה רק ורסיבית שמקבלת את המחרוזת הנוכחית, אינדקס במחרוזת, החלק הנוכחי של כתובת ה-IP, וכמות החלקים שנותרה להוספה.
    2.  **מקרה בסיס:** אם כמות החלקים הנותרים שווה לאפס, והאינדקס מצביע על סוף המחרוזת, אזי מוסיפים את כתובת ה-IP לרשימת התוצאות.
    3.  **יצירת חלקים:**
        *   בכל שלב, עוברים על כל תתי-המחרוזות האפשריות עד סוף המחרוזת המקורית בצעדים של 1, שהן מועמדות פוטנציאליות לחלק מכתובת ה-IP.
        *   **תיקוף:** בודקים כל תת-מחרוזת לגבי נכונותה, כלומר, היא חייבת להתאים לטווח הערכים (0 עד 255) ואסור להכיל אפסים מובילים (למעט המקרה שתת-המחרוזת היא `0`).
        *   **קריאה רק ורסיבית:** אם תת-המחרוזת תקפה, קוראים לפונקציה רק ורסיבית כדי למצוא את החלק הבא של ה-IP.
        *   **בקטראקינג:** לאחר כל קריאה רק ורסיבית, מבטלים את החלק הנוכחי של ה-IP (או לא מוסיפים אותו), כדי לחזור למצב הקודם ולבדוק אפשרויות אחרות.

**דוגמאות יישום ב-Python:**

```python
def restore_ip_addresses(s):
    result = []
    def backtrack(index, current_ip, remaining_parts):
        if remaining_parts == 0:
            if index == len(s):
                result.append(".".join(current_ip))
            return
        for i in range(index, len(s)):
            sub = s[index : i + 1]
            if len(sub) > 1 and sub[0] == '0':
                continue
            if int(sub) <= 255:
               backtrack(i+1, current_ip+[sub], remaining_parts-1)

    backtrack(0, [], 4) # קריאה לפונקציה הרקורסיבית
    return result


s1 = "25525511135"
print(f"קלט: s = '{s1}'")
print(f"פלט: {restore_ip_addresses(s1)}") #  ['255.255.11.135', '255.255.111.35']

s2 = "0000"
print(f"קלט: s = '{s2}'")
print(f"פלט: {restore_ip_addresses(s2)}")  # יוצג: ['0.0.0.0']

s3 = "101023"
print(f"קלט: s = '{s3}'")
print(f"פלט: {restore_ip_addresses(s3)}") # ['1.0.10.23', '1.0.102.3', '10.1.0.23', '10.1.2.3', '10.10.2.3', '101.0.2.3', '101.0.23']
```
