**שאלה 645.** מהו ההבדל בין השיטות `remove()`, `del` ו-`pop()` בעת הסרת איברים מרשימה בפייתון?

-   A. `remove()` מסירה איבר לפי האינדקס שלו, `del` מסירה את הערך התואם הראשון, ו-`pop()` מסירה את האיבר האחרון.
-  B. `remove()`, `del` ו-`pop()` שקולים וניתנים לשימוש להסרת איבר.
-   C. `remove()` מסירה את הערך התואם הראשון, `del` מסירה איבר לפי האינדקס שלו, ו-`pop()` מסירה איבר לפי אינדקס ומחזירה את ערכו.
-   D. `remove()` מסירה איבר לפי האינדקס שלו, `del` מסירה מספר איברים ברשימה, ו-`pop()` מסירה את האיבר הראשון.

**תשובה נכונה: C**

**הסבר:**

בפייתון, קיימות שלוש דרכים עיקריות להסרת איברים מרשימות — שיטת ה-`remove()`, הוראת ה-`del`, ושיטת ה-`pop()`. לכל אחת מהן מאפיינים ייחודיים, ובחירת השיטה המתאימה תלויה במשימה הנדרשת.

*   **שיטת `remove()`:**
    *   **מחיקה לפי ערך:** מוחקת את המופע הראשון של איבר בעל הערך שצוין.
    *   **שינוי הרשימה:** משנה את הרשימה המקורית, ולא מחזירה רשימה חדשה.
    *   **קריאה:** `list.remove(value)`.
    *   **שגיאה:** אם האיבר אינו נמצא ברשימה, תיגרם חריגת `ValueError`.

*   **הוראת `del`:**
    *   **מחיקה לפי אינדקס:** מוחקת איבר מהרשימה לפי האינדקס שלו.
    *   **שינוי הרשימה:** משנה את הרשימה המקורית.
    *   **קריאה:** `del list[index]`.
    *   **מחיקת פרוסות:** יכולה למחוק פרוסות של רשימות, לדוגמה, `del list[0:3]`.
*   **שיטת `pop()`:**
    *   **מחיקה לפי אינדקס:** מוחקת איבר מהרשימה לפי האינדקס שלו, בדומה ל-`del`.
    *   **החזרת ערך:** מחזירה את האיבר שנמחק.
    *   **שינוי הרשימה:** משנה את הרשימה המקורית.
    *   **קריאה:** `list.pop(index)`.
    *   **מחיקת האיבר האחרון (ללא אינדקס):** אם אינדקס אינו מצוין, מוחקת ומחזירה את האיבר האחרון.

**דוגמאות:**

```python
# דוגמה 1: שימוש ב-remove() להסרת איבר לפי ערך
my_list = ['a', 'b', 'c', 'd', 'b']
my_list.remove('b')
print(f"הרשימה אחרי remove('b'): {my_list}") # פלט: ['a', 'c', 'd', 'b']

# דוגמה 2: שימוש ב-del להסרת איבר לפי אינדקס
my_list2 = ['a', 'b', 'c', 'd']
del my_list2[0]
print(f"הרשימה אחרי del my_list2[0]: {my_list2}") # פלט: ['b', 'c', 'd']

# דוגמה 3: שימוש ב-pop() להסרת איבר לפי אינדקס והחזרת ערכו
my_list3 = ['a', 'b', 'c', 'd']
popped_element = my_list3.pop(2)
print(f"הרשימה אחרי pop(2): {my_list3}") # פלט: ['a', 'b', 'd']
print(f"האיבר שהוחזר מ-pop(2): {popped_element}") # יוצג: c

# דוגמה 4: pop() ללא פרמטר
my_list4 = ["apple", "banana", "cherry"]
last_element = my_list4.pop()
print(f"הרשימה אחרי pop(): {my_list4}") #  ['apple', 'banana']
print(f"Last element: {last_element}") # יוצג Last element: cherry

# דוגמה 5: גרימת ValueError אם האיבר אינו ברשימה
try:
  my_list5 = ["a", "b", "c"]
  my_list5.remove("d") # raises Value Error
except ValueError as e:
  print(f"שגיאה: {e}") # יוצג: Error: list.remove(x): x not in list

```

**ניתוח האפשרויות:**
*   **A. `remove()` מסירה איבר לפי האינדקס שלו, `del` מסירה את הערך התואם הראשון, ו-`pop()` מסירה את האיבר האחרון.:** שגוי.
*   **B. `remove()`, `del` ו-`pop()` שקולים וניתנים לשימוש להסרת איבר.:** שגוי.
*   **C. `remove()` מסירה את הערך התואם הראשון, `del` מסירה איבר לפי האינדקס שלו, ו-`pop()` מסירה איבר לפי אינדקס ומחזירה את ערכו.:** נכון.
*   **D. `remove()` מסירה איבר לפי האינדקס שלו, `del` מסירה מספר איברים ברשימה, ו-`pop()` מסירה את האיבר הראשון.:** שגוי.

**לסיכום:**
*   שיטת `remove()` מסירה את המופע הראשון של הערך.
*   `del` מוחקת איבר לפי אינדקס או פרוסה.
*   שיטת `pop()` מוחקת ומחזירה את האיבר לפי אינדקס, או את האיבר האחרון אם אינדקס לא הועבר.
*   בחירת השיטה תלויה באופן שבו ברצונך למחוק את האיבר, והאם נחוץ לך לקבל את הערך שנמחק.

לפיכך, התשובה הנכונה היא **C. `remove()` מסירה את הערך התואם הראשון, `del` מסירה איבר לפי האינדקס שלו, ו-`pop()` מסירה איבר לפי אינדקס ומחזירה את ערכו.**
