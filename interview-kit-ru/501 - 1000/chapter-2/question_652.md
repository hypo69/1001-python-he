**שאלה 652.** מהו PyTest ולשם מה הוא משמש?

- א. PyTest היא ספרייה לוויזואליזציה של גרפים.
- ב. PyTest היא פריימוורק ליצירת יישומי רשת.
- ג. PyTest היא פריימוורק פופולרית לבדיקת קוד ב-Python.
- ד. PyTest היא ספרייה לעבודה עם מסדי נתונים.

**תשובה נכונה: ג**

**הסבר:**

PyTest הוא אחד מפריימוורקי בדיקת הקוד הפופולריים והעוצמתיים ביותר ב-Python. הוא מספק מגוון רחב של כלים ויכולות לכתיבה, לארגון ולביצוע בדיקות, וכן לניהול סביבות בדיקה ודוחות.

*   **מאפיינים עיקריים של PyTest:**
    *   **בדיקת קוד:** PyTest מיועד לבדיקת קוד ב-Python, מאפשר כתיבה, הרצה ובדיקה של סוגי בדיקות שונים.
    *   **פישוט תהליך הבדיקה:** מספק תחביר נוח ותמציתי לכתיבת בדיקות.
    *  **איתור אוטומטי של בדיקות:** מזהה בדיקות באופן אוטומטי על בסיס תחיליות בשמות פונקציות.
    *  **פיקסטורות:** מאפשר להגדיר פיקסטורות המספקות סביבות בדיקה ונתונים לבדיקות.
    *    **פרמטריזציה:** מאפשר לבצע פרמטריזציה לבדיקות, ולהריץ אותן מספר פעמים עם ערכות נתונים שונות.
     *  **סמנים:** מאפשר ליצור סמנים עבור בדיקות ולשלוט באופן גמיש בבחירת הבדיקות להרצה.
    *  **תוספים:** כולל מערכת תוספים המאפשרת להרחיב את הפונקציונליות שלו.
    *   **דיווח:** מספק דוחות אודות תוצאות הבדיקות.

*  **רכיבים עיקריים של PyTest:**
    *  **פונקציות בדיקה:** מתחילות בתחילית `test_` (לדוגמה, `test_my_function()`).
    *  **פיקסטורות (fixtures):** פונקציות שמכינות נתונים לבדיקות ומנהלות את הסביבה.
     *  **פרמטריזציה (parametrize):** באמצעות ה-decorator ניתן להריץ בדיקות מספר פעמים עם ערכים שונים של פרמטרים.
     *  **סמנים (markers):** משמשים לקטלוג בדיקות.

**דוגמאות:**

```python
# דוגמה 1: פונקציית בדיקה פשוטה
def test_add():
  assert 2 + 2 == 4

# דוגמה 2:  שימוש בפיקסטורה
import pytest

@pytest.fixture
def my_data():
  return [1,2,3]
def test_my_list(my_data):
  assert len(my_data) == 3

# דוגמה 3: שימוש בפרמטריזציה

@pytest.mark.parametrize("input, expected", [
    (1,2),
    (2,3),
    (3,4)
])
def test_add_parametrize(input, expected):
   assert input + 1 == expected

```
**ניתוח האפשרויות:**
*   **א. PyTest היא ספרייה לוויזואליזציה של גרפים.:** שגוי.
*   **ב. PyTest היא פריימוורק ליצירת יישומי רשת.:** שגוי.
*   **ג. PyTest היא פריימוורק פופולרית לבדיקת קוד ב-Python.:** נכון.
*   **ד. PyTest היא ספרייה לעבודה עם מסדי נתונים.:** שגוי.

**לסיכום:**
*   PyTest הוא כלי עוצמתי ונוח לבדיקת קוד Python.
*  מספק מגוון רחב של יכולות ליצירה, להרצה ולניהול בדיקות.

לפיכך, התשובה הנכונה היא **ג. PyTest היא פריימוורק פופולרית לבדיקת קוד ב-Python.**
