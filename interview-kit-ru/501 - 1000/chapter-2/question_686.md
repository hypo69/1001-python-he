**שאלה 686.** נתון מערך של מספרים שלמים `nums` ושני מספרים שלמים `minK` ו-`maxK`. נדרש לפתח אלגוריתם הסופר את כמות תת-המערכים (חלקים רציפים של המערך) ב-`nums` שמקיימים את התנאים הבאים:

*   הערך המינימלי בתת-המערך שווה ל-`minK`.
*   הערך המקסימלי בתת-המערך שווה ל-`maxK`.

**דוגמאות:**

```
Ввод: nums = [1,3,5,2,7,5], minK = 1, maxK = 5
Вывод: 2

Ввод: nums = [1,1,1,1], minK = 1, maxK = 1
Вывод: 10
```

-   A. לפתרון הבעיה ניתן למיין את תת-המערכים ולבדוק את התאמתם ל-minK ול-maxK, אם מתאים, מגדילים את המונה.
-   B. לפתרון הבעיה, יש לעבור על המערך באיטרציה וליצור את כל התת-מערכים האפשריים, ובכל פעם לבדוק את תנאי המינימום והמקסימום.
-   C. לפתרון הבעיה יש להשתמש בגישת חלון גולש (sliding window) ולתחזק מצביעים left ו-right, תוך שימוש בתכנות דינמי לאופטימיזציה.
-   D. לפתרון הבעיה ניתן להשתמש בשני מצביעים `left` ו-`right`, להזיז את הימני ולעקוב אחר המינימום והמקסימום בטווח `left`, `right`, וכן אחר אינדקסים `last_min` ו-`last_max`, מה שמאפשר לעשות שימוש חוזר בנתונים כדי לא לחשב מחדש את המינימום והמקסימום.

**תשובה נכונה: D**

**הסבר:**

לפתרון יעיל של בעיית ספירת תת-מערכים המקיימים את תנאי המינימום והמקסימום, ניתן להשתמש בגישה עם שני מצביעים ומעקב אחר מיקום `minK` ו-`maxK`, שיאפשר שימוש חוזר בתוצאות, ולהימנע מחיפוש חוזר של המינימום והמקסימום בכל צעד.

*   **אלגוריתם שני מצביעים:**
    1.  **אתחול:**
         *  `left` - מצביע על הגבול השמאלי של התת-מערך הנוכחי.
        *  `right` - מצביע על הגבול הימני של התת-מערך הנוכחי.
         * `result` - כמות תת-המערכים המקיימים את התנאי.
        * `last_min` ו-`last_max` - אינדקסים של המופעים האחרונים של `minK` ו-`maxK`.
    2.   **הזזת הגבול הימני:**
          *  מזיזים את `right` ימינה עד שמגיעים לסוף הרשימה.
        *   **עדכון `last_min` ו-`last_max`:**
           *  מעדכנים את מיקומי `last_min` ו-`last_max` אם פוגשים אלמנטים מינימליים ומקסימליים חדשים.
       *  **בדיקת תנאי:**
         *  אם בחלון הנוכחי, `last_min` ו-`last_max` נמצאים מחוץ לטווח החלון, אז מעבירים את הגבול השמאלי אחריהם (כלומר `left = min(last_min, last_max)+1`), מכיוון שמשמאל אין אפשרויות לתת-מערך מתאים, וכמו כן מאפסים את כמות התת-מערכים המתאימים עבור הגבול השמאלי.
      * **ספירה:** אם שני האינדקסים `last_min` ו-`last_max` נמצאים בחלון הנוכחי, אז מוסיפים לתוצאה את כמות התת-מערכים המתאימים, שבהם המצביע השמאלי יכול להימצא עד לאינדקס שאחרי המופעים האחרונים של `last_min` ו-`last_max` (כלומר `min(last_min, last_max)+1`) - הנוסחה `max(0, min(last_min, last_max)-left+1)`.
   3. **החזרת תוצאה:** מחזירים את כמות תת-המערכים שנספרה.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות לינארית:** לאלגוריתם סיבוכיות זמן O(n), מכיוון ששני המצביעים עוברים על המערך לא יותר מפעם אחת.
    *   **יעילות:** אין צורך לעבור על כל התת-מערכים, מה שמאפשר לקבל את התוצאה הנדרשת בזמן לינארי.

**דוגמאות (פסאודו-קוד):**

```
function count_fixed_bound_subarrays(nums, minK, maxK):
    left = 0
    result = 0
    last_min = -1
    last_max = -1
    for right in range(length(nums)):
        if nums[right] < minK or nums[right] > maxK: # בודקים הגבלות
          last_min = -1
          last_max = -1
          left = right + 1 # מאפסים את המונה, מכיוון שלא יכולים להיות יותר תת-מערכים נכונים משמאל.
          continue
        if nums[right] == minK:
            last_min = right
        if nums[right] == maxK:
            last_max = right
        if last_min != -1 and last_max != -1:
          result += max(0, min(last_min, last_max)-left+1 )
    return result
```

def count_fixed_bound_subarrays(nums, minK, maxK):
    left = 0
    result = 0
    last_min = -1
    last_max = -1
    for right, val in enumerate(nums):
        if val < minK or val > maxK:
           last_min = -1
           last_max = -1
           left = right + 1
           continue
        if val == minK:
           last_min = right
        if val == maxK:
           last_max = right
        if last_min != -1 and last_max != -1:
          result += max(0, min(last_min, last_max)-left+1 )
    return result

nums = [1,3,5,2,7,5]
minK = 1
maxK = 5
print(f"קלט: nums = {nums}, minK = {minK}, maxK = {maxK}")
print(f"פלט: {count_fixed_bound_subarrays(nums, minK, maxK)}") # יוצג: פלט: 2
nums2 = [1,1,1,1]
minK2 = 1
maxK2 = 1
print(f"קלט: nums = {nums2}, minK = {minK2}, maxK = {maxK2}")

minK3 = 1

print(f"קלט: nums = {nums3}, minK = {minK3}, maxK = {maxK3}")
print(f"פלט: {count_fixed_bound_subarrays(nums3, minK3, maxK3)}") # יוצג: פלט: 10
