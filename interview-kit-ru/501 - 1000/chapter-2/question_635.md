### `question_635.md`

**שאלה 635.** מהם ההבדלים בין מילות המפתח `pass`, `continue` ו-`break` ב-Python, וכיצד הן משפיעות על ביצוע לולאות?

- א. `pass` משמשת להפסקת לולאה, `continue` לדילוג על האיטרציה הנוכחית, `break` לביצוע לולאה בסדר הפוך.
- ב. `pass` משמשת לדילוג על קוד, `continue` לדילוג רק על אלמנטים שאינם עומדים בתנאי, ו-`break` להפסקת לולאה וסיום ביצועה.
- ג. `pass` משמשת לדילוג על קוד, `continue` למעבר לאיטרציה הבאה של הלולאה, ו-`break` לסיום מיידי של הלולאה.
- ד. `pass` משמשת לאיתור באגים (debugging), `continue` לביצוע אינסופי, `break` לדילוג על האלמנט הנוכחי.

**תשובה נכונה: ג**

**הסבר:**

ב-Python, מילות המפתח `pass`, `continue` ו-`break` משמשות לשליטה בזרימת הביצוע בלולאות, אך יש להן מטרות שונות לחלוטין.

*   **`pass`:**
    *   **הוראת 'עשה כלום' (No-operation statement):** `pass` היא הוראה שאינה עושה דבר.
    *   **תחביר:** משמשת במקומות שבהם נדרש ביטוי תחבירי מסוים, אך אין צורך לבצע קוד כלשהו.
    *   **שימוש:**
        *   בבלוקי קוד ריקים, לדוגמה, במחלקות, פונקציות או משפטי תנאי שטרם יושמו.
        *   להגדרת קוד מילוי (placeholder) עבור קוד שיישום ביצועו מאוחר יותר.
    *   **אינה משפיעה על זרימת הביצוע:** אינה משנה את זרימת הביצוע הרגילה של התוכנית.

*   **`continue`:**
    *   **דילוג על האיטרציה הנוכחית:** מסיימת את האיטרציה הנוכחית של הלולאה ועוברת לאיטרציה הבאה.
    *   **תחביר:** משמשת בתוך לולאות `for` ו-`while`.
    *   **מעבר לאיטרציה הבאה:** בעת הגעה ל-`continue` בלולאה, כל ההוראות שנותרו עד לסוף האיטרציה הנוכחית אינן מבוצעות, וזרימת הבקרה עוברת לתחילת האיטרציה הבאה.

*   **`break`:**
    *   **סיום הלולאה:** מפסיקה את ביצוע הלולאה ומעבירה את זרימת הבקרה להוראה הבאה אחרי הלולאה.
    *   **תחביר:** משמשת בתוך לולאות `for` ו-`while`.
    *   **סיום מיידי:** בעת הגעה ל-`break` בלולאה, כל האיטרציות הנותרות של הלולאה אינן מבוצעות, וזרימת הבקרה יוצאת מהלולאה.

**דוגמאות:**
```python
# דוגמה 1: שימוש ב-pass

for i in range(5):
  if i > 3:
     pass # לא עושים כלום
  print(i)
# פלט:
# 0
# 1
# 2
# 3
# 4

# דוגמה 2: שימוש ב-continue
for i in range(5):
    if i < 2:
      continue # עובר לאיטרציה הבאה
    print(i)
# פלט
# 2
# 3
# 4

# דוגמה 3: שימוש ב-break
for i in range(5):
    if i == 3:
        break # יציאה מהלולאה
    print(i)
# פלט
# 0
# 1
# 2

# דוגמה pass בתוך if

def my_func():
   if True:
     pass # קוד מילוי, אין קוד כרגע
    # print("inside if") - אם מבטלים את ההערה, אז pass מיותר
   print("outside if")

my_func() # יפלוט: outside if
```
**ניתוח האפשרויות:**
*   **א. `pass` משמשת להפסקת לולאה, `continue` לדילוג על האיטרציה הנוכחית, `break` לביצוע לולאה בסדר הפוך.:** שגוי.
*   **ב. `pass` משמשת לדילוג על קוד, `continue` לדילוג רק על אלמנטים שאינם עומדים בתנאי, ו-`break` להפסקת לולאה וסיום ביצועה.:** שגוי.
*   **ג. `pass` משמשת לדילוג על קוד, `continue` למעבר לאיטרציה הבאה של הלולאה, ו-`break` לסיום מיידי של הלולאה.:** נכון.
*   **ד. `pass` משמשת לאיתור באגים (debugging), `continue` לביצוע אינסופי, `break` לדילוג על האלמנט הנוכחי.:** שגוי.

**לסיכום:**
*   `pass` משמשת כהוראת 'עשה כלום' במקומות שבהם נדרש תחביר אך אין קוד.
*   `continue` עוברת לאיטרציה הבאה של הלולאה.
*   `break` עוצרת את ביצוע הלולאה.

לכן, התשובה הנכונה היא **ג. `pass` משמשת לדילוג על קוד, `continue` למעבר לאיטרציה הבאה של הלולאה, ו-`break` לסיום מיידי של הלולאה.**
