**שאלה 667.** כיצד ניתן להסיר כפילויות מרשימה מקושרת לא ממוינת בפייתון? תאר שני גישות לפתרון בעיה זו: שימוש בשני מצביעים ושימוש בטבלת גיבוב (מבנה נתונים מסוג קבוצה).

-   א. כדי להסיר כפילויות מרשימה מקושרת יש למיין תחילה את הרשימה, ולאחר מכן להסיר את הכפילויות.
-   ב. הסרת כפילויות אפשרית רק באמצעות רקורסיה, שכן יש לעקוב אחר אלמנטים שכבר בוקרו.
-   ג. הסרת כפילויות אפשרית באמצעות מעבר על הרשימה תוך שימוש בשני מצביעים, או באמצעות שימוש בטבלאות גיבוב (מבנה נתונים מסוג קבוצה).
-   ד. להסרת כפילויות משתמשים בשיטה `remove_duplicates()`, שניתן להשתמש בה רק עבור רשימות ממוינות.

**תשובה נכונה: ג**

**הסבר:**

להסרת כפילויות מרשימה מקושרת לא ממוינת בפייתון, ניתן להשתמש בשתי גישות עיקריות: שיטת שני המצביעים ושיטת השימוש בטבלת גיבוב (מבנה נתונים מסוג קבוצה).

1.  **שיטת שני המצביעים:**
    *   **שני מצביעים:** משתמשים בשני מצביעים למעבר על הרשימה.
    *   **`nextone` (מצביע חיצוני):** מתקדם על הרשימה ומצביע על האלמנט הנוכחי.
    *   **`runner` (מצביע פנימי):** משמש למעבר על הרשימה, החל מ-`nextone`, וחיפוש כפילויות עבור האלמנט הנוכחי.
    * **השוואת ערכים:** אם `runner` מוצא ערך כפול, הוא מוסר.
    * **לולאות מקוננות:** משתמשים בשתי לולאות `while` מקוננות למעבר על האלמנטים.
    *   **שינוי מצביעים:** כאשר כפילות מוסרת, המצביעים מופנים מחדש, להסרה נכונה של האלמנטים.
2.  **שיטה באמצעות טבלת גיבוב (מבנה נתונים מסוג קבוצה):**
    *   **קבוצה למעקב אחר ערכים ייחודיים:** נוצרת `set()` כדי לעקוב אילו אלמנטים כבר נבדקו (ערכים ייחודיים).
    *   **מעבר על הרשימה:** עוברים על הרשימה תוך שימוש בשלושה מצביעים: `current` עבור האלמנט הנוכחי, `prev` עבור הקודם, ו-`seen` עבור הערכים שכבר נבדקו.
    *   **בדיקה והסרה:** אם הערך הנוכחי קיים בקבוצה `seen` (כפילות), האלמנט הנוכחי מוסר; אם לא, הוא מתווסף לקבוצה ועוברים לאלמנט הבא.

**דוגמאות:**
```python
# Пример структуры данных - ListNode для связного списка
# דוגמה למבנה נתונים - ListNode עבור רשימה מקושרת
class ListNode:
    def __init__(self, val, next=None):
        self.val = val
        self.next = next

# Пример 1: Удаление дубликатов с помощью двух указателей
# דוגמה 1: הסרת כפילויות באמצעות שני מצביעים
def remove_duplicates_two_pointers(first):
   if not first:
       return

   nextone = first

   while nextone:
       runner = nextone
       while runner.next:
           if runner.next.val == nextone.val:
               runner.next = runner.next.next
           else:
               runner = runner.next
       nextone = nextone.next
   return first

# Пример 2: Удаление дубликатов с помощью хеш таблицы (множества)
# דוגמה 2: הסרת כפילויות באמצעות טבלת גיבוב (קבוצה)

def remove_duplicates_hash_table(list_head):
    if not list_head:
        return

    seen = set()
    current = list_head
    prev = None

    while current:
        if current.val in seen:
            # Remove the duplicate node
            # הסרת הצומת הכפול
            prev.next = current.next
        else:
            # Add to seen and move prev forward
            # הוספה ל-seen והזזת prev קדימה
            seen.add(current.val)
            prev = current
        current = current.next

    return list_head


# Пример списка для тестирования
# דוגמה לרשימה לבדיקה
list1 = ListNode(1, ListNode(2, ListNode(3, ListNode(2, ListNode(4, ListNode(1))))))

list2 = ListNode(1, ListNode(1, ListNode(1, ListNode(1))))


# Вывод и удаление дубликатов с двух указателей
# הדפסה והסרת כפילויות עם שני מצביעים
def print_list(head):
   while head:
      print(head.val, end = " ")
      head = head.next
   print("")
print("רשימה מקורית 1:")
print_list(list1)
remove_duplicates_two_pointers(list1)
print("רשימה 1 ללא כפילויות (שני מצביעים):")
print_list(list1) #  1 2 3 4
print("רשימה מקורית 2:")
print_list(list2)
remove_duplicates_two_pointers(list2)
print("רשימה 2 ללא כפילויות (שני מצביעים):")
print_list(list2) # 1


# Вывод и удаление дубликатов с хеш таблицей (множеством)
# הדפסה והסרת כפילויות עם טבלת גיבוב (קבוצה)
list1_copy = ListNode(1, ListNode(2, ListNode(3, ListNode(2, ListNode(4, ListNode(1))))))
list2_copy = ListNode(1, ListNode(1, ListNode(1, ListNode(1)))))

print("רשימה מקורית 1:")
print_list(list1_copy)
remove_duplicates_hash_table(list1_copy)
print("רשימה 1 ללא כפילויות (טבלת גיבוב):")
print_list(list1_copy) #  1 2 3 4

print("רשימה מקורית 2:")
print_list(list2_copy)
remove_duplicates_hash_table(list2_copy)
print("רשימה 2 ללא כפילויות (טבלת גיבוב):")
print_list(list2_copy) # 1
```
