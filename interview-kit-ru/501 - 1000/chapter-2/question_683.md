**שאלה 683.** פתח פונקציית Python המקבלת מספר שלם `n` כקלט ומחזירה את מספר האפסים בסוף עצרת המספר `n!` (n עצרת).

-   א. כדי לפתור את הבעיה יש לחשב את העצרת, ולאחר מכן לספור את מספר האפסים בסוף.
-   ב. כדי לפתור את הבעיה יש להשתמש בלולאה עם בדיקה של התחלקות ב-10, ובכל שלב שבו יש התחלקות ב-10, להגדיל את מונה האפסים.
-   ג. מספר האפסים בסוף עצרת המספר n! נקבע על ידי מספר זוגות הגורמים הראשוניים 2 ו-5, ומכיוון שתמיד יש פחות חמשות, מספיק למצוא את מספר החמשות בפיתוח לגורמים של n!.
-   ד. מספר האפסים בסוף עצרת n! שווה ל- n-1.

**התשובה הנכונה: ג**

**הסבר:**

מספר האפסים בסוף עצרת המספר `n!` נקבע על ידי מספר הפעמים שהמספר 10 מופיע בפיתוח העצרת לגורמים ראשוניים. מכיוון ש- 10 = 2 * 5, ובדרך כלל יש יותר שתיים מאשר חמשות, מספיק לקבוע את מספר החמשות בפיתוח לגורמים של n!

*   **עצרת:**
     *   עצרת המספר `n` (מסומן כ-`n!`) היא מכפלת כל המספרים הטבעיים עד `n`. לדוגמה `5! = 5 * 4 * 3 * 2 * 1 = 120`.
*   **מספר האפסים:**
    *  מספר האפסים בסוף מספר תלוי במספר העשרות.
    *   כל זוג של 2 * 5 יוצר כופל 10.
   *  בדרך כלל יש פחות חמשות בפיתוח העצרת מאשר שתיים, ולכן הן קובעות את מספר האפסים בסוף העצרת.
*   **ספירת מספר ה-5:**
    *  כדי לספור את מספר החמשות, נספור את מספר הכפולות של 5, מספר הכפולות של 25, מספר הכפולות של 125 וכן הלאה, עד n, ונחבר את התוצאות לפי הנוסחה: `floor(n / 5) + floor(n / 25) + floor(n/125) + ...`

**דוגמאות:**
```python
def count_trailing_zeros(n):
    count = 0
    i = 5
    while n // i >= 1: # כל עוד יש כפולות של i (5, 25, 125 וכן הלאה)
       count += n // i
       i *= 5
    return count


print(f"מספר האפסים בסוף 5!: {count_trailing_zeros(5)}")  # הפלט: מספר האפסים בסוף 5!: 1
print(f"מספר האפסים בסוף 10!: {count_trailing_zeros(10)}") # הפלט: מספר האפסים בסוף 10!: 2
print(f"מספר האפסים בסוף 25!: {count_trailing_zeros(25)}") # הפלט: מספר האפסים בסוף 25!: 6
print(f"מספר האפסים בסוף 125!: {count_trailing_zeros(125)}")  # הפלט: מספר האפסים בסוף 125!: 31

```
