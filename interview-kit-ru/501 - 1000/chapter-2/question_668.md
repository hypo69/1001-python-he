**שאלה 668.** כיצד לבדוק האם סוגריים זוגיים `()`, `[]`, `{}`, `<>` מסודרים נכונה במחרוזת, ואיזה אלגוריתם (שיטה) אופטימלי לכך?

- A. לבדיקת סידור הסוגריים הנכון, יש להשתמש בשיטת replace אחת בלבד להסרת כל זוגות הסוגריים.
- B. הפתרון האופטימלי לבדיקת סידור הסוגריים הנכון הוא שימוש במחסנית והסרה מתמדת של סוגריים זוגיים מהמחסנית.
- C. לבדיקת סידור הסוגריים הנכון, יש להשתמש במונה בלבד עבור כל סוג סוגרים, ואם ההפרש בין סוגריים פותחים לסוגרים סוגרים אינו שווה ל-0, אז המחרוזת אינה תקינה.
- D. בדיקת סידור הסוגריים הנכון אינה אפשרית ואין מנגנון ליישומה ב-Python.

**תשובה נכונה: B**

**הסבר:**

לבדיקת סידור הסוגריים הנכון במחרוזת (כאשר לכל סוגר פותח צריך להיות סוגר סוגר מתאים), הפתרון האופטימלי הוא שימוש במחסנית. גישה זו מאפשרת לעקוב ביעילות אחר רצף הסוגריים הפותחים והסוגרים.

* **אלגוריתם עם שימוש במחסנית:**
    1. **אתחול:** יוצרים מחסנית ריקה.
    2. **איטרציה:** עוברים על כל תו במחרוזת.
    3. **סוגר פותח:** אם התו הנוכחי הוא סוגר פותח (`(`, `[`, `{`, `<`), מכניסים אותו למחסנית.
    4. **סוגר סוגר:** אם התו הנוכחי הוא סוגר סוגר, מבצעים את הפעולות הבאות:
          * בודקים האם המחסנית אינה ריקה. אם היא ריקה, אז המחרוזת אינה תקינה.
          * מחפשים את האינדקס של הסוגר הסוגר, ואת הסוגר הפותח המתאים,
         * בודקים האם הסוגר הפותח האחרון במחסנית תואם לסוגר הסוגר. אם לא - אז המחרוזת אינה תקינה.
         * אם תואם, אז מסירים את הסוגר האחרון מהמחסנית.
    5. **תו לא תקין:** אם התו אינו סוגר, מדלגים עליו.
    6. **מחרוזת תקינה:** לאחר עיבוד כל המחרוזת, אם המחסנית ריקה, פירושו שכל הסוגריים מסודרים נכון, אחרת - לא.

**דוגמאות:**
```python
def check(string):
    brackets_open = ('(', '[', '{', '<')
    brackets_closed = (')', ']', '}', '>')
    stack = []
    for i in string:
        if i in brackets_open:
            stack.append(i)
        if i in brackets_closed:
            if len(stack) == 0:
                return False
            index = brackets_closed.index(i)
            open_bracket = brackets_open[index]
            if stack[-1] == open_bracket:
                stack = stack[:-1]
            else:
                return False
    return (not stack)

str1 = '[{([[[<>]]])(<>)(){}}]'
str2 = ']()(){<>}[[()]]'

print(f"Строка '{str1}' валидна: {check(str1)}")  # פלט: המחרוזת '[{([[[<>]]])(<>)(){}}]' תקינה: True
print(f"Строка '{str2}' валидна: {check(str2)}") # פלט: המחרוזת ']()(){<>}[[()]]' תקינה: False
str3 = "((((("
print(f"Строка '{str3}' валидна: {check(str3)}") # False, סוגריים פותחים רבים אך ללא סוגרים סוגרים
str4 = "))(("
print(f"Строка '{str4}' валидна: {check(str4)}") # False, אין סוגר פותח לפני הסוגרים הסוגרים.
str5 =  "({)}[]"
print(f"Строка '{str5}' валидна: {check(str5)}") # False, הסדר מופר.
str6 = "<><>"
print(f"Строка '{str6}' валидна: {check(str6)}") # True

```
