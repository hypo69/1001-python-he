### `question_834.md`

**שאלה 834.** מה יודפס למסך כתוצאה מהרצת קוד ה-Python הבא?

```python
def foo():
  return total + 1

total = 0
print(foo())
```

א. `0`
ב. `1`
ג. `שגיאה`
ד. אחר

**תשובה נכונה: B**

**הסבר:**

קוד זה מדגים את היקף (scope) המשתנים ב-Python וכיצד פונקציה יכולה לגשת למשתנים גלובליים.

1.  **`def foo():`**: מוגדרת הפונקציה `foo`.
2.  **`return total + 1`**: בתוך הפונקציה `foo` מתבצעת גישה למשתנה `total`, שאינו מוגדר מקומית בתוך `foo`. Python יחפש אותו ב*היקף הגלובלי*.
3.  **`total = 0`**: המשתנה הגלובלי `total` מאותחל לערך `0`. חשוב לציין שזה קורה *לפני הקריאה* לפונקציה `foo`.
4.  **`print(foo())`**:
    *   נקראת הפונקציה `foo`.
    *   הפונקציה `foo` מקבלת גישה למשתנה הגלובלי `total` (ששווה ל-0), מוסיפה לו 1, ומחזירה את התוצאה (1).
    *   `print()` מדפיסה את הערך המוחזר למסך.

*   **אפשרות א' שגויה:** הפונקציה מוסיפה `1` לערך של `total`.
*   **אפשרות ב' נכונה:** `foo` מחזירה את הערך `1`, והוא זה שיודפס.
*   **אפשרות ג' שגויה:** הקוד תקין ולא אמור לגרום לשגיאה. היתה מתעוררת בעיה אם `total` לא היה מוגדר לפני הקריאה ל-`foo`.
*   **אפשרות ד' שגויה:** התוצאה צפויה ואין סיבה להניח שתהיה אפשרות אחרת.

**היקף וסדר ביצוע:**

1.  אם משתנה אינו נמצא בהיקף המקומי של הפונקציה, Python מחפש אותו בהיקף הגלובלי.
2.  חשוב שהמשתנה יהיה מוגדר *לפני* שפונים אליו.

**לסיכום:**

מכיוון שהמשתנה `total` מוגדר בהיקף הגלובלי לפני הקריאה ל-`foo()`, הפונקציה תקבל אליו גישה בהצלחה ותחזיר `1`, שיוצג על המסך.

לפיכך, אפשרות ב' היא הנכונה.
