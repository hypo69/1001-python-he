### `question_885.md`

**שאלה 885.** בפייתון, אם מיושמים מספר דקורטורים על אותה פונקציה, באיזה סדר הם יבוצעו?

א. הדקורטורים מבוצעים לפי סדר הכרזתם, כלומר מלמעלה למטה. הדקורטור העליון ביותר מיושם ראשון.
ב. הדקורטורים מבוצעים בסדר הפוך מזה של הכרזתם, כלומר מלמטה למעלה. הדקורטור התחתון ביותר מיושם ראשון.
ג. סדר ביצוע הדקורטורים נקבע לפי סדר הייבוא שלהם לקובץ, ולא לפי סדר הכתיבה בקוד.
ד. סדר ביצוע הדקורטורים תלוי בסדר העדיפות שלהם, שנקבע לפי שמות הדקורטורים.

**תשובה נכונה: B**

**הסבר:**

בפייתון, כאשר מיושמים מספר דקורטורים על אותה פונקציה, הם מיושמים בסדר *הפוך* לזה של הכרזתם, כלומר *מלמטה למעלה*. הסיבה לכך היא שכל דקורטור עוטף את הפונקציה בשכבה חדשה של פונקציונליות.

*   **כיצד פועלים דקורטורים:**
    *   דקורטור הוא פונקציה שמקבלת פונקציה אחרת כארגומנט, מוסיפה לה פונקציונליות מסוימת ומחזירה פונקציה חדשה.
    *   כאשר דקורטור מיושם על פונקציה, למעשה הפונקציה מוחלפת בתוצאת הקריאה לדקורטור עם פונקציה זו.

*   **סדר ביצוע:**
    *   אם מיושמים מספר דקורטורים על פונקציה, הרי שקודם כל מיושם הדקורטור *התחתון ביותר*, ולאחר מכן, ברצף, כל שאר הדקורטורים, עד שמגיעים לדקורטור העליון ביותר.
    *   כל דקורטור כאילו "עוטף" את התוצאה של הדקורטור הקודם.

*   **אפשרות א' אינה נכונה:** קודם כל מבוצע הדקורטור התחתון.
*   **אפשרות ב' נכונה:** זו התשובה הנכונה.
*   **אפשרות ג' אינה נכונה:** הסדר נקבע רק לפי המיקום בקוד.
*   **אפשרות ד' אינה נכונה:** הסדר תלוי במיקום בקוד, ולא בסדר עדיפות.

**דוגמה:**

```python
def decorator1(func):
    def wrapper(*args, **kwargs):
        print("Decorator 1 is running")
        return func(*args, **kwargs)
    return wrapper

def decorator2(func):
    def wrapper(*args, **kwargs):
        print("Decorator 2 is running")
        return func(*args, **kwargs)
    return wrapper

@decorator1
@decorator2
def my_function():
    print("my_function is running")

my_function()
# Output:
# Decorator 1 is running
# Decorator 2 is running
# my_function is running
```

**כתוצאה מכך:**

הדקורטורים מבוצעים מלמטה למעלה.

לפיכך, אפשרות ב' היא הנכונה.

**שינוי:**

*   אפשרות ג' הוחלפה ב"סדר ביצוע הדקורטורים נקבע לפי סדר הייבוא שלהם לקובץ, ולא לפי סדר הכתיבה בקוד". אפשרות זו כבר נשמעת מספיק סבירה ועשויה להטעות אם לא יודעים את התשובה הנכונה.
