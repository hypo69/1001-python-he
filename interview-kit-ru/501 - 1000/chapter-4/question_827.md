### `question_827.md`

**שאלה 827**

איזו מהמבנים המיוצגים של קוד פייתון משתמש בצורה *הנכונה ביותר* בבלוק `try...except KeyboardInterrupt` לטיפול בלחיצה על שילוב המקשים CTRL+C ולביצוע פעולות מסוימות לפני סיום התוכנית?

- A
    ```python
    try:
        while True:
            print("Работаю...")
    except:
        print("Прервано!")
    ```

- B
    ```python
    while True:
        try:
            print("Работаю...")
        except KeyboardInterrupt:
            print("Прервано!")
            break
    ```

- C
    ```python
    try:
        while True:
            print("Работаю...")
    except KeyboardInterrupt:
        pass
    finally:
        print("Прервано!")
    ```

- D
    ```python
    try:
        print("Работаю...")
    except KeyboardInterrupt:
        print("Прервано!")
        break
    ```

**תשובה נכונה:**

*   **B**

**הסבר:**

אפשרות B היא הטובה ביותר, מכיוון שהיא מאפשרת ללולאה הראשית להמשיך לפעול עד ללחיצה על שילוב המקשים CTRL+C. חריגת `KeyboardInterrupt` מתרחשת בלחיצה על CTRL+C. בלוק `except` מטפל בחריגה זו, מדפיס את ההודעה "Прервано!", והפקודה `break` קוטעת את הלולאה האינסופית `while True`. אפשרות A לוכדת את כל החריגות, וזה אינו מומלץ, מכיוון שעלולים לפספס שגיאות חשובות. אפשרות C מתעלמת מחריגת `KeyboardInterrupt`, ובלוק `finally` יתבצע בכל מקרה, ללא קשר אם נלחץ CTRL+C. אפשרות D תדפיס "Работаю...", ולאחר מכן בלוק `except KeyboardInterrupt` יתבצע מיד, מכיוון שהלולאה אינה אינסופית ומסתיימת מיידית.

**דוגמה:**

```python
import time

try:
    while True:
        print("Работаю...")
        time.sleep(1)  # הדמיית עבודה
except KeyboardInterrupt:
    print("\nהתוכנית הופסקה על ידי המשתמש.")
    print("מבצע פעולות סיום...")
    # כאן ניתן למקם קוד לשמירת נתונים, סגירת קבצים וכו'.
    print("סיום...")
```

דוגמה זו תדפיס אינסופית "Работаю...", עד שהמשתמש ילחץ על CTRL+C. לאחר לחיצה על CTRL+C, הביצוע יעבור לבלוק `except`, שם תודפס הודעה על ההפסקה ויתבצע קוד פעולות הסיום.

**מסקנה:**

`try...except KeyboardInterrupt` היא דרך פשוטה ויעילה לטפל בלחיצה על CTRL+C בפייתון. נקודת המפתח היא למקם את כל הקוד שעלול להיקטע בבלוק `try`, ואת קוד פעולות הסיום בבלוק `except`. ניתן להשתמש בפקודה `break` ליציאה מהלולאה לאחר הטיפול בחריגה. גישה זו מאפשרת לתוכנית לסיים את פעולתה בצורה תקינה, ולא רק להיעצר באופן בלתי מבוקר.
