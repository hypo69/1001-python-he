**שאלה 564.** מהי הערכה עצלה (lazy evaluation) ב-Python, וכיצד היא פועלת? אנא הצג דוגמה המדגימה שימוש בחישובים עצלים.

-   א. הערכה עצלה היא שיטת הערכה שבה ביטויים מוערכים מיד לאחר הגדרתם.
-   ב. הערכה עצלה היא אסטרטגיית חישוב שבה ביטוי מוערך רק כאשר ערכו נדרש. ב-Python ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.
-   ג. הערכה עצלה היא כאשר נעשה שימוש רק במשתנים שאליהם קיימת פנייה.
-   ד. הערכה עצלה היא שיטה להאצת חישובים על ידי ביצועם במקביל.

**תשובה נכונה: ב**

**הסבר:**

הערכה עצלה (lazy evaluation) היא אסטרטגיית חישוב שבה ביטוי אינו מוערך באופן מיידי, אלא נדחה עד לרגע שבו ערכו נחוץ באמת. אסטרטגיה זו מאפשרת חיסכון במשאבים חישוביים, שכן חישובים אינם מבוצעים עד אשר התוצאה שלהם נדרשת בפועל.

*   **מושגי יסוד בהערכה עצלה:**
    *   **חישובים מושהים:** ערכי ביטויים אינם מוערכים מיד, אלא רק כאשר הם נחוצים.
    *   **חיסכון במשאבים:** מאפשר לחסוך בזיכרון ובזמן מעבד על ידי חישוב רק של הערכים הנחוצים.
    *   **סדרות אינסופיות:** מאפשר הצגת סדרות נתונים אינסופיות, שכן ערכים מוערכים לפי דרישה.
    *   **שימוש בגנרטורים ואיטרטורים:** Python מאפשרת מימוש חישובים עצלים באמצעות גנרטורים ואיטרטורים.

*   **גנרטורים וחישובים עצלים:**
    *   גנרטורים הן פונקציות מיוחדות המשתמשות ב-`yield` כדי להחזיר ערכים בזה אחר זה.
    *   גנרטורים מחשבים את הערך הבא רק כאשר הפונקציה `next` נקראת או מתרחשת איטרציה בלולאת `for`, ולא מחשבים את כל הערכים בבת אחת.

**דוגמה (מטקסט השאלה):**

```python
def fibonacci():
    # פונקציית גנרטור המייצרת סדרה אינסופית של מספרי פיבונאצ'י.
    a, b = 0, 1
    while True:
        # לולאה אינסופית
        yield a
        # מחזירה את מספר פיבונאצ'י הבא ומשהה את ביצוע הפונקציה.
        a, b = b, a + b

fib_gen = fibonacci()
# יוצר אובייקט גנרטור.
for i in range(10):
# לולאה שתתבצע 10 פעמים.
    print(next(fib_gen))
    # קורא לגנרטור, גורם לו לחשב ולהחזיר את ערך פיבונאצ'י הבא.
# הגנרטור מייצר ערכים לפי דרישה ורק כאשר הם נחוצים ללולאה.
```

**תיאור הדוגמה:**

*   **`fibonacci()`:** פונקציית גנרטור המייצרת סדרה אינסופית של מספרי פיבונאצ'י.
    *   `while True`: לולאה אינסופית
    *   `yield a`: מחזירה את מספר פיבונאצ'י הבא ומשהה את ביצוע הפונקציה.
*   `fib_gen = fibonacci()`: יוצר אובייקט-גנרטור.
*   `for i in range(10)`: לולאה שתתבצע 10 פעמים.
*   `next(fib_gen)`: קורא לגנרטור, גורם לו לחשב ולהחזיר את ערך פיבונאצ'י הבא.
*   הגנרטור מייצר ערכים לפי דרישה ורק כאשר הם נחוצים ללולאה.

**דוגמאות נוספות:**
```python
# חישוב עצל באמצעות range
numbers = range(10000000000) # יוצר range, אך המספרים אינם מחושבים
print(numbers[0]) # חישובים אינם מתרחשים עד לרגע זה
print(numbers[1]) # range מחשב באופן עצלני אלמנטים לפי דרישה.

# חישובים עצלים באמצעות map
numbers = [1,2,3,4]
squared_numbers = map(lambda x: x**2, numbers) # map אינו מחשב את הערכים, אלא רק מחזיר איטרטור
print(list(squared_numbers)) # ממירים לרשימה כדי לחשב את כל ריבועי המספרים.
```
