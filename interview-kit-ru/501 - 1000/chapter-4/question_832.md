### `question_89.md`

**שאלה 89.** כיצד ניתן לאחד שתי רשימות ב-Python?

- א. `list3 = list1 + list2`
- ב. `list3 = list1.append(list2)`
- ג. `list3 = concat(list1, list2)`
- ד. `list3 = list1.extend(list2)`

**תשובה נכונה: א**

**הסבר:**

ב-Python, לאיחוד שתי רשימות משתמשים באופרטור `+` או במתודה `extend()`.

*   **אפשרות א** נכונה: האופרטור `+` יוצר רשימה חדשה, שהיא התוצאה של איחוד שתי הרשימות המקוריות.

*   **אפשרות ב** אינה נכונה: המתודה `append()` מוסיפה את הרשימה השנייה כאובייקט יחיד בסוף הרשימה הראשונה, מבלי לאחד את האלמנטים שלהן.

*   **אפשרות ג** אינה נכונה: הפונקציה `concat()` אינה פונקציה מובנית ב-Python.

*   **אפשרות ד** נכונה: המתודה `extend()` מוסיפה את כל האלמנטים מהרשימה השנייה לסוף הרשימה הראשונה, ובכך משנה את הרשימה המקורית.

**אופרטור `+`:**

1.  יוצר רשימה חדשה, המאחדת את כל האלמנטים משתי הרשימות לרשימה חדשה אחת.
2.  אינו משנה את הרשימות המקוריות.

**מתודת `extend()`:**

1.  מוסיפה את כל האלמנטים מהרשימה השנייה לסוף הרשימה הראשונה, *תוך שינוי* הרשימה הראשונה.
2.  אינה מחזירה רשימה חדשה, אלא משנה את הרשימה הקיימת.

**דוגמה:**

```python
# Пример с оператором +
list1: list[int] = [1, 2, 3]
list2: list[int] = [4, 5, 6]
list3: list[int] = list1 + list2
print(f"list1: {list1}")
print(f"list2: {list2}")
print(f"list1 + list2: {list3}")
# Вывод:
# list1: [1, 2, 3]
# list2: [4, 5, 6]
# list1 + list2: [1, 2, 3, 4, 5, 6]

# Пример с методом extend()
list4: list[int] = [1, 2, 3]
list5: list[int] = [4, 5, 6]
list4.extend(list5)
print(f"list4 после extend(list5): {list4}") # Вывод: list4 после extend(list5): [1, 2, 3, 4, 5, 6]
```

**לסיכום:**
*  האופרטור `+` יוצר רשימה *חדשה* עם אלמנטים מ-`list1` ומ-`list2`, מבלי לשנות את המקוריות.
*   `extend` משנה את `list4` במקום.

לפיכך, **אפשרות א** היא האפשרות הנכונה ליצירת רשימה חדשה על ידי איחוד שתי רשימות מקוריות. אפשרות ד גם נכונה, אך היא משנה את הרשימה המקורית ולא יוצרת רשימה חדשה.
