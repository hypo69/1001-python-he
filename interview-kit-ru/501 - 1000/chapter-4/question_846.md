### `question_75.md`

**פרק 1. שאלה 75.** בפייתון, מה ההבדל בין הפונקציה `sorted()` לבין המתודה `sort()`?

- א. ניתן להשתמש ב-`sorted()` עבור כל אובייקט בר-חזרה (iterable), בעוד ש-`sort()` היא מתודה המיושמת באופן ספציפי רק לרשימות.
- ב. `sorted()` ממיינת רשימות בסדר יורד, בעוד ש-`sort()` ממיינת רשימות בסדר עולה.
- ג. `sorted()` מחזירה רשימה חדשה, בעוד ש-`sort()` משנה את הרשימה במקום.
- ד. א' ו-ג' נכונים.

**תשובה נכונה: ד'**

**הסבר:**

בפייתון, קיימות שתי דרכים עיקריות למיון רצפים: הפונקציה `sorted()` והמתודה `sort()`. יש להן הבדלים חשובים בהתנהגותן.

* אפשרות א' נכונה: `sorted()` יכולה לעבוד עם כל אובייקט בר-חזרה (רשימה, טאפל, מחרוזת, סט), ו-`sort()` היא מתודה ספציפית לרשימות.
* אפשרות ב' אינה נכונה: `sorted()` ו-`sort()` ממיינות כברירת מחדל בסדר עולה. למיון בסדר הפוך, יש להשתמש בארגומנט `reverse=True`.
* אפשרות ג' נכונה: `sorted()` מחזירה רשימה חדשה ממוינת, מבלי לשנות את הרשימה המקורית, בעוד ש-`sort()` משנה את הרשימה במקום, מבלי להחזיר חדשה.
* אפשרות ד' נכונה: אפשרויות א' ו-ג' נכונות, ולכן אפשרות ד' היא התשובה הנכונה.

**הפונקציה `sorted()`:**

1. מקבלת כארגומנט אובייקט בר-חזרה (רשימה, טאפל, מחרוזת וכו').
2. מחזירה רשימה ממוינת *חדשה*, והאובייקט המקורי אינו משתנה.
3. אינה מתודה של רשימה, אלא פונקציה מובנית.

**המתודה `sort()`:**

1. נקראת על אובייקט מסוג רשימה.
2. ממיינת את הרשימה *עצמה במקום*, משנה את סדר האיברים ברשימה המקורית.
3. אינה מחזירה רשימה חדשה (מחזירה `None`).

**דוגמאות:**

```python
# דוגמה עם sorted()
my_list: list[int] = [3, 1, 4, 1, 5, 9, 2, 6]
sorted_list: list[int] = sorted(my_list)
print(f"Исходный список: {my_list}")   # פלט: רשימה מקורית: [3, 1, 4, 1, 5, 9, 2, 6]
print(f"Отсортированный список: {sorted_list}") # פלט: רשימה ממוינת: [1, 1, 2, 3, 4, 5, 6, 9]
print(type(sorted_list)) # פלט: <class 'list'>
# דוגמה עם sort()
my_list: list[int] = [3, 1, 4, 1, 5, 9, 2, 6]
my_list.sort() # ממיין במקום
print(f"Список после sort(): {my_list}") # פלט: רשימה לאחר sort(): [1, 1, 2, 3, 4, 5, 6, 9]
```

**בסיכום:**

* `sorted(my_list)` יוצרת רשימה חדשה ממוינת. הרשימה המקורית `my_list` אינה משתנה.
* `my_list.sort()` ממיינת את הרשימה `my_list` במקום, משנה את סדר האיברים שלה.

לפיכך, **אפשרות ד'** היא התשובה הנכונה, שכן אפשרויות א' ו-ג' נכונות.
