**פרק 2. שאלה 4.** מה עושה האופרטור `break` בתוך לולאה בפייתון?

- A.  הוא יוצא מהלולאה הנוכחית וממשיך את ביצוע הפקודה הבאה מחוץ ללולאה.
- B.  הוא מדלג על האיטרציה הנוכחית ועובר ישירות לאיטרציה הבאה בלולאה.
- C.  הוא משעה זמנית את ביצוע הלולאה ומחדש אותו רק כאשר תנאי מסוים משתנה.
- D.  הוא מפעיל מחדש את הלולאה מהשורה הראשונה עם התנאי המקורי.

**תשובה נכונה: A**

**הסבר:**

האופרטור `break` משמש להפסקת (סיום) מיידית של הלולאה הנוכחית שבה הוא נמצא.

*   **אפשרות A** נכונה: האופרטור `break` מסיים את ביצוע הלולאה ומעביר את השליטה לפקודה הבאה לאחר הלולאה.
*   **אפשרות B** אינה נכונה: לדילוג על איטרציה משתמשים ב-`continue`, לא ב-`break`.
*   **אפשרות C** אינה נכונה: `break` אינו משעה את הלולאה, אלא מסיים אותה.
*   **אפשרות D** אינה נכונה: `break` אינו מפעיל מחדש את הלולאה, אלא יוצא ממנה.

**כיצד פועל `break`:**

1.  כאשר פייתון נתקל באופרטור `break` בתוך לולאה (`for` או `while`), ביצוע הלולאה נפסק באופן מיידי.
2.  השליטה מועברת לפקודה הבאה לאחר הלולאה.
3.  כל האיטרציות הנותרות של הלולאה מדלגות.

**מקרים אופייניים לשימוש:**

*   **חיפוש אלמנט:** סיום הלולאה מיד לאחר מציאת האלמנט המבוקש.
*   **בדיקת תנאים:** יציאה מהלולאה אם מתקיים תנאי המחייב סיום מיידי.
*   **מניעת לולאה אינסופית:** יציאה מלולאה אינסופית.

**דוגמה:**

```python
# דוגמה לשימוש ב-break בלולאת for
numbers: list[int] = [1, 2, 3, 4, 5, 6]
for number in numbers:
    if number > 4:
        print("המספר גדול מ-4. יציאה מהלולאה.")
        break
    print(number)

# פלט:
# 1
# 2
# 3
# 4
# המספר גדול מ-4. יציאה מהלולאה.

# דוגמה לשימוש ב-break בלולאת while
x: int = 0
while True:
    x += 1
    if x > 5:
      break
    print(f"מספר: {x}")
# פלט:
# מספר: 1
# מספר: 2
# מספר: 3
# מספר: 4
# מספר: 5
```

**בתוצאה:**
*   בלולאת `for`, ברגע שמשתנה הלולאה הופך להיות גדול מ-4, מבוצע `break` והלולאה מסתיימת, כאשר המספר 5 לא יודפס.
*   בלולאת `while`, ברגע שמשתנה `x` הופך להיות גדול מ-5, מבוצע `break` והלולאה מסתיימת.

לפיכך, **אפשרות A** היא הנכונה.
