### `question_274_interview.md`

**שאלה 274.** האם קוד הפייתון הבא תקין, ומה יודפס למסך אם יבוצע?

```python
a = 2, 3, 4, 5
a
```

A. כן, יוצג במסוף `2`
B. כן, יוצג במסוף `[2, 3, 4, 5]`
C. כן, יוצג במסוף `(2, 3, 4, 5)`
D. לא, יותר מדי ערכים

**תשובה נכונה: C**

**הסבר:**

בפייתון, כאשר מספר ערכים מופרדים בפסיקים בעת השמה, נוצר אוטומטית *קורטז* (tuple). בקוד הנתון, למשתנה `a` יוּשׁם קורטז המורכב מהמספרים `2`, `3`, `4` ו-`5`. הביטוי `a` במסוף יוביל להצגת ייצוגו של קורטז זה.

*   **אפשרות A שגויה:** יוצג קורטז, ולא רק הערך הראשון.
*   **אפשרות B שגויה:** יוצג קורטז, ולא רשימה.
*   **אפשרות C נכונה:** הקוד תקין, והתוצאה תהיה הצגת הקורטז `(2, 3, 4, 5)`.
*  **אפשרות D שגויה:** הקוד תקין. פייתון אורז אוטומטית מספר רב של ערכים המופרדים בפסיקים לקורטז.

**איך עובדת השמת מספר ערכים בפייתון:**

1.  כאשר השמה בפייתון נראית כך: `variable = value1, value2, value3`, פייתון יוצר אוטומטית קורטז `(value1, value2, value3)` ומבצע השמה שלו למשתנה `variable`.
2.  הצגת שם המשתנה במצב אינטראקטיבי (במסוף) תציג את הקורטז בסוגריים עגולים.

**דוגמה:**

```python
a = 2, 3, 4, 5
print(type(a)) # Output: <class 'tuple'>
print(a) # Output: (2, 3, 4, 5)
b = [2, 3, 4, 5]
print(type(b)) # Output: <class 'list'>
print(b) # Output: [2, 3, 4, 5]
```
**לסיכום:**

הקוד תקין, מכיוון שפייתון יוצר אוטומטית קורטז בעת השמת מספר ערכים המופרדים בפסיקים. בעת הצגת ערך המשתנה `a` במסוף, יוצג הקורטז `(2, 3, 4, 5)`.

לפיכך, אפשרות C היא הנכונה.
