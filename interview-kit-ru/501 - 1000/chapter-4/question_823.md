### `question_278_interview.md`

**שאלה 278.** מה יודפס למסך כתוצאה מהרצת קוד Python הבא?

```python
result = 0
for i in range(5):
    result += 2 * i
print(result)
```

A. 10
B. 15
C. 20
D. 25

**תשובה נכונה: C**

**הסבר:**

קוד זה מחשב את סכום הערכים, המתקבלים כתוצאה מהכפלת מספרים בטווח שבין 0 ל-4 פי 2.

1.  **`result = 0`**:
    *   מאותחל המשתנה `result` בערך 0. משתנה זה ישמש לצבירת הסכום.

2.  **`for i in range(5):`**:
    *   לולאת `for` עוברת על הערכים, המוחזרים על ידי פונקציית `range(5)`.
    *   פונקציית `range(5)` יוצרת רצף של מספרים שלמים מ-0 (כולל) עד 5 (לא כולל), כלומר, 0, 1, 2, 3, 4.
    *   המשתנה `i` יקבל לסירוגין ערכים אלה.
3.  **`result += 2 * i`**:
    *   מפעיל ההשמה המורכב `+=` שקול ל-`result = result + 2 * i`.
    *   בכל איטרציה של הלולאה:
        *   מחושב `2 * i`.
        *   תוצאת הכפל מתווספת לערך הנוכחי של `result`.

4.  **`print(result)`**:
    *   מודפס הערך הסופי של המשתנה `result` למסך לאחר סיום הלולאה.

**חישוב ערך המשתנה `result`:**

*   איטרציה 1: `i = 0`, `result += 2 * 0`, `result = 0`
*   איטרציה 2: `i = 1`, `result += 2 * 1`, `result = 2`
*   איטרציה 3: `i = 2`, `result += 2 * 2`, `result = 6`
*   איטרציה 4: `i = 3`, `result += 2 * 3`, `result = 12`
*   איטרציה 5: `i = 4`, `result += 2 * 4`, `result = 20`

*   **אפשרות A אינה נכונה:** 10 — תוצאת חישובים שגויה.
*   **אפשרות B אינה נכונה:** 15 — תוצאת חישובים שגויה.
*  **אפשרות C נכונה:** 20 — תוצאת חישובים נכונה.
*   **אפשרות D אינה נכונה:** 25 — תוצאת חישובים שגויה.

**לסיכום:**

לאחר ביצוע הלולאה, יודפס למסך ערך המשתנה `result`, השווה ל-`20`.

לפיכך, אפשרות C הינה התשובה הנכונה.
