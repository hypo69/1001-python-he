**שאלה 826.** מה יודפס למסך כתוצאה מהרצת קוד הפייתון הבא?

```python
print(not None != "")
```

A.  `None`
B.  `False`
C.  `True`
D.  `Error`

**תשובה נכונה: C**

**הסבר:**

קוד זה מדגים שימוש באופרטור הלוגי `not`, באופרטור אי-שוויון `!=`, ובבדיקת האמת של `None` ומחרוזת ריקה `""` בפייתון.

1.  **`not None`**:
    *   `None` - היא מילת מפתח מיוחדת בפייתון המייצגת היעדר ערך. בהקשר בוליאני, ל-`None` יש ערך `False`.
    *   האופרטור `not` הופך את הערך הבוליאני, ולכן `not None` שווה ל-`True`.

2.  **`""`**:
    *   למחרוזת הריקה `""`, בהקשר בוליאני, יש ערך `False`.

3.  **`True != False`**:
    *   האופרטור `!=` בודק אי-שוויון. `True` אינו שווה ל-`False`, ולכן ביטוי זה מוערך ל-`True`.

4.  **`print(True)`**:
    *   הערך `True` יודפס למסך.

*   **אפשרות A שגויה:** הביטוי לא יחזיר `None`, אלא יחזיר `True` או `False`.
*   **אפשרות B שגויה:** תוצאת הביטוי היא `True`.
*   **אפשרות C נכונה:** זוהי התוצאה הנכונה.
*   **אפשרות D שגויה:** הקוד תקין תחבירית ולא יגרום לשגיאה.

**סדר הערכת הביטוי:**

1.  ראשית מוערך `not None`, התוצאה היא `True`.
2.  לאחר מכן המחרוזת הריקה `""` מומרת לערך בוליאני `False`.
3.  משווים לאי-שוויון: `True != False` שתוצאתו היא `True`.
4.  הפונקציה `print` מדפיסה את הערך `True`.

**כתוצאה מכך:**
למסך יודפס `True`, מכיוון שהביטוי `not None != ""` מוערך כ-`True != False` ולאחר מכן מוערך כ-`True`.

לפיכך, אפשרות C היא הנכונה.
