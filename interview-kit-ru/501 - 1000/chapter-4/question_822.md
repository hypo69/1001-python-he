### `question_277_interview.md`

**שאלה 277.** מה יוצג על המסך כתוצאה מהרצת קוד ה-Python הבא?

```python
a = {4, 5, 6}
b = {2, 8, 6}
print(a + b)
```

A.  `{4, 5, 6, 2, 8}`
B.  `{4, 5, 6, 2, 8, 6}`
C.  `Error`
D.  `{6, 13, 12}`

**תשובה נכונה: C**

**הסבר:**

קוד זה מדגים ניסיון שגוי להשתמש באופרטור החיבור `+` לאיחוד קבוצות ב-Python. האופרטור `+` אינו מוגדר עבור קבוצות. לאיחוד קבוצות משתמשים במתודה `union()` או באופרטור `|`.

1.  **`a = {4, 5, 6}`:**
    * נוצרת קבוצה `a` עם האיברים 4, 5 ו-6.
2.  **`b = {2, 8, 6}`:**
    * נוצרת קבוצה `b` עם האיברים 2, 8 ו-6.
3.  **`print(a + b)`:**
    * מתבצע ניסיון לחבר את הקבוצות `a` ו-`b` באמצעות האופרטור `+`.
    * האופרטור `+` אינו פועל עבור קבוצות, ולכן Python תעלה שגיאת `TypeError`.

*   **אפשרות A אינה נכונה:** האופרטור `+` לא יבצע איחוד קבוצות.
*   **אפשרות B אינה נכונה:** האופרטור `+` לא יבצע איחוד קבוצות.
*   **אפשרות C נכונה:** הקוד יגרום לשגיאת `TypeError`, מכיוון שהאופרטור `+` אינו מוגדר עבור קבוצות.
*   **אפשרות D אינה נכונה:** האופרטור `+` אינו מבצע חיבור איברים.

**דרכים נכונות לאיחוד קבוצות:**

1.  **באמצעות האופרטור `|` (איחוד):**
    ```python
    a = {4, 5, 6}
    b = {2, 8, 6}
    c = a | b # or a.union(b)
    print(c) # Output: {2, 4, 5, 6, 8}
    ```
    האופרטור `|` או המתודה `.union()` מחזירים קבוצה חדשה המכילה את כל האיברים משתי הקבוצות.

2.  **באמצעות המתודה `.union()`:**
    ```python
        a = {4, 5, 6}
        b = {2, 8, 6}
        c = a.union(b) # or a | b
        print(c) # Output: {2, 4, 5, 6, 8}
    ```
    בדומה לאופרטור `|`.

**לסיכום:**

ניסיון לחבר קבוצות באמצעות האופרטור `+` גורם לשגיאת `TypeError`. לאיחוד קבוצות יש להשתמש באופרטור `|` או במתודה `union()`.

לכן, אפשרות C היא הנכונה.
