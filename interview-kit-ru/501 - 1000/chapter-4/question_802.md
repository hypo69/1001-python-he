### `question_802.md`

**שאלה 802.** מה עושה המתודה `__init__` במחלקה בפייתון?

א. יוצרת מופע חדש של אובייקט המחלקה.
ב. מחזירה ייצוג מחרוזתי של האובייקט.
ג. מאתחלת את האובייקט בעת יצירתו.
ד. מסירה את האובייקט מהזיכרון.

**התשובה הנכונה: C**

**הסבר:**

המתודה `__init__` היא מתודה מיוחדת (קונסטרוקטור) בפייתון, הנקראת אוטומטית בעת יצירת מופע (אובייקט) חדש של המחלקה. המטרה העיקרית של מתודה זו היא אתחול אטריבוטי האובייקט והגדרת מצבו ההתחלתי.

*   **אפשרות א** אינה נכונה: על יצירת אובייקט חדש אחראית המתודה `__new__`, ואילו `__init__` רק מאתחלת אותו.
*   **אפשרות ב** אינה נכונה: ייצוג מחרוזתי של האובייקט נוצר על ידי המתודה `__str__()` או `__repr__()`.
*   **אפשרות ג** נכונה: המתודה `__init__` משמשת לאתחול האובייקט.
*   **אפשרות ד** אינה נכונה: להסרת אובייקט מהזיכרון משתמשים במתודה `__del__()`.

**כיצד פועלת `__init__`:**

1.  המתודה `__init__` נקראת אוטומטית על ידי מפרש פייתון בעת יצירת מופע חדש של המחלקה.
2.  הפרמטר הראשון של `__init__` הוא תמיד `self`, המפנה לאובייקט שנוצר.
3.  בתוך המתודה `__init__` לרוב מוגדרים הערכים ההתחלתיים של אטריבוטי האובייקט.
4. המתודה יכולה לקבל פרמטרים נוספים, שיעברו בעת יצירת מופע האובייקט.

**דוגמה:**

```python
class Person:
    def __init__(self, name: str, age: int):
        self.name = name  # אתחול אטריבוט name
        self.age = age    # אתחול אטריבוט age
        print("האובייקט נוצר!")

person1: Person = Person("Alice", 30) # פלט: האובייקט נוצר!
print(f"שם: {person1.name}, גיל: {person1.age}") # פלט: שם: Alice, גיל: 30
```

**כתוצאה מכך:**

*   בעת יצירת האובייקט `person1`, נקראת אוטומטית המתודה `__init__()`, המאתחלת את האטריבוטים `name` ו-`age`.

לפיכך, **אפשרות ג** היא התשובה הנכונה.
