### `question_176.md`

**שאלה 176.** מה תפקידן של מחרוזות תיעוד (docstrings) בפייתון?

א. הן משמשות להוספת הערות לקוד ואין להן משמעות פונקציונלית.
ב. הן משמשות להגדרת טיפוסי משתנים בתוך פונקציות.
ג. הן משמשות ליצירה אוטומטית של תיעוד עבור פונקציות, מחלקות ומודולים.
ד. הן משמשות לאופטימיזציה של קוד, מה שהופך אותו ליעיל יותר.

**התשובה הנכונה: ג**

**הסבר:**

מחרוזות תיעוד (docstrings) הן מחרוזות מרובות שורות המשמשות לתיעוד קוד בפייתון. הן נכתבות בתוך הגדרות של פונקציות, מחלקות, מודולים ואלמנטים אחרים של קוד.

*   **אפשרות א** אינה נכונה: מחרוזות תיעוד אינן משמשות רק להוספת הערות לקוד, אלא גם משמשות ליצירת תיעוד.
*   **אפשרות ב** אינה נכונה: להגדרת טיפוסי משתנים משתמשים באנוטציות טיפוס (type hints).
*   **אפשרות ג** נכונה: מחרוזות תיעוד משמשות ליצירת תיעוד.
*   **אפשרות ד** אינה נכונה: מחרוזות תיעוד אינן משפיעות על ביצועי הקוד.

**כיצד משתמשים במחרוזות תיעוד:**

1.  מחרוזות תיעוד הן מחרוזות המוקפות בגרשיים משולשים (`"""Docstring"""` או `'''Docstring'''`).
2.  הן נכתבות מיד לאחר כותרת הפונקציה, המחלקה או המודול.
3.  כלי תיעוד, כגון Sphinx, יכולים להשתמש במחרוזות תיעוד ליצירת תיעוד.
4.  ניתן לגשת למחרוזות תיעוד באמצעות המאפיין `__doc__` של האובייקט.

**דוגמה:**

```python
def my_function(x: int) -> int:
  """פונקציה זו מכפילה מספר ב-2.
  
  Args:
    x: המספר להכפלה.
  Returns:
    המספר מוכפל ב-2.
  """
  return x * 2

print(my_function.__doc__)

# פלט:
#  פונקציה זו מכפילה מספר ב-2.
#
#  Args:
#    x: המספר להכפלה.
#  Returns:
#    המספר מוכפל ב-2.
```
**לסיכום:**
*   מחרוזות תיעוד מאפשרות הוספת תיעוד ישירות לקוד, מה שמסייע בהבנת מטרתו.
*   מחרוזות תיעוד נגישות באמצעות מאפיין מיוחד `__doc__`.

לפיכך, **אפשרות ג** היא התשובה הנכונה.
