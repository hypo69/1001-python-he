### `question_84.md`

**שאלה 84.** מהי תוצאת הביטוי `all([0, 1, 2, 3])` בפייתון?

- A. `True`
- B. `False`
- C. `None`
- D. מתרחשת שגיאה

**תשובה נכונה: B**

**הסבר:**

הפונקציה `all()` בפייתון בודקת האם כל האלמנטים באובייקט ניתן לאיטרציה הם ערכי אמת (במובן בוליאני). היא מחזירה `True` אם כל האלמנטים באובייקט הניתן לאיטרציה הם ערכי אמת, ו-`False` אחרת.

*   **אפשרות A** אינה נכונה: התוצאה תהיה `False`, מכיוון שיש אלמנט שקר.
*   **אפשרות B** נכונה: הפונקציה `all()` תחזיר `False` בשל נוכחות של אלמנט שקר.
*   **אפשרות C** אינה נכונה: הפונקציה `all()` מחזירה או `True` או `False`.
*   **אפשרות D** אינה נכונה: התחביר תקין, שגיאה לא תתרחש.

**כיצד `all()` פועלת:**

1.  מקבלת כקלט אובייקט הניתן לאיטרציה (רשימה, טאפל, מחרוזת, קבוצה).
2.  בודקת כל אלמנט עבור ערך האמת (truthy). מחרוזת ריקה, המספר 0, `None`, ומבנים ריקים נחשבים לערכי *שקר* (falsy), וכל השאר לערכי *אמת* (truthy).
3.  מחזירה `True` אם *כל* האלמנטים הם ערכי אמת, אחרת מחזירה `False`.
4.   אם האובייקט הניתן לאיטרציה ריק, מוחזר `True`.

**דוגמה:**

```python
# דוגמה עם all() ורשימה עם איבר שקר אחד
my_list: list[int] = [0, 1, 2, 3]
result_all_false: bool = all(my_list)
print(f"Результат all({my_list}): {result_all_false}") # פלט: Результат all([0, 1, 2, 3]): False

# דוגמה עם all() ורשימה ללא איברי שקר
my_list: list[int] = [1, 2, 3]
result_all_true: bool = all(my_list)
print(f"Результат all({my_list}): {result_all_true}") # פלט: Результат all([1, 2, 3]): True

# דוגמה עם all() ורשימה ריקה
empty_list: list[int] = []
result_all_empty: bool = all(empty_list)
print(f"Результат all({empty_list}): {result_all_empty}") # פלט: Результат all([]): True
```
**כתוצאה מכך:**

*   `all([0, 1, 2, 3])` מחזירה `False`, מכיוון שהרשימה מכילה את 0, שהוא ערך שקר.
*   `all([1, 2, 3])` מחזירה `True`, מכיוון שכל האלמנטים הם ערכי אמת.
*  `all([])` מחזירה `True`, מכיוון שרשימה ריקה עומדת בתנאים.

לפיכך, **אפשרות B** היא הנכונה.
