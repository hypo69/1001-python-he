### `question_893.md`

**שאלה 893.** מהי המטרה העיקרית של ההוראה `from __future__ import annotations` בפייתון, ואילו יתרונות היא מספקת באנוטציות (annotations) טיפוסים?

א. ההוראה `from __future__ import annotations` מבטלת אנוטציות טיפוסים בפייתון, ומאפשרת שימוש בתחביר ישן ללא ציון טיפוסים.
ב. ההוראה `from __future__ import annotations` כופה על פייתון לבצע בדיקת טיפוסים בזמן ריצת התוכנית, מה שמאפשר לזהות שגיאות בשלבים מוקדמים.
ג. ההוראה `from __future__ import annotations` מחליפה את כל אנוטציות הטיפוסים בייצוגים מחרוזתיים, מה שדוחה את חישוב הטיפוסים עד לרגע השימוש בהם ומאפשר להימנע מבעיות עם תלויות מעגליות והפניות לטיפוסים שטרם הוגדרו בזמן הקומפילציה.
ד. ההוראה `from __future__ import annotations` מוסיפה תמיכה באנוטציות טיפוסים לגרסאות פייתון ישנות, שבהן פונקציונליות זו לא הייתה קיימת במקור.

**תשובה נכונה: ג**

**הסבר:**

ההוראה `from __future__ import annotations` משנה את אופן העיבוד של אנוטציות טיפוסים בפייתון.

*   **התנהגות רגילה (ללא `__future__`):** אנוטציות טיפוסים מחושבות בזמן הגדרת הפונקציה או המחלקה. זה עלול להוביל לבעיות:
    *   **תלויות מעגליות:** אם מחלקה A משתמשת במחלקה B באנוטציה, ומחלקה B משתמשת במחלקה A, עלולה להיווצר שגיאה, מכיוון שאחת המחלקות טרם הוגדרה.
    *   **הפניות לטיפוסים שטרם הוגדרו:** אם אנוטציה מכילה הפניה לטיפוס שטרם הוגדר בזמן הקומפילציה, הדבר יוביל לשגיאה.

*   **עם `from __future__ import annotations`:**
    *   אנוטציות טיפוסים מומרות לייצוגים מחרוזתיים.
    *   החישוב בפועל של הטיפוסים נדחה עד לרגע שבו הם נדרשים באמת (לדוגמה, בזמן ריצה, באמצעות `typing.get_type_hints()`).
    *   נפתרות בעיות עם תלויות מעגליות והפניות לטיפוסים שטרם הוגדרו בזמן הקומפילציה.

*   **אפשרות א' אינה נכונה:** אנוטציות, להפך, מופעלות/נכללות.
*   **אפשרות ב' אינה נכונה:** בדיקת הטיפוסים נשארת סטטית (MyPy).
*   **אפשרות ג' נכונה:** מתארת את מהות פעולת ההוראה.
*   **אפשרות ד' אינה נכונה:** אנוטציות טיפוסים פועלות גם ללא ייבוא זה.

**דוגמה:**

```python
from __future__ import annotations
import typing

class A:
    def __init__(self, b: 'B'): # אנוטציה לטיפוס B
        self.b = b

class B:
    def __init__(self, a: A): # אנוטציה לטיפוס A
        self.a = a

print(typing.get_type_hints(A))
print(typing.get_type_hints(B))
```

**תוצאה:**

`from __future__ import annotations` דוחה את חישוב אנוטציות הטיפוסים, על ידי המרתן לייצוגים מחרוזתיים. הדבר מאפשר להימנע מבעיות עם תלויות מעגליות והפניות לטיפוסים שטרם הוגדרו בזמן הקומפילציה.

לסיכום, אפשרות ג' היא הנכונה.
