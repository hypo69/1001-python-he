**שאלה 843.** כיצד ניתן להסיר ב-Python *את כל הרווחים הרגילים* (רווח ASCII) ממחרוזת? הצג לפחות שתי שיטות לפתרון בעיה זו.

-   A.  כדי להסיר רווחים ממחרוזת, ניתן להשתמש רק במתודה `strip()` או בשילוב של המתודות `lstrip()` ו-`rstrip()`.
-   B.  כדי להסיר רווחים ממחרוזת, *חובה* להשתמש בלולאת `for` למעבר על כל התווים והסרה מותנית של רווחים.
-   C.  כדי להסיר רווחים, ניתן להשתמש במתודה `replace(' ', '')` או בשילוב של המתודות `split()` ו-`join('')`.
-   D.  כדי להסיר רווחים, *יש להשתמש באופרטור delete* ( `del` ) כדי לנקות את הרווחים מהמחרוזת.

**תשובה נכונה: C**

**הסבר:**

ב-Python, להסרת כל הרווחים *הרגילים* (רווח ASCII) ממחרוזת קיימות מספר שיטות. הנפוצות והנוחות ביותר הן שימוש במתודה `replace()` ושילוב של המתודות `split()` ו-`join()`.

*  **מתודת `replace()`:**
    *  **החלפת תת-מחרוזות:** מאפשרת להחליף את כל המופעים של תת-מחרוזת אחת בתת-מחרוזת אחרת.
    *   **הסרת רווחים:** כדי להסיר רווחים, מחליפים את כל הרווחים במחרוזת ריקה.
    *  **מחזירה מחרוזת חדשה:** המתודה מחזירה מחרוזת חדשה, המחרוזת המקורית אינה משתנה.
    * **תחביר:** `string.replace(old, new)`.

*   **מתודות `split()` ו-`join()`:**
    *   **`split()`:** מפצלת מחרוזת לרשימת מילים, תוך שימוש בתו מפריד (ברירת המחדל היא רווח).
    *   **`join()`:** מאחדת רשימת מחרוזות למחרוזת אחת, תוך שימוש ב-`""` כתו מפריד (בין המחרוזות).
     *  מאפשרת להסיר את כל הרווחים במחרוזת, על ידי פיצולה למילים, ואיחודן מחדש ללא רווחים.
    * **מחזירה מחרוזת חדשה**: המתודה מחזירה מחרוזת חדשה, המחרוזת המקורית אינה משתנה.
    *   **תחביר:**
         *   `string.split(separator)`
         *   `separator.join(iterable)`

**דוגמאות:**
```python
# דוגמה 1: שימוש ב-replace() להסרת רווחים
my_string = ' A string with white space '
new_string = my_string.replace(' ', '')
print(f"Исходная строка: '{my_string}'") # Original string
print(f"Строка без пробелов (replace): '{new_string}'") # String without spaces (replace)

# דוגמה 2: שימוש ב-split() ו-join() להסרת רווחים
my_string2 = ' A string with white space '
new_string2 = ''.join(my_string2.split())
print(f"Исходная строка: '{my_string2}'") # Original string
print(f"Строка без пробелов (split/join): '{new_string2}'") # String without spaces (split/join)
```

**ניתוח האפשרויות:**

*  **A. כדי להסיר רווחים ממחרוזת, ניתן להשתמש רק במתודה `strip()` או בשילוב של המתודות `lstrip()` ו-`rstrip()`. :** לא נכון. מתודות אלו מסירות רווחים רק מתחילת המחרוזת או מסופה.
*  **B. כדי להסיר רווחים ממחרוזת, *חובה* להשתמש בלולאת `for` למעבר על כל התווים והסרה מותנית של רווחים.:** לא נכון. למרות שניתן להשתמש בלולאת `for`, קיימות מתודות מובנות נוחות ויעילות יותר.
*   **C. כדי להסיר רווחים, ניתן להשתמש במתודה `replace(' ', '')` או בשילוב של המתודות `split()` ו-`join('')`.:** נכון.
*  **D. כדי להסיר רווחים, *יש להשתמש באופרטור delete* (`del` ), כדי לנקות את הרווחים מהמחרוזת.:** לא נכון. האופרטור `del` משמש למחיקת משתנים או איברים מאוספים הניתנים לשינוי (לדוגמה, איברים מרשימה). לא ניתן להשתמש בו כדי לשנות מחרוזות בצורה זו. מחרוזות הן בלתי ניתנות לשינוי (immutable) ב-Python.

**לסיכום:**

להסרת רווחים *רגילים* ב-Python, השיטות המתאימות ביותר הן המתודות `replace()` ו-`split()`/`join()`. אין להשתמש באופרטור `del` להסרת תווים ממחרוזת.
