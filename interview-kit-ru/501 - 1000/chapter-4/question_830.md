**שאלה 3.** ב-Python, לתחומי הכיסוי (scopes) של משתנים תפקיד חשוב בהבנת האופן שבו משתנים נגישים או ניתנים לשינוי בתוך בלוקים מקוננים של קוד. אם משתנה בעל אותו שם מוגדר הן בתוך פונקציה והן מחוצה לה, מה יקרה כאשר הפונקציה תנסה לשנות משתנה זה ללא שימוש במילת מפתח נוספת כלשהי?

- A.  הפונקציה תשנה ישירות את המשתנה הגלובלי ללא כל שגיאה.

- B.  Python תגרום לשגיאה מסוג `SyntaxError` עקב התנגשות בתחומי הכיסוי של המשתנים.

- C.  הפונקציה תיצור משתנה לוקלי חדש בעל שם זהה למשתנה הגלובלי, תוך השארת המשתנה הגלובלי ללא שינוי.

- D.  המשתנה הגלובלי יעבור הצללה, וערכו ישתנה רק לאחר שהפונקציה תסיים את פעולתה.

**תשובה נכונה: C**

**הסבר:**

*   **משתנים לוקליים:** ב-Python, משתנים המוגדרים בתוך פונקציה נחשבים כברירת מחדל ללוקליים לאותה פונקציה. משמעות הדבר היא שהם קיימים ונגישים רק בתוך אותה פונקציה.

*   **משתנים גלובליים:** משתנים גלובליים הם משתנים המוגדרים מחוץ לפונקציות (ברמת המודול), והם נגישים בכל קוד המודול.

*   **שינוי משתנים גלובליים:** אם בתוך פונקציה נעשה שימוש בשם של משתנה הקיים גם בתחום הכיסוי הגלובלי, ובמקביל לא נעשה שימוש במילת המפתח `global`, אזי פייתון ייצור אוטומטית משתנה לוקלי בעל אותו שם בתוך פונקציה זו. משמעות הדבר היא שהשינויים שיבוצעו במשתנה זה בתוך הפונקציה לא ישפיעו על המשתנה הגלובלי.

**דוגמה:**

```python
global_variable: int = 10  # משתנה גלובלי

def modify_variable() -> None:
    global_variable: int = 5 # נוצר משתנה לוקלי עם אותו שם
    print(f"ערך המשתנה בתוך הפונקציה: {global_variable}")  # פלט: ערך המשתנה בתוך הפונקציה: 5

modify_variable()
print(f"ערך המשתנה הגלובלי לאחר קריאה לפונקציה: {global_variable}") # פלט: ערך המשתנה הגלובלי לאחר קריאה לפונקציה: 10
```

**תוצאה:**

*   בתוך הפונקציה `modify_variable` נוצר משתנה לוקלי `global_variable` בעל ערך `5`. הוא *מצל* (shadows) את המשתנה הגלובלי בעל אותו שם, אך *אינו משנה* אותו.
*   כאשר הפונקציה מסיימת את פעולתה, המשתנה הלוקלי מפסיק להתקיים.
*   לאחר קריאה לפונקציה, המשתנה הגלובלי `global_variable` שומר על ערכו המקורי `10`.

לפיכך, תשובה **C** היא הנכונה, שכן היא מתארת במדויק את מנגנון יצירת המשתנה הלוקלי בתוך פונקציה כאשר קיים משתנה גלובלי בעל אותו שם.
