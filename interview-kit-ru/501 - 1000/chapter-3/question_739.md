**שאלה 431.** נתון גרף המיוצג כרשימת טיסות `flights`, כאשר `flights[i] = [from_i, to_i, price_i]` מציין קיום טיסה מעיר `from_i` לעיר `to_i` בעלות `price_i`. כמו כן, נתונים עיר מוצא `src`, עיר יעד `dst` ומספר עצירות מרבי `k`. פתח אלגוריתם שיחזיר את העלות המינימלית של טיסה מ-`src` ל-`dst` עם לא יותר מ-`k` עצירות. אם מסלול כזה אינו קיים, יש להחזיר `-1`.

**ייצוג הגרף ב-ASCII:**

```
      0
     /  \
  100/   \100
   1     2
 600 \ /100 \200
     3
```

**דוגמאות:**
```
Ввод: n = 4, flights = [[0,1,100],[1,2,100],[2,0,100],[1,3,600],[2,3,200]], src = 0, dst = 3, k = 1
Вывод: 700
Объяснение: המסלול הזול ביותר הוא 0 -> 1 -> 3 (0 -> 1 בעלות 100 ו-1 -> 3 בעלות 600, עצירה אחת, סך הכל 700)

Ввод: n = 3, flights = [[0,1,100],[1,2,100],[0,2,500]], src = 0, dst = 2, k = 1
Вывод: 200
Объяснение: המסלול הזול ביותר הוא 0 -> 1 -> 2 (0 -> 1 בעלות 100 ו-1 -> 2 בעלות 100, עצירה אחת, סך הכל 200)
```

-   א. כדי לפתור את הבעיה יש להשתמש באלגוריתם חמדן.
-   ב. כדי לפתור את הבעיה יש להשתמש בחיפוש לרוחב (BFS), אך מבלי להתחשב באילוץ k.
-   ג. כדי לפתור את הבעיה יש להשתמש באלגוריתם דייקסטרה, מבלי להתחשב באילוץ על מספר העצירות.
-   ד. כדי לפתור את הבעיה ניתן להשתמש באלגוריתם חיפוש לרוחב (BFS), ולעקוב לא רק אחר המרחק (העלות), אלא גם אחר מספר העצירות.

**תשובה נכונה: ד**

**הסבר:**

לפתרון בעיית מציאת המסלול הקצר ביותר עם אילוץ על מספר העצירות (k), נעשה שימוש באלגוריתם חיפוש לרוחב (BFS) עם שימוש בתור. גישה זו מאפשרת לא רק למצוא את המסלול הקצר ביותר, אלא גם להתחשב באילוצים על העצירות, ומספקת פתרון אופטימלי.

*   **אלגוריתם (BFS עם אילוץ עצירות):**
    1.  **אתחול:**
        *   נוצר מילון `distances`, שיכיל את עלות הטיסה לכל עיר (הערך ההתחלתי של כל המפתחות, למעט נקודת ההתחלה, הוא אינסוף). `distances[src]` = 0.
        *   נוצר תור `queue` ומוסף אליו העיר ההתחלתית ומספר העצירות `(src, 0)`.
        *   נשמר `k` במשתנה `stops`.
    2.  **BFS:** כל עוד התור אינו ריק:
        *   שולפים מהתור את העיר הנוכחית `current_city` ואת מספר העצירות `current_stops`.
        *   **בדיקה:** אם מספר העצירות עולה על `k`, אז המסלול הנוכחי אינו מבטיח.
        *   **מעבר על הטיסות:** עוברים על כל הטיסות `from_city, to_city, price` מתוך `flights`.
        *   **בדיקת התאמה:** אם `from_city` שווה ל-`current_city`, אז מעדכנים את העלות לעיר `to_city`, אם `distances[to_city] > distances[current_city] + price`, ומוסיפים את `to_city` לתור עם מספר עצירות `stops+1`.
    3.  **תוצאה:** לאחר עיבוד כל האפשרויות מחזירים את `distances[dst]`, או `-1` אם עיר היעד לא הושגה.

*   **יתרונות האלגוריתם:**
    *   **מסלול קצר ביותר:** BFS מבטיח מציאת המסלול הקצר ביותר.
    *   **התחשבות בעצירות:** האלגוריתם מתחשב באילוץ על מספר העצירות המרבי.
    *   **סיבוכיות אופטימלית:** בעל סיבוכיות זמן של `O(n*m)`, כאשר n - מספר הצמתים, ו-`m` - מספר הטיסות.

**דוגמאות (פסאודו-קוד):**
```
function find_cheapest_price(n, flights, src, dst, k):
    distances = a dictionary with keys of city, default to infinity
    distances[src] = 0
    queue = a queue with initial (src, 0)  # (city, stops)
    while queue is not empty:
      current_city, stops = queue.pop()
        if stops > k:
             continue
        for from_i, to_i, price in flights:
            if from_i == current_city:
              if distances[to_i] > distances[current_city] + price:
                distances[to_i] = distances[current_city] + price
                queue.add((to_i, stops +1))

    if distances[dst] is not infinity:
      return distances[dst]
    else:
      return -1
```
**דוגמאות ליישום בפייתון:**

```python
from collections import deque

def find_cheapest_price(n, flights, src, dst, k):
    distances = {city: float('inf') for city in range(n)}
    distances[src] = 0
    queue = deque()
    queue.append((src, 0))  # (city, stops)

    while queue:
        current_city, stops = queue.popleft()
        if stops > k:
            continue
        for from_i, to_i, price in flights:
            if from_i == current_city:
                if distances[to_i] > distances[current_city] + price:
                   distances[to_i] = distances[current_city] + price
                   queue.append((to_i, stops +1))

    return distances[dst] if distances[dst] != float('inf') else -1


n1 = 4
flights1 = [[0,1,100],[1,2,100],[2,0,100],[1,3,600],[2,3,200]]
src1 = 0
dst1 = 3
k1 = 1
print(f"Ввод: n = {n1}, flights = {flights1}, src = {src1}, dst = {dst1}, k = {k1}")
print(f"Вывод: {find_cheapest_price(n1, flights1, src1, dst1, k1)}") # Выведет: 700


n2 = 3
flights2 = [[0,1,100],[1,2,100],[0,2,500]]
src2 = 0
dst2 = 2
k2 = 1
print(f"Ввод: n = {n2}, flights = {flights2}, src = {src2}, dst = {dst2}, k = {k2}")
print(f"Вывод: {find_cheapest_price(n2, flights2, src2, dst2, k2)}") # Выведет: 200
```
