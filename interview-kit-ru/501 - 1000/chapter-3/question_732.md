**שאלה 732.** נתונה לך מחרוזת `s` המכילה מספר מילים המופרדות באמצעות רווחים (ייתכנו מספר רווחים בין מילים, וכן רווחים בתחילת או בסוף המחרוזת). פתח אלגוריתם בפייתון שמבצע היפוך סדר המילים במחרוזת. במחרוזת התוצאה, בין המילים אמור להיות רווח בודד בלבד, ולא אמורים להיות רווחים בתחילת ובסוף המחרוזת.

*   זמן ביצוע הסקריפט לא אמור לעלות על שנייה אחת.

**דוגמאות:**
```
Ввод: s = " Быстрый автомобиль "
Вывод: "автомобиль Быстрый"
```

-  A. לפתרון הבעיה יש להשתמש ברקורסיה ולהחליף מילים במחרוזת באופן סדרתי, ולאחר מכן להוציא אותן מהזיכרון.
-   B. לפתרון הבעיה יש להשתמש בלולאת for וליצור מחרוזת חדשה, על ידי הוספת מילים בסדר הפוך.
-   C. לפתרון הבעיה משתמשים בשני מצביעים (מסוף המחרוזת), וכן בלולאה מקוננת לבחירת המילה והוספתה למחרוזת התוצאה.
-   D. לפתרון הבעיה יש למיין קודם את המילים לפי אורך ולאחר מכן לבנות את המחרוזת בסדר הפוך.

**תשובה נכונה: C**

**הסבר:**

לפתרון בעיית היפוך סדר המילים במחרוזת, עם טיפול ברווחים מיותרים, האופטימלי ביותר הוא שימוש באלגוריתם המשתמש בשני מצביעים, הנעים מסוף המחרוזת לתחילתה, ובו בזמן חולצים מילים ומוסיפים אותן למחרוזת התוצאה. אלגוריתם כזה מאפשר לפתור את הבעיה בזמן ליניארי O(n), ובו בזמן להימנע משימוש בזיכרון מיותר.

*   **אלגוריתם (שני מצביעים):**
    1.  **בדיקה:** אם המחרוזת המקורית ריקה או מורכבת מרווחים בלבד, מחזירים מחרוזת ריקה.
    2.  **אתחול:** מאתחלים מצביע `start` לסוף המחרוזת המקורית `len(str)-1`, וכן מחרוזת ריקה `ans`.
    3.  **מעבר מהסוף:** לולאת `while` מתבצעת כל עוד `start` לא הגיע לתחילת המחרוזת:
        *   **דילוג על רווחים:** אם התו הנוכחי (עליו מצביע `start`) הוא רווח, ממשיכים להזיז את `start` שמאלה עד שהוא מפסיק להיות רווח.
        *   **הוספת רווח:** אם מחרוזת התוצאה `ans` אינה ריקה, מוסיפים רווח בסוף `ans`.
        *   **חליצת מילה:** לאחר מכן יש לולאת `while` פנימית, בה `j` נע מ-`start` עד רווח, ויוצר את המילה.
        *   **הוספה למחרוזת התוצאה:** מוסיפים את המילה שנחתכה למחרוזת התוצאה `ans`.
        *   מעדכנים את `start` למיקום של `j`.
    4.   **תוצאה:** לאחר עיבוד כל התווים, מחזירים את מחרוזת התוצאה `ans` (שתכיל את המילים בסדר הפוך).

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות ליניארית:** האלגוריתם עובר על המחרוזת פעם אחת בלבד, מה שמבטיח סיבוכיות זמן של O(n), כאשר n - אורך המחרוזת.
    *   **יעילות:** האלגוריתם אינו דורש מבנים נוספים, זיכרון או מיון.
    *   **טיפול ברווחים:** לוקח בחשבון רווחים מרובים.

**דוגמאות (פסאודו-קוד):**

```
function reverse_words(str):
    if str is empty or all chars are spaces
        return ""

    ans = ""
    start = length(str) - 1

    while start >= 0:
      if str[start] is space:
           start = start - 1
       else:
         if length(ans)> 0 :
           ans = ans + " "
         j = start;
        while j >= 0 and str[j] is not space:
          j = j -1;
         ans = ans + substring(str, j+1, start) # add word to ans
        start = j
    return ans
```

def reverseString(str: str) -> str:
    if(str == "" or str == " "):
        return ""
    ans = ''

    start = len(str) - 1

    while(start >= 0):
        if(str[start] == ' '):
            start-=1
        else:
            if(len(ans) > 0):
                ans += (' ')

            j = start

            while(j >= 0 and str[j] != ' '):
                j-=1

            ans +=  (str[j+1: j+1+start-j])
            start = j

    return ans
