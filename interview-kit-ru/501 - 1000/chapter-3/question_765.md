### `question_711.md`

**שאלה 711.** פתחו פונקציה בפייתון המקבלת אובייקט איטרבילי המכיל ערכים נומריים, ומחזירה את מכפלת כל האיברים.

**דוגמה:**
```
קלט: multiply((8, 2, 3, -1, 7))
פלט: -336
```

-   א. כדי לפתור את הבעיה, יש להשתמש בלולאת `for`, תוך הכפלת כל האיברים ברצף.
-   ב. כדי לפתור את הבעיה, יש להשתמש אך ורק באלגוריתם רקורסיבי.
-   ג. כדי לפתור את הבעיה, יש להשתמש במתודה `reduce()` מהמודול `functools`, ולהעביר אליה פונקציית למדא להכפלת כל האיברים.
-   ד. כדי לפתור את הבעיה, יש להשתמש במתודה `prod()`, אליה מועבר האובייקט האיטרבילי.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית הכפלת כל האיברים באובייקט איטרבילי, הדרך האופטימלית ביותר היא שימוש בפונקציה `reduce()` מהמודול `functools` בשילוב עם פונקציית למדא. גישה זו מאפשרת מימוש יעיל של הכפלה ומבטיחה קוד תמציתי.

*   **מושגי יסוד:**
    *   **`functools.reduce()`:** הפונקציה `reduce` מיישמת פונקציה על איברי אובייקט איטרבילי, ומביאה בסופו של דבר לערך יחיד.
    *   **פונקציית למדא:** אנו משתמשים בפונקציית למדא לביצוע פעולת ההכפלה.
    *   **צבירת תוצאה:** בכל איטרציה מתרחשת הכפלה של התוצאה הנוכחית באיבר הבא מהאובייקט האיטרבילי, והתוצאה עצמה הופכת לערך החדש של האקומולטור, מה שבסופו של דבר ייתן את מכפלת כל האיברים.

*   **יתרונות האלגוריתם:**
    *   **שימוש ב-`reduce()`:** `reduce()` מאפשרת "לצמצם" אובייקט איטרבילי לערך יחיד.
    *   **פונקציות למדא:** פונקציית למדא מאפשרת לתאר בקצרה את לוגיקת ההכפלה.
    *   **יעילות:** האלגוריתם מבצע את הכפלת האיברים בזמן לינארי, כלומר ב-`O(n)`.

**דוגמאות (פסאודו-קוד):**
```
function multiply(iterable):
  return reduce (lambda x, y : x*y, iterable)
```
**דוגמאות למימוש בפייתון:**
```python
from functools import reduce
import operator

def multiply(iterable):
    return reduce(lambda x, y: x * y, iterable)

def multiply2(iterable): # even shorter form
     return reduce(operator.mul, iterable)

data1 = (8, 2, 3, -1, 7)
print(f"קלט: multiply({data1})")
print(f"פלט: {multiply(data1)}")

data2 = [1, 2, 3, 4, 5]
print(f"קלט: multiply2({data2})")
print(f"פלט: {multiply2(data2)}")

data3 = [1,2,3]
print(f"קלט: multiply2({data3})")
print(f"פלט: {multiply2(data3)}")
```
**ניתוח אפשרויות:**
*   **א. כדי לפתור את הבעיה, יש להשתמש בלולאת `for`, תוך הכפלת כל האיברים ברצף.:** שגוי.
*   **ב. כדי לפתור את הבעיה, יש להשתמש אך ורק באלגוריתם רקורסיבי.:** שגוי. שימוש ב-`reduce()` הוא גישה תמציתית יותר.
*   **ג. כדי לפתור את הבעיה, יש להשתמש במתודה `reduce()` מהמודול `functools`, ולהעביר אליה פונקציית למדא להכפלת כל האיברים.:** נכון.
*   **ד. כדי לפתור את הבעיה, יש להשתמש במתודה `prod()`, אליה מועבר האובייקט האיטרבילי.:** שגוי, אין מתודה מובנית כזו בפייתון.

**לסיכום:**
*   הפונקציה `reduce()` מהמודול `functools` ופונקציית למדא מאפשרות קבלת מכפלת כל האיברים באובייקט איטרבילי.
*   לאלגוריתם מורכבות לינארית של `O(n)`.

לפיכך, התשובה הנכונה היא **ג. כדי לפתור את הבעיה, יש להשתמש במתודה `reduce()` מהמודול `functools`, ולהעביר אליה פונקציית למדא להכפלת כל האיברים.**
