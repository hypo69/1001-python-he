**שאלה 705.** נתון מספר שלם `n`. פתח אלגוריתם שמפרק מספר זה לסכום של לפחות שני מספרים שלמים חיוביים באופן כזה שמכפלת המספרים הללו תהיה מקסימלית. האלגוריתם יחזיר את המכפלה המקסימלית.

**דוגמאות:**

```
קלט: n = 2
פלט: 1
הסבר: 2 = 1 + 1, 1 * 1 = 1.

קלט: n = 10
פלט: 36
הסבר: 10 = 3 + 3 + 4, 3 * 3 * 4 = 36.
```

-   א. על מנת לפתור את הבעיה, יש להשתמש באלגוריתם כוח גס מלא (full enumeration) של כל הפירוקים האפשריים, ולהחזיר את המכפלה המקסימלית.
-   ב. על מנת לפתור את הבעיה, ניתן להשתמש באלגוריתם חמדן (greedy), שתמיד בוחר את המספר הקרוב ביותר, ומכפיל אותם עד שמגיעים למספר היעד.
-   ג. על מנת לפתור את הבעיה, ניתן להשתמש באלגוריתם תכנון דינמי, ולבנות טבלת תוצאות על סמך פתרונות קטנים יותר.
-   ד. על מנת לפתור את הבעיה, ניתן להשתמש בתכונות מתמטיות הקשורות לפירוק מספרים לשלשות וזוגות, ובאופן זה לחשב במהירות את המכפלה המקסימלית.

**תשובה נכונה: ד**

**הסבר:**

לפתרון בעיה זו, הגישה האופטימלית היא שימוש בתכונות מתמטיות הקשורות לפירוק אופטימלי של מספר לסכום של מחוברים. מתברר שכדי לקבל את המכפלה הגדולה ביותר, יש צורך לפרק את המספר למחוברים קרובים ככל האפשר ל-`e` (מספר אוילר ≈ 2.71828). המספרים השלמים הקרובים ביותר אליו הם 2 ו-3, כאשר שלשה יעילה יותר.

*   **אלגוריתם (איטרטיבי עם תכונות מתמטיות):**
    1.  **מקרים מיוחדים:** אם המספר שווה 2, החזר 1. אם המספר שווה 3, החזר 2.
    2.  **פירוק לשלשות:** פרק את `n` למספר מרבי אפשרי של מחוברים השווים ל-`3`.
        *   לשם כך, בדוק את שארית החלוקה של `n` ב-3.
        *   אם השארית שווה 0 - אזי ניתן לפרק את המספר לשלשות `3 * 3 * 3 ...`.
        *   אם השארית שווה 1 - אזי את ה-`3` האחרון ואת ה-`3` שלפניו מחליפים ב-`4`, ואת השאר משאירים `3`, כפי שהיה קודם (מכיוון ש-`4= 2*2` במקום `1*3`) - `3 * 3 * 4`.
        *   אם השארית שווה 2 - אזי מוסיפים בסוף עוד `2` (ואת שאר השלשות) - `3 * 3 * 3 ... * 2`.
    3.  **החזרת תוצאה:** חשב את מכפלת המחוברים שהתקבלו.
*   **יתרונות האלגוריתם:**
    *   **יעילות:** משיג סיבוכיות אופטימלית של (O(log n מבחינת זמן ריצה.
    *   **גישה מבוססת מתמטית:** מבוססת על תכונות מתמטיות של פירוק אופטימלי.
    *   **פשטות:** האלגוריתם פשוט וברור.

**דוגמאות מימוש ב-Python:**
```python
import math

def integer_break(n):
    if n == 2:
        return 1
    if n == 3:
        return 2
    num_threes = n // 3
    remainder = n % 3
    if remainder == 0:
        return 3 ** num_threes
    if remainder == 1:
        return 3 ** (num_threes - 1) * 4
    return 3 ** num_threes * 2

print(f"קלט: n = {2}")
print(f"פלט: {integer_break(2)}") # פלט: 1

print(f"קלט: n = {10}")
print(f"פלט: {integer_break(10)}")   # פלט: 36

print(f"קלט: n = {3}")
print(f"פלט: {integer_break(3)}") # פלט: 2
print(f"קלט: n = {12}")
print(f"פלט: {integer_break(12)}")  # פלט: 81
```
