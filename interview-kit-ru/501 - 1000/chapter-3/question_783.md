**שאלה 783.** פתח/י פונקציה ב-Python שמקבלת כקלט מספר שלם ומחזירה את המחרוזת `"Even"` אם המספר זוגי, ואת המחרוזת `"Odd"` אם המספר אי-זוגי.

**דוגמאות:**

```
קלט: 2
פלט: "Even"

קלט: 1
פלט: "Odd"

קלט: -46
פלט: "Even"

קלט: 71
פלט: "Odd"

קלט: 0
פלט: "Even"

קלט: -123
פלט: "Odd"
```

- א. כדי לפתור את הבעיה יש להשתמש באופרטורים בינאריים בלבד.
- ב. כדי לפתור את הבעיה יש להשתמש באופרטור התנאי `if/else` ובשארית החלוקה ב-`2`.
- ג. כדי לפתור את הבעיה יש להשתמש בשיטה `is_even()` בלבד.
- ד. כדי לפתור את הבעיה יש צורך לפרק את המספר לגורמים ראשוניים.

**תשובה נכונה: ב**

**הסבר:**

כדי לפתור את הבעיה של בדיקת מספר לזוגיות והחזרת המחרוזת המתאימה ("Even" או "Odd"), נעשה שימוש באופרטור `%` (מודולו) ובאופרטור התנאי `if...else`.

*   **אופרטור מודולו `%`:**
    *   **שארית החלוקה:** האופרטור `%` מחזיר את שארית החלוקה של מספר אחד באחר.
   *  **זוגיות:** אם שארית החלוקה של מספר ב-2 שווה ל-`0`, אזי המספר הוא זוגי.

*   **אופרטור תנאי `if...else`:**
    *   **בדיקת התנאי:** מאפשר לבצע אחד משני בלוקי קוד בהתאם לערך התנאי.

* **אלגוריתם:**
    1. **בדיקת זוגיות:** נשתמש באופרטור `%` כדי לקבוע אם המספר זוגי.
    2.   **החזרת התוצאה:**
        *  אם המספר זוגי, אזי נחזיר את המחרוזת `"Even"`.
       *  אם המספר אינו זוגי, אזי נחזיר את המחרוזת `"Odd"`.

**דוגמאות (קוד מדומה):**
```
function check_parity(number):
  if number  % 2 == 0:
        return "Even"
   else:
       return "Odd"
```
**דוגמאות למימוש ב-Python:**

```python
def reverse(num: int) -> str:
    if num % 2 == 0:
      return "Even"
    return "Odd"


print(f"קלט: 2")
print(f"פלט: {reverse(2)}")  # פלט: Even

print(f"קלט: 1")
print(f"פלט: {reverse(1)}") # פלט: Odd

print(f"קלט: -46")
print(f"פלט: {reverse(-46)}") # פלט: Even

print(f"קלט: 71")
print(f"פלט: {reverse(71)}") # פלט: Odd
print(f"קלט: 0")
print(f"פלט: {reverse(0)}") # פלט: Even
print(f"קלט: -123")
print(f"פלט: {reverse(-123)}") # פלט: Odd
```
**ניתוח האפשרויות:**
*  **א. כדי לפתור את הבעיה יש להשתמש באופרטורים בינאריים בלבד.:** לא נכון.
*  **ב. כדי לפתור את הבעיה יש להשתמש באופרטור התנאי `if/else` ובשארית החלוקה ב-`2`. :** נכון.
*  **ג. כדי לפתור את הבעיה יש להשתמש בשיטה `is_even()` בלבד.:** לא נכון, שיטה כזו אינה קיימת.
*   **ד. כדי לפתור את הבעיה יש צורך לפרק את המספר לגורמים ראשוניים.:** לא נכון.

**לסיכום:**
*  באופרטור % (מודולו) נעשה שימוש כדי לקבוע את זוגיות המספר.
*  אופרטור התנאי `if/else` מאפשר להגדיר ערכים שונים המוחזרים בהתאם לתוצאת הבדיקה.
