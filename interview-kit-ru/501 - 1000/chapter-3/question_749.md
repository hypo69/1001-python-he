**דוגמאות:**
```

Вывод: "One Hundred Twenty Three"

Вывод: "Twelve Thousand Three Hundred Forty Five"
```

**הסבר:**

*   **אלגוריתם (רקורסיבי):**
    1.  **מילונים:** יוצרים מילונים לאחסון הייצוג האותי של המספרים מ-0 עד 19, וכן לעשרות 20, 30, ... 90 ולדרגות (אלפים, מיליונים, מיליארדים).
    2.  **מקרי בסיס:**
        *   אם `num` שווה ל-0, מחזירים `"Zero"`.
        *   אם `num` קטן מ-20, מחזירים את המילה מהמילון הראשון.
    3.  **טיפול במאות:** אם `num` גדול מ-100, אז
        *   מבודדים את המאות, קוראים פונקציה זו רקורסיבית עבורן ומוסיפים "Hundred".
        *   מבודדים את שארית החלוקה ב-100 (לטיפול בעשרות ויחידות), וקוראים רקורסיבית פונקציה זו עבורה, אם היא אינה שווה לאפס.
    4.  **טיפול בעשרות ויחידות:** אם `num` גדול מ-20, אז
        *   מפצלים את המספר לעשרות ויחידות.
        *   באמצעות המילונים ממירים את העשרות ואת היחידות (אם אינן שוות ל-`0`) לאותיות.
        *   מחזירים את התוצאה.
    5.  **טיפול בדרגות:** אם `num` גדול או שווה ל-`1000`:
        *   מפרקים לדרגות (אלפים, מיליונים, מיליארדים).
        *   קוראים רקורסיבית לפונקציה עבור המספרים בדרגה הנוכחית ומוסיפים את שם הדרגה לתוצאה.

*   **יתרונות האלגוריתם:**
    *   **רקורסיה:** מאפשרת טיפול קל במספרים באורכים שונים.
    *   **קוד נקי:** הופכת את הקוד למובן ותמציתי יותר.
    *   **אופטימליות:** לאלגוריתם סיבוכיות O(log n).

**דוגמאות מימוש בפייתון:**
```python
def numberToWords(num):
    small_numbers = {
        0: "Zero", 1: "One", 2: "Two", 3: "Three", 4: "Four", 5: "Five", 6: "Six", 7: "Seven", 8: "Eight",
        9: "Nine", 10: "Ten", 11: "Eleven", 12: "Twelve", 13: "Thirteen", 14: "Fourteen", 15: "Fifteen",
        16: "Sixteen", 17: "Seventeen", 18: "Eighteen", 19: "Nineteen"
    }
    tens_ = {2: "Twenty", 3: "Thirty", 4: "Forty", 5: "Fifty", 6: "Sixty", 7: "Seventy", 8: "Eighty", 9: "Ninety"}
    thousands_ = {1000: "Thousand", 1000000: "Million", 1000000000: "Billion"}

    def helper(number):
        if number == 0:
            return ""
        if number < 20:
            return small_numbers[number]
        if number >= 100:
             hundreds_part = number // 100
             rem = number % 100
             res = small_numbers[hundreds_part] + " Hundred" # הוספנו רווח כאן
             if rem != 0:
                res+= " " + helper(rem) # הוספנו רווח לפני הקריאה הרקורסיבית
             return res
        if number >= 20:
            tens_part = number // 10
            units = number % 10
            res = tens_[tens_part]
            if units != 0:
                 res += " " + helper(units)
            return res
        return ""
        
    if num == 0: # מקרה בסיס עבור המספר 0 כולו
        return "Zero"

    result = ""
    # עוברים על הדרגות הגדולות בסדר יורד (מיליארד, מיליון, אלף)
    for thousands, word in sorted(thousands_.items(), reverse=True): # סידור בסדר יורד כדי לטפל קודם בדרגות הגדולות
        if num >= thousands:
            # קוראים לפונקציה helper עבור החלק של המספר בדרגה הנוכחית
            result_part = helper(num // thousands)
            # מוסיפים את החלק המתורגם + שם הדרגה + רווח
            result += result_part + " " + word + " "
            # מעדכנים את num לשארית אחרי הטיפול בדרגה הנוכחית
            num %= thousands
            
    # אחרי הטיפול בדרגות הגדולות, נשאר החלק שקטן מ-1000. מטפלים בו עם helper.
    result += helper(num)

    # מסירים רווחים מיותרים בתחילת ובסוף התוצאה
    return result.strip()
    
num1 = 123
print(f"Ввод: num = {num1}") # קלט: num = 123
print(f"Вывод: '{numberToWords(num1)}'") # פלט: 'One Hundred Twenty Three'

num2 = 12345
print(f"Ввод: num = {num2}") # קלט: num = 12345
print(f"Вывод: '{numberToWords(num2)}'")   # פלט: 'Twelve Thousand Three Hundred Forty Five'

num3 = 10

print(f"Вывод: '{numberToWords(num4)}'") # פלט: 'One Million Two Hundred Three Thousand One'
```
