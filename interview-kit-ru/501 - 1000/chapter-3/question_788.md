### `question_788.md`

**שאלה 788.** האם אפשר ב-Python, בעת ירושת מחלקות, לשלב מתודות של מחלקת האב ולא רק להחליף אותן במחלקה היורשת? אנא ספק דוגמה קוד המדגימה שימוש ב-`super()` לקריאה למתודה של מחלקת האב מתוך מתודה שנדרסה במחלקה היורשת.

-   א.  בעת ירושת מתודות מחלקה ב-Python, אי אפשר לשלב מתודות של מחלקת האב והמחלקה היורשת, ואפשרי רק לשכתב (לדרוס) מתודות.
-   ב.  בעת ירושת מתודות מחלקה ב-Python, אפשר לקרוא למתודה של מחלקת האב באמצעות `self.parent_method()`, מה שיאפשר לשלב אותן.
-   ג. בעת ירושת מתודות מחלקה ב-Python, מתודות שנדרסו מחליפות את מתודות מחלקת האב, אך אפשר לקרוא למתודה ממחלקת האב באמצעות הפונקציה `super()`, מה שמאפשר לשלב את הלוגיקה של שתי המתודות.
-   ד.  בעת ירושת מתודות מחלקה ב-Python, כדי לשלב מתודות יש להשתמש בירושה מרובה.

**תשובה נכונה: ג**

**הסבר:**

ב-Python, בעת ירושה, מתודה במחלקה היורשת בעלת אותו שם כמו מתודה במחלקת האב - דורסת (מחליפה) את מתודת מחלקת האב. עם זאת, Python מספקת מנגנון באמצעות הפונקציה `super()` כדי לקרוא למתודה של מחלקת האב מתוך מתודה שנדרסה במחלקה היורשת. זה מאפשר לשלב את הלוגיקה של שתי המתודות, ולא רק לדרוס את מתודת ההורה.

*   **דריסת מתודות (Method Overriding):**
    *   כאשר מחלקה יורשת מגדירה מתודה בעלת אותו שם כמו במחלקת האב, המתודה של המחלקה היורשת *דורסת* (מחליפה) את מתודת מחלקת האב.
    *   בעת קריאה למתודה באמצעות מופע של המחלקה היורשת, נקראת המתודה של המחלקה היורשת.

*   **הפונקציה `super()`:**
    *    **גישה למתודות מחלקת האב:** משמשת לקבלת גישה למתודות מחלקת האב.
     *    **שילוב לוגיקה:** מאפשרת לקרוא למתודת ההורה ולאחר מכן להוסיף לוגיקה משלך.

*   **איך `super()` עובדת:**
    *   בשימוש בתוך מתודה של מחלקה יורשת, `super()` מחזירה אובייקט proxy, המייצג את מחלקת האב, שממנו ניתן לקרוא למתודה.
    *  `super().method_name()` קורא למתודה של מחלקת האב בשם `method_name`.

**דוגמאות:**

```python
# Example 1: Method Overriding
class Parent:
    def test(self):
        print('inside parent')


class Child(Parent):
    def test(self):
        print('inside child')


b1 = Child()
b1.test() # Output: inside child
# Example 2: Using super() to combine methods

class Parent2:
    def test(self):
        print('inside parent')


class Child2(Parent2):
    def test(self):
        super().test() # Call the parent class method
        print('inside child')

b2 = Child2()
b2.test()
# Output:
# inside parent
# inside child
# Example of multiple inheritance with super
class A:
    def some_method(self):
        print("A method")

class B(A):
     def some_method(self):
        super().some_method()
        print("B method")

class C(A):
    def some_method(self):
       super().some_method()
       print("C method")

class D(B,C):
    def some_method(self):
       super().some_method()
       print("D method")
d=D()
d.some_method()
# A method
# C method
# B method
# D method
```
