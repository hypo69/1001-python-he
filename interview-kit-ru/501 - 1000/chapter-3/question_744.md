**שאלה 701.** נתונה מחרוזת לא ריקה `s`. פיתחו אלגוריתם ב-Python אשר מוצא ומחזיר את כל התמורות הייחודיות של התווים במחרוזת `s`.

**דוגמאות:**

```
Ввод: s = "a"
Вывод: ["a"]

Ввод: s = "ab"
Вывод: ["ab", "ba"]

Ввод: s = "aabb"
Вывод: ["aabb", "abab", "abba", "baab", "baba", "bbaa"]
```

- A. כדי לפתור את הבעיה יש להשתמש באלגוריתם חמדן (greedy), ובכל שלב להחליף מקומות רק לשני תווים.
- B. כדי לפתור את הבעיה יש להשתמש באלגוריתם רקורסיבי עם Backtracking (נסיגה), למציאת כל התמורות, תוך הימנעות משילובים כפולים, באמצעות קבוצה (set).
- C. כדי לפתור את הבעיה יש להשתמש באלגוריתם חיפוש לרוחב (BFS) לסריקת כל התמורות האפשריות.
- D. כדי לפתור את הבעיה יש להשתמש אך ורק בשיטת החלפת מיקום של אלמנטים באמצעות לולאות.

**תשובה נכונה: B**

**הסבר:**

לצורך פתרון הבעיה של מציאת כל התמורות הייחודיות של תווים במחרוזת, הפתרון האופטימלי הוא שימוש באלגוריתם רקורסיבי עם שימוש בטכניקת Backtracking (נסיגה). שימוש ב-Backtracking מאפשר לאלגוריתם לעקוב אחר כל האפשרויות האפשריות, ושימוש במערך קבוצות (set) מסייע במניעת כפילויות.

*   **האלגוריתם (רקורסיבי עם Backtracking):**
    1.  **אתחול:** נוצרת קבוצה (`set`) בשם `result` לאחסון התמורות הייחודיות.
    2. **פונקציה רקורסיבית:**
        *   הפונקציה מקבלת את התמורה הנוכחית `current_combination`, את התווים שנותרו לשימוש `remaining`, ואת הקבוצה התוצאתית `result`.
        *   **מקרה בסיס:** אם `remaining` ריק (כלומר, לא נותרו תווים), נוסיף את `current_combination` ל-`result`.
        *   **מעבר (איטרציה):** בכל שלב, נעבור (נעשה איטרציה) על התווים ב-`remaining`:
            *   נקבע את התו הנוכחי כ-`next_char`.
            *   נקרא לפונקציה הרקורסיבית, כאשר ל-`current_combination` נוסף `next_char`, ו-`remaining` הופך למחרוזת `remaining` ללא התו הנוכחי `next_char`.
            *   **Backtracking (נסיגה):** לאחר הקריאה הרקורסיבית, נסיר את `next_char` מ-`current_combination`.
    3.  **קריאה ראשונית:** נקרא לרקורסיה עם הערכים `""`, `s` ו-`result`.
    4. **תוצאה:** לאחר סיום המעבר הרקורסיבי, נחזיר את התוצאה מהקבוצה, תוך המרתה לרשימה.

*   **יתרונות האלגוריתם:**
    *   **יצירת כל התמורות:** האלגוריתם סורק את כל התמורות האפשריות של התווים במחרוזת.
    *   **Backtracking (נסיגה):** מאפשר לחתוך מסלולים לא מתאימים ולשלוט בלוגיקת הסריקה.
    *   **קבוצה (Set) לייחודיות:** הקבוצה `result` מבטיחה שכל התמורות יהיו ייחודיות, גם בנוכחות תווים כפולים.
    *   **יעילות:** לוקוס המורכבות (complexity) של האלגוריתם הוא `O(n*n!)`.

**דוגמאות (קוד מדומה - Pseudocode):**
```
function all_permutations_unique(s):
   result = new Set()
    function backtrack(current_permutation, remaining_chars):
       if remaining_chars is empty:             // מקרה בסיס: אם לא נותרו תווים
           add current_permutation to the result // הוסף את התמורה השלמה לתוצאה
           return
       for index from 0 to length(remaining_chars) -1: // עבור כל תו שנותר
           new_permutation = current_permutation + remaining_chars[index] // בנה תמורה חדשה עם התו הנוכחי
           new_remaining = remaining_chars without remaining_chars[index] // צור רשימת תווים שנותרו ללא התו הנוכחי
            backtrack(new_permutation, new_remaining) // קריאה רקורסיבית עם המצב החדש

    backtrack("",s) // קריאה ראשונית
    return convert set result to list // החזרת התוצאה כרשימה
```

**דוגמאות מימוש ב-Python:**
```python
def all_permutations_unique(s):
    result = set()
    def backtrack(current_combination, remaining_chars):
        if not remaining_chars:
            result.add("".join(current_combination))
            return

        for index, char in enumerate(remaining_chars):
            # הקריאה הרקורסיבית בונה שילוב חדש ומעבירה את שאר התווים
            # אין צורך ב-Backtracking מפורש מכיוון שמשתמשים ברשימות חדשות
            backtrack(current_combination + [char], remaining_chars[:index] + remaining_chars[index+1:])


    # התחלה עם מחרוזת ריקה ורשימת תווים מתוך s
    backtrack("", list(s))
    return list(result)

s1 = "a"
print(f"Ввод: s = '{s1}'")
print(f"Вывод: {all_permutations_unique(s1)}") # Выведет: Вывод: ['a']

s2 = "ab"
print(f"Ввод: s = '{s2}'")
print(f"Вывод: {all_permutations_unique(s2)}") # Выведет: Вывод: ['ab', 'ba']

s3 = "aabb"
print(f"Ввод: s = '{s3}'")
print(f"Вывод: {all_permutations_unique(s3)}") # Выведет: Вывод: ['aabb', 'abab', 'baba', 'abba', 'baab', 'bbaa']

```
