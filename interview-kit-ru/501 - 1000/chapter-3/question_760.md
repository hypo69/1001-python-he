### `question_760.md`

**שאלה 760.** נתון מערך של מספרים שלמים `nums`. פתח אלגוריתם ב-Python שמוצא ומחזיר את הסכום המקסימלי שניתן להשיג מתת-מערך רציף במערך `nums`.

*   **תת-מערך (subarray)**: חלק רציף ממערך.

**דוגמאות:**
```
קלט: nums = [-2,1,-3,4,-1,2,1,-5,4]
פלט: 6
הסבר: לתת-מערך [4,-1,2,1] יש את הסכום הגדול ביותר, 6.

קלט: nums = [5,4,-1,7,8]
פלט: 23
הסבר: לתת-מערך [5,4,-1,7,8] יש את הסכום הגדול ביותר, 23.
```

-   א. כדי לפתור את הבעיה, יש להשתמש רק באלגוריתם של חיפוש ממצה של כל תת-המערכים ולבחור את הסכום המקסימלי.
-   ב. כדי לפתור את הבעיה, יש להשתמש רק באלגוריתם תכנות דינמי, ולהרכיב טבלת ערכים עם הסכום המקסימלי.
-   ג. כדי לפתור את הבעיה, יש להשתמש רק באלגוריתם רקורסיבי.
-   ד. כדי לפתור את הבעיה, ניתן להשתמש באלגוריתם קאדן (Kadane), אשר מחשב באופן איטרטיבי את הסכום המקסימלי של התת-מערך הנוכחי ואת הסכום המקסימלי הכולל של תת-המערך.

**תשובה נכונה: ד**

**הסבר:**

כדי לפתור את בעיית מציאת הסכום המקסימלי של תת-מערך רציף במערך של מספרים שלמים, השיטה האופטימלית היא שימוש באלגוריתם קאדן (Kadane's Algorithm). אלגוריתם זה מאפשר לפתור את הבעיה ביעילות במעבר יחיד על המערך, ומספק סיבוכיות זמן ליניארית של O(n).

*   **אלגוריתם קאדן (Kadane's Algorithm):**
    1.  **אתחול:**
        *   מאתחלים את `max_so_far` (הסכום המקסימלי עד כה) עם הערך הנמוך ביותר האפשרי (למשל `float('-inf')`).
        *   מאתחלים את `max_ending_here` (הסכום המקסימלי של התת-מערך הנוכחי שמסתיים באיבר הנוכחי) עם הערך 0.
    2.  **איטרציה על המערך:**
        *   עוברים על המערך `nums`:
            *   מעדכנים את `max_ending_here` כך שיהיה המקסימום בין האיבר הנוכחי `num` לבין סכום של `max_ending_here` והאיבר הנוכחי `num`.
            *   אם `max_ending_here` נעשה קטן מ-0, מאפסים אותו ל-`0`. (הערה: גרסה אחרת ומקובלת יותר של קאדן שומרת את האיבר עצמו אם הוא גדול יותר מהסכום הנוכחי שמסתיים כאן פלוס האיבר, כפי שמוצג בפסאודו-קוד ובקוד הפייתון למטה. זה מטפל גם במקרים של מערכים עם מספרים שליליים בלבד).
            *   מעדכנים את `max_so_far` כך שיהיה המקסימום בין `max_so_far` לבין `max_ending_here`.
    3.  **תוצאה:** לאחר שעוברים על כל האיברים, מחזירים את הערך של `max_so_far`.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות ליניארית:** בעל סיבוכיות זמן `O(n)`, כאשר n הוא אורך המערך, מכיוון שהמעבר על הרשימה נעשה פעם אחת בלבד.
    *   **זיכרון קבוע:** משתמש רק בנפח קבוע של זיכרון נוסף, ולא במערך דינמי.
    *   **יעילות:** נמנע מחישובים חוזרים ועוקב ביעילות אחר הסכומים המקסימליים.

**דוגמאות (פסאודו-קוד):**
```
function find_max_subarray_sum(nums):
  max_so_far = - infinity
  max_ending_here = 0
  for num from nums:
      max_ending_here = max(num , num + max_ending_here)
     max_so_far =  max(max_so_far, max_ending_here)
   return max_so_far
```

**דוגמאות מימוש ב-Python:**

```python
def maxSubArraySum(nums):
    # אתחול הסכום המקסימלי הכולל (עד כה) למינוס אינסוף
    max_so_far = float('-inf')
    # אתחול הסכום המקסימלי של התת-מערך הנוכחי שמסתיים באיבר הנוכחי ל-0
    max_ending_here = 0

    # מעבר על המערך
    for num in nums:
        # חישוב הסכום המקסימלי שמסתיים באיבר הנוכחי
        # זהו המקסימום בין התחלת תת-מערך חדש באיבר הנוכחי
        # לבין הרחבת התת-מערך הנוכחי עם האיבר הנוכחי
        max_ending_here = max(num, num + max_ending_here)
        # עדכון הסכום המקסימלי הכולל אם הסכום שמסתיים כאן גדול יותר
        max_so_far = max(max_so_far, max_ending_here)
    # החזרת הסכום המקסימלי הכולל
    return max_so_far

nums1 = [-2,1,-3,4,-1,2,1,-5,4]
print(f"קלט: nums = {nums1}")
print(f"פלט: {maxSubArraySum(nums1)}") # יפלוט: פלט: 6

nums2 = [5,4,-1,7,8]
print(f"קלט: nums = {nums2}")
print(f"פלט: {maxSubArraySum(nums2)}") # יפלוט: פלט: 23

nums3 = [-1,-2,-3]
print(f"קלט: nums = {nums3}")
print(f"פלט: {maxSubArraySum(nums3)}") # יפלוט: פלט: -1

```

*   **ב. כדי לפתור את הבעיה, יש להשתמש רק באלגוריתם חמדן, תוך בחירה עקבית של הערכים הגדולים ביותר העונים על התנאי.:** שגוי, אלגוריתם חמדן אינו מתאים לבעיה זו.
*   **ג. כדי לפתור את הבעיה, יש להשתמש רק באלגוריתם רקורסיבי.:** שגוי, רקורסיה פשוטה אינה מתאימה ובעלת סיבוכיות של O(2^n).
*   **ד. כדי לפתור את הבעיה, ניתן להשתמש באלגוריתם קאדן (Kadane), אשר מחשב באופן איטרטיבי את הסכום המקסימלי של התת-מערך הנוכחי ואת הסכום המקסימלי הכולל של תת-המערך.:** נכון.
