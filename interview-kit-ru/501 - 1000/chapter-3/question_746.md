**שאלה 746.** פתח/י פונקציה ב-Python המקבלת מילון קלט בשם `data`, המכיל מידע על מוצר: `cost_price` (עלות יחידת מוצר), `sell_price` (מחיר מכירה ליחידת מוצר), ו-`inventory` (מלאי התחלתי). הפונקציה צריכה להחזיר את הרווח הכולל שהתקבל ממכירת כל המלאי, מעוגל למספר שלם.

*   יש להניח שכל המלאי נמכר.
*   `רווח = סך הכנסות ממכירות – סך עלות המכר`

**דוגמאות:**

```
Ввод: profit({"cost_price": 32.67, "sell_price": 45.00, "inventory": 1200})
Вывод: 14796

Ввод: profit({"cost_price": 225.89, "sell_price": 550.00, "inventory": 100})
Вывод: 32411

Ввод: profit({"cost_price": 2.77, "sell_price": 7.95, "inventory": 8500})
Вывод: 44030
```

- א. כדי לפתור את הבעיה, יש לעבור על כל המפתחות במילון ולהחיל את הנוסחה.
- ב. כדי לפתור את הבעיה, יש לחשב תחילה את סכום המחירים, ואז לחסר את עלות המכר, ואז לעגל למספר שלם.
- ג. כדי לפתור את הבעיה, יש לחלץ את כל הערכים מהמילון ולהשתמש בנוסחה `(sell_price - cost_price) * inventory`, ולאחר מכן לעגל את התוצאה למספר שלם.
- ד. כדי לפתור את הבעיה, יש להשתמש בלולאה כדי לעבור על כל הערכים במילון ולהחיל את הנוסחה עבור כל ערך.

*   **אלגוריתם (חישוב ישיר):**
    1.  **חילוץ נתונים:** מחלצים את הערכים `cost_price`, `sell_price`, ו-`inventory` ממילון הקלט `data`.

        * מחשבים את הרווח `profit = (sell_price - cost_price) * inventory`.

    *   **סיבוכיות קבועה:** בעל סיבוכיות זמן של `O(1)`.

**דוגמאות (פסאודו-קוד):**
```
function calculate_profit(data):
    cost_price = data["cost_price"]
    sell_price = data["sell_price"]
    inventory = data["inventory"]
    profit = (sell_price - cost_price) * inventory
    return round(profit)

```
**דוגמאות ליישום ב-Python:**
```python
def profit(data):
    cost_price = data["cost_price"]
    sell_price = data["sell_price"]
    inventory = data["inventory"]
    profit = (sell_price - cost_price) * inventory
    return round(profit)

data1 = {"cost_price": 32.67, "sell_price": 45.00, "inventory": 1200}
print(f"Ввод: data = {data1}")
print(f"Вывод: {profit(data1)}") # יציג: 14796

data2 = {"cost_price": 225.89, "sell_price": 550.00, "inventory": 100}
print(f"Ввод: data = {data2}")
print(f"Вывод: {profit(data2)}")  # יציג: 32411

data3 = {"cost_price": 2.77, "sell_price": 7.95, "inventory": 8500}
print(f"Ввод: data = {data3}")
print(f"Вывод: {profit(data3)}") # יציג: 44030
```
