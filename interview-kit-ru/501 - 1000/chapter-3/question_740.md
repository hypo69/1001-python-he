**שאלה 740.** נתונה מערך של מרווחי זמן לקיום פגישות `intervals`, כאשר `intervals[i] = [start_i, end_i]`. פתח/י אלגוריתם בפייתון לקביעת המספר המינימלי של חדרי ישיבות הנדרשים לקיום כל הפגישות ללא התנגשויות.

**דוגמאות:**

```
Ввод: intervals = [[0,30],[5,10],[15,20]]
Вывод: 2

Ввод: intervals = [[7,10],[2,4]]
Вывод: 1
```

- א. כדי לפתור את הבעיה, יש להשתמש באלגוריתם מיון, לאחר מכן לסנן את המרווחים החופפים, ולספור את כמותם.
- ב. כדי לפתור את הבעיה ניתן להשתמש רק בחיפוש בינארי ולבדוק חפיפות במערך ממוין.
- ג. כדי לפתור את הבעיה יש למיין את המרווחים לפי זמן התחלה ולעקוב אחר המספר המינימלי של מרווחים חופפים, תוך שמירת ערכי זמני הסיום של החדרים התפוסים.
- ד. כדי לפתור את הבעיה יש להשתמש באלגוריתם חיפוש לעומק (DFS).

**תשובה נכונה: ג**

**הסבר:**

לצורך פתרון בעיית מציאת המספר המינימלי של חדרי ישיבות הנדרשים לקיום כל הפגישות ללא התנגשויות, הגישה האופטימלית היא שימוש באלגוריתם המבוסס על מיון המרווחים לפי זמן התחלה ומעקב אחר חפיפות באמצעות רשימת זמני סיום.

*   **אלגוריתם (מיון וגישה חמדנית):**
    1.  **מיון:** ממיינים את רשימת המרווחים לפי זמן התחלה.
    2.  **מעקב אחר חפיפות:** יוצרים רשימה `end_times`, שתעקוב אחר זמני הסיום של החדרים התפוסים, ובתחילה היא ריקה.
    3.  **איטרציה:** עוברים באיטרציה על כל המרווחים:
        *   **הסרת חדרים שאינם נחוצים:** בכל צעד, מסירים מ-`end_times` את כל ערכי זמני הסיום שקטנים או שווים מזמן ההתחלה הנוכחי (מאחר שניתן לעשות שימוש חוזר בחדרים אלו).
        *   **הוספת זמן סיום:** מוסיפים את זמן הסיום של המרווח הנוכחי ל-`end_times`, תוך שמירה על הסדר שלה.
        *   **מספר החדרים:** בכל צעד, קובעים את אורך הרשימה `end_times`, המתאים למספר הנוכחי של החדרים התפוסים.
    4.  **החזרת תוצאה:** מחזירים את האורך המרבי של `end_times` שנצפה בכל שלב.

*   **יתרונות האלגוריתם:**
    *   **מיון:** מיון לפי תחילת המרווח מאפשר לעבור על המרווחים בסדר התחלתם.
    *   **יעילות:** לאלגוריתם סיבוכיות של `O(n log n)` עקב הצורך במיון המערך.
    *   **ספירה נכונה:** מפשט את המעקב אחר מספר החדרים התפוסים הנוכחי.
    *   **גישה חמדנית:** האלגוריתם החמדני מאפשר מציאה יעילה של המספר המינימלי של חדרים.
**דוגמאות (פסאודו-קוד):**
```
function min_meeting_rooms(intervals):
    sort intervals by start time
    end_times = [] # array for tracking ends of meetings
    for start, end in intervals
        remove all end_times less than or equal than start
        add current end time to end_times
    return max length of end_times
```

**דוגמאות ליישום בפייתון:**

```python
import heapq

def min_meeting_rooms(intervals):
    intervals.sort(key=lambda x: x[0])
    end_times = []  # using heapq to maintain order
    for start, end in intervals:
        while end_times and end_times[0] <= start:
            heapq.heappop(end_times)
        heapq.heappush(end_times, end)
    return len(end_times)

intervals1 = [[0,30],[5,10],[15,20]]
print(f"Ввод: intervals = {intervals1}")
print(f"Вывод: {min_meeting_rooms(intervals1)}") # Выведет: Вывод: 2

intervals2 = [[7,10],[2,4]]
print(f"Ввод: intervals = {intervals2}")
print(f"Вывод: {min_meeting_rooms(intervals2)}") # Выведет: Вывод: 1

intervals3 = [[1, 3],[4, 8],[2, 3]]
print(f"Ввод: intervals = {intervals3}")
print(f"Вывод: {min_meeting_rooms(intervals3)}")  # Выведет: Вывод: 2

intervals4 = [[0, 3], [2, 5], [3, 7], [6, 10]]
print(f"Ввод: intervals = {intervals4}")
print(f"Вывод: {min_meeting_rooms(intervals4)}")  # Выведет: 2

intervals5 = [[0, 30], [5, 10], [15, 20], [30,35], [31,32], [40,50]]
print(f"Ввод: intervals = {intervals5}")
print(f"Вывод: {min_meeting_rooms(intervals5)}") # Выведет: 3
```
