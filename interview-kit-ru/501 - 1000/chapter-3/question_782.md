**שאלה 782.** פיתוח אלגוריתם בפייתון ליישום שיטת "כוח גס" (brute-force) לפיצוח סיסמה. הפונקציה צריכה לקבל כקלט:
*   `password` - מחרוזת המכילה את הסיסמה,
*   `characters` - מחרוזת המכילה את כל התווים המותרים לסיסמה,
*   `max_length` - מספר שלם, האורך המקסימלי של הסיסמה.

הפונקציה צריכה לייצר את כל הצירופים האפשריים של סיסמאות, עד שתמצא את הנדרשת, ולאחר מכן להחזיר אותה, תוך הבטחה שהסיסמה תימצא.

*   מומלץ להגביל את אורך הסיסמה ל-4 תווים לבדיקה.

**דוגמאות:**

```
# בהנחה ש characters = "abc"
קלט: password = "c", max_length = 3
פלט: "c"

קלט: password = "abc", max_length = 3
פלט: "abc"

קלט: password = "ad", max_length = 3
פלט: "ad"
```

-   א. לפתרון הבעיה יש להשתמש באלגוריתם חמדן ולהוסיף בכל פעם את התווים הסבירים ביותר, עד להגעה לסיסמה, מבלי לעבור על כל האפשרויות.
-   ב. לפתרון הבעיה, יש להשתמש בגנרטורים, על מנת לקבל את כל וריאנטים הסיסמאות ולאחר מכן לעבור עליהם.
- ג. לפתרון הבעיה יש להשתמש באלגוריתם רקורסיבי (DFS) עם בחזרה לאחור, כאשר בכל שלב של האלגוריתם מוסיפים כל תו מתוך התווים המותרים, ובודקים אם הצירוף הנוכחי שווה לסיסמה, ואז מחזירים אותה.
-   ד. לפתרון הבעיה יש לעבור על כל המחרוזות האפשריות באמצעות לולאות מקוננות.

**תשובה נכונה: ג**

**הסבר:**

לצורך יישום שיטת "כוח גס" (brute-force), המבטיחה את מציאת הסיסמה במגבלות הנתונות של תווים ואורך, יש ליישם אלגוריתם רקורסיבי תוך שימוש במנגנון בחזרה לאחור. רקורסיה מאפשרת יצירה קלה של כל הצירופים האפשריים, תוך כדי בדיקת התאמתם לערך הקלט.

*   **אלגוריתם (רקורסיבי עם מנגנון בחזרה לאחור):**
    1.  **פונקציה רקורסיבית:** יוצרים פונקציה רקורסיבית, המקבלת:
        *   `current_combination` - המחרוזת שנוצרה כרגע.
       *   `characters` - מחרוזת עם התווים המותרים.
        *   `max_length` - האורך המקסימלי של המחרוזת.
    2. **מקרה בסיס:**
          *  אם `current_combination` שווה לסיסמה המבוקשת, מחזירים את `current_combination`.
          *   אם האורך של `current_combination` גדול מ-`max_length`, אז מחזירים `None`.
    3.  **איטרציה:** עוברים על התווים מתוך מחרוזת `characters`.
        *    מוסיפים למחרוזת הנוכחית `current_combination` תו אחד `char` , וקוראים לפונקציה הרקורסיבית, עם תת-המחרוזת החדשה.
          *  אם התוצאה של הקריאה הרקורסיבית אינה שווה ל-`None`, אז זו הסיסמה הנכונה, ומחזירים אותה.
        *   לאחר הקריאה הרקורסיבית, מסירים את `char` מ-`current_combination`.
    4.  **תוצאה:** אם הלולאה הסתיימה, והסיסמה לא נמצאה, פירוש הדבר שסיסמה כזו אינה קיימת, ומחזירים `None` (מה שסותר את תנאי השאלה).

*   **יתרונות האלגוריתם:**
    *   **מעבר על כל האפשרויות:** האלגוריתם עובר על כל המחרוזות האפשריות (עם הפרמטרים הנתונים).
    *   **אופי רקורסיבי:** רקורסיה מאפשרת יצירה קלה של צירופי מחרוזות באורכים שונים.
   *  **מנגנון בחזרה לאחור:** דוחה מסלולים שגויים, בחזרה למצבים קודמים במקרה של אי-התאמה.
   *   **ורסטיליות:** מאפשר שימוש בכל קבוצה של תווים.

**דוגמאות ליישום בפייתון:**
```python
def brute_force(password, characters, max_length):
    def backtrack(current_combination):
        if current_combination == password:
            return current_combination
        if len(current_combination) > max_length:
            return None
        for char in characters:
          new_comb = current_combination + char
          result = backtrack(new_comb)
          if result is not None:
              return result
        return None
    return backtrack("", characters)


password1 = "c"
characters1 = "abc"
max_length1 = 3
print(f"קלט: password = '{password1}', characters = '{characters1}', max_length = {max_length1}")
print(f"פלט: '{brute_force(password1, characters1, max_length1)}'") # יוצג: 'c'
password2 = "abc"
characters2 = "abc"
max_length2 = 3
print(f"קלט: password = '{password2}', characters = '{characters2}', max_length = {max_length2}")
print(f"פלט: '{brute_force(password2, characters2, max_length2)}'")  # יוצג:  'abc'
password3 = "ad"
characters3 = "abcde"
max_length3 = 3
print(f"קלט: password = '{password3}', characters = '{characters3}', max_length = {max_length3}")
print(f"פלט: '{brute_force(password3, characters3, max_length3)}'") # יוצג:  'ad'
```
