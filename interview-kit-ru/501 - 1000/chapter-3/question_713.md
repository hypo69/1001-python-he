**שאלה 713.** נתונות שתי מחרוזות `s` ו-`t`. פתח אלגוריתם הקובע האם מחרוזות אלה זהות לאחר עיבוד התווים `#`, המציינים פעולת Backspace (מחיקת התו הקודם).

**דוגמאות:**
```
Ввод: s = "ab#c", t = "ad#c"
Вывод: True
Объяснение: Обе строки после обработки backspace образуют строку "ac".

Ввод: s = "ab##", t = "c#d#"
Вывод: True
Объяснение: Обе строки после обработки backspace образуют пустую строку ""

Ввод: s = "a#c", t = "b"
Вывод: False
Объяснение: строка "a#c" преобразуется в "c", а строка "b" остаётся "b"

```

- א. כדי לפתור את הבעיה יש ליצור פונקציה רקורסיבית שתחליף את "#" במחרוזת, ולאחר מכן להשוות את המחרוזות שהתקבלו.
- ב. כדי לפתור את הבעיה יש פשוט למחוק את כל תווי ה-# ולהשוות את המחרוזות שהתקבלו.
- ג. כדי לפתור את הבעיה יש להשתמש במחסנית כדי לדמות את פעולת ה-Backspace עבור כל מחרוזת, ולאחר מכן להשוות את המחרוזות הסופיות.
- ד. כדי לפתור את הבעיה יש להשתמש במתודות str.replace() ו-del למחיקת תווים.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיה זו, הגישה האופטימלית היא שימוש במחסנית כדי לדמות את פעולת ה-Backspace. מחסניות מאפשרות מעקב אחר התווים הנמחקים ובנייה יעילה של המחרוזות התוצאתיות.

*   **אלגוריתם (מחסנית):**
    1.  **יצירת פונקציית עיבוד:**
        *   יוצרים פונקציה המקבלת מחרוזת ומעבדת אותה.
        *   יוצרים מחסנית ריקה לאחסון התווים.
        *   מבצעים איטרציה על כל תווי המחרוזת:
            *   אם התו הנוכחי אינו `#`, מוסיפים אותו למחסנית.
            *   אם התו הנוכחי הוא `#` והמחסנית אינה ריקה, מסירים את האלמנט האחרון מהמחסנית (pop).
        *   לאחר מעבר על כל המחרוזת, ממירים את המחסנית למחרוזת.
    2.  **השוואת מחרוזות:** קוראים לפונקציית העיבוד עבור שתי המחרוזות, ומשווים את התוצאות שהתקבלו.

*   **יתרונות האלגוריתם:**
    *   **יעילות:** האלגוריתם עובר על המחרוזות פעם אחת בלבד.
    *   **דימוי Backspace:** המחסנית מדמה את פעולת ה-Backspace.
    *   **פשטות:** קל למימוש.

**דוגמאות (פסאודו-קוד):**
```
function process_string(string):
    stack = []
    for char in string:
        if char != '#':
            stack.append(char)
        elif stack is not empty:
            stack.pop()
    return stack to string

function compare_strings_with_backspace(s1,s2):
    str1 =  process_string(s1)
    str2 = process_string(s2)
   return str1 == str2
```
**דוגמאות מימוש ב-Python:**

```python
def process_string(string):
    stack = []
    for char in string:
        if char != '#':
            stack.append(char)
        elif stack:
            stack.pop()
    return "".join(stack)


def compare_strings_with_backspace(s1,s2):
    str1 =  process_string(s1)
    str2 = process_string(s2)
    return str1 == str2


s1 = "ab#c"
t1 = "ad#c"
print(f"Ввод: s = '{s1}', t = '{t1}'")
print(f"Вывод: {compare_strings_with_backspace(s1, t1)}")  # Output: True

s2 = "ab##"
t2 = "c#d#"
print(f"Ввод: s = '{s2}', t = '{t2}'")
print(f"Вывод: {compare_strings_with_backspace(s2, t2)}") # Output: True

s3 = "a#c"
t3 = "b"
print(f"Ввод: s = '{s3}', t = '{t3}'")
print(f"Вывод: {compare_strings_with_backspace(s3, t3)}") # Output: False

s4 = "abcd###"
t4 = "ab"
print(f"Ввод: s = '{s4}', t = '{t4}'")
print(f"Вывод: {compare_strings_with_backspace(s4,t4)}") # Output: False

s5 = "#a#c"
t5 = "c"
print(f"Ввод: s = '{s5}', t = '{t5}'")
print(f"Вывод: {compare_strings_with_backspace(s5,t5)}") # Output: True

```
