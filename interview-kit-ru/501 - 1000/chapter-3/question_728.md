**שאלה 728.** נתון מערך שלמים `nums`. פיתחו אלגוריתם למציאה והחזרה של כל תתי-הסדרות הלא-יורדות (לאו דווקא רציפות) הייחודיות במערך `nums`, שאורכן לכל הפחות שניים.

*   **תת-סדרה לא-יורדת:** רצף של איברים, כאשר כל איבר הבא גדול או שווה לקודמו (חזרות מותרות).
*   צירופים ייחודיים.

**דוגמאות:**

```
קלט: nums = [4,6,7,7]
פלט: [[4,6],[4,6,7],[4,6,7,7],[4,7],[4,7,7],[6,7],[6,7,7],[7,7]]

קלט: nums = [4,4,3,2,1]
פלט: [[4,4]]
```

-   א. כדי לפתור את הבעיה יש למיין את המערך ולעבור על כל תתי-המערכים.
- ב.  כדי לפתור את הבעיה יש להשתמש באלגוריתם חמדני, על ידי בחירת האיבר המינימלי והוספת איברים גדולים ממנו או שווים לו.
-  ג. כדי לפתור את הבעיה יש להשתמש באלגוריתם תכנות דינמי, על ידי שמירת התוצאות בכל שלב.
-  ד. כדי לפתור את הבעיה ניתן להשתמש באלגוריתם רקורסיבי עם Backtracking (נסיגה), כדי לעבור על כל הצירופים האפשריים.

**התשובה הנכונה: ד**

**הסבר:**

לפתרון בעיית מציאת כל תתי-הסדרות הלא-יורדות הייחודיות במערך, הגישה המתאימה ביותר היא שימוש באלגוריתם רקורסיבי עם Backtracking (נסיגה), המאפשר לעבור ביעילות על כל האפשרויות, וכן להשתמש ב-`set` (מבנה נתונים מסוג קבוצה) לסילוק כפילויות בשלב מציאת תת-הסדרה.

*   **האלגוריתם (רקורסיבי עם Backtracking):**
    1.  **יצירת קבוצה:**  יוצרים קבוצה `result`  לאחסון כל הרצפים הייחודיים.
    2. **פונקציה רקורסיבית:**  יוצרים פונקציה רקורסיבית אשר:
         *   מקבלת את האינדקס של האיבר הנוכחי `index`, את תת-הסדרה הנוכחית `current_subsequence` ואת קבוצת התוצאות `result`.
         *  **מקרה בסיס:** אם אורך `current_subsequence`  גדול או שווה ל-2, מוסיפים את תת-הסדרה לקבוצה  `result`,  לאחר שהפכנו אותה ל-`tuple` (זוג סדור / קורטז', מכיוון ש-`set` אינו יכול להכיל רשימות).
      *   **איטרציה:** מבצעים איטרציה על איברי `nums`  החל מהאינדקס הנוכחי `index`:
         *   **בדיקת תנאי:** אם האיבר הנוכחי `nums[i]` גדול או שווה לאיבר האחרון ב-`current_subsequence` (או אם  `current_subsequence`  ריק), אז
              *   מוסיפים את האיבר `nums[i]` ל-`current_subsequence`.
               * קוראים לפונקציה באופן רקורסיבי  עבור  האינדקס הבא.
             * **Backtracking (נסיגה):**  לאחר הקריאה הרקורסיבית מסירים את האיבר האחרון  מ-`current_subsequence` (פעולת נסיגה), כדי  לבדוק צירופים נוספים.
    3.   **קריאה רקורסיבית:** קוראים  לפונקציה  עם הפרמטרים ההתחלתיים, וכתשואה מכך תוחזר קבוצה (אותה ממירים לאחר מכן לרשימה).

*   **יתרונות האלגוריתם:**
    *   **מעבר מלא על כל האפשרויות:** מאפשר לבדוק את כל האפשרויות האפשריות.
    *   **Backtracking (נסיגה):** מאפשר לפסול אפשרויות שאינן תקינות.
    *   **סילוק כפילויות:** הקבוצה מסלקת כפילויות.

**דוגמאות מימוש ב-Python:**
```python
def find_all_non_decreasing_subsequences(nums):
    result = set()

    def backtrack(index, current_combination):
      # מקרה בסיס: אם אורך התת-סדרה לפחות 2, הוסף לקבוצת התוצאות
      if len(current_combination) >= 2:
         result.add(tuple(current_combination)) # הוספה כ-tuple לקבוצה לסילוק כפילויות
      
      # איטרציה על האלמנטים החל מהאינדקס הנוכחי
      for i in range(index, len(nums)):
        # בדיקת תנאי לא-יורד או אם התת-סדרה ריקה (למקרה של האיבר הראשון)
        if not current_combination or nums[i] >= current_combination[-1]:
         current_combination.append(nums[i]) # הוספת האיבר לתת-הסדרה הנוכחית
         backtrack(i + 1, current_combination) # קריאה רקורסיבית עם האינדקס הבא
         current_combination.pop() # ביצוע backtracking (נסיגה) - הסרת האיבר האחרון

    backtrack(0, []) # קריאה התחלתית עם אינדקס 0 ותת-סדרה ריקה
    return [list(seq) for seq in result] # המרת הקבוצה של tuples לרשימה של רשימות והחזרה


nums1 = [4,6,7,7]
print(f"קלט: nums = {nums1}")
print(f"פלט: {find_all_non_decreasing_subsequences(nums1)}") # פלט: [[4, 6], [4, 6, 7], [4, 6, 7, 7], [4, 7], [4, 7, 7], [6, 7], [6, 7, 7], [7, 7]]

nums2 = [4,4,3,2,1]
print(f"קלט: nums = {nums2}")
print(f"פלט: {find_all_non_decreasing_subsequences(nums2)}") # פלט: [[4, 4]]

nums3 = [1,2,3]
print(f"קלט: nums = {nums3}")
print(f"פלט: {find_all_non_decreasing_subsequences(nums3)}") # פלט: [[1, 2], [1, 2, 3], [1, 3], [2, 3]]
```
