**שאלה 717.** נתון מערך של מספרים שלמים `nums`, כאשר כל איבר נמצא בטווח מ-1 עד `n` (כאשר `n` הוא אורך המערך). פתחו אלגוריתם שמוצא ומחזיר את כל המספרים השלמים מהטווח `[1, n]` אשר *נעדרים* מהמערך `nums`.

**דוגמאות:**

```
קלט: nums = [4, 3, 2, 7, 8, 2, 3, 1]
פלט: [5, 6]

קלט: nums = [1, 1]
פלט: [2]
```

- A. כדי לפתור את הבעיה, יש למיין את הרשימה ולאחר מכן ליצור רשימה של המספרים החסרים.
- B. כדי לפתור את הבעיה, יש להשתמש בטבלת גיבוב או בקבוצה, ולבדוק עבור כל מספר האם הוא קיים במערך.
- C. כדי לפתור את הבעיה, יש ליצור מערך חדש של מספרים [1,n] ולסנן החוצה את כל האיברים מ-nums.
- D. כדי לפתור את הבעיה, יש להשתמש במערך מונים, שבו האינדקס מייצג מספר בטווח [1,n], והערך - מספר המופעים של מספר זה, ובסוף להחזיר את כל האינדקסים שערכיהם שווים ל-0.

**תשובה נכונה: D**

**הסבר:**

לפתרון הבעיה של מציאת מספרים חסרים בטווח נתון, הגישה האופטימלית היא שימוש במערך מונים (או לחלופין, קבוצה/סט), המאפשרת לעקוב אחר נוכחות או היעדר מספרים במעבר יחיד, ודורשת סיבוכיות לינארית בזמן ובזיכרון.

*   **אלגוריתם (עם שימוש במערך מונים):**
    1.  **יצירת מערך מונים:**
        *   נוצר מערך `counters` עם אפסים, שגודלו שווה לאורך המערך `nums`.
        *   האינדקס במערך מתאים לערכים במערך `nums`.
    2.  **איטרציה על המערך `nums`**:
        *   עוברים באיטרציה על המערך `nums`.
        *   עבור כל איבר, מגדילים את הערך במערך `counters` באחד באינדקס `element-1` (היסט הנובע מכך שהמונה צריך להתאים לערכים מ-1 עד `n`, ולא מ-0 עד `n-1`).
    3.  **מציאת החסרים:** עוברים באיטרציה על מערך המונים.
        *   אם הערך במונה שווה לאפס, אז מוסיפים את האינדקס + 1 (מטופל חזרה בהיסט) לרשימת התוצאות.
    4.  **תוצאה:** לאחר איטרציה על מערך המונים, מחזירים את רשימת המספרים החסרים.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות לינארית:** לאלגוריתם יש סיבוכיות זמן של O(n), מכיוון שהוא עובר על המערך פעם אחת בלבד.
    *   **פשטות:** האלגוריתם פשוט ליישום.
    *   **שימוש בזיכרון מוגבל:** נפח הזיכרון הדרוש למערך `counters` תלוי בגודל המערך `nums`.

**דוגמאות (פסאודו קוד):**

```
function find_missing_numbers(nums):
    n = length(nums)
    counters = an array initialized with zeros of size n
    for element in nums:
       counters[element-1] += 1
    result = []
   for i from 0 to n:
       if counters[i] == 0:
            result.append(i + 1)
    return result
```
**דוגמאות ליישום ב-Python:**

```python
def find_missing_numbers(nums):
    n = len(nums)
    counters = [0] * n # מערך מונים
    for el in nums:
        counters[el - 1] += 1
    net = []
    for i in range(n):
        if counters[i] == 0:
            net.append(i + 1)
    return net

nums1 = [4, 3, 2, 7, 8, 2, 3, 1]
print(f"קלט: nums = {nums1}")
print(f"פלט: {find_missing_numbers(nums1)}") # ידפיס: פלט: [5, 6]

nums2 = [1,1]
print(f"קלט: nums = {nums2}")
print(f"פלט: {find_missing_numbers(nums2)}") # ידפיס: פלט: [2]
```
