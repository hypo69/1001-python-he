**שאלה 792.** בשורה עומדים זה אחר זה `n` אנשים, כאשר גובהו של האדם ה-`i` הוא `a[i]` יחידות מותנות. עליך למצוא מיקום `p` בשורה, כך שערך הפונקציה `f(p)` יהיה מקסימלי.

*   `f(p)` = `[מספר האנשים משמאלך בעלי אותו גובה כמו שלך]` * `[מספר האנשים מימינך בעלי גובה שונה משלך]`.
*   באפשרותך לעמוד בתחילת השורה, בסופה או בין כל שני אנשים קיימים.
*   ידועות לך `m` השערות לגבי גובהך האפשרי, ועבור כל השערה עליך למצוא את המיקום האופטימלי.

פַּתֵּח אלגוריתם בפייתון המקבל כקלט:
*   `n` - רשימת גבהי האנשים.
*   `m` - רשימה של וריאנטים אפשריים לגובהך.

האלגוריתם יחזיר רשימה של מיקומים אופטימליים עבור כל גובה מרשימת `m`, כאשר המיקום צריך להיות בין 1 ל-n כולל (0 - תחילת הרשימה, n - סוף הרשימה, אחרת אינדקס + 1).

**דוגמאות:**
```
Ввод: n = [1,2,3,2,4], m = [2, 4]
Вывод: [2, 4]
Объяснение:
Для 2 - лучшая позиция = 2 (1 * 1)
Для 4 - лучшая позиция = 4 (0 * 2)

Ввод:  n = [4,4,3,2,1,4], m = [4]
Вывод:  [2]
Объяснение:
Для 4, вставить надо на позицию 2 (1*2)
```

-   א. כדי לפתור את הבעיה יש לעבור על כל המיקומים האפשריים ולכל מיקום לחשב מחדש את ערכי `f(p)`, ולבחור את המקסימלי.
-   ב. כדי לפתור את הבעיה יש להשתמש במערך דו-ממדי, שבו יאוחסנו כל הערכים שחושבו ולבחור את המקסימלי.
-   ג. כדי לפתור את הבעיה ניתן להשתמש במעבר ליניארי על המערך, ועבור כל השערה לעקוב אחר כמויות שוות ושונות, ולהחזיר את המיקום שבו `f(p)` יהיה מקסימלי.
-   ד. כדי לפתור את הבעיה יש למיין את המערך, ולאחר מכן להפעיל חיפוש בינארי.

**תשובה נכונה: ג**

**הסבר:**

כדי לפתור את בעיית מציאת המיקום האופטימלי עם ערך מקסימלי של `f(p)` עבור כל אחת מהשערות הגובה, הגישה היעילה ביותר היא שימוש במעבר ליניארי על המערך `n`, תוך שמירה וחישוב של `equal` (מספר האיברים השווים משמאל) ו-`notequal` (מספר האיברים הלא שווים מימין), מה שמאפשר קבלת פתרון במעבר יחיד, הימנעות מחישובים מיותרים.

*   **אלגוריתם (ליניארי עם מעבר):**
    1.  **אתחול:**
        *   יוצרים רשימות `equal` ו-`notequal` עם ערך התחלתי של `0` ועם ערך התחלתי של מספר האיברים הלא שווים (מתחילת הרשימה ועד סופה), בהתאמה, עבור כל אחד מ-`m` הגבהים האפשריים.
        *   כמו כן, יוצרים רשימות `valsave` (ערך מקסימלי) ו-`possave` (מיקום מיטבי) עם ערכים התחלתיים של `0`.
    2.  **איטרציה על המערך `n`:**
        *   עוברים באיטרציה על המערך `n` (המיקומים האפשריים בשורה).
        *   **עוברים באיטרציה על `m`:** עבור כל אחד מ-m וריאנטים של גובה בודקים:
            *   **השוואה עם האיבר הנוכחי:** אם האיבר הנוכחי `n[i]` שווה לערך הנוכחי מ-`m[x]`, מגדילים את המונה `equal[x]` ב-1.
            *   אם האיבר הנוכחי `n[i]` אינו שווה לערך הנוכחי מ-`m[x]`, מקטינים את המונה `notequal[x]` ב-1.
            *   **חישוב `f(p)`:** מחשבים `val = equal[x] * notequal[x]`.
            *   **עדכון:** אם `val` גדול מ-`valsave[x]`, שומרים את הערך החדש `valsave[x] = val` ואת המיקום `possave[x] = i+1`.
    3.  **תוצאה:** לאחר מעבר על כל איברי `n`, מחזירים את מערך `possave` (עם המיקום האופטימלי עבור כל ערך של `m`).

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות ליניארית:** האלגוריתם עובר על המערך רק פעם אחת, וסיבוכיות הזמן שלו היא O(n*m).
    *   **נוחות:** קל ליישם את האלגוריתם ולעבור על כל הווריאנטים.
    *   **מעקב:** מאפשר בכל שלב של האיטרציה על המערך `n` לעקוב אחר `equal`, `notequal` ו-`possave` ולשמור את ערכיהם הנכונים.

**דוגמאות (פסאודו-קוד):**
```
function find_positions(n, m):
  equal = new array with length m, default 0
  notequal = new array with length m. initialized with n - sum of same m
  valsave =  new array with length m, default 0
    possave =  new array with length m, default 0

    for i from 0 to length(n)-1
        for x from 0 to length(m) - 1
            if n[i] is equal to m[x]
                increase equal[x]
            if n[i] is not equal to m[x]
                reduce notequal[x]
           val  = equal[x] * notequal[x]
         if val > valsave[x]
             valsave[x] = val
             possave[x]= i+1

     return possave
```

**דוגמאות ליישום בפייתון:**

```python
def find_positions(n, m):
    equal = [0 for i in range(len(m))]
    notequal = [len(list(filter(lambda x: x != m[i], n))) for i in range(len(m))]
    valsave = [0 for i in range(len(m))]
    possave = [0 for i in range(len(m))]

    for i in range(len(n)):
        for x in range(len(m)):
            if n[i] == m[x]:
                equal[x] += 1
            if n[i] != m[x]:
                notequal[x] -= 1

            val = equal[x] * notequal[x]
            if val > valsave[x]:
                valsave[x] = val
                possave[x] = i+1
    return possave

n1 = [1,2,3,2,4]
m1 = [2, 4]
print(f"קלט: n = {n1}, m = {m1}")
print(f"פלט: {find_positions(n1,m1)}") # יציג: [2, 4]

n2 = [4,4,3,2,1,4]
m2 = [4]
print(f"קלט: n = {n2}, m = {m2}")
print(f"פלט: {find_positions(n2,m2)}") # יציג [2]
```
