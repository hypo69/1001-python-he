**שאלה 701.** נתון מערך של מספרים שלמים `nums`. פתח אלגוריתם לחישוב האורך המרבי של תת-סדרה עולה לחלוטין (longest increasing subsequence, LIS).

**דוגמאות:**
```
קלט: nums = [10,9,2,5,3,7,101,18]
פלט: 4
הסבר: תת-הסדרה [2,3,7,101] היא בעלת האורך המרבי.

קלט: nums = [1, 3, 2, 4, 5]
פלט: 4
הסבר: תת-הסדרה [1, 2, 4, 5] היא בעלת האורך המרבי.
```

-   A.  לפתרון הבעיה יש למיין את המערך ולקבל את אורכו.
-   B.  לפתרון הבעיה ניתן להשתמש רק באלגוריתם רקורסיבי, ולהחזיר את הערך המקסימלי מבין כל המסלולים.
-   C.  לפתרון הבעיה יש להשתמש באלגוריתם חמדן, לבחור ברציפות את הערכים המרביים שמקיימים את התנאי.
-   D.  לפתרון הבעיה ניתן להשתמש בתכנות דינמי, כדי לבנות את הסדרה, תוך שמירת ערכים עבור כל תת-הסדרות האפשריות.

**תשובה נכונה: D**

**הסבר:**

למציאת האורך המרבי של תת-סדרה עולה לחלוטין (LIS) במערך של מספרים שלמים, משתמשים באלגוריתם תכנות דינמי. אלגוריתם זה מאפשר לפתור את הבעיה ביעילות, תוך הימנעות מחישובים חוזרים.

*   **אלגוריתם (תכנות דינמי):**
    1.  **יצירת מערך:** יוצרים מערך `dp`, כאשר `dp[i]` יציין את האורך של תת-הסדרה העולה הגדולה ביותר שמסתיימת באיבר `nums[i]`.
    2.  **אתחול:** מאפסים את כל ערכי המערך `dp` לערך `1`, מכיוון שכל איבר בפני עצמו מהווה תת-סדרה מינימלית באורך אחד.
    3.  **מילוי המערך:**
        *   מבצעים איטרציה על המערך `nums`, ובכל איטרציה מבצעים איטרציה פנימית על כל האיברים הקודמים.
        *   אם האיבר הנוכחי `nums[i]` גדול מהאיבר הקודם `nums[j]`, אז `dp[i]` יהיה הערך המקסימלי מבין `dp[i]` ו-`dp[j] + 1`.
        *   בדרך זו, `dp[i]` יתאים לאורך של תת-הסדרה העולה הגדולה ביותר שמסתיימת באיבר `nums[i]`.
    4.  **תוצאה:** לאחר מעבר על כל האיברים, לוקחים את הערך המקסימלי מתוך `dp`, אשר מתאים לאורך של LIS.

*   **יתרונות האלגוריתם:**
    *   **אופטימליות:** האלגוריתם מוצא את הפתרון האופטימלי ביותר.
    *   **תכנות דינמי:** שימוש בטבלת dp מאפשר להימנע מחישובים חוזרים.
    *   **סיבוכיות אופטימלית:** לאלגוריתם סיבוכיות זמן של `O(n^2)`.

**דוגמאות (קוד פסאודו):**
```
function longest_increasing_subsequence(nums):
  n = length(nums)
    dp = מערך מאותחל ל-1, באורך של nums.
    for i from 1 to n-1:
        for j from 0 to i-1:
          if nums[i] > nums[j]:
            dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```
**דוגמאות מימוש בפייתון:**

```python
def longest_increasing_subsequence(nums):
    n = len(nums)
    dp = [1] * n

    for i in range(1, n):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

nums1 = [10, 9, 2, 5, 3, 7, 101, 18]
print(f"קלט: nums = {nums1}")
print(f"פלט: {longest_increasing_subsequence(nums1)}") # יציג: פלט: 4

nums2 = [1, 3, 2, 4, 5]
print(f"קלט: nums = {nums2}")
print(f"פלט: {longest_increasing_subsequence(nums2)}") # יציג: פלט: 4

nums3 = [1, 1, 1, 1]
print(f"קלט: nums = {nums3}")
print(f"פלט: {longest_increasing_subsequence(nums3)}") # יציג 1
```
