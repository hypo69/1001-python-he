**שאלה 776.** נתון מערך שלם `height` באורך `n`. דמיינו שעל המישור שורטטו `n` קווים אנכיים, כאשר הקואורדינטות של נקודות הקצה של הקו ה-`i` הן `(i, 0)` ו-`(i, height[i])`. מצאו שני קווים אשר יחד עם ציר ה-X יוצרים מיכל שמכיל את כמות המים הגדולה ביותר. החזירו את כמות המים המקסימלית שמיכל כזה יכול להכיל. אסור להטות את המיכל.

**דוגמאות:**

```
קלט: height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
פלט: 49
הסבר: שטח המים המקסימלי הוא 49, בין הקווים בגבהים 8 ו-7, ברוחב 7. (חישוב: min(8, 7) * 7 = 49)

קלט: height = [1, 1]
פלט: 1
```
-   א.  כדי לפתור את הבעיה יש להשתמש באלגוריתם של בדיקת כל הקומבינציות האפשריות של שני קווים, לחשב את השטח ולהחזיר מתוכם את השטח המקסימלי.
-   ב. כדי לפתור את הבעיה יש להשתמש באלגוריתם של חיפוש בינארי ולבדוק רק ערכים אמצעיים.
-  ג.  כדי לפתור את הבעיה יש להשתמש באלגוריתם של תכנון דינמי, בניית טבלה עבור כל תת-מערכים.
-  ד. כדי לפתור את הבעיה יש להשתמש בשיטת שני מצביעים, הנעים לכיוון האמצע, ותוך כדי כך לשמור על השטח המקסימלי.

**תשובה נכונה: ד'**

**הסבר:**

כדי לפתור את הבעיה של מציאת השטח המקסימלי של מיכל, הנוצר על ידי שני קווים במערך, הגישה האופטימלית היא שימוש בשיטת שני מצביעים, המתקדמים זה לקראת זה ומחשבים את השטח הנוכחי בכל שלב, ובכך עוברים רק על האפשרויות הנחוצות.

*   **אלגוריתם (שני מצביעים):**
    1.  **אתחול:**
          *  הגדרת מצביע שמאלי `left` לתחילת המערך (אינדקס 0).
        *   הגדרת מצביע ימני `right` לסוף המערך (אינדקס `len(height)-1`).
        *  הגדרת `max_area` ל-`0`.
    2.   **איטרציה:**
           *  כל עוד המצביע השמאלי `left` קטן מהמצביע הימני `right`:
             *   **חישוב שטח:** חישוב השטח בין שני הקווים הנוכחיים כ-`min(height[left], height[right]) * (right - left)`.
             *  **עדכון מקסימום:** השוואת השטח שהתקבל עם הערך הנוכחי של `max_area`, ואם השטח הנוכחי גדול יותר - עדכון `max_area`.
              *   **הזזת מצביעים:** אם הקו השמאלי נמוך יותר מהקו הימני, יש להזיז את `left` ימינה (`left++`), אחרת יש להזיז את `right` שמאלה (`right--`).
    3.  **תוצאה:** לאחר השלמת הלולאה, יש להחזיר את `max_area`.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות ליניארית:** לאלגוריתם סיבוכיות זמן של O(n), מכיוון שהמצביעים עוברים על המערך פעם אחת בלבד.
    *  **שימוש בשני מצביעים:** מאפשר סינון אפשרויות לא נחוצות ובדיקת פתרונות מבטיחים בלבד.
     *  **פשטות:** אלגוריתם קל ליישום.

**דוגמאות (פסאודו-קוד):**
```
function maxWater(height):
   left = 0
    right = length(height)-1
    max_area = 0
     while left < right:
        current_area = min(height[left], height[right]) * (right - left)
        max_area = max(max_area, current_area);
      if height[left] < height[right]
           left = left +1;
      else:
           right = right -1
    return max_area;
```
**דוגמאות מימוש בפייתון:**
```python
def maxWater(height):
    first = 0
    end = len(height) - 1
    ans = 0

    while first < end:
        ans = max(ans, min(height[first], height[end]) * (end - first))
        if height[first] < height[end]:
            first += 1
        else:
            end -= 1
    return ans


height1=[1,8,6,2,5,4,8,3,7]
print(f"קלט: height = {height1}")
print(f"פלט: {maxWater(height1)}")  # יוצג: 49

height2 = [1, 1]
print(f"קלט: height = {height2}")
print(f"פלט: {maxWater(height2)}") # יוצג: 1
```
