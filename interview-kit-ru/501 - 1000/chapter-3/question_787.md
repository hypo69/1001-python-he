**שאלה 787.** פתח פונקציה ב-Python שמקבלת כקלט שנה `year` (מספר שלם) ומספר חודש `month` (מספר שלם, כאשר 1 הוא ינואר), ומחזירה `True` אם בחודש זה קיים יום שישי ה-13, ו-`False` אחרת.

```
קלט: has_friday_13(2023, 3)
פלט: True

קלט: has_friday_13(2017, 10)
פלט: True

קלט: has_friday_13(1985, 1)
פלט: False
```
-   א. כדי לפתור את הבעיה, יש להשתמש באלגוריתם רקורסיבי שיעבור על הימים עד שימצא יום שישי ה-13.
-   ב. כדי לפתור את הבעיה, יש להשתמש בספריית `calendar` ולבדוק את יום השבוע עבור היום ה-13 בחודש הנתון.
-   ג. כדי לפתור את הבעיה, יש להשתמש בלולאת `for` שתעבור על כל ימי החודש ותבדוק אותם.
-   ד. כדי לפתור את הבעיה, יש לעבור על כל השנים ולבחור רק את אלו שבהן ה-13 חל ביום שישי.

**תשובה נכונה: ב**

**הסבר:**

לצורך פתרון הבעיה של זיהוי קיום יום שישי ה-13 בחודש נתון, השיטה המתאימה והיעילה ביותר היא שימוש במודול המובנה `calendar` ב-Python. מודול זה מספק כלים לעבודה עם לוחות שנה ומאפשר לקבוע בנוחות את יום השבוע עבור כל תאריך.

*   **מתודה `calendar.weekday(year, month, day)`:**
    *   **קביעת יום השבוע:** מחזירה את מספר יום השבוע עבור התאריך הנתון.
        *   `0` - שני, 1 - שלישי, 2 - רביעי, 3 - חמישי, 4 - שישי, 5 - שבת, 6 - ראשון.
    *   **שלושה ארגומנטים:** מקבלת `year`, `month`, ו-`day`.
*   **עקרונות יסוד של האלגוריתם:**
    *   יוצרים תאריך עבור היום ה-13 ובודקים את יום השבוע, אם זה יום שישי, מחזירים `True`.
    *   אחרת, מחזירים `False`.
*   **יתרונות האלגוריתם:**
    *   **דיוק:** מבטיח קביעה נכונה של יום השבוע.
    *   **פשטות:** משתמש במודול מובנה `calendar` ואינו דורש חישובים מורכבים.
    *   **יעילות:** לאלגוריתם סיבוכיות קבועה O(1), מאחר ואין איטרציות.

**דוגמאות ליישום ב-Python:**
```python
import calendar
def has_friday_13(year, month):
    day = 13
    weekday = calendar.weekday(year, month, day) # 0 - שני and 6 - ראשון
    return weekday == 4
year1 = 2023
month1 = 3
print(f"קלט: month = {month1}, year = {year1}")
print(f"פלט: {has_friday_13(year1, month1)}") # ידפיס: פלט: True

year2 = 2017
month2 = 10
print(f"קלט: month = {month2}, year = {year2}")
print(f"פלט: {has_friday_13(year2, month2)}")  # ידפיס: פלט: True
year3 = 1985
month3 = 1
print(f"קלט: month = {month3}, year = {year3}")
print(f"פלט: {has_friday_13(year3, month3)}")  # ידפיס: פלט: False
```

*   **ג. כדי לפתור את הבעיה, יש להשתמש במתודה `datetime.weekday()` ולחשב את יום שלישי הראשון.:** נכון, אך לא `datetime.weekday()` אלא `calendar.weekday()`.
*   **ד. כדי לפתור את הבעיה, יש לעבור על כל ימי החודש ולמצוא את ההופעה הראשונה של יום שלישי.:** לא נכון. אלגוריתם זה ידרוש יותר זמן ואינו אופטימלי.
