**שאלה 723.** נתונה מערך מחרוזות `words`, המובטח שיכיל לפחות מילה אחת. פתח אלגוריתם ב-Python שימצא ויחזיר את הקידומת המשותפת הארוכה ביותר (התחלה משותפת) לכל המחרוזות במערך. אם לא קיימת קידומת משותפת, החזר מחרוזת ריקה "".

**דוגמאות:**

```
קלט: words = ["дог", "домен", "домра", "доширак"]
פלט: "до"

קלט: words = ["документ", "кот", "кум", "ум"]
פלט: ""
```

-   A. לפתרון הבעיה יש לבנות עץ המכיל את כל תתי-המחרוזות ולבדוק אותן.
-  B.  לפתרון הבעיה יש למיין תחילה את המילים ולהשוות את הראשונה והאחרונה.
- C.  לפתרון הבעיה יש להשוות את הקידומות של המילה הראשונה עם כל שאר המילים במערך, תו-אחר-תו, עד ההבדל הראשון, או עד שמסתיימת אחת המילים.
-  D. לפתרון הבעיה יש להשתמש ברקורסיה בלבד.

**תשובה נכונה: C**

**הסבר:**

לפתרון בעיית מציאת הקידומת המשותפת הארוכה ביותר במערך מחרוזות, הגישה האופטימלית היא שימוש באלגוריתם השוואה תו-אחר-תו. גישה זו מאפשרת להימנע מבדיקות מיותרות ומבטיחה הגעה לפתרון בזמן מינימלי.

*   **האלגוריתם (השוואה תו-אחר-תו):**
    1.  **מערך ריק:** אם המערך `words` ריק, החזר מחרוזת ריקה.
    2.  **בחירת ייחוס:** ניקח את המילה הראשונה `words[0]` כמחרוזת ייחוס `first_word` להשוואות הבאות. אם ישנה מחרוזת ריקה (במערך), אז נחזיר מיד מחרוזת ריקה.
    3.  **השוואה תו-אחר-תו:** עוברים על תווי `first_word` לפי אינדקס:
        *  עוברים על כל המילים במערך, ומשווים את התו הנוכחי `first_word[index]` עם התו באותו אינדקס `other_word[index]`.
            *   אם האינדקס גדול או שווה לאורך `other_word` (לא לכל המילים אורך זהה) או שהתווים אינם שווים, אז הקידומת הנוכחית הפסיקה להיות משותפת. נחזיר את הפרוסה (slice) של `first_word` עד האינדקס הנוכחי.
     4.   **תוצאה:** אם הבדיקה לכל תווי מילת הייחוס הושלמה בהצלחה, משמע שמילת הייחוס היא הקידומת המשותפת הארוכה ביותר, ואותה נחזיר.

*   **יתרונות האלגוריתם:**
    *  **יעילות:** לאלגוריתם סיבוכיות זמן לינארית של `O(m*n)`, כאשר `n` הוא מספר המילים, ו-`m` הוא אורך המילה הקצרה ביותר, שכן הוא סורק את המחרוזת עד הכישלון הראשון.
   * **פשטות:** אלגוריתם קל ליישום וברור.

**דוגמאות (פסאודו-קוד):**
```
function longest_common_prefix(words):
    if length(words) == 0:
      return ""
    first_word = words[0]
    for index from 0 to length(first_word) -1:
      for other_word in words:
        if index >= length(other_word) or first_word[index] != other_word[index]:
              return first_word[0:index]
    return first_word
```

**דוגמאות למימוש ב-Python:**
```python
def longest_common_prefix(words):
    if not words:
        return ""
    first_word = words[0]
    for index, char in enumerate(first_word):
        for other_word in words:
            if index >= len(other_word) or other_word[index] != char:
                return first_word[:index]
    return first_word


words1 = ["дог", "домен", "домра", "доширак"]
print(f"קלט: words = {words1}")
print(f"פלט: '{longest_common_prefix(words1)}'") # ידפיס: 'до'

words2 = ["документ", "кот", "кум", "ум"]
print(f"קלט: words = {words2}")
print(f"פלט: '{longest_common_prefix(words2)}'")  # ידפיס: ''

words3 = ["flower","flow","flight"]
print(f"קלט: words = {words3}")
print(f"פלט: '{longest_common_prefix(words3)}'")  # ידפיס: 'fl'
words4 = ["aaa","aa","aaa"]
print(f"קלט: words = {words4}")
print(f"פלט: '{longest_common_prefix(words4)}'")  # ידפיס: 'aa'
```

*   **B. לפתרון הבעיה ניתן להשתמש באלגוריתם רקורסיבי שיבדוק את קיום הקידומת המשותפת באופן רקורסיבי.:** שגוי. אלגוריתם רקורסיבי אינו נחוץ ופחות יעיל.
