**שאלה 796.** נתונה מילון `results`, המכילה נתונים אודות עלויות והכנסות של קמפיינים פרסומיים ממקורות שונים. יש להשלים את מבנה הנתונים המקורי עם מדד ROI (החזר השקעה) על פי הנוסחה: `(revenue / cost - 1) * 100` (יש לעגל ל-2 ספרות לאחר הנקודה העשרונית), ולהחזיר את המילון ששונה.

**דוגמה:**

```python
results = {
    'vk': {'revenue': 103, 'cost': 98},
    'yandex': {'revenue': 179, 'cost': 153},
    'ok': {'revenue': 103, 'cost': 110},
    'adwords': {'revenue': 35, 'cost': 34},
    'twitter': {'revenue': 11, 'cost': 24},
}
```
**פלט:**

```
{'adwords': {'revenue': 35, 'cost': 34, 'ROI': 2.94},
 'ok':      {'revenue': 103, 'cost': 110, 'ROI': -6.36},

 'vk':      {'revenue': 103, 'cost': 98, 'ROI': 5.1},
 'yandex':  {'revenue': 179, 'cost': 153, 'ROI': 16.99}}
```

-  A. לפתרון הבעיה יש ליצור מילון חדש, ולהעתיק אליו את כל הנתונים ולהוסיף את ערכי ה-ROI.
-  B. לפתרון הבעיה יש לחסר את עלות המכר ממחיר המכירה, ולאחר מכן להכפיל את התוצאה בכמות המלאי, ולאחר מכן לעגל.
-  C.  לפתרון הבעיה יש להשתמש במתודה `dict.update()` ובלולאות.
-  D. לפתרון הבעיה יש להשתמש בלולאת `for`, תוך כדי פירוק `key` ו-`value` מהמילון, ולחשב את ה-ROI, ולהוסיף אותו למילון הנוכחי.

**תשובה נכונה: D**

**הסבר:**

לפתרון הבעיה של הוספת מדד ה-ROI למילון קיים, האופן היעיל ביותר יהיה להשתמש בלולאת `for` כדי לעבור על המילון ולהוסיף את ערך ה-ROI ישירות למילון עצמו, וכן להשתמש במילונים מקוננים לאחסון ערכים נוספים.

*  **אלגוריתם:**
    1.   **מעבר על המילון:**  עוברים על איברי המילון `results`:
           *    מחשבים את ה-ROI על פי הנוסחה `(revenue / cost - 1) * 100`,  ושומרים במשתנה `roi`.
          *    מוסיפים למילון הנוכחי מפתח חדש `ROI` עם הערך של `roi`, מעוגל לשתי ספרות לאחר הנקודה העשרונית.
    2.  **החזרת המילון:** לאחר מעבר על כל איברי המילון, מחזירים אותו.

*   **יתרונות האלגוריתם:**
    *   **פשטות:**  משתמש בפעולות אריתמטיות פשוטות ובלולאת `for` בסיסית.
     *   **משנה את המילון במקום:**  עורך ישירות את המילון המקורי, ללא יצירת עותק.
   *    **סיבוכיות לינארית:** בעל סיבוכיות זמן של O(n), כאשר n הוא מספר האיברים במילון.

**דוגמאות למימוש ב-Python:**
```python
def calculate_roi(results):
    for key, value in results.items():
        revenue = value["revenue"]
        cost = value["cost"]
        roi = (revenue / cost - 1) * 100
        value["ROI"] = round(roi, 2)
    return results

results = {
    'vk': {'revenue': 103, 'cost': 98},
    'yandex': {'revenue': 179, 'cost': 153},
    'ok': {'revenue': 103, 'cost': 110},
    'adwords': {'revenue': 35, 'cost': 34},
    'twitter': {'revenue': 11, 'cost': 24},
}

print(f"Ввод: results = {results}")
print(f"Вывод: {calculate_roi(results)}")
# יודפס: Вывод: {'vk': {'revenue': 103, 'cost': 98, 'ROI': 5.1}, 'yandex': {'revenue': 179, 'cost': 153, 'ROI': 16.99}, 'ok': {'revenue': 103, 'cost': 110, 'ROI': -6.36}, 'adwords': {'revenue': 35, 'cost': 34, 'ROI': 2.94}, 'twitter': {'revenue': 11, 'cost': 24, 'ROI': -54.17}}
```
