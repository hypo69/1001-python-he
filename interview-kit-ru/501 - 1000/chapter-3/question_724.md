**שאלה 724.** פתח/פתחי פונקציה בפייתון שמקבלת מילון כקלט בשם `data`, המכיל מידע אודות מוצר: `cost_price` (עלות ליחידת מוצר), `sell_price` (מחיר מכירה ליחידת מוצר) ו-`inventory` (מלאי התחלתי). הפונקציה צריכה להחזיר את הרווח הכולל שהתקבל ממכירת המוצר, מעוגל למספר שלם.

*   ההנחה היא שכל המלאי נמכר.
*   רווח = סך ההכנסות ממכירות – סך עלות המכר

**דוגמאות:**
```
קלט: profit({"cost_price": 32.67, "sell_price": 45.00, "inventory": 1200})
פלט: 14796

קלט: profit({"cost_price": 225.89, "sell_price": 550.00, "inventory": 100})
פלט: 32411

קלט: profit({"cost_price": 2.77, "sell_price": 7.95, "inventory": 8500})
פלט: 44030
```

-   א. כדי לפתור את הבעיה, יש לעגל כל ערך לפני חישוב הרווח, ובסוף להשליך את השארית.
-   ב. כדי לפתור את הבעיה, יש לחסר את עלות המכר ממחיר המכירה, ולאחר מכן להכפיל את התוצאה בכמות המלאי, ולאחר מכן לעגל.
-   ג. כדי לפתור את הבעיה, יש להכפיל את המחיר בכמות המלאי, וגם את עלות המכר בכמות המלאי, ולאחר מכן לחסר אותם, ולאחר מכן לעגל את התוצאה למספר שלם.
-   ד. כדי לפתור את הבעיה, יש להשתמש רק בלולאות ובאופרטורים מותנים.

**הסבר:**

כדי לפתור את בעיית חישוב הרווח הכולל, יש לבצע ברצף מספר שלבים: חישוב סך ההכנסות ממכירה, חישוב סך עלות המכר, מציאת ההפרש בין גדלים אלו ועיגול התוצאה.

*   **אלגוריתם (חישוב ישיר):**
    1.  **חילוץ נתונים:** מקבלים את הערכים `cost_price`, `sell_price` ו-`inventory` מהמילון `data`.
    2.  **חישוב סך ההכנסות:** מחשבים את סך ההכנסות ממכירות `total_revenue`, כתוצאת המכפלה של `sell_price` ב-`inventory`.
    3.  **חישוב סך עלות המכר:** מחשבים את סך עלות המכר `total_cost` כתוצאת המכפלה של `cost_price` ב-`inventory`.
    4.  **חישוב רווח:** מוצאים את הרווח `profit`, על ידי חיסור `total_cost` מ-`total_revenue`.
    5.  **עיגול:** מעגלים את `profit` למספר השלם הקרוב ביותר באמצעות הפונקציה `round()`.
    6.  **החזרת התוצאה:** מחזירים את הרווח המעוגל.

**דוגמאות (פסאודו-קוד):**
```
function profit(data):
    cost_price = data["cost_price"]
    sell_price = data["sell_price"]
    inventory = data["inventory"]
    total_revenue = sell_price * inventory
    total_cost = cost_price * inventory
    profit = total_revenue - total_cost
    return round(profit)
```
**דוגמאות מימוש בפייתון:**

    cost_price = data["cost_price"]
    sell_price = data["sell_price"]
    inventory = data["inventory"]
    total_revenue = sell_price * inventory
    total_cost = cost_price * inventory
    profit = total_revenue - total_cost
    return round(profit)
