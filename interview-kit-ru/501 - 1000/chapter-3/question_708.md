### `question_68.md`

**פרק 1. שאלה 68.** מהי הפונקציונליות של המתודה `pop()` כאשר היא משמשת ברשימת פייתון?

- A. היא מוסיפה איבר לסוף הרשימה.
- B. היא מסירה ומחזירה את האיבר האחרון ברשימה.
- C. היא בוחרת ומסירה איבר מהרשימה באופן אקראי.
- D. היא ממיינת את הרשימה ולאחר מכן מסירה את האיבר הקטן ביותר מהרשימה.

**תשובה נכונה: B**

**הסבר:**

המתודה `pop()` בפייתון משמשת להסרה ו*החזרה* של איבר מרשימה. כברירת מחדל, אם אינדקס אינו מצוין, המתודה מסירה ומחזירה את האיבר האחרון ברשימה.

*   **אפשרות A** אינה נכונה: המתודה `append()` משמשת להוספת איברים לסוף הרשימה.
*   **אפשרות B** נכונה: `pop()` מסירה ומחזירה את האיבר האחרון ברשימה, אם אינדקס אינו מצוין.
*   **אפשרות C** אינה נכונה: `pop()` אינה בוחרת איבר באופן אקראי, ויכולה להסיר רק לפי אינדקס.
*   **אפשרות D** אינה נכונה: `pop()` אינה ממיינת את הרשימה, אלא רק מסירה איבר.

**כיצד פועלת `pop()`:**

1.  `pop()` נקראת על רשימה.
2.  אם אינדקס אינו מצוין, המתודה מסירה את האיבר האחרון ברשימה ומחזירה את ערכו.
3.  אם אינדקס מצוין, אזי המתודה מסירה את האיבר לפי אינדקס זה ומחזירה את ערכו.
4.  אם הרשימה ריקה, `pop()` תגרום לשגיאת `IndexError`.

**דוגמה:**

```python
my_list: list[int] = [1, 2, 3, 4, 5]

# הסרת האיבר האחרון
last_element: int = my_list.pop()
print(f"הרשימה לאחר pop(): {my_list}")  # פלט: הרשימה לאחר pop(): [1, 2, 3, 4]
print(f"האיבר שהוסר: {last_element}")  # פלט: האיבר שהוסר: 5

# הסרת איבר לפי אינדקס
second_element: int = my_list.pop(1)
print(f"הרשימה לאחר pop(1): {my_list}")   # פלט: הרשימה לאחר pop(1): [1, 3, 4]
print(f"האיבר שהוסר: {second_element}")  # פלט: האיבר שהוסר: 2
```

**תוצאה:**

*   קריאה ל-`my_list.pop()` תסיר ותחזיר את האיבר האחרון (5), ותשאיר את הרשימה במצב `[1, 2, 3, 4]`.
*   קריאה ל-`my_list.pop(1)` תסיר ותחזיר את האיבר באינדקס 1 (2), ותשאיר את הרשימה במצב `[1, 3, 4]`.

לפיכך, **אפשרות B** נכונה.
