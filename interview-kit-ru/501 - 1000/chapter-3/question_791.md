**דוגמה:**

```
קלט: s = '1222311'
פלט: '(1, 1) (3, 2) (1, 3) (2, 1)'

קלט: s = "111"
פלט: '(3, 1)'
```

-   ב. כדי לפתור את הבעיה יש להשתמש באלגוריתם רקורסיבי ולעבור על התווים ברצף.
-   ג. כדי לפתור את הבעיה, יש להשתמש בשיטת הקיבוץ `itertools.groupby`, בספירת אורך הקבוצה והאיבר הראשון שלה, ובעיצוב התוצאה למחרוזת.
-   ד. כדי לפתור את הבעיה יש להשתמש באלגוריתם חלון הזזה, שישמור על גבול שמאלי וימני לאיסוף נתונים על איברים חוזרים.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית הפיכת מחרוזת על ידי קיבוץ תווים חוזרים, השימוש האופטימלי הוא בפונקציה `itertools.groupby()`. שיטה זו מאפשרת לקבץ ביעילות תווים חוזרים ברצף, והיא תמציתית יותר בהשוואה לשיטות המשתמשות בלולאות.

*   **מודול `itertools` והפונקציה `groupby()`:**
    *   `itertools.groupby(iterable, key)` משמשת לקיבוץ איברים חוזרים רצופים מתוך `iterable`.
    *   `key` - ארגומנט אופציונלי, משמש לקביעה לפי איזה קריטריון לקבץ את האיברים.
    *   מחזירה איטרטור, שמחזיר זוגות (מפתח, איטרטור קבוצתי).
*   **בניית מחרוזת התוצאה:**
    *   מבצעים איטרציה על הקבוצות שהתקבלו מ-`groupby()`.
    *   בכל איטרציה: מחשבים את אורך הקבוצה ואת האיבר הראשון בקבוצה.
    *   בונים מחרוזת בצורה `(n, x)`, ומוסיפים למחרוזת התוצאה.

*   **יתרונות האלגוריתם:**
    *   **תמציתיות:** `itertools.groupby()` מאפשרת לתאר את קוד קיבוץ האיברים בקצרה.
    *   **יעילות:** עובר על המחרוזת פעם אחת בלבד.

**דוגמאות (פסאודו-קוד):**
```
function transform_string(s):
    result = ""
    for  key, group from groupby(s):
        count = length of group
        result = result +  (count,key)
    return result
```

**דוגמאות מימוש בפייתון:**
```python
import itertools

def occurrence(s):
    result = ""
    for key, group in itertools.groupby(s):
        result += f"({len(list(group))}, {key}) "
    return result.strip()

s1 = '1222311'
print(f"Ввод: s = '{s1}'")
print(f"Вывод: {occurrence(s1)}") # יפלט: (1, 1) (3, 2) (1, 3) (2, 1)

s2 = "111"
print(f"Ввод: s = '{s2}'")
print(f"Вывод: {occurrence(s2)}") # יפלט: (3, 1)

```
