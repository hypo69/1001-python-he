### `question_780.md`

**שאלה 780.** כיצד ניתן לבצע חיפוש תת-מחרוזת במחרוזת ב-Python באמצעות ביטויים רגולריים? תאר כיצד פועלות המתודות `re.findall()` ו-`re.match()`.

-   א. המתודה `re.findall()` מחפשת את המופע הראשון של תת-המחרוזת, והמתודה `re.match()` בודקת התאמה של המחרוזת כולה לתבנית.
-   ב. המתודה `re.findall()` משמשת לחיפוש כל המופעים החופפים, ואילו `re.match()` - לחיפוש מופעים שאינם חופפים.
-   ג. המתודה `re.findall()` משמשת לחיפוש כל המופעים של תת-המחרוזת במחרוזת, ואילו המתודה `re.match()` משמשת לחיפוש התאמה מתחילת המחרוזת.
-   ד. המתודה `re.findall()` מחזירה רק את האלמנט הראשון, ואילו `re.match()` מחזירה `None` אם אין התאמות.

**תשובה נכונה: ג**

**הסבר:**

ב-Python, לעבודה עם ביטויים רגולריים (regular expressions) נעשה שימוש במודול `re`. מודול זה מכיל את המתודות `re.findall()` ו-`re.match()`, אשר מבצעות סוגים שונים של חיפוש תת-מחרוזות.

*   **מודול `re`:**
    *   **ביטויים רגולריים:** משמשים לחיפוש ועיבוד טקסט לפי תבנית.
    *   **פונקציות:** מספק מתודות לחיפוש, התאמה, פיצול ושינוי מחרוזות בהתבסס על תבניות.
*   **מתודה `re.findall(pattern, string)`:**
    *   **חיפוש כל המופעים:** מחפשת את כל ההתאמות שאינן חופפות של התבנית (`pattern`) במחרוזת (`string`).
    *   **רשימת תוצאות:** מחזירה רשימה של תת-מחרוזות המתאימות לתבנית.
    *   **התאמות מרובות:** יכולה למצוא מספר התאמות.
*   **מתודה `re.match(pattern, string)`:**
    *   **חיפוש מתחילת המחרוזת:** מנסה להתאים את התבנית `pattern` לתחילת המחרוזת `string`.
    *   **התאמה:** אם התבנית מתאימה לתחילת המחרוזת, היא מחזירה אובייקט `Match`.
    *   **`None`:** אם אין התאמות, היא מחזירה `None`.
    *   **קיבוץ:** באמצעות סוגריים עגולים בתבנית ניתן ליצור קבוצות, אותן ניתן לקבל באמצעות `m.groups()`.

**דוגמאות:**
```python
import re

# דוגמה 1: חיפוש כל המופעים באמצעות re.findall
text = "He was carefully disguised but captured quickly by police."
result1 = re.findall(r"\w+ly\b", text)
print(f"המחרוזת '{text}'. תוצאה מ-re.findall: {result1}") # יציג: המחרוזת 'He was carefully disguised but captured quickly by police.'. תוצאה מ-re.findall: ['carefully', 'quickly']

# דוגמה 2: התאמה מתחילת המחרוזת עם re.match() וקיבוץ
m = re.match(r"(\d+)\.(\d+)", "24.1632")
if m:
    print(f"התאמה מתחילת המחרוזת (קבוצות): {m.groups()}")  # יציג: התאמה מתחילת המחרוזת (קבוצות): ('24', '1632')

# דוגמה 3: התאמה מתחילת המחרוזת ללא קיבוץ
m2 = re.match(r"a+", "aaaaabc")
if m2:
    print(f"התאמה מתחילת המחרוזת (קבוצה 0): {m2.group(0)}")  # יציג: התאמה מתחילת המחרוזת (קבוצה 0): aaaa

# דוגמה 4: חיפוש במחרוזת ללא match
text3 = "this is a string with numbers 1234 and 5678"
print(f"re.findall(\d+ , {text3}) : {re.findall(r"\d+", text3)}") # יציג re.findall(\d+ , this is a string with numbers 1234 and 5678) : ['1234', '5678']

# דוגמה 5: match באמצע המחרוזת
text4 = "some other 123 string"
m3 = re.match(r"\w+\s+(\d+)", text4)
if m3:
   print(f"התאמה באמצעות match: {m3.groups()}") # לא יוצג, כיוון ש-match מיושם לתחילת המחרוזת
text5 = "123 some other string"
m4 = re.match(r"(\d+)\s+\w+", text5)
if m4:
   print(f"התאמה באמצעות match: {m4.groups()}") # יציג: התאמה באמצעות match: ('123',)

```
**ניתוח האפשרויות:**
*   **א. המתודה `re.findall()` מחפשת את המופע הראשון של תת-המחרוזת, והמתודה `re.match()` בודקת התאמה של המחרוזת כולה לתבנית.:** שגוי.
*   **ב. המתודה `re.findall()` משמשת לחיפוש כל המופעים החופפים, ואילו `re.match()` - לחיפוש מופעים שאינם חופפים.:** שגוי.
*   **ג. המתודה `re.findall()` משמשת לחיפוש כל המופעים של תת-המחרוזת במחרוזת, ואילו המתודה `re.match()` משמשת לחיפוש התאמה מתחילת המחרוזת.:** נכון.
*   **ד. המתודה `re.findall()` מחזירה רק את האלמנט הראשון, ואילו `re.match()` מחזירה `None` אם אין התאמות.:** שגוי.

**לסיכום:**
*   `re.findall` מאפשר לקבל את כל ההתאמות במחרוזת.
*   `re.match` בודק התאמה מתחילת המחרוזת, ומאפשר לקבץ ערכים.
*   מתודות שונות ב-`re` מאפשרות עבודה גמישה עם מחרוזות.

לפיכך, התשובה הנכונה היא **ג. המתודה `re.findall()` משמשת לחיפוש כל המופעים של תת-המחרוזת במחרוזת, ואילו המתודה `re.match()` משמשת לחיפוש התאמה מתחילת המחרוזת.**
