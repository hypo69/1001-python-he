**שאלה 778.** כיצד ניתן למיין תווים של מחרוזת ב-Python בסדר אלפביתי?

*   א. למיון מחרוזת בסדר אלפביתי יש להשתמש במתודה `sort()`.
*   ב. למיון מחרוזת בסדר אלפביתי יש להשתמש בפונקציה `sorted()` ישירות, ללא מפתחות.
*   ג. למיון מחרוזת בסדר אלפביתי יש להשתמש בפונקציה `sorted()` עם הפרמטר האופציונלי `key`, שישמש למיון ללא תלות ברישיות.
*   ד. למיון מחרוזת יש לעבור על כל התווים ולהשתמש באלגוריתם מיון בועות.

**תשובה נכונה: ג**

**הסבר:**

למיון תווים של מחרוזת ב-Python משתמשים בפונקציה `sorted()`. פונקציה זו מחזירה רשימה ממוינת חדשה של תווים (לא ניתן למיין מחרוזת מכיוון שהיא טיפוס נתונים בלתי ניתן לשינוי), וניתן להשתמש בפרמטר נוסף `key` לציון פונקציה מותאמת אישית להשוואה, למשל למיון ללא תלות ברישיות.

*   **פונקציה `sorted()`:**
    *   **מחזירה רשימה חדשה:** הפונקציה מחזירה רשימה חדשה המכילה את התווים הממוינים של המחרוזת המקורית.
    *   **פרמטר `key`:** הפרמטר `key` מקבל פונקציה המשמשת לחישוב מפתח המיון, למשל `key=str.lower` למיון מחרוזות ללא תלות ברישיות.
    *   **אובייקט איטרבילי:** הפונקציה מקבלת כל אובייקט איטרבילי, כולל מחרוזות, רשימות, טיפוסים (tuples).
    *   **מיון:** ממיינת אלמנטים בסדר עולה (כברירת מחדל).

*   **מחרוזות ב-Python:**
    *   מחרוזות ב-Python הן בלתי ניתנות לשינוי.
    *   למיון תווים של מחרוזת, יש להמיר אותה לאובייקט איטרבילי, למשל לרשימה באמצעות פונקציית `list()`.

**דוגמאות:**

```python
# Example 1: Sorting a string in alphabetical order
word = 'Python'
sorted_word = sorted(word) # מחזיר רשימה של תווים ['P', 'h', 'n', 'o', 't', 'y']
print(f"Исходная строка: '{word}'") # מחרוזת מקורית: 'Python'
print(f"Отсортированная строка: {sorted_word}") # ידפיס: רשימה ממוינת: ['P', 'h', 'n', 'o', 't', 'y']
print(f"Отсортированная строка в виде строки: {''.join(sorted_word)}") # ידפיס: מחרוזת ממוינת בצורת מחרוזת: 'Phnoty'
# Example 2: Case-insensitive sorting
word = 'Python'
sorted_word_case_insensitive = sorted(word, key=lambda x: x.lower())
print(f"Исходная строка: '{word}'") # מחרוזת מקורית: 'Python'
print(f"Отсортированная строка (без учета регистра): {sorted_word_case_insensitive}")  # ידפיס: רשימה ממוינת (ללא תלות ברישיות): ['h', 'n', 'o', 'P', 't', 'y']

# Example 3: Sorting using str.lower()
word = 'Python'
sorted_word_case_insensitive_str_lower = sorted(word, key=str.lower)
print(f"Отсортированная строка (с  str.lower): {sorted_word_case_insensitive_str_lower}") #  ידפיס: רשימה ממוינת (עם str.lower): ['h', 'n', 'o', 'P', 't', 'y']

# Example 4: Original string remains unchanged
word2 = "test"
print(f"Исходная строка: {word2}") # מחרוזת מקורית: test
print(f"Итог  : {word2}") # ידפיס test

# Example 5: Empty string
word3 = ""
sorted_word3 = sorted(word3)
print(f"Исходная строка: {word3}") # מחרוזת מקורית:
print(f"Отсортированная строка (пустая): {sorted_word3}") # ידפיס: רשימה ממוינת (ריקה): []
```

**ניתוח האפשרויות:**
*   **א. למיון מחרוזת בסדר אלפביתי יש להשתמש במתודה `sort()`.:** לא נכון. המתודה `sort` משנה את הרשימה במקום ואינה ישימה למחרוזות.
*   **ב. למיון מחרוזת בסדר אלפביתי יש להשתמש במתודה `sorted()` ישירות, ללא מפתחות.:** לא נכון, מכיוון שהרישיות תילקח בחשבון.
*   **ג. למיון מחרוזת בסדר אלפביתי יש להשתמש בפונקציה `sorted()` עם הפרמטר האופציונלי `key`, שישמש למיון ללא תלות ברישיות.:** נכון.
*   **ד. למיון מחרוזת יש לעבור על כל התווים ולהשתמש באלגוריתם מיון בועות.:** לא נכון.

**לסיכום:**
*   פונקציית `sorted()` מאפשרת למיין אובייקטים איטרביליים, וכן מחרוזות.
*   הפרמטר `key` מאפשר להגדיר כלל מיון.
*   `str.lower()` משמש להשוואת תווים ללא תלות ברישיות.

לפיכך, התשובה הנכונה היא **ג. למיון מחרוזת בסדר אלפביתי יש להשתמש בפונקציה `sorted()` עם הפרמטר האופציונלי `key`, שישמש למיון ללא תלות ברישיות.**
