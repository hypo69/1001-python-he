### `question_748.md`

**שאלה 748.** נתון לכם מערך של מספרים שלמים `students`, המייצג את דירוג ההצלחה בלימודים של תלמידים. חשבו את המספר המינימלי של איברים שיש להסיר מהמערך על מנת לקבל מערך "שמוצא חן בעיני" מורה מדעי המחשב, איוואן איוואנוביץ' (א"א).

מערך ה-`students` מוצא חן בעיני א"א אם:
1. אורך המערך הוא מספר זוגי.
2. עבור כל התלמידים בעמדות זוגיות `students[i] != students[i+1]` (כלומר, לשכנים לספסל הלימודים צריך להיות דירוג שונה).

מערך ריק גם מוצא חן בעיני א"א.

*   חשוב: אין להחליף מקומות בין איברים, ניתן רק להסיר. כאשר איבר מוסר, האיברים מימינו מוזזים עמדה אחת שמאלה.

**דוגמאות:**

```
Ввод: students = [1, 1, 2, 3, 5]
Вывод: 1
Объяснение: ניתן להסיר את students[0] או את students[1] כדי לקבל את [1, 2, 3, 5].
```

```
Ввод: students = [1,1,1,2,3]
Вывод: 1
Объяснение: נסיר 1, ונקבל [1,2,3], לאחר מכן נסיר 1, נקבל [2,3]. ואם נסיר את ה-1 השלישי, נקבל: [1,1,2,3], וזו אינה אפשרות נכונה.
```

```
Ввод: students = [1,1,2,2,3,3]
Вывод: 3
Объяснение: יש להסיר לפחות אחד מכל זוג.
```

- A. לפתרון הבעיה יש לעבור על כל האפשרויות להסרה, לבחור את המספר המינימלי של הסרות, ולבדוק את התקפות של כל הפתרונות.
- B. לפתרון הבעיה יש להשתמש באלגוריתם חמדן (greedy), תוך הסרת איברים כפולים אחד אחד והסרת איבר אם אורך המערך אינו מספר זוגי.
- C. לפתרון הבעיה יש להשתמש בתכנות דינמי ולחפש את סדרת ההסרות האופטימלית, תוך שמירה של ערכים שחושבו קודם לכן.
- D. לפתרון הבעיה יש להשתמש באלגוריתם חמדן ולבדוק זוגות איברים בעמדות זוגיות, ולפי הצורך להסיר איברים, תוך מעבר על המערך משמאל לימין.

**התשובה הנכונה: C**

**הסבר:**

לפתרון הבעיה של מציאת המספר המינימלי של הסרות מהמערך במטרה ש"ימצא חן בעיני" א"א, יש להשתמש בתכנות דינמי, דבר שיאפשר לבנות את הפתרון על בסיס חישובים קודמים. במקביל, ניתן לבחור באופן אופטימלי איזה איבר להסיר אם קיימת התנגשות בעמדה זוגית.

*   **אלגוריתם (תכנות דינמי):**
    1.  **אתחול:**
        *   יוצרים מערך `dp`, באורך `n+1` עם ערכים 0. הערך `dp[i]` יתאים למספר המינימלי של איברים שהוסרו עד אינדקס `i`, כאשר תת-המערך `nums[:i]` מוצא חן בעיני א"א.
    2.  **איטרציה על המערך:** עוברים על המערך `students` מ-`2` עד הסוף:
        *   **בדיקת זוגיות:** אם האינדקס `i` זוגי, אז:
            *   אם `students[i-2]` ו-`students[i-1]` אינם שווים, אז לוקחים את הערך הקודם `dp[i-2]`.
            *   אם `students[i-2]` ו-`students[i-1]` שווים, אז לוקחים את המינימלי מבין `dp[i-1]+1` (מסירים את הקודם) ו-`dp[i-2]+1` (מסירים את הנוכחי).
        *   **אינדקס אי-זוגי:** אם האינדקס `i` אי-זוגי, אז פשוט מעתיקים את הערך הקודם `dp[i] = dp[i-1]`
    3.  **בדיקת זוגיות:** אם אורך הרשימה `n` אינו זוגי, אז מוסיפים `1` לערך ב-`dp[n-1]`.
    4.  **תוצאה:** מחזירים את הערך `dp[n]`, שמתאים למספר המינימלי של איברים שהוסרו.

*   **יתרונות האלגוריתם:**
    *   **תכנות דינמי:** משתמש במערך `dp` לאחסון פתרונות ביניים ולהימנעות מחישובים חוזרים.
    *   **אופטימליות:** מוצא את המספר המינימלי של הסרות הדרושות להשגת המצב הרצוי.
    *   **שימוש חוזר:** משתמש בתוצאות שחושבו קודם לכן.
    *   **סיבוכיות לינארית:** לאלגוריתם סיבוכיות זמן של `O(n)`.

**דוגמאות (פסאודו-קוד):**

```
function min_removals(students):
     n = length(students)
     dp = an array with length n+1 with defaults 0
    for i = 2 to n-1
       if i is even
           if students[i-2] != students[i-1]
                dp[i] = dp[i-2]
          else
             dp[i] = min(dp[i-1]+1, dp[i-2] +1)
        else
           dp[i] = dp[i-1]
     if len is odd
         dp[n] = dp[n-1] + 1
    return dp[n]
```

**דוגמאות מימוש ב-Python:**

```python
def min_removals(students):
    n = len(students)
    dp = [0] * (n + 1)
    for i in range(2, n + 1):
        if i % 2 == 0:
            if students[i-2] != students[i-1]:
               dp[i] = dp[i-2]
            else:
                dp[i] = min(dp[i-1] + 1, dp[i-2]+1)
        else:
             dp[i] = dp[i-1]
    if n % 2 != 0:
        dp[n] = dp[n-1] + 1

    return dp[n]

students1 = [1, 1, 2, 3, 5]
print(f"Ввод: students = {students1}")
print(f"Вывод: {min_removals(students1)}") # יוצא לאור: 1

students2 = [1,1,1,2,3]
print(f"Ввод: students = {students2}")
print(f"Вывод: {min_removals(students2)}") # יוצא לאור: 1

students3 = [1,1,2,2,3,3]
print(f"Ввод: students = {students3}")
print(f"Вывод: {min_removals(students3)}") # יוצא לאור: 3
```
