### `question_712.md`

**שאלה 712.** נתונה מערך של רשימות בשם `rooms`, כאשר `rooms[i]` מייצג רשימת מפתחות הנמצאים בחדר שמספרו `i`. בתחילה, רק חדר `0` פתוח. פתח אלגוריתם בפייתון שיקבע האם ניתן לבקר בכל החדרים, תוך שימוש במפתחות מכל חדר שבו ביקרנו כדי לגשת לחדרים אחרים.

**דוגמאות:**
```
Ввод: rooms = [[1], [2], [3], []]
Вывод: True
```
**הסבר:** ניתן לעבור דרך החדרים: 0 -> 1 -> 2 -> 3.
```
Ввод: rooms = [[1,3],[3,0,1],[2],[0]]
Вывод: False
```
**הסבר:** בלתי אפשרי להגיע לחדר 3.

*   א. לפתרון הבעיה נדרש להשתמש באלגוריתם חיפוש לעומק (DFS) ולעבור על כל המסלולים האפשריים, ולהחזיר True אם ניתן להגיע לכל החדרים.
*   ב. לפתרון הבעיה יש להשתמש באלגוריתם חמדן, להתחיל מהחדר האפס ולבחור בכל חדר את המסלול הקצר ביותר.
*   ג. לפתרון הבעיה יש להשתמש באלגוריתם חיפוש לרוחב (BFS), להתחיל מחדר 0 ולסמן חדרים שבוקרו, להמשיך את החיפוש עד שהתור יתרוקן.
*   ד. לפתרון הבעיה יש להשתמש באלגוריתם חיפוש בינארי.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית בדיקת האפשרות לבקר בכל החדרים בהינתן מפתחות בכל חדר, הדרך האופטימלית היא שימוש באלגוריתם חיפוש לרוחב (BFS). BFS מאפשר מעבר על גרף (במקרה שלנו, מבנה החדרים והמפתחות) בסדר עולה של מרחק מהצומת ההתחלתי, ומבטיח מציאת המסלול הקצר ביותר לכל צומת בר השגה.

*   **אלגוריתם (BFS):**
    1.  **אתחול:**
        *   יוצרים תור `queue` ומוסיפים אליו את הצומת ההתחלתי `0` (החדר ההתחלתי).
        *   יוצרים קבוצה `visited` כדי לעקוב אחר חדרים שכבר בוקרו, ומסמנים את החדר ההתחלתי `0` כחדר שבוקר.
    2.  **BFS:** כל עוד התור `queue` אינו ריק, מבצעים את הפעולות הבאות:
         *   מוציאים את מספר החדר הנוכחי `current_room` מהתור.
           *  **עיבוד מפתחות:** עוברים על כל המפתחות הכלולים בחדר `current_room`, בלולאת `for`:
               *  **בדיקת ביקור:** אם המפתח `next_room` טרם בוקר, מוסיפים אותו לתור ומסמנים אותו כבוקר.
        *   **מספר החדרים שבוקרו:** לאחר המעבר, בודקים האם כל החדרים בוקרו (אורך `visited` צריך להיות שווה ל-`n`).
     3. **תוצאה:**  מחזירים `True` אם כל החדרים ברי השגה, אחרת מחזירים `False`.

*   **יתרונות האלגוריתם:**
    *   **מעבר על גרף:** BFS מבצע מעבר יעיל על הגרף לפי רמות, ומבטיח מציאת כל החדרים הנגישים.
    *   **מסלול קצר ביותר:** מאפשר מציאת מסלול באורך מינימלי (מספר חדרים), אם קיים כזה.
    *   **נכונות:** מאפשר קביעה נכונה האם ניתן לבקר בכל החדרים.
      *  **יעילות:** בעל סיבוכיות זמן של O(n + m), כאשר n הוא מספר החדרים, ו-m הוא מספר המפתחות (הקשתות).

**דוגמאות (פסאודו-קוד):**
```
function can_visit_all_rooms(rooms):
  queue = new queue add 0
    visited = set add 0
    while queue is not empty:
        current_room = queue.removeHead()
         for key in rooms[current_room]:
            if key is not in visited:
                queue.add(key)
                 visited.add(key)
   return length(visited) == length(rooms)
```

**דוגמאות יישום בפייתון:**

```python
from collections import deque

def can_visit_all_rooms(rooms):
    n = len(rooms)
    queue = deque()
    queue.append(0)
    visited = {0}
    while queue:
        current_room = queue.popleft()
        for next_room in rooms[current_room]:
           if next_room not in visited:
               queue.append(next_room)
               visited.add(next_room)
    return len(visited) == n
rooms1 = [[1],[2],[3],[]]
print(f"Ввод: rooms = {rooms1}")
print(f"Вывод: {can_visit_all_rooms(rooms1)}") #  Выведет: Вывод: True
rooms2 = [[1,3],[3,0,1],[2],[0]]
print(f"Ввод: rooms = {rooms2}")
print(f"Вывод: {can_visit_all_rooms(rooms2)}")  # Выведет: Вывод: False
```

*   **ב. לפתרון הבעיה יש להשתמש באלגוריתם חמדן, להתחיל מהחדר האפס ולבחור בכל חדר את המסלול הקצר ביותר.:** שגוי. אלגוריתם חמדן אינו מבטיח ביקור בכל החדרים.
*   **ג. לפתרון הבעיה יש להשתמש באלגוריתם חיפוש לרוחב (BFS), להתחיל מחדר 0 ולסמן חדרים שבוקרו, להמשיך את החיפוש עד שהתור יתרוקן.:** נכון.
