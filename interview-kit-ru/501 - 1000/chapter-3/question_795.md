**שאלה 795.** פתחו פונקציה ב-Python בשם `find_median`, המקבלת כקלט רשימת מספרים `arr` ומחזירה את ערך חציון הסדרה המספרית הזו.

*   אם במדגם מספר זוגי של איברים, החציון נחשב לממוצע החשבוני של שני האיברים המרכזיים לאחר סידור.

**דוגמאות:**

```
Ввод: arr = [1, 5, 2, 3, 6]
Вывод: 3

Ввод: arr = [100, 5, 2, 4, 3, 6]
Вывод: 4.5
```

-   B. כדי לפתור את הבעיה יש להשתמש באלגוריתם רקורסיבי עם Backtracking, ולבחור את הערך הממוצע בעת מציאת המיקום הנכון של האיברים.

-  D. כדי לפתור את הבעיה ניתן להשתמש אך ורק במתודה `numpy.median()`.

**הסבר:**

כדי לפתור את הבעיה של מציאת החציון במערך, הגישה האופטימלית תהיה שימוש באלגוריתם מיון, עם בחירה עוקבת של איבר אחד או שני איברים ברשימה הממוינת למציאת החציון.

*   **אלגוריתם:**
    1.  **מיון המערך:** ממיינים את המערך `arr` באמצעות המתודה `sort()`, כדי למיין אותו בסדר עולה.
    2.  **קביעת אורך המערך:** קובעים את אורך המערך `n` באמצעות הפונקציה `len()`.
    3.  **חיפוש החציון:**
        *  **אורך אי-זוגי:** אם מספר האיברים `n` אי-זוגי, החציון הוא האיבר באמצע הרשימה הממוינת (עם האינדקס `n // 2`).
        *   **אורך זוגי:** אם מספר האיברים `n` זוגי, החציון מחושב כממוצע החשבוני של שני האיברים, הממוקמים באמצע הרשימה הממוינת. (עם האינדקסים `n // 2 - 1` ו-`n // 2`).
     4.  **החזרת התוצאה:** מחזירים את תוצאת החציון.

*  **יתרונות האלגוריתם:**
    *   **מיון:** מיון מבטיח את סדר האיברים במערך, מה שנדרש לחישוב נכון של החציון.
    * **פשטות:** הקוד מתקבל תמציתי ופשוט ליישום.
    *   **יעילות:** לאלגוריתם סיבוכיות של `O(n log n)` בשל המיון.

**דוגמאות (קוד מדומה):**
```
function find_median(arr):
    sort array arr in increasing order
    n = length of array
     if n is odd:
       median = arr[n // 2]
     else:
       median  =  (arr[n // 2 - 1]  + arr[n // 2]) / 2
    return median;
```

**דוגמאות מימוש ב-Python:**

```python
def find_median(arr):
    arr.sort()
    n = len(arr)
    if n % 2 == 1:
      return arr[n // 2]
    else:
      return (arr[n // 2 - 1] + arr[n // 2]) / 2

arr1 = [1, 5, 2, 3, 6]
print(f"Ввод: arr = {arr1}")
print(f"Вывод: {find_median(arr1)}")  # Выведет: Вывод: 3

arr2 = [100, 5, 2, 4, 3, 6]
print(f"Ввод: arr = {arr2}")
print(f"Вывод: {find_median(arr2)}") # Выведет: Вывод: 4.5
```
