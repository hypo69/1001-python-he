### `question_794.md`

**שאלה 794.** פתח/י דקורטור `RepeatDecorator` ב-Python, אשר בודק האם הפונקציה שעוטרה נקראה בעבר עם ארגומנטים פוזיציונליים נתונים. אם קריאה עם ארגומנטים כאלה כבר התבצעה, אזי הדקורטור צריך להדפיס למסך את ההודעה `"Функция с такими аргументами уже запускалась!"`. אחרת - הוא קורא לפונקציה המעוטרת.

*   הערה: הדקורטור צריך להתנהג בצורה נכונה במקרה בו מועטרות פונקציות שונות, עם אותם ארגומנטים.

**דוגמאות:**
```python
@RepeatDecorator
def func1(x, y):
    pass

@RepeatDecorator
def func2(x, y):
    pass
func1(1, 2)
# אין הדפסה
func1(2, 1)
# אין הדפסה
func1(3, 1)
# אין הדפסה
func1(1, 2)
# Функция с такими аргументами уже запускалась!

func2(1, 2)
# אין הדפסה, כי הפונקציה שונה
func2(1, 2)
# Функция с такими аргументами уже запускалась!
```

- א. לפתרון הבעיה יש להשתמש רק במשתנה גלובלי.
- ב. לפתרון הבעיה יש להשתמש רק במחסנית למעקב אחר פונקציות שנקראו בעבר.
- ג. לפתרון הבעיה יש להשתמש במילון (טבלת גיבוב), כאשר המפתח יהיה שם הפונקציה וטאפל של הארגומנטים הפוזיציונליים, ולבדוק האם מפתח כזה קיים.
- ד. לפתרון הבעיה יש להשתמש רק ברקורסיה ולשמור ערכים במחסנית.

**תשובה נכונה: ג**

**הסבר:**

לשם יישום דקורטור העוקב אחר קריאות חוזרות לפונקציה עם ארגומנטים פוזיציונליים זהים, הגישה האופטימלית היא שימוש במילון (טבלת גיבוב). הדבר מאפשר בדיקה יעילה של הימצאות פונקציות שנקראו בעבר עם הארגומנטים הנתונים.

*   **מושגי יסוד:**
    *   **מילון (טבלת גיבוב):** משמש לאחסון מידע אודות פונקציות שנקראו בעבר והארגומנטים הפוזיציונליים שלהן (בצורת טאפלים).
        * המפתח במילון הוא שם הפונקציה וטאפל של הארגומנטים הפוזיציונליים, מה שמאפשר לשמור נתונים מבודדים עבור כל פונקציה.
    *   **דקורטור:**
        * מקבל פונקציה כארגומנט.
        * מחזיר פונקציית עטיפה, שמבוצעת בעת קריאה לפונקציה המעוטרת.
    *   **בדיקה:** לפני קריאה לפונקציה המעוטרת, מתבצעת בדיקה במילון על מנת לקבוע האם כבר התבצעה קריאה בעבר עם אותם פרמטרים.
     *  **הדפסת הודעה:** אם המפתח כבר קיים במילון, מודפסת ההודעה `"Функция с такими аргументами уже запускалась!"`.
     *  **שמירת מצב:** אם המפתח אינו קיים, הפונקציה נקראת, ומפתח חדש נרשם בטבלת הגיבוב.

**דוגמאות (פסאודו-קוד):**

```
function RepeatDecorator(func):
    cache = new dictionary

    function wrapper(*args, **kwargs):
        key = (func.__name__, args) # create unique key (func name + args)
        if key is in cache:
            print that method with given args was called before
        else:
           cache[key] = True
            func(*args, **kwargs)
       
  return wrapper
```

**דוגמאות ליישום ב-Python:**

```python
from functools import wraps
def RepeatDecorator(func):
    cache = {}
    @wraps(func)
    def wrapper(*args, **kwargs):
        key = (func.__name__, *args)
        if key in cache:
            print("Функция с такими аргументами уже запускалась!")
        else:
            cache[key] = True
        return func(*args, **kwargs)
    return wrapper

@RepeatDecorator
def func1(x, y):
    pass

@RepeatDecorator
def func2(x, y):
    pass
func1(1, 2)
# אין הדפסה
func1(2, 1)
# אין הדפסה
func1(3, 1)
# אין הדפסה
func1(1, 2)
# Функция с такими аргументами уже запускалась!

func2(1, 2)
# אין הדפסה, כי הפונקציה שונה
func2(1, 2)
# Функция с такими аргументами уже запускалась!

```
**ניתוח האפשרויות:**

*   **א. לפתרון הבעיה יש להשתמש רק באופרטורי השוואה ואופרטורים תנאיים, ולעקוב אחר הפרמטרים שהועברו לפונקציה.:** לא נכון.
*    **ב. לפתרון הבעיה יש להשתמש בפונקציה מקוננת, ובמשתנה גלובלי, כדי לשמור את ערכי הפרמטרים בעת הקריאה.:** לא נכון, משתנים גלובליים לא יאפשרו מעקב אחר פרמטרים של פונקציות שונות.
*    **ג. לפתרון הבעיה יש להשתמש במילון, שבו המפתח יהיה טאפל של הארגומנטים ושם הפונקציה, ולבדוק האם מפתח כזה קיים.:** נכון.
*   **ד. לפתרון הבעיה ניתן להשתמש רק בסט, ולהוסיף אליו את ערכי כל הארגומנטים.:** לא נכון. סט לא מבטיח שארגומנטים שונים בסדר שונה לא ייתנו תוצאה זהה.

**לסיכום:**
*   שימוש במילון מאפשר מעקב אחר ייחודיות קריאה לפונקציה עם פרמטרים נתונים ולהימנע מקריאה חוזרת.
*  טאפל מאפשר שימוש בארגומנטים כמפתח למילון.
*   דקורטור מאפשר ליישם היגיון זה ללא שינוי קוד הפונקציות המעוטרות.

לפיכך, התשובה הנכונה היא **ג. לפתרון הבעיה יש להשתמש במילון, שבו המפתח יהיה טאפל של הארגומנטים ושם הפונקציה, ולבדוק האם מפתח כזה קיים.**
