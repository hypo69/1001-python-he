**שאלה 768.** פתח אלגוריתם בפייתון למשחק "פרים ופרות" (Bulls and Cows). האלגוריתם מקבל שתי מחרוזות באורך זהה: `secret` (המספר הסודי) ו-`guess` (הניחוש). הפונקציה צריכה להחזיר מחרוזת משוב בפורמט `"xAyB"`, כאשר `x` הוא מספר ה"פרים" (ספרות במיקומים הנכונים), ו-`y` הוא מספר ה"פרות" (ספרות הקיימות בשני המספרים אך במיקומים שונים).

**דוגמאות:**

```
Ввод: secret = "1807", guess = "7810"
Вывод: "1A3B"
Объяснение: 
    - 8 - это бык(на правильной позиции)
    - 1, 0 и 7 -  это коровы (присутствуют в обоих числах, но не на правильных позициях).
    
Ввод: secret = "1123", guess = "0111"
Вывод: "1A1B"
Объяснение:
    - 1 - это бык (одна  правильная позиция)
    - 1 и 1 - две коровы (так как в secret есть два 1, то надо  показать только 1 корову (то есть количество  цифр, которые не совпадают по индексу, но присутствуют в обоих строках).
```

**תשובה נכונה: C**

**הסבר:**

לפתרון הבעיה של מציאת מספר ה"פרים" וה"פרות" במשחק "פרים ופרות", הגישה האופטימלית היא שימוש בטבלת גיבוב (מילון) לספירת כמות כל תו במספר הסודי, מה שמאפשר להימנע מבדיקות והשוואות חוזרות ונשנות.

*   **אלגוריתם (טבלת גיבוב ואיטרציה):**
    1.  **אתחול:**
        *   יוצרים מילון ריק `secret_counts` לאחסון ספירת הספרות ב-`secret`.
        *   מאפסים את המונים `bulls = 0` ו-`cows = 0`.
    2.  **איתור פרים:** עוברים על המחרוזות, ועבור כל תו באותו אינדקס, אם התווים זהים, מגדילים את מונה ה-`bulls` ב-1.
    3.  **ספירת פרות:** לאחר מכן, מבצעים איטרציה על מחרוזת ה-`guess`, ובודקים האם תו זה קיים במילון `secret_counts`:
        *   אם הוא קיים, אך האינדקס של התו אינו תואם לאינדקס במחרוזת `secret`, וספירת התווים הללו גדולה מ-0, מוסיפים 1 לספירת ה-`cows` ומקטינים את הערך במילון `counts[symbol] = counts[symbol] -1`.
    4.  **עיצוב התוצאה:** יוצרים את מחרוזת התשובה בפורמט "xAyB", באמצעות ערכי `bulls` ו-`cows`.

*   **יתרונות האלגוריתם:**
    *   **יעילות:** משתמש בטבלת גיבוב (מילון), מה שמזרז את איתור ההתאמות.
    *   **סיבוכיות לינארית:** לאלגוריתם יש סיבוכיות O(n), כאשר n הוא אורך המחרוזת.
    *   **פשטות:** הקוד מובן וקל ליישום.

**דוגמאות (פסאודו-קוד):**
```
function get_bulls_and_cows(secret, guess):
    counts = dictionary to keep counts of chars in secret
    bulls = 0
    cows = 0

    for index and symbol of secret:
        add or increase number of this element to counts
    for index and symbol of guess:
        if symbol  at same index with symbol of secret are equal:
          increment bulls
        else:
           if symbol in counts and counts[symbol] > 0:
                increment cows
                reduce count by 1 in hash table.

    return bulls + "A" + cows + "B"
```
**דוגמאות ליישום בפייתון:**

```python
from collections import Counter

def get_bulls_and_cows(secret, guess):
    counts = Counter(secret)
    bulls = 0
    cows = 0
    for index, ch in enumerate(guess):
      if ch == secret[index]:
          bulls+=1
      else:
         if ch in counts and counts[ch] > 0:
              cows+=1
              counts[ch]-=1

    return f"{bulls}A{cows}B"
secret1 = "1807"
guess1 = "7810"
print(f"Ввод: secret = '{secret1}', guess = '{guess1}'")
print(f"Вывод: {get_bulls_and_cows(secret1, guess1)}")  # Вывод: 1A3B

secret2 = "1123"
guess2 = "0111"
print(f"Ввод: secret = '{secret2}', guess = '{guess2}'")
print(f"Вывод: {get_bulls_and_cows(secret2, guess2)}")   # Вывод: 1A1B

```
