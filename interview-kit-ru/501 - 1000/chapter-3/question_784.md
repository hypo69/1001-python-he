### `question_784.md`

**שאלה 784.** פתח פונקציה בפייתון שמקבלת כקלט רשימה של tuples, כאשר כל tuple מורכב ממחרוזת (שם המקצוע) ומספר שלם (הציון). הפונקציה צריכה למיין את רשימת ה-tuples הזו לפי הציון בסדר עולה, ולהחזיר את הרשימה הממוינת, תוך שימוש בפונקציית למדא כמפתח למיון.

**דוגמאות:**

```
קלט: [('Русский', 88), ('Физика', 90), ('Алгебра', 97), ('История', 82)]
פלט: [('История', 82), ('Русский', 88), ('Физика', 90), ('Алгебра', 97)]
```
- א. לפתרון הבעיה יש להשתמש באלגוריתם מיון רקורסיבי.
- ב. לפתרון הבעיה יש להשתמש במתודת `sort()` למיון הרשימה, ולהשתמש בפונקציית למדא להשוואה.
- ג. לפתרון הבעיה יש להשתמש בפונקציה `sorted()` ולציין בפונקציית למדא כמפתח שתחלץ את הציון מה-tuple.
- ד. לפתרון הבעיה יש להשתמש במתודות `min()` ו-`max()` לבחירת הערך המינימלי והמקסימלי, ולאחר מכן ליצור רשימה חדשה ממוינת.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית מיון רשימה של tuples לפי ערך מספרי (ציון), יש להשתמש בפונקציה `sorted()`, בציון פונקציית למדא כארגומנט `key` שתחזיר את האלמנט השני מה-tuple, אשר ישמש כמפתח להשוואה. גישה זו מאפשרת למיין באופן תמציתי רשימה של tuples לפי הציון.

*   **הפונקציה `sorted()`:**
    *   **מיון אובייקטים איטרביליים:** מחזירה רשימה ממוינת המבוססת על האלמנטים של אובייקט איטרבילי (רשימה, tuple, וכדומה) שהועבר אליה.
    *   **הפרמטר `key`:** משמש להגדרת פונקציה שמחזירה את המפתח לפיו יש להשוות את האלמנטים.
*   **פונקציית למדא (lambda function):**
    *   מאפשרת להגדיר פונקציה אנונימית שמחזירה את המפתח למיון.
    *   לדוגמה, `lambda x: x[1]` תחזיר את האלמנט השני של ה-tuple (את הציון).

**דוגמאות (פסאודו-קוד):**
```
function sort_list_of_tuples(list):
   return  sort list with key parameter as lambda function, returning element under index 1.
```

**דוגמאות ליישום בפייתון:**
```python
def sort_list_of_tuples(tuples_list):
    return sorted(tuples_list, key=lambda x: x[1])


data1 = [('Русский', 88), ('Физика', 90), ('Алгебра', 97), ('История', 82)]
print(f"קלט: {data1}")
print(f"פלט: {sort_list_of_tuples(data1)}")
# תוצאה:  [('История', 82), ('Русский', 88), ('Физика', 90), ('Алгебра', 97)]

data2 =  [('математика', 70), ('физика', 100), ('химия', 90), ('биология', 50), ('география', 70)]
print(f"קלט: {data2}")
print(f"פלט: {sort_list_of_tuples(data2)}") # תוצאה: : [('биология', 50), ('математика', 70), ('география', 70), ('химия', 90), ('физика', 100)]
```

**ניתוח האפשרויות:**
*   **א. לפתרון הבעיה יש להשתמש באלגוריתם מיון רקורסיבי.:** שגוי, זה לא יעיל.
*   **ב. לפתרון הבעיה יש להשתמש במתודת `sort()` למיון הרשימה, ולהשתמש בפונקציית למדא להשוואה.:** שגוי, המתודה `sort` משנה את הרשימה במקום, ואילו הפונקציה sorted יוצרת חדש.
*   **ג. לפתרון הבעיה יש להשתמש בפונקציה `sorted()` ולציין בפונקציית למדא כמפתח שתחלץ את הציון מה-tuple.:** נכון.
*   **ד. לפתרון הבעיה יש להשתמש במתודות `min()` ו-`max()` לבחירת הערך המינימלי והמקסימלי, ולאחר מכן ליצור רשימה חדשה ממוינת.:** שגוי.

**לסיכום:**
*   הפונקציה `sorted()` מאפשרת למיין אובייקטים איטרביליים.
*   פונקציית למדא בפרמטר `key` מאפשרת להגדיר לפי איזה פרמטר יש למיין.

לפיכך, התשובה הנכונה היא **ג. לפתרון הבעיה יש להשתמש בפונקציה `sorted()` ולציין בפונקציית למדא כמפתח שתחלץ את הציון מה-tuple.**
