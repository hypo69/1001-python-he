**פרק 1. שאלה 76.** מהו סוג הערך המוחזר של פונקציית למדא ב-Python?

- A. מספר שלם (Integer)
- B. מחרוזת (String)
- C. אובייקט מסוג פונקציה (Object of function type)
- D. ערך בוליאני (Boolean)

**תשובה נכונה: C**

**הסבר:**

פונקציות למדא ב-Python הן פונקציות אנונימיות (חסרות שם) המוגדרות באמצעות מילת המפתח `lambda`. הן נוצרות כאובייקטים מסוג פונקציה, כלומר הסוג שלהן הוא *function*.

*   **אפשרות A** אינה נכונה: פונקציית למדא אינה מחזירה בהכרח מספר שלם.
*   **אפשרות B** אינה נכונה: פונקציית למדא אינה מחזירה בהכרח מחרוזת.
*   **אפשרות C** נכונה: פונקציית למדא מחזירה אובייקט מסוג פונקציה.
*   **אפשרות D** אינה נכונה: פונקציית למדא אינה מחזירה בהכרח ערך בוליאני.

**כיצד פועלות פונקציות למדא:**
*   פונקציית למדא מוגדרת באמצעות מילת המפתח `lambda`, אחריה מופיעים הארגומנטים (אם ישנם), נקודתיים `:`, וביטוי המחושב ומחזיר ערך.
*   פונקציות למדא נוצרות באופן אנונימי במהלך זמן הריצה, והסוג שלהן הוא `function`.
*   הן מהוות אובייקטים מסוג "first-class", מה שמאפשר להעביר אותן לפונקציות אחרות, להחזיר אותן מפונקציות וכו'.

**דוגמה:**

```python
# פונקציית למדא להכפלה ב-2
multiply_by_2: Callable[[int], int] = lambda x: x * 2

# בדיקת סוג
print(f"סוג multiply_by_2: {type(multiply_by_2)}")  # פלט: סוג multiply_by_2: <class 'function'>

# קריאה לפונקציית הלמדא
result: int = multiply_by_2(5)
print(f"תוצאה: {result}") # פלט: תוצאה: 10
```

**בתוצאה:**
*   `type(multiply_by_2)` מראה ש-`multiply_by_2` הוא אובייקט מסוג `function`.
*   בעת קריאה לפונקציית הלמדא `multiply_by_2(5)`, היא מחזירה את המספר `10`.

לפיכך, **אפשרות C** נכונה.
