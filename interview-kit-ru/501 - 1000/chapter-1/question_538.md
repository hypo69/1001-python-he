**שאלה 540.** מה תהיה התוצאה שתתקבל בעת הרצת קוד Python הבא, המיישם את אלגוריתם מיון ערימה?

```python
import heapq

def heap_sort(arr):
    # ממיר את הרשימה arr לערימת מינימום "במקום", כלומר משנה את סדר האיברים במערך עצמו.
    heapq.heapify(arr)
    # מאותחל רשימה ריקה לאחסון האיברים הממוינים.
    sorted_arr = []
    # הלולאה מתבצעת כל עוד יש איברים בערימה.
    while arr:
        # שולף את האיבר הקטן ביותר מהערימה (השורש), מוסיף אותו ל- sorted_arr ובנוסף בונה מחדש את הערימה.
        sorted_arr.append(heapq.heappop(arr))
    # מחזיר את המערך הממוין.
    return sorted_arr

# דוגמה לשימוש
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
sorted_arr = heap_sort(arr)
print("Sorted array:", sorted_arr)
```

- A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`
- B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`
- C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`
- D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`

**תשובה נכונה: B**

**הסבר:**

הקוד המוצג מממש את אלגוריתם מיון ערימה (Heap Sort) תוך שימוש במודול `heapq` ב-Python.

*   **אלגוריתם מיון ערימה:**
    *   ממיר את המערך המקורי לערימת מינימום (min-heap).
    *   שולף את האיברים מהערימה בזה אחר זה וממקם אותם במערך הממוין.
    *   המודול `heapq` ב-Python מממש ערימת מינימום.
*   **תיאור הקוד:**
    *   `import heapq`: מייבא את המודול `heapq` לעבודה עם ערימה.
    *   `heap_sort(arr)`: פונקציה שמקבלת מערך `arr` כארגומנט.
        *   `heapq.heapify(arr)`: ממיר את הרשימה `arr` לערימת מינימום "במקום", כלומר משנה את סדר האיברים במערך עצמו.
        *   `sorted_arr = []`: מאותחל רשימה ריקה לאחסון האיברים הממוינים.
        *   `while arr:`: הלולאה מתבצעת כל עוד יש איברים בערימה.
        *   `sorted_arr.append(heapq.heappop(arr))`: שולף את האיבר הקטן ביותר מהערימה (השורש), מוסיף אותו ל-`sorted_arr` ובנוסף בונה מחדש את הערימה.
        *   `return sorted_arr`: מחזיר את המערך הממוין.

**ניתוח האפשרויות:**
*   **A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`:** לא נכון. זוהי תוצאה של מיון בסדר יורד.
*   **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`:** נכון. זוהי תוצאה של מיון בסדר עולה.
*   **C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`:** לא נכון.
*   **D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`:** לא נכון.

**לסיכום:**
*   הקוד מממש נכונה את אלגוריתם מיון ערימה למיון מערך בסדר עולה.
*   המודול `heapq` מפשט את העבודה עם ערימה ב-Python.
*   מיון ערימה מבטיח סיבוכיות זמן O(n log n) והוא אלגוריתם יעיל.
*   האלגוריתם ממיין נתונים "במקום" (in place), מה שאומר שהוא משתמש במינימום זיכרון נוסף.

לסיכום, התשובה הנכונה היא **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`**.
