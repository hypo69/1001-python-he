**שאלה 565.** מהי הערכה עצלה (lazy evaluation) ב-Python, וכיצד היא פועלת? ספק דוגמה המדגימה שימוש בחישובים עצלים.

-   א. הערכה עצלה היא שיטת הערכה שבה ביטויים מחושבים מיד לאחר הגדרתם.
-   ב. הערכה עצלה היא אסטרטגיית חישובים שבה ביטוי מחושב רק כאשר ערכו נחוץ. ב-Python ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.
-   ג. הערכה עצלה היא כאשר נעשה שימוש רק במשתנים שיש אליהם הפניה.
-   ד. הערכה עצלה היא שיטה להאצת חישובים על ידי ביצועם במקביל.

**תשובה נכונה: B**

**הסבר:**

הערכה עצלה (lazy evaluation) היא אסטרטגיית חישובים שבה ביטוי אינו מחושב מיד, אלא נדחה לרגע שבו ערכו אכן נחוץ. אסטרטגיה זו מאפשרת לחסוך במשאבי חישוב, שכן חישובים אינם מבוצעים עד שהתוצאה שלהם אכן נדרשת.

*   **מושגי יסוד בהערכה עצלה:**
    *   **חישובים נדחים:** ערכי ביטויים אינם מחושבים מיד, אלא רק כאשר הם נחוצים.
    *   **חיסכון במשאבים:** מאפשר לחסוך בזיכרון ובזמן מעבד, מכיוון שרק הערכים הנחוצים מחושבים.
    *   **סדרות אינסופיות:** מאפשר לייצג סדרות נתונים אינסופיות, שכן הערכים מחושבים לפי דרישה.
    *   **שימוש בגנרטורים ואיטרטורים:** Python מאפשר לממש חישובים עצלים באמצעות גנרטורים ואיטרטורים.

*   **גנרטורים וחישובים עצלים:**
    *   גנרטורים הם פונקציות מיוחדות המשתמשות ב-`yield` כדי להחזיר ערכים בזה אחר זה.
    *   גנרטורים מחשבים את הערך הבא רק כאשר נקראת הפונקציה next או מתבצע איטרציה בלולאת for, ולא מחשבים את כל הערכים מיד.

**דוגמה (מתוך טקסט השאלה):**

```python
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib_gen = fibonacci()
for i in range(10):
    print(next(fib_gen))
```

**תיאור הדוגמה:**

*   **`fibonacci()`:** פונקציית גנרטור, המייצרת סדרה אינסופית של מספרי פיבונאצ'י.
    *   `while True`: לולאה אינסופית
    *   `yield a`: מחזיר את מספר פיבונאצ'י הבא ומשהה את ביצוע הפונקציה.
*   `fib_gen = fibonacci()`: יוצר אובייקט מסוג גנרטור.
*   `for i in range(10)`: לולאה שתרוץ 10 פעמים.
*   `next(fib_gen)`: קורא לגנרטור, ומכריח אותו לחשב ולהחזיר את ערך פיבונאצ'י הבא.
*   הגנרטור מייצר ערכים לפי דרישה ורק כאשר הם נחוצים ללולאה.

**דוגמאות נוספות:**
```python
# הערכה עצלה באמצעות range
numbers = range(10000000000) # יוצר אובייקט range, אך המספרים אינם מחושבים מיד
print(numbers[0]) # חישובים אינם מתרחשים עד לרגע זה
print(numbers[1]) # range מחשב את האיברים בעצלות לפי דרישה.

# חישובים עצלים באמצעות map
numbers = [1,2,3,4]
squared_numbers = map(lambda x: x**2, numbers) # map אינו מחשב את הערכים, אלא רק מחזיר איטרטור
print(list(squared_numbers)) # ממירים לרשימה כדי לחשב את כל ריבועי המספרים.
```

**ניתוח האפשרויות:**
*   **א. הערכה עצלה היא שיטת הערכה שבה ביטויים מחושבים מיד לאחר הגדרתם.:** שגוי. זוהי ההפך מהערכה עצלה (הערכה חמה/eager evaluation).
*   **ב. הערכה עצלה היא אסטרטגיית חישובים שבה ביטוי מחושב רק כאשר ערכו נחוץ. ב-Python ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.:** נכון.
*   **ג. הערכה עצלה היא כאשר נעשה שימוש רק במשתנים שיש אליהם הפניה.:** שגוי. זה קשור לתחום ההכרזה (scope) של משתנים.
*   **ד. הערכה עצלה היא שיטה להאצת חישובים על ידי ביצועם במקביל.:** שגוי.
