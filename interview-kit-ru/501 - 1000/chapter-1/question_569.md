**שאלה 568.** מהי תבנית העיצוב הפונקציונלית "Reduce" (צמצום), וכיצד היא משמשת ב-Python? הבא דוגמה לשימוש בתבנית זו לחישוב סכום איברים באוסף.

-   א. תבנית "Reduce" (צמצום) משמשת לבחירת איברים מאוסף על סמך תנאי נתון.
-   ב. תבנית "Reduce" (צמצום) משמשת להמרת איברי אוסף לאוסף חדש.
-   ג. תבנית "Reduce" (צמצום) משמשת לשילוב כל איברי האוסף לערך תוצאה יחיד באמצעות פונקציה נתונה.
-   ד. תבנית "Reduce" (צמצום) משמשת למיון איברי אוסף בסדר מסוים.

**תשובה נכונה: ג**

**הסבר:**

תבנית "Reduce" (צמצום) היא תבנית פונקציונלית המשמשת לשילוב (צמצום) כל איברי האוסף לערך תוצאה יחיד. לצורך כך משתמשים בפונקציה המקבלת את ערך התוצאה הנוכחי (מצבר) ואת האיבר הנוכחי של האוסף, ומחזירה ערך תוצאה חדש.

*   **מושגי יסוד של תבנית "Reduce":**
    *   **צמצום אוסף:** יישום הדרגתי של פונקציה על איברי האוסף על מנת לקבל ערך סופי יחיד.
    *   **מצבר (Accumulator):** הפונקציה מקבלת מצבר (תוצאה) ואת האיבר הנוכחי ומחזירה תוצאה חדשה.
    *   **ערך התחלתי (אופציונלי):** ניתן להגדיר ערך התחלתי למצבר.
    *   **שילוב איברים:** יישום סדרתי של הפונקציה על איברים ותוצאות ביניים.
    *   **ערך תוצאה:** הערך הסופי לאחר עיבוד כל איברי האוסף.

*   **שימוש ב-Python:**
    *   תבנית "Reduce" ב-Python ניתנת למימוש באמצעות הפונקציה `functools.reduce()`.
    *   הפונקציה `functools.reduce()` מקבלת פונקציה ואוסף כארגומנטים.

**דוגמאות:**

```python
from functools import reduce

# דוגמה לחישוב סכום מספרים
numbers = [1, 2, 3, 4, 5]
sum_of_numbers = reduce(lambda x, y: x + y, numbers) # מבצעים איטרציה על הרשימה, מסכמים את המספרים
print(sum_of_numbers)  # יודפס: 15

# דוגמה לחישוב מכפלת מספרים
numbers = [1, 2, 3, 4, 5]
product_of_numbers = reduce(lambda x, y: x * y, numbers)
print(product_of_numbers)  # יודפס: 120

# דוגמה למציאת הערך המקסימלי
numbers = [1, 8, 3, 5, 2]
max_value = reduce(lambda x, y: x if x > y else y, numbers)
print(max_value) # יודפס: 8

# דוגמה לשרשור מחרוזות
strings = ["a", "b", "c", "d"]
concatenated_string = reduce(lambda x, y: x + y, strings)
print(concatenated_string) # יודפס: abcd

# דוגמה עם ערך התחלתי
numbers = [1, 2, 3, 4, 5]
sum_with_initial = reduce(lambda acc, x: acc + x, numbers, 10) # ערך התחלתי 10
print(sum_with_initial) # יודפס: 25

# דוגמה לשימוש ב-reduce עם list comprehension ו-filter

numbers = [1,2,3,4,5,6]
filtered_sum = reduce(lambda x,y: x+y, [i for i in numbers if i%2 == 0], 0 )
print(filtered_sum) # יודפס 12, (סכום המספרים הזוגיים מהרשימה)
```

**ניתוח דוגמאות:**
1.  **דוגמה לחישוב סכום מספרים**: הפונקציה `reduce` מקבלת את הפונקציה `lambda x, y: x + y`, המסכמת שני מספרים, ואת הרשימה `numbers`. לאחר מכן, היא מיישמת את הפונקציה באופן סדרתי על האיברים, מחזירה את התוצאה בכל איטרציה, ובסוף מחזירה את סכום כל המספרים.
2.  **דוגמה לחישוב מכפלת מספרים**: הפונקציה `reduce` מקבלת את הפונקציה `lambda x, y: x * y`, המכפילה שני מספרים, ואת הרשימה `numbers`. לאחר מכן, היא מיישמת את הפונקציה באופן סדרתי על האיברים, מחזירה את התוצאה בכל איטרציה, ובסוף מחזירה את מכפלת כל המספרים.
3.  **דוגמה למציאת הערך המקסימלי**: הפונקציה `reduce` מקבלת את הפונקציה `lambda x, y: x if x > y else y` המחזירה את הגדול מבין שני מספרים, ואת הרשימה `numbers`. לאחר מכן, היא מיישמת את הפונקציה באופן סדרתי על האיברים, מחזירה את התוצאה בכל איטרציה, ובסוף מחזירה את המספר המקסימלי מהרשימה.
4.  **דוגמה לשרשור מחרוזות:** הפונקציה `reduce` מקבלת את הפונקציה `lambda x, y: x + y` המשרשרת שתי מחרוזות, ואת רשימת המחרוזות `strings`. לאחר מכן, היא מיישמת את הפונקציה באופן סדרתי על האיברים, מחזירה את התוצאה בכל איטרציה, ובסוף מחזירה מחרוזת אחת כוללת.
5.  **דוגמה עם ערך התחלתי**: `reduce` עם ערך התחלתי `10`, שיהיה הארגומנט הראשון של פונקציית ה-lambda. לאחר מכן, הפונקציה תיושם על כל איברי הרשימה ועל תוצאות האיטרציות הקודמות.
6.  **דוגמה לשימוש ב-`reduce` עם list comprehension ו-`filter`**: באמצעות `list comprehension` ו-`filter` מחלצים רק את המספרים הזוגיים מהרשימה, ולאחר מכן באמצעות הפונקציה `reduce` מסכמים אותם.
