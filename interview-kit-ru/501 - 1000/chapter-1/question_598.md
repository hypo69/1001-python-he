**שאלה 598.** כיצד פועלת הפונקציה `reduce()` ב-Python? תאר את מטרתה והבא דוגמה לשימוש בה לצבירת (אגרגציה) איברים של רצף.

*   א. הפונקציה `reduce()` משמשת לבחירת איברים מרצף העונים על תנאי מסוים.
*   ב. הפונקציה `reduce()` משמשת ליצירת רשימות חדשות מרשימות קיימות עם איברים שעברו המרה (טרנספורמציה).
*   ג. הפונקציה `reduce()` משמשת לאיחוד כל איברי הרצף לערך תוצאה יחיד באמצעות פונקציה נתונה.
*   ד. הפונקציה `reduce()` משמשת למיון אוטומטי של איברים ברצף.

**תשובה נכונה: ג**

**הסבר:**

הפונקציה `reduce()` ב-Python היא פונקציה מהמודול `functools`, המשמשת לאיחוד סדרתי של איברי אובייקט איטרבילי לערך תוצאה יחיד, תוך שימוש בפונקציה נתונה. בניגוד ל-`map`, הפונקציה `reduce()` אינה יוצרת איטרטור עם תוצאות, אלא מחזירה תוצאה אחת.

*   **מאפיינים עיקריים של `reduce()`:**
    *   **צמצום (Reduction):** מבצעת פעולת צמצום, כלומר, לוקחת אובייקט איטרבילי ומצמצמת אותו לתוצאה אחת.
    *   **שני ארגומנטים:** מקבלת פונקציה (המקבלת שני ארגומנטים) ואובייקט איטרבילי.
    *   **צבר ואיבר:** בכל איטרציה, הפונקציה מקבלת את ערך הצבר ואת האיבר הבא ברצף.
    *   **ערך תוצאה יחיד:** מחזירה ערך סופי יחיד לאחר עיבוד כל האיברים.
    *   **ערך התחלתי (אופציונלי):** ניתן להעביר ארגומנט אופציונלי ל-`reduce` המשמש כערך התחלתי עבור הצבר.

*   **תחביר של `reduce()`:**
    *   `functools.reduce(function, iterable[, initializer])`
        *   `function` - פונקציה המקבלת שני ארגומנטים, שתיושם על איברי ה-`iterable`.
        *   `iterable` - אובייקט איטרבילי (רשימה, tuple וכו').
        *   `initializer` (לא חובה) - ערך התחלתי עבור הצבר.

**דוגמאות:**

```python
from functools import reduce

# Пример 1: вычисление суммы списка чисел
# דוגמה 1: חישוב סכום רשימת מספרים
numbers = [1, 2, 3, 4, 5]
sum_result = reduce(lambda x, y: x + y, numbers)
print(f"Сумма элементов: {sum_result}")  # Вывод: 15
# פלט: 15
print(f"סכום האיברים: {sum_result}")

# Пример 2: вычисление произведения списка чисел
# דוגמה 2: חישוב מכפלת רשימת מספרים
numbers = [1, 2, 3, 4]
product_result = reduce(lambda x, y: x * y, numbers)
print(f"Произведение элементов: {product_result}")  # Вывод: 24
# פלט: 24
print(f"מכפלת האיברים: {product_result}")

# Пример 3: поиск максимального элемента в списке
# דוגמה 3: מציאת האיבר המקסימלי ברשימה
numbers = [1, 8, 3, 5, 2]
max_result = reduce(lambda x, y: x if x > y else y, numbers)
print(f"Максимальное значение: {max_result}")  # Вывод: 8
# פלט: 8
print(f"הערך המקסימלי: {max_result}")

# Пример 4: объединение строк
# דוגמה 4: איחוד מחרוזות (שרשור)
strings = ["a", "b", "c", "d"]
combined_string = reduce(lambda x, y: x + y, strings)
print(f"Соединенная строка: {combined_string}")  # Вывод: abcd
# פלט: abcd
print(f"המחרוזת המאוחדת: {combined_string}")

# Пример с initial value
# דוגמה עם ערך התחלתי
numbers = [1, 2, 3, 4, 5]
sum_initial_result = reduce(lambda x, y: x + y, numbers, 10) # начальное значение аккумулятора равно 10
# ערך התחלתי של הצבר שווה ל-10
print(f"Сумма элементов с начальным значением: {sum_initial_result}")  # Вывод: 25
# פלט: 25
print(f"סכום האיברים עם ערך התחלתי: {sum_initial_result}")
```
**ניתוח דוגמאות:**

1.  **דוגמה 1 (סכום):**
    *   `reduce(lambda x, y: x + y, numbers)`: `reduce` מיישמת את פונקציית ה-lambda על איברי הרשימה `numbers`, ומחשבת את סכומם. באיטרציה הראשונה ה-lambda תקבל `1` ו-`2`, בשנייה את תוצאת הסכום `1 + 2 = 3` ואת האיבר הבא `3`, וכן הלאה.
2.  **דוגמה 2 (מכפלה):**
    *   `reduce(lambda x, y: x * y, numbers)`: מיישמת את פונקציית ה-lambda על איברי הרשימה `numbers`, ומחשבת את מכפלתם.
3.  **דוגמה 3 (מציאת מקסימום):**
    *   `reduce(lambda x, y: x if x > y else y, numbers)`: מיישמת את פונקציית ה-lambda על איברי הרשימה `numbers`, מחזירה את המספר הגדול מבין זוג מספרים, ובסוף מחזירה את המספר המקסימלי.
4.  **דוגמה 4 (מחרוזות):**
    *   `reduce(lambda x, y: x + y, strings)`: מיישמת את פונקציית ה-lambda לשרשור מחרוזות, ומחזירה מחרוזת אחת.
5.  **דוגמה עם ערך התחלתי:**
    *   `reduce(lambda x, y: x + y, numbers, 10)`: מחשבת את סכום האיברים עם ערך התחלתי של הצבר השווה ל-`10`.

**ניתוח אפשרויות:**

*   **ג. הפונקציה `reduce()` משמשת לאיחוד כל איברי הרצף לערך תוצאה יחיד באמצעות פונקציה נתונה.:** נכון.

*   בדרך כלל משמשת לצבירת נתונים (אגרגציה), כגון סכומים, מכפלות, מציאת מקסימום, מינימום וכו'.
*   הפונקציה `reduce` מקבלת כארגומנטים פונקציה ואובייקט איטרבילי. קיימת גם אפשרות להעביר ערך התחלתי עבור הצבר.
