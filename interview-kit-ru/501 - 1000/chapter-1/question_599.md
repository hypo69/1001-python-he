**שאלה 597.** כיצד פועלת הפונקציה `filter()` ב-Python? תאר/י את מטרתה והבא/י דוגמה לשימוש בה לבחירת אלמנטים מתוך סדרה בהתבסס על תנאי מסוים.

-   א. הפונקציה `filter()` משמשת ליצירת רשימות חדשות שבהן כל האלמנטים עברו טרנספורמציה באמצעות פונקציה נתונה.
-   ב. הפונקציה `filter()` משמשת למיון אלמנטים בסדרה.
-   ג. הפונקציה `filter()` משמשת לבחירת אלמנטים מתוך סדרה בהתבסס על תנאי נתון, ומחזירה איטרטור המכיל את האלמנטים שעבורם התנאי מתקיים (הוא True).
-   ד. הפונקציה `filter()` משמשת להמרה של כל אלמנטי הסדרה לערך יחיד.

**תשובה נכונה: ג**

**הסבר:**

הפונקציה `filter()` ב-Python היא פונקציה מובנית המשמשת לבחירת אלמנטים מתוך אובייקט איטרבילי (כגון רשימה, טאפל, וכו') בהתבסס על תנאי נתון. `filter()` מחזירה איטרטור המכיל רק את אותם אלמנטים שעבורם התנאי הנתון (הפונקציה) מחזיר `True`.

*   **מאפיינים עיקריים של `filter()`:**
    *   **אובייקט איטרבילי:** עובדת עם כל אובייקט איטרבילי.
    *   **פונקציית תנאי:** מקבלת פונקציה הקובעת אם אלמנט מתאים (מחזירה `True` או `False`).
    *   **חישובים עצלים (Lazy evaluation):** מחזירה איטרטור ולא רשימה, מה שמאפשר לחסוך במשאבים בעת עבודה עם כמות גדולה של נתונים.
    *   **בחירת אלמנטים:** מאפשרת ליצור אוספים חדשים בהתבסס על תנאי שיש לבדוק עבור אלמנטי האוסף המקורי.

*   **תחביר של `filter()`:**
    *   `filter(function, iterable)`
        *   `function`: הפונקציה המופעלת על כל אלמנט ומחזירה `True` או `False`.
        *   `iterable`: אובייקט איטרבילי (רשימה, טאפל וכו').
        *   מחזירה איטרטור.

**דוגמאות:**

```python
# דוגמה 1: סינון מספרים זוגיים
def is_even(x):
    return x % 2 == 0

numbers = [1, 2, 3, 4, 5, 6, 7, 8]
even_numbers = list(filter(is_even, numbers))
print(f"מספרים זוגיים: {even_numbers}")  # פלט: [2, 4, 6, 8]

# דוגמה 2: סינון מספרים גדולים מ-3
numbers = [1, 2, 3, 4, 5, 6, 7, 8]
filtered_numbers = list(filter(lambda x: x > 3, numbers))
print(f"מספרים גדולים מ-3: {filtered_numbers}")  # פלט: [4, 5, 6, 7, 8]


# דוגמה 3: סינון מחרוזות לפי אורך
strings = ["apple", "banana", "kiwi", "orange", "pear"]
filtered_strings = list(filter(lambda s: len(s) > 5, strings))
print(f"מחרוזות ארוכות מ-5 תווים: {filtered_strings}")  # פלט: ['banana', 'orange']

# דוגמה 4: סינון מחרוזות ריקות
strings = ["hello", "", "world", "", "python"]
non_empty_strings = list(filter(None, strings)) # אם None - אז מסוננים ערכי False (מחרוזות ריקות, 0, None)
print(f"מחרוזות ללא מחרוזות ריקות: {non_empty_strings}") # פלט: ['hello', 'world', 'python']

# דוגמה 5: סינון לפי טיפוס נתונים:
mixed_list = [1, "hello", 2, 3.14, "world", 4]
only_int = list(filter(lambda x : isinstance(x,int), mixed_list))
print(f"רשימת מספרים שלמים: {only_int}") # פלט: [1, 2, 4]
```

**ניתוח הדוגמאות:**
1.  **דוגמה 1:**
    *   `is_even(x)` - הפונקציה מחזירה `True` אם המספר זוגי.
    *   `filter(is_even, numbers)` - מחזירה איטרטור, שבו יהיו רק מספרים זוגיים.
2.  **דוגמה 2:**
    *   פונקציית למדא `lambda x: x > 3` בודקת האם המספר גדול מ-`3`.
    *   `filter(lambda x: x > 3, numbers)` - מחזירה איטרטור, שבו יהיו רק מספרים גדולים מ-3.
3.  **דוגמה 3:**
    *   פונקציית למדא `lambda s: len(s) > 5` בודקת האם אורך המחרוזת ארוך מ-5 תווים.
    *   `filter(lambda s: len(s) > 5, strings)` - מחזירה איטרטור, שבו יהיו רק מחרוזות שאורכן גדול מ-5.
4.  **דוגמה 4:** שימוש ב-`filter` עם `None` מאפשר להיפטר מערכי `False`, כגון מחרוזות ריקות.
5.  **דוגמה 5:** באמצעות פונקציית למדא `lambda x : isinstance(x,int)` ניתן לסנן רשימה ולהשאיר רק מספרים שלמים.

**ניתוח האפשרויות:**

*   **א. הפונקציה `filter()` משמשת ליצירת רשימות חדשות שבהן כל האלמנטים עברו טרנספורמציה באמצעות פונקציה נתונה.:** שגוי. זהו תיאור של הפונקציה `map()`.
*   **ב. הפונקציה `filter()` משמשת למיון אלמנטים בסדרה.:** שגוי. זהו תיאור של פונקציות מיון כמו `sorted()`.
*   **ג. הפונקציה `filter()` משמשת לבחירת אלמנטים מתוך סדרה בהתבסס על תנאי נתון, ומחזירה איטרטור המכיל את האלמנטים שעבורם התנאי מתקיים (הוא True).:** נכון. זהו ההסבר המדויק לפעולת הפונקציה.
*   **ד. הפונקציה `filter()` משמשת להמרה של כל אלמנטי הסדרה לערך יחיד.:** שגוי. זהו תיאור של פונקציות כמו `reduce()`.

**לסיכום:**
*   הפונקציה `filter()` היא כלי נוח לבחירת אלמנטים מאוסף בהתבסס על תנאי נתון.
*   `filter()` מחזירה איטרטור ולא רשימה.
*   ניתן להשתמש בפונקציות רגילות או בפונקציות למדא לבדיקת האלמנטים.

לפיכך, התשובה הנכונה היא **ג. הפונקציה `filter()` משמשת לבחירת אלמנטים מתוך סדרה בהתבסס על תנאי נתון, ומחזירה איטרטור המכיל את האלמנטים שעבורם התנאי מתקיים (הוא True).**
