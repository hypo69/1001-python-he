**שאלה 585.** מהם מודולים בפייתון?

-   A. מודולים בפייתון הם דרך ליצירת ממשק גרפי עבור יישומים.
-   B. מודולים בפייתון הם קבצים נפרדים המכילים הגדרות של פונקציות, מחלקות, משתנים וקוד אחר, שניתן להשתמש בהם בתוכניות אחרות.
-   C. מודולים בפייתון הם דרך לעבודה עם מסדי נתונים.
-   D. מודולים בפייתון הם דרך לבדיקת קוד באופן אוטומטי.

**תשובה נכונה: B**

**הסבר:**

בפייתון, מודולים הם קבצים נפרדים המכילים הגדרות של פונקציות, מחלקות, משתנים ואלמנטים אחרים של פייתון. הם מהווים את יחידת הארגון הבסיסית של קוד ומסייעים בחלוקת פרויקטים גדולים לחלקים קטנים יותר וניתנים לניהול.

*   **מאפיינים עיקריים של מודולים:**
    *   **קבצי פייתון:** כל מודול הוא קובץ פייתון רגיל עם סיומת `.py`.
    *   **מכילים הגדרות:** מודולים יכולים להכיל כל הגדרות פייתון (פונקציות, מחלקות, משתנים וכו').
    *   **ייבוא:** ניתן לייבא מודולים למודולים או סקריפטים אחרים, לצורך שימוש חוזר בקוד.
    *   **ארגון קוד:** מודולים מאפשרים לחלק קוד גדול לחלקים קטנים יותר והגיוניים.
    *   **שימוש חוזר בקוד:** מאפשרים שימוש חוזר בקוד בפרויקטים שונים.

*   **דוגמאות למודולים:**
    *   **הספרייה הסטנדרטית של פייתון:** לדוגמה, `os`, `sys`, `math`, `datetime`.
    *   **מודולים מותאמים אישית (User-defined):** מודולים שנוצרים על ידי מפתחים למשימות ספציפיות.
    *   **ספריות צד שלישי (Third-party):** מודולים שמותקנים באמצעות `pip`, לדוגמה `requests`, `numpy`, `pandas`.

*   **כיצד להשתמש במודולים:**
    *   לייבא מודול באמצעות אופרטור `import my_module`.
    *   לייבא אובייקטים ספציפיים ממודול: `from my_module import my_function`.
    *   לייבא את כל האובייקטים ממודול: `from my_module import *` (אינו מומלץ עקב התנגשויות שמות אפשריות).
    *   לייבא מודול עם כינוי (alias): `import my_module as mm`

**דוגמאות:**

```python
# דוגמה לשימוש במודול מובנה math
import math
print(math.sqrt(25)) # 5.0

# דוגמה לייבוא אובייקט ספציפי ממודול
from datetime import datetime
now = datetime.now()
print(now)

# דוגמה לייבוא מודול עם כינוי
import os as operating_system
print(operating_system.getcwd())

# דוגמה למודול מותאם אישית
# קובץ my_module.py
def my_function(x):
    return x * 2
# קובץ main.py
import my_module
print(my_module.my_function(5)) # 10
```
**ניתוח האפשרויות:**
*   **A. מודולים בפייתון — זהו דרך ליצירת ממשק גרפי עבור יישומים.:** לא נכון.
*   **B. מודולים בפייתון — זהו קבצים נפרדים המכילים הגדרות של פונקציות, מחלקות, משתנים וקוד אחר, שניתן להשתמש בהם בתוכניות אחרות.:** נכון.
*   **C. מודולים בפייתון - זהו דרך לעבודה עם מסדי נתונים.:** לא נכון.
*   **D. מודולים בפייתון - זהו דרך לבדיקת קוד באופן אוטומטי.:** לא נכון.

**לסיכום:**
*   מודולים מהווים את יחידת הארגון הבסיסית של קוד בפייתון.
*   מודולים מאפשרים כתיבת קוד מובנה יותר וניתן לשימוש חוזר.

לפיכך, התשובה הנכונה היא **B. מודולים בפייתון הם קבצים נפרדים המכילים הגדרות של פונקציות, מחלקות, משתנים וקוד אחר, שניתן להשתמש בהם בתוכניות אחרות.**
