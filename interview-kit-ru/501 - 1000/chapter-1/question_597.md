**שאלה 597.** כיצד פועלת הפונקציה `map()` בפייתון? תאר/י את ייעודה והצג/י דוגמה לשימוש בה.

-   A. הפונקציה `map()` משמשת לסינון אלמנטים ברצף על בסיס תנאי מוגדר.
-   B. הפונקציה `map()` משמשת לאיחוד כל האלמנטים ברצף לערך יחיד.
-   C. הפונקציה `map()` מיושמת למיון אלמנטים ברצף בסדר עולה.
-   D. הפונקציה `map()` מיישמת פונקציה מוגדרת על כל אלמנט באובייקט איטרבילי ומחזירה איטרטור עם התוצאות.

**תשובה נכונה: D**

**הסבר:**

הפונקציה `map()` בפייתון היא פונקציה מובנית, המיושמת לביצוע פונקציה מוגדרת על כל אלמנט באובייקט איטרבילי (לדוגמה, רשימה, טאפל, קבוצה, מילון, וכו'). `map()` מחזירה איטרטור, המכיל את תוצאות יישום הפונקציה.

*   **מאפיינים עיקריים של `map()`:**
    *   **אובייקט איטרבילי:** הפונקציה `map()` עובדת עם כל אובייקט איטרבילי.
    *   **פונקציית טרנספורמציה:** מקבלת פונקציה כארגומנט הראשון, אשר תיושם על אלמנטים של האובייקט האיטרבילי.
    *   **חישוב עצלני:** `map()` אינה מחשבת את התוצאות באופן מיידי, אלא יוצרת איטרטור המייצר אותן לפי דרישה.
    *   **איטרטור תוצאות:** הפונקציה מחזירה איטרטור המכיל את האלמנטים שעברו טרנספורמציה.
    *   **טרנספורמציה:** משמשת לטרנספורמציה של אלמנטים באוסף.

*   **תחביר `map()`:**
    *   `map(function, iterable, ...)`
         *  `function` - הפונקציה המיושמת על כל אלמנט של `iterable`.
         *   `iterable` - אובייקט איטרבילי (רשימה, טאפל, מחרוזת ואחרים). ניתן להעביר מספר אובייקטים איטרביליים.
         * מחזירה איטרטור.

**דוגמאות:**
```python
# דוגמה 1: שימוש ב-map() עם פונקציה רגילה
def add_5(x):
    return x + 5

numbers = [1, 2, 3, 4, 5]
result = map(add_5, numbers)
print(list(result))  # פלט: [6, 7, 8, 9, 10]

# דוגמה 2: שימוש ב-map() עם פונקציית למדא
numbers = [1, 2, 3, 4, 5]
result = map(lambda x: x * 2, numbers)
print(list(result))  # פלט: [2, 4, 6, 8, 10]

# דוגמה 3: שימוש ב-map() עם מספר אובייקטים איטרביליים
list1 = [1, 2, 3]
list2 = [4, 5, 6]
result = map(lambda x, y: x + y, list1, list2)
print(list(result))  # פלט: [5, 7, 9]

# דוגמה עם המרת סוגים
strings = ["1", "2", "3"]
int_numbers = list(map(int, strings))
print(int_numbers) # פלט: [1, 2, 3]
```

**ניתוח האפשרויות:**
*   **A. הפונקציה `map()` משמשת לסינון אלמנטים ברצף על בסיס תנאי מוגדר.:** לא נכון.
*   **B. הפונקציה `map()` משמשת לאיחוד כל האלמנטים ברצף לערך יחיד.:** לא נכון. זה מבוצע על ידי הפונקציה `reduce`.
*  **C. הפונקציה `map()` מיושמת למיון אלמנטים ברצף בסדר עולה.:** לא נכון.
*   **D. הפונקציה `map()` מיישמת פונקציה מוגדרת על כל אלמנט באובייקט איטרבילי ומחזירה איטרטור עם התוצאות.:** נכון.

**לסיכום:**
*  `map` היא פונקציה נוחה ליישום טרנספורמציה על כל האלמנטים של אובייקט איטרבילי.
*  מחזירה איטרטור.
* `map` יכולה להיות מיושמת על אובייקט איטרבילי אחד או יותר.

לפיכך, התשובה הנכונה היא **D. הפונקציה `map()` מיישמת פונקציה מוגדרת על כל אלמנט באובייקט איטרבילי ומחזירה איטרטור עם התוצאות.**
