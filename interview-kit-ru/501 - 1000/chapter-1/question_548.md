**שאלה 548.** מהן הדרכים העיקריות להעתקת רשימות ב-Python, ומה ההבדל בין העתקה שטחית להעתקה עמוקה? הצג דוגמאות המדגימות את שיטות ההעתקה השונות ואת השפעתן.

-   A. ב-Python קיימת רק דרך אחת להעתקת רשימות - שימוש בשיטה `copy()`, כל שאר הדרכים יוצרות הפניות לרשימה המקורית.
-   B. קיימות דרכים להעתקה שטחית (`[:]`, `list()`, `copy()`) ולהעתקה עמוקה (`deepcopy()`). העתקה שטחית יוצרת רשימה חדשה, אך האלמנטים של רשימות מקוננות נשארים הפניות לרשימה המקורית. העתקה עמוקה יוצרת עותקים בלתי תלויים לחלוטין של הרשימות, כולל כל האובייקטים המקוננים.
-   C. קיימות שלוש דרכים להעתקת רשימות: `[:]`, `list()` ו-`copy()`, אשר מבצעות העתקה עמוקה.
-   D. ב-Python לא ניתן להעתיק רשימות, ניתן רק לשנות את שמן.

**תשובה נכונה: B**

**הסבר:**

ב-Python קיימות מספר דרכים להעתקת רשימות, אשר יוצרות עותקים *שטחיים* או עותקים *עמוקים*. הבנת ההבדל ביניהם חיונית למניעת שגיאות הקשורות לשינויים לא רצויים בנתונים.

1.  **העתקה שטחית (Shallow Copy):**
    *   יוצרת אובייקט רשימה חדש, אך אינה מעתיקה באופן רקורסיבי את האובייקטים הכלולים ברשימה המקורית.
    *   אם אלמנטים ברשימה המקורית הם עצמם אובייקטים ניתנים לשינוי (לדוגמה, רשימות מקוננות), הרי שברשימה שהועתקה ישמרו הפניות לאובייקטים אלו, ולא עותקים שלהם.
    *   דרכים ליצירת עותק שטחי:
        *   **פרוסה (`[:]`):** `new_list = old_list[:]`
        *   **פונקציה `list()`:** `new_list = list(old_list)`
        *   **שיטה `copy()`:** `new_list = old_list.copy()`
2.  **העתקה עמוקה (Deep Copy):**
    *   יוצרת אובייקט רשימה חדש ומעתיקה באופן רקורסיבי את כל האובייקטים הכלולים ברשימה המקורית, כולל אובייקטים מקוננים ניתנים לשינוי.
    *   שינויים ברשימה שהועתקה באופן עמוק לא ישפיעו על הרשימה המקורית ולהיפך.
    *   משתמשים בפונקציה `deepcopy()` מהמודול `copy`.
        *   `import copy; new_list = copy.deepcopy(old_list)`

**דוגמאות קוד (העתקה שטחית):**

```python
import copy

old_list = [1, 2, [3, 4]]

# העתקה שטחית (באמצעות פרוסה)
new_list1 = old_list[:]

# העתקה שטחית (באמצעות list())
new_list2 = list(old_list)

# העתקה שטחית (באמצעות .copy())
new_list3 = old_list.copy()

print(f"Original list id: {id(old_list)}")
print(f"new_list1 id: {id(new_list1)}")
print(f"new_list2 id: {id(new_list2)}")
print(f"new_list3 id: {id(new_list3)}")


old_list[0] = 100 # שינוי הרשימה הישנה
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

old_list[2][0] = 5 # שינוי ברשימה המקוננת
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

```

**פלט:**
```
Original list id: 140493391085632
new_list1 id: 140493391092032
new_list2 id: 140493391091520
new_list3 id: 140493391091456
new_list1: [1, 2, [3, 4]]
new_list2: [1, 2, [3, 4]]
new_list3: [1, 2, [3, 4]]
new_list1: [1, 2, [5, 4]]
new_list2: [1, 2, [5, 4]]
new_list3: [1, 2, [5, 4]]
```

**דוגמאות קוד (העתקה עמוקה):**

```python
import copy

old_list = [1, 2, [3, 4]]

# העתקה עמוקה (באמצעות deepcopy)
new_list = copy.deepcopy(old_list)
print(f"Original list id: {id(old_list)}")
print(f"new_list id: {id(new_list)}")


old_list[0] = 100
old_list[2][0] = 5
print(f"old_list: {old_list}")
print(f"new_list: {new_list}")
```

**פלט:**

```
Original list id: 140493391086464
new_list id: 140493391085952
old_list: [100, 2, [5, 4]]
new_list: [1, 2, [3, 4]]
```
