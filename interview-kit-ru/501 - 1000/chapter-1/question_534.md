**שאלה 534.** מהי התוצאה שתתקבל בעת ביצוע קוד הפייתון הבא, המיישם את אלגוריתם מיון ההכנסה?

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

# דוגמת שימוש
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
sorted_arr = insertion_sort(arr)
print("Sorted array:", sorted_arr)
```

-   A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`
-   B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`
-   C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`
-   D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`

**תשובה נכונה: B**

**הסבר:**

הקוד המסופק מיישם את אלגוריתם מיון ההכנסה (insertion sort).

*   **אלגוריתם מיון ההכנסה:**
    *   עובר על המערך החל מהאיבר השני.
    *   עבור כל איבר, משווה אותו לאיברים בחלק הממוין של המערך (משמאלו).
    *   מזיז איברים הגדולים מהאיבר הנוכחי בעמדה אחת ימינה, על מנת למצוא מקום להכנסת האיבר הנוכחי.
    *   מכניס את האיבר הנוכחי למקום שנמצא.
*   **תיאור הקוד:**
    *   `insertion_sort(arr)`: פונקציה שמקבלת את המערך `arr` כארגומנט.
    *   `for i in range(1, len(arr))`: הלולאה החיצונית עוברת על המערך, החל מהאיבר השני.
    *   `key = arr[i]`: שומרת את האיבר הנוכחי שיש להכניס למקומו הנכון.
    *   `j = i - 1`: מגדירה את האינדקס להשוואה עם איברי החלק הממוין של המערך.
    *   `while j >= 0 and key < arr[j]`: הלולאה מזיזה איברים הגדולים מהנוכחי ימינה.
    *   `arr[j + 1] = arr[j]`: מזיזה את האיבר ימינה.
    *   `j -= 1`: מקטינה את האינדקס להשוואה עם האיבר הקודם.
    *   `arr[j + 1] = key`: מכניסה את האיבר הנוכחי למקום שנמצא.
    *   `return arr`: מחזירה את המערך הממוין.

**ניתוח האפשרויות:**
*   **A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`:** שגוי, זהו סדר מיון יורד.
*   **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`:** נכון. תוצאת המיון בסדר עולה.
*   **C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`:** שגוי.
*   **D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`:** שגוי.

**לסיכום:**
*   הקוד מיישם נכונה את אלגוריתם מיון ההכנסה למיון מערך בסדר עולה.
*   מיון הכנסה עשוי להיות יעיל יותר ממיון בועות במערכים קטנים וכמעט ממוינים.

לפיכך, התשובה הנכונה היא **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`**.
