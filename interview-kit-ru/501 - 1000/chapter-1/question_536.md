**שאלה 536.** איזו תוצאה תתקבל בעת הרצת קוד ה-Python הבא, המממש את אלגוריתם המיון המהיר?

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)


# Пример использования
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
sorted_arr = quick_sort(arr)
print("Sorted array:", sorted_arr)
```

-   A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`
-   B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`
-   C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`
-   D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`

**תשובה נכונה: B**

**הסבר:**

הקוד המוצג מממש את אלגוריתם המיון המהיר (quick sort), אשר משתמש בעקרון "הפרד ומשול".

*   **אלגוריתם המיון המהיר:**
    *   בוחר איבר ציר (pivot) מתוך המערך.
    *   מפצל את המערך לשלושה חלקים: איברים הקטנים מאיבר הציר, איברים השווים לאיבר הציר, ואיברים הגדולים מאיבר הציר.
    *   ממיין באופן רקורסיבי את תת-המערכים משמאל ומימין לאיבר הציר.
    *   מאחד את תת-המערכים הממוינים יחד עם איבר הציר.
*   **תיאור הקוד:**
    *   `quick_sort(arr)`: פונקציה שמקבלת את המערך `arr` כארגומנט.
        *   `if len(arr) <= 1`: מקרה בסיס לרקורסיה: אם אורך המערך קטן או שווה ל-1, הוא מוחזר כפי שהוא.
        *   `pivot = arr[len(arr) // 2]`: בוחר איבר ציר מאמצע המערך (באופן פשטני).
        *   `left = [x for x in arr if x < pivot]`: יוצר רשימה של איברים הקטנים מאיבר הציר.
        *   `middle = [x for x in arr if x == pivot]`: יוצר רשימה של איברים השווים לאיבר הציר.
        *   `right = [x for x in arr if x > pivot]`: יוצר רשימה של איברים הגדולים מאיבר הציר.
        *   `return quick_sort(left) + middle + quick_sort(right)`: קורא באופן רקורסיבי ל-`quick_sort` עבור `left` ו-`right` ומחזיר את המערך הממוין המאוחד.

**ניתוח האפשרויות:**
*   **A. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [90, 64, 34, 25, 22, 12, 11]`:** שגוי. זהו מיון בסדר יורד.
*   **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`:** נכון. זוהי תוצאת מיון בסדר עולה.
*   **C. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 90, 12, 64, 22, 34, 25]`:** שגוי. התוצאה אינה ממוינת.
*   **D. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [64, 34, 25, 22, 12, 11, 90]`:** שגוי. זוהי הרשימה המקורית, לא ממוינת.

**לסיכום:**
*   הקוד מממש נכונה את אלגוריתם המיון המהיר למיון מערך בסדר עולה.
*   מיון מהיר הוא אחד מאלגוריתמי המיון היעילים ביותר.
*   בחירת איבר הציר יכולה להשפיע על ביצועי האלגוריתם, אך בדוגמה זו מדובר בגרסה פשוטה.

לפיכך, התשובה הנכונה היא **B. `Original array: [64, 34, 25, 12, 22, 11, 90] Sorted array: [11, 12, 22, 25, 34, 64, 90]`**.
