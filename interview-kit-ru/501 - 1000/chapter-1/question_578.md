**שאלה 578.** מהו פולימורפיזם בתכנות מונחה עצמים (תמ"ע), וכיצד ניתן לממש אותו בפייתון? הצג דוגמה המדגימה שימוש בפולימורפיזם באמצעות ירושה ודריסת מתודות.

-   א. פולימורפיזם הוא היכולת ליצור עצמים בעלי שמות שונים.
-   ב. פולימורפיזם הוא היכולת לאחד מספר מחלקות לאחת.
-   ג. פולימורפיזם הוא עיקרון בתמ"ע המאפשר לעצמים ממחלקות שונות להגיב לאותה מתודה באופן שונה. בפייתון, ניתן לממש פולימורפיזם באמצעות ירושה ודריסת מתודות.
-   ד. פולימורפיזם הוא היכולת ליצור מופעים של מחלקה בזמן ריצת התוכנית.

**התשובה הנכונה: ג**

**הסבר:**

פולימורפיזם (polymorphism) הוא אחד מעקרונות המפתח בתכנות מונחה עצמים, המאפשר לעצמים ממחלקות שונות להגיב לאותה מתודה (או ממשק) באופן שונה. הוא מאפשר לעבוד עם עצמים ממחלקות שונות כאילו היו עצמים מטיפוס משותף אחד, מה שתורם לגמישות וליכולת הרחבת הקוד.

*   **מושגי יסוד של פולימורפיזם:**
    *   **ממשק אחיד, מימוש שונה:** לעצמים ממחלקות שונות יכולות להיות מתודות עם שמות זהים, אך המימושים שלהן יכולים להיות שונים.
    *   **ירושה ודריסה:** פולימורפיזם ממומש לעיתים קרובות באמצעות ירושה ודריסת מתודות במחלקות בנות.
    *   **גמישות הקוד:** מאפשר לכתוב קוד שאינו תלוי בטיפוסים ספציפיים של עצמים ויכול לעבוד עם עצמים ממחלקות שונות דרך ממשק משותף.
    *   **יכולת הרחבה:** מאפשר להוסיף טיפוסים חדשים של עצמים שיכולים לעבוד עם הקוד הקיים מבלי להכניס בו שינויים.

*   **מימוש פולימורפיזם בפייתון:**
    *   שימוש במחלקות היורשות ממחלקת בסיס משותפת אחת.
    *   דריסת מתודות במחלקות בנות (תת-מחלקות).
    *   שימוש ב-Duck Typing (אם לעצם יש את המתודות הנדרשות, אז ניתן להשתמש בו).

**דוגמה (מטקסט השאלה):**

```python
class Animal:
    def speak(self):
        # המתודה זורקת חריגה מסוג NotImplementedError מכיוון שהיא חייבת להיות ממומשת במחלקות הבנות.
        raise NotImplementedError

class Dog(Animal):
    def speak(self):
        # המחלקה Dog דורסת את המתודה speak() כדי להחזיר את המחרוזת "Woof!".
        return "Woof!"

class Cat(Animal):
    def speak(self):
        # המחלקה Cat דורסת את המתודה speak() כדי להחזיר את המחרוזת "Meow!".
        return "Meow!"

# נוצרת רשימה המכילה עצמים מהמחלקות Dog ו-Cat.
animals = [Dog(), Cat()]
# בלולאת for נקראת המתודה speak() עבור כל עצם מהרשימה, ולמרות שהעצמים הם מטיפוסים שונים, נקרא המימוש המתאים של המתודה speak.
for animal in animals:
    print(animal.speak())  # פלט: Woof!, Meow!
```
**תיאור הדוגמה:**

1.  **`class Animal`:** מחלקת בסיס `Animal` עם המתודה `speak()`. המתודה זורקת חריגה מסוג `NotImplementedError` מכיוון שהיא חייבת להיות ממומשת במחלקות הבנות.
2.  **`class Dog(Animal)`:** המחלקה `Dog`, שיורשת מ-`Animal` ודורסת את המתודה `speak()` כדי להחזיר את המחרוזת `"Woof!"`.
3.  **`class Cat(Animal)`:** המחלקה `Cat`, שיורשת מ-`Animal` ודורסת את המתודה `speak()` כדי להחזיר את המחרוסטת `"Meow!"`.
4.  **`animals`**: נוצרת רשימה המכילה עצמים מהמחלקות `Dog` ו-`Cat`.
5.  בלולאת `for` נקראת המתודה `speak()` עבור כל עצם מהרשימה, ולמרות שהעצמים הם מטיפוסים שונים, נקרא המימוש המתאים של המתודה `speak`.

```python
# דוגמה נוספת המשתמשת ב-duck typing:
class Duck:
    def quack(self):
        print("Quack!")

class NotDuck:
    def quack(self):
         print("Imitation quack")

def make_noise(obj):
  obj.quack()

duck = Duck()
not_duck = NotDuck()

make_noise(duck) # פלט: Quack!
make_noise(not_duck) # פלט: Imitation quack

# פולימורפיזם באמצעות מחלקות מופשטות
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
       # זוהי מתודה מופשטת שחייבת להיות ממומשת במחלקות היורשות.
       pass

class Square(Shape):
    def __init__(self, side):
        self.side = side
    def area(self):
        # מימוש המתודה area עבור ריבוע.
        return self.side * self.side

class Circle(Shape):
    def __init__(self, radius):
      self.radius = radius
    def area(self):
        # מימוש המתודה area עבור מעגל.
        return 3.14* self.radius * self.radius


def print_area(shape):
  # הפונקציה יכולה לקבל כל עצם שיש לו מתודת area().
  print(f"Area of the shape {shape.area()}")

square = Square(5)
circle = Circle(4)

print_area(square) # פלט: 25
print_area(circle) # פלט: 50.24 (בערך)
```
