### `question_551.md`

**שאלה 551.** מהן פונקציות למדא (lambda functions) בפייתון, ובאילו מצבים השימוש בהן הכי יעיל?

-   א. פונקציות למדא הן פונקציות שיכולות להכיל מספר ביטויים ומשמשות למימוש אלגוריתמים מורכבים.
-   ב. פונקציות למדא הן פונקציות אנונימיות (חסרות שם), המורכבות מביטוי יחיד בלבד. הן נוחות ליצירת פונקציות קצרות המועברות כארגומנטים לפונקציות אחרות.
-   ג. פונקציות למדא הן פונקציות שיכולות לבצע אופטימיזציה אוטומטית לקוד פייתון ולהגדיל את ביצועי התוכניות.
-   ד. פונקציות למדא הן פונקציות מיוחדות לעבודה עם קבצים ומסדי נתונים.

**תשובה נכונה: ב**

**הסבר:**

פונקציות למדא (או פונקציות אנונימיות) בפייתון הן פונקציות קטנות, חד-שורתיות, שאין להן שם. הן מוגדרות באמצעות מילת המפתח `lambda` ובדרך כלל משמשות כאשר יש צורך בפונקציה פשוטה לזמן קצר וללא צורך להגדיר אותה בנפרד (באמצעות `def`).

*   **תכונות עיקריות של פונקציות למדא:**
    *   **אנונימיות:** אין להן שם (חסרות שם).
    *   **ביטוי יחיד:** מורכבות מביטוי אחד בלבד, שמחושב ומוחזר.
    *   **קומפקטיות:** מיועדות ליצירת פונקציות קצרות ופשוטות.
    *   **שימוש עם פונקציות מסדר גבוה:** לעתים קרובות משמשות כארגומנטים לפונקציות מסדר גבוה, כמו `map()`, `filter()`, `sorted()`, ואחרות.

*   **תחביר פונקציות למדא:**

    `lambda arguments: expression`

**מתי כדאי להשתמש בפונקציות למדא:**

*   כאשר יש צורך בפונקציה לביצוע פעולה פשוטה.
*   כאשר הפונקציה משמשת במקום אחד ואין צורך להשתמש בה שוב.
*   כארגומנט לפונקציות מסדר גבוה.
*   לצמצום קוד והגברת קריאותו במקרים מסוימים.

**דוגמה (מתוך טקסט השאלה):**

```python
numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(lambda x: x**2, numbers))

print(squared_numbers)  # [1, 4, 9, 16, 25]
```

**תיאור הדוגמה:**

*   **`lambda x: x**2`:** פונקציית למדא שמקבלת ארגומנט יחיד `x` ומחזירה את הריבוע שלו (`x**2`).
*   **`map(lambda x: x**2, numbers)`:** הפונקציה map מיישמת את פונקציית הלמדא על כל אלמנט ברשימה `numbers` ומחזירה איטרטור.
*   `list(...)`: ממיר את האיטרטור לרשימה.
*   פונקציית למדא כאן משמשת לחישוב קומפקטי של ריבוע כל מספר ברשימה.

**דוגמאות נוספות:**

```python
# למדא לחיבור שני מספרים
add = lambda x, y: x + y
print(add(5, 3)) # תדפיס 8


# למדא לבדיקת זוגיות מספר
is_even = lambda x: x % 2 == 0
print(is_even(4)) # תדפיס True
print(is_even(3)) # תדפיס False

# למדא עם filter()
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers) # תדפיס [2, 4, 6]
```

**ניתוח האפשרויות:**
*   **א. פונקציות למדא הן פונקציות שיכולות להכיל מספר ביטויים ומשמשות למימוש אלגוריתמים מורכבים.:** לא נכון. לפונקציות למדא יש ביטוי יחיד בלבד והן מיועדות לפעולות פשוטות.
*   **ב. פונקציות למדא הן פונקציות אנונימיות (חסרות שם), המורכבות מביטוי יחיד בלבד. הן נוחות ליצירת פונקציות קצרות המועברות כארגומנטים לפונקציות אחרות.:** נכון.
*   **ג. פונקציות למדא הן פונקציות שיכולות לבצע אופטימיזציה אוטומטית לקוד פייתון ולהגדיל את ביצועי התוכניות.:** לא נכון. פונקציות למדא אינן מבצעות אופטימיזציה של קוד.
*   **ד. פונקציות למדא הן פונקציות מיוחדות לעבודה עם קבצים ומסדי נתונים.:** לא נכון. פונקציות למדא משמשות במשימות שונות, לא רק בעבודה עם קבצים ומסדי נתונים.

**לסיכום:**
*   פונקציות למדא מספקות דרך תמציתית להגדרת פונקציות פשוטות בפייתון.
*   הן שימושיות במיוחד בעת עבודה עם פונקציות מסדר גבוה ובכתיבת קוד קצר ואקספרסיבי.

לפיכך, התשובה הנכונה היא **ב. פונקציות למדא הן פונקציות אנונימיות (חסרות שם), המורכבות מביטוי יחיד בלבד. הן נוחות ליצירת פונקציות קצרות המועברות כארגומנטים לפונקציות אחרות.**
