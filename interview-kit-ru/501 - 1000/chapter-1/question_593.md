**שאלה 593.** כיצד ניתן להגדיר מחלקה `Car` ב-Python עם תכונות (`attributes`) `color` ו-`speed`, ליצור מופע (`instance`) שלה, ולקבל את ערך התכונה `speed`?

-   A. להגדרת מחלקה נעשה שימוש במילת המפתח `type`, והתכונות נוצרות כמשתנים גלובליים הזמינים מחוץ למחלקה. כדי לגשת לתכונה `speed`, יש לפנות ל-`car['speed']`.
-   B. להגדרת מחלקה נעשה שימוש במילת המפתח `class`, והתכונות נוצרות תוך שימוש ב-`self` בבנאי (`constructor`) `__init__`. כדי לגשת לתכונה `speed`, נעשה שימוש בתחביר `car.speed`.
-   C. להגדרת מחלקה נעשה שימוש במילת המפתח `def`, והתכונות נוצרות בתוך שיטה (`method`) של המחלקה. כדי לגשת לתכונה `speed`, יש לקרוא לשיטת המחלקה שתחזיר את הערך הזה.
-   D. להגדרת מחלקה נעשה שימוש במילת המפתח `object`, והתכונות נוצרות כמשתנים סטטיים (`static variables`). כדי לגשת לתכונה `speed`, יש להשתמש בתחביר `object.speed`.**

**תשובה נכונה: B**

**הסבר:**

ב-Python, מחלקות מוגדרות באמצעות מילת המפתח `class`, ולצורך אתחול מופעי המחלקה נעשה שימוש בשיטה מיוחדת `__init__`. בתוך שיטות המחלקה, כדי לגשת לתכונות ולשיטות של המופע, נעשה שימוש במילת המפתח `self`.

*   **הגדרת מחלקה:**
    *   מחלקות מוגדרות באמצעות מילת המפתח `class`, ולאחריה שם המחלקה.
    *   לדוגמה `class Car:`
*   **בנאי `__init__`:**
    *   השיטה `__init__()` היא הבנאי של המחלקה.
    * היא נקראת אוטומטית בעת יצירת מופע חדש של המחלקה.
    * הפרמטר הראשון `self` הוא הפניה למופע שזה עתה נוצר.
    *   `self` משמש לאתחול תכונות המופע.
*   **יצירת מופע:**
    *   ליצירת מופע של המחלקה, יש לקרוא למחלקה כפונקציה, ולהעביר לה את הארגומנטים הנדרשים: `car = Car("red", "100mph")`.
*   **גישה לתכונות:**
    *   תכונות המופע נגישות באמצעות תחביר נקודה (`dot notation`), תוך שימוש בשם המופע: `car.speed`.

**דוגמאות:**
```python
class Car:
    def __init__(self, color, speed):
        self.color = color
        self.speed = speed


car = Car('red','100mph') # יצירת מופע מחלקה
print(car.speed) # פלט: 100mph

# דוגמה עם סדר ארגומנטים שונה, אך עם שמות פרמטרים

car2 = Car(speed='200mph', color='green')
print(car2.speed) # פלט: 200mph

# דוגמה עם סוגי נתונים שונים

class Bike:
    def __init__(self, brand: str, price: float):
       self.brand = brand
       self.price = price
bike = Bike('BMX', 150.5)
print(f"Brand: {bike.brand}, price: {bike.price}") # פלט: Brand: BMX, price: 150.5
```
**ניתוח האפשרויות:**
*   **A. להגדרת מחלקה נעשה שימוש במילת המפתח `type`, והתכונות נוצרות כמשתנים גלובליים הזמינים מחוץ למחלקה. כדי לגשת לתכונה `speed`, יש לפנות ל-`car['speed']`. :** שגוי.
*   **B. להגדרת מחלקה נעשה שימוש במילת המפתח `class`, והתכונות נוצרות תוך שימוש ב-`self` בבנאי `__init__`. כדי לגשת לתכונה `speed`, נעשה שימוש בתחביר `car.speed`. :** נכון.
*   **C. להגדרת מחלקה נעשה שימוש במילת המפתח `def`, והתכונות נוצרות בתוך שיטה של המחלקה. כדי לגשת לתכונה `speed`, יש לקרוא לשיטת המחלקה שתחזיר את הערך הזה.:** שגוי.
*   **D. להגדרת מחלקה נעשה שימוש במילת המפתח `object`, והתכונות נוצרות כמשתנים סטטיים. כדי לגשת לתכונה `speed`, יש להשתמש בתחביר `object.speed`.** שגוי.

**לסיכום:**
*   `class` היא מילת המפתח ליצירת מחלקה.
*   `__init__()` משמשת לאתחול תכונות.
*   `self` מאפשרת גישה למופע מתוך השיטה.
*   `car.speed` מאפשר גישה לתכונה.
