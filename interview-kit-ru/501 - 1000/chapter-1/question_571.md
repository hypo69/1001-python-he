### question_570.md

**שאלה 570.** מהי תבנית העיצוב הפונקציונלית "Currying" (קאריינג) וכיצד היא משמשת ב-Python? אנא ספקו דוגמה לשימוש בקאריינג ליצירת גרסאות מתמחות של פונקציה.

-   א. קאריינג הוא תהליך שבו פונקציה מבוצעת פעם אחת בלבד.
-  ב. קאריינג הוא תהליך שבו פונקציה המקבלת מספר ארגומנטים מומרת לפונקציה המקבלת ארגומנט יחיד ומחזירה פונקציה חדשה שמצפה לארגומנטים הנותרים.
-  ג. קאריינג הוא תהליך שבו פונקציה מיושמת על סדרה של מספרים.
-  ד. קאריינג הוא תהליך שבו נוצרים סגורים (closures).

**תשובה נכונה: ב**

**הסבר:**

קאריינג (currying) הוא תבנית פונקציונלית המאפשרת להמיר פונקציה המקבלת מספר ארגומנטים לסדרה של פונקציות, שכל אחת מהן מקבלת ארגומנט יחיד בלבד. תהליך זה מאפשר ליצור גרסאות מתמחות יותר של פונקציות ומקל על שימוש חוזר בהן.

*   **מושגי יסוד בקאריינג:**
    *   **המרת פונקציה:** קאריינג ממיר פונקציה בעלת מספר ארגומנטים לשרשרת של פונקציות בעלות ארגומנט יחיד.
    *   **יישום סדרתי:** קריאה לפונקציה עם מספר ארגומנטים מתבצעת באופן סדרתי, על ידי קריאה לפונקציות המוחזרות מהקריאות הקודמות.
    *   **יישום חלקי:** כל קריאה לפונקציה שבוצעה לה קאריינג מקבעת ערכים עבור אחד הארגומנטים, ומחזירה פונקציה עם ארייטי (מספר הארגומנטים המצופים) מופחת.

*  **מימוש קאריינג ב-Python:**
    *  ב-Python, ניתן לממש קאריינג באמצעות יצירת שרשרת של פונקציות מקוננות.

**דוגמאות:**

```python
# Example of currying using nested functions
def curried_add(x):
    def add_y(y):
        def add_z(z):
            return x + y + z
        return add_z
    return add_y

add_5_and_7 = curried_add(5)(7)
print(add_5_and_7(10)) # Will output: 22
print(curried_add(5)(7)(10)) # Will output: 22

# Example of currying using lambda functions

def curried_multiplier(x):
   return lambda y : lambda z: x * y * z


multiply_by_2_and_3 = curried_multiplier(2)(3)
print(multiply_by_2_and_3(5))  # Will output: 30
print(curried_multiplier(2)(3)(5)) # Will output: 30
```

**ניתוח הדוגמאות:**

1.  **`curried_add(x)`:**
    *   מחזירה את הפונקציה המקוננת `add_y(y)`.
2.  **`add_y(y)`:**
     *   מחזירה את הפונקציה המקוננת `add_z(z)`.
3.  **`add_z(z)`:**
      *  מסכמת את `x`, `y` ו-`z` ומחזירה את התוצאה.
    *   `curried_add(5)`  מחזירה פונקציה `add_y`, שבה `x` שווה ל-5.
     * `curried_add(5)(7)` מחזירה פונקציה `add_z`, שבה `x = 5` ו-`y = 7`.
     *   `curried_add(5)(7)(10)` קוראת ל-`add_z(10)` עם `x=5`, `y=7` ו-`z=10`.
4. **`curried_multiplier(x)`:**
    *   מקבלת ארגומנט יחיד `x` ומחזירה פונקציית למדא שמקבלת `y`.
    *  מחזירה פונקציית למדא שנייה שמקבלת `z` ומחזירה את התוצאה `x*y*z`.
5.   `curried_multiplier(2)(3)(5)`: קריאות סדרתיות מחזירות פונקציות שמחשבות `2 * 3 * 5`.

**ניתוח האפשרויות:**
*   **א. קאריינג - זה תהליך שבו פונקציה מבוצעת פעם אחת בלבד:** לא נכון.

*  **ג. קאריינג - זה תהליך שבו פונקציה מיושמת על סדרה של מספרים:** לא נכון.
*   **ד. קאריינג - זה תהליך שבו נוצרים סגורים (closures):** לא נכון, אף על פי שבקאריינג נעשה שימוש תכוף בסגורים, לא כל סגור הוא קאריינג.
