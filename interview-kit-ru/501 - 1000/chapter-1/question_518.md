### `question_401.md`

**שאלה 401.** איזו תוצאה תודפס על המסך על ידי קוד Python הבא?

```python
my_str = "python"
result = my_str[::-2]
print(result)
```

-   A. `"nhy"`
-   B. `"otp"`
-   C. `"pto"`
-   D. `Error`

**תשובה נכונה: B**

**הסבר:**

קוד זה מדגים שימוש בטכניקת חיתוך (slicing) מחרוזות עם צעד שלילי ב-Python.

1.  **מחרוזת מקורית:** למשתנה `my_str` מושם הערך "python".
2.  **חיתוך מחרוזת:** הביטוי `my_str[::-2]` יוצר חיתוך חדש של המחרוזת.
    *   הנקודתיים הראשונות `:` ללא ציון אינדקס התחלה פירושן שהחיתוך מתחיל מתחילת המחרוזת.
    *   הנקודתיים השניות `:` ללא ציון אינדקס סיום פירושן שהחיתוך ממשיך עד סוף המחרוזת.
    *   `-2` – זהו הצעד, המציין שיש לבחור אלמנטים בסדר הפוך בדילוג של אלמנט אחד בכל פעם.
3.  **תוצאת החיתוך:**
    *   החיתוך יתחיל מהתו האחרון 'n', לאחר מכן ייקח את התו שלפניו 'o', אחר כך ידלג על התו 'h' וכן הלאה.
    *   לפיכך, כתוצאה מהחיתוך, ייבחרו התווים "n", "t", "p", כלומר התווים באינדקסים 5, 3, 1 בהתאמה.
    *   מאחר שהחיתוך מתבצע בסדר הפוך, התוצאה תהיה `"otp"`.

4.  **פלט:** הפקודה `print(result)` מדפיסה את חיתוך המחרוזת שהתקבל.

**ביצוע צעד-אחר-צעד של חיתוך `[::-2]`:**

| אינדקס | תו   | כלול בתוצאה |
| :---- | :--- | :---------- |
| 0     | p    |             |
| 1     | y    | o           |
| 2     | t    |             |
| 3     | h    | t           |
| 4     | o    |             |
| 5     | n    | p           |

**ניתוח האפשרויות:**

*   **A. `"nhy"`**: שגוי, מכיוון שסדר התווים אינו נלקח בחשבון כהלכה.
*   **B. `"otp"`**: נכון, החיתוך מחזיר את המחרוזת בסדר הפוך עם צעד 2.
*   **C. `"pto"`**: שגוי, מכיוון שסדר התווים אינו נלקח בחשבון כהלכה.
*   **D. `Error`**: שגוי, הקוד יתבצע ללא שגיאות.

**לסיכום:**

*   חיתוכים של מחרוזות עם צעד שלילי מאפשרים לקבל בקלות תת-מחרוזות בסדר הפוך.
*   הבנת אופן פעולת החיתוכים חשובה לעיבוד מחרוזות יעיל ב-Python.

לפיכך, התשובה הנכונה היא **B. `"otp"`**.
