### `question_81.md`

**פרק 1. שאלה 81.** מהו ערך ההחזרה המוגדר כברירת מחדל של פונקציה ב-Python, אם פקודת ה-`return` אינה כלולה באופן מפורש?

- A.  `0`
- B.  `None`
- C.  `False`
- D.  מחרוזת ריקה `""`

**תשובה נכונה: B**

**הסבר:**

ב-Python, אם פונקציה אינה מכילה פקודת `return` מפורשת, או אם פקודת ה-`return` משמשת ללא ציון ערך החזרה, אז כברירת מחדל הפונקציה מחזירה את הערך `None`.

*   **אפשרות A** אינה נכונה: הפונקציה אינה מחזירה `0`, אלא אם צוין ערך החזרה מפורש.
*   **אפשרות B** נכונה: Python מחזירה `None` כברירת מחדל אם הפונקציה אינה מחזירה ערך באופן מפורש.
*   **אפשרות C** אינה נכונה: `False` אינו מוחזר כברירת מחדל.
*   **אפשרות D** אינה נכונה: מחרוזת ריקה (`""`) אינה מוחזרת כברירת מחדל.

**כיצד עובד `return` כברירת מחדל:**

1.  אם פונקציה מסיימת את פעולתה כשהיא מגיעה לסוף בלוק הקוד, היא תחזיר `None` באופן מרומז.
2.  אם פונקציה מכילה `return` ללא ערך, הפונקציה תחזיר גם כן `None`.

**דוגמאות:**

```python
# פונקציה ללא return מפורש
def my_function_1() -> None:
    x: int = 10 # סתם אתחול של משתנה לוקאלי
    # כאן מתבצע return None באופן מרומז
res1 = my_function_1()
print(f"החזרת ערך מפונקציה 1: {res1}") # פלט: החזרת ערך מפונקציה 1: None

# פונקציה עם return מפורש
def my_function_2() -> None | int:
  return

res2: None = my_function_2()
print(f"החזרת ערך מפונקציה 2: {res2}") # פלט: החזרת ערך מפונקציה 2: None

def my_function_3(num: int) -> int:
  if num > 0:
      return num * 2
res3: int | None = my_function_3(4)
print(f"החזרת ערך מפונקציה 3: {res3}") # פלט: החזרת ערך מפונקציה 3: 8
res4: int | None = my_function_3(-1)
print(f"החזרת ערך מפונקציה 3: {res4}") # פלט: החזרת ערך מפונקציה 3: None
```
**בתוצאה:**

* הפונקציה `my_function_1` אינה מכילה פקודת `return`, ולכן מחזירה `None`.
* הפונקציה `my_function_2` מכילה פקודת return ללא ערך, ולכן גם היא מחזירה `None`.
* הפונקציה `my_function_3` מחזירה `num*2` אם התנאי `num > 0` מתקיים, אחרת היא מחזירה `None` באופן מרומז.

לפיכך, **אפשרות B** היא הנכונה.
