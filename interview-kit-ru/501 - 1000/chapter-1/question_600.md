**שאלה 604.** מה התפקיד של מילת המפתח `self` במתודות של מחלקות ב-Python? הצג/י דוגמה המדגימה את השימוש ב-`self` לגישה למאפיינים (attributes) של מופע (instance) של מחלקה.

-   A. `self` היא מילת מפתח המשמשת להגדרת מתודות סטטיות של מחלקה.
-   B. `self` היא מילת מפתח המשמשת ליצירת מחלקות חדשות.
-   C. `self` היא מילת מפתח המשמשת כהפניה (reference) למופע הנוכחי (אובייקט) של המחלקה ומשמשת לגישה למאפיינים ולמתודות של מופע זה.
-  D. `self` היא מילת מפתח המשמשת לקבלת הערך של מאפיין של מחלקת אב.

**תשובה נכונה: C**

**הסבר:**

ב-Python, מילת המפתח `self` משמשת במתודות של מחלקה כדי להתייחס למופע (אובייקט) ספציפי של אותה מחלקה. כאשר מתודת מופע נקראת, Python מעבירה אוטומטית את מופע המחלקה כאגרגומנט הראשון, אשר מקובל לקרוא לו `self`.

*   **מאפיינים עיקריים של `self`:**
    *   **הפניה למופע:** `self` מספקת גישה למאפיינים ולמתודות של המופע הנוכחי.
    *   **פרמטר ראשון:** זהו הפרמטר הראשון בכל מתודת מופע (הקונסטרקטור `__init__` הוא גם מתודת מופע), אך לא במתודות סטטיות.
    *  **שימוש בתוך מתודות:** משמש בתוך מתודות לפניה למאפיינים ולמתודות של המופע.
    *   **הבחנה:** מסייע ל-Python להבחין בין מאפייני מחלקה לבין משתנים מקומיים (local variables) של מתודות.
    *    **העברה בעת קריאה**: Python מעבירה אוטומטית הפניה למופע בעת קריאה למתודה דרך מופע (לדוגמה `obj.my_method()`).

*   **שימוש ב-`self` ב-`__init__` (קונסטרקטור):**
    *   במתודת `__init__()`, `self` מפנה למופע *שזה עתה נוצר* של המחלקה.
    *   `self` משמש להקצאת ערכים למאפיינים של המופע החדש.

*   **שימוש ב-`self` במתודות אחרות:**
    *  במתודות אחרות, `self` מפנה למופע של המחלקה *שכבר נוצר*.
    *   `self` משמש לגישה ושינוי מאפיינים, או לקריאה למתודות אחרות של אותו מופע.

**דוגמאות:**
```python
class Shirt:
    def __init__(self, color, size):
        self.color = color # self.color מגדיר את מאפיין המופע color
        self.size = size # self.size מגדיר את מאפיין המופע size

    def display_info(self):
        print(f"This shirt is {self.color} and size is {self.size}") # גישה למאפיינים באמצעות self
        self.another_method() # קריאה למתודה אחרת באמצעות self
    def another_method(self):
       print("Another method was called")

s = Shirt('yellow', 'M')  # יצירת מופע
print(s.color) # צהוב
s.display_info()
# החולצה הזו בצבע צהוב וגודל M
# נקראה מתודה אחרת

class MyClass:
    def __init__(self, x):
        self.x = x

    def get_x(self):
        return self.x

obj1 = MyClass(5)
print(obj1.get_x()) # 5
try:
    MyClass.get_x() # get_x() missing 1 required positional argument: 'self'
except Exception as e:
    print(e)
# get_x() missing 1 required positional argument: 'self'

# הוספת מאפיין חדש באמצעות self במתודה
class MyClass2:
    def __init__(self, x):
        self.x = x

    def set_y(self, y):
        self.y = y

obj2 = MyClass2(10)
obj2.set_y(20)
print(obj2.y) # 20
```

**ניתוח האפשרויות:**
*   **A. `self` היא מילת מפתח המשמשת להגדרת מתודות סטטיות של מחלקה.:** שגוי. מתודות סטטיות אינן מקבלות את הפרמטר `self`.
*   **B. `self` היא מילת מפתח המשמשת ליצירת מחלקות חדשות.:** שגוי.
*  **C. `self` היא מילת מפתח המשמשת כהפניה (reference) למופע הנוכחי (אובייקט) של המחלקה ומשמשת לגישה למאפיינים ולמתודות של מופע זה.:** נכון.
*   **D. `self` היא מילת מפתח המשמשת לקבלת הערך של מאפיין של מחלקת אב.:** שגוי.

**לסיכום:**
*   מילת המפתח `self` מאפשרת למתודות של מחלקה לפנות למאפיינים ולמתודות של המופע שדרכו נקראה המתודה.
*  השימוש ב-`self` הוא חלק חשוב מיצירת מחלקות ב-Python.

לפיכך, התשובה הנכונה היא **C. `self` היא מילת מפתח המשמשת כהפניה (reference) למופע הנוכחי (אובייקט) של המחלקה ומשמשת לגישה למאפיינים ולמתודות של מופע זה.**
