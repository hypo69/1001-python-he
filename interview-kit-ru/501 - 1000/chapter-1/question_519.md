**פרק 1. שאלה 82.** איזו מילת מפתח בשפת Python משמשת ליצירת פונקציה אנונימית?

- A. `func`
- B. `def`
- C. `lambda`
- D. `anon`

**תשובה נכונה: C**

**הסבר:**

ב-Python, פונקציות אנונימיות, הידועות גם כפונקציות למדא, נוצרות באמצעות מילת המפתח `lambda`.

*   **אפשרות A** אינה נכונה: `func` אינה מילת מפתח ב-Python ליצירת פונקציות.
*   **אפשרות B** אינה נכונה: `def` משמשת להגדרת פונקציות רגילות, בעלות שם.
*   **אפשרות C** נכונה: `lambda` משמשת ליצירת פונקציות אנונימיות.
*   **אפשרות D** אינה נכונה: `anon` אינה מילת מפתח ב-Python.

**פונקציות למדא ב-Python:**

1.  **אנונימיות:** לפונקציות למדא אין שם, בשונה מפונקציות המוגדרות באמצעות `def`.
2.  **תמציתיות:** הן נכתבות בשורה אחת בלבד ויכולות להכיל ביטוי יחיד.
3.  **תחום שימוש:** משמשות בדרך כלל לפעולות פשוטות שיש להעביר כארגומנט לפונקציה אחרת (לדוגמה, `map()`, `filter()`, `sorted()`).

**דוגמה:**

```python
# פונקציית למדא להכפלה ב-2
multiply_by_2 = lambda x: x * 2

# קריאה לפונקציית למדא
result: int = multiply_by_2(5)
print(f"Результат: {result}") # פלט: תוצאה: 10
print(f"Тип multiply_by_2: {type(multiply_by_2)}")  # פלט: טיפוס multiply_by_2: <class 'function'>
```

**לסיכום:**
*  פונקציית הלמדא `lambda x: x * 2` נוצרה כדי להכפיל ערך ב-2 והוקצתה למשתנה `multiply_by_2`.
*  טיפוס פונקציית הלמדא הוא `function`.

לפיכך, **אפשרות C** נכונה.
