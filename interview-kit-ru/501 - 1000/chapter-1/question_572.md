### `question_571.md`

**שאלה 571.** מהי תבנית העיצוב הפונקציונלית "Partial application" (יישום חלקי) וכיצד היא משמשת בפייתון? הצג/הציגי דוגמה לשימוש ביישום חלקי ליצירת גרסאות מיוחדות של פונקציות.

-   א. Partial application היא תבנית לטיפול בשגיאות בעת קריאה לפונקציות.
-   ב. Partial application היא תבנית המאפשרת שרשור של פונקציות.
-   ג. Partial application היא תבנית שממירה פונקציה המקבלת מספר ארגומנטים לפונקציה המקבלת ארגומנט אחד.
-   ד. Partial application היא תבנית שמקבעת חלק מהארגומנטים של פונקציה, ומחזירה פונקציה חדשה באריות נמוכה יותר (עם פחות ארגומנטים).

**תשובה נכונה: D**

**הסבר:**

יישום חלקי (Partial application) הוא תבנית פונקציונלית המאפשרת לקבע (לנעול) חלק מהארגומנטים של פונקציה, ובכך ליצור פונקציה חדשה באריות נמוכה יותר (כלומר, פונקציה המקבלת פחות ארגומנטים). הדבר מאפשר יצירת גרסאות מיוחדות יותר של פונקציות, שעשויות להיות נוחות יותר לשימוש בהקשרים מסוימים.

*   **מושגי מפתח ביישום חלקי:**
    *   **קיבוע ארגומנטים:** נעילת ערכים עבור חלק מהארגומנטים של הפונקציה המקורית.
    *   **יצירת פונקציה חדשה:** החזרת פונקציה חדשה, שמצפה רק לארגומנטים הנותרים (שלא קובעו).
    *  **הפחתת אריות:** האריות של הפונקציה החדשה נמוכה יותר מזו של הפונקציה המקורית.
    *   **נוחות ושימוש חוזר:** מאפשר יצירת פונקציות שניתן לעשות בהן שימוש חוזר עם ערכי ארגומנטים שונים.

*   **שימוש בפייתון:**
    *   בפייתון ניתן לממש יישום חלקי באמצעות `functools.partial()`.

**דוגמאות:**

```python
from functools import partial

def power(base, exponent):
    return base ** exponent

# יוצרים פונקציה מיוחדת להעלאה בריבוע
square = partial(power, exponent=2) # מקבעים את ה-exponent
print(square(5))  # 25

# יוצרים פונקציה מיוחדת להעלאה בחזקת שלוש
cube = partial(power, exponent=3)
print(cube(3))  # 27

# דוגמה לקיבוע הארגומנט הראשון
def multiply(a, b):
    return a * b

multiply_by_5 = partial(multiply, 5)
print(multiply_by_5(6))  # 30

# דוגמה לקיבוע שני ארגומנטים
def calculate_discount(price, discount, tax):
   return price * (1 - discount) * (1+ tax)

discount_with_tax = partial(calculate_discount, discount=0.2, tax=0.1)
print(discount_with_tax(100)) # 88
```

**ניתוח הדוגמאות:**

1.  **`power(base, exponent)`:** פונקציה להעלאת `base` בחזקת `exponent`.
2.  **`square = partial(power, exponent=2)`:** יוצרת פונקציה חדשה `square` באמצעות `partial`, כאשר ערך ה-`exponent` מקובע ל-`2`. לפיכך, `square(5)` שקול/ה ל-`power(5, 2)`.
3.   **`cube = partial(power, exponent=3)`:** יוצרת פונקציה חדשה `cube` באמצעות `partial`, כאשר ערך ה-`exponent` מקובע ל-`3`. לפיכך `cube(3)` שקול/ה ל-`power(3, 3)`.
4.  **`multiply(a, b)`:** פונקציה המכפילה את `a` ב-`b`.
5.  **`multiply_by_5 = partial(multiply, 5)`:** יוצרת פונקציה `multiply_by_5` באמצעות `partial`, כאשר ערך ה-`a` מקובע ל-5. לפיכך `multiply_by_5(6)` שקול/ה ל-`multiply(5, 6)`.
6.   **`calculate_discount(price, discount, tax)`**: פונקציה לחישוב המחיר תוך התחשבות בהנחה ובמס.
7.   **`discount_with_tax = partial(calculate_discount, discount=0.2, tax=0.1)`:** יוצרת פונקציה `discount_with_tax` באמצעות `partial`, כאשר הערכים של `discount=0.2` ו-`tax=0.1` מקובעים.

**ניתוח האפשרויות:**
*   **א. Partial application היא תבנית לטיפול בשגיאות בעת קריאה לפונקציות.:** לא נכון.
*  **ב. Partial application היא תבנית המאפשרת שרשור של פונקציות.:** לא נכון. זה מתייחס לתבנית Compose.
*  **ג. Partial application היא תבנית שממירה פונקציה המקבלת מספר ארגומנטים לפונקציה המקבלת ארגומנט אחד.:** לא נכון.
*   **ד. Partial application היא תבנית שמקבעת חלק מהארגומנטים של פונקציה, ומחזירה פונקציה חדשה באריות נמוכה יותר (עם פחות ארגומנטים).:** נכון.

**לסיכום:**
*   יישום חלקי מאפשר יצירת גרסאות מיוחדות יותר של פונקציות, על ידי קיבוע חלק מהארגומנטים.
*  הפונקציה `functools.partial()` מאפשרת לממש יישום חלקי בפייתון.

לפיכך, התשובה הנכונה היא **ד. Partial application היא תבנית שמקבעת חלק מהארגומנטים של פונקציה, ומחזירה פונקציה חדשה באריות נמוכה יותר (עם פחות ארגומנטים).**
