**שאלה 529.** מה תהיה הפלט של קוד Python הבא?

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.__height = height

    def area(self):
        return self.width * self.__height

rect = Rectangle(5, 3)
rect.width = 10
rect.__height = 5
print(rect.area())
```

-   A. 50
-   B. 15
-   C. 30
-   D. 5

**תשובה נכונה: C**

**הסבר:**

קוד זה מדגים עבודה עם מחלקות, תכונות, אנקפסולציה (באמצעות מנגנון עיבוד השם - name mangling) ושיטות ב-Python.

1.  **מחלקה `Rectangle`:**
    *   **`__init__` (בנאי):** מאתחל את התכונות `width` ו-`__height`. התכונה `__height` היא תכונה פרטית הודות לקו תחתון כפול בתחילת השם. Python מיישמת פרטיות באמצעות מנגנון "עיבוד השם" (name mangling).
        *   `self.width` מקבל את הערך של `width` שהועבר לבנאי.
        *   `self.__height` מקבל את הערך של `height` שהועבר לבנאי.
    *   **`area` (שיטה):** מחזירה את שטח המלבן על ידי הכפלת `width` ב-`__height`.

2.  **יצירת אובייקט:**
    *   `rect = Rectangle(5, 3)`: נוצר אובייקט `rect` מהמחלקה `Rectangle` עם ערכים התחלתיים `width = 5` ו-`__height = 3`.
3.  **שינוי תכונות:**
    *   `rect.width = 10`: משנה את התכונה `width` של האובייקט `rect` ל-`10`.
    *   `rect.__height = 5`: מנסה לשנות את התכונה `__height` ל-`5`, אך מכיוון ש-`__height` היא תכונה פרטית, נוצרת תכונה חדשה בשם `__height` באובייקט `rect`. התכונה המקורית `__height` עם השם שעבר עיבוד (`_Rectangle__height`) נשארת ללא שינוי.
4.  **קריאה ל-`area`:**
    *   `print(rect.area())`: קורא לשיטה `area` של האובייקט `rect` ומדפיס את התוצאה. השיטה `area` משתמשת בתכונות המקוריות שאותחלו דרך הבנאי, ובתכונה הפומבית `width` ששונתה.

**כתוצאה מכך:**
*   ב-Python, תכונות עם קו תחתון כפול (`__attr`) עוברות עיבוד שם (name mangling) - Python משנה אוטומטית את שמותיהן כדי למנוע התנגשויות שמות מקריות ומספקת צורה מסוימת של "פרטיות" ברמת המוסכמה.
*   התכונה `__height` לא תשתנה על ידי השורה `rect.__height = 5` בטווח הגלובלי, אלא תיצור תכונה חדשה שלא תשמש בשיטה `area`.
*   השיטה `area` משתמשת ב-`self.width`, ששונה ל-`10`, ובתכונה המקורית, שלא שונתה, `self.__height`, שאותחלה בעת יצירת האובייקט `rect` בבנאי לערך 3.

**חישוב שטח:**

שטח = `rect.width` \* `rect.__height` = 10 \* 3 = 30

**ניתוח האפשרויות:**
*   **A. 50:** שגוי.
*   **B. 15:** שגוי.
*   **C. 30:** נכון.
*   **D. 5:** שגוי.

לפיכך, התשובה הנכונה היא **C. 30**.
