**שאלה 560.** מהם מילונים ב-Python? תאר/י את הפעולות העיקריות שניתן לבצע עם מילונים, והבא/י דוגמת קוד המדגימה פעולות אלו.

-   א.  מילונים ב-Python הם אוספים מסודרים של אלמנטים המאונדקסים באמצעות מספרים שלמים, בדומה לרשימות.
-   ב.  מילונים ב-Python הם מבני נתונים המאחסנים זוגות מפתח-ערך, כאשר המפתחות חייבים להיות אובייקטים ייחודיים ובלתי-ניתנים לשינוי (כגון מחרוזות, מספרים, טאפלים), והערכים יכולים להיות מכל סוג.
-   ג.  מילונים ב-Python הם אוספים המכילים רק אלמנטים ייחודיים, המאוחסנים בסדר יורד.
-   ד.  מילונים הם אובייקטים ב-Python המאחסנים אך ורק מחרוזות ומספקים שיטות לעיצובן.

**תשובה נכונה: ב**

**הסבר:**

מילונים (`dict`) ב-Python הם מבני נתונים גמישים וחזקים המאפשרים לאחסן זוגות מפתח-ערך. הם אוספים לא מסודרים, כאשר כל מפתח מקושר לערך ספציפי. מפתחות המילון חייבים להיות אובייקטים ייחודיים ובלתי-ניתנים לשינוי, כגון מחרוזות, מספרים או טאפלים, ואילו הערכים יכולים להיות מכל סוג נתונים.

*   **מאפיינים עיקריים של מילונים:**
    *   **אחסון זוגות מפתח-ערך:** לכל מפתח מתאים ערך מסוים.
    *   **אי-סדר:** סדר האלמנטים במילון אינו מובטח (החל מגרסה Python 3.7 סדר המפתחות מובטח להתאים לסדר הוספתם).
    *   **ייחודיות המפתחות:** המפתחות חייבים להיות ייחודיים ובלתי-ניתנים לשינוי.
    *   **שינוי (Mutability):** מילונים ניתנים לשינוי (mutable), כלומר ניתן להוסיף, לשנות או למחוק בהם אלמנטים.

*   **פעולות עיקריות עם מילונים:**
    *   **הוספת אלמנט:** השמת ערך למפתח חדש (לדוגמה, `my_dict['new_key'] = 'new_value'`).
    *   **עדכון אלמנט:** השמת ערך חדש למפתח קיים.
    *   **מחיקת אלמנט:** שימוש באופרטור `del` עם ציון המפתח (לדוגמה, `del my_dict['key']`).
    *   **קבלת ערך לפי מפתח:** שימוש במפתח בתוך סוגריים מרובעים (לדוגמה, `value = my_dict['key']`).
        *   שימוש במפתח שאינו קיים יגרום לשגיאה `KeyError`.
        *   לצורך בדיקת קיום מפתח ניתן להשתמש ב-`if "key" in my_dict` או להשתמש בשיטה `.get("key", default)`.
    *   **איטרציה על מפתחות:** שימוש בלולאת `for` על המילון, אשר כברירת מחדל מבצעת איטרציה על המפתחות.
    *   **איטרציה על מפתחות וערכים:** שימוש בשיטה `items()` למעבר על זוגות מפתח-ערך (לדוגמה, `for key, value in my_dict.items():`).
    *   **איטרציה על ערכים:** שימוש בשיטה `values()`, אשר מחזירה ייצוג של ערכי המילון.

**דוגמה (מתוך טקסט השאלה):**

```python
my_dict = {'apple': 1, 'banana': 2, 'cherry': 3}

# הוספת אלמנט
my_dict['date'] = 4

# עדכון אלמנט
my_dict['apple'] = 10

# מחיקת אלמנט
del my_dict['banana']

# קבלת ערך לפי מפתח
print(my_dict['apple'])  # 10

# איטרציה על מפתחות
for key in my_dict:
    print(key) #  "apple", "cherry", "date"
```
**דוגמאות נוספות**
```python
# קבלת ערך לפי מפתח, עם בדיקה על קיום המפתח
my_dict = {'apple': 1, 'cherry': 3}
if 'banana' in my_dict:
   print(my_dict['banana'])
else:
  print("No key 'banana'")

# איטרציה על מפתחות וערכים באמצעות השיטה items()
for key, value in my_dict.items():
  print(f"Key: {key}, Value: {value}")

# איטרציה על ערכים באמצעות השיטה values()
for value in my_dict.values():
   print(f"Value: {value}")
```

*   **ב. מילונים ב-Python - זה מבני נתונים המאחסנים זוגות מפתח-ערך, כאשר המפתחות חייבים להיות אובייקטים ייחודיים ובלתי-ניתנים לשינוי (לדוגמה, מחרוזות, מספרים, טאפלים), והערכים יכולים להיות מכל סוג.:** נכון.
*   **ג. מילונים ב-Python - זה אוספים המכילים רק אלמנטים ייחודיים, המאוחסנים בסדר יורד.:** לא נכון. מילונים אינם מאחסנים אלמנטים ייחודיים בסדר יורד.
*   **ד. מילונים - זה אובייקטים ב-Python, המאחסנים אך ורק מחרוזות ומספקים שיטות לעיצובן.:** לא נכון. מילונים יכולים לאחסן כל סוג נתונים.
