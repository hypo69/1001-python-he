**שאלה 568.** מהו תבנית העיצוב הפונקציונלית "Map" (מיפוי) וכיצד היא משמשת ב-Python? ספק דוגמה לשימוש בתבנית זו לצורך המרה של אלמנטים באוסף.

-   A. תבנית "Map" היא דרך למיון אלמנטים באוסף, באמצעות פונקציה נתונה.
-   B. תבנית "Map" היא דרך לסינון אלמנטים באוסף, באמצעות תנאי נתון.
-   C. תבנית "Map" היא דרך לאיחוד אלמנטים באוסף לערך יחיד, באמצעות פונקציה נתונה.
-   D. תבנית "Map" היא דרך להחלת פונקציה על כל אלמנט באוסף ויצירת אוסף חדש מהתוצאות המתקבלות.

**התשובה הנכונה: D**

**הסבר:**

תבנית "Map" (מיפוי) — היא תבנית פונקציונלית נפוצה, המשמשת להמרת כל אלמנט באוסף (לדוגמה, רשימה, טאפל) לאלמנט חדש באמצעות פונקציה מוגדרת. התוצאה היא אוסף חדש, המכיל את האלמנטים שהומרו.

*   **מושגי יסוד של תבנית "Map":**
    *   **החלת פונקציה:** פונקציית "המיפוי" מוחלת על כל אלמנט באוסף הקלט.
    *   **אוסף חדש:** תוצאות החלת הפונקציה נשמרות באוסף חדש.
    *   **שמירת הסדר:** בדרך כלל נשמר סדר האלמנטים (אם הדבר רלוונטי למבנה הנתונים).
    *   **שימוש עם פונקציות מסדר גבוה:** לעתים קרובות ממומשת באמצעות פונקציות מסדר גבוה, כמו `map()` ב-Python.

*   **אופן פעולת תבנית "Map":**
    1. מקבל אוסף קלט של אלמנטים.
    2. מקבל פונקציית המרה.
    3. מחיל את הפונקציה על כל אלמנט באוסף.
    4. אוסף את תוצאות ההמרה לאוסף חדש.
    5. מחזיר את האוסף החדש עם הערכים המומרים.

**דוגמאות:**

```python
# דוגמה לשימוש ב-map עם פונקציה רגילה
def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(square, numbers))
print(f"הרשימה המקורית: {numbers}")
print(f"רשימת הריבועים: {squared_numbers}") # [1, 4, 9, 16, 25]

# דוגמה לשימוש ב-map עם פונקציית למדא
numbers = [1, 2, 3, 4, 5]
cubed_numbers = list(map(lambda x: x ** 3, numbers))
print(f"הרשימה המקורית: {numbers}")
print(f"רשימת המעוקבים: {cubed_numbers}") # [1, 8, 27, 64, 125]

# דוגמה לשימוש ב-map עם המרת מחרוזות
strings = ["apple", "banana", "cherry"]
upper_strings = list(map(lambda x: x.upper(), strings))
print(f"הרשימה המקורית: {strings}")
print(f"רשימת מחרוזות באותיות גדולות: {upper_strings}") # ['APPLE', 'BANANA', 'CHERRY']

# דוגמה לשימוש ב-map עם מספר אוספים
list1 = [1, 2, 3]
list2 = [4, 5, 6]

sums_list = list(map(lambda x,y: x+y, list1, list2))
print(f"רשימה 1: {list1}")
print(f"רשימה 2: {list2}")
print(f"רשימת הסכומים: {sums_list}") # [5, 7, 9]
```

**ניתוח הדוגמאות:**

1.  **שימוש עם פונקציה רגילה `square`:**
    *   הפונקציה `square(x)` מחשבת את הריבוע של מספר.
    *  הפונקציה `map(square, numbers)` מחילה את הפונקציה `square` על כל אלמנט ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    * מציגה את הרשימה המקורית ואת הרשימה החדשה.
2.  **שימוש עם פונקציית למדא:**
      * פונקציית הלמדא `lambda x: x ** 3` מחשבת את המעוקב של מספר.
    *  הפונקציה `map(lambda x: x ** 3, numbers)` מחילה את הלמדא על כל אלמנט ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    *  מציגה את הרשימה המקורית ואת הרשימה החדשה.
3. **שימוש ב-`map` עם המרת מחרוזות:**
     *  פונקציית הלמדא `lambda x: x.upper()` ממירה מחרוזת לאותיות גדולות (upper case).
     * הפונקציה `map(lambda x: x.upper(), strings)` מחילה את פונקציית הלמדא על כל מחרוזת ברשימה `strings`.
     * מציגה את הרשימה המקורית ואת הרשימה החדשה.
4.  **שימוש ב-`map` עם מספר אוספים:**
     *  פונקציית הלמדא `lambda x,y: x+y` מסכמת שני מספרים.
     *  הפונקציה `map(lambda x,y: x+y, list1, list2)` מחילה את פונקציית הלמדא על כל אלמנט ברשימות `list1` ו-`list2` (לוקחת אלמנטים עם אינדקס זהה).
     *  מציגה את הרשימות המקוריות ואת רשימת הסכומים.

**ניתוח האפשרויות:**
*   **A. תבנית "Map" היא דרך למיון אלמנטים באוסף, באמצעות פונקציה נתונה.:** לא נכון. מיון היא פעולה אחרת.
*  **B. תבנית "Map" היא דרך לסינון אלמנטים באוסף, באמצעות תנאי נתון.:** לא נכון. סינון משמש לבחירת אלמנטים לפי תנאי.
*   **C. תבנית "Map" היא דרך לאיחוד אלמנטים באוסף לערך יחיד, באמצעות פונקציה נתונה.:** לא נכון. זהו תיאור של תבנית Reduce.
*   **D. תבנית "Map" היא דרך להחלת פונקציה על כל אלמנט באוסף ויצירת אוסף חדש מהתוצאות המתקבלות.:** נכון.

**לסיכום:**
*  תבנית "Map" מאפשרת ליצור רשימה חדשה באמצעות המרה של אלמנטים מהרשימה המקורית בעזרת פונקציית המרה.
*  באמצעות שימוש בפונקציה `map` ופונקציות למדא, ניתן ליישם בקלות את תבנית Map.
* הפונקציה `map` מאפשרת להחיל פונקציה על מספר רשימות בו-זמנית.

לפיכך, התשובה הנכונה היא **D. תבנית "Map" היא דרך להחלת פונקציה על כל אלמנט באוסף ויצירת אוסף חדש מהתוצאות המתקבלות.**
