### `question_532.md`

**שאלה 532.** איזה מבין קטעי הקוד המוצגים מדגים דרך אידיומטית (פיית'ונית) יותר ליישם פונקציה על כל איבר ברשימה ומדוע?

**קטע 1:**

```python
def square(num):
    return num**2

x = [1, 2, 3, 4, 5]
s = list(map(square, x))
print(s)
```

**קטע 2:**

```python
def square(num):
    return num**2

x = [1, 2, 3, 4, 5]
s = []
for i in x:
    s.append(square(i))
print(s)
```

-   א. קטע 2, מכיוון שהוא מדגים באופן מפורש יותר את תהליך האיטרציה.
-   ב. קטע 1, מכיוון שהוא משתמש בלולאת `for` באופן מפורש.
-   ג. קטע 1, מכיוון שהוא קצר יותר ומשתמש בפונקציה המובנית `map()`.
-   ד. שני הקטעים טובים ושקולים במידה שווה.

**תשובה נכונה: ג**

**הסבר:**

שאלה זו נועדה לבחון את ההבנה של אופן השימוש בפונקציות מובנות של פיית'ון לכתיבת קוד תמציתית ובעלת הבעה רבה יותר.

*   **קטע 1 (שימוש ב־`map()`):**
    *   משתמש בפונקציה `map()`, המקבלת כארגומנטים פונקציה ואובייקט ניתן לאיטרציה.
    *   `map()` מפעיל את הפונקציה הנתונה על כל איבר באובייקט הניתן לאיטרציה, ומחזיר איטרטור של התוצאות.
    *   התוצאה מומרת לרשימה באמצעות `list()`.
    *   גישה זו דקלרטיבית יותר, מכיוון שהיא מציינת באופן מפורש את התוצאה הרצויה ללא לולאת `for` מפורשת.
*   **קטע 2 (שימוש בלולאת `for`):**
    *   משתמש באופן מפורש בלולאת `for` לאיטרציה על פני הרשימה.
    *   בכל איטרציה קורא לפונקציה `square()` ומוסיף את התוצאה לרשימה חדשה.
    *   זהו סגנון תכנות אימפרטיבי יותר, מכיוון שהוא מתאר באופן מפורש את הצעדים לביצוע המשימה.
    *   יש צורך לאתחל רשימה ריקה ולהוסיף לשם ערכים, מה שהופך את הקוד למילולי יותר.

**ניתוח האפשרויות:**
*   **א. קטע 2, מכיוון שהוא מדגים באופן מפורש יותר את תהליך האיטרציה:** לא נכון. הצגת האיטרציה באופן מפורש במקרה זה הופכת את הקוד למילולי יותר ובעל הבעה פחותה.
*   **ב. קטע 1, מכיוון שהוא משתמש בלולאת `for` באופן מפורש:** לא נכון. קטע 1 אינו משתמש בלולאת for באופן מפורש, אלא משתמש בפונקציה map().
*   **ג. קטע 1, מכיוון שהוא קצר יותר ומשתמש בפונקציה המובנית `map()`:** נכון. `map` היא דרך פיית'ונית יותר ליישם פונקציה על כל איבר ברצף.
*   **ד. שני הקטעים טובים ושקולים במידה שווה:** לא נכון. קטע 1 עדיף יותר מבחינת קריאות ותמציתיות.

**לסיכום:**
*   `map()` היא דרך תמציתית ובעלת הבעה רבה יותר ליישם פונקציה על כל איבר ברצף.
*   `map()` תמציתית יותר מלולאת `for`.
*   קוד פיית'וני הוא קוד העוקב אחר המוסכמות המקובלות בפיית'ון ומשתמש ביכולות המובנות של השפה להשגת הבעה ותמציתיות רבה יותר.
*   הפונקציה המובנית `map` מפשטת את הקוד והופכת אותו לקריא ובעל הבעה רבה יותר.

לפיכך, התשובה הנכונה היא **ג. קטע 1, מכיוון שהוא קצר יותר ומשתמש בפונקציה המובנית `map()`**.
