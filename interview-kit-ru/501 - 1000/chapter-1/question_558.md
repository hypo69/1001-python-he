**שאלה 558.** מהם מילונים ב-Python? תאר את הפעולות העיקריות שניתן לבצע על מילונים, והצג דוגמת קוד המדגימה פעולות אלה.

-   A. מילונים ב-Python הם אוספים מסודרים של אלמנטים, שמאינדקסים באמצעות מספרים שלמים, כמו ברשימות.
-   B. מילונים ב-Python הם מבני נתונים המאחסנים זוגות מפתח-ערך, כאשר המפתחות חייבים להיות אובייקטים ייחודיים ובלתי ניתנים לשינוי (לדוגמה, מחרוזות, מספרים, טאפלים), והערכים יכולים להיות מכל סוג.
-   C. מילונים ב-Python הם אוספים המכילים רק אלמנטים ייחודיים, המאוחסנים בסדר יורד.
-   D. מילונים הם אובייקטים ב-Python המאחסנים אך ורק מחרוזות ומספקים שיטות לעיצובן.

**תשובה נכונה: B**

**הסבר:**

מילונים (`dict`) ב-Python הם מבני נתונים גמישים ועוצמתיים המאפשרים אחסון זוגות מפתח-ערך. הם אוספים בלתי מסודרים, שבהם כל מפתח מקושר לערך ספציפי. מפתחות המילון חייבים להיות אובייקטים ייחודיים ובלתי ניתנים לשינוי, כגון מחרוזות, מספרים או טאפלים, בעוד שהערכים יכולים להיות מכל סוג נתונים.

*   **מאפיינים עיקריים של מילונים:**
    *   **אחסון זוגות מפתח-ערך:** כל מפתח מתאים לערך מסוים.
    *   **חוסר סדר:** סדר האלמנטים במילון אינו מובטח (החל מ-Python 3.7, סדר המפתחות מובטח להתאים לסדר הוספתם).
    *   **ייחודיות המפתחות:** מפתחות חייבים להיות ייחודיים ובלתי ניתנים לשינוי.
    *   **שינוי (Mutability):** מילונים ניתנים לשינוי (mutable), כלומר ניתן להוסיף, לשנות או למחוק מהם אלמנטים.

*   **פעולות עיקריות על מילונים:**
    *   **הוספת אלמנט:** הקצאת ערך למפתח חדש (לדוגמה, `my_dict['new_key'] = 'new_value'`).
    *   **עדכון אלמנט:** הקצאת ערך חדש למפתח קיים.
    *   **מחיקת אלמנט:** שימוש באופרטור `del` עם ציון המפתח (לדוגמה, `del my_dict['key']`).
    *   **קבלת ערך לפי מפתח:** שימוש במפתח בסוגריים מרובעים (לדוגמה, `value = my_dict['key']`).
        *   שימוש במפתח שאינו קיים יגרום לשגיאת `KeyError`.
        *   כדי לבדוק קיום מפתח, ניתן להשתמש ב-`if "key" in my_dict` או בשיטה `.get("key", default)`.
    *   **איטרציה על מפתחות:** שימוש בלולאת `for` על המילון, אשר כברירת מחדל מבצעת איטרציה על המפתחות.
    *   **איטרציה על מפתחות וערכים:** שימוש בשיטה `items()` למעבר על זוגות מפתח-ערך (לדוגמה, `for key, value in my_dict.items():`).
    *   **איטרציה על ערכים:** שימוש בשיטה `values()`, המחזירה ייצוג של ערכי המילון.

**דוגמה (מתוך טקסט השאלה):**

```python
my_dict = {'apple': 1, 'banana': 2, 'cherry': 3}

# הוספת אלמנט
my_dict['date'] = 4

# עדכון אלמנט
my_dict['apple'] = 10

# מחיקת אלמנט
del my_dict['banana']

# קבלת ערך לפי מפתח
print(my_dict['apple'])  # 10

# איטרציה על מפתחות
for key in my_dict:
    print(key) #  "apple", "cherry", "date"
```
**דוגמאות נוספות**
```python
# קבלת ערך לפי מפתח, עם בדיקה לקיום המפתח
my_dict = {'apple': 1, 'cherry': 3}
if 'banana' in my_dict:
   print(my_dict['banana'])
else:
  print("No key 'banana'")

# איטרציה על מפתחות וערכים באמצעות שיטת items()
for key, value in my_dict.items():
  print(f"Key: {key}, Value: {value}")

# איטרציה על ערכים באמצעות שיטת values()
for value in my_dict.values():
   print(f"Value: {value}")
```
