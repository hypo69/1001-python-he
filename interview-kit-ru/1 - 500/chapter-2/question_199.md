### `question_199.md`

**שאלה 199.** מהי הפלט שתציג פיסת הקוד הבאה?

```python
numbers: list[int] = [1, 2, 3, 4, 5]
for i, num in enumerate(numbers):
  if i % 2 == 0:
    continue
  print(num)
```

A. 1 2 3 4 5
B. 2 4
C. 1 3 5
D. 2 3 4 5

**התשובה הנכונה: B**

**הסבר:**

בקוד זה נעשה שימוש בלולאת `for` יחד עם הפונקציה `enumerate()` ובאופרטור תנאי `if` יחד עם האופרטור `continue`.

1. **אתחול:** נוצרת הרשימה `numbers`.
2. **לולאת `for` עם `enumerate()`:**
   * הפונקציה `enumerate()` מחזירה זוגות של אינדקס-ערך עבור כל איבר ברשימה `numbers`.
   * המשתנה `i` מייצג את אינדקס האיבר, והמשתנה `num` מייצג את ערך האיבר.
3. **אופרטור התנאי `if`:**
   * `if i % 2 == 0`: בודק האם האינדקס `i` הוא מספר זוגי.
   * אם האינדקס זוגי, האופרטור `continue` מדלג על הדפסת האיבר הנוכחי ועובר לאיטרציה הבאה של הלולאה.
4. **הדפסה**: אחרת (אם האינדקס אינו זוגי, כלומר אי-זוגי), יודפס ערך `num`.

נסקור את הלולאה שלב אחר שלב:
* i = 0, num = 1. התנאי `i % 2 == 0` נכון, ולכן מדלגים על ההדפסה.
* i = 1, num = 2. התנאי `i % 2 == 0` שגוי, מדפיסים את ערך num = 2.
* i = 2, num = 3. התנאי `i % 2 == 0` נכון, מדלגים על ההדפסה.
* i = 3, num = 4. התנאי `i % 2 == 0` שגוי, מדפיסים את ערך num = 4.
* i = 4, num = 5. התנאי `i % 2 == 0` נכון, מדלגים על ההדפסה.

כעת נסקור את אפשרויות התשובה:

* **אפשרות A** אינה נכונה: לא כל המספרים מודפסים.
* **אפשרות B** נכונה: בלולאה מודפסים הערכים שאינדקסיהם אינם זוגיים (2 ו-4).
* **אפשרות C** אינה נכונה: מודפסים מספרים מאינדקסים שאינם אלה הנכונים.
* **אפשרות D** אינה נכונה: מודפסים ערכים שאינם אלה הנכונים וגם לא בסדר הנכון.

**דוגמה:**

```python
numbers: list[int] = [1, 2, 3, 4, 5]
for i, num in enumerate(numbers):
  if i % 2 == 0:
    continue
  print(num) # פלט:
           # 2
           # 4
```

**לסיכום:**

* מודפסים רק ערכי הרשימה בעלי אינדקס אי-זוגי.

לפיכך, **אפשרות B** היא התשובה הנכונה.
