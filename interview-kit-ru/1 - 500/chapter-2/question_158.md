**שאלה 58.** איזו תכונה של Python מאפשרת לשנות את טיפוס המשתנה במהלך ריצת התוכנית?

A. טיפוּת סטטית.
B. טיפוּת חזקה.
C. טיפוּת דינמית.
D. טיפוּת מפורשת.

**תשובה נכונה: C**

**הסבר:**

Python משתמשת בטיפוּת דינמית, כלומר, טיפוס המשתנה נקבע במהלך ריצת התוכנית ויכול להשתנות על ידי השמת ערך מטיפוס אחר למשתנה.

*   **אפשרות A** אינה נכונה: טיפוּת סטטית דורשת שטיפוסי המשתנים יוצהרו בעת יצירתם ולא ישתנו.
*   **אפשרות B** אינה נכונה: טיפוּת חזקה פירושה שפעולות בין טיפוסים שונים דורשות המרה מפורשת של טיפוסים, אך היא אינה קובעת את מועד הצהרת הטיפוסים.
*   **אפשרות C** נכונה: טיפוּת דינמית מאפשרת לשנות את טיפוס המשתנה במהלך הריצה.
*   **אפשרות D** אינה נכונה: טיפוּת מפורשת פירושה שטיפוסי המשתנים חייבים להיות מצוינים באופן מפורש.

**כיצד פועלת טיפוּת דינמית:**

1.  ב-Python, אין צורך להצהיר על טיפוס המשתנה לפני השימוש בו.
2.  טיפוס המשתנה נקבע אוטומטית בעת השמת ערך.
3.  אותו משתנה יכול להכיל בזמנים שונים נתונים מטיפוסים שונים.

**דוגמה:**

```python
x = 10 # x - מספר שלם
print(f"טיפוס x: {type(x)}") # פלט: <class 'int'>

x = "hello" # x - מחרוזת
print(f"טיפוס x: {type(x)}") # פלט: <class 'str'>

x = [1, 2, 3] # x - רשימה
print(f"טיפוס x: {type(x)}") # פלט: <class 'list'>
```

**כתוצאה מכך:**
*   המשתנה `x` יכול לקבל טיפוסי נתונים שונים במהלך ביצוע התוכנית.

לפיכך, **אפשרות C** היא הנכונה.
