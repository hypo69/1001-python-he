**שאלה 54.** מהו "ארגומנט מילת מפתח" (keyword argument) בעת קריאה לפונקציה בפייתון?

א. ארגומנט שחייב תמיד להיות מצוין בעת קריאה לפונקציה.
ב. ארגומנט המועבר לפונקציה כרצף של אלמנטים.
ג. ארגומנט המועבר לפונקציה עם ציון שם הפרמטר שאליו הוא מתייחס.
ד. ארגומנט שניתן להשמיט בעת קריאה לפונקציה.

**תשובה נכונה: ג**

**הסבר:**

ארגומנט מילת מפתח בפייתון הוא ארגומנט המועבר לפונקציה עם ציון שם הפרמטר שאליו הוא מתייחס. זאת מאפשר להעביר ארגומנטים בכל סדר רצוי, וכן הופך את קריאות הפונקציה לקריאות יותר.

*   **אפשרות א** אינה נכונה: אין חובה לציין ארגומנטים של מילות מפתח בעת קריאה לפונקציה, הם יכולים לקבל ערכי ברירת מחדל.
*   **אפשרות ב** אינה נכונה: ארגומנטים של מילות מפתח אינם מועברים כרצף של אלמנטים.
*   **אפשרות ג** נכונה: ארגומנטים של מילות מפתח מועברים עם ציון שם הפרמטר.
*   **אפשרות ד** אינה נכונה: ניתן להשמיט ארגומנט בעל ערך ברירת מחדל בעת הקריאה.

**אופן השימוש בארגומנטים של מילות מפתח:**

1.  בעת קריאה לפונקציה, לאחר שם הפרמטר מוצב הסימן `=` ומועבר הערך (לדוגמה, `my_function(name="John", age=30)`).
2.  ניתן להעביר ארגומנטים של מילות מפתח בכל סדר.
3.  ניתן לשלב ארגומנטים של מילות מפתח עם ארגומנטים מיקומיים, אך ארגומנטים של מילות מפתח חייבים להופיע לאחר הארגומנטים המיקומיים.
4.  ארגומנטים של מילות מפתח מאפשרים להשמיט ארגומנטים בעלי ערכי ברירת מחדל.

**דוגמה:**

```python
def my_function(name: str, age: int = 18, city: str = "Москва"):
    print(f"שם: {name}, גיל: {age}, עיר: {city}")

# שימוש בארגומנטים של מילות מפתח:
my_function(name="Алиса", age=25) # פלט: שם: Алиса, גיל: 25, עיר: Москва
my_function(age=20, name="Боб") # פלט: שם: בוב, גיל: 20, עיר: Москва
my_function("Чарли", city="Париж") # פלט: שם: צ'ארלי, גיל: 18, עיר: פריז
my_function(name="Даша", city="Лондон", age = 35) # פלט: שם: Даша, גיל: 35, עיר: לונדון
```

**לסיכום:**

*   בעת שימוש בארגומנטים של מילות מפתח ניתן להעביר ערכים בכל סדר רצוי.
*  ניתן להשמיט ארגומנטים אופציונליים.
*  קריאות הקוד משתפרת, שכן ברור לאיזה פרמטר מתייחס הערך שהועבר.

לפיכך, **אפשרות ג** היא התשובה הנכונה.
