**שאלה 177.** בפייתון, כיצד ניתן ליצור מחרוזת רב-שורתית?

א. שימוש בגרשיים בודדים (') עבור כל שורה.
ב. שימוש בגרשיים כפולים (") עבור כל שורה.
ג. שימוש בגרשיים בודדים משולשים או בגרשיים כפולים משולשים (''' או """).
ד. שימוש בסימן לוכסן אחורי (\\) בסוף כל שורה.

**תשובה נכונה: C**

**הסבר:**

בפייתון, מחרוזות רב-שורתיות ניתנות ליצירה באמצעות גרשיים בודדים משולשים (`'''`) או גרשיים כפולים משולשים (`"""`). שיטה זו מאפשרת לכתוב טקסט המשתרע על פני מספר שורות, ללא צורך בשימוש בסימנים מיוחדים עבור מעברי שורה.

*   **אפשרות א** אינה נכונה: גרשיים בודדים אינם תומכים בריבוי שורות באופן זה.
*   **אפשרות ב** אינה נכונה: גרשיים כפולים אינם תומכים בריבוי שורות ללא שימוש בסימנים מיוחדים.
*   **אפשרות ג** נכונה: גרשיים משולשים מאפשרים יצירת מחרוזות רב-שורתיות.
*   **אפשרות ד** אינה נכונה: לוכסן אחורי מאפשר יצירת מעבר שורה בתוך מחרוזת, אך אינו הופך אותה למחרוזת רב-שורתית באופן שבו תלת-גרשיים עושים זאת.

**דוגמאות:**

```python
# שימוש בגרשיים בודדים משולשים
multiline_str1: str = '''Это
многострочная
строка
'''
print(multiline_str1)
# Вывод:
# Это
# многострочная
# строка

# שימוש בגרשיים כפולים משולשים
multiline_str2: str = """Это
тоже
многострочная
строка
"""
print(multiline_str2)
# Вывод:
# Это
# тоже
# многострочная
# строка
```

**לסיכום:**
*   גרשיים משולשים שומרים על עיצוב המחרוזת, כולל מעברי שורה והזחות.

לפיכך, **אפשרות C** היא התשובה הנכונה.
