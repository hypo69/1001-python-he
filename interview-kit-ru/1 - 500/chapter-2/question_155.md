**שאלה 54.** מהו "ארגומנט מפתח" (keyword argument) בעת קריאה לפונקציה ב-Python?

א. ארגומנט שתמיד חייב להיות מצוין בעת קריאה לפונקציה.
ב. ארגומנט המועבר לפונקציה כרצף של איברים.
ג. ארגומנט המועבר לפונקציה עם ציון שם הפרמטר אליו הוא מתייחס.
ד. ארגומנט שניתן להשמיט בעת קריאה לפונקציה.

**תשובה נכונה: C**

**הסבר:**

ארגומנט מפתח ב-Python הוא ארגומנט המועבר לפונקציה תוך ציון שם הפרמטר אליו הוא מתייחס. גישה זו מאפשרת להעביר ארגומנטים בכל סדר רצוי, וכן משפרת את קריאות הקוד של קריאות לפונקציות.

*   **אפשרות A** אינה נכונה: אין הכרח לציין ארגומנטים מפתח בעת קריאה לפונקציה; הם יכולים להכיל ערכי ברירת מחדל.
*   **אפשרות B** אינה נכונה: ארגומנטים מפתח אינם מועברים כרצף של איברים.
*   **אפשרות C** נכונה: ארגומנטים מפתח מועברים עם ציון שם הפרמטר.
*   **אפשרות D** אינה נכונה: ניתן להשמיט ארגומנט בעל ערך ברירת מחדל בעת קריאה לפונקציה.

**אופן השימוש בארגומנטים מפתח:**

1.  בעת קריאה לפונקציה, לאחר שם הפרמטר מוצב סימן `=` ומועבר הערך (לדוגמה, `my_function(name="John", age=30)`).
2.  ניתן להעביר ארגומנטים מפתח בכל סדר רצוי.
3.  ניתן לשלב ארגומנטים מפתח עם ארגומנטים פוזיציונליים, אך ארגומנטים מפתח חייבים להופיע לאחר הארגומנטים הפוזיציונליים.
4.  ארגומנטים מפתח מאפשרים להשמיט ארגומנטים בעלי ערכי ברירת מחדל.

**דוגמה:**

```python
def my_function(name: str, age: int = 18, city: str = "Москва"):
    print(f"Имя: {name}, Возраст: {age}, Город: {city}")

# Использование ключевых аргументов:
my_function(name="Алиса", age=25) # Вывод: Имя: Алиса, Возраст: 25, Город: Москва
my_function(age=20, name="Боб") # Вывод: Имя: Боб, Возраст: 20, Город: Москва
my_function("Чарли", city="Париж") # Вывод: Имя: Чарли, Возраст: 18, Город: Париж
my_function(name="Даша", city="Лондон", age = 35) # Вывод: Имя: Даша, Возраст: 35, Город: Лондон
```

**לסיכום:**

*   בעת שימוש בארגומנטים מפתח, ניתן להעביר ערכים בכל סדר רצוי.
*   ניתן להשמיט ארגומנטים אופציונליים.
*   קריאות הקוד משתפרת, מכיוון שברור לאיזה פרמטר מתייחס הערך שהועבר.

לפיכך, **אפשרות C** היא התשובה הנכונה.
