**שאלה 49.** מה תפקידו של המתודה `__str__()` במחלקה בפייתון?

א. היא משמשת ליצירת מופע חדש של המחלקה.
ב. היא משמשת לייצוג אובייקט בצורת מחרוזת נוחה לקריאה.
ג. היא משמשת להשוואת שני אובייקטים מהמחלקה.
ד. היא משמשת למחיקת מופע של המחלקה מהזיכרון.

**תשובה נכונה: ב**

**הסבר:**

המתודה `__str__()` היא מתודה מיוחדת בפייתון המשמשת להחזרת ייצוג מחרוזתי של אובייקט. ייצוג זה מיועד לקריאה נוחה על ידי המשתמש.

*   אפשרות א' אינה נכונה: יצירת מופע חדש מבוצעת על ידי המתודה `__init__()` (הקונסטרוקטור).
*   אפשרות ב' נכונה: המתודה `__str__()` מיועדת לקבלת ייצוג מחרוזתי קריא של אובייקט.
*   אפשרות ג' אינה נכונה: להשוואת אובייקטים משתמשים במתודות כמו `__eq__()` (שווה).
*   אפשרות ד' אינה נכונה: למחיקת אובייקט משתמשים במתודה `__del__()`.

**כיצד פועלת `__str__()`:**

1.  המתודה `__str__()` נקראת באופן מרומז כאשר אובייקט מועבר לפונקציה `print()` או בשימוש בפונקציה `str()`.
2.  היא חייבת להחזיר ייצוג מחרוזתי של האובייקט.
3.  ייצוג זה צריך להיות נוח לקריאה על ידי אדם.

**דוגמה:**

```python
class Person:
    def __init__(self, name: str, age: int):
        self.name = name
        self.age = age

    def __str__(self):
        return f"Имя: {self.name}, Возраст: {self.age}"

person1: Person = Person("Алиса", 30)
print(person1) # פלט: שם: Алиса, גיל: 30
print(str(person1)) # פלט: שם: Алиса, גיל: 30
```

**כתוצאה מכך:**

*   כאשר נקראות `print(person1)` או `str(person1)`, נקראת באופן מרומז המתודה `__str__()` של האובייקט `person1`, המחזירה מחרוזת מעוצבת המכילה את השם והגיל.

לפיכך, **אפשרות ב'** היא הנכונה.
