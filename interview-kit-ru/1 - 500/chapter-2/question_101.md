### `question_101.md`

**שאלה 101.** איזו מההצהרות הבאות מתארת במדויק את הפונקציונליות של הוראת `if` בתכנות בפייתון?

- א. היא מאפשרת לבצע סדרת הוראות באופן מותנה, על ידי הפעלת בלוק הקוד שלה רק אם התנאי מוערך כערך שאינו אפס או שאינו Null.
- ב. היא עוברת בלוק קוד בלולאה כל עוד התנאי נכון.
- ג. היא מסיימת לולאה או מדלגת על איטרציה בעת עמידה בתנאים מסוימים.
- ד. היא מבצעת בלוק קוד מספר פעמים, לרוב באמצעות מונה.

**תשובה נכונה: א**

**הסבר:**

אופרטור ה-`if` בפייתון משמש לביצוע מותנה של בלוק קוד, בהתבסס על אמיתות תנאי נתון.

*   **אפשרות א** נכונה: בלוק הקוד שבתוך `if` מבוצע אם התנאי נכון (ובפייתון פירושו של דבר שהתנאי אינו `0`, `None`, `False`, או קונטיינר ריק).

*   **אפשרות ב** אינה נכונה: לביצוע קוד באופן מחזורי משתמשים ב-`while` או ב-`for`.
*   **אפשרות ג** אינה נכונה: לדילוג על איטרציה או סיום לולאה משתמשים ב-`continue` וב-`break`.
*  **אפשרות ד** אינה נכונה: לביצוע בלוק קוד מספר פעמים משתמשים בלולאות.

**כיצד פועל `if`:**

1.  ראשית, מוערך התנאי המצוין לאחר מילת המפתח `if`.
2.  אם התנאי נכון (`True`), אזי מבוצע בלוק הקוד הבא לאחר אופרטור ה-`if`.
3.  אם התנאי שגוי (`False`), אזי בלוק הקוד אינו מבוצע.

**דוגמה:**

```python
x: int = 10

if x > 5:
  print("x יותר גדול מ-5") # פלט: x יותר גדול מ-5

if x > 100:
   print("x יותר גדול מ-100") # לא יודפס דבר, מכיוון ש-x > 100 == False

y: int = 0
if y:
  print("y הוא ערך אמיתי") # לא יודפס דבר, מכיוון ש-0 הוא ערך שגוי
if 1:
  print("1 הוא ערך אמיתי") # פלט: 1 הוא ערך אמיתי
```
**כתוצאה מכך:**

*   התנאי הראשון `x > 5` נכון, ולכן מודפס "x יותר גדול מ-5".
*  התנאי השני `x > 100` אינו נכון, ולכן קטע הקוד אינו מודפס.
* `0` הוא ערך שגוי, ולכן `if y:` אינו מבוצע.
* `1` הוא ערך אמיתי, ולכן `if 1:` מבוצע.

לפיכך, **אפשרות א** היא הנכונה.
