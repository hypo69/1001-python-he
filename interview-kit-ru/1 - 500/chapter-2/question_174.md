### `question_174.md`

**שאלה 174.** לשם מה משמשת מילת המפתח `as` בפייתון באופרטור `import`?

א. ליצירת כינוי למודול או לפונקציה המיובאים.
ב. לייבוא כל המשתנים ממודול, למעט אלו שצוינו.
ג. לייבוא מודול רק כאשר מתקיים תנאי.
ד. לקריאה לפונקציה מתוך מודול.

**תשובה נכונה: א**

**הסבר:**

מילת המפתח `as` באופרטור `import` משמשת ליצירת כינוי (alias) למודול או לפונקציה המיובאים. הדבר מאפשר שימוש בשם קצר ונוח יותר לפניה ליסודות המיובאים.

*   **אפשרות א** נכונה: מילת המפתח `as` יוצרת כינוי.
*   **אפשרות ב** אינה נכונה: להחרגת ייבוא נעשה שימוש בייבוא עם רשימה מפורשת של יסודות לייבוא, ולא ב־`as`.
*   **אפשרות ג** אינה נכונה: לייבוא מותנה לרוב נעשה שימוש בבלוקי `if` ובמבנה `__import__()`.
*   **אפשרות ד** אינה נכונה: מילת המפתח `as` משפיעה על השם אך אינה קוראת לפונקציה מתוך המודול.

**כיצד `as` פועלת:**

1.  תחביר השימוש ב־`as`: `import module_name as alias_name` או `from module_name import function_name as alias_name`.
2.  המודול או הפונקציה המיובאים מקבלים שם חדש (`alias_name`), שניתן להשתמש בו בקוד.
3.  הדבר מסייע להימנע מהתנגשויות שמות, מפשט את הקוד ומשפר את קריאותו.

**דוגמאות:**

```python
# יצירת כינוי למודול
import numpy as np
print(np.array([1, 2, 3])) # פלט: [1 2 3]

# יצירת כינוי לפונקציה
from math import sqrt as square_root
print(square_root(16)) # פלט: 4.0
```

**סיכום:**

*   בדוגמה הראשונה, המודול `numpy` מיובא כ־`np`.
*   בדוגמה השנייה, הפונקציה `sqrt` מהמודול `math` מיובאת כ־`square_root`.

לפיכך, **אפשרות א** היא התשובה הנכונה.
