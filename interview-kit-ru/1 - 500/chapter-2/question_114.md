### `question_114.md`

**שאלה 114.** כיצד פועל תנאי ה-`else` בתוך לולאת `while` בפייתון?

- א. הוא מבצע את בלוק הקוד פעם אחת כאשר תנאי לולאת ה-`while` אינו עוד אמיתי, וזאת בתנאי שהלולאה לא נקטעה על ידי `break`.
- ב. הוא מבוצע בכל פעם שתנאי לולאת ה-`while` מוערך כ-`False`.
- ג. הוא מספק תנאי חלופי שיכול להאריך את ביצוע לולאת ה-`while` מעבר לתנאי המקורי שלה.
- ד. הוא גורם לסיום מיידי של לולאת ה-`while` כאשר התנאי שלה הופך להיות שקרי.

**תשובה נכונה: א**

**הסבר:**

בפייתון, לולאת `while` יכולה לכלול בלוק `else` אופציונלי, המבוצע רק לאחר שהלולאה מסתיימת באופן טבעי, כלומר כאשר התנאי שלה הופך שקרי, וזאת בתנאי שהלולאה לא נקטעה באמצעות אופרטור `break`.

*   **אפשרות א** נכונה: בלוק ה-`else` מבוצע פעם אחת לאחר שלולאת ה-`while` מסתיימת באופן טבעי (לא באמצעות `break`).
*   **אפשרות ב** אינה נכונה: בלוק ה-`else` מבוצע פעם אחת בלבד כאשר תנאי ה-`while` הופך ל-`False`, ולא בכל פעם שהוא שקרי.
*   **אפשרות ג** אינה נכונה: `else` אינו מרחיב את תנאי ה-`while`, אלא פועל כפעולת סיום (post-processing).
*   **אפשרות ד** אינה נכונה: הלולאה מסתיימת כאשר התנאי הוא `False`, וה-`else` מבוצע לאחר מכן. `else` אינו גורם לסיום.

**כיצד פועל `while...else`:**

1.  לולאת ה-`while` מבוצעת כל עוד התנאי שלה אמיתי.
2.  אם תנאי ה-`while` הופך שקרי, והלולאה לא נקטעה על ידי אופרטור `break`, אזי מבוצע בלוק הקוד המגיע לאחר ה-`else`.
3.  לעומת זאת, אם הלולאה נקטעת באמצעות `break`, בלוק הקוד של ה-`else` אינו מבוצע.

**שימושים אופייניים ב-`else` בלולאות `while`:**

*   **חיפוש פריטים:** ביצוע קוד ספציפי אם פריט לא נמצא במהלך חיפוש בלולאת `while`.
*   **ביצוע קוד לאחר סיום מוצלח של הלולאה:** ביצוע קוד שאמור להתבצע רק כאשר הלולאה מסתיימת באופן טבעי, ללא קטיעה מוקדמת.

**דוגמה:**

```python
# דוגמה עם while ו-else, כאשר הלולאה מסתיימת ללא break
x: int = 0
while x < 5:
    x += 1
else:
    print(f"לולאת while הסתיימה, x = {x}")  # פלט: לולאת while הסתיימה, x = 5

# דוגמה עם while ו-break
x: int = 0
while x < 10:
    x += 1
    if x == 3:
      break
else:
    print(f"לולאת while הסתיימה, x = {x}") # פלט לא יודפס
```
**תוצאה:**

* בדוגמה הראשונה, לולאת ה-`while` הסתיימה כאשר `x` שווה ל-`5`, ולאחר מכן מבוצע בלוק ה-`else`.
* בדוגמה השנייה, כאשר `x` הפך שווה ל-3, `break` הופעל וקטע את הלולאה, ובלוק ה-`else` אינו מבוצע.

לפיכך, **אפשרות א** היא הנכונה.
