**שאלה 114.** כיצד פועל תנאי `else` בתוך לולאת `while` ב-Python?

- א. הוא מבצע את בלוק הקוד פעם אחת כאשר תנאי לולאת ה-`while` אינו עוד `True`, ובתנאי שהלולאה לא נקטעה על ידי `break`.
- ב. הוא מבוצע בכל פעם שתנאי לולאת ה-`while` מוערך כ-`False`.
- ג. הוא מספק תנאי חלופי שיכול להאריך את ביצוע לולאת ה-`while` מעבר לתנאי המקורי שלה.
- ד. הוא גורם לסיום מיידי של לולאת ה-`while` כאשר התנאי שלה הופך ל-`False`.

**תשובה נכונה: א**

**הסבר:**

ב-Python, לולאת `while` יכולה לכלול בלוק `else` אופציונלי, המבוצע רק לאחר שהלולאה מסתיימת באופן טבעי, כלומר כאשר התנאי שלה הופך ל-`False`, ובתנאי שהלולאה לא נקטעה על ידי אופרטור `break`.

*   **אפשרות א** נכונה: בלוק ה-`else` מבוצע פעם אחת לאחר שלולאת ה-`while` מסתיימת באופן טבעי (לא על ידי `break`).
*   **אפשרות ב** אינה נכונה: בלוק ה-`else` מבוצע רק פעם אחת, כאשר תנאי ה-`while` הופך ל-`False`, ולא בכל פעם שהוא `False`.
*  **אפשרות ג** אינה נכונה: ה-`else` אינו מרחיב את תנאי ה-`while`, אלא פועל כעיבוד לאחר סיום.
*   **אפשרות ד** אינה נכונה: הלולאה מסתיימת כאשר התנאי הופך ל-`False`, וה-`else` מבוצע לאחר מכן. ה-`else` אינו גורם לסיום.

**אופן הפעולה של `while...else`:**

1.  לולאת ה-`while` מבוצעת כל עוד התנאי שלה `True`.
2.  אם תנאי ה-`while` הופך ל-`False`, והלולאה לא נקטעה על ידי אופרטור `break`, אזי מבוצע בלוק הקוד הבא אחרי ה-`else`.
3.  אם הלולאה נקטעת באמצעות `break`, אזי בלוק הקוד של ה-`else` אינו מבוצע.

**מקרים אופייניים לשימוש ב-`else` בלולאות `while`:**

*   **חיפוש אלמנטים:** ביצוע קוד מסוים, אם אלמנט לא נמצא במהלך חיפוש בלולאת `while`.
*   **ביצוע קוד לאחר השלמה מוצלחת של הלולאה:** ביצוע קוד שאמור להתבצע רק בעת סיום טבעי של הלולאה, ללא הפסקה מוקדמת.

**דוגמה:**

```python
# דוגמה ל-while ו-else, כאשר הלולאה מסתיימת ללא break
x: int = 0
while x < 5:
    x += 1
else:
    print(f"לולאת while הסתיימה, x = {x}")  # פלט: לולאת while הסתיימה, x = 5

# דוגמה ל-while ו-break
x: int = 0
while x < 10:
    x += 1
    if x == 3:
      break
else:
    print(f"לולאת while הסתיימה, x = {x}") # הפלט לא יוצג
```
**בתוצאה:**

*   בדוגמה הראשונה, לולאת ה-`while` הסתיימה כאשר `x` שווה ל-`5`, ולאחר מכן מבוצע בלוק ה-`else`.
*   בדוגמה השנייה, כאשר `x` הפך שווה ל-`3`, הופעל ה-`break` שקטע את הלולאה, ובלוק ה-`else` לא בוצע.

לפיכך, **אפשרות א** היא הנכונה.
