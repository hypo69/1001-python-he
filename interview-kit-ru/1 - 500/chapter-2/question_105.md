**שאלה 105.** מה מבצע האופרטור `continue` כאשר הוא מופעל בתוך לולאה פנימית?

- א. גורם לתוכנית לצאת מהלולאה הפנימית ולחזור לשורה הראשונה של הלולאה החיצונית.
- ב. מדלג על יתרת הקוד בתוך הלולאה הפנימית עבור האיטרציה הנוכחית וממשיך לאיטרציה הבאה של הלולאה הפנימית.
- ג. מסיים מיד את שתי הלולאות ועובר לשורה הבאה של הקוד מחוץ ללולאות המקוננות.
- ד. מאפס את התנאי של הלולאה הפנימית ומפעיל את הלולאה מהתחלה.

**תשובה נכונה: ב**

**הסבר:**

האופרטור `continue` בפייתון משמש בתוך לולאות כדי לדלג על האיטרציה הנוכחית ולעבור לתחילת האיטרציה הבאה. במקרה של לולאות מקוננות, האופרטור `continue` משפיע אך ורק על הלולאה שבה הוא נמצא באופן ישיר.

*   **אפשרות א'** אינה נכונה: האופרטור `continue` אינו יוצא מהלולאה הפנימית ואינו חוזר ללולאה החיצונית.
*   **אפשרות ב'** נכונה: האופרטור `continue` מדלג על יתרת הקוד *באיטרציה הנוכחית* של הלולאה הפנימית ועובר לאיטרציה הבאה של לולאה פנימית זו.
*   **אפשרות ג'** אינה נכונה: האופרטור `continue` אינו מסיים את שתי הלולאות. לסיום לולאה משתמשים ב-`break`.
*   **אפשרות ד'** אינה נכונה: האופרטור `continue` אינו מפעיל מחדש את הלולאה, אלא עובר לאיטרציה הבאה.

**כיצד פועל `continue` בלולאות מקוננות:**

1.  האופרטור `continue`, הנמצא בלולאה הפנימית, מדלג על הקוד הנותר באיטרציה הנוכחית של הלולאה הפנימית.
2.  השליטה עוברת לתחילת *הלולאה הפנימית*  כדי לעבור לאיטרציה הבאה.
3.  הלולאה החיצונית ממשיכה את פעולתה במקביל.

**דוגמה:**

```python
for i in range(3):
    print(f"Внешний цикл: i = {i}")
    for j in range(3):
        if j == 1:
            continue  # מדלג על יתרת הקוד בלולאה הפנימית אם j שווה 1
        print(f"  Внутренний цикл: j = {j}")
    print("После внутреннего цикла")
# פלט:
# Внешний цикл: i = 0
#   Внутренний цикл: j = 0
# После внутреннего цикла
# Внешний цикл: i = 1
#   Внутренний цикл: j = 0
# После внутреннего цикла
# Внешний цикл: i = 2
#   Внутренний цикл: j = 0
# После внутреннего цикла
```
**כתוצאה מכך:**

*   הלולאה החיצונית מבוצעת שלוש פעמים.
*   הלולאה הפנימית מבוצעת שלוש פעמים עבור כל ערך של הלולאה החיצונית, למעט המקרים שבהם הערך של `j` שווה ל-`1`, ואז הקוד הנותר של הלולאה הפנימית מדולג.

לפיכך, **אפשרות ב'** היא הנכונה.
