**שאלה 105.** מה עושה האופרטור `continue` בעת ביצוע בלולאה פנימית?

- A. גורם לתוכנית לצאת מהלולאה הפנימית ולחזור לשורה הראשונה של הלולאה החיצונית.
- B. מדלג על שאר קוד הלולאה הפנימית עבור האיטרציה הנוכחית וממשיך לאיטרציה הבאה של הלולאה הפנימית.
- C. מסיים מיד את שתי הלולאות וממשיך לשורת הקוד הבאה מחוץ ללולאות המקוננות.
- D. מאפס את תנאי הלולאה הפנימית ומפעיל את הלולאה מההתחלה.

**תשובה נכונה: B**

**הסבר:**

האופרטור `continue` בפייתון משמש בתוך לולאות כדי לדלג על האיטרציה הנוכחית ולעבור לתחילת האיטרציה הבאה. במקרה של לולאות מקוננות, האופרטור `continue` משפיע רק על הלולאה שבה הוא ממוקם ישירות.

*   **אפשרות A** אינה נכונה: האופרטור `continue` אינו יוצא מהלולאה הפנימית ואינו חוזר ללולאה החיצונית.
*   **אפשרות B** נכונה: האופרטור `continue` מדלג על הקוד הנותר *באיטרציה הנוכחית* של הלולאה הפנימית וממשיך לאיטרציה הבאה של לולאה פנימית זו.
*   **אפשרות C** אינה נכונה: האופרטור `continue` אינו מסיים את שתי הלולאות. לסיום לולאה משתמשים ב-`break`.
*   **אפשרות D** אינה נכונה: האופרטור `continue` אינו מפעיל מחדש את הלולאה, אלא עובר לאיטרציה הבאה.

**כיצד פועל `continue` בלולאות מקוננות:**

1.  האופרטור `continue`, בהיותו בלולאה פנימית, מדלג על הקוד הנותר באיטרציה הנוכחית של הלולאה הפנימית.
2.  השליטה עוברת לתחילת *הלולאה הפנימית* למעבר לאיטרציה הבאה.
3.  הלולאה החיצונית ממשיכה בביצוע.

**דוגמה:**

```python
for i in range(3):
    print(f"Внешний цикл: i = {i}") # לולאה חיצונית: i = {i}
    for j in range(3):
        if j == 1:
            continue  # מדלגים על הקוד הנותר בלולאה הפנימית אם j==1
        print(f"  Внутренний цикл: j = {j}") # לולאה פנימית: j = {j}
    print("После внутреннего цикла") # אחרי הלולאה הפנימית
# פלט:
# Внешний цикл: i = 0
#   Внутренний цикл: j = 0
# После внутреннего цикла
# Внешний цикл: i = 1
#   Внутренний цикл: j = 0
# После внутреннего цикла
# Внешний цикл: i = 2
#   Внутренний цикл: j = 0
# После внутреннего цикла
```
**תוצאה:**

*   הלולאה החיצונית מתבצעת שלוש פעמים.
*   הלולאה הפנימית מתבצעת שלוש פעמים עבור כל ערך של הלולאה החיצונית, למעט המקרים שבהם הערך של `j` שווה ל-`1`, ואז הקוד הנותר של הלולאה הפנימית מדולג.

לפיכך, **אפשרות B** היא הנכונה.
