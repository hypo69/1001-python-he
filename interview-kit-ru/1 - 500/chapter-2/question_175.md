### `question_175.md`

**שאלה 175.** בפייתון, מהי "שיטת מחלקה" (class method), וכיצד היא שונה מ"שיטה סטטית" (static method)?

A. שיטת מחלקה יכולה לשנות את מצב המופע, בעוד ששיטה סטטית אינה יכולה.
B. שיטת מחלקה ניתנת לקריאה רק דרך מופע של המחלקה, שיטה סטטית ניתנת לקריאה דרך המחלקה.
C. שיטת מחלקה מקבלת כארגומנט ראשון את המחלקה, בעוד ששיטה סטטית אינה מקבלת ארגומנט כלשהו הקשור למחלקה או למופע.
D. שיטת מחלקה אינה קשורה לאף מחלקה, בעוד ששיטה סטטית שייכת רק למופע של המחלקה.

**תשובה נכונה: C**

**הסבר:**

שיטות מחלקה (`@classmethod`) ושיטות סטטיות (`@staticmethod`) הן סוגים מיוחדים של שיטות בתוך מחלקה, אך הן שונות באופן הקריאה שלהן ובהתאם לארגומנטים שהן מקבלות.

*   **אפשרות A** אינה נכונה: שיטת מחלקה יכולה לשנות את מצב המחלקה, אך לא את מצב המופע, ושיטה סטטית אינה משנה את מצב המחלקה וגם לא את מצב המופע.
*   **אפשרות B** אינה נכונה: שיטת מחלקה נקראת דרך המחלקה, ולא דרך מופע. שיטה סטטית גם כן נקראת דרך המחלקה.
*   **אפשרות C** נכונה: שיטת מחלקה מקבלת את `cls` (הפניה למחלקה) כארגומנט ראשון, בעוד ששיטה סטטית אינה מקבלת לא את `self` (הפניה למופע) ולא את `cls`.
*   **אפשרות D** אינה נכונה: שיטת מחלקה קשורה למחלקה, ושיטה סטטית קשורה מבחינה לוגית למחלקה.

**כיצד פועלות שיטות מחלקה ושיטות סטטיות:**

1.  **שיטת מחלקה (`@classmethod`)**:
    *   משתמשת בדקורטור `@classmethod`.
    *   מקבלת כארגומנט ראשון את המחלקה ששימשה לקריאה לשיטה (מקובל לקרוא לו `cls`).
    *   משמשת ליצירת בנאים חלופיים או לשינוי תכונות המחלקה.
    *   ניתנת לקריאה גם דרך המחלקה וגם דרך מופע של המחלקה.

2.  **שיטה סטטית (`@staticmethod`)**:
    *   משתמשת בדקורטור `@staticmethod`.
    *   אינה מקבלת לא את `self` ולא את `cls` כארגומנט.
    *   משמשת עבור פונקציות עזר שאינן עובדות עם מופע של המחלקה או עם המחלקה עצמה.
    *   ניתנת לקריאה גם דרך המחלקה וגם דרך מופע של המחלקה.

**דוגמה:**

```python
class MyClass:
    count: int = 0 # משתנה מחלקה
    def __init__(self, name: str):
        self.name = name  # תכונת מופע

    @classmethod
    def increase_count(cls): # משתמש ב-cls כדי לשנות את משתנה המחלקה
        cls.count += 1
        print(f"מספר המופעים: {cls.count}")

    @staticmethod
    def say_hello(name: str): # שיטה סטטית ללא גישה למחלקה או למופע
        print(f"שלום {name}!")


obj1: MyClass = MyClass("Alice")
MyClass.increase_count() # פלט: מספר המופעים: 1
obj1.increase_count() # פלט: מספר המופעים: 2
MyClass.say_hello("Bob") # פלט: שלום Bob!
obj1.say_hello("Charlie") # פלט: שלום Charlie!
```

**כתוצאה מכך:**
* `increase_count` היא שיטת מחלקה, יש לה גישה לתכונה `count` של המחלקה.
* `say_hello` היא שיטה סטטית, אין לה גישה לא למופע ולא למחלקה.
לפיכך, **אפשרות C** היא הנכונה.
