**שאלה 112.** ב-Python, מה המשמעות של קינון מבני בקרה זה בתוך זה?

- א. הצבת מבנה בקרה בתוך מבנה אחר, למשל משפט `if` בתוך לולאת `for`, כדי להרחיב תהליכי קבלת החלטות בהתבסס על תנאים שונים.
- ב. שימוש במבנה בקרה יחיד כדי לשלוט בכל דרישות הלולאות והתנאים של התוכנית.
- ג. ארגון מספר אופרטורי בקרה במקביל כדי להבטיח את ביצועם בסדר מוגדר מראש.
- ד. הפרדת אופרטורי בקרה לפונקציות או מודולים נפרדים לשיפור קריאות הקוד.

**תשובה נכונה: א**

**הסבר:**

קינון מבני בקרה (nested control structures) ב-Python פירושו הצבת מבנה בקרה אחד (למשל, לולאת `for`, לולאת `while`, משפט `if`, משפט `try...except`) בתוך מבנה בקרה אחר. הדבר מאפשר יצירת לוגיקה מורכבת יותר לשליטה בזרימת ביצוע התוכנית.

*   **אפשרות א** נכונה: קינון מבנה בקרה בתוך אחר מאפשר לסבך את לוגיקת התוכנית, לדוגמה, כדי לעבד נתונים תוך התחשבות בתנאים שונים, על ידי יצירת לולאות מקוננות ומשפטי תנאי מקוננים.
*   **אפשרות ב** אינה נכונה: להפך, קינון משמעו שימוש במספר מבני בקרה עבור חלקים שונים בלוגיקת התוכנית.
*   **אפשרות ג** אינה נכונה: מבנים מקוננים מבוצעים ברצף, לא במקביל.
*   **אפשרות ד** אינה נכונה: הוצאת מבנים לפונקציות או מודולים נפרדים משמשת למודולריות הקוד, ולא לקינון.

**דוגמאות טיפוסיות למבנים מקוננים:**

1.  **לולאות מקוננות:** לולאת `for` בתוך לולאת `for` או לולאת `while` בתוך לולאת `for` וכו'.
2.  **משפטי תנאי מקוננים:** משפט `if` בתוך משפט `if` אחר או בתוך בלוק `elif`, או `else`.
3.  **לולאה בתוך משפט תנאי**: לולאת `for` או `while` בתוך בלוק `if`, או `else`.
4.  **חריגות**: `try-except` בתוך לולאות או משפטי תנאי.

**דוגמה:**
```python
# דוגמה ללולאה מקוננת
for i in range(2):
    for j in range(2):
        print(f"External loop (i={i}), Inner loop (j={j})")
# פלט:
# External loop (i=0), Inner loop (j=0)
# External loop (i=0), Inner loop (j=1)
# External loop (i=1), Inner loop (j=0)
# External loop (i=1), Inner loop (j=1)

# דוגמה ל-if מקונן
x: int = 10
if x > 5:
    if x > 8:
        print("x is greater than 8 and greater than 5") # פלט: x is greater than 8 and greater than 5
    else:
       print("x is greater than 5, but not greater than 8")
```
**בתוצאה:**
* בדוגמה של לולאות מקוננות, הלולאה הפנימית מבוצעת במלואה עבור כל ערך של הלולאה החיצונית.
* בדוגמה של בלוקי `if` מקוננים, התנאי הפנימי נבדק רק לאחר שהתנאי ב-`if` החיצוני מתקיים.

לפיכך, **אפשרות א** נכונה, מכיוון שהיא מתארת במדויק את השימוש במבני בקרה מקוננים.
