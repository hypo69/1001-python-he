### `question_112.md`

**שאלה 112.** בפייתון, מה הכוונה בקינון מבני בקרה אחד בתוך השני?

- א. הצבת מבנה בקרה אחד בתוך אחר, לדוגמה פסוקית `if` בתוך לולאת `for`, על מנת להרחיב את תהליכי קבלת ההחלטות בהתבסס על תנאים שונים.
- ב. שימוש במבנה בקרה יחיד לניהול כל דרישות הלולאות והתנאים של התוכנית.
- ג. מבנוּת מספר פסוקיות בקרה במקביל על מנת להבטיח את ביצוען בסדר קבוע מראש.
- ד. הפרדת פסוקיות בקרה לפונקציות או מודולים שונים לשם הגברת קריאות הקוד.

**תשובה נכונה: א**

**הסבר:**

קינון מבני בקרה (nested control structures) בפייתון פירושו הצבת מבנה בקרה אחד (לדוגמה, לולאת `for`, לולאת `while`, פסוקית `if`, פסוקית `try...except`) בתוך מבנה בקרה אחר. זה מאפשר יצירת לוגיקת בקרה מורכבת יותר של זרימת ביצוע התוכנית.

*   **אפשרות א' נכונה:** קינון מבנה בקרה אחד בתוך אחר מאפשר לסבך את לוגיקת התוכנית, לדוגמה, כדי לעבד נתונים תוך התחשבות בתנאים שונים, על ידי יצירת לולאות מקוננות ופסוקיות תנאי מקוננות.
*   **אפשרות ב' אינה נכונה:** להיפך, קינון פירושו שימוש במספר מבני בקרה לחלקים שונים של לוגיקת התוכנית.
*   **אפשרות ג' אינה נכונה:** מבנים מקוננים מבוצעים בסדר רציף (sequential), ולא במקביל (parallel).
*   **אפשרות ד' אינה נכונה:** הוצאת מבנים לפונקציות או מודולים נפרדים משרתת את מודולריות הקוד, ולא את הקינון.

**דוגמאות אופייניות למבנים מקוננים:**

1.  **לולאות מקוננות:** לולאת `for` בתוך לולאת `for`, או לולאת `while` בתוך לולאת `for`, וכן הלאה.
2.  **פסוקיות תנאי מקוננות:** פסוקית `if` בתוך פסוקית `if` אחרת או בתוך בלוק `elif` או `else`.
3.  **לולאה בתוך פסוקית תנאי:** לולאת `for` או `while` בתוך בלוק `if` או `else`.
4.  **חריגות:** `try-except` בתוך לולאות או פסוקיות תנאי.

**דוגמה:**
```python
# דוגמה ללולאה מקוננת
for i in range(2):
    for j in range(2):
        print(f"לולאה חיצונית (i={i}), לולאה פנימית (j={j})")
# פלט:
# לולאה חיצונית (i=0), לולאה פנימית (j=0)
# לולאה חיצונית (i=0), לולאה פנימית (j=1)
# לולאה חיצונית (i=1), לולאה פנימית (j=0)
# לולאה חיצונית (i=1), לולאה פנימית (j=1)

# דוגמה לפסוקית if מקוננת
x: int = 10
if x > 5:
    if x > 8:
        print("x גדול מ-8 וגדול מ-5") # פלט: x גדול מ-8 וגדול מ-5
    else:
       print("x גדול מ-5, אך לא גדול מ-8")
```
**תוצאה:**
*   בדוגמה של לולאות מקוננות, הלולאה הפנימית מבוצעת במלואה עבור כל ערך של הלולאה החיצונית.
*   בדוגמה של בלוקי `if` מקוננים, התנאי הפנימי נבדק רק לאחר שהתנאי בבלוק ה-`if` החיצוני מתקיים.

לפיכך, **אפשרות א'** נכונה, מכיוון שהיא מתארת במדויק את השימוש במבני בקרה מקוננים.
