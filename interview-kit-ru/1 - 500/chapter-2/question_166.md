### `question_166.md`

**שאלה 66.** מה קורה כאשר מנסים לגשת למפתח שאינו קיים במילון פייתון?

א. מוחזר `None`.
ב. מוחזרת מחרוזת ריקה.
ג. נזרקת חריגה מסוג `KeyError`.
ד. נוצר אלמנט חדש עם המפתח שצוין.

**התשובה הנכונה: C**

**הסבר:**

בפייתון, אם מנסים לגשת למפתח שאינו קיים במילון, נזרקת חריגה מסוג `KeyError`.

*   **אפשרות א** אינה נכונה: `None` מוחזר על ידי המתודה `get()`, ולא על ידי אופרטור גישה למפתח `[]`.
*   **אפשרות ב** אינה נכונה: מילון אינו מחזיר מחרוזת ריקה בעת גישה למפתח שאינו קיים.
*   **אפשרות ג** נכונה: בעת גישה למפתח שאינו קיים נזרקת `KeyError`.
*   **אפשרות ד** אינה נכונה: אלמנט חדש אינו נוצר בעת ניסיון גישה.

**כיצד פועלת גישה למילון:**

1.  לקבלת ערך לפי מפתח משתמשים בתחביר `dictionary[key]`.
2.  אם המפתח `key` קיים במילון, יוחזר הערך המתאים.
3.  אם המפתח `key` אינו קיים, תיזרק חריגה מסוג `KeyError`.

**דוגמה:**

```python
my_dict: dict[str, int] = {"a": 1, "b": 2}

try:
    value: int = my_dict["c"] # מנסים לקבל ערך לפי מפתח "c"
except KeyError as e:
    print(f"Ошибка: Ключ не найден! {e}") # פלט: שגיאה: מפתח לא נמצא! 'c'


value = my_dict.get("c", None) # משתמשים במתודת get כדי לקבל ערך לפי מפתח או None
print(value) # פלט: None

value = my_dict.get("c", 0) # ניתן להגדיר ערך ברירת מחדל
print(value) # פלט: 0
```

**כתוצאה מכך:**

*   גישה ישירה באמצעות מפתח שאינו קיים (`my_dict["c"]`) גורמת לזריקת `KeyError`.
*   בעת שימוש במתודת `get()` והמפתח אינו קיים - מוחזר ערך ברירת המחדל (אם עבר) או `None` אם לא עבר.

לפיכך, **אפשרות C** היא התשובה הנכונה.
