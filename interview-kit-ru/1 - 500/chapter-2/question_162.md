**שאלה 62.** מה יהיה הפלט של הקוד הבא?

```python
x: int = 10
y: int = 20
if x > 5:
    if y < 15:
        print("Оба условия верны") # Keep original string in code block as per rule 2
    else:
        print("x больше 5, но y не меньше 15") # Keep original string
else:
    print("x не больше 5") # Keep original string
```

A. `x не больше 5`
B. `Оба условия верны`
C. `x больше 5, но y не меньше 15`
D. שגיאה

**תשובה נכונה: C**

**הסבר:**

קוד זה מדגים אופרטורים תנאיים מקוננים מסוג `if-else`.

1.  **אתחול:** `x` מוגדר להיות שווה ל-10, ו-`y` מוגדר להיות שווה ל-20.
2.  **ה-`if` החיצוני:** `if x > 5:` בודק ש-`x` גדול מ-5. תנאי זה אמיתי, מכיוון ש-`x` שווה ל-10.
3.  **ה-`if` הפנימי:** בלוק הקוד של האופרטור `if` הפנימי מבוצע. `if y < 15:` בודק ש-`y` קטן מ-15. תנאי זה שקרי, מכיוון ש-`y` שווה ל-20.
4.  **ה-`else` הפנימי:** מכיוון שהתנאי של ה-`if` הפנימי שקרי, מבוצע הקוד המקונן בתוך ה-`else`.

עתה נתבונן באפשרויות התשובה:

*   **אפשרות A** אינה נכונה: התנאי החיצוני `x > 5` אמיתי, ולכן בלוק ה-`else` (החיצוני) לא יבוצע.
*   **אפשרות B** אינה נכונה: התנאי של ה-`if` הפנימי שקרי, ולכן בלוק `"Оба условия верны"` לא יבוצע.
*   **אפשרות C** נכונה: ה-`else` הפנימי מדפיס את המחרוזת `"x больше 5, но y не меньше 15"`, מכיוון שהתנאי `y<15` שקרי.
*   **אפשרות D** אינה נכונה: הקוד תקין תחבירית ולא תהיה שגיאה.

**דוגמה:**

```python
x: int = 10
y: int = 20
if x > 5:
    if y < 15:
        print("Оба условия верны")
    else:
        print("x больше 5, но y не меньше 15") # פלט: x גדול מ-5, אך y אינו קטן מ-15
else:
    print("x не больше 5")
```

**כתוצאה מכך:**
*   מודפסת המחרוזת `"x больше 5, но y не меньше 15"`.

לפיכך, **אפשרות C** היא הנכונה.
