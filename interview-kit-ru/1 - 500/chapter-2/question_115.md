**שאלה 115.** מהו תפקידם של אופרטורי `if` מקוננים ב-Python?

- א. הם מאפשרים לבצע סט נוסף של אופרטורים אם יותר מתנאי אחד הוא אמיתי, ובכך מספקים סינון רב-רמות.
- ב. הם מפחיתים את יעילות התוכנית על ידי הגדלת מספר הבדיקות המבוצעות.
- ג. הם מאפשרים למתכנת לציין בלוק קוד ברירת מחדל שמבוצע ללא תלות בתנאים.
- ד. אופרטורי `if` מקוננים משמשים ליציאה מתנאי `if` חיצוני מוקדם מהרגיל.

**תשובה נכונה: א**

**הסבר:**

אופרטורי `if` מקוננים ב-Python משמשים כאשר יש צורך לבדוק תנאי אחד או יותר בתוך בלוק קוד שמבוצע לאחר בדיקת תנאי אחר. הדבר מאפשר יצירת לוגיקת הסתעפות מורכבת.

*   **אפשרות א** נכונה: אופרטורי `if` מקוננים מאפשרים להוסיף בדיקות נוספות אם מספר תנאים אמיתיים, מה שמאפשר לסנן נתונים במספר רמות.
*   **אפשרות ב** אינה נכונה: אופרטורי `if` מקוננים אינם מפחיתים ביצועים, אך הם הופכים את הקוד למורכב יותר.
*   **אפשרות ג** אינה נכונה: אופרטור `if` אינו מאפשר להגדיר "בלוק קוד ברירת מחדל" שמבוצע תמיד. זאת עושה הצהרת `else`.
*   **אפשרות ד** אינה נכונה: אופרטור `if` אינו משמש ליציאה מבלוק, לשם כך משתמשים ב-`break` או `return`.

**כיצד פועלים אופרטורי `if` מקוננים:**

1.  אופרטור ה-`if` הראשון בודק את התנאי שלו.
2.  אם התנאי הראשון אמיתי, מבוצע בלוק הקוד שלו, שיכול להכיל אופרטורי `if` מקוננים שיבדקו תנאים נוספים.
3.  אופרטור `if` מקונן בודק את התנאי שלו, ואם הוא אמיתי, מבצע את בלוק הקוד שלו.
4.  ניתן לקנן אופרטורי `if` זה בתוך זה מספר פעמים.

**מתי להשתמש באופרטורי `if` מקוננים:**
*   **סינון רב-שכבתי:** כאשר יש צורך לבדוק מספר תנאים קשורים הדדית, כאשר ביצוע הקוד בבלוק אחד תלוי בבדיקות קודמות.
*   **לוגיקה מורכבת:** לבניית לוגיקה מורכבת יותר, שבה תנאי אחד משפיע על בדיקת תנאי אחר.

**דוגמה:**

```python
x: int = 10
y: int = 5
if x > 0:
    print("x положительное") # פלט: x חיובי
    if y > 0:
        print("y положительное") # פלט: y חיובי
        if x > y:
           print("x больше y") # פלט: x גדול מ-y
        else:
           print("y больше или равен x")
    else:
       print("y не положительное")
else:
  print("x не положительное")
```
**בתוצאה:**
*  אופרטור ה-`if` החיצוני בודק ש-`x` גדול מ-`0`. אם זה נכון, מבוצעים אופרטורי התנאי הפנימיים.
*  בהמשך מבוצע אופרטור ה-`if` המקונן, שם נבדק ש-`y` גדול מ-`0`. אם זה נכון, מבוצעת הבדיקה המקוננת הבאה.

לפיכך, **אפשרות א** היא הנכונה.
