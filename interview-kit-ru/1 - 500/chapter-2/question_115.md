### `question_115.md`

**שאלה 115.** מהי הפונקציה של שימוש באופרטורי `if` מקוננים בפייתון?

- A. הם מאפשרים לבצע סט נוסף של אופרטורים אם יותר מתנאי אחד מתקיים (True), ובכך מספקים סינון רב-שכבתי.
- B. הם מפחיתים את יעילות התוכנית עקב הגדלת מספר הבדיקות המתבצעות.
- C. הם מאפשרים למתכנת לציין בלוק קוד ברירת מחדל, אשר יתבצע ללא קשר לתנאים.
- D. אופרטורי `if` מקוננים משמשים ליציאה מוקדמת מהתנאי החיצוני של `if`.

**תשובה נכונה: A**

**הסבר:**

אופרטורי `if` מקוננים בפייתון משמשים כאשר יש צורך לבדוק תנאי אחד או יותר בתוך בלוק קוד המתבצע לאחר בדיקת תנאי אחר. זה מאפשר ליצור לוגיקת הסתעפות מורכבת.

*   **אפשרות A** נכונה: `if` מקוננים מאפשרים להוסיף בדיקות נוספות אם מספר תנאים מתקיימים, מה שמאפשר סינון נתונים במספר רמות.
*   **אפשרות B** אינה נכונה: `if` מקוננים אינם מפחיתים את הביצועים, אך הופכים את הקוד למורכב יותר.
*   **אפשרות C** אינה נכונה: `if` אינו מאפשר להגדיר "בלוק קוד ברירת מחדל" שמתבצע תמיד. לשם כך משמש `else`.
*   **אפשרות D** אינה נכונה: `if` אינו משמש ליציאה מבלוק, לשם כך משמשים `break` או `return`.

**כיצד פועלים `if` מקוננים:**

1.  אופרטור ה-`if` הראשון בודק את התנאי שלו.
2.  אם התנאי הראשון מתקיים (True), מתבצע בלוק הקוד שלו, שעשוי להכיל `if` מקוננים שיבדקו תנאים נוספים.
3.  ה-`if` המקונן בודק את התנאי שלו, ואם הוא מתקיים (True), הוא מבצע את בלוק הקוד שלו.
4. ניתן לקנן `if` זה בתוך זה מספר פעמים.

**מתי להשתמש ב-`if` מקוננים:**
*   **סינון רב-שכבתי:** כאשר יש לבדוק מספר תנאים הקשורים זה לזה, כאשר ביצוע קוד בבלוק אחד תלוי בבדיקות קודמות.
*   **לוגיקה מורכבת:** לבניית לוגיקה מורכבת יותר, שבה תנאי אחד משפיע על בדיקת תנאי אחר.

**דוגמה:**

```python
x: int = 10
y: int = 5
if x > 0:
    print("x חיובי") # פלט: x חיובי
    if y > 0:
        print("y חיובי") # פלט: y חיובי
        if x > y:
           print("x גדול מ-y") # פלט: x גדול מ-y
        else:
           print("y גדול או שווה ל-x")
    else:
       print("y אינו חיובי")
else:
  print("x אינו חיובי")
```
**בתוצאה:**
*   אופרטור ה-`if` החיצוני בודק ש-`x` גדול מ-`0`. אם זה נכון, מתבצעים האופרטורים המותנים הפנימיים.
*   לאחר מכן מתבצע אופרטור ה-`if` המקונן, שבו נבדק ש-`y` גדול מ-`0`. אם זה נכון, מתבצעת הבדיקה המקוננת הבאה.

לפיכך, **אפשרות A** היא הנכונה.
