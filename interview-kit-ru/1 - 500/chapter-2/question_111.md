### `question_111.md`

**שאלה 111.** מה ההבדל בין שרשרת `if-elif-else` לבין סדרה של הצהרות `if` בפייתון בעת טיפול בתנאים מרובים?

- A. `if-elif-else` מבצעת רק בלוק קוד אחד מבין מספר תנאים, בעוד שסדרה של הצהרות `if` עשויה לבצע את כל בלוקי הקוד שהתנאים שלהם מתקיימים.
- B. אין הבדל; שני האופנים מעריכים את כל התנאים שסופקו ומבצעים את אותו בלוק קוד.
- C. סדרה של הצהרות `if` משמשת לבדיקת תנאי יחיד, בעוד ש-`if-elif-else` משמשת לתנאים מרובים ובלתי קשורים.
- D. `if-elif-else` עשויה לבצע מספר בלוקי קוד ברצף ללא הערכה מחדש של התנאים.

**תשובה נכונה: A**

**הסבר:**

בפייתון, פסוקי התנאי `if`, `elif` (קיצור של "else if") ו-`else` מאפשרים ליצור מבנים מורכבים לבדיקת מספר תנאים. ההבדל המרכזי בין שימוש בשרשרת `if-elif-else` לבין סדרה של הצהרות `if` נפרדות נעוץ באופן שבו הן מטפלות בבדיקה ובביצוע של בלוקי הקוד.

*   **אפשרות A** נכונה: `if-elif-else` מבצעת רק בלוק קוד אחד מכלל האפשרויות, בעוד שמספר הצהרות `if` יבצעו את כל הבלוקים שהתנאים שלהם יהיו `True`.
*   **אפשרות B** אינה נכונה: כל אחת מהאפשרויות מתנהגת באופן שונה, בניגוד לבחירה בין `if-elif-else` לבין סדרה של הצהרות `if`.
*   **אפשרות C** אינה נכונה: גם `if-elif-else` וגם הצהרות `if` נפרדות יכולות לטפל בכל סוגי התנאים.
*   **אפשרות D** אינה נכונה: `if-elif-else` מבצעת רק אחד מהבלוקים, ולא מבצעת אותם ברצף.

**כיצד פועלת שרשרת `if-elif-else`:**

1.  בתחילה מוערך התנאי בהצהרת `if`.
2.  אם תנאי ה-`if` הוא אמת, בלוק הקוד המתאים מתבצע, וכל שאר בלוקי ה-`elif` וה-`else` נדלגים.
3.  אם תנאי ה-`if` הוא שקר, נבדק התנאי בהצהרת ה-`elif` הראשונה.
4.  אם תנאי ה-`elif` הוא אמת, בלוק הקוד המתאים מתבצע, וכל שאר הבלוקים נדלגים.
5.  תהליך זה חוזר על עצמו עבור כל `elif` עד שנמצא תנאי שהוא אמת, או עד שמגיעים לבלוק ה-`else`.
6.  אם אף אחד מתנאי ה-`if` או ה-`elif` אינו אמת, בלוק ה-`else` מתבצע.

**כיצד פועלות הצהרות `if` רצופות:**

1.  כל הצהרת `if` נבדקת באופן בלתי תלוי.
2.  בלוק הקוד של כל הצהרת `if` מתבצע רק אם התנאי שלה הוא אמת.
3.  אם מספר תנאים הם אמת, אזי יתבצעו מספר בלוקי קוד מתאימים.

**דוגמה:**

```python
x: int = 10
print("Пример if-elif-else:")
if x > 15:
  print("x больше 15")
elif x > 5:
  print("x больше 5, но не больше 15") # פלט: x גדול מ-5, אך לא גדול מ-15
elif x > 0:
  print("x больше 0, но не больше 5")
else:
  print("x <= 0")

print("Пример последовательных if:")
if x > 15:
  print("x больше 15")
if x > 5:
  print("x больше 5") # פלט: x גדול מ-5
if x > 0:
    print("x больше 0") # פלט: x גדול מ-0
```
**תוצאה:**
*   בדוגמה של `if-elif-else`, רק בלוק אחד מתבצע. לאחר שהתנאי `x > 5` היה אמת, שאר ה-`elif` וה-`else` אינם נבדקים.
*   בדוגמה של הצהרות `if` נפרדות, כל התנאים נבדקים, והתוצאות המתאימות מוצגות בפלט.

לפיכך, **אפשרות A** היא הנכונה.
