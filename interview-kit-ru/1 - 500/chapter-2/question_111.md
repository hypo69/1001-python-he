### `question_111.md`

**שאלה 111.** מה ההבדל בין שרשרת `if-elif-else` לסדרה של אופרטורי `if` בפייתון בעת טיפול במספר תנאים?

- א. `if-elif-else` מבצע רק בלוק קוד אחד מבין מספר תנאים, בעוד שאופרטורי `if` מרובים יכולים לבצע את כל בלוקי הקוד שהתנאים שלהם מתקיימים.
- ב. אין הבדל; שתי האפשרויות מעריכות את כל התנאים שסופקו ומבצעות את אותו בלוק קוד.
- ג. אופרטורי `if` מרובים משמשים לבדיקת תנאי יחיד, בעוד ש-`if-elif-else` משמש למספר תנאים שאינם קשורים זה לזה.
- ד. `if-elif-else` יכול לבצע מספר בלוקי קוד ברצף ללא הערכה מחדש של התנאים.

**תשובה נכונה: א**

**הסבר:**

בפייתון, אופרטורים מותנים `if`, `elif` (קיצור של "else if") ו-`else` מאפשרים ליצור מבנים מורכבים לבדיקת מספר תנאים. ההבדל המרכזי בין שימוש בשרשרת `if-elif-else` לבין סדרה של אופרטורי `if` נפרדים טמון באופן שבו הם מטפלים בבדיקת התנאים וביצוע בלוקי הקוד.

*   **אפשרות א** נכונה: `if-elif-else` מבצע רק בלוק קוד אחד מכלל התנאים, בעוד שמספר אופרטורי `if` יבצעו את כל הבלוקים שהתנאים שלהם יהיו `True`.
*   **אפשרות ב** אינה נכונה: כל אחת מהאפשרויות מתנהגת אחרת, בשונה מהבחירה בין `if-elif-else` לבין סדרה של אופרטורי `if`.
*   **אפשרות ג** אינה נכונה: גם `if-elif-else` וגם אופרטורי `if` נפרדים יכולים לטפל בכל תנאי.
*   **אפשרות ד** אינה נכונה: `if-elif-else` מבצע רק אחד מהבלוקים, ולא מבצע אותם ברצף.

**כיצד פועל `if-elif-else`:**

1.  ראשית, התנאי באופרטור ה-`if` מוערך.
2.  אם התנאי של ה-`if` אמיתי, אזי בלוק הקוד המתאים מבוצע, וכל שאר בלוקי ה-`elif` וה-`else` מדלגים.
3.  אם התנאי של ה-`if` שקרי, נבדק התנאי באופרטור ה-`elif` הראשון.
4.  אם התנאי של ה-`elif` אמיתי, בלוק הקוד המתאים מבוצע, וכל שאר הבלוקים מדלגים.
5.  תהליך זה חוזר על עצמו עבור כל `elif`, עד שיימצא תנאי אמיתי, או עד שיגיעו לבלוק ה-`else`.
6. אם אף אחד מהתנאים של ה-`if` או ה-`elif` אינו אמיתי, בלוק ה-`else` מבוצע.

**כיצד פועלים אופרטורי `if` רציפים:**

1.  כל אופרטור `if` נבדק באופן עצמאי.
2.  בלוק הקוד של כל אופרטור `if` מבוצע רק אם התנאי שלו אמיתי.
3.  אם מספר תנאים אמיתיים, אזי יבוצעו מספר בלוקי קוד מתאימים.

**דוגמה:**

```python
x: int = 10
print("דוגמה if-elif-else:")
if x > 15:
  print("x גדול מ-15")
elif x > 5:
  print("x גדול מ-5, אך לא גדול מ-15") # פלט: x גדול מ-5, אך לא גדול מ-15
elif x > 0:
  print("x גדול מ-0, אך לא גדול מ-5")
else:
  print("x <= 0")

print("דוגמה של אופרטורי if רציפים:")
if x > 15:
  print("x גדול מ-15")
if x > 5:
  print("x גדול מ-5") # פלט: x גדול מ-5
if x > 0:
    print("x גדול מ-0") # פלט: x גדול מ-0
```
**בתוצאה:**
* ב-`if-elif-else` של הדוגמה, רק בלוק אחד מבוצע. לאחר שהתנאי `x > 5` היה אמיתי, שאר ה-`elif` וה-`else` אינם נבדקים.
* בדוגמה עם אופרטורי `if` נפרדים, כל התנאים נבדקים, והתוצאות המתאימות מודפסות.

לסיכום, **אפשרות א** נכונה.
