### `question_178.md`

**שאלה 178.** מהו ייעוד המתודה `get()` ביחס למילונים בפייתון?

A.  היא מוסיפה זוג מפתח-ערך חדש למילון.
B.  היא מוחקת זוג מפתח-ערך מהמילון לפי מפתח שצוין.
C.  היא מחזירה את הערך לפי מפתח שצוין ומחזירה `None` (או ערך ברירת מחדל), אם המפתח לא נמצא.
D.  היא מחזירה רשימה של כל המפתחות במילון.

**תשובה נכונה: C**

**הסבר:**

המתודה `get()` של מילון בפייתון מיועדת לקבלת ערך לפי מפתח, אך היא גם מאפשרת להגדיר ערך ברירת מחדל, שיוחזר אם המפתח אינו קיים במילון. זה מאפשר להימנע משגיאות `KeyError`.

*   **אפשרות A** אינה נכונה: להוספת אלמנט משתמשים ב-`dictionary[key] = value` או ב-`dictionary.update()`.
*   **אפשרות B** אינה נכונה: למחיקה משתמשים במתודה `pop()` או ב-`del`.
*   **אפשרות C** נכונה: `get()` מחזירה ערך לפי מפתח או ערך ברירת מחדל אם המפתח אינו קיים.
*   **אפשרות D** אינה נכונה: `keys()` מחזירה רשימה של כל המפתחות.

**כיצד עובדת `get()`:**

1.  `get(key)`: מחזירה את הערך המשויך למפתח `key`. אם המפתח לא נמצא, מחזירה `None`.
2.  `get(key, default)`: מחזירה את הערך המשויך למפתח `key`. אם המפתח לא נמצא, מחזירה את הערך `default`.

**דוגמה:**

```python
my_dict: dict[str, int] = {"a": 1, "b": 2}

value1: int | None = my_dict.get("a")
print(f"ערך עבור 'a': {value1}")  # פלט: ערך עבור 'a': 1

value2: int | None = my_dict.get("c")
print(f"ערך עבור 'c' (לא נמצא): {value2}")  # פלט: ערך עבור 'c' (לא נמצא): None

value3: int = my_dict.get("c", 0)
print(f"ערך עבור 'c' (לא נמצא, ברירת מחדל 0): {value3}")  # פלט: ערך עבור 'c' (לא נמצא, ברירת מחדל 0): 0
```

**בתוצאה:**

*   המתודה `get("a")` מחזירה את הערך עבור המפתח הקיים `"a"`.
*   המתודה `get("c")` מחזירה `None`, מכיוון שהמפתח `"c"` אינו במילון.
*  המתודה `get("c", 0)` מחזירה את הערך 0, כערך ברירת המחדל.
*   גישה ישירה למילון באמצעות מפתח שאינו קיים הייתה גורמת לשגיאה `KeyError`.

לכן, **אפשרות C** היא הנכונה.
