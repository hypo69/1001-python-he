**שאלה 181.** איזו שיטה משמשת להסרת כל האיברים מרשימה בפייתון?

A. `remove()`
B. `clear()`
C. `pop()`
D. `delete()`

**תשובה נכונה: B**

**הסבר:**

השיטה `clear()` בפייתון משמשת להסרת כל האיברים מרשימה, והופכת אותה לריקה.

*   **אפשרות A אינה נכונה:** השיטה `remove()` מסירה רק את המופע הראשון של הערך שצוין.
*   **אפשרות B נכונה:** השיטה `clear()` מסירה את כל האיברים מהרשימה.
*   **אפשרות C אינה נכונה:** השיטה `pop()` מסירה איבר לפי האינדקס שצוין או את האיבר האחרון ברשימה.
*   **אפשרות D אינה נכונה:** `delete()` אינה שיטה עבור רשימות.

**אופן הפעולה של `clear()`:**

1.  השיטה `clear()` אינה מקבלת ארגומנטים.
2.  היא מסירה את כל האיברים מהרשימה, ומותירה רשימה ריקה.
3.  `clear()` אינה מחזירה שום ערך.

**דוגמה:**

```python
my_list: list[int] = [1, 2, 3, 4, 5]
print(f"הרשימה המקורית: {my_list}") # פלט: הרשימה המקורית: [1, 2, 3, 4, 5]

my_list.clear()
print(f"הרשימה לאחר clear(): {my_list}")  # פלט: הרשימה לאחר clear(): []
```

**כתוצאה מכך:**
*   לאחר הפעלת השיטה `clear()` הרשימה הופכת לריקה.

לסיכום, **אפשרות B** היא התשובה הנכונה.
