### `שאלה_113.md`

**שאלה 113.** איזו השפעה יש לקינון של יאמרת `break` בתוך מספר לולאות על מבנה הלולאה?

- א. היא מביאה לסיום רק של הלולאה הפנימית ביותר, בה היא ממוקמת, ומאפשרת ללולאות החיצוניות להמשיך את ביצוען.
- ב. היא יוצאת מיד מכל הלולאות המקוננות, ומסיימת את מבנה הלולאה כולו.
- ג. היא משעה זמנית את ביצוע הלולאה ומחדשת אותו רק כאשר התנאי הנתון משתנה.
- ד. היא מאפסת את תנאי הלולאה הפנימית ומפעילה את הלולאה מחדש מההתחלה.

**תשובה נכונה: א**

**הסבר:**

ב-Python, יאמרת `break` משמשת ליציאה מיידית מלולאה. כאשר היא נמצאת בשימוש בלולאות מקוננות, יאמרת `break` משפיעה רק על הלולאה הפנימית ביותר, בה היא ממוקמת באופן ישיר.

*   אפשרות א' נכונה: `break` מביאה לסיום רק של הלולאה בה היא נמצאת.

*   אפשרות ב' אינה נכונה: `break` אינה מביאה לסיום של כל הלולאות המקוננות.

*   אפשרות ג' אינה נכונה: `break` אינה משעה את הלולאה, אלא מביאה לסיומה.
*   אפשרות ד' אינה נכונה: `break` אינה מפעילה מחדש את הלולאה.

**אופן הפעולה של `break` בלולאות מקוננות:**

1.  כאשר מפרש ה-Python נתקל ב-`break` בתוך לולאה מקוננת, ביצוע לולאה זו נפסק באופן מיידי.
2.  הבקרה מועברת לפקודה הבאה, הממוקמת מיד לאחר הלולאה *הפנימית*.
3.  הלולאות החיצוניות ממשיכות את ביצוען.

**דוגמה:**

```python
for i in range(3):
    print(f"Внешний цикл: i = {i}")
    for j in range(3):
        if j == 1:
            print("Выход из внутреннего цикла")
            break
        print(f"  Внутренний цикл: j = {j}")
    print("Продолжение внешнего цикла")
# פלט:
# Внешний цикл: i = 0
#   Внутренний цикл: j = 0
# Выход из внутреннего цикла
# Продолжение внешнего цикла
# Внешний цикл: i = 1
#   Внутренний цикл: j = 0
# Выход из внутреннего цикла
# Продолжение внешнего цикла
# Внешний цикл: i = 2
#   Внутренний цикл: j = 0
# Выход из внутреннего цикла
# Продолжение внешнего цикла
```

**כתוצאה מכך:**

*   כאשר `j` בלולאה הפנימית הופך שווה ל-1, הלולאה נקטעת על ידי יאמרת `break`.
*   הלולאה החיצונית ממשיכה את עבודתה עד הסוף.

לפיכך, **אפשרות א'** היא הנכונה.
