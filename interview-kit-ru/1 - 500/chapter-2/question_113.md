**שאלה 113.** איזו השפעה יש להכללת האופרטור `break` בתוך מספר לולאות מקוננות על מבנה הלולאה?

- א. הוא מסיים רק את הלולאה הפנימית ביותר שבה הוא ממוקם, ומאפשר ללולאות החיצוניות להמשיך לפעול.
- ב. הוא יוצא מיד מכל הלולאות המקוננות, ומסיים את כל מבנה הלולאה.
- ג. הוא משהה זמנית את ביצוע הלולאה ומחדש אותו רק כאשר התנאי הנתון משתנה.
- ד. הוא מאפס את תנאי הלולאה הפנימית ומפעיל את הלולאה מההתחלה.

**תשובה נכונה: א**

**הסבר:**

ב-Python, האופרטור `break` משמש ליציאה מיידית מלולאה. כאשר הוא משמש בלולאות מקוננות, האופרטור `break` משפיע רק על הלולאה הפנימית ביותר שבה הוא ממוקם ישירות.

*   **אפשרות א'** נכונה: `break` מסיים רק את הלולאה שבה הוא נמצא.

*   **אפשרות ב'** אינה נכונה: `break` אינו מסיים את כל הלולאות המקוננות.

*   **אפשרות ג'** אינה נכונה: `break` אינו משהה את הלולאה, אלא מסיים אותה.
*   **אפשרות ד'** אינה נכונה: `break` אינו מפעיל מחדש את הלולאה.

**כיצד פועל `break` בלולאות מקוננות:**

1.  כאשר מפרש Python נתקל ב-`break` בתוך לולאה מקוננת, ביצוע לולאה זו נפסק באופן מיידי.
2.  השליטה עוברת לפקודה הבאה, הממוקמת מיד לאחר הלולאה ה*פנימית*.
3.  הלולאות החיצוניות ממשיכות את ביצוען.

**דוגמה:**

```python
for i in range(3):
    print(f"Внешний цикл: i = {i}") # Outer loop: i = {i}
    for j in range(3):
        if j == 1:
            print("Выход из внутреннего цикла") # Exiting inner loop
            break
        print(f"  Внутренний цикл: j = {j}") # Inner loop: j = {j}
    print("Продолжение внешнего цикла") # Continuation of outer loop
# Вывод: # Output:
# Внешний цикл: i = 0 # Outer loop: i = 0
#   Внутренний цикл: j = 0 # Inner loop: j = 0
# Выход из внутреннего цикла # Exiting inner loop
# Продолжение внешнего цикла # Continuation of outer loop
# Внешний цикл: i = 1 # Outer loop: i = 1
#   Внутренний цикл: j = 0 # Inner loop: j = 0
# Выход из внутреннего цикла # Exiting inner loop
# Продолжение внешнего цикла # Continuation of outer loop
# Внешний цикл: i = 2 # Outer loop: i = 2
#   Внутренний цикл: j = 0 # Inner loop: j = 0
# Выход из внутреннего цикла # Exiting inner loop
# Продолжение внешнего цикла # Continuation of outer loop
```

**כתוצאה מכך:**

*   כאשר `j` בלולאה הפנימית הופך שווה ל-1, הלולאה נקטעת על ידי האופרטור `break`.
*   הלולאה החיצונית ממשיכה לפעול עד סופה.

לפיכך, **אפשרות א'** נכונה.
