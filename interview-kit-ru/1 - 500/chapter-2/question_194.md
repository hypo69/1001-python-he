**שאלה 194.** ב-Python, מה עושה המתודה `remove()` בהקשר של רשימה (list)?

א. היא מסירה את כל האיברים מהרשימה.
ב. היא מסירה איבר לפי האינדקס שלו.
ג. היא מסירה איבר לפי הערך שלו, המופע הראשון.
ד. היא מסירה את האיבר האחרון ברשימה ומחזירה אותו.

**תשובה נכונה: ג**

**הסבר:**

המתודה `remove()` ב-Python משמשת להסרת ה*מופע הראשון* של איבר עם ערך מסוים מרשימה.

*   **אפשרות א'** אינה נכונה: להסרת כל האיברים משתמשים במתודה `clear()`.
*   **אפשרות ב'** אינה נכונה: להסרה לפי אינדקס משתמשים במתודה `pop()`.
*   **אפשרות ג'** נכונה: המתודה `remove()` מסירה איבר מרשימה לפי ערך.
*   **אפשרות ד'** אינה נכונה: להסרת האיבר האחרון משתמשים במתודה `pop()` ללא ציון אינדקס.

**כיצד פועלת remove():**

1.  המתודה `remove()` מקבלת ארגומנט אחד: ערך האיבר שיש להסיר.
2.  אם נמצא איבר עם הערך שצוין, הוא מוסר מהרשימה. אם קיימים ברשימה מספר איברים עם אותו ערך, יוסר רק המופע הראשון.
3.  אם האיבר לא נמצא, מתרחשת שגיאת `ValueError`.
4.  `remove()` אינה מחזירה את האיבר שהוסר, אלא משנה את הרשימה במקום.

**דוגמה:**

```python
my_list: list[int] = [1, 2, 3, 2, 4]
print(f"Исходный список: {my_list}") # Output: Initial list: [1, 2, 3, 2, 4]

my_list.remove(2)
print(f"Список после remove(2): {my_list}") # Output: List after remove(2): [1, 3, 2, 4]

# Error:
# my_list.remove(10) # ValueError: list.remove(x): x not in list
```

**תוצאה:**

*   המתודה `remove()` מסירה רק את המופע הראשון של הערך 2 מהרשימה.

לסיכום, **אפשרות ג'** היא התשובה הנכונה.
