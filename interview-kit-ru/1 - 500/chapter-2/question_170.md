### `question_170.md`

**שאלה 170.** מה עושה הפונקציה `sorted()` בפייתון?

א. ממיינת רשימה בסדר יורד.
ב. ממיינת רשימה בסדר עולה.
ג. משנה את סדר האיברים ברשימה המקורית להיפוך.
ד. יוצרת רשימה ממוינת חדשה, מבלי לשנות את המקור.

**תשובה נכונה: ד**

**הסבר:**

הפונקציה `sorted()` בפייתון משמשת ליצירת רשימה ממוינת חדשה מהאיברים של כל אובייקט איטרבילי (כגון רשימה, טיפל או מחרוזת). היא אינה משנה את האובייקט המקורי.

*   **אפשרות א** אינה נכונה: `sorted()` ממיינת כברירת מחדל בסדר עולה, אך קיימת אפשרות לשנות את סדר המיון.
*   **אפשרות ב** אינה נכונה: `sorted()` ממיינת כברירת מחדל בסדר עולה.
*   **אפשרות ג** אינה נכונה: עבור סדר הפוך משתמשים במתודה `reversed()`.
*   **אפשרות ד** נכונה: הפונקציה `sorted()` מחזירה רשימה ממוינת חדשה, ומשאירה את המקור ללא שינוי.

**כיצד פועלת `sorted()`:**

1.  הפונקציה `sorted()` מקבלת אובייקט איטרבילי ומחזירה רשימה חדשה המכילה את האיברים של האובייקט האיטרבילי שהוכנס, בסדר ממוין.
2.  כברירת מחדל, המיון מתבצע בסדר עולה, אך ניתן לשנותו על ידי העברת הפרמטר `reverse=True`.
3.  למיון אובייקטים מורכבים (כגון רשימות של מילונים) ניתן להעביר מפתח מיון (המפתח `key`) - פונקציה שמחזירה את הערך שלפיו יש למיין.

**דוגמאות:**

```python
numbers: list[int] = [3, 1, 4, 1, 5, 9, 2, 6]
sorted_numbers: list[int] = sorted(numbers)
print(f"רשימה ממוינת (בסדר עולה): {sorted_numbers}") # פלט: רשימה ממוינת (בסדר עולה): [1, 1, 2, 3, 4, 5, 6, 9]
print(f"רשימה מקורית: {numbers}") # פלט: רשימה מקורית: [3, 1, 4, 1, 5, 9, 2, 6]


sorted_numbers_reverse: list[int] = sorted(numbers, reverse=True)
print(f"רשימה ממוינת (בסדר יורד): {sorted_numbers_reverse}") # פלט: רשימה ממוינת (בסדר יורד): [9, 6, 5, 4, 3, 2, 1, 1]
```

**כתוצאה מכך:**

*   `sorted()` מחזירה רשימה ממוינת חדשה, והרשימה המקורית `numbers` נשארת ללא שינוי.
*   ניתן לקבוע את סדר המיון באמצעות הפרמטר `reverse = True`.

לפיכך, **אפשרות ד** היא הנכונה.
