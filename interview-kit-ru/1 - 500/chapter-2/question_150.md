### `question_150.md`

**שאלה 50.** מהי הבנת רשימה (list comprehension) ב-Python?

A. זו פונקציה להמרת רשימה לטאפל.
B. זו דרך להגדרת סוגי נתונים ברשימה.
C. זו דרך תמציתית ליצירת רשימות על בסיס אובייקטים ניתנים לאיטרציה אחרים.
D. זו מתודה להוספת מספר איברים לרשימה בו זמנית.

**Правильный ответ: C**

**הסבר:**

הבנת רשימה (list comprehension) היא דרך תמציתית ליצירת רשימות חדשות ב-Python על בסיס אובייקטים קיימים ניתנים לאיטרציה (לדוגמה, רשימות, טאפלים, מחרוזות).

*   **אפשרות A** שגויה: הבנת רשימה יוצרת *רשימה*, ולא טאפל.
*   **אפשרות B** שגויה: הבנת רשימה אינה מגדירה סוגי נתונים, אלא יוצרת רשימה, המגדירה את סוג האיברים מתוך הביטוי.
*   **אפשרות C** נכונה: הבנת רשימה מאפשרת ליצור רשימות על בסיס אובייקטים ניתנים לאיטרציה אחרים.
*   **אפשרות D** שגויה: להוספת איברים משתמשים במתודת `extend()` או `append()`, ולא בהבנת רשימה.

**כיצד פועלת הבנת רשימה:**

1.  התחביר של הבנת רשימה מורכב מביטוי, שאחריו באה לולאת `for`, ו(אופציונלי) ביטוי אחד או יותר של `if`.
2.  הביטוי מבוצע עבור כל איבר באובייקט הניתן לאיטרציה, שצוין לאחר ה-`for`.
3.  אם צוין תנאי `if`, אזי הביטוי מבוצע רק עבור אותם איברים העומדים בתנאי זה.
4.  תוצאות ביצוע הביטוי נשמרות ברשימה חדשה.

**דוגמאות:**

```python
# יצירת רשימה של ריבועי מספרים
squares: list[int] = [x**2 for x in range(5)]
print(squares) # פלט: [0, 1, 4, 9, 16]

# יצירת רשימה של מספרים זוגיים מרשימה אחרת
numbers: list[int] = [1, 2, 3, 4, 5, 6]
even_numbers: list[int] = [x for x in numbers if x % 2 == 0]
print(even_numbers) # פלט: [2, 4, 6]
```

**לסיכום:**
*   הבנת רשימה יוצרת רשימה חדשה `squares` על בסיס סדרת מספרים מ-0 עד 4.
*   הבנת רשימה יוצרת רשימה חדשה `even_numbers` על בסיס הרשימה `numbers`, לוקחת רק את המספרים הזוגיים.

לפיכך, **אפשרות C** מהווה את התשובה הנכונה.
