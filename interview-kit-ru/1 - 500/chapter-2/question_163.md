**שאלה 63.** מהו ייעודו של המתודה `__init__` במחלקת Python?

א.  היא נקראת בעת מחיקת אובייקט מהזיכרון.
ב.  היא נקראת בעת יצירת אובייקט חדש של המחלקה.
ג.  היא משמשת להגדרת משתנים סטטיים של המחלקה.
ד.  היא משמשת להשוואת שני אובייקטים.

**תשובה נכונה: ב**

**הסבר:**

המתודה `__init__` היא מתודה מיוחדת (בנאי) ב-Python, הנקראת אוטומטית בעת יצירת מופע חדש של המחלקה (אובייקט). היא משמשת לאתחול האובייקט והקצאת ערכי התחלה עבורו.

*   **אפשרות א** אינה נכונה: המתודה `__del__` נקראת בעת מחיקת אובייקט.
*   **אפשרות ב** נכונה: `__init__` היא הבנאי ונקראת בעת יצירת אובייקט.
*   **אפשרות ג** אינה נכונה: משתנים סטטיים של המחלקה (class variables) מוגדרים ישירות בגוף המחלקה, ולא בתוך `__init__`.
*   **אפשרות ד** אינה נכונה: להשוואת אובייקטים משתמשים במתודות אחרות, למשל `__eq__()`.

**כיצד פועלת `__init__`:**

1.  המתודה `__init__` נקראת אוטומטית לאחר יצירת אובייקט חדש של המחלקה.
2.  הארגומנט הראשון של המתודה הוא תמיד `self`, המפנה לאובייקט שנוצר.
3.  בתוך המתודה `__init__` בדרך כלל מוגדרים ערכי ההתחלה של התכונות (attributes) של האובייקט.
4.  המתודה יכולה לקבל פרמטרים נוספים, המועברים בעת יצירת האובייקט.

**דוגמה:**

```python
class Car:
    def __init__(self, model: str, color: str):
        self.model = model    # אתחול התכונה model
        self.color = color    # אתחול התכונה color

car1: Car = Car("Tesla", "red") # נקראת המתודה __init__
print(f"Модель: {car1.model}, Цвет: {car1.color}") # פלט: Модель: Tesla, Цвет: red
```

**בתוצאה:**

*  בעת יצירת האובייקט `car1`, נקראת אוטומטית המתודה `__init__()`, אשר מאתחלת את התכונות `model` ו-`color`.

לפיכך, **אפשרות ב** היא הנכונה.
