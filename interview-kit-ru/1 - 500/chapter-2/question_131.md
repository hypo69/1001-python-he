### `question_131.md`

**שאלה 31.** מה מהבאות היא הדרך הנכונה ליישם מבנה דמוי switch-case בפייתון, בהתחשב בכך שפייתון אינה תומכת ב-switch-case ישירות?

א. להשתמש בשרשרת `if-elif-else`.
ב. להשתמש במילון לצורך מיפוי ערכים לפונקציות.
ג. להשתמש במספר אופרטורי `if` ברצף.
ד. אפשרויות א' וב'.

**תשובה נכונה: ד**

**הסבר:**

בפייתון אין אופרטור `switch-case` כמו בשפות תכנות אחרות (לדוגמה, ++C או Java). עם זאת, ניתן ליישם פונקציונליות דומה באמצעות מבנים אחרים.

*   **אפשרות א'**: שרשרת `if-elif-else` היא אחת מהדרכים הסטנדרטיות ליישום לוגיקה הדומה ל-`switch-case`. כל `elif` בודק את התנאי שלו, וכאשר אחד התנאים הופך ל-`True`, מבוצע בלוק הקוד המתאים.
*   **אפשרות ב'**: שימוש במילון לצורך מיפוי ערכים לפונקציות הוא דרך נוספת ליישם מבנה דמוי switch-case. בגישה זו, הערך משמש כמפתח במילון, והפונקציה (או הפעולה) המתאימה נקראת במקרה של התאמה.

נבחן בפירוט כיצד ניתן ליישם זאת:

**דוגמה עם `if-elif-else`:**

```python
command: str = "утро"

if command == "утро":
    print("Доброе утро!")
elif command == "день":
    print("Добрый день!")
elif command == "вечер":
    print("Добрый вечер!")
else:
    print("Неизвестное время суток.")
```

**דוגמה עם מילון:**

```python
def greet_morning():
    print("Доброе утро!")

def greet_afternoon():
    print("Добрый день!")

def greet_evening():
    print("Добрый вечер!")

def greet_default():
    print("Неизвестное время суток.")

command_map: dict[str, callable] = {
    "утро": greet_morning,
    "день": greet_afternoon,
    "вечер": greet_evening
}

command: str = "день"
action = command_map.get(command, greet_default) # Если нет ключа, вызывается greet_default
action() # Вывод: Добрый день!
```

כעת ננתח את אפשרויות התשובה:

*   **אפשרות א'** נכונה, מכיוון ששרשרת `if-elif-else` היא גישה סטנדרטית.
*   **אפשרות ב'** גם נכונה, מכיוון ששימוש במילון הוא דרך חלופית ליישום.
*   **אפשרות ג'** אינה הפתרון המיטבי, מכיוון שאופרטורי `if` עוקבים אינם מספקים קשר לוגי כמו ב-`switch-case`.
*   **אפשרות ד'** נכונה, מכיוון שהיא כוללת את שתי הגישות הישימות (א' וב').

לפיכך, **אפשרות ד'** היא התשובה הנכונה, שכן הן שימוש ב-`if-elif-else` והן שימוש במילון למיפוי ערכים לפונקציות הן דרכים לדמות את ה-`switch-case`.
