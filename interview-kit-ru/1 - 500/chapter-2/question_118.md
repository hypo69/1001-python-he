**שאלה 118.** כיצד פועלת לולאת `while True` בפייתון?

- A. היא יוצרת לולאה אינסופית שתרוץ ללא הגבלה, אלא אם תופסק על ידי האופרטור `break` או עקב שגיאה.
- B. היא בודקת את התנאי `True` פעם אחת ומסתיימת לאחר האיטרציה הראשונה.
- C. היא מתפקדת כלולאה עם איטרציה אחת, בדומה לאופרטור `if`.
- D. זוהי שגיאת תחביר, מכיוון ש-`True` אינו תנאי תקין.

**תשובה נכונה: A**

**הסבר:**

לולאת `while True` בפייתון היא דרך ליצור לולאה אינסופית. היא ממשיכה לרוץ עד שהיא מסתיימת באופן מפורש באמצעות `break` או כאשר מתרחשת שגיאה בלתי מטופלת.

*   **אפשרות A** נכונה: `while True` יוצרת לולאה שרצה ללא הגבלה, אלא אם תופסק באופן יזום.

*   **אפשרות B** אינה נכונה: הלולאה לא תסתיים לאחר איטרציה אחת.

*   **אפשרות C** אינה נכונה: `while True` אינה מתנהגת כמו `if`.
*   **אפשרות D** אינה נכונה: `True` הוא תנאי תקין בהחלט, ולא יגרום לשגיאת תחביר.

**כיצד פועלת `while True`:**

1.  התנאי `True` תמיד נכון.
2.  לולאת `while` רצה ללא הגבלה, אלא אם תופסק באופן מפורש על ידי האופרטור `break` או כאשר מתרחשת חריגה שתסיים את ריצת התוכנית.

**שימושים ב-`while True`:**

1.  **המתנה לאירועים:** משמש כאשר התוכנית צריכה להמתין להתרחשות אירוע כלשהו, למשל קלט משתמש.
2.  **לולאות עיבוד נתונים:** לעיבוד זרם נתונים עד לסיומו.
3.  **יישומי שרת:** בתוכנות הפועלות במצב שרת ומטפלות בבקשות.
4.  **יצירת לולאות מבוקרות:** כאשר יש צורך לבצע לולאה עד שתנאי מורכב כלשהו מתקיים.

**דוגמה:**

```python
import time

x: int = 0
while True:
  x += 1
  print(f"איטרציה: {x}") # תרגום טקסט הפלט, מבנה השורה נשמר
  time.sleep(0.5) # השהיה לצורך הדגמה
  if x >= 5:
     break

# פלט:
# איטרציה: 1
# איטרציה: 2
# איטרציה: 3
# איטרציה: 4
# איטרציה: 5
```
**תוצאה:**
הלולאה רצה עד שהתנאי `x>=5` הופך להיות `True`, ואז נקרא האופרטור `break`, שמפסיק את הלולאה.

לפיכך, אפשרות **A** היא התשובה הנכונה.
