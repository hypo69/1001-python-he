### `question_169.md`

**שאלה 69.** מה עושה המתודה `__repr__()` במחלקה בפייתון?

A. היא יוצרת מופע חדש של האובייקט.
B. היא מחזירה ייצוג מחרוזתי של האובייקט, המיועד למפתחים.
C. היא קוראת לפונקציית הבנאי.
D. היא משמשת להשוואת שני מופעים של האובייקט.

**תשובה נכונה: B**

**הסבר:**

המתודה `__repr__()` בפייתון משמשת להחזרת ייצוג מחרוזתי "רשמי" של אובייקט. ייצוג זה מיועד למפתחים ולניפוי שגיאות.

*   **אפשרות A** אינה נכונה: ליצירת מופע חדש משתמשים במתודה `__new__()` או בקריאה לבנאי (לדוגמה, `MyClass()`).
*   **אפשרות B** נכונה: `__repr__()` מחזירה ייצוג מחרוזת של האובייקט, המיועד למפתחים.
*   **אפשרות C** אינה נכונה: המתודה `__init__()` היא הבנאי, לא `__repr__()`.
*   **אפשרות D** אינה נכונה: להשוואת אובייקטים משתמשים במתודות אחרות, כגון `__eq__()`, `__lt__()` וכו'.

**כיצד פועלת `__repr__()`:**

1.  מתודה זו נקראת כאשר אובייקט מודפס בקונסולת הפייתון האינטראקטיבית, או כאשר משתמשים בפונקציה `repr()`.
2.  היא צריכה להחזיר מחרוזת שניתן להשתמש בה כדי ליצור אובייקט עם אותן תכונות.
3.  המחרוזת צריכה להיות חד-משמעית ולהכיל מידע מלא לשחזור האובייקט.

**דוגמה:**

```python
class Point:
    def __init__(self, x: int, y: int):
        self.x = x
        self.y = y

    def __repr__(self):
        return f"Point(x={self.x}, y={self.y})"

point: Point = Point(3, 5)
print(point) # פלט: <__main__.Point object at 0x...> - (נקראת __str__, אם היא לא מוגדרת אזי __repr__)
print(repr(point)) # פלט: Point(x=3, y=5) (נקראת __repr__)
point # פלט: Point(x=3, y=5) (נקראת __repr__)
```
**בתוצאה:**
* אם המתודה `__str__()` אינה מוגדרת במחלקה ומשתמשים ב-`print()`, פייתון יקרא אוטומטית ל-`__repr__()`.
*   בעת קריאה ל-`repr()` או בעת הדפסה ישירה של אובייקט במצב אינטראקטיבי, נקראת המתודה `__repr__()`, אשר מחזירה מחרוזת המייצגת את האובייקט.

לפיכך, **אפשרות B** היא התשובה הנכונה.
