**שאלה 186.** מה עושה האופרטור `del` בפייתון?

א. הוא מסיר איבר מרשימה לפי ערך.
ב. הוא מסיר את האיבר האחרון מרשימה ומחזיר אותו.
ג. הוא מסיר משתנה או איבר ממבנה נתונים.
ד. הוא משמש לסיום ביצוע לולאה.

**התשובה הנכונה: C**

**הסבר:**

האופרטור `del` בפייתון משמש למחיקת אובייקטים, משתנים או איברים ממבני נתונים (לדוגמה, מרשימה, ממילון או מ-tuple).

*   **אפשרות א** אינה נכונה: למחיקה לפי ערך משתמשים בשיטה `remove()`.
*   **אפשרות ב** אינה נכונה: למחיקה והחזרת האיבר האחרון משתמשים ב-`pop()`.
*   **אפשרות ג** נכונה: `del` משמש למחיקת אובייקטים, משתנים או איברים ממבני נתונים.
*   **אפשרות ד** אינה נכונה: האופרטור `break` משמש לסיום ביצוע לולאה.

**כיצד פועל `del`:**

1.  `del variable`: מוחק משתנה. לאחר מכן, המשתנה לא יהיה נגיש.
2.  `del list[index]`: מוחק איבר לפי האינדקס המצוין מהרשימה.
3.  `del dict[key]`: מוחק איבר לפי המפתח המצוין מהמילון.
4.  `del tuple[index]`: לא ניתן למחוק איברים מ-tuple.

**דוגמאות:**

```python
# מחיקת משתנה
x: int = 10
del x
# print(x) # NameError: name 'x' is not defined


my_list: list[int] = [1, 2, 3, 4, 5]
del my_list[2]  # מחיקת איבר לפי אינדקס
print(my_list) # פלט: [1, 2, 4, 5]


my_dict: dict[str, int] = {"a": 1, "b": 2, "c": 3}
del my_dict["b"] # מחיקת איבר לפי מפתח
print(my_dict) # פלט: {'a': 1, 'c': 3}
```

**לסיכום:**
* האופרטור `del` מוחק משתנה, איבר מרשימה או ערך במילון.

לפיכך, **אפשרות ג** מהווה את התשובה הנכונה.
