### `question_165.md`

**שאלה 65.** מהו ההבדל העיקרי בין המתודה `append()` לבין המתודה `extend()` בעת שימוש בהן עם רשימות בפייתון?

A.  המתודה `append()` מוסיפה איברים מרשימה אחת לרשימה אחרת, בעוד שהמתודה `extend()` מוסיפה רק איבר אחד לסוף הרשימה.
B.  המתודה `append()` מוסיפה איבר יחיד לסוף הרשימה, בעוד שהמתודה `extend()` מוסיפה איברים מאובייקט איטרבילי לסוף הרשימה.
C.  המתודה `append()` מוסיפה איברים לתחילת הרשימה, בעוד שהמתודה `extend()` מוסיפה לסוף.
D.  המתודה `append()` יוצרת רשימה חדשה, בעוד שהמתודה `extend()` משנה את הרשימה הקיימת.

**תשובה נכונה: B**

**הסבר:**

המתודות `append()` ו-`extend()` משמשות להוספת איברים לסוף רשימה, אך הן פועלות באופן שונה.

*   **אפשרות A** אינה נכונה: `append()` מוסיפה איבר יחיד, בעוד ש-`extend()` מוסיפה מספר איברים.
*   **אפשרות B** נכונה: `append()` מוסיפה *איבר יחיד*, `extend()` מוסיפה *איברים* מאובייקט איטרבילי.
*   **אפשרות C** אינה נכונה: שתי המתודות מוסיפות איברים לסוף הרשימה, לא לתחילתה.
*   **אפשרות D** אינה נכונה: שתי המתודות משנות את הרשימה הקיימת, ואינן יוצרות רשימה חדשה.

**כיצד פועלות `append()` ו-`extend()`:**

1.  `append(element)`: מוסיפה את `element` כישות אחת שלמה לסוף הרשימה. אם `element` הוא רשימה, הוא יתווסף כרשימה מקוננת.
2.  `extend(iterable)`: מוסיפה את האיברים מתוך האובייקט האיטרבילי (רשימה, tuple, מחרוזת, וכו') לסוף הרשימה, והופכת אותם לחלק מהרשימה הנוכחית.

**דוגמה:**

```python
list1: list[int] = [1, 2, 3]
list2: list[int] = [4, 5, 6]

# דוגמה ל-append()
list1.append(list2)
print(f"list1 אחרי append: {list1}") # פלט: list1 אחרי append: [1, 2, 3, [4, 5, 6]]


list3: list[int] = [1, 2, 3]
list4: list[int] = [4, 5, 6]
# דוגמה ל-extend()
list3.extend(list4)
print(f"list3 אחרי extend: {list3}") # פלט: list3 אחרי extend: [1, 2, 3, 4, 5, 6]

```

**בתוצאה:**

*   המתודה `append()` מוסיפה את `list2` כאיבר אחד לסוף `list1`, מה שמוביל לקינון של הרשימה.
*   המתודה `extend()` מוסיפה את האיברים מתוך `list4` לסוף `list3` בנפרד.

לפיכך, **אפשרות B** היא התשובה הנכונה.
