### `question_151.md`

**שאלה 51.** מה עושים המתודות `__str__` ו-`__repr__` במחלקות בפייתון ומתי כל אחת מהן נקראת?

א. המתודה `__str__` מחזירה ייצוג של האובייקט המיועד לפיתוח, והמתודה `__repr__` מחזירה ייצוג למשתמש הקצה.
ב. המתודה `__str__` מחזירה ייצוג של האובייקט המיועד למשתמש הקצה, והמתודה `__repr__` מחזירה ייצוג למטרות איתור באגים.
ג. המתודה `__str__` משמשת להשוואת שני אובייקטים, והמתודה `__repr__` משמשת לגיבובם.
ד. המתודה `__str__` משמשת ליצירת עותק של האובייקט, והמתודה `__repr__` למחיקת אובייקט.

**תשובה נכונה: B**

**הסבר:**

בפייתון, המתודות המיוחדות `__str__` ו-`__repr__` אחראיות על ייצוג מחרוזתי של אובייקטים, אך משמשות בהקשרים שונים.

*   **`__str__()`:**
    *   מתודה זו נקראת כאשר יש צורך לקבל ייצוג מחרוזתי "לא רשמי" או "קריא לבני אדם" של האובייקט.
    *   זה קורה כאשר האובייקט מועבר לפונקציות `print()` או `str()`.
    *   הוא מיועד למשתמש הקצה.
*   **`__repr__()`:**
    *   מתודה זו נקראת לקבלת ייצוג מחרוזתי "רשמי" או "לא חד משמעי" של האובייקט.
    *   ייצוג זה אמור להכיל את כל הנתונים הדרושים ליצירת עותק של האובייקט.
    *   הוא משמש לאיתור באגים ולרישום (לוגינג).
    *   מתודה זו נקראת כאשר האובייקט מוצג במפרש פייתון (ללא `print`).

כעת נבחן את אפשרויות התשובה:

*   **אפשרות א** אינה נכונה: תפקידי המתודות `__str__` ו-`__repr__` מתוארים בסדר הפוך.
*   **אפשרות ב** נכונה: `__str__` מיועד לייצוג האובייקט עבור המשתמש, `__repr__` למטרות איתור באגים.
*   **אפשרות ג** אינה נכונה: להשוואה משתמשים במתודות מיוחדות אחרות (`__eq__` וכו').
*   **אפשרות ד** אינה נכונה: על יצירת עותקים אחראית המתודה `__copy__`, ועל מחיקת אובייקט המתודה `__del__`.

**דוגמה:**

```python
class Point:
    def __init__(self, x: int, y: int):
        self.x = x
        self.y = y

    def __str__(self):
      return f"Point({self.x}, {self.y})"

    def __repr__(self):
      return f"Point(x={self.x}, y={self.y})"


point: Point = Point(3, 5)
print(point)  # Output: Point(3, 5)  (__str__ is called)
print(str(point)) # Output: Point(3, 5) (__str__ is called)
point  # Output: Point(x=3, y=5)  (__repr__ is called)
```

**תוצאה:**
*   `print(point)` ו-`str(point)` קוראים ל-`__str__()` כדי לקבל את הייצוג ה"קריא לבני אדם".
*   פלט פשוט של המשתנה `point` קורא למתודה `__repr__()` לקבלת ייצוג האובייקט למטרות איתור באגים.

לפיכך, **אפשרות ב** היא התשובה הנכונה.
