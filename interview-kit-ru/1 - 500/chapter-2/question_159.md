**שאלה 59.** מהי מטרת השימוש בבלוקים try ו-except בפייתון?

A. לשיפור קריאות הקוד.
B. להגברת מהירות ביצוע הקוד.
C. לטיפול בחריגות (Exceptions) ומניעת סיום עבודה בלתי תקין של התוכנית.
D. ליצירת איטרטורים.

**תשובה נכונה: C**

**הסבר:**

בלוקים try ו-except משמשים לטיפול בחריגות בפייתון. זהו מנגנון חשוב ליצירת תוכניות אמינות, המסוגלות לטפל בצורה תקינה בשגיאות במהלך ריצה.

*   **אפשרות A** שגויה: try-except אינם משמשים לשיפור קריאות הקוד.
*   **אפשרות B** שגויה: try-except אינם מאיצים את הקוד.
*   **אפשרות C** נכונה: try-except משמשים ליירוט חריגות (Exceptions) וטיפול תקין בשגיאות.
*   **אפשרות D** שגויה: ליצירת איטרטורים משתמשים במחוללים (generators) (yield) או במחלקות המממשות את פרוטוקול האיטרטור.

**כיצד פועל try-except:**

1.  קוד שעלול באופן פוטנציאלי לגרום לחריגה ממוקם בבלוק try.
2.  אם במהלך ביצוע הקוד בבלוק try מתרחשת חריגה, פייתון מפסיק את ביצוע בלוק try ומחפש בלוק except מתאים לטיפול בחריגה.
3.  אם נמצא בלוק except מתאים, הקוד שבתוכו מתבצע, מה שמאפשר לטפל בשגיאה בצורה תקינה. אם לא, התוכנית תסתיים עם חריגה לא מטופלת.
4.  אם לא היו שגיאות בבלוק try, הקוד בבלוק except אינו מתבצע.

**דוגמה:**

```python
try:
    result: int = 10 / 0
except ZeroDivisionError:
    print("אירעה שגיאה: חלוקה באפס!")

try:
    result: int = int("abc")
except ValueError as e:
   print(f"אירעה שגיאה: {e}")
```

**התוצאה:**

* בדוגמה הראשונה, בלוק try גורם לחריגה מסוג ZeroDivisionError, המטופלת בבלוק except.
* בדוגמה השנייה, בלוק try גורם לחריגה מסוג ValueError, המטופלת בבלוק except, עם אפשרות לקבל את ההודעה.

לסיכום, **אפשרות C** היא התשובה הנכונה.
