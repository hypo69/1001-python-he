### `question_146.md`

**שאלה 46.** מהו ייעוד השיטה `pop()` כאשר היא מיושמת על רשימה בפייתון?

א. היא מוסיפה איבר לתחילת הרשימה.
ב. היא מוסיפה איבר לסוף הרשימה.
ג. היא מסירה איבר מהרשימה לפי ערכו.
ד. היא מסירה ומחזירה איבר מהרשימה לפי האינדקס המצוין (או את האיבר האחרון, אם לא צוין אינדקס).

**תשובה נכונה: ד**

**הסבר:**

השיטה `pop()` בפייתון משמשת להסרת איבר מרשימה לפי אינדקס מצוין, והיא מחזירה את האיבר שהוסר. אם לא צוין אינדקס, השיטה `pop()` מסירה ומחזירה את האיבר האחרון ברשימה.

*   **אפשרות א** אינה נכונה: להוספת איבר לתחילת רשימה משתמשים בשיטה `insert()` או ב-`appendleft()` מהמודול `collections.deque`.
*   **אפשרות ב** אינה נכונה: להוספת איבר לסוף רשימה משתמשים בשיטה `append()`.
*   **אפשרות ג** אינה נכונה: להסרה לפי ערך משתמשים בשיטה `remove()`.
*   **אפשרות ד** נכונה: `pop()` מסירה ומחזירה איבר לפי אינדקס או את האיבר האחרון אם לא צוין אינדקס.

**כיצד עובדת `pop()`:**

1.  `pop(index)`: מסירה את האיבר לפי האינדקס המצוין ומחזירה אותו.
2.  `pop()`: מסירה את האיבר האחרון ברשימה ומחזירה אותו.
3.  אם הרשימה ריקה ונקראת השיטה `pop()`, תיגרם שגיאת `IndexError`.

**דוגמה:**

```python
my_list: list[int] = [10, 20, 30, 40]

# הסרת האיבר האחרון
last_element: int = my_list.pop()
print(f"איבר שהוסר: {last_element}, רשימה לאחר ההסרה: {my_list}")
# פלט: איבר שהוסר: 40, רשימה לאחר ההסרה: [10, 20, 30]


# הסרת איבר לפי אינדקס
removed_element: int = my_list.pop(1)
print(f"איבר שהוסר: {removed_element}, רשימה לאחר ההסרה: {my_list}")
# פלט: איבר שהוסר: 20, רשימה לאחר ההסרה: [10, 30]

# ניסיון להסיר איבר מרשימה ריקה (יגרום לשגיאה)
# my_list = []
# my_list.pop() # IndexError: pop from empty list
```

**לסיכום:**
*   `pop()` יכולה להסיר ולהחזיר את האיבר האחרון או איבר לפי אינדקס.
*   אם לא צוין אינדקס, מוסר ומוחזר האיבר האחרון.

לפיכך, **אפשרות ד** היא הנכונה.
