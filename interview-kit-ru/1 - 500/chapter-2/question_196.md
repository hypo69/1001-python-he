**שאלה 196.** מה יקרה אם תנסה להשתמש בפקודת ה-`break` מחוץ ללולאה ב-Python?

א. התוכנית תסתיים בשגיאה `SyntaxError`.
ב. התוכנית תמשיך את הביצוע, תוך התעלמות מפקודת ה-`break`.
ג. פקודת ה-`break` תפורש כ-`pass`.
ד. שורת הקוד הבאה לאחר הלולאה הקרובה ביותר תבוצע.

**תשובה נכונה: א**

**הסבר:**

פקודת ה-`break` ב-Python מיועדת לשימוש בתוך לולאות (`for` ו-`while`). אם תנסה להשתמש בה מחוץ ללולאה, Python תוציא שגיאה מסוג `SyntaxError`.

*   **אפשרות א** נכונה: פקודת ה-`break` מחוץ ללולאה תגרום לשגיאת `SyntaxError`.
*   **אפשרות ב** אינה נכונה: Python לא תתעלם מ-`break`.
*   **אפשרות ג** אינה נכונה: `break` אינה מפורשת כ-`pass`, יש להן מטרות שונות.
*   **אפשרות ד** אינה נכונה: יציאה מהלולאה לא תתרחש, מכיוון ש-`break` מופעלת מחוץ ללולאה.

**אופן פעולת `break`:**

1.  פקודת ה-`break` משמשת ליציאה מיידית מהלולאה שבה היא נמצאת.
2.  אם היא משמשת מחוץ ללולאה, Python אינה יודעת איזו לולאה יש להפסיק, ומוציאה שגיאה.

**דוגמה:**

```python
# Example of incorrect break usage
try:
  break  # SyntaxError: 'break' outside loop
except SyntaxError as e:
  print(f"Ошибка! {e}") # Output: Ошибка! 'break' outside loop
```

**כתוצאה מכך:**

*   בעת שימוש ב-`break` מחוץ ללולאה, נזרקת חריגה מסוג `SyntaxError`.

לפיכך, **אפשרות א** היא הנכונה.
