**שאלה 21.** מהי ההשפעה של הצבת אופרטור `return` בתוך לולאת `for` בפונקציה?

-   א. זה גורם לפונקציה לשלוח ערך חזרה ומסיים מיידית את הלולאה ואת הפונקציה עצמה.
-   ב. זה משהה את ביצוע הפונקציה, בהמתנה לקיום תנאי מסוים.
-   ג. זה מחזיר את הבקרה לתחילת הלולאה עבור האיטרציה הבאה.
-   ד. זה פועל כאופרטור `continue`, מדלג על יתרת האיטרציות.

**תשובה נכונה: א**

**הסבר:**

האופרטור `return` ב-Python משמש לסיום ביצוע פונקציה ולהחזרת ערך ממנה. אם האופרטור `return` נתקל בתוך לולאת `for`, ביצוע הלולאה והפונקציה נפסק מיידית.

*   **אפשרות א** נכונה: `return` מחזיר ערך מהפונקציה, ובה בעת גם קוטע את הלולאה שבה הוא נמצא.
*   **אפשרות ב** אינה נכונה: `return` אינו משהה את ביצוע הפונקציה. לשם כך משתמשים ב-`yield`.
*   **אפשרות ג** אינה נכונה: למעבר לאיטרציה הבאה של הלולאה משתמשים באופרטור `continue`, ולא ב-`return`.
*   **אפשרות ד** אינה נכונה: `return` אינו מדלג על איטרציות. לדילוג משתמשים באופרטור `continue`.

**כיצד פועל `return` בתוך לולאת `for`:**

1.  לולאת `for` מתחילה ביצוע.
2.  עם ההגעה לאופרטור `return`, Python מסיים מיידית את ביצוע הפונקציה שבה נמצאת הלולאה.
3.  אם `return` מחזיר ערך, אז ערך זה מוחזר מהפונקציה.
4.  האיטרציות הנותרות של הלולאה, וכן כל הקוד בפונקציה אחרי `return`, אינם מבוצעים.

**דוגמה:**

```python
def find_even_number(numbers: list[int]) -> int | None:
    """פונקציה המוצאת את המספר הזוגי הראשון ברשימה."""
    for number in numbers:
        if number % 2 == 0:
            print(f"Найдено четное число: {number}")
            return number # יציאה מהפונקציה מיד לאחר איתור מספר זוגי
    return None # אם אף מספר אינו זוגי, יחזיר None

result1 = find_even_number([1, 3, 5, 7, 9])
print(f"Результат 1: {result1}") # פלט: Результат 1: None

result2 = find_even_number([1, 2, 3, 4, 5])
print(f"Результат 2: {result2}")
# פלט: Найдено четное число: 2
# פלט: Результат 2: 2
```

**בתוצאה:**
* בשיחה הראשונה של `find_even_number`, מכיוון שאין מספרים זוגיים, הלולאה תסתיים, ויוחזר `None`.
* בשיחה השנייה של `find_even_number`, הלולאה נקטעת באיבר השני (2) בעת ביצוע `return number`, ומוחזר המספר 2.

לפיכך, **אפשרות א** היא התשובה הנכונה.
