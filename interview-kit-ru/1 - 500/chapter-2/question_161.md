**שאלה 61.** מהי המשמעות של `None` בפייתון?

A. זהו ערך אפס בחישובים נומריים.
B. זהו טיפוס נתונים מיוחד שניתן להקצות רק למשתני מחרוזת.
C. זהו ערך המייצג היעדר ערך או ערך ריק במשתנה.
D. זהו סמן המשמש להגדרת לולאה אינסופית.

**תשובה נכונה: C**

**הסבר:**

בפייתון, `None` הוא קבוע מיוחד המייצג היעדר ערך או ערך ריק. זהו אובייקט מטיפוס `NoneType`.

*   **אפשרות A** אינה נכונה: `None` אינו ערך נומרי אפס.
*   **אפשרות B** אינה נכונה: `None` אינו מוגבל למשתני מחרוזת בלבד, ניתן להקצותו לכל טיפוס.
*   **אפשרות C** נכונה: `None` משמש לייצוג היעדר ערך.
*   **אפשרות D** אינה נכונה: עבור לולאה אינסופית משתמשים ב-`while True`.

**דוגמאות לשימוש ב-`None`:**

1.  **ערך ברירת מחדל:** משמש כערך ברירת מחדל עבור פרמטרים של פונקציה.
2.  **ערך מוחזר:** פונקציה יכולה להחזיר `None` אם אין ערך מתאים להחזרה.
3.  **ייצוג היעדר ערך:** משמש לייצוג היעדר ערך במקום בו נדרש משתנה.

**דוגמה:**

```python
def my_function(x: int | None = None):
    if x is None:
        # פלט: ערך x לא הוגדר
        print("Значение x не задано")
        return None
    # פלט: ערך x: 5
    print(f"Значение x: {x}")
    return x * 2

# פלט: ערך x לא הוגדר \n None
print(my_function())
# פלט: ערך x: 5 \n 10
print(my_function(5))
```

**תוצאה:**

*   בדוגמה לעיל, הפונקציה `my_function()` בודקת האם הועבר ערך עבור `x`, ואם לא, היא מציגה הודעה ומחזירה `None`.
*   אם הועבר ערך עבור `x`, הפונקציה תציג אותו ותחזיר את תוצאת החישוב.

לפיכך, **אפשרות C** היא התשובה הנכונה.
