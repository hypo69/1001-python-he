**שאלה 40.** בפייתון, איזו מילה שמורה משמשת ליציאה מלולאה לפני שהיא מסיימת את כל האיטרציות?

A. `exit`
B. `stop`
C. `break`
D. `continue`

**תשובה נכונה: C**

**הסבר:**

המילה השמורה `break` משמשת בפייתון ליציאה מוקדמת מלולאה, בין אם זו לולאת `for` או לולאת `while`.

*   **אפשרות A** אינה נכונה: `exit` היא פונקציה שמסיימת את ריצת התוכנית כולה, לא רק את הלולאה.
*   **אפשרות B** אינה נכונה: `stop` אינה מילה שמורה המשמשת לקטיעת לולאה.
*   **אפשרות C** נכונה: `break` משמשת ליציאה מיידית מהלולאה.
*   **אפשרות D** אינה נכונה: `continue` משמשת למעבר לאיטרציה הבאה של הלולאה, לא לקטיעתה.

**אופן פעולת `break`:**

1.  כאשר נתקלים באופרטור `break`, האיטרציה הנוכחית של הלולאה מסתיימת באופן מיידי.
2.  ביצוע התוכנית ממשיך מהשורה הבאה של הקוד המופיעה אחרי הלולאה.

**דוגמה:**

```python
for i in range(10):
    if i == 5:
      print("הלולאה נקטעה כאשר i שווה ל-5") # הלולאה נקטעה, כאשר i שווה ל-5
      break
    print(i)
print("הלולאה הסתיימה") # הלולאה הסתיימה

# Вывод:
# 0
# 1
# 2
# 3
# 4
# הלולאה נקטעה כאשר i שווה ל-5
# הלולאה הסתיימה
```

**תוצאה:**
*   הלולאה תסתיים ברגע שהמשתנה i יהיה שווה ל-5.

לפיכך, **אפשרות C** היא התשובה הנכונה.
