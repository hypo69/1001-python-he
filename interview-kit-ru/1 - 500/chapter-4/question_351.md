### `question_351.md`

**שאלה 351.** מהו הייעוד של המתודה `__init__` במחלקת Python וכיצד היא קשורה ליצירה ולאתחול של אובייקטים, במיוחד כאשר אובייקט נוצר ממחלקה זו?

א. היא משמשת כמתודת השמדה (destructor) שמפנה את הזיכרון של האובייקט לאחר השימוש בו.
ב. היא משמשת לאתחול התכונות (attributes) של האובייקט ולהגדרת האובייקט בעת יצירתו.
ג. היא בודקת שגיאות במהלך יצירת המחלקה.
ד. היא מחזירה את ערך האובייקט בעת גישה אליו.

**תשובה נכונה: ב**

**הסבר:**

המתודה `__init__` במחלקת Python היא קונסטרקטור (constructor). היא נקראת אוטומטית בעת יצירת אובייקט חדש (מופע - instance) של מחלקה זו. מטרתה העיקרית היא לאתחל את התכונות (attributes) של האובייקט, כלומר להגדיר את המצב ההתחלתי של האובייקט החדש.

*   **אפשרות א שגויה:** ה-destructor, שמפנה משאבים וזיכרון, הוא המתודה `__del__`, ולא `__init__`.
*   **אפשרות ב נכונה:** המתודה `__init__` משמשת ל*אתחול* התכונות (attributes) של האובייקט, וכן עבור *הגדרות* נוספות של האובייקט בעת יצירתו.
*   **אפשרות ג שגויה:** בדיקת שגיאות מתבצעת לרוב בתוך המתודה `__init__` עצמה או במתודות אחרות של המחלקה, ולא באמצעות בדיקה מיוחדת בעת יצירת המחלקה.
*   **אפשרות ד שגויה:** המתודה `__init__` אינה מיועדת להחזרת ערך של האובייקט בעת גישה אליו.

**כיצד `__init__` עובדת:**

1.  בעת יצירת אובייקט חדש של מחלקה (לדוגמה, `obj = MyClass(arg1, arg2)`):
    *   Python מקצה זיכרון עבור האובייקט.
    *   קורא אוטומטית למתודה `__init__` של מחלקה זו.
    *   מעביר למתודה `__init__` את האובייקט עצמו (`self`) ואת הארגומנטים שצוינו בעת יצירת האובייקט (`arg1`, `arg2`).
    *   המתודה `__init__` משתמשת ב-`self` כדי להגדיר את הערכים ההתחלתיים של תכונות האובייקט.

**דוגמה:**

```python
class Person:
    def __init__(self, name, age):
        self.name = name  # אתחול התכונה name
        self.age = age    # אתחול התכונה age

person1 = Person("Alice", 30)
print(person1.name) # פלט: Alice
print(person1.age)  # פלט: 30

```

**לסיכום:**

המתודה `__init__` היא מתודת קונסטרקטור מיוחדת ב-Python, הנקראת אוטומטית בעת יצירת אובייקט של מחלקה. היא משמשת לאתחול תכונות האובייקט, הגדרת מצבו ההתחלתי והכנתו לעבודה עתידית.

לפיכך, אפשרות ב' היא הנכונה.
