### `question_302.md`

**שאלה 302.** במודול Python, כיצד ניתן להפוך פונקציות ומשתנים המוגדרים בתוכו לזמינים מחוץ למודול?

A. באמצעות הפונקציה `__init__`.
B. על ידי ייבוא המודול.
C. על ידי כתיבת `global` לפני המשתנה.
D. על ידי הגדרת בלוק `__main__`.

**תשובה נכונה: B**

**הסבר:**

כדי להפוך פונקציות ומשתנים המוגדרים במודול Python לזמינים מחוץ למודול, יש *לייבא* מודול זה בקובץ Python אחר. פעולה זו מאפשרת להשתמש בתוכן המודול על ידי פנייה אליו דרך שם המודול.

*   **אפשרות A אינה נכונה:** הפונקציה `__init__` משמשת לאתחול *אובייקטים* של מחלקה, ולא לייצוא אלמנטים של מודול.
*   **אפשרות B נכונה:** ייבוא המודול הופך את תוכנו לזמין.
*   **אפשרות C אינה נכונה:** מילת המפתח `global` הופכת משתנה לגלובלי *בתוך* המודול, אך אינה הופכת אותו לזמין מחוץ לו.
*   **אפשרות D אינה נכונה:** בלוק `__main__` מורץ רק כאשר המודול מופעל ישירות, ואינו משפיע על זמינותו בעת ייבוא.

**כיצד פועל ייבוא מודול:**

1.  השתמש באופרטור `import module_name` בקובץ Python אחר.
2.  Python מוצא את הקובץ `module_name.py`, טוען ומבצע את הקוד מתוך קובץ זה.
3.  נוצר אובייקט מודול, והוא הופך לזמין לשימוש באמצעות `module_name.element`.

**דוגמה:**

```python
# קובץ my_module.py
def my_function():
  return "Hello from my_module"

my_variable = 10

# קובץ main.py
import my_module

print(my_module.my_function())  # פלט: Hello from my_module
print(my_module.my_variable)  # פלט: 10
```

בדוגמה זו, לאחר ייבוא המודול `my_module` בקובץ `main.py`, הפונקציות והמשתנים המוגדרים בתוך `my_module.py` הופכים לזמינים לשימוש דרך `my_module.my_function` ו-`my_module.my_variable`.

**לסיכום:**

כדי להפוך פונקציות ומשתנים של מודול Python לזמינים מחוץ לו, יש להשתמש באופרטור `import` כדי לייבא מודול זה בקובץ Python אחר.

לפיכך, אפשרות B היא הנכונה.
