### `question_201.md`

**שאלה 201.** בפייתון, איזו מילת מפתח משמשת ליצירת גנרטור?

A. `return`
B. `yield`
C. `generator`
D. `yield from`

**תשובה נכונה: B**

**הסבר:**

מילת המפתח `yield` משמשת ליצירת גנרטורים בפייתון. גנרטורים הם פונקציות מיוחדות שיכולות להשהות את ביצוען ולהחזיר ערך, תוך שמירה על המצב הפנימי שלהן.

*   **אפשרות א'** אינה נכונה: `return` מסיימת את הפונקציה ומחזירה ערך בודד.
*   **אפשרות ב'** נכונה: `yield` משמשת ליצירת גנרטורים.
*   **אפשרות ג'** אינה נכונה: `generator` אינה מילת מפתח בפייתון.
*   **אפשרות ד'** אינה נכונה: `yield from` משמשת להאצלת איטרציות בגנרטורים.

**כיצד פועלת `yield`:**

1.  פונקציה המכילה `yield` הופכת לגנרטור.
2.  כאשר קוראים לפונקציית גנרטור, קוד הפונקציה אינו מתבצע מיידית.
3.  הגנרטור מחזיר אובייקט איטרטור, המנהל את ביצוע הפונקציה.
4.  כאשר נקראת שיטת `next()` של האיטרטור, קוד הפונקציה מתבצע עד לאופרטור `yield`.
5.  לאחר `yield`, הביצוע מושהה ומוחזר ערך.
6.  בקריאה הבאה לשיטת `next()`, הביצוע מתחדש מנקודת ההשהיה.

**דוגמה:**

```python
def my_generator(n: int):
    for i in range(n):
      yield i * 2 # השהיה והחזרת ערך

gen = my_generator(5)
print(next(gen)) # פלט: 0
print(next(gen)) # פלט: 2
print(next(gen)) # פלט: 4
print(next(gen)) # פלט: 6
print(next(gen)) # פלט: 8
# print(next(gen)) # StopIteration
for value in my_generator(3):
    print(value) # פלט:
              # 0
              # 2
              # 4

```
**לסיכום:**
* בפונקציה `my_generator`, מילת המפתח `yield` משהה את ביצוע הפונקציה ומחזירה את הערך הבא.
* באמצעות שימוש ב-`next()` ניתן לקבל את הערך הבא של הגנרטור.
* ניתן לבצע איטרציה על הגנרטור באמצעות לולאת `for`.

לפיכך, **אפשרות ב'** היא הנכונה.
