### `question_209.md`

**שאלה 209.** מהי הפלט של הקוד הבא?

```python
numbers: list[int] = [1, 2, 3, 4, 5]
for i, num in enumerate(numbers):
  if i % 2 == 0:
    continue
  print(num)
```

א. 1 2 3 4 5
ב. 2 4
ג. 1 3 5
ד. 2 3 4 5

**תשובה נכונה: ב'**

**הסבר:**

קוד זה משתמש בלולאת `for` עם הפונקציה `enumerate()` ובמשפט תנאי `if` עם האופרטור `continue`.

1.  **אתחול:** נוצרת רשימה בשם `numbers`.
2.  **לולאת `for` עם `enumerate()`:**
    *   הפונקציה `enumerate()` מחזירה זוגות של אינדקס-ערך עבור כל איבר ברשימה `numbers`.
    *   המשתנה `i` מייצג את אינדקס האיבר, והמשתנה `num` מייצג את ערך האיבר.
3.  **משפט תנאי `if`:**
    *   `if i % 2 == 0`: בודק אם האינדקס `i` הוא מספר זוגי.
    *  אם האינדקס זוגי, האופרטור `continue` מדלג על הדפסת האיבר הנוכחי ועובר לאיטרציה הבאה של הלולאה.
4. **פלט**: אחרת (אם האינדקס אינו זוגי), מודפס ערך `num`.

נתפרק את הלולאה צעד אחר צעד:
* `i = 0`, `num = 1`. התנאי `i % 2 == 0` הוא אמת, לכן מדלגים על ההדפסה.
* `i = 1`, `num = 2`. התנאי `i % 2 == 0` הוא שקר, מדפיסים את הערך `num = 2`.
* `i = 2`, `num = 3`. התנאי `i % 2 == 0` הוא אמת, לכן מדלגים על ההדפסה.
* `i = 3`, `num = 4`. התנאי `i % 2 == 0` הוא שקר, מדפיסים את הערך `num = 4`.
* `i = 4`, `num = 5`. התנאי `i % 2 == 0` הוא אמת, לכן מדלגים על ההדפסה.

כעת נבחן את אפשרויות התשובה:

*   **אפשרות א'** אינה נכונה: לא מודפסים כל המספרים.
*   **אפשרות ב'** נכונה: בלולאה מודפסים הערכים שהאינדקסים שלהם אינם זוגיים (2 ו-4).
*   **אפשרות ג'** אינה נכונה: מודפסים מספרים מאינדקסים שונים.
*   **אפשרות ד'** אינה נכונה: מודפסים ערכים שונים ולא בסדר הנכון.

**דוגמה:**

```python
numbers: list[int] = [1, 2, 3, 4, 5]
for i, num in enumerate(numbers):
  if i % 2 == 0:
    continue
  print(num)
```
# פלט:

# 4

**כתוצאה מכך:**

* מודפסים רק ערכי הרשימה שהאינדקס שלהם אינו זוגי.

לפיכך, **אפשרות ב'** היא התשובה הנכונה.
