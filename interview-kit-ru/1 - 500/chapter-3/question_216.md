### `question_216.md`

**שאלה 216.** איזו מההצהרות הבאות מתארת נכונה את השימוש באופרטור `yield` בתוך פונקציית Python?

א. האופרטור `yield` מסיים את ביצוע הפונקציה ומחזיר ערך.
ב. האופרטור `yield` משהה את ביצוע הפונקציה ומחזיר ערך, ובעת הקריאה הבאה, הפונקציה ממשיכה את ביצועה מהמקום בו הושעתה.
ג. האופרטור `yield` הופך את הפונקציה לשגרת משנה רב-תהליכית.
ד. האופרטור `yield` משמש לטיפול בחריגות.

**תשובה נכונה: ב**

**הסבר:**

מילת המפתח `yield` משמשת ב-Python ליצירת גנרטורים. גנרטור הוא פונקציה שיכולה להשהות את ביצועה ולהחזיר ערך, תוך שמירת מצבה לקריאה עתידית.

*   אפשרות א' שגויה: האופרטור `return` מסיים את ביצוע הפונקציה.
*   אפשרות ב' נכונה: האופרטור `yield` משהה את הביצוע ומחזיר ערך, תוך שמירת מצב הפונקציה.
*   אפשרות ג' שגויה: `yield` אינו הופך את הפונקציה לרב-תהליכית.
*   אפשרות ד' שגויה: `try-except` משמש לטיפול בחריגות.

**כיצד פועל `yield`:**

1.  כאשר פונקציית גנרטור נתקלת באופרטור `yield`, היא משהה את ביצועה ומחזירה את הערך שצוין אחרי `yield`.
2.  בעת הקריאה הבאה לפונקציית הגנרטור, ביצועה ימשיך מהמקום שבו עצרה (כלומר, אחרי האופרטור `yield`), והיא תשתמש בערכים של המשתנים שנשמרו קודם לכן.
3.  פונקציה יכולה להכיל מספר ביטויי `yield`.
4.  עם סיום הגנרטור (בהגעה לסוף הפונקציה או פגישה באופרטור `return`), תיזרק חריגה מסוג `StopIteration`.

**דוגמה:**

```python
def my_generator(n: int):
    for i in range(n):
        print(f"Перед yield: i = {i}")
        yield i * 2
        print(f"После yield: i = {i}") # יבוצע לאחר הקריאה הבאה
    print("Генератор закончил работу") # יבוצע לאחר סיום הלולאה

gen = my_generator(3)

print(next(gen)) # פלט: Перед yield: i = 0
                # פלט: 0
print(next(gen)) # פלט: После yield: i = 0
                #  פלט: Перед yield: i = 1
                # פלט: 2
print(next(gen)) # פלט: После yield: i = 1
                # פלט: Перед yield: i = 2
                # פלט: 4
print(next(gen)) # פלט: После yield: i = 2
                # פלט: Генератор закончил работу
                # StopIteration
```
**תוצאה:**

*   `yield` מחזיר ערך ומשהה את הפונקציה, תוך שמירת מצבה.
*   בכל קריאה ל-`next` הפונקציה ממשיכה את עבודתה מהמקום בו הושעתה.

לפיכך, **אפשרות ב'** היא הנכונה.
