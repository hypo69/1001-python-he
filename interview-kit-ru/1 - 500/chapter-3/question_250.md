### `question_250.md`

**שאלה 250.** מהו התפקיד של המתודה `__init__` במחלקת Python וכיצד היא קשורה ליצירה ולאתחול של אובייקטים, במיוחד כאשר אובייקט נוצר ממחלקה זו?

A. היא משמשת כמתודת הורס (destructor) שמפנה זיכרון של אובייקט לאחר השימוש בו.
B. היא משמשת לאתחול מאפייני האובייקט ולהגדרת האובייקט בעת יצירתו.
C. היא בודקת שגיאות במהלך יצירת המחלקה.
D. היא מחזירה את ערך האובייקט בעת גישה אליו.

**תשובה נכונה: B**

**הסבר:**

המתודה `__init__` במחלקת Python היא בנאי (constructor). היא נקראת באופן אוטומטי בעת יצירת אובייקט (מופע) חדש של מחלקה זו. מטרתה העיקרית היא לאתחל את מאפייני האובייקט, כלומר, להגדיר את המצב ההתחלתי של האובייקט החדש.

*   **אפשרות A אינה נכונה:** הורס, המפנה משאבים וזיכרון, הוא המתודה `__del__`, ולא `__init__`.
*   **אפשרות B נכונה:** המתודה `__init__` משמשת ל*אתחול* מאפייני האובייקט, וכן ל*הגדרות* נוספות של האובייקט בעת יצירתו.
*   **אפשרות C אינה נכונה:** בדיקת שגיאות מתבצעת בדרך כלל בתוך המתודה `__init__` עצמה או במתודות אחרות של המחלקה, ולא באמצעות בדיקה מיוחדת בעת יצירת המחלקה.
*   **אפשרות D אינה נכונה:** המתודה `__init__` אינה מיועדת להחזיר את ערך האובייקט בעת גישה אליו.

**כיצד פועלת `__init__`:**

1.  בעת יצירת אובייקט חדש של המחלקה (לדוגמה, `obj = MyClass(arg1, arg2)`):
    *   פייתון מקצה זיכרון עבור האובייקט.
    *   פייתון קורא אוטומטית למתודה `__init__` של מחלקה זו.
    *   פייתון מעביר למתודה `__init__` את האובייקט עצמו (`self`) ואת הארגומנטים שצוינו בעת יצירת האובייקט (`arg1`, `arg2`).
    *   המתודה `__init__` משתמשת ב-`self` כדי להגדיר ערכים התחלתיים למאפייני האובייקט.

**דוגמה:**

```python
class Person:
    def __init__(self, name, age):
        self.name = name  # אתחול המאפיין name
        self.age = age    # אתחול המאפיין age

person1 = Person("Alice", 30)
print(person1.name) # פלט: Alice
print(person1.age)  # פלט: 30

```

**כתוצאה מכך:**

המתודה `__init__` היא מתודת בנאי מיוחדת בפייתון, הנקראת אוטומטית בעת יצירת אובייקט של מחלקה. היא משמשת לאתחול מאפייני האובייקט, הגדרת מצבו ההתחלתי והכנתו להמשך עבודה.

לפיכך, אפשרות B היא הנכונה.
