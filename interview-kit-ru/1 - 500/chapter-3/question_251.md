### `question_251.md`

**שאלה 251.** כיצד פועל אופרטור `import` בפייתון בעת ייבוא מודול, ומה הוא מרמז ביחס למרחב השמות של התוכנית?

א. הוא יוצר כינוי גלובלי עבור המודול ומייבא אוטומטית את כל תוכנו למרחב השמות.
ב. הוא מייבא את המודול למרחב השמות הגלובלי, אך מגביל גישה לפונקציות ולמשתנים שלו.
ג. הוא טוען את המודול והופך את פונקציותיו לזמינות, באמצעות פנייה אליהן עם שם המודול כקידומת.
ד. הוא מסיר את כל ההפניות הקודמות למודול ממרחב השמות.

**תשובה נכונה: C**

**הסבר:**

אופרטור `import` בפייתון משמש לטעינת מודולים לתוכנית הנוכחית. כאשר אתה משתמש ב-`import module_name`, פייתון טוען את המודול המצוין ויוצר הפניה אליו במרחב השמות הנוכחי. לגישה לפונקציות, מחלקות ומשתנים של המודול יש להשתמש בשם המודול כקידומת.

*   **אפשרות A שגויה:** `import module_name` אינו יוצר כינוי גלובלי ואינו מייבא אוטומטית *הכל* למרחב השמות הנוכחי.
*   **אפשרות B שגויה:** `import` אינו *מגביל* גישה, אלא להיפך, *מאפשר גישה*, אך דורש שימוש בקידומת.
*   **אפשרות C נכונה:** `import` טוען את המודול ודורש שימוש בשם המודול כקידומת בעת פנייה לאלמנטים שלו.
*   **אפשרות D שגויה:** `import` אינו מסיר אף הפניה, אלא יוצר חדשות.

**כיצד פועל אופרטור import:**

1.  אופרטור `import module_name` מאתר את הקובץ `module_name.py` בנתיבי חיפוש המודולים.
2.  טוען ומבצע את הקוד מהקובץ `module_name.py`.
3.  יוצר במרחב השמות הנוכחי משתנה בשם `module_name`, המפנה למודול שנטען.
4.  לשימוש באלמנטים של המודול (פונקציות, מחלקות, משתנים) יש להשתמש בקידומת `module_name.element`, כאשר `element` הוא שם האלמנט.

**דוגמה:**

```python
# my_module.py
def my_function():
    return "Hello from my_module!"

value = 10

# main.py
import my_module

result = my_module.my_function() # גישה לפונקציה באמצעות הקידומת `my_module`
print(result)
print(my_module.value)
```
**מרחב השמות:**
אופרטור `import` אינו מזהם את מרחב השמות הנוכחי בשמות מהמודול. הוא שומר את אלמנטי המודול במרחב השמות שלו, שגישה אליו מתבצעת באמצעות הקידומת.

**לסיכום:**

אופרטור `import` טוען את המודול והופך את פונקציותיו ומשתניו לזמינים באמצעות קידומת עם שם המודול, תוך שמירה על ניקיון מרחב השמות הנוכחי.

לפיכך, אפשרות C היא הנכונה.
