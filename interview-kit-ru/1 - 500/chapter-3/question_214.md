**שאלה 214.** איזו תוצאה יודפס על ידי הקוד הבא?

```python
x: int = 10
y: int = 20
if x > 5:
    if y < 15:
        print("Оба условия верны")
    else:
        print("x больше 5, но y не меньше 15")
else:
    print("x не больше 5")
```

A.  `x אינו גדול מ-5`
B.  `שני התנאים מתקיימים`
C.  `x גדול מ-5, אך y אינו קטן מ-15`
D.  שגיאה

**תשובה נכונה: C**

**הסבר:**

קוד זה מדגים את פעולתם של ביטויי תנאי מקוננים if-else ב-Python.

1.  **אתחול משתנים**: ל-x מושם הערך 10, ול-y מושם הערך 20.
2.  **if החיצוני**: נבדק התנאי `x > 5`. מאחר ש-`x` שווה ל-10, התנאי הוא אמת.
3.  **if הפנימי**: מבוצע ביטוי התנאי המקונן `if y < 15`. מאחר ש-`y` שווה ל-20, תנאי זה הוא שקר.
4.  **else המקונן**: מאחר שה-`if` הפנימי הוא שקר, מבוצע בלוק הקוד תחת `else`, שידפיס את `"x больше 5, но y не меньше 15"`.

כעת נסתכל על אפשרויות התשובה:

*   **אפשרות A** אינה נכונה: הפלט `"x не больше 5"` היה מתקבל אילו התנאי `x > 5` היה שקר.
*   **אפשרות B** אינה נכונה: הפלט `"Оба условия верны"` היה מתקבל אילו גם `x > 5` וגם `y < 15` היו אמת.
*   **אפשרות C** נכונה: מתארת נכונה את הפלט, בהתחשב בכך שהתנאי `x > 5` הוא אמת, אך התנאי `y < 15` הוא שקר.
*   **אפשרות D** אינה נכונה: אין שגיאות תחביר בקוד.

**דוגמה:**

```python
x: int = 10
y: int = 20
if x > 5:
    if y < 15:
        print("Оба условия верны")
    else:
        print("x больше 5, но y не меньше 15") # פלט: x גדול מ-5, אך y אינו קטן מ-15
else:
    print("x не больше 5")
```

**לסיכום:**

*   התוכנית תדפיס למסך: `"x больше 5, но y не меньше 15"`.

לפיכך, **אפשרות C** היא התשובה הנכונה.
