**שאלה 255.** כיצד פועל האופרטור `import` ב-Python בעת ייבוא מודול ומה הוא מרמז בנוגע למרחב השמות של התוכנית?

א. הוא יוצר כינוי גלובלי עבור המודול ומייבא אוטומטית את כל תוכנו לתוך מרחב השמות.
ב. הוא מייבא את המודול למרחב השמות הגלובלי, אך מגביל את הגישה לפונקציות ולמשתנים שלו.
ג. הוא טוען את המודול והופך את פונקציותיו לזמינות, תוך פנייה אליהן באמצעות שם המודול כקידומת.
ד. הוא מסיר את כל ההפניות הקודמות למודול ממרחב השמות.

**תשובה נכונה: C**

**הסבר:**

האופרטור `import` ב-Python משמש לטעינת מודולים לתוכנית הנוכחית. כאשר משתמשים ב-`import module_name`, Python טוען את המודול המצוין ויוצר הפניה אליו במרחב השמות הנוכחי. כדי לגשת לפונקציות, מחלקות ומשתנים של המודול, יש להשתמש בשם המודול כקידומת.

*   **אפשרות א' אינה נכונה:** `import module_name` לא יוצר כינוי גלובלי ואינו מייבא אוטומטית *הכול* למרחב השמות הנוכחי.
*   **אפשרות ב' אינה נכונה:** `import` אינו *מגביל* גישה, אלא להפך, *מאפשר גישה*, אך דורש שימוש בקידומת.
*   **אפשרות ג' נכונה:** `import` טוען את המודול ודורש שימוש בשם המודול כקידומת בעת פנייה לרכיביו.
*   **אפשרות ד' אינה נכונה:** `import` אינו מסיר הפניות כלשהן, אלא יוצר חדשות.

**כיצד פועל האופרטור `import`:**

1.  האופרטור `import module_name` מוצא את הקובץ `module_name.py` בנתיבי חיפוש המודולים.
2.  טוען ומבצע את הקוד מהקובץ `module_name.py`.
3.  יוצר במרחב השמות הנוכחי משתנה בשם `module_name`, המפנה למודול שנטען.
4.  כדי להשתמש ברכיבי המודול (פונקציות, מחלקות, משתנים), יש להשתמש בקידומת `module_name.element`, כאשר `element` הוא שם הרכיב.

**דוגמה:**

```python
# my_module.py
def my_function():
    return "Hello from my_module!"

value = 10

# main.py
import my_module

result = my_module.my_function() # גישה לפונקציה דרך הקידומת `my_module`
print(result)  # פלט: Hello from my_module!
print(my_module.value) # פלט: 10
```
**מרחב שמות:**
האופרטור `import` אינו מזהם את מרחב השמות הנוכחי בשמות מהמודול. הוא שומר את רכיבי המודול במרחב השמות שלו עצמו, וגישה אליהם מתבצעת דרך קידומת.

**לסיכום:**

האופרטור `import` טוען את המודול והופך את פונקציותיו ומשתניו לזמינים באמצעות קידומת עם שם המודול, תוך שמירה על ניקיון מרחב השמות הנוכחי.

לפיכך, אפשרות C היא הנכונה.
