**שאלה 222.** מהי "הפשטה" (abstraction) בתכנות מונחה עצמים (תמ"ע)?

א. זו היכולת ליצור מחלקות חדשות המבוססות על מחלקות קיימות.
ב. זו היכולת של אובייקטים ממחלקות שונות להתנהג באופן אחיד באמצעות ממשק משותף.
ג. זהו עיקרון המאפשר להסתיר את פרטי המימוש של אובייקט ולספק גישה אליו דרך ממשק ציבורי.
ד. זהו מנגנון לטיפול בשגיאות וחריגים.

**תשובה נכונה: ג**

**הסבר:**

בתכנות מונחה עצמים (תמ"ע), הפשטה היא עקרון של הסתרת מימוש פנימי מורכב של אובייקט ומתן ממשק פשוט וברור לעבודה איתו. הפשטה מאפשרת לעבוד עם אובייקט מבלי לדעת את פרטי המבנה הפנימי שלו.

*   **אפשרות א** שגויה: זו ההגדרה של ירושה.
*   **אפשרות ב** שגויה: זו ההגדרה של פולימורפיזם.
*   **אפשרות ג** נכונה: הפשטה היא הסתרת מימוש פנימי ומתן גישה אליו דרך ממשק ציבורי.
*   **אפשרות ד** שגויה: זו הגדרה עבור בלוקים של try-except.

**כיצד פועלת הפשטה:**

1.  הפשטה מאפשרת להסתיר פרטי מימוש מורכבים של אובייקט, ולהותיר רק את הממשק הנחוץ לאינטראקציה.
2.  היא מאפשרת למפתחים לעבוד עם אובייקט ברמה גבוהה יותר, מבלי להתעמק בפרטים.
3.  היא תורמת למודולריות ומפשטת את התחזוקה והשינוי של קוד.
4. ב-Python, הפשטה יכולה להיות ממומשת דרך מחלקות מופשטות או ממשקים.

**דוגמה:**
```python
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self) -> float:
       """מתודה מופשטת"""
       pass
class Circle(Shape):
    def __init__(self, radius: int):
        self.radius = radius

    def area(self) -> float: # מממשים את מתודת המחלקה Shape
        """חישוב שטח העיגול."""
        return 3.14 * self.radius**2

class Square(Shape):
  def __init__(self, side: int):
    self.side = side

  def area(self) -> float: # מממשים את מתודת המחלקה Shape
    """חישוב שטח הריבוע"""
    return self.side**2

circle: Circle = Circle(5)
square: Square = Square(10)

def display_area(shape: Shape):
   print(f"Площадь: {shape.area()}") # קריאה פולימורפית למתודת area()

display_area(circle) # פלט: Площадь: 78.5
display_area(square) # פלט: Площадь: 100
```
**בתוצאה:**

*   אנו יכולים לעבוד עם `circle` ו-`square` באמצעות הממשק המשותף `shape` (המתודה `area`), וזאת מבלי לדעת את פרטי המימוש שלהם.

לפיכך, **אפשרות ג** היא התשובה הנכונה.
