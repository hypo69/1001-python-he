### `question_231.md`

**שאלה 231.** מהי התועלת העיקרית בשימוש במתודת `__init__()` במחלקות בפייתון וכיצד היא משפיעה על אתחול אובייקטים?

א. היא משמשת רק לאתחול משתני מחלקה, ולא משתני מופע.
ב. היא קוראת לעצמה אוטומטית בכל פעם שנוצר מופע של המחלקה.
ג. זוהי מתודת בנאי הנקראת בעת יצירת אובייקט חדש של המחלקה כדי לאתחל את משתני המופע.
ד. היא משמשת לירושת מאפיינים ממחלקת אב בעת יצירת אובייקט.

**תשובה נכונה: C**

**הסבר:**

מתודת `__init__()` בפייתון היא הבנאי של המחלקה. תפקידה העיקרי הוא לאתחל את *מאפייני המופע* (instance variables) של האובייקט בעת יצירתו.

*   **אפשרות A אינה נכונה:** `__init__` משמשת לאתחול משתני *מופע*, ולא רק משתני *מחלקה*.
*   **אפשרות B אינה נכונה:** `__init__` אינה קוראת לעצמה, היא נקראת אוטומטית על ידי פייתון בעת יצירת *אובייקט חדש* של המחלקה.
*   **אפשרות C נכונה:** מתודת `__init__()` היא בנאי והיא נקראת בעת יצירת אובייקט חדש. היא מאתחלת את *משתני המופע*.
*   **אפשרות D אינה נכונה:** ירושת מאפיינים ממחלקת אב מושגת באמצעות מנגנונים אחרים (ירושת מחלקות), ולא ישירות דרך מתודת `__init__`.

**כיצד פועל `__init__()`:**

1.  בעת יצירת אובייקט חדש של מחלקה (לדוגמה, `obj = MyClass(arg1, arg2)`):
    *   פייתון מקצה זיכרון עבור האובייקט החדש.
    *   קורא אוטומטית למתודת `__init__` של מחלקה זו.
    *   מעביר ל-`__init__` את האובייקט עצמו (`self`) ואת הארגומנטים שצוינו בעת יצירת האובייקט (`arg1`, `arg2`).
    *   `__init__` משתמש ב-`self` כדי להגדיר את הערכים של *מאפייני המופע* של האובייקט.

**דוגמה:**

```python
class Car:
    def __init__(self, model, color):
        self.model = model  # אתחול מאפיין model
        self.color = color  # אתחול מאפיין color

my_car = Car("Tesla Model S", "Red")
print(my_car.model)  # פלט: Tesla Model S
print(my_car.color)  # פלט: Red
```

בדוגמה זו, `__init__` מגדירה את המאפיינים `model` ו-`color` עבור כל *מופע ספציפי* של מחלקת `Car`.

**לסיכום:**

`__init__()` – זהו הבנאי של המחלקה, המשמש לאתחול מאפייני *המופע* של האובייקט בעת יצירתו, ומבטיח שכל אובייקט יתחיל במצב תקין ועם הנתונים הנדרשים.

לפיכך, אפשרות C היא הנכונה.
