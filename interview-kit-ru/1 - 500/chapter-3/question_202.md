### `question_202.md`

**שאלה 202.** מהו `__name__` בפייתון וכיצד ערכו תלוי באופן הפעלת הסקריפט?

A. זהו שם המודול, והוא תמיד `__main__`.
B. זהו שם הפונקציה הנוכחית, והוא תמיד זהה לשם הפונקציה.
C. זהו שם המחלקה והוא תמיד שם המחלקה.
D. זהו משתנה מיוחד ששווה ל-`__main__` כאשר הסקריפט מופעל ישירות, ולשם המודול אם הסקריפט מיובא.

**התשובה הנכונה: D**

**הסבר:**

`__name__` הוא משתנה מובנה בפייתון המציין כיצד הופעל הקובץ הנוכחי: כסקריפט ראשי או כמודול מיובא.

*   **אפשרות A** אינה נכונה: `__name__` אינו תמיד `__main__`.
*   **אפשרות B** אינה נכונה: `__name__` מכיל את שם המודול, לא את שם הפונקציה.
*  **אפשרות C** אינה נכונה: `__name__` מכיל את שם המודול, לא את שם המחלקה.
*  **אפשרות D** נכונה: `__name__` שווה ל-`__main__` אם הסקריפט מופעל כראשי.

**כיצד `__name__` פועל:**

1.  כאשר סקריפט פייתון מופעל ישירות, המפרש מקצה למשתנה `__name__` את הערך `"__main__"`.
2.  אם הסקריפט מיובא כמודול לסקריפט אחר, אזי למשתנה `__name__` מוקצה שם קובץ המודול (ללא הסיומת `.py`).
3.  זה מאפשר לבדוק האם הסקריפט מתבצע כראשי או כמודול מיובא.

**דוגמה:**

```python
# my_module.py
print(f"שם המודול: {__name__}") # פלט: שם המודול: __main__ אם הקובץ מופעל כסקריפט

def my_function():
    print("זו פונקציה מהמודול")
# main_script.py
import my_module # מייבאים את my_module
print(f"שם המודול: {__name__}") # פלט: שם המודול: __main__

my_module.my_function() # פלט: זו פונקציה מהמודול
```

**תוצאה:**
*   בעת הפעלת `my_module.py` באופן עצמאי, `__name__` יהיה שווה ל-`__main__`.
*   בעת הפעלת `main_script.py`, `__name__` יהיה שווה ל-`__main__`, ואילו במודול `my_module` המשתנה `__name__` יקבל את הערך `my_module`.

לפיכך, **אפשרות D** היא התשובה הנכונה.
