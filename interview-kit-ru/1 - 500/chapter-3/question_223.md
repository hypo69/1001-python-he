**שאלה 223.** מהי "אבסטרקציה" בתכנות מונחה עצמים (OOP)?

א. זוהי האפשרות ליצור מחלקות חדשות המבוססות על מחלקות קיימות.
ב. זוהי היכולת של אובייקטים ממחלקות שונות להתנהג באופן אחיד באמצעות ממשק משותף.
ג. זהו עיקרון המאפשר להסתיר פרטי מימוש של אובייקט ולספק גישה אליו באמצעות ממשק ציבורי.
ד. זהו מנגנון לטיפול בשגיאות וחריגים.

**התשובה הנכונה: C**

**הסבר:**

בתכנות מונחה עצמים (OOP), אבסטרקציה היא עיקרון של הסתרת המימוש הפנימי המורכב של אובייקט ומתן ממשק פשוט ומובן לעבודה עמו. אבסטרקציה מאפשרת לעבוד עם אובייקט מבלי לדעת את פרטי המבנה הפנימי שלו.

*   **אפשרות א'** שגויה: זוהי ההגדרה של ירושה (inheritance).
*   **אפשרות ב'** שגויה: זוהי ההגדרה של פולימורפיזם (polymorphism).
*   **אפשרות ג'** נכונה: אבסטרקציה היא הסתרת המימוש הפנימי ומתן גישה אליו באמצעות ממשק ציבורי.
*   **אפשרות ד'** שגויה: זוהי הגדרה עבור בלוקים של try-except.

**כיצד פועלת אבסטרקציה:**

1.  אבסטרקציה מאפשרת להסתיר פרטי מימוש מורכבים של אובייקט, ולהותיר רק את הממשק ההכרחי לאינטראקציה.
2.  היא מאפשרת למפתחים לעבוד עם האובייקט ברמה גבוהה יותר, מבלי להתעמק בפרטים.
3.  היא תורמת למודולריות ומפשטת את התחזוקה והשינויים בקוד.
4.  בפייתון, אבסטרקציה יכולה להיות ממומשת באמצעות מחלקות מופשטות (abstract classes) או ממשקים (interfaces).

**דוגמה:**
```python
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self) -> float:
       """שיטה מופשטת"""
       pass
class Circle(Shape):
    def __init__(self, radius: int):
        self.radius = radius

    def area(self) -> float: # Реализуем метод класса Shape
        """חישוב שטח המעגל."""
        return 3.14 * self.radius**2

class Square(Shape):
  def __init__(self, side: int):
    self.side = side

  def area(self) -> float: # Реализуем метод класса Shape
    """חישוב שטח הריבוע."""
    return self.side**2

circle: Circle = Circle(5)
square: Square = Square(10)

def display_area(shape: Shape):
   print(f"Площадь: {shape.area()}") # Полиморфный вызов метода area()

display_area(circle) # Вывод: Площадь: 78.5
display_area(square) # Вывод: Площадь: 100
```
**התוצאה:**

*   אנו יכולים לעבוד עם `circle` ו-`square`, תוך שימוש בממשק המשותף `shape` (השיטה `area`), מבלי לדעת את פרטי המימוש שלהם.

לפיכך, **אפשרות ג'** היא התשובה הנכונה.
