**שאלה 103.** בפייתון, כיצד ניתן ליצור פונקציה עם ארגומנטים בעלי ערכי ברירת מחדל?

A. על ידי הקצאת ערך לארגומנט בגוף הפונקציה.
B. על ידי שימוש במילת המפתח `default` בעת הגדרת הארגומנט.
C. על ידי הקצאת ערך לארגומנט בהגדרת הפונקציה.
D. על ידי הגדרת הערך של הארגומנט כ-`None` וטיפול במקרה זה בתוך הפונקציה.

**תשובה נכונה: C**

**הסבר:**

בפייתון, כדי להגדיר ארגומנטים בעלי ערכי ברירת מחדל בפונקציה, יש להקצות ערך לארגומנטים אלו ישירות בהגדרת הפונקציה.

*   **אפשרות A** אינה נכונה: הקצאת ערך לארגומנט בגוף הפונקציה אינה יוצרת ערך ברירת מחדל.
*   **אפשרות B** אינה נכונה: בפייתון אין מילת מפתח `default` להגדרת ערך ברירת מחדל.
*   **אפשרות C** נכונה: הקצאת ערך לארגומנט בהגדרת הפונקציה יוצרת ערך ברירת מחדל.
*   **אפשרות D** אינה נכונה: ניתן להשתמש ב-`None` כברירת מחדל, אך זה אינו הופך אותו לערך ברירת מחדל בפני עצמו (השיטה ליצירת ברירת מחדל היא ההקצאה בהגדרה).

**כיצד פועלים ארגומנטים בעלי ערך ברירת מחדל:**

1.  בעת הגדרת פונקציה, ניתן להקצות ערכי ברירת מחדל לארגומנטים.
2.  אם בעת קריאה לפונקציה ארגומנט בעל ערך ברירת מחדל אינו מצוין, נעשה שימוש בערך שהוגדר כברירת מחדל.
3.  ערך ברירת המחדל מחושב פעם אחת בלבד בעת הגדרת הפונקציה.
4.  ארגומנטים בעלי ערכי ברירת מחדל חייבים להופיע לאחר ארגומנטים רגילים (חובה).

**דוגמה:**

```python
def my_function(name: str, age: int = 18, city: str = "Москва"):
    print(f"Имя: {name}, Возраст: {age}, Город: {city}")

my_function("Alice")     # שימוש בערך ברירת המחדל עבור age ו-city. פלט: Имя: Alice, Возраст: 18, Город: Москва
my_function("Bob", 25)    # שימוש בערך ברירת המחדל עבור city. פלט: Имя: Bob, Возраст: 25, Город: Москва
my_function("Charlie", 30, "London") # כל הערכים הועברו. פלט: Имя: Charlie, Возраст: 30, Город: London
```
**תוצאה:**

בעת קריאה לפונקציה ללא העברת הארגומנטים `age` ו-`city`, נעשה שימוש בערכי ברירת המחדל שלהם.

לפיכך, **אפשרות C** היא התשובה הנכונה.
