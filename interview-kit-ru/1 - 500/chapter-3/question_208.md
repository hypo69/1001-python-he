**שאלה 208.** מהו ייעודו של המתודה `__repr__()` ב-Python?

א. היא משמשת ליצירת אובייקט חדש.
ב. היא מחזירה ייצוג מחרוזתי "רשמי" של האובייקט, שניתן להשתמש בו כדי לשחזרו.
ג. היא מחזירה ייצוג מחרוזתי "ידידותי למשתמש" של האובייקט.
ד. היא מסירה את האובייקט מהזיכרון.

**תשובה נכונה: B**

**הסבר:**

המתודה `__repr__()` ב-Python משמשת להחזרת ייצוג מחרוזתי "רשמי" של האובייקט. ייצוג זה משמש לניפוי שגיאות (debugging) ויכול לשמש ליצירת עותק של האובייקט.

*   **אפשרות א** אינה נכונה: ליצירת אובייקט חדש משתמשים בבנאים (constructors) (`__init__()`, `__new__()`).
*   **אפשרות ב** נכונה: `__repr__()` מספקת ייצוג מחרוזתי עבור מפתחים.
*   **אפשרות ג** אינה נכונה: המתודה `__str__()` משמשת לייצוג המיועד למשתמש הקצה.
*   **אפשרות ד** אינה נכונה: להסרת אובייקט משתמשים במתודה `__del__()`.

**כיצד פועלת `__repr__()`:**

1.  מתודה זו נקראת אוטומטית בעת שימוש בפונקציה `repr()` או כאשר אובייקט מוצג במפרש (interpreter) של Python.
2.  היא אמורה להחזיר מחרוזת המייצגת את האובייקט, ומתאימה לניפוי שגיאות.
3.  המחרוזת צריכה להכיל את המידע הדרוש לשחזור האובייקט.

**דוגמה:**

```python
class Point:
    def __init__(self, x: int, y: int):
        self.x = x
        self.y = y

    def __repr__(self) -> str:
        return f"Point(x={self.x}, y={self.y})"

point: Point = Point(3, 5)
print(point) # פלט: <__main__.Point object at 0x...> (נקראת __str__, אם לא מוגדרת, אז __repr__)
print(repr(point)) # פלט: Point(x=3, y=5) (נקראת __repr__)
point # פלט: Point(x=3, y=5) (נקראת __repr__)
```

**כתוצאה מכך:**
* אם בנאי המחלקה אינו מגדיר את `__str__()` ונעשה שימוש ב-`print()`, Python יקרא אוטומטית ל-`__repr__()`.
*   בעת קריאה ל-`repr()` או בעת הדפסה ישירה של אובייקט במצב אינטראקטיבי, נקראת המתודה `__repr__()`, המחזירה מחרוזת המייצגת את האובייקט.

לפיכך, **אפשרות B** היא התשובה הנכונה.
