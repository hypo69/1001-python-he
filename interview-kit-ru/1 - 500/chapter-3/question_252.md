### `question_252.md`

**שאלה 252.** כיצד ניתן להגדיר מודול ב-Python ומה מטרתו בארגון ושימוש חוזר בקוד בתוכניות Python שונות?

A. מודול מוגדר באמצעות פונקציה, האורזת קוד לשימוש חוזר לביצוע בתוכנית הנוכחית.
B. מודול נוצר על ידי כתיבת קוד Python בקובץ נפרד, ותכולתו (פונקציות, מחלקות) ניתנת לשימוש חוזר בסקריפטים אחרים באמצעות ייבוא.
C. מודול הוא ספרייה חיצונית שיש להוריד ולהתקין לפני השימוש בה.
D. מודול הוא משתנה המאחסן סוגי נתונים שונים וניתן להשתמש בו לפעולות דינמיות.

**תשובה נכונה: B**

**הסבר:**

ב-Python, מודול הוא פשוט קובץ המכיל קוד Python (פונקציות, מחלקות, משתנים). מודולים משמשים לארגון קוד ושימוש חוזר בו בתוכניות שונות.

*   **אפשרות A אינה נכונה:** מודול אינו פונקציה, אלא קובץ.
*   **אפשרות B נכונה:** זהו התיאור הנכון של אופן יצירת מודול ושימוש בו.
*   **אפשרות C אינה נכונה:** מודול אינו בהכרח ספרייה חיצונית. הוא יכול להיות קוד מותאם אישית.
*   **אפשרות D אינה נכונה:** מודול אינו משתנה.

**כיצד להגדיר מודול:**

1.  צור קובץ עם סיומת `.py`.
2.  כתוב בו קוד Python (הגדרות של פונקציות, מחלקות, משתנים וכו').
3.  שמור את הקובץ. שם הקובץ (ללא הסיומת `.py`) יהיה שם המודול.

**כיצד להשתמש במודול:**

1.  בקובץ Python אחר, השתמש באופרטור `import module_name` לייבוא המודול.
2.  השתמש בפונקציות, מחלקות ומשתנים של המודול באמצעות התחביר `module_name.element`, כאשר `element` הוא שם הרכיב המיובא.

**דוגמה:**

```python
# my_module.py (מוּדוּל)
def my_function():
    return "Hello from my_module"

my_variable = 10

# main.py (קובץ Python אחר)
import my_module

print(my_module.my_function())
print(my_module.my_variable)

```
**לסיכום:**

מודול ב-Python הוא קובץ המכיל קוד Python, שניתן להשתמש בו לארגון ושימוש חוזר בקוד בתוכניות שונות, על ידי ייבוא המודול.

לפיכך, אפשרות B היא הנכונה.
