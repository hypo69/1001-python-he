**שאלה 210.** מהי "אינקפסולציה" בתכנות מונחה-עצמים (תמ"ע) ואיזה תועלת היא מביאה?

A. אינקפסולציה היא מנגנון ירושה של מחלקות.
B. אינקפסולציה היא היכולת של עצמים ממחלקות שונות להתנהג באופן אחיד באמצעות ממשק משותף.
C. אינקפסולציה היא עקרון שילוב נתונים ושיטות הפועלות על נתונים אלה, בתוך גבולות המחלקה, וכן הסתרת המימוש הפנימי של העצם ומתן גישה אליו באמצעות ממשק ציבורי.
D. אינקפסולציה היא דרך ליצירת שיטות סטטיות השייכות למחלקה.

**תשובה נכונה: C**

**הסבר:**

אינקפסולציה היא אחד מהעקרונות היסודיים של תכנות מונחה-עצמים. היא מתבטאת בשילוב נתונים (תכונות) ושיטות הפועלות על נתונים אלה, בתוך מחלקה. היא גם כוללת הסתרת פרטי המימוש ומתן גישה לנתונים רק דרך שיטות המחלקה, כלומר, דרך "ממשק ציבורי".

*   **אפשרות A** שגויה: ירושה היא מנגנון המאפשר למחלקה אחת לרשת תכונות ממחלקה אחרת.
*   **אפשרות B** שגויה: זוהי הגדרת הפולימורפיזם.
*   **אפשרות C** נכונה: אינקפסולציה היא שילוב נתונים עם שיטות וניהול גישה.
*   **אפשרות D** שגויה: זוהי הגדרת שיטות סטטיות.

**כיצד פועלת אינקפסולציה:**

1.  הנתונים והשיטות הקשורים לעצם מאוחדים למחלקה.
2.  נתונים פנימיים (תכונות) מוכרזים כ"מוגנים" (protected) או "פרטיים" (private), כדי להגביל גישה ישירה מבחוץ למחלקה.
3.  לצורך גישה לנתונים כאלה, נוצרות שיטות (גטרים וסטרים) המספקות גישה מבוקרת.
4.  אינקפסולציה מאפשרת להסתיר פרטי מימוש ומספקת הגנה על הנתונים.

**דוגמה:**

```python
class BankAccount:
    def __init__(self, balance: int):
      self._balance: int = balance # משתנה מוגן

    def get_balance(self) -> int:
        """קבלת היתרה הנוכחית."""
        return self._balance

    def deposit(self, amount: int) -> None:
        """הפקדת כסף לחשבון."""
        if amount > 0:
           self._balance += amount
        else:
            print("הסכום חייב להיות חיובי!")

account: BankAccount = BankAccount(1000)
print(f"באלאנס: {account.get_balance()}") # פלט: באלאנס: 1000

# גישה ישירה למשתנה balance, אך אינה מומלצת:
# print(account._balance)
```

**בתוצאה:**
*   בדוגמה לעיל, התכונה `_balance` היא מוגנת, ואין אליה גישה ישירה.
*   הניהול מתבצע דרך השיטות `get_balance`, `deposit` ו-`withdraw`, מה שמאפשר להגן על מצב העצם.

לפיכך, **אפשרות C** היא הנכונה.
