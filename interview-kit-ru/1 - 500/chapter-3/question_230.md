### `question_230.md`

**שאלה 230.** מה ההבדל בין מילת המפתח `def` ל-`lambda` בעת הגדרת פונקציות ב-Python, בפרט, מבחינת הצהרה ופונקציונליות?

א. `def` מגדירה פונקציה שחייבת להחזיר ערך, ואילו `lambda` אינה יכולה להחזיר כל ערך.
ב. `def` מגדירה פונקציה מלאה עם שם ומספר ביטויים, ואילו `lambda` מגדירה פונקציה אנונימית עם ביטוי יחיד.
ג. `lambda` מגדירה פונקציה שניתן להשתמש בה כגנרטור, ואילו ב-`def` - לא.
ד. `def` משמשת למחלקות ואובייקטים, ואילו `lambda` שמורה רק לפונקציות.

**תשובה נכונה: B**

**הסבר:**

מילות המפתח `def` ו-`lambda` שתיהן משמשות להגדרת פונקציות ב-Python, אך הן נבדלות באופיין ובשימושיהן.

*   **`def`:**
    *   משמשת להגדרת פונקציות *מלאות*.
    *   לפונקציה המוגדרת באמצעות `def` יש *שם*.
    *   יכולה להכיל *מספר ביטויים* ו*מספר שורות קוד שרירותי*.
    *   מתאימה להגדרת *לוגיקה מורכבת*.
    *   יכולה להשתמש באופרטור `return` כדי להחזיר ערך.
*   **`lambda`:**
    *   משמשת להגדרת פונקציות *אנונימיות* (חסרות שם).
    *   מכילה *רק ביטוי יחיד*.
    *   בדרך כלל משמשת לפונקציות *פשוטות* ו*קצרות*, המשמשות לעיתים קרובות כארגומנטים לפונקציות אחרות (למשל, `map`, `filter`, `sorted`).
    *   אינה דורשת שימוש ב-`return`; הביטוי מחזיר את התוצאה אוטומטית.

*   **אפשרות א' אינה נכונה:** גם `def` וגם `lambda` יכולות להחזיר ערכים.
*   **אפשרות ב' נכונה:** זהו ההבדל המרכזי בין `def` ל-`lambda`. `def` יוצרת פונקציה מלאה עם שם ובלוק קוד, `lambda` - פונקציה אנונימית, עם פונקציונליות מוגבלת.
*   **אפשרות ג' אינה נכונה:** `lambda` אינה מיועדת ליצירת גנרטורים. גנרטורים נוצרים באמצעות שימוש ב-`yield` בתוך פונקציות המוגדרות באמצעות `def`.
*   **אפשרות ד' אינה נכונה:** `def` משמשת להגדרת פונקציות, שיכולות לשמש במחלקות ומחוצה להן. `lambda` משמשת להגדרת פונקציות אנונימיות.

**דוגמאות:**

```python
# פונקציה שהוגדרה באמצעות def:
def add(a, b):
    result = a + b
    return result

# פונקציה אנונימית שהוגדרה באמצעות lambda:
multiply = lambda x, y: x * y

# שימוש ב-lambda בפונקציה filter:
numbers = [1, 2, 3, 4, 5]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))

```

**מתי להשתמש ב-`lambda`:**

*   כאשר יש צורך בפונקציה פשוטה וקצרה, המשמשת פעם אחת.
*   כאשר יש צורך להעביר פונקציה כארגומנט לפונקציה אחרת.
*   כאשר אין צורך לתת שם לפונקציה.

**לסיכום:**

`def` משמשת להגדרת פונקציות מלאות ושמיות עם אפשרות לכתיבת מבני קוד מורכבים. `lambda` משמשת ליצירת פונקציות אנונימיות פשוטות, המשמשות לעיתים קרובות בתכנות פונקציונלי או כארגומנטים לפונקציות אחרות.

לפיכך, אפשרות ב' היא הנכונה.
