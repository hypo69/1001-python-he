### `question_300.md`

**שאלה 300.** מהו הייעוד של הפונקציה `filter()` ב-Python ובמה היא שונה מהפונקציה `map()` מבחינת מה שהיא מחזירה?

A. הפונקציה `filter()` מיישמת פונקציה על כל איבר באובייקט איטרבילי ומחזירה גרסה מסוננת עם האיברים שעבורם התנאי מתקיים (True). הפונקציה `map()` מיישמת פונקציה על כל איבר ומחזירה גרסה משתנה של האובייקט האיטרבילי המקורי.
B. הפונקציה `filter()` מיישמת פונקציה על אובייקט איטרבילי ומחזירה tuple של כל האיברים באובייקט האיטרבילי. הפונקציה `map()` מחזירה list של כל הערכים באובייקט האיטרבילי.
C. הפונקציה `filter()` מבצעת חישובים על אובייקט איטרבילי ומחזירה ערך מספרי, בעוד ש-`map()` מבצעת פעולות לוגיות.
D. גם `filter()` וגם `map()` מחזירות תוצאה זהה, ללא הבדל פונקציונלי ביניהן.

**תשובה נכונה: A**

**הסבר:**

הפונקציות `filter()` ו-`map()` הן פונקציות מובנות ב-Python המשמשות לעיבוד אובייקטים איטרביליים (למשל, רשימות, tuples), אך הן מבצעות משימות שונות ומחזירות תוצאות שונות.

*   **`filter(function, iterable)`:**
    *   מיישמת את הפונקציה `function` על *כל איבר* ב-`iterable`. הפונקציה צריכה להחזיר `True` או `False`.
    *   מחזירה *איטרטור*, המכיל רק את האיברים מתוך `iterable` שעבורם הפונקציה החזירה `True`.
    *   מיועדת ל*סינון* איברים ברצף.
*   **`map(function, iterable)`:**
     *   מיישמת את הפונקציה `function` על *כל איבר* ב-`iterable`.
     *   מחזירה *איטרטור*, המכיל את התוצאות של יישום `function` על כל איבר ב-`iterable`.
     *   מיועדת ל*טרנספורמציה* של כל איבר ברצף.

*   **אפשרות A נכונה:** זהו תיאור נכון של ההבדלים בייעוד ובתוצאות של הפונקציות `filter()` ו-`map()`.
*   **אפשרות B אינה נכונה:** `filter` מחזירה *איטרטור* (לא tuple), ולא את כל האיברים אלא רק את המסוננים. `map()` מחזירה איטרטור, ולא list.
*   **אפשרות C אינה נכונה:** `filter` מבצעת סינון לוגי, ו-`map` - טרנספורמציה של איברים. אף אחת מהפונקציות לא מחזירה ערך מספרי אלא אם כן הפונקציה המופעלת מחזירה מספר.
*   **אפשרות D אינה נכונה:** `filter()` ו-`map()` מבצעות משימות שונות ומחזירות תוצאות שונות.

**דוגמה:**

```python
numbers = [1, 2, 3, 4, 5, 6]

# שימוש ב-filter() לבחירת מספרים זוגיים
even_numbers = list(filter(lambda x: x % 2 == 0, numbers)) # Returns [2,4,6]
print(even_numbers) # Output: [2, 4, 6]

# שימוש ב-map() להכפלת כל המספרים ב-2
multiplied_numbers = list(map(lambda x: x * 2, numbers)) # Returns [2,4,6,8,10,12]
print(multiplied_numbers) # Output: [2, 4, 6, 8, 10, 12]
```
**לסיכום:**

`filter()` מיושמת לבחירת איברים מרצף על סמך תנאי נתון, בעוד ש-`map()` מיושמת לטרנספורמציה של כל איבר ברצף. `filter()` מחזירה איטרטור עם איברים מסוננים, בעוד ש-`map()` מחזירה איטרטור עם איברים שעברו טרנספורמציה.

לפיכך, אפשרות A היא הנכונה.
