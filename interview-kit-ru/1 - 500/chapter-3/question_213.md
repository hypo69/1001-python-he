**שאלה 13.** מהו "פולימורפיזם" בהקשר של תכנות מונחה עצמים?

A. זוהי היכולת לרשת מתודות רק ממחלקת אב אחת.
B. זוהי היכולת של עצמים ממחלקות שונות לקיים מבנה נתונים משותף.
C. זוהי היכולת של עצמים ממחלקות שונות להגיב לאותה מתודה באופנים שונים.
D. זוהי היכולת להסתיר פרטים פנימיים של מחלקה.

**תשובה נכונה: C**

**הסבר:**

פולימורפיזם (מיוונית: "רב-צורתיות") בתכנות מונחה עצמים (תמו"ע) הוא היכולת של עצמים ממחלקות שונות להגיב לאותה מתודה בצורה ייחודית להם. משמעות הדבר היא שניתן להשתמש באותו ממשק (מתודה) עבור סוגים שונים של עצמים, וכל עצם יבצע את הפעולות המתאימות לו.

*   **אפשרות A** אינה נכונה: מושג זה מתייחס לירושה, ולא לפולימורפיזם.
*   **אפשרות B** אינה נכונה: מבנה נתונים משותף אינו מתייחס לפולימורפיזם, אלא יותר לירושה.
*   **אפשרות C** נכונה: פולימורפיזם מאפשר לעצמים ממחלקות שונות להגיב באופנים שונים לאותה מתודה.
*   **אפשרות D** אינה נכונה: מושג זה מתייחס לאנקפסולציה (כימוס), ולא לפולימורפיזם.

**כיצד פועל פולימורפיזם:**

1.  למחלקות שונות יכולות להיות מתודות בעלות שם זהה.
2.  המתודה נקראת דרך ממשק, והפעולה שתתבצע נקבעת על סמך סוג העצם.

**דוגמה:**

```python
class Animal:
    def make_sound(self):
        print("Generic animal sound")

class Dog(Animal):
    def make_sound(self):
        print("Woof!")

class Cat(Animal):
    def make_sound(self):
      print("Meow!")

def animal_speak(animal: Animal):
    animal.make_sound() # Полиморфный вызов метода make_sound() - Polymorphic call to the make_sound() method

animal: Animal = Animal()
dog: Dog = Dog()
cat: Cat = Cat()
animal_speak(animal) # Вывод: Generic animal sound - Output: Generic animal sound
animal_speak(dog)  # Вывод: Woof! - Output: Woof!
animal_speak(cat)  # Вывод: Meow! - Output: Meow!
```

**תוצאה:**
* הפונקציה `animal_speak()` יכולה לקרוא למתודה `make_sound()` על עצמים שונים, וכל עצם קורא למימוש הייחודי לו של המתודה.

לפיכך, **אפשרות C** היא התשובה הנכונה.
