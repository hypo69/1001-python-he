**שאלה 218.** איזה תפקיד ממלא הפרמטר `self` במתודות של מחלקה ב-Python?

א. זהו הארגומנט הראשון, המייצג את המחלקה בה הוגדרה המתודה.
ב. זהו הארגומנט האחרון, המכיל את הערכים שהועברו עבור הפרמטרים.
ג. זהו ארגומנט חובה, המייצג את מופע המחלקה (instance) שעבורו נקראה המתודה.
ד. זהו ארגומנט אופציונלי, המשמש לגישה לתכונות סטטיות (static attributes) של המחלקה.

**תשובה נכונה: ג**

**הסבר:**

הפרמטר `self` במתודות של מחלקה ב-Python הוא ארגומנט ראשון חובה, המשמש להתייחסות למופע (אובייקט) ספציפי של המחלקה שעבורו נקראה המתודה.

*   **אפשרות א** אינה נכונה: הפרמטר `cls` מייצג את המחלקה עצמה, לא `self`.
*   **אפשרות ב** אינה נכונה: `self` הוא הארגומנט הראשון, לא האחרון.
*   **אפשרות ג** נכונה: `self` מתייחס למופע ומאפשר לעבוד עם התכונות והמתודות שלו.
*   **אפשרות ד** אינה נכונה: `self` הוא חובה במתודה של מופע.

**כיצד פועל `self`:**

1.  כאשר נקראת מתודה של מופע, Python מעביר אוטומטית הפניה למופע זה כארגומנט הראשון, אשר לפי מוסכמה נקרא `self`.
2.  באמצעות `self`, המתודה יכולה לגשת לתכונות ולמתודות אחרות של מופע זה.
3.  `self` מאפשר למתודות להבחין לאיזה אובייקט ספציפי הן מתייחסות, גם אם לכמה אובייקטים מאותה מחלקה ישנן מתודות זהות.

**דוגמה:**

```python
class Dog:
    def __init__(self, name: str, age: int):
        self.name = name # תכונה של האובייקט
        self.age = age  # תכונה של האובייקט

    def describe(self): # מתודה של האובייקט
        print(f"Собака: {self.name}, Возраст: {self.age}")


dog1: Dog = Dog("Buddy", 3)
dog1.describe()  # Output: Собака: Buddy, Возраст: 3
```

**בתוצאה:**

*   `self` במתודה `describe` מאפשר גישה לתכונות `name` ו-`age` של האובייקט `dog1`.

לפיכך, **אפשרות ג** היא התשובה הנכונה.
