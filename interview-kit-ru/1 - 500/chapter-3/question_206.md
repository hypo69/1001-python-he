### `question_206.md`

**שאלה 206.** מה מבצעת המתודה `__repr__()` בפייתון?

A.  היא משמשת ליצירת אובייקט חדש.
B.  היא מחזירה ייצוג מחרוזתי "רשמי" של האובייקט, שניתן להשתמש בו ליצירתו מחדש.
C.  היא מחזירה ייצוג מחרוזתי "קריא למשתמש" של האובייקט.
D.  היא מסירה את האובייקט מהזיכרון.

**תשובה נכונה: B**

**הסבר:**

המתודה `__repr__()` בפייתון מיועדת לספק ייצוג מחרוזתי "רשמי" של אובייקט. ייצוג זה משמש לאיתור באגים (debugging) וניתן להשתמש בו, ככלל, ליצירת עותק של האובייקט.

*   **אפשרות A** אינה נכונה: ליצירת אובייקט משתמשים בבנאים (`__init__()`, `__new__()`).
*   **אפשרות B** נכונה: `__repr__()` מספקת ייצוג מחרוזזתי המיועד למפתחים.
*   **אפשרות C** אינה נכונה: המתודה `__str__()` משמשת לייצוג המיועד למשתמש הקצה.
*   **אפשרות D** אינה נכונה: להסרת אובייקט משתמשים במתודה `__del__()`.

**כיצד פועלת `__repr__()`:**

1.  המתודה `__repr__()` נקראת באופן אוטומטי בעת שימוש בפונקציה `repr()`.
2.  היא צריכה להחזיר מחרוזת המייצגת את האובייקט, המתאימה לאיתור באגים.
3.  המחרוזת צריכה, ככלל, להכיל מידע הנחוץ לצורך יצירתו מחדש של האובייקט.

**דוגמה:**

```python
class Point:
    def __init__(self, x: int, y: int):
        self.x = x
        self.y = y

    def __repr__(self) -> str:
        return f"Point(x={self.x}, y={self.y})"

point: Point = Point(3, 5)
print(point) # פלט: <__main__.Point object at 0x...> (נקראת __str__, אם אינה מוגדרת, אז __repr__)
print(repr(point)) # פלט: Point(x=3, y=5) (נקראת __repr__)
point # פלט: Point(x=3, y=5) (נקראת __repr__)
```

**לסיכום:**

*   הפונקציה `print(point)` קוראת למתודה `__str__()`, ואם היא אינה מוגדרת, פייתון קורא ל-`__repr__()`.
*  הפונקציה `repr(point)` וכן הדפסה ישירה של המשתנה `point` במצב אינטראקטיבי קוראות למתודה `__repr__()`, המחזירה מחרוזת בפורמט המתאים לאיתור באגים.

לפיכך, **אפשרות B** היא התשובה הנכונה.
