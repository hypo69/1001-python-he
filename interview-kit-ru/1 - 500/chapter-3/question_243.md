**שאלה 243.** איזו מהפונקציות הבאות של Python משמשת למציאת האיבר הגדול ביותר ברשימה?

A. `max()`
B. `largest()`
C. `biggest()`
D. `max_value()`

**תשובה נכונה: A**

**הסבר:**

ב-Python, למציאת האיבר הגדול ביותר ברשימה (או כל סדרת איברים הניתנת לאיטרציה אחרת) משתמשים בפונקציה המובנית `max()`.

*   **אפשרות A נכונה:** `max()` – זו פונקציה מובנית, שמוצאת את האיבר המקסימלי בסדרת איברים.
*   **אפשרות B אינה נכונה:** הפונקציה `largest()` אינה קיימת ב-Python.
*   **אפשרות C אינה נכונה:** הפונקציה `biggest()` אינה קיימת ב-Python.
*   **אפשרות D אינה נכונה:** הפונקציה `max_value()` אינה קיימת ב-Python.

**כיצד להשתמש בפונקציה `max()`:**

1.  העבירו את הרשימה (או סדרת איברים הניתנת לאיטרציה אחרת) כארגומנט לפונקציה `max()`.
2.  הפונקציה `max()` תחזיר את האיבר הגדול ביותר מסדרת איברים זו.

**דוגמה:**

```python
my_list = [10, 5, 25, 8, 15]
max_element = max(my_list)
# פלט: 25
print(max_element) 

my_tuple = (1, 2, 3, 4, 5)
max_tuple = max(my_tuple)
# Output 5
print(max_tuple) 
```
**לסיכום:**

למציאת האיבר הגדול ביותר ברשימה ב-Python יש להשתמש בפונקציה המובנית `max()`.

לפיכך, אפשרות A היא הנכונה.
