**שאלה 229.** כיצד ניתן לייבא פונקציה ספציפית מתוך מודול Python ולהשתמש בה ללא צורך בהוספת קידומת של שם המודול?

A. `import module_name.function_name`
B. `from module_name import function_name`
C. `import function_name from module_name`
D. `use module_name.function_name`

**התשובה הנכונה: B**

**הסבר:**

ב-Python, על מנת לייבא פונקציה ספציפית ממודול ולהשתמש בה ללא קידומת של שם המודול, משתמשים בתחביר `from module_name import function_name`. תחביר זה מאפשר לגשת לפונקציה ישירות במרחב השמות הנוכחי.

*   **אפשרות A שגויה:** התחביר `import module_name.function_name` אינו תקין לייבוא פונקציה ספציפית.
*   **אפשרות B נכונה:** התחביר `from module_name import function_name` מאפשר לייבא פונקציה ספציפית ולהשתמש בה ישירות, ללא קידומת של שם המודול.
*   **אפשרות C שגויה:** התחביר `import function_name from module_name` אינו תקין.
*   **אפשרות D שגויה:** התחביר `use module_name.function_name` אינו קיים ב-Python.

**כיצד פועל `from ... import ...`:**

1.  התחביר `from module_name import function_name` מבצע את הפעולות הבאות:
    *   מייבא את המודול `module_name`.
    *   מחלץ מתוך מודול זה את האובייקט `function_name`.
    *   יוצר במרחב השמות הנוכחי הפניה לאובייקט `function_name`, מה שמאפשר להשתמש בפונקציה `function_name` ללא ציון שם המודול.

**דוגמה:**

```python
# נניח שיש מודול math.py המכיל את הפונקציה sqrt()

# ייבוא תקין של פונקציה ספציפית
from math import sqrt

# כעת ניתן להשתמש ב-sqrt() ישירות
result = sqrt(16)
print(result) # פלט: 4.0

# ייבוא לא תקין
# import math.sqrt # שגיאה
# result = sqrt(16) # שגיאה, אין פונקציה sqrt במרחב השמות הנוכחי
```

**יתרונות השימוש ב-`from ... import ...`:**

*   **נוחות:** מאפשר להשתמש בפונקציות ישירות ללא קידומת, מה שהופך את הקוד לתמציתי יותר.
*   **בהירות:** מראה בבירור אילו פונקציות ספציפיות מיובאות מהמודול.
*   **מניעת התנגשויות:** בשימוש ב-`from ... import ...` רק הפונקציות הנחוצות מיובאות למרחב השמות הנוכחי, מה שמפחית את הסבירות להתנגשויות שמות, אם קיימות לכם פונקציות או משתנים עם שמות זהים.

**לסיכום:**

התחביר `from module_name import function_name` הוא דרך נוחה ועדיפה לייבוא פונקציות ספציפיות ממודולים ב-Python, והופך את הקוד לנקי וקריא יותר.

לפיכך, אפשרות B היא הנכונה.
