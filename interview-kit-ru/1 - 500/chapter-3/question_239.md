### `question_239.md`

**שאלה 239.** איזו מהאפשרויות הבאות היא התחביר הנכון להגדרת פונקציה בפייתון, שמקבלת שני פרמטרים ומחזירה את סכומם?

A. `def function add(a, b): return a + b`
B. `function def add(a, b): return a + b`
C. `def add(a, b) { return a + b }`
D. `def add(a, b): return a + b`

**תשובה נכונה: D**

**הסבר:**

בפייתון, להגדרת פונקציה נעשה שימוש במילת המפתח `def`, אחריה מגיע שם הפונקציה, רשימת פרמטרים בסוגריים עגולים ונקודתיים. גוף הפונקציה חייב להיות עם הזחה. הפקודה `return` משמשת להחזרת ערך מהפונקציה.

*   **אפשרות A שגויה:** התחביר `def function add(a, b):` אינו תקין. המילה `function` אינה הכרחית.
*   **אפשרות B שגויה:** התחביר `function def add(a, b):` אינו תקין. מילת המפתח `def` חייבת להיות ראשונה.
*   **אפשרות C שגויה:** בפייתון לא נעשה שימוש בסוגריים מסולסלים `{}` להגדרת גוף הפונקציה. נעשה שימוש בהזחות.
*   **אפשרות D נכונה:** זהו התחביר הנכון להגדרת פונקציה `add`, שמקבלת שני פרמטרים `a` ו-`b`, ומחזירה את סכומם.

**כיצד להגדיר פונקציות באופן תקין בפייתון:**

1.  התחילו במילת המפתח `def`.
2.  לאחר `def` ציינו את *שם* הפונקציה.
3.  בהמשך, בסוגריים עגולים, ציינו את *הפרמטרים* (ארגומנטים) של הפונקציה, מופרדים בפסיקים. אם אין פרמטרים, הסוגריים חייבים להיות ריקים.
4.  שימו נקודתיים `:` בסוף השורה.
5.  גוף הפונקציה חייב להיות עם הזחה.
6.  השתמשו בפקודת `return` להחזרת ערך מהפונקציה.

**דוגמה:**

```python
def add(x, y):
    result = x + y
    return result

sum_of_numbers = add(5, 10)
print(sum_of_numbers)  # פלט: 15
```

**לסיכום:**

התחביר הנכון להגדרת פונקציה בפייתון, שמקבלת שני פרמטרים ומחזירה את סכומם, הוא `def add(a, b): return a + b`.

לפיכך, אפשרות D היא התשובה הנכונה.
