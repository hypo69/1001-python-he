### `question_248.md`

**שאלה 248.** כיצד ניתן למנוע ביצוע של קוד מסוים במודול Python בעת ייבואו במקום אחר?

א. על ידי הצבת הקוד בתוך פונקציה.
ב. באמצעות שימוש בבלוק `if __name__ == '__main__':`.
ג. על ידי הפיכת הקוד להערה.
ד. באמצעות שימוש באופרטור `import` במודול.

**תשובה נכונה: ב**

**הסבר:**

ב-Python, בלוק `if __name__ == '__main__':` משמש כדי למנוע ביצוע של קוד מסוים במודול, כאשר הוא מיובא במקום אחר. קוד שנמצא בתוך בלוק זה יתבצע רק כאשר המודול מופעל כתוכנית עצמאית, ולא כאשר הוא מיובא כמודול לקובץ אחר.

*   **אפשרות א' אינה נכונה:** הצבת קוד בתוך פונקציה לא תמנע את ביצועו בעת ייבוא. הפונקציה פשוט תוגדר, אך לא תקרא.
*   **אפשרות ב' נכונה:** `if __name__ == '__main__':` היא אידיומה לשליטה על ביצוע קוד בעת ייבוא.
*   **אפשרות ג' אינה נכונה:** קוד שהוצמד לו סימן הערה (comment) לא יתבצע *לעולם*, לא בעת ייבוא ולא בעת הפעלה ישירה.
*   **אפשרות ד' אינה נכונה:** אופרטור `import` אינו מונע ביצוע של קוד.

**כיצד פועל `if __name__ == '__main__':`:**

1.  כאשר קובץ Python מופעל ישירות (לדוגמה, `python my_module.py`), למשתנה `__name__` מוקצה הערך `"__main__"`.
2.  כאשר קובץ Python מיובא לקובץ אחר (לדוגמה, `import my_module`), למשתנה `__name__` באותו קובץ מוקצה הערך של שם המודול (לדוגמה, `"my_module"`).
3.  קוד המוצב בתוך בלוק `if __name__ == '__main__':` יתבצע רק במקרה שהקובץ מופעל *ישירות*, כלומר כאשר `__name__ == '__main__'`.

**דוגמה:**

```python
# קובץ my_module.py
print("קוד מחוץ לבלוק ה-if")

def my_function():
    print("Function in my_module is called")

if __name__ == '__main__':
    print("קוד בתוך בלוק if __name__ == '__main__':")
    my_function()

# קובץ main.py
import my_module
print("after import in main")
my_module.my_function()
```
בעת הרצת `python my_module.py` הפלט יהיה:
```
קוד מחוץ לבלוק ה-if
קוד בתוך בלוק if __name__ == '__main__':
Function in my_module is called
```
בעת הרצת `python main.py` הפלט יהיה:
```
קוד מחוץ לבלוק ה-if
after import in main
Function in my_module is called
```
שימו לב שהקוד *בתוך* הבלוק `if __name__ == '__main__':` מתבצע רק בעת הפעלה *ישירה* של `my_module.py`, אך לא בעת ייבוא.

**לסיכום:**

באמצעות שימוש בבלוק `if __name__ == '__main__':`, ניתן למנוע ביצוע של קוד מסוים במודול Python בעת ייבואו במקום אחר, מה שהופך את המודול לגמיש יותר ומתאים לשימוש חוזר.

לפיכך, אפשרות ב' היא הנכונה.
