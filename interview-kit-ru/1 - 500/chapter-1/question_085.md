### `question_085.md`

**שאלה 85.** מה עושה מילת המפתח `continue` בלולאה?

- A. יוצאת מהלולאה באופן מיידי.
- B. מדלגת על יתרת הקוד בתוך הלולאה עבור האיטרציה הנוכחית.
- C. משהה את ביצוע הלולאה.
- D. אף אחד מהנ"ל.

**תשובה נכונה: B**

**הסבר:**

מילת המפתח `continue` ב-Python משמשת בתוך לולאות (לדוגמה, `for` ו-`while`) על מנת לדלג על יתרת האיטרציה הנוכחית ולעבור לאיטרציה הבאה של הלולאה.

*   **אפשרות A** אינה נכונה: `continue` אינה מפסיקה את הלולאה, אלא רק את האיטרציה הנוכחית. להפסקת הלולאה משתמשים ב-`break`.
*   **אפשרות B** נכונה: `continue` מדלגת על יתרת הקוד באיטרציה הנוכחית ועוברת לבאה.
*   **אפשרות C** אינה נכונה: `continue` אינה משהה את ביצוע הלולאה.
*   **אפשרות D** אינה נכונה: אפשרות B נכונה.

**כיצד פועל `continue`:**

1.  כאשר מפרש Python נתקל במילה `continue` בתוך לולאה, הוא מדלג על כל הפקודות הנותרות באותה איטרציה.
2.  ביצוע התוכנית עובר לאיטרציה הבאה של הלולאה, כלומר לבדיקת התנאי בלולאת `while` או לאלמנט הבא בלולאת `for`.
3.  הדילוג על האיטרציה מתרחש רק עבור האיטרציה הנוכחית; שאר האיטרציות של הלולאה ממשיכות כרגיל.

**דוגמאות שימוש אופייניות:**

1.  **דילוג על איטרציה ספציפית:** כאשר יש צורך לדלג על איטרציות מסוימות, בהתבסס על תנאי כלשהו.
2.  **סינון נתונים:** כאשר יש צורך לדלג על אלמנטים מסוימים במבנה נתונים בר איטרציה.

**דוגמה:**

```python
numbers: list[int] = [1, 2, 3, 4, 5, 6]
for number in numbers:
    if number % 2 == 0:
      continue
    print(f"Нечетное число: {number}")
# פלט:
# Нечетное число: 1
# Нечетное число: 3
# Нечетное число: 5

x: int = 0
while x < 10:
   x +=1
   if x % 3 == 0:
      continue
   print(x)
# פלט:
# 1
# 2
# 4
# 5
# 7
# 8
# 10
```

**לסיכום:**
* בלולאת `for` מודפסים רק מספרים אי-זוגיים, מכיוון שמילת `continue` מדלגת על האיטרציה עבור מספרים זוגיים.
* בלולאת `while` מודפסים כל המספרים, למעט אלה המתחלקים ב-3.

לפיכך, **אפשרות B** היא הנכונה.
