**שאלה 11.** ב-Python, שמות משתנים רגישים לרישיות (case-sensitive), מה שאומר ש-`variable` ו-`Variable` נחשבים על ידי המפרש לשונים. בהינתן מידע זה, איזו מהקביעות הבאות תהיה נכונה, אם `variable` ו-`Variable` הוגדרו כמספרים שלמים בסקריפט Python?

- A. `print(variable)` ו-`print(Variable)` ידפיסו את אותו הערך, אם לשני המשתנים יש את אותו הערך.
- B. תתרחש שגיאה, המודיעה ששמות משתנים אינם יכולים להיות דומים, למעט ההבדל ברישיות.
- C. Python ישכתב אוטומטית את ערך המשתנה הראשון (`variable`) בערך המשתנה השני (`Variable`) בכל הסקריפט.
- D. בלתי אפשרי להשתמש באותו שם עם רישיות שונות עבור משתנים שונים באותו סקריפט.

**תשובה נכונה: A**

**הסבר:**

ב-Python, מזהים (כולל שמות משתנים) רגישים לרישיות. משמעות הדבר היא ש-`variable` ו-`Variable` נתפסים על ידי המפרש כשני משתנים שונים לחלוטין.

*   **אפשרות A** נכונה: אם לשני המשתנים, `variable` ו-`Variable`, יש את אותו ערך מספרי, הפלט שלהם באמצעות הפונקציה `print()` יציג ערך זה, ללא תלות בשם המשתנה. Python לא יתייחס אליהם כאל אותו משתנה עקב ההבדלים ברישיות.

*   **אפשרות B** אינה נכונה: Python לא יפיק שגיאה המציינת ששמות משתנים אינם יכולים להיות דומים, למעט ההבדל ברישיות. Python מפרש אותם כשני משתנים נפרדים.

*   **אפשרות C** אינה נכונה: Python לא ישכתב אוטומטית את ערך `variable` בערך `Variable`. הם ימשיכו להתקיים כשני משתנים נפרדים.

*   **אפשרות D** אינה נכונה: Python מאפשר שימוש באותם שמות עם רישיות שונות עבור משתנים שונים באותו סקריפט.

**דוגמה:**

```python
variable: int = 10
Variable: int = 10

print(f"Значение variable: {variable}")   # פלט: ערך של variable: 10
print(f"Значение Variable: {Variable}")   # פלט: ערך של Variable: 10

Variable: int = 20 # משנים את הערך של משתנה אחד בלבד

print(f"Значение variable после изменения Variable: {variable}")   # פלט: ערך של variable לאחר שינוי Variable: 10
print(f"Значение Variable после изменения: {Variable}")  # פלט: ערך של Variable לאחר שינוי: 20
```

**לסיכום:**

כפי שניתן לראות מהדוגמה, `variable` ו-`Variable` הם שני משתנים שונים, גם אם יש להם כתיבה דומה עם רישיות שונות ובתחילה הכילו את אותו הערך. כאשר אנו משנים את הערך של `Variable`, ערך המשתנה `variable` אינו משתנה.

לפיכך, **אפשרות A** נכונה, מכיוון שהיא מתארת במדויק את ההתנהגות של Python ביחס למשתנים הרגישים לרישיות.
