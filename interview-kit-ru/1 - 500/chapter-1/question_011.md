**שאלה 11.** ב-Python, שמות משתנים רגישים לאותיות רישיות/קטנות (case-sensitive), כלומר שהמפרש (interpreter) מחשיב את `variable` ואת `Variable` כשונים. בהתחשב במידע זה, איזו מהטענות הבאות נכונה, אם `variable` ו-`Variable` הוגדרו כמספרים שלמים בתסריט Python?

- א. `print(variable)` ו-`print(Variable)` ידפיסו את אותו ערך, אם לשני המשתנים יש אותו ערך.
- ב. תתרחש שגיאה המציינת ששמות המשתנים אינם יכולים להיות דומים, למעט הבדל באותיות רישיות/קטנות.
- ג. Python ידרוס אוטומטית את הערך של המשתנה הראשון (`variable`) עם הערך של השני (`Variable`) בכל התסריט.
- ד. לא ניתן להשתמש באותו שם עם אותיות רישיות/קטנות שונות עבור משתנים שונים באותו תסריט.

**תשובה נכונה: א**

**הסבר:**

ב-Python, מזהים (identifiers), כולל שמות משתנים, רגישים לאותיות רישיות/קטנות. משמעות הדבר היא שהמפרש תופס את `variable` ואת `Variable` כשני משתנים שונים לחלוטין.

*   **אפשרות א'** נכונה: אם לשני המשתנים, `variable` ו-`Variable`, יש אותו ערך מספרי, הפלט שלהם על ידי הפונקציה `print()` יציג ערך זה, ללא תלות בשם המשתנה. Python לא יתייחס אליהם כאותו משתנה בשל הבדלי הרישיות.

*   **אפשרות ב'** לא נכונה: Python לא יפיק שגיאה ששמות המשתנים אינם יכולים להיות דומים, למעט הבדל באותיות רישיות/קטנות. Python מפרש אותם כשני משתנים נפרדים.

*   **אפשרות ג'** לא נכונה: Python לא ידרוס אוטומטית את הערך של `variable` עם הערך של `Variable`. הם ימשיכו להתקיים כשני משתנים נפרדים.

*   **אפשרות ד'** לא נכונה: Python מאפשר להשתמש באותם שמות עם אותיות רישיות/קטנות שונות עבור משתנים שונים באותו תסריט.

**דוגמה:**

```python
variable: int = 10
Variable: int = 10

print(f"הערך של variable: {variable}")   # פלט: הערך של variable: 10
print(f"הערך של Variable: {Variable}")   # פלט: הערך של Variable: 10

Variable: int = 20 # משנים את הערך של משתנה אחד בלבד

print(f"הערך של variable לאחר שינוי Variable: {variable}")   # פלט: הערך של variable לאחר שינוי Variable: 10
print(f"הערך של Variable לאחר השינוי: {Variable}")  # פלט: הערך של Variable לאחר השינוי: 20
```

**לסיכום:**

כפי שניתן לראות בדוגמה, `variable` ו-`Variable` הם שני משתנים שונים, גם אם יש להם כתיב דומה עם אותיות רישיות/קטנות שונות ובתחילה הכילו את אותו ערך. כשמשנים את הערך של `Variable`, הערך של המשתנה `variable` אינו משתנה.

לכן, **אפשרות א'** נכונה, מכיוון שהיא מתארת בדיוק את ההתנהגות של Python ביחס למשתנים רגישים לאותיות רישיות/קטנות.
