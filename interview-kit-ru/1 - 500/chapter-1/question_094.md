**שאלה 94.** מהי הפלט של קטע הקוד הבא?

```python
x = ['Python', 'is', 'awesome']
print(''.join(x))
```

- A.  `Pythonisawesome`
- B.  `Python is awesome`
- C.  `['Python', 'is', 'awesome']`
- D.  `None`

**תשובה נכונה: A**

**הסבר:**

ב-Python, מתודת ה-`join()` משמשת לאיחוד רכיבים של מבנה איטרבילי (למשל, רשימת מחרוזות) למחרוזת אחת באמצעות מפריד. במקרה זה, מתודת ה-`join()` נקראת על מחרוזת ריקה `''`, מה שאומר שכל רכיבי הרשימה `x` יאוחדו ללא מפרידים כלשהם.

*   **אפשרות A נכונה**: `join()` ללא מפריד יאחד את כל המחרוזות לאחת: `"Pythonisawesome"`
*   **אפשרות B אינה נכונה**: `join()` עם רווח כמפריד יפיק `"Python is awesome"`, אך לא בדוגמה זו.
*   **אפשרות C אינה נכונה**: הפונקציה `join()` לא תחזיר רשימה, אלא מחרוזת.
*   **אפשרות D אינה נכונה**: הפונקציה `join()` לא תחזיר `None`.

**כיצד פועלת `join()`:**

1.  מתודת ה-`join()` נקראת על אובייקט מחרוזת (המפריד).
2.  היא מקבלת אובייקט איטרבילי (רשימה, טאפל, קבוצה) כארגומנט.
3.  היא מאחדת את כל רכיבי האובייקט האיטרבילי למחרוזת אחת, תוך שימוש במחרוזת שעליה נקראה המתודה, כמפריד.

**דוגמה:**

```python
x: list[str] = ['Python', 'is', 'awesome']

result: str = ''.join(x)
print(f"Результат join(''): {result}")  # פלט: תוצאת join(''): Pythonisawesome

result: str = ' '.join(x)
print(f"Результат join(' '): {result}")  # פלט: תוצאת join(' '): Python is awesome
```

**לסיכום:**

*   `''.join(x)` מאחדת את רכיבי הרשימה `x` למחרוזת אחת ללא רווחים.
*  `' '.join(x)` מאחדת את רכיבי הרשימה `x` למחרוזת אחת עם רווחים בין הרכיבים.

לפיכך, **אפשרות A** היא הנכונה.
