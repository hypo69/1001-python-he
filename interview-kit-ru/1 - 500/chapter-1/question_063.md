**שאלה 063.** מהי תבנית העיצוב הפונקציונלית "מפה" (Map) וכיצד היא משמשת בפייתון? הצג דוגמה לשימוש בתבנית זו להמרת איברי אוסף.

-   א. תבנית "מפה" היא דרך למיין איברים באוסף, תוך שימוש בפונקציה נתונה.
-   ב. תבנית "מפה" היא דרך לסנן איברים באוסף, תוך שימוש בתנאי נתון.
-   ג. תבנית "מפה" היא דרך לאחד איברי אוסף לערך יחיד, תוך שימוש בפונקציה נתונה.
-   ד. תבנית "מפה" היא דרך ליישם פונקציה על כל איבר באוסף וליצור אוסף חדש מהתוצאות שהתקבלו.

**תשובה נכונה: ד**

**הסבר:**

תבנית "מפה" (Map) היא תבנית פונקציונלית נפוצה המשמשת להמרת כל איבר באוסף (כגון רשימה, טיפל) לאיבר חדש באמצעות פונקציה מוגדרת. התוצאה היא אוסף חדש המכיל את האיברים שהומרו.

*   **מושגי יסוד של תבנית "מפה":**
    *   **יישום פונקציה:** פונקציית ה"מפה" מיושמת על כל איבר באוסף הקלט.
    *   **אוסף חדש:** תוצאות יישום הפונקציה נשמרות באוסף חדש.
    *   **שמירת הסדר:** בדרך כלל הסדר של האיברים נשמר (אם רלוונטי למבנה הנתונים).
    *   **שימוש עם פונקציות מסדר גבוה:** לרוב מיושמת באמצעות פונקציות מסדר גבוה, כגון `map()` בפייתון.

*   **כיצד פועלת תבנית "מפה":**
    1. מקבלת אוסף קלט של איברים.
    2. מקבלת פונקציית המרה.
    3. מיישמת את הפונקציה על כל איבר באוסף.
    4. אוספת את תוצאות ההמרה לאוסף חדש.
    5. מחזירה את האוסף החדש עם הערכים שהומרו.

**דוגמאות:**

```python
# Пример использования map с обычной функцией
# דוגמה לשימוש ב-map עם פונקציה רגילה
def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(square, numbers))
print(f"Исходный список: {numbers}") # רשימה מקורית
print(f"Список с квадратами: {squared_numbers}")  # רשימת הריבועים: [1, 4, 9, 16, 25]

# Пример использования map с лямбда функцией
# דוגמה לשימוש ב-map עם פונקציית למבדה
numbers = [1, 2, 3, 4, 5]
cubed_numbers = list(map(lambda x: x ** 3, numbers))
print(f"Исходный список: {numbers}") # רשימה מקורית
print(f"Список с кубами: {cubed_numbers}")  # רשימת החזקות בשלישית: [1, 8, 27, 64, 125]

# Пример использования map с преобразованием строк
# דוגמה לשימוש ב-map עם המרת מחרוזות
strings = ["apple", "banana", "cherry"]
upper_strings = list(map(lambda x: x.upper(), strings))
print(f"Исходный список: {strings}") # רשימה מקורית
print(f"Список строк в верхнем регистре: {upper_strings}")  # רשימת מחרוזות באותיות גדולות: ['APPLE', 'BANANA', 'CHERRY']

# Пример использования map с несколькими коллекциями
# דוגמה לשימוש ב-map עם מספר אוספים
list1 = [1, 2, 3]
list2 = [4, 5, 6]

sums_list = list(map(lambda x, y: x + y, list1, list2))
print(f"Список 1: {list1}") # רשימה 1
print(f"Список 2: {list2}") # רשימה 2
print(f"Список сумм: {sums_list}") # רשימת הסכומים: [5, 7, 9]


# Пример использования map с функцией, которая возвращает кортеж
# דוגמה לשימוש ב-map עם פונקציה המחזירה טיפל
strings = ["apple", "banana", "cherry"]
indexed_strings = list(map(lambda s, i: (i, s), strings, range(len(strings))))
print(f"Исходный список: {strings}") # רשימה מקורית
print(f"Список проиндексированных строк: {indexed_strings}")  # רשימת מחרוזות ממוספרות: [(0, 'apple'), (1, 'banana'), (2, 'cherry')]


# Пример использования map с различными типами данных в списке
# דוגמה לשימוש ב-map עם סוגי נתונים שונים ברשימה
mixed_list = [1, "two", 3.0, True]
types = list(map(lambda x: type(x), mixed_list))
print(f"Исходный список: {mixed_list}") # רשימה מקורית
print(f"Список типов: {types}") # רשימת הסוגים: [<class 'int'>, <class 'str'>, <class 'float'>, <class 'bool'>]
```

    *  הפונקציה `map(square, numbers)` מיישמת את הפונקציה `square` על כל איבר ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    * מודפסות הרשימה המקורית והרשימה החדשה.
2.  **שימוש עם פונקציית למבדה:**

    *  הפונקציה `map(lambda x: x ** 3, numbers)` מיישמת את הלמבדה על כל איבר ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    *  מודפסות הרשימה המקורית והרשימה החדשה.
3. **שימוש ב-`map` עם המרת מחרוזות:**
     *  פונקציית הלמבדה `lambda x: x.upper()` ממירה מחרוזת לאותיות גדולות.
     * הפונקציה `map(lambda x: x.upper(), strings)` מיישמת את פונקציית הלמבדה על כל מחרוזת ברשימה `strings`.
     * מודפסות הרשימה המקורית והרשימה החדשה.
4.  **שימוש ב-`map` עם מספר אוספים:**
     *  פונקציית הלמבדה `lambda x,y: x+y` מסכמת שני מספרים.
     *  הפונקציה `map(lambda x,y: x+y, list1, list2)` מיישמת את פונקציית הלמבדה על כל איבר ברשימות `list1` ו-`list2` (לוקחת איברים עם אותו אינדקס).
     *  מודפסות הרשימות המקוריות ורשימת הסכומים.

       * פונקציית הלמבדה `lambda s, i: (i, s)` יוצרת זוג (אינדקס, ערך) עבור כל מחרוזת ברשימה.
       * הפונקציה `map` יוצרת רשימה של טיפלים.
     *  מודפסות הרשימה המקורית ורשימה עם מחרוזות ממוספרות.
6.  **שימוש ב-`map` עם סוגי נתונים שונים:**
      *  פונקציית הלמבדה `lambda x: type(x)` מחזירה את סוג האובייקט.
      *  הפונקציה `map` מיישמת פונקציה זו על כל איבר ברשימה, מה שמוביל ליצירת רשימת סוגים.
      *  מודפסות הרשימה המקורית ורשימת הסוגים.
