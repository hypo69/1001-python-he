**שאלה 063.** מהי תבנית העיצוב הפונקציונלית "Map" (מיפוי) וכיצד היא משמשת ב-Python? הבא דוגמה לשימוש בתבנית זו להמרת איברים באוסף.

-   A. תבנית "Map" היא שיטה למיון איברים באוסף, באמצעות פונקציה נתונה.
-   B. תבנית "Map" היא שיטה לסינון איברים באוסף, באמצעות תנאי נתון.
-   C. תבנית "Map" היא שיטה לאיחוד איברי אוסף לערך יחיד, באמצעות פונקציה נתונה.
-   D. תבנית "Map" היא שיטה להחלת פונקציה על כל איבר באוסף ויצירת אוסף חדש מהתוצאות שהתקבלו.

**תשובה נכונה: D**

**הסבר:**

תבנית "Map" (מיפוי) היא תבנית פונקציונלית נפוצה המשמשת להמרת כל איבר באוסף (למשל, רשימה, טאפל) לאיבר חדש תוך שימוש בפונקציה מוגדרת. התוצאה היא אוסף חדש המכיל את האיברים המומרים.

*   **מושגי יסוד של תבנית "Map":**
    *   **החלת פונקציה:** פונקציית ה"מיפוי" מוחלת על כל איבר באוסף הקלט.
    *   **אוסף חדש:** תוצאות החלת הפונקציה נשמרות באוסף חדש.
    *   **שימור סדר:** בדרך כלל נשמר סדר האיברים (אם הדבר רלוונטי למבנה הנתונים).
    *   **שימוש עם פונקציות מסדר גבוה:** לעיתים קרובות מיושמת באמצעות פונקציות מסדר גבוה, כגון `map()` ב-Python.

*   **כיצד פועלת תבנית "Map":**
    1. מקבלת אוסף קלט של איברים.
    2. מקבלת פונקציית טרנספורמציה.
    3. מחילה את הפונקציה על כל איבר באוסף.
    4. אוספת את תוצאות הטרנספורמציה לאוסף חדש.
    5. מחזירה את האוסף החדש עם הערכים המומרים.

**דוגמאות:**

```python
# דוגמה לשימוש ב-map עם פונקציה רגילה
def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(square, numbers))
print(f"רשימה מקורית: {numbers}")
print(f"רשימה עם ריבועים: {squared_numbers}")  # פלט: [1, 4, 9, 16, 25]

# דוגמה לשימוש ב-map עם פונקציית למדא
numbers = [1, 2, 3, 4, 5]
cubed_numbers = list(map(lambda x: x ** 3, numbers))
print(f"רשימה מקורית: {numbers}")
print(f"רשימה עם מעוקבים: {cubed_numbers}")  # פלט: [1, 8, 27, 64, 125]

# דוגמה לשימוש ב-map עם טרנספורמציה של מחרוזות
strings = ["apple", "banana", "cherry"]
upper_strings = list(map(lambda x: x.upper(), strings))
print(f"רשימה מקורית: {strings}")
print(f"רשימת מחרוזות באותיות גדולות: {upper_strings}")  # פלט: ['APPLE', 'BANANA', 'CHERRY']

# דוגמה לשימוש ב-map עם מספר אוספים
list1 = [1, 2, 3]
list2 = [4, 5, 6]

sums_list = list(map(lambda x, y: x + y, list1, list2))
print(f"רשימה 1: {list1}")
print(f"רשימה 2: {list2}")
print(f"רשימת סכומים: {sums_list}") # פלט: [5, 7, 9]


# דוגמה לשימוש ב-map עם פונקציה שמחזירה טאפל
strings = ["apple", "banana", "cherry"]
indexed_strings = list(map(lambda s, i: (i, s), strings, range(len(strings))))
print(f"רשימה מקורית: {strings}")
print(f"רשימת מחרוזות עם אינדקס: {indexed_strings}")  # פלט: [(0, 'apple'), (1, 'banana'), (2, 'cherry')]


# דוגמה לשימוש ב-map עם טיפוסי נתונים שונים ברשימה
mixed_list = [1, "two", 3.0, True]
types = list(map(lambda x: type(x), mixed_list))
print(f"רשימה מקורית: {mixed_list}")
print(f"רשימת טיפוסים: {types}") # יפלט: [<class 'int'>, <class 'str'>, <class 'float'>, <class 'bool'>]
```

**ניתוח הדוגמאות:**
1.  **שימוש עם פונקציה רגילה `square`:**
    *   הפונקציה `square(x)` מחזירה את הריבוע של מספר.
    *  הפונקציה `map(square, numbers)` מחילה את הפונקציה `square` על כל איבר ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    *  מדפיס את הרשימה המקורית ואת הרשימה החדשה.
2.  **שימוש עם פונקציית למדא:**
      *  פונקציית הלמדא `lambda x: x ** 3` מחשבת את המעוקב של מספר.
    *  הפונקציה `map(lambda x: x ** 3, numbers)` מחילה את פונקציית הלמדא על כל איבר ברשימה `numbers` ומחזירה איטרטור. לאחר מכן, האיטרטור מומר לרשימה.
    *  מדפיס את הרשימה המקורית ואת הרשימה החדשה.
3. **שימוש ב-`map` עם טרנספורמציה של מחרוזות:**
     *  פונקציית הלמדא `lambda x: x.upper()` ממירה מחרוזת לאותיות גדולות.
     * הפונקציה `map(lambda x: x.upper(), strings)` מחילה את פונקציית הלמדא על כל מחרוזת ברשימה `strings`.
     *  מדפיס את הרשימה המקורית ואת הרשימה החדשה.
4.  **שימוש ב-`map` עם מספר אוספים:**
     *  פונקציית הלמדא `lambda x,y: x+y` מסכמת שני מספרים.
     *  הפונקציה `map(lambda x,y: x+y, list1, list2)` מחילה את פונקציית הלמדא על כל איבר ברשימות `list1` ו-`list2` (לוקחת איברים עם אותו אינדקס).
     *  מדפיס את הרשימות המקוריות ואת רשימת הסכומים.
5.  **שימוש ב-`map` עם פונקציה שמחזירה טאפל:**
       *  פונקציית הלמדא `lambda s, i: (i, s)` יוצרת זוג (אינדקס, ערך) עבור כל מחרוזת ברשימה.
       * הפונקציה map יוצרת רשימה של טאפלים.
     *  מדפיס את הרשימה המקורית ואת רשימת המחרוזות עם אינדקס.
6.  **שימוש ב-`map` עם טיפוסי נתונים שונים:**
      *  פונקציית הלמדא `lambda x: type(x)` מחזירה את הטיפוס של האובייקט.
      *  הפונקציה `map` מחילה פונקציה זו על כל איבר ברשימה, מה שמוביל ליצירת רשימת טיפוסים.
      *  מדפיס את הרשימה המקורית ואת רשימת הטיפוסים.

**ניתוח האפשרויות:**
*   **A. תבנית "Map" היא שיטה למיון איברים באוסף, באמצעות פונקציה נתונה.:** לא נכון. מיון הוא פעולה שונה.
*  **B. תבנית "Map" היא שיטה לסינון איברים באוסף, באמצעות תנאי נתון.:** לא נכון. סינון משמש לבחירת איברים לפי תנאי.
*   **C. תבנית "Map" היא שיטה לאיחוד איברי אוסף לערך יחיד, באמצעות פונקציה נתונה.:** לא נכון. זהו תיאור של תבנית Reduce.
*   **D. תבנית "Map" היא שיטה להחלת פונקציה על כל איבר באוסף ויצירת אוסף חדש מהתוצאות שהתקבלו.:** נכון.

**לסיכום:**
*  תבנית "Map" מאפשרת ליצור רשימה חדשה על ידי המרת איברי הרשימה המקורית באמצעות פונקציית טרנספורמציה.
*  באמצעות הפונקציה `map` ופונקציות למדא ניתן ליישם בקלות את תבנית Map.
*  הפונקציה `map` מאפשרת להחיל פונקציה על מספר רשימות בו זמנית.
*  הפונקציה `map` יכולה לשמש עם פונקציות המחזירות כל ערך (כולל טאפלים), ויכולה לקבל כל טיפוס נתונים.
