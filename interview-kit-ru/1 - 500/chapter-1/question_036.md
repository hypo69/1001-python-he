### `question_036.md`

**שאלה 36.** בהתייחס לשימוש במתודה `__init__` בהגדרות מחלקות ב-Python, כיצד היא מתפקדת במחזור החיים של אובייקט לצורך אתחול מצב או הגדרת מאפיינים נחוצים מיד לאחר יצירת מופע?

- א. המתודה `__init__` נקראת אוטומטית בכל פעם שמחלקה יורשת ממחלקה אחרת, כדי להבטיח שהמחלקה הנגזרת יורשת את כל המאפיינים והמתודות של מחלקת האב ללא צורך בהצהרה מפורשת.
- ב. היא פועלת כקונסטרוקטור עבור המחלקה, ונקראת אוטומטית בעת יצירת מופע חדש של אובייקט כדי לאתחל את מאפייני המופע או לבצע כל הגדרה ראשונית נחוצה אחרת.
- ג. מתודה זו משמשת כדסטרוקטור (destructor) עבור מופעי המחלקה, האחראית על שחרור משאבים שהאובייקט עשוי היה להחזיק במהלך חייו.
- ד. המתודה `__init__` ב-Python נקראת בכל פעם שאובייקט מועתק, ומבטיחה שהמופע החדש מקבל עותק נפרד של המצב ההתחלתי של האובייקט.

**תשובה נכונה: ב**

**הסבר:**

המתודה `__init__` ב-Python היא הקונסטרוקטור של המחלקה. היא נקראת אוטומטית בעת יצירת מופע (אובייקט) חדש של המחלקה.

*   **אפשרות א** אינה נכונה: `__init__` אינה קשורה לירושה. היא נקראת לצורך אתחול האובייקט שנוצר.
*   **אפשרות ב** נכונה: `__init__` היא הקונסטרוקטור המאתחל את האובייקט, מספקת מצב התחלתי או מאפיינים נחוצים.
*   **אפשרות ג** אינה נכונה: דסטרוקטורים ב-Python מטופלים על ידי המתודה `__del__`, ולא על ידי `__init__`.
*   **אפשרות ד** אינה נכונה: המתודה `__init__` אינה נקראת בעת העתקת אובייקט; היא נקראת בעת יצירת אובייקט.

**נקודות מפתח לגבי `__init__`:**

*   **קריאה אוטומטית:** היא נקראת אוטומטית על ידי מפרש Python בעת יצירת מופע חדש של מחלקה.
*   **אתחול:** המטרה העיקרית היא לאתחל את מאפייני האובייקט עם ערכים התחלתיים.
*   **הגדרה:** בתוך מתודה זו ניתן לבצע כל הגדרה נחוצה או עיבוד מקדים של נתונים הדרושים לתפקוד האובייקט.
*   **פרמטרים:** היא מקבלת את הארגומנט `self`, המתייחס לאובייקט שנוצר, וכן כל פרמטרים נוספים שיש להעביר בעת יצירת המופע.

**דוגמה:**

```python
class Person:
    def __init__(self, name: str, age: int):
        """קונסטרוקטור עבור המחלקה Person"""
        self.name = name  # אתחול המאפיין name
        self.age = age   # אתחול המאפיין age
        print(f"Объект {self.name} создан") # אובייקט {self.name} נוצר
    def greet(self) -> None:
        print(f"Привет, меня зовут {self.name} и мне {self.age} лет.") # שלום, שמי {self.name} ואני בן {self.age} שנים.

# יצירת מופע של המחלקה
person1: Person = Person("Alice", 30)  # פלט: Объект Alice создан # פלט: אובייקט Alice נוצר
person1.greet() # פלט: Привет, меня зовут Alice и мне 30 лет. # פלט: שלום, שמי Alice ואני בן 30 שנים.
```

**כתוצאה מכך:**

*   כאשר אנו יוצרים את האובייקט `person1`, המתודה `__init__` נקראת אוטומטית.
*   בתוך המתודה `__init__`, אנו יכולים לאתחל את `self.name` ואת `self.age`.
*   לאחר מכן ניתן להשתמש באובייקט על ידי קריאה למתודה `greet()`.

לכן, **אפשרות ב** היא הנכונה, מכיוון שהיא מתארת בדיוק את תפקיד המתודה `__init__` כקונסטרוקטור עבור המחלקה.
