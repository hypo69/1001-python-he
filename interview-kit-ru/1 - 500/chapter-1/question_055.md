### `question_55.md`

**שאלה 55.** בהינתן קטע הקוד הבא בשפת פייתון, מהי ההתנהגות הצפויה של התוכנית?

```python
for i in range(5):
    if i == 3:
        break
    print(i)
```

- א. התוכנית מדפיסה את המספרים מ-0 עד 4 ללא הפרעה.
- ב. היא מדפיסה את המספרים מ-0 עד 2, ולאחר מכן עוצרת לפני הדפסת המספר 3.
- ג. התוכנית מחזירה שגיאה מכיוון שהפקודה `break` אינה בשימוש נכון מחוץ ללולאה.
- ד. היא מדפיסה ברציפות את המספר 3 בלולאה אינסופית.

**תשובה נכונה: ב**

**הסבר:**

בקוד זה נעשה שימוש בלולאת `for` ובפקודה `break`. הבה ננתח את פעולת הקוד צעד אחר צעד.

1.  **לולאת `for`**: `for i in range(5):` - לולאה זו עוברת באיטרציות על המספרים בטווח שבין 0 ל-4 כולל (range(5) יוצר סדרה 0, 1, 2, 3, 4).
2.  **תנאי:** `if i == 3:` - בכל איטרציה נבדק האם הערך הנוכחי של משתנה הלולאה `i` שווה למספר 3.
3.  **פקודת `break`:** אם התנאי `i == 3` מתקיים, מופעלת הפקודה `break`, שמסיימת את הלולאה באופן מיידי.
4.  **פלט:** `print(i)` - ערכו של המשתנה `i` מודפס לפני בדיקת התנאי.

*   אפשרות א' אינה נכונה: הלולאה לא תתבצע עד סופה, מכיוון שישנה פקודת `break`.

*   אפשרות ב' נכונה: הלולאה תבצע איטרציות מ-0 עד 4, אך כאשר i == 3 הלולאה תופסק, ולכן יודפסו הערכים מ-0 עד 2 כולל.

*   אפשרות ג' אינה נכונה: הפקודה `break` נמצאת בשימוש נכון בתוך הלולאה.

*   אפשרות ד' אינה נכונה: הלולאה לא תהיה אינסופית, מכיוון שפקודת `break` מפסיקה את ביצועה.

**דוגמה:**
```python
for i in range(5):
    if i == 3:
        break
    print(i)
# פלט:
# 0
# 1
# 2
```
**לסיכום:**

הלולאה תתחיל איטרציות כאשר `i = 0`. יודפסו המספרים 0, 1, 2. כאשר `i` יהיה שווה ל-`3`, תתבצע הפקודה `break`, והלולאה תסתיים, והמספר 3 לא יודפס.

לפיכך, **אפשרות ב'** היא התשובה הנכונה.
