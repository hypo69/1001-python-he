**שאלה 55.** בהינתן קטע קוד Python הבא, מהי ההתנהגות הצפויה של התוכנית?

```python
for i in range(5):
    if i == 3:
        break
    print(i)
```

- א. התוכנית מדפיסה את המספרים מ-0 עד 4 ללא הפרעה.
- ב. היא מדפיסה את המספרים מ-0 עד 2, ואז נעצרת לפני הדפסת 3.
- ג. התוכנית מציגה שגיאה, מכיוון שהאופרטור `break` אינו בשימוש נכון מחוץ ללולאה.
- ד. היא מדפיסה את המספר 3 ברציפות בלולאה אינסופית.

**תשובה נכונה: ב**

**הסבר:**

בקטע קוד זה נעשה שימוש בלולאת `for` ובאופרטור `break`. ננתח את פעולת הקוד צעד אחר צעד.

1.  **לולאת `for`**: `for i in range(5):` - לולאה זו מבצעת איטרציה על המספרים בטווח מ-0 עד 4 כולל (range(5) יוצר את הרצף 0, 1, 2, 3, 4).
2.  **תנאי:** `if i == 3:` - בכל איטרציה נבדק האם הערך הנוכחי של משתנה הלולאה `i` שווה ל-3.
3.  **אופרטור `break`:** אם התנאי `i == 3` מתקיים, נקרא האופרטור `break`, שמסיים את הלולאה באופן מיידי.
4.  **פלט:** `print(i)` - הערך של משתנה הלולאה `i` מודפס לפני בדיקת התנאי.

*   **אפשרות א** אינה נכונה: הלולאה לא תתבצע עד סופה, מכיוון שקיים אופרטור `break`.

*   **אפשרות ב** נכונה: הלולאה תבצע איטרציה מ-0 עד 4, אך כאשר i == 3, הלולאה תיקטע, ולכן יודפסו הערכים מ-0 עד 2 כולל.

*   **אפשרות ג** אינה נכונה: האופרטור `break` נמצא בשימוש נכון בתוך הלולאה.

*   **אפשרות ד** אינה נכונה: הלולאה לא תהיה אינסופית, מכיוון ש-`break` מפסיק את ביצועה.

**דוגמה:**
```python
for i in range(5):
    if i == 3:
        break
    print(i)
# פלט:
# 0
# 1
# 2
```
**לסיכום:**

הלולאה תתחיל איטרציות עם `i = 0`. יודפסו המספרים 0, 1, 2. כאשר `i` יהפוך ל-`3`, יתבצע `break`, והלולאה תסתיים, והמספר 3 לא יודפס.

לפיכך, **אפשרות ב** היא התשובה הנכונה.
