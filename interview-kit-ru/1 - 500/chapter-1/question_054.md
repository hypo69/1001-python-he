**שאלה 054.** נתון מערך של מחרוזות `words`. פתח אלגוריתם שמחזיר רשימה חדשה המכילה רק את המחרוזות שניתן להקליד באמצעות מקשים משורה אחת בלבד במקלדת QWERTY סטנדרטית.

**דוגמה:**
```
קלט: words = ["Hello","Alaska","Dad","Peace"]
פלט: ["Alaska","Dad"]

קלט: words = ["omk"]
פלט: []
```

-   A. לפתרון הבעיה יש להשתמש בלולאות מקוננות ולבדוק עבור כל מילה, באיזו שורה היא נמצאת.
-   B. לפתרון הבעיה יש להשתמש בביטויים רגולריים.
-   C. לפתרון הבעיה יש ליצור קבוצת תווים עבור כל שורת מקלדת ולבדוק עבור כל מילה שכל התווים שלה נמצאים רק באחת מקבוצות אלה.
-   D. לפתרון הבעיה ניתן תחילה למיין את רשימת המילים לפי אורכן ולאחר מכן לבדוק אותן.

**תשובה נכונה: C**

**הסבר:**

לפתרון בעיית סינון המילים שניתן להקליד בשורת מקלדת אחת, הגישה האופטימלית היא יצירת קבוצות עבור כל שורה ובדיקה עוקבת של כל מילה האם תויה שייכים לקבוצה אחת בלבד.

*   **אלגוריתם המבוסס על קבוצות:**
    1.  **יצירת קבוצות:** יוצרים קבוצות (sets) של תווים עבור כל שורת מקלדת.
    2.  **מעבר על מילים:** עוברים על כל מילה במערך `words`.
    3.  **בדיקת השתייכות לשורה:** בודקים שכל התווים של כל מילה שייכים לאחת הקבוצות.
        *   לשם כך, ממירים את המילה לקבוצה ובודקים אם היא תת-קבוצה של אחת מקבוצות השורות (באמצעות issubset).
        *   אם תנאי זה מתקיים עבור אחת השורות, מוסיפים את המילה לרשימת התוצאה.
    4.  **תוצאה:** מחזירים את רשימת המחרוזות המסוננת.

*   **יתרונות האלגוריתם:**
    *   **חיפוש יעיל:** קבוצות מספקות חיפוש מהיר (מורכבות זמן ממוצעת של (O(1) בעת בדיקת הימצאות איברים.
    *   **פשטות:** הקוד הופך קריא ותמציתי יותר.

**דוגמאות (פסאודו-קוד):**
```
function find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
```
**דוגמאות מימוש ב-Python:**
```python
def find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
words = ["Hello","Alaska","Dad","Peace"]
print(f"קלט: words = {words}")
print(f"פלט: {find_words_from_keyboard_row(words)}") # יציג ['Alaska', 'Dad']

words2 = ["omk"]
print(f"קלט: words = {words2}")
print(f"פלט: {find_words_from_keyboard_row(words2)}")  # יציג: []

words3 = ["adsf","qwer","mxcv","asdfghjkl", "qwertyuiop"]
print(f"קלט: words = {words3}")
print(f"פלט: {find_words_from_keyboard_row(words3)}") # יציג: ['adsf', 'qwer', 'mxcv', 'asdfghjkl', 'qwertyuiop']
```
