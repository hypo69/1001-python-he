**שאלה 054.** נתונה מערך מחרוזות `words`. פתח אלגוריתם המחזיר רשימה חדשה המכילה רק את המחרוזות שניתן להקליד באמצעות מקשים משורה אחת בלבד במקלדת QWERTY סטנדרטית.

**דוגמה:**
```
קלט: words = ["Hello","Alaska","Dad","Peace"]
פלט: ["Alaska","Dad"]

קלט: words = ["omk"]
פלט: []
```

- א. לפתרון הבעיה יש להשתמש בלולאות מקוננות ולבדוק עבור כל מילה מאיזו שורה היא.
- ב. לפתרון הבעיה יש להשתמש בביטויים רגולריים.
- ג. לפתרון הבעיה יש ליצור קבוצת תווים עבור כל שורת מקלדת ולבדוק עבור כל מילה שכל התווים שלה נמצאים באחת הקבוצות הללו בלבד.
- ד. לפתרון הבעיה ניתן למיין קודם את רשימת המילים לפי אורכן ולאחר מכן לבדוק אותן.

**תשובה נכונה: ג**

**הסבר:**

לפתרון בעיית סינון המילים שניתן להקליד על שורת מקלדת אחת, הגישה האופטימלית היא יצירת קבוצות עבור כל שורה ולאחר מכן בדיקת כל מילה לגבי שייכות התווים שלה לקבוצה אחת.

*   **אלגוריתם עם קבוצות:**
    1.  **יצירת קבוצות:** נוצרות קבוצות (sets) של תווים עבור כל שורת מקלדת.
    2.  **מעבר על המילים:** עוברים על כל מילה במערך `words`.
    3.  **בדיקת שייכות לשורה:** בודקים שכל התווים של כל מילה שייכים לאחת הקבוצות.
        *   לשם כך, ממירים את המילה לקבוצה ובודקים שהחיתוך עם כל שורת מקלדת נותן את קבוצת תווים המילה, או קבוצה ריקה, מכיוון שהיא אמורה להיות כלולה בשורה אחת בלבד.
        *   אם תנאי זה מתקיים, מוסיפים את המילה לרשימה התוצאתית.
    4.  **תוצאה:** מחזירים את רשימת המחרוזות המסוננת.

*   **יתרונות האלגוריתם:**
    *   **חיפוש יעיל:** קבוצות מאפשרות חיפוש מהיר (סיבוכיות זמן ממוצעת O(1)) בעת בדיקת קיום איברים.
    *   **פשטות:** הקוד הופך קריא ותמציתי יותר.

**דוגמאות (פסאודו-קוד):**
```
function find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
```
**דוגמאות למימוש ב-Python:**
```python
def find_words_from_keyboard_row(words):
    row1 = {'q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'}
    row2 = {'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'}
    row3 = {'z', 'x', 'c', 'v', 'b', 'n', 'm'}

    result = []
    for word in words:
       word_set = set(word.lower())
       if word_set.issubset(row1) or  word_set.issubset(row2) or word_set.issubset(row3):
         result.append(word)
    return result
words = ["Hello","Alaska","Dad","Peace"]
print(f"Вход: words = {words}")
print(f"Выход: {find_words_from_keyboard_row(words)}")  # Will print ['Alaska', 'Dad']

words2 = ["omk"]
print(f"Вход: words = {words2}")
print(f"Выход: {find_words_from_keyboard_row(words2)}")  # Will print: []

words3 = ["adsf","qwer","mxcv","asdfghjkl", "qwertyuiop"]
print(f"Вход: words = {words3}")
print(f"Выход: {find_words_from_keyboard_row(words3)}") # Will print: ['adsf', 'qwer', 'mxcv', 'asdfghjkl', 'qwertyuiop']
```
