**שאלה 29.** נתון מילון Python הבא, כיצד ניתן לגשת לערך המשויך למפתח `'color'`?

```python
car = {"brand": "Ford", "model": "Mustang", "year": 1964, "color": "red"}
```

- A. `car[1]`
- B. `car.get("color")`
- C. `car[color]`
- D. `car['color']`

**תשובה נכונה: D**

**הסבר:**

ב-Python, גישה לערכים במילון מתבצעת באמצעות המפתחות, הנתונים בסוגריים מרובעים.

*   **אפשרות A** אינה נכונה: `car[1]` - תנסה לגשת לאיבר לפי אינדקס מספרי `1`, ולא לפי המפתח `'color'`. הדבר יוביל לשגיאה, מכיוון שמילונים אינם תומכים באינדקסציה מספרית.

*   **אפשרות B** נכונה: `car.get("color")` - זוהי דרך נכונה לגשת לערך לפי מפתח באמצעות המתודה `get()`, אשר גם מאפשרת להחזיר ערך ברירת מחדל במקרה שהמפתח אינו קיים.

*   **אפשרות C** אינה נכונה: `car[color]` - תנסה להשתמש במשתנה בשם `color`, ולא במפתח המחרוזתי `"color"`. הדבר יוביל לשגיאה אם המשתנה `color` אינו מוכרז.

*   **אפשרות D** נכונה: `car['color']` - זוהי הדרך הישירה והנפוצה ביותר לגשת לערך לפי מפתח, כשהמפתח נתון במירכאות.

**דוגמה:**

```python
car: dict[str, str | int] = {"brand": "Ford", "model": "Mustang", "year": 1964, "color": "red"}

# דרך נכונה (אפשרות D)
color_value_d: str = car['color']
print(f"הערך של המפתח 'color': {color_value_d}") # פלט: הערך של המפתח 'color': red

# דרך נכונה (אפשרות B)
color_value_b: str = car.get("color")
print(f"הערך של המפתח 'color': {color_value_b}")  # פלט: הערך של המפתח 'color': red

# דרך שגויה (אפשרות A) - תגרום לשגיאת KeyError
try:
    color_value_a: str = car[1]
except KeyError as e:
    print(e) # פלט: 1

# דרך שגויה (אפשרות C) - תגרום לשגיאת NameError אם המשתנה color אינו מוכרז
# try:
#   color_value_c = car[color] # NameError: name 'color' is not defined
# except NameError as e:
#   print(e)
```

**לסיכום:**

*   אפשרויות `B` ו-`D` מספקות את התוצאה הנכונה ומאפשרות לקבל את ערך המפתח `'color'`, ולהדפיס למסך את המחרוזת `"red"`.
*   אפשרות `A` גורמת לחריגת `KeyError`, מכיוון שבמילון אין מפתחות מספריים.
*   אפשרות `C` גורמת לחריגת `NameError`, אם המשתנה `color` אינו מוכרז.

לפיכך, **אפשרות D** היא הדרך הנפוצה והישירה ביותר לגשת לערך לפי מפתח במילון, ואילו **אפשרות B** נכונה גם היא, אך מעט יותר מפורטת.
