**שאלה 29.** נתון מילון Python הבא, כיצד תגשו לערך המשויך למפתח `'color'`?

```python
car = {"brand": "Ford", "model": "Mustang", "year": 1964, "color": "red"}
```

- A. `car[1]`
- B. `car.get("color")`
- C. `car[color]`
- D. `car['color']`

**תשובה נכונה: D**

**הסבר:**

ב-Python, על מנת לגשת לערכים במילון, משתמשים במפתחות סגורים בסוגריים מרובעים.

*   **אפשרות A** אינה נכונה: `car[1]` - ניסיון לגשת לאלמנט לפי אינדקס מספרי `1`, ולא לפי המפתח `'color'`. פעולה זו תוביל לשגיאה, מכיוון שמילונים אינם תומכים באינדקס מספרי.

*   **אפשרות B** נכונה: `car.get("color")` - זוהי דרך נכונה לגשת לערך באמצעות מפתח תוך שימוש במתודה `get()`, אשר מאפשרת גם להחזיר ערך ברירת מחדל במקרה שהמפתח אינו קיים.

*   **אפשרות C** אינה נכונה: `car[color]` - ניסיון להשתמש במשתנה `color`, ולא במפתח המחרוזתי `"color"`. פעולה זו תוביל לשגיאה אם המשתנה `color` לא הוכרז.

*   **אפשרות D** נכונה: `car['color']` - זוהי הדרך הישירה והנפוצה ביותר לגשת לערך באמצעות מפתח, הסגור במרכאות.

**דוגמה:**

```python
car: dict[str, str | int] = {"brand": "Ford", "model": "Mustang", "year": 1964, "color": "red"}

# הדרך הנכונה (אפשרות D)
color_value_d: str = car['color']
print(f"הערך של המפתח 'color': {color_value_d}") # פלט: הערך של המפתח 'color': red

# הדרך הנכונה (אפשרות B)
color_value_b: str = car.get("color")
print(f"הערך של המפתח 'color': {color_value_b}")  # פלט: הערך של המפתח 'color': red

# הדרך הלא נכונה (אפשרות A) - תגרום ל-KeyError
try:
    color_value_a: str = car[1]
except KeyError as e:
    print(e) # פלט: 1

# הדרך הלא נכונה (אפשרות C) - תגרום ל-NameError אם המשתנה color לא הוכרז
# try:
#   color_value_c = car[color] # NameError: name 'color' is not defined
# except NameError as e:
#   print(e)
```

**לסיכום:**

*   אפשרויות `B` ו-`D` נותנות תוצאה נכונה ומאפשרות לקבל את הערך של המפתח `'color'`, ומציגות על המסך את המחרוזת `"red"`.
*   אפשרות `A` גורמת לחריגה מסוג `KeyError`, מכיוון שבמילון אין מפתחות מספריים.
*   אפשרות `C` גורמת לחריגה מסוג `NameError`, אם המשתנה `color` לא הוכרז.

לפיכך, **אפשרות D** היא הדרך הנפוצה והישירה ביותר לגשת לערך באמצעות מפתח במילון, בעוד ש**אפשרות B** היא נכונה, אף כי מעט יותר מפורטת.
