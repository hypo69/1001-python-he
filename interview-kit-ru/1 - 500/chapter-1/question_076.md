**שאלה 76.** מה עושה קוד פייתון הבא, המיישם אלגוריתם מיון?

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# Пример использования
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
sorted_arr = bubble_sort(arr)
print("Sorted array:", sorted_arr)
```

-   א. ממיין את המערך בסדר הפוך.
-   ב. ממיין את המערך בסדר עולה באמצעות אלגוריתם מיון בועות.
-   ג. ממיין את המערך בסדר יורד באמצעות אלגוריתם מיון הכנסה.
-   ד. מסיר כפילויות מהמערך וממיין אותו.

**תשובה נכונה: ב**

**הסבר:**

הקוד המוצג מיישם את אלגוריתם מיון בועות (bubble sort).

*   **אלגוריתם מיון בועות:**
    *   משווה איברים סמוכים במערך.
    *   אם האיבר הנוכחי גדול מהאיבר הבא, הם מוחלפים.
    *   כתוצאה מכל מעבר, האיבר הגדול ביותר "צף" לסוף המערך.
    *   המעברים חוזרים על עצמם, ומקטינים בכל שלב את גבול החלק הנבדק במערך.

*   **תיאור הקוד:**
    *   `bubble_sort(arr)`: פונקציה שמקבלת מערך `arr` כארגומנט.
    *   `n = len(arr)`: מגדיר את אורך המערך.
    *   `for i in range(n)`: הלולאה החיצונית עוברת על המערך `n` פעמים.
    *   `for j in range(0, n-i-1)`: הלולאה הפנימית עוברת עד לגבול החלק הממוין של המערך.
    *   `if arr[j] > arr[j+1]`: בודק האם יש צורך להחליף איברים סמוכים.
    *   `arr[j], arr[j+1] = arr[j+1], arr[j]`: מחליף את האיברים הסמוכים.
    *   `return arr`: מחזיר את המערך הממוין.

**ניתוח האפשרויות:**
*   **א. ממיין את המערך בסדר הפוך:** שגוי.
*   **ב. ממיין את המערך בסדר עולה באמצעות אלגוריתם מיון בועות:** נכון.
*   **ג. ממיין את המערך בסדר יורד באמצעות אלגוריתם מיון הכנסה:** שגוי.
*   **ד. מסיר כפילויות מהמערך וממיין אותו:** שגוי.

**לסיכום:**
*   הקוד מיישם נכונה את אלגוריתם מיון בועות למיון מערך בסדר עולה.
*   מיון בועות הוא אלגוריתם מיון פשוט, אך לא היעיל ביותר.

לפיכך, התשובה הנכונה היא **ב. ממיין את המערך בסדר עולה באמצעות אלגוריתם מיון בועות.**
