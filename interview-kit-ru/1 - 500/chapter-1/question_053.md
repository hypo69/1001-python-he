### `question_053.md`

**שאלה 053.** כיצד ניתן לשלב שתי רשימות או יותר ב-Python? תאר שיטות שונות וספק דוגמאות לשימוש בהן.

-   א. לשילוב רשימות ב-Python משתמשים רק במתודה `concat()` או בפונקציה `merge()`.
-   ב. לשילוב רשימות ב-Python ניתן להשתמש באופרטור החיבור `+`, במתודה `extend()`, או בלולאות עם הוספת אלמנטים.
-   ג. לשילוב רשימות משתמשים רק במתודה `union()`, המאחדת שתי רשימות לאחת.
-   ד. לשילוב רשימות יש להשתמש רק בגנרטורי רשימות (list comprehensions).

**תשובה נכונה: ב**

**הסבר:**

ב-Python קיימות מספר דרכים לשלב (שרשר) רשימות, מה שמאפשר גמישות בעבודה עם מצבים שונים.

*   **אופרטור `+` (חיבור):**
    *   יוצר רשימה חדשה על ידי שילוב האלמנטים מהרשימות המקוריות.
    *   הרשימות המקוריות אינן משתנות.
    *   יכול לשלב שתי רשימות או יותר, אך בזוגות.
    *   דוגמה: `list1 + list2`

*   **מתודה `extend()`:**
    *   משנה את הרשימה המקורית על ידי הוספת אלמנטים מרשימה אחרת בסופה.
    *   אינה יוצרת רשימה חדשה.
    *   דוגמה: `list1.extend(list2)`

*   **לולאות ו-`append()`:**
    *   ניתן להשתמש בלולאות `for` לעבור על הרשימות ועם המתודה `append()` להוסיף אלמנטים לרשימת התוצאה.
    *   זו לא הדרך התמציתית ביותר.
*   **גנרטור רשימה (List comprehension):**
    *   ניתן להשתמש בגנרטור רשימה כדי ליצור רשימה חדשה.

**דוגמאות:**

```python
# דוגמה 1: שימוש באופרטור +
list1 = [1, 2]
list2 = [3, 4, 5]
combined_list1 = list1 + list2
print(f"רשימה 1: {list1}")
print(f"רשימה 2: {list2}")
print(f"רשימה משולבת (+): {combined_list1}") # פלט: [1, 2, 3, 4, 5]

# דוגמה 2: שימוש במתודה extend()
list3 = [1, 2]
list4 = [3, 4, 5]
print(f"רשימה 3 לפני extend: {[1,2]}")
print(f"רשימה 4: {list4}")
list3.extend(list4)
print(f"רשימה משולבת (extend): {list3}") # פלט: [1, 2, 3, 4, 5]

# דוגמה 3: שימוש בלולאת for
list5 = [1, 2]
list6 = [3,4,5]

combined_list2 = []
for item in list5:
  combined_list2.append(item)
for item in list6:
   combined_list2.append(item)
print(f"רשימה משולבת (לולאת for): {combined_list2}") # פלט: [1, 2, 3, 4, 5]


# דוגמה 4: שימוש בגנרטור רשימה (list comprehension)
list7 = [1, 2]
list8 = [3,4,5]
combined_list3 =  [item for list_ in [list7, list8] for item in list_]
print(f"רשימה משולבת (list comprehension): {combined_list3}") # פלט: [1, 2, 3, 4, 5]

# דוגמה עם מספר רשימות:
list9 = [1, 2]
list10 = [3,4,5]
list11 = [6,7]
combined_list4 = list9 + list10 + list11
print(f"רשימה משולבת (אופרטור +): {combined_list4}") # פלט: [1, 2, 3, 4, 5, 6, 7]

# דוגמה עם אופרטור +=
list12 = [1, 2]
list12 += [3,4]
print(f"רשימה משולבת (+=): {list12}") # פלט: [1, 2, 3, 4]
```
**ניתוח האפשרויות:**
*   **א. לשילוב רשימות ב-Python משתמשים רק במתודה `concat()` או בפונקציה `merge()`.:** לא נכון.
*   **ב. לשילוב רשימות ב-Python ניתן להשתמש באופרטור החיבור `+`, במתודה `extend()`, או בלולאות עם הוספת אלמנטים.:** נכון.
*   **ג. לשילוב רשימות משתמשים רק במתודה `union()`, המאחדת שתי רשימות לאחת.:** לא נכון.
*   **ד. לשילוב רשימות יש להשתמש רק בגנרטורי רשימות (list comprehensions).:** לא נכון.

**לסיכום:**
*   האופרטור `+` והמתודה `extend()` הם הדרכים הנפוצות ביותר לשילוב רשימות ב-Python.
*   האופרטור `+` מחזיר רשימה חדשה.
*   המתודה `extend()` משנה את הרשימה הקיימת.
*   כמו כן, ניתן להשתמש בלולאות ובגנרטורי רשימות ליצירת רשימה חדשה ממספר רשימות ישנות.
