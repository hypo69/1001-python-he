**שאלה 053.** כיצד ניתן לאחד שתי רשימות או יותר בפייתון? תאר/י שיטות שונות והצג/י דוגמאות לשימוש בהן.

-   א. לאיחוד רשימות בפייתון משתמשים אך ורק במתודה `concat()` או בפונקציה `merge()`.
-   ב. לאיחוד רשימות בפייתון ניתן להשתמש באופרטור החיבור `+`, במתודה `extend()`, או בלולאות עם הוספת איברים.
-   ג. לאיחוד רשימות משתמשים אך ורק במתודה `union()`, המשלבת שתי רשימות לאחת.
-   ד. לאיחוד רשימות יש להשתמש אך ורק בביטויי רשימה (list comprehensions).

**תשובה נכונה: ב**

**הסבר:**

בפייתון קיימות מספר דרכים לאיחוד (שרשור) רשימות, המאפשרות גמישות בעבודה עם מצבים שונים.

*   **אופרטור `+` (חיבור):**
    *   יוצר רשימה חדשה המאחדת את איברי הרשימות המקוריות.
    *   הרשימות המקוריות אינן משתנות.
    *   יכול לאחד שתי רשימות או יותר, אך בזוגות.
    *   דוגמה: `list1 + list2`

*   **מתודה `extend()`:**
    *   משנה את הרשימה המקורית על ידי הוספת איברי רשימה אחרת לסופה.
    *   אינה יוצרת רשימה חדשה.
    *   דוגמה: `list1.extend(list2)`

*   **לולאות ו-`append()`:**
    *   ניתן להשתמש בלולאות `for` לאיטרציה על הרשימות ובמתודה `append()` להוספת איברים לרשימה התוצאתית.
    *   זו לא הדרך התמציתית ביותר.
*   **ביטוי רשימה (List comprehension):**
    *   ניתן להשתמש בביטוי רשימה ליצירת רשימה חדשה.

**דוגמאות:**

```python
# דוגמה 1: שימוש באופרטור +
list1 = [1, 2]
list2 = [3, 4, 5]
combined_list1 = list1 + list2
print(f"רשימה 1: {list1}")
print(f"רשימה 2: {list2}")
print(f"רשימה מאוחדת (+): {combined_list1}") # פלט: [1, 2, 3, 4, 5]

# דוגמה 2: שימוש במתודה extend()
list3 = [1, 2]
list4 = [3, 4, 5]
print(f"רשימה 3 לפני extend: {[1,2]}")
list3.extend(list4)
print(f"רשימה 4: {list4}")
print(f"רשימה מאוחדת (extend): {list3}") # פלט: [1, 2, 3, 4, 5]

# דוגמה 3: שימוש בלולאת for
list5 = [1, 2]
list6 = [3,4,5]

combined_list2 = []
for item in list5:
  combined_list2.append(item)
for item in list6:
   combined_list2.append(item)
print(f"רשימה מאוחדת (לולאת for): {combined_list2}") # פלט: [1, 2, 3, 4, 5]


# דוגמה 4: שימוש ב-list comprehension
list7 = [1, 2]
list8 = [3,4,5]
combined_list3 =  [item for list_ in [list7, list8] for item in list_]
print(f"רשימה מאוחדת (list comprehension): {combined_list3}") # פלט: [1, 2, 3, 4, 5]

# דוגמה עם מספר רשימות:
list9 = [1, 2]
list10 = [3,4,5]
list11 = [6,7]
combined_list4 = list9 + list10 + list11
print(f"רשימה מאוחדת (אופרטור +): {combined_list4}") # פלט [1, 2, 3, 4, 5, 6, 7]

# דוגמה עם אופרטור +=
list12 = [1, 2]
list12 += [3,4]
print(f"רשימה מאוחדת (+=): {list12}") # פלט [1, 2, 3, 4]

```
**ניתוח האפשרויות:**
*   **א. לאיחוד רשימות בפייתון משתמשים אך ורק במתודה `concat()` או בפונקציה `merge()`.:** לא נכון.
*   **ב. לאיחוד רשימות בפייתון ניתן להשתמש באופרטור החיבור `+`, במתודה `extend()`, או בלולאות עם הוספת איברים.:** נכון.
*   **ג. לאיחוד רשימות משתמשים אך ורק במתודה `union()`, המשלבת שתי רשימות לאחת.:** לא נכון.
*   **ד. לאיחוד רשימות יש להשתמש אך ורק בביטויי רשימה (list comprehensions).:** לא נכון.

**לסיכום:**
*   האופרטור `+` והמתודה `extend()` הן הדרכים הנפוצות ביותר לאיחוד רשימות בפייתון.
*   האופרטור `+` מחזיר רשימה חדשה.
*   המתודה `extend()` משנה את הרשימה הקיימת.
*   ניתן גם להשתמש בלולאות ובביטויי רשימה (list comprehension) כדי ליצור רשימה חדשה ממספר רשימות ישנות.
