**שאלה 86.** כיצד יוצרים קבוצה (`set`) בפייתון?

- A.  `set = {1, 2, 3}`
- B.  `set = [1, 2, 3]`
- C.  `set = (1, 2, 3)`
- D.  `set = '123'`

**תשובה נכונה: A**

**הסבר:**

בפייתון, קבוצה (`set`) נוצרת באמצעות שימוש בסוגריים מסולסלים `{}`, בתוכם ממוקמים איברים ייחודיים הניתנים לגיבוב (hashable).

*   **אפשרות A** נכונה: `set = {1, 2, 3}` - זוהי דרך תקינה ליצירת קבוצה.
*   **אפשרות B** אינה נכונה: `set = [1, 2, 3]` - יוצרת *רשימה* (list), ולא קבוצה.
*   **אפשרות C** אינה נכונה: `set = (1, 2, 3)` - יוצרת *טיפל* (tuple), ולא קבוצה.
*   **אפשרות D** אינה נכונה: `set = '123'` - יוצרת *מחרוזת* (str), ולא קבוצה.

**מאפיינים עיקריים של קבוצה (`set`):**

1.  **ייחודיות איברים:** קבוצה מכילה רק איברים ייחודיים, כפילויות מוסרות אוטומטית.
2.  **אי-סדר:** איברים בקבוצה אינם מאוחסנים בסדר מוגדר כלשהו.
3.  **יכולת שינוי:** קבוצות הן ניתנות לשינוי (mutable), מה שמאפשר להוסיף ולהסיר איברים לאחר יצירת הקבוצה.

**דוגמה:**

```python
# יצירת קבוצה
my_set: set[int] = {1, 2, 3}
print(f"קבוצה: {my_set}") # פלט: קבוצה: {1, 2, 3}

# יצירת קבוצה עם כפילויות
my_set_dupl: set[int] = {1, 2, 3, 2, 1}
print(f"קבוצה עם כפילויות: {my_set_dupl}") # פלט: קבוצה עם כפילויות: {1, 2, 3}
print(f"סוג my_set: {type(my_set)}") # פלט: סוג my_set: <class 'set'>
```
**בתוצאה:**

*   `my_set = {1, 2, 3}` יוצרת קבוצה המכילה את המספרים השלמים 1, 2 ו-3.
*   בעת יצירת הקבוצה `my_set_dupl`, כפילויות מוסרות אוטומטית.

לפיכך, **אפשרות A** היא התשובה הנכונה.
