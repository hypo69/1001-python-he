**שאלה 068.** נתון מערך של מספרים שלמים `nums`, המכיל `n` איברים, ומספר שלם `k`. פתח אלגוריתם למציאת תת-מערך רציף (`subarray`) באורך `k` במערך `nums`, שלו הממוצע המקסימלי. החזר ערך ממוצע מקסימלי זה.

**דוגמאות:**

```
קלט: nums = [1,12,-5,-6,50,3], k = 4
פלט: 12.75000

קלט: nums = [5], k = 1
פלט: 5.00000
```

-  A. עבור פתרון הבעיה יש להשתמש באלגוריתם חמדן (greedy) ולבחור בכל שלב את האיבר הגדול ביותר מהרשימה.
-  B. עבור פתרון הבעיה יש להשתמש באלגוריתם חיפוש לרוחב (BFS).
- C. עבור פתרון הבעיה יש להשתמש בשיטת "חלון מתגלגל" (sliding window) לחיפוש יעיל של הממוצע המקסימלי.
- D. עבור פתרון הבעיה, יש להשתמש במעבר רקורסיבי על כל האפשרויות.

**תשובה נכונה: C**

**הסבר:**

עבור פתרון הבעיה של מציאת תת-מערך עם ממוצע מקסימלי ואורך קבוע `k` במערך מספרים, השיטה האופטימלית היא שימוש בשיטת "חלון מתגלגל" (sliding window). שיטה זו מבטיחה סיבוכיות זמן לינארית `O(n)`, ומאפשרת לעבור על המערך פעם אחת בלבד.

*   **אלגוריתם (חלון מתגלגל):**
    1.  **אתחול:**
         *  מחשבים את סכום `k` האיברים הראשונים של המערך, ושומרים אותם במשתנה `current_sum`.
         *    הסכום המקסימלי `max_sum`  מאותחל  לערך של `current_sum`.
    2. **חלון מתגלגל:**
         *    החל מהאיבר ה-`k` עד סוף המערך (עבור אינדקס  `i`),  אנו שומרים על חלון בגודל k:
              *  מתוך `current_sum` אנו מחסירים את האיבר מתחילת החלון (`nums[i-k]`).
             * ל-`current_sum` אנו מוסיפים את האיבר החדש בסוף החלון `nums[i]`.
         *   משווים  את  `current_sum`  עם `max_sum` וְאִם  הוא גדול יותר אז  `max_sum = current_sum`
    3.  **החזרת התוצאה:**  לאחר מעבר על כל האיברים, אנו מחזירים את `max_sum / k`,  שמתאים לערך הממוצע המקסימלי של תת-המערך.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות לינארית:**  מבטיח סיבוכיות זמן O(n).
    *   **זיכרון קבוע:**  משתמש בכמות קבועה של זיכרון נוסף.
    *   **פשטות:** קל ליישום.

**דוגמאות (קוד מדומה):**

```
function find_max_avg(nums, k):
    n = length(nums)
    current_sum = sum(nums[0] to nums[k-1])
    max_sum = current_sum
    for i in range from k to n:
        current_sum = current_sum - nums[i - k] + nums[i]
        max_sum = max(max_sum, current_sum)
    return max_sum/k
```

**דוגמאות מימוש ב-Python:**
```python
def find_max_avg(nums, k):
    n = len(nums)
    current_sum = sum(nums[:k])
    max_sum = current_sum
    for i in range(k, n):
      current_sum = current_sum - nums[i - k] + nums[i]
      max_sum = max(max_sum, current_sum)
    return max_sum / k

nums1 = [1, 12, -5, -6, 50, 3]
k1 = 4
# קלט: nums = {nums1}, k = {k1}
print(f"קלט: nums = {nums1}, k = {k1}")
# פלט: {find_max_avg(nums1, k1)} # יוצג: 12.75
print(f"פלט: {find_max_avg(nums1, k1)}") # יוצג: 12.75

nums2 = [5]
k2 = 1
# קלט: nums = {nums2}, k = {k2}
print(f"קלט: nums = {nums2}, k = {k2}")
# פלט: {find_max_avg(nums2, k2)}  # יוצג: 5.0
print(f"פלט: {find_max_avg(nums2, k2)}")  # יוצג: 5.0
```
