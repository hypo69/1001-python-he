**שאלה 068.** נתונה מערכת (array) של מספרים שלמים `nums`, המכילה `n` אלמנטים, ומספר שלם `k`. פתח אלגוריתם למציאת תת-מערכת רציפה (contiguous subarray) באורך `k` במערכת `nums`, בעלת ממוצע הערך המקסימלי. החזר את ממוצע הערך המקסימלי הזה.

**דוגמאות:**

```
קלט: nums = [1,12,-5,-6,50,3], k = 4
פלט: 12.75000

קלט: nums = [5], k = 1
פלט: 5.00000
```

- A. כדי לפתור את הבעיה, יש להשתמש באלגוריתם חמדן (greedy) ולבחור בכל צעד את האלמנט הגדול ביותר מהרשימה.
- B. כדי לפתור את הבעיה, יש להשתמש באלגוריתם חיפוש לרוחב (BFS).
- C. כדי לפתור את הבעיה, יש להשתמש בשיטת "חלון הזזה" (sliding window) לחיפוש יעיל של הממוצע המקסימלי.
- D. כדי לפתור את הבעיה, יש להשתמש במעבר רקורסיבי על כל האפשרויות.

**תשובה נכונה: C**

**הסבר:**

כדי לפתור את הבעיה של מציאת תת-מערכת עם ממוצע הערך המקסימלי ואורך קבוע `k` במערכת מספרים, השיטה האופטימלית היא יישום שיטת "חלון הזזה". שיטה זו מבטיחה סיבוכיות זמן לינארית `O(n)`, ומאפשרת מעבר על המערכת פעם אחת בלבד.

*   **אלגוריתם (חלון הזזה):**
    1.  **אתחול:**
        *   חשב את סכום `k` האלמנטים הראשונים של המערכת, ושמור אותם במשתנה `current_sum`.
        *   סכום מקסימלי `max_sum` מאותחל עם ערך `current_sum`.
    2.  **חלון הזזה:**
        *   החל מהאלמנט ה-`k`-י ועד סוף המערכת (עבור אינדקס `i`), שמור על חלון בגודל k:
            *   מתוך `current_sum`, החסר את האלמנט מתחילת החלון (`nums[i-k]`).
            *   אל `current_sum`, הוסף את האלמנט החדש בסוף החלון (`nums[i]`).
        *   השווה את `current_sum` עם `max_sum`, ואם הוא גדול יותר, אז `max_sum = current_sum`.
    3.  **החזרת התוצאה:** לאחר מעבר על כל האלמנטים, החזר את `max_sum / k`, אשר מתאים לממוצע הערך המקסימלי של תת-המערכת.

*   **יתרונות האלגוריתם:**
    *   **סיבוכיות לינארית:** מבטיח סיבוכיות זמן של O(n).
    *   **זיכרון קבוע:** משתמש בכמות קבועה של זיכרון נוסף.
    *   **פשטות:** קל ליישום.

**דוגמאות (פסאודו-קוד):**

```
function find_max_avg(nums, k):
    n = length(nums)
    current_sum = sum(nums[0] to nums[k-1])
    max_sum = current_sum
    for i in range from k to n:
        current_sum = current_sum - nums[i - k] + nums[i]
        max_sum = max(max_sum, current_sum)
    return max_sum/k
```

**דוגמאות ליישום ב-Python:**

```python
def find_max_avg(nums, k):
    n = len(nums)
    current_sum = sum(nums[:k])
    max_sum = current_sum
    for i in range(k, n):
      current_sum = current_sum - nums[i - k] + nums[i]
      max_sum = max(max_sum, current_sum)
    return max_sum / k

nums1 = [1, 12, -5, -6, 50, 3]
k1 = 4
print(f"קלט: nums = {nums1}, k = {k1}")
print(f"פלט: {find_max_avg(nums1, k1)}") # יוצג: 12.75

nums2 = [5]
k2 = 1
print(f"קלט: nums = {nums2}, k = {k2}")
print(f"פלט: {find_max_avg(nums2, k2)}")  # יוצג: 5.0
```
