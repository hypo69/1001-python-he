**שאלה 18.** בהינתן קוד Python הבא שמבצע שינוי ברשימה, מה תהיה הפלט הסופי?

```python
items = [2, 4, 6, 8, 10]
for i in range(len(items)):
    items[i] += 3
print(items)
```

- א.  `[5, 7, 9, 11, 13]`
- ב.  `[2, 4, 6, 8, 10]`
- ג.  `[3, 6, 9, 12, 15]`
- ד.  `[5, 7, 9, 11, 15]`

**תשובה נכונה: א**

**הסבר:**

קוד זה מדגים שינוי של הרשימה `items` באמצעות לולאת `for`. כך פועל קוד זה:

1.  **אתחול הרשימה:** `items = [2, 4, 6, 8, 10]` יוצר רשימה עם ערכים התחלתיים.
2.  **לולאת `for`:** `for i in range(len(items)):` לולאה זו מבצעת איטרציה על פי האינדקסים של הרשימה `items`. הפונקציה `range(len(items))` יוצרת סדרה של מספרים שלמים מ-0 ועד אורך הרשימה פחות 1 (0, 1, 2, 3, 4 במקרה זה).
3.  **שינוי האלמנטים:** `items[i] += 3` מוסיף `3` לערך האלמנט ברשימה באינדקס הנוכחי `i`.
4.  **פלט:** `print(items)` מדפיס את הרשימה שהשתנתה למסך.

כעת נפרט כל שלב בלולאה:
* i = 0: items[0] = 2 + 3 = 5
* i = 1: items[1] = 4 + 3 = 7
* i = 2: items[2] = 6 + 3 = 9
* i = 3: items[3] = 8 + 3 = 11
* i = 4: items[4] = 10 + 3 = 13

**דוגמה:**

```python
items: list[int] = [2, 4, 6, 8, 10]
for i in range(len(items)):
    items[i] += 3
print(items)  # Output: [5, 7, 9, 11, 13]
```

**כתוצאה מכך:**

כל אלמנט ברשימה המקורית `items` יוגדל ב-3, מה שיוביל ליצירת רשימה חדשה `[5, 7, 9, 11, 13]`.

לפיכך, **אפשרות א** היא הנכונה.
