**שאלה 072.** מהן מונאדות ב-Python, וכיצד הן משמשות? הצג דוגמה לשימוש במונאדת `Maybe` לטיפול בשגיאות אפשריות או בהיעדר ערכים.

-   A. מונאדות הן טיפוסי נתונים מובנים ב-Python לאחסון סדרות של מספרים.
-   B. מונאדות הן דרך לארגן קוד כדי להשתמש ביותר משתנים, מבלי לשנות את ביצועי התוכנית.
-   C. מונאדות הן מבנה מופשט, המשמש בתכנות פונקציונלי לבניית שרשרת של חישובים, ומונאדת `Maybe` משמשת לטיפול בשגיאות אפשריות או בהיעדר ערכים.
-   D. מונאדות הן אובייקטים המשמשים לתיאור היררכיית מחלקות ב-Python.

**תשובה נכונה: C**

**הסבר:**

מונאדות ב-Python (כמו בשפות תכנות אחרות השואבות השראה מגישה פונקציונלית) הן מושג מופשט המאפשר לארגן חישובים ולבנות שרשראות של פעולות עוקבות. מונאדות מספקות דרך ליצור קומפוזיציה של חישובים, במיוחד כאשר חישובים אלו עשויים להיות לא מוגדרים או שעשויים להכיל תופעות לוואי.

*   **מושגי מפתח של מונאדות:**
    *   **מכולות (Containers):** מונאדות לרוב מייצגות מכולות לערכים.
    *   **קומפוזיציה:** מונאדות מאפשרות לאחד (ליצור קומפוזיציה של) פונקציות לשרשראות, כך שהפלט של פונקציה אחת הופך לקלט עבור הבאה.
    *   **ניהול תופעות לוואי:** מונאדות עוזרות לבודד ולנהל תופעות לוואי (לדוגמה, שגיאות, קלט/פלט) בתוך שרשרת החישובים.
*   **מונאדת `Maybe` (או `Option`):**
    *   משמשת לייצוג ערך שעשוי להתקיים או לא (כלומר, זהו ערך מטיפוס מסוים, או "כלום" (`None`)).
    *   מאפשרת טיפול בשגיאות אפשריות או בהיעדר ערכים (לדוגמה, כאשר תוצאת חלוקה עלולה להוביל לחלוקה באפס).
    *   עוזרת להימנע מחריגות, אם אחד משלבי החישוב עשוי להחזיר ערך לא חוקי.

**דוגמה (מתוך טקסט השאלה):**

```python
class Maybe:
    def __init__(self, value=None):
        self.value = value

    def bind(self, func):
        if self.value is None:
            return Maybe()
        else:
            return func(self.value)

def safe_divide(x, y):
    if y == 0:
        return Maybe()
    else:
        return Maybe(x / y)

result = Maybe(10).bind(lambda x: safe_divide(x, 2)).bind(lambda x: safe_divide(x, 0))

print(result.value)  # None
```

**ניתוח הדוגמה:**

1.  `class Maybe:` יוצרים מחלקה שמממשת את מונאדת `Maybe`.
2.  `__init__`: הקונסטרוקטור של המחלקה.
3.  `bind(self, func)`: מתודה שמקשרת `Maybe` עם פונקציה אחרת. אם הערך הנוכחי של `Maybe` הוא `None`, אזי היא מחזירה `Maybe()` (מה שאומר `None`). אחרת – היא תפעיל את הפונקציה `func` עם הערך הנוכחי.
4.  `safe_divide(x, y)`: פונקציה שמחלקת את `x` ב-`y` באופן בטוח, ומחזירה `Maybe()` אם `y` שווה לאפס.
5.  `result`: מאותחל עם `Maybe(10)`, ולאחר מכן מתבצעת שרשרת של פעולות, כאשר כל `bind` מקשר את התוצאה של השלב הקודם עם החישוב הבא.
6.  מכיוון שהחלוקה באפס בסוף תחזיר `None`, וכל הערכים העוקבים של `bind` יחזירו `None`, בסופו של דבר `result.value` יהיה שווה ל-`None`.

**ניתוח האפשרויות:**

*   **A. מונאדות — זהו טיפוס נתונים מובנה ב-Python לאחסון סדרות של מספרים:** שגוי. מונאדות הן מושג מופשט, לא טיפוס נתונים מובנה.
*   **B. מונאדות — זוהי דרך לארגן קוד כדי להשתמש ביותר משתנים, מבלי לשנות את ביצועי התוכנית:** שגוי. מונאדות אינן קשורות לניהול כמות המשתנים.
*   **C. מונאדות — זהו מבנה מופשט, המשמש בתכנות פונקציונלי לבניית שרשרת של חישובים, ומונאדת `Maybe` משמשת לטיפול בשגיאות אפשריות או בהיעדר ערכים:** נכון.
