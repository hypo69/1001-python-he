### `question_56.md`

**שאלה 56.** מהו ההבדל העיקרי בין מבני הנתונים `list` ו-`tuple` בפייתון?

- א. רשימות הן ניתנות לשינוי (mutable), מה שמאפשר לשנות אותן לאחר יצירתן, בעוד טאפלים הם בלתי ניתנים לשינוי (immutable) ולא ניתנים לשינוי לאחר יצירתם.
- ב. טאפלים יכולים לאחסן איברים מסוגי נתונים שונים, אך רשימות אינן יכולות.
- ג. רשימות משמשות בדרך כלל ללולאות, ועל טאפלים אי אפשר לבצע איטרציה.
- ד. לטאפלים יש זמן גישה מהיר יותר, אך רשימות עובדות לאט יותר מכיוון שהן מוצפנות לצורך אבטחה.

**תשובה נכונה: א**

**הסבר:**

בפייתון, רשימות (`list`) וטאפלים (`tuple`) הם רצפים (sequences), אך הם נבדלים במוטביליות (mutable/immutable) שלהם.

*   **אפשרות א** נכונה: רשימות הן *ניתנות לשינוי* (mutable), בעוד טאפלים הם *בלתי ניתנים לשינוי* (immutable).
*   **אפשרות ב** אינה נכונה: גם רשימות וגם טאפלים יכולים לאחסן איברים מסוגים שונים.
*   **אפשרות ג** אינה נכונה: גם על רשימות וגם על טאפלים ניתן לבצע איטרציה.
*   **אפשרות ד** אינה נכונה: טאפלים לרוב מהירים יותר מרשימות עבור פעולות מסוימות, מכיוון שהם אינם יכולים להשתנות, ולכן פייתון יכולה לייעל את השימוש בהם טוב יותר, אך רשימות אינן מוצפנות.

**הסבר מפורט:**

*   **רשימות (`list`):**
    *   **מוטביליות:** רשימות הן ניתנות לשינוי, כלומר לאחר יצירתן ניתן להוסיף, למחוק או לשנות איברים ברשימה.
    *   **תחביר:** רשימות נוצרות באמצעות סוגריים מרובעים `[]`.
    *   **שימוש:** משמשות לאחסון אוספים דינמיים של איברים, כאשר יש צורך לשנות את הסדר, להוסיף או למחוק איברים.

*   **טאפלים (`tuple`):**
    *   **בלתי ניתנים לשינוי:** טאפלים הם בלתי ניתנים לשינוי, כלומר לאחר יצירתם אי אפשר להוסיף, למחוק או לשנות איברים.
    *   **תחביר:** טאפלים נוצרים באמצעות סוגריים עגולים `()`.
    *   **שימוש:** משמשים לייצוג אוספים קבועים של איברים, כאשר אין צורך בשינוי התוכן (לדוגמה, קואורדינטות, רשומות). טאפלים יכולים לשמש כמפתחות מילון, דבר שלא ניתן לעשות עם רשימות.

**דוגמה:**

```python
# דוגמה עם רשימה (ניתנת לשינוי)
my_list: list[int] = [1, 2, 3]
my_list.append(4)
my_list[0] = 0
print(f"רשימה: {my_list}") # פלט: רשימה: [0, 2, 3, 4]

# דוגמה עם טאפל (בלתי ניתן לשינוי)
my_tuple: tuple[int,int,int] = (1, 2, 3)
# my_tuple[0] = 0 # תגרום ל-TypeError: 'tuple' object does not support item assignment
# my_tuple.append(4) # תגרום ל-AttributeError: 'tuple' object has no attribute 'append'
print(f"טאפל: {my_tuple}")  # פלט: טאפל: (1, 2, 3)
```
**בתוצאה:**
* ניתן לשנות ( לבצע מוטציה ב-) רשימות כפי שעשינו בדוגמה, הוספנו ושינינו איבר.
* לא ניתן לשנות טאפלים לאחר יצירתם.

לפיכך, **אפשרות א** היא הנכונה, כיוון שהיא משקפת במדויק את ההבדל העיקרי בין רשימות לטאפלים בפייתון.
