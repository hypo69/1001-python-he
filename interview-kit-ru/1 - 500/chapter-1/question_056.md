**שאלה 56.** מה ההבדל העיקרי בין מבני הנתונים `list` ו-`tuple` בפייתון?

- A. רשימות הן ניתנות לשינוי, מה שמאפשר לשנות אותן לאחר יצירתן, בעוד שקורטז'ים הם בלתי ניתנים לשינוי ולא ניתן לשנות אותם לאחר יצירתם.
- B. קורטז'ים יכולים לאחסן אלמנטים מסוגי נתונים שונים, אך רשימות לא יכולות.
- C. רשימות משמשות בדרך כלל ללולאות, וקורטז'ים אינם ניתנים למעבר.
- D. לקורטז'ים זמן גישה מהיר יותר, אך רשימות עובדות לאט יותר מכיוון שהן מוצפנות למען האבטחה.

**תשובה נכונה: A**

**הסבר:**

בפייתון, רשימות (`list`) וקורטז'ים (`tuple`) הם רצפים, אך הם שונים בהיבט ה-mutability (יכולת השינוי) שלהם.

*   **אפשרות A** נכונה: רשימות הן *ניתנות לשינוי* (mutable), בעוד שקורטז'ים הם *בלתי ניתנים לשינוי* (immutable).
*   **אפשרות B** אינה נכונה: גם רשימות וגם קורטז'ים יכולים לאחסן אלמנטים מסוגים שונים.
*   **אפשרות C** אינה נכונה: גם על רשימות וגם על קורטז'ים ניתן לעבור (iterate).
*   **אפשרות D** אינה נכונה: קורטז'ים בדרך כלל מהירים יותר מרשימות עבור פעולות מסוימות, מכיוון שלא ניתן לשנות אותם, ולכן פייתון יכול לבצע אופטימיזציה טובה יותר לשימוש בהם, אך רשימות אינן מוצפנות.

**הסבר מפורט:**

*   **רשימות (`list`):**
    *   **יכולת שינוי (Mutability):** רשימות הן ניתנות לשינוי, כלומר לאחר יצירתן ניתן להוסיף, להסיר או לשנות אלמנטים ברשימה.
    *   **תחביר:** רשימות נוצרות באמצעות סוגריים מרובעים `[]`.
    *   **שימוש:** משמשות לאחסון אוספים דינמיים של אלמנטים, כאשר נדרש לשנות סדר, להוסיף או להסיר אלמנטים.

*   **קורטז'ים (`tuple`):**
    *   **אי-יכולת שינוי (Immutability):** קורטז'ים הם בלתי ניתנים לשינוי, כלומר לאחר יצירתם לא ניתן להוסיף, להסיר או לשנות אלמנטים.
    *   **תחביר:** קורטז'ים נוצרים באמצעות סוגריים עגולים `()`.
    *   **שימוש:** משמשים לייצוג אוספים קבועים של אלמנטים, כאשר אין דרישה לשינוי התוכן (לדוגמה, קואורדינטות, רשומות). קורטז'ים יכולים לשמש כמפתחות במילון, דבר שלא ניתן לעשות עם רשימות.

**דוגמה:**

```python
# דוגמה עם רשימה (ניתנת לשינוי)
my_list: list[int] = [1, 2, 3]
my_list.append(4)
my_list[0] = 0
print(f"רשימה: {my_list}") # פלט: רשימה: [0, 2, 3, 4]

# דוגמה עם קורטז' (בלתי ניתן לשינוי)
my_tuple: tuple[int,int,int] = (1, 2, 3)
# my_tuple[0] = 0 # יגרום ל-TypeError: 'tuple' object does not support item assignment
# my_tuple.append(4) # יגרום ל-AttributeError: 'tuple' object has no attribute 'append'
print(f"קורטז': {my_tuple}")  # פלט: קורטז': (1, 2, 3)
```
**לסיכום:**
* רשימות ניתן לשנות (לעשות להן מוטציה) כפי שעשינו בדוגמה, על ידי הוספה ושינוי אלמנט.
* קורטז'ים לאחר יצירתם אינם ניתנים לשינוי.

לפיכך, **אפשרות A** היא הנכונה, שכן היא משקפת במדויק את ההבדל העיקרי בין רשימות לקורטז'ים בפייתון.
