**שאלה 92.** כיצד ניתן להמיר רשימה של מספרים שלמים לרשימה של מחרוזות?

- א. שימוש בפונקציה `str()` באופן פרטני עבור כל אלמנט.
- ב. שימוש בפונקציה `map(str, list)`.
- ג. שרשור כל אלמנט עם מחרוזת ריקה.
- ד. גם א' וגם ב' נכונים.

**תשובה נכונה: ד'**

**הסבר:**

קיימות מספר דרכים להמיר רשימה של מספרים שלמים לרשימה של מחרוזות ב-Python.

*   **אפשרות א'** נכונה: ניתן להחיל את הפונקציה `str()` על כל אלמנט ברשימה בנפרד כדי להמיר אותו למחרוזת.
*   **אפשרות ב'** נכונה: הפונקציה `map()` מאפשרת להחיל את הפונקציה `str()` על כל אלמנט ברשימה, מה שממיר ביעילות את כל האלמנטים למחרוזות.
*   **אפשרות ג'** אינה נכונה: שרשור עם מחרוזת ריקה אמנם יעבוד במקרה זה, אך השיטה המומלצת והקריאה יותר להמרה למחרוזת היא באמצעות הפונקציה `str()`.
*   **אפשרות ד'** נכונה: מאחר שאפשרויות א' וב' נכונות.

**כיצד זה עובד:**

1.  **`str()`:** הפונקציה `str()` מקבלת כל אובייקט כקלט וממירה אותו לייצוג מחרוזתי שלו.
2.  **לולאה:** בלולאה, ניתן להמיר בזה אחר זה כל אלמנט ברשימה למחרוזת.
3.  **`map()`:** הפונקציה `map()` מקבלת פונקציה ואובייקט איטרבילי (רשימה) ומחילה פונקציה זו על כל אלמנט ברשימה.
4.  **הבנת רשימה (List Comprehension):** ניתן גם להשתמש בהבנת רשימה (list comprehension) עם המרת טיפוסים.

**דוגמאות:**

```python
numbers: list[int] = [1, 2, 3, 4, 5]

# המרה באמצעות לולאה ו-str()
string_list_loop: list[str] = []
for number in numbers:
    string_list_loop.append(str(number))

print(f"Список строк (цикл): {string_list_loop}")  # פלט: Список строк (цикл): ['1', '2', '3', '4', '5']

# המרה באמצעות map() ו-str()
string_list_map: list[str] = list(map(str, numbers))
print(f"Список строк (map): {string_list_map}") # פלט: Список строк (map): ['1', '2', '3', '4', '5']

# המרה באמצעות הבנת רשימה (list comprehension)
string_list_comp: list[str] = [str(number) for number in numbers]
print(f"Список строк (list comprehension): {string_list_comp}") # פלט: Список строк (list comprehension): ['1', '2', '3', '4', '5']

# המרה פחות מומלצת באמצעות שרשור עם מחרוזת ריקה (תעבוד, אך פחות קריאה)
string_list_concat: list[str] = [str(number) + "" for number in numbers] # נדרש להשתמש ב-str() תחילה!
print(f"Список строк (concat): {string_list_concat}") # פלט: Список строк (concat): ['1', '2', '3', '4', '5']
```
**כתוצאה מכך:**
*  כל שלוש השיטות (לולאה, `map()` והבנת רשימה) יובילו ליצירת רשימה של מחרוזות.
*  שרשור עם מחרוזת ריקה יעבוד, אך נחשב לפחות קריא.

לפיכך, **אפשרות ד'** היא התשובה הנכונה, מכיוון שאפשרויות א' וב' שתיהן דרכים נכונות להמיר רשימה של מספרים שלמים לרשימה של מחרוזות.
