### `question_047.md`

**שאלה 47.** כיצד פונקציית למדא ב-Python מקלה על הגדרה מהירה של פונקציות, ומהם מקרי השימוש האופייניים שלה בתכנות?

- א. פונקציות למדא ב-Python משמשות ליצירת אובייקטי פונקציה חדשים לשימוש קבוע ביישומים, עם אותן יכולות כמו פונקציות המוגדרות באמצעות `def`.
- ב. הן מאפשרות הגדרה של פונקציות אנונימיות קטנות בשורה אחת, המשמשות בדרך כלל במקומות שבהם נדרשים אובייקטי פונקציה לתקופות קצרות, למשל עם פונקציות כמו `map()` או `filter()`.
- ג. סוג זה של פונקציה מנהל אוטומטית הקצאת זיכרון ואיסוף זבל, ומשפר משמעותית את ביצועי היישום.
- ד. פונקציות למדא משמשות בעיקר להצהרה וניהול של משתנים גלובליים בטווחי ראייה מקומיים (local scopes) לשיפור מודולריות ושימוש חוזר בקוד.

**תשובה נכונה: B**

**הסבר:**

פונקציות למדא ב-Python הן פונקציות אנונימיות, כלומר פונקציות ללא שם. הן מוגדרות באמצעות מילת המפתח `lambda` ומשמשות ליצירת פונקציות פשוטות "תוך כדי תנועה" בשורה אחת.

*   **אפשרות א** אינה נכונה: פונקציות למדא אינן מיועדות לשימוש קבוע ויש להן יכולות מוגבלות לעומת פונקציות רגילות שנוצרו באמצעות `def`.

*   **אפשרות ב** נכונה: פונקציות למדא הן פונקציות אנונימיות, המוגדרות בשורה אחת, והן משמשות בדרך כלל כאשר נדרשת פונקציה פשוטה לשימוש קצר מועד (למשל, בעת העברה לפונקציות כמו `map()`, `filter()`, `sort()`).

*   **אפשרות ג** אינה נכונה: פונקציות למדא אינן מנהלות זיכרון אוטומטית. הן פועלות באופן דומה לפונקציות רגילות מבחינת ניהול זיכרון.

*   **אפשרות ד** אינה נכונה: פונקציות למדא אינן קשורות לניהול משתנים גלובליים.

**מאפיינים עיקריים של פונקציות למדא:**

1.  **אנונימיות:** לפונקציות למדא אין שם, בשונה מפונקציות רגילות המוגדרות באמצעות `def`.
2.  **קומפקטיות:** הן נכתבות בשורה אחת ויכולות להכיל רק ביטוי יחיד.
3.  **תחום שימוש:** פונקציות למדא משמשות לעתים קרובות כארגומנטים לפונקציות מסדר גבוה (higher-order functions), כגון `map()`, `filter()`, `sorted()` וכו', כאשר נדרש להעביר פונקציה פשוטה לעיבוד נתונים.
4.  **מגבלות:** פונקציות למדא אינן יכולות להכיל ביטויים מרובים או מבנים מורכבים, למשל לולאות או תנאים.

**דוגמה:**

```python
# דוגמה לשימוש ב-lambda עם map()
numbers: list[int] = [1, 2, 3, 4, 5]
squared_numbers = list(map(lambda x: x**2, numbers))
print(f"ריבועי המספרים: {squared_numbers}") # פלט: ריבועי המספרים: [1, 4, 9, 16, 25]

# דוגמה לשימוש ב-lambda עם filter()
numbers: list[int] = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(f"מספרים זוגיים: {even_numbers}") # פלט: מספרים זוגיים: [2, 4, 6]

# דוגמה לשימוש ב-lambda למיון רשימה
points: list[tuple[int, int]] = [(3, 1), (1, 2), (2, 3)]
sorted_points = sorted(points, key=lambda p: p[1])
print(f"מיון לפי האיבר השני של ה-tuple: {sorted_points}") # פלט: מיון לפי האיבר השני של ה-tuple: [(3, 1), (1, 2), (2, 3)]
```

**כתוצאה מכך:**
*   פונקציית למדא משמשת להעלאה בריבוע של כל איבר ברשימה `numbers` באמצעות `map()`.
*   פונקציית למדא משמשת לסינון מספרים זוגיים מתוך הרשימה `numbers` באמצעות `filter()`.
*   פונקציית למדא משמשת למיון רשימת ה-`points` (tuples) לפי האיבר השני.

לפיכך, **אפשרות B** היא התשובה הנכונה.
