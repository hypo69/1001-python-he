### `question_047.md`

**שאלה 47.** כיצד פונקציית למדא ב-Python מאפשרת הגדרה מהירה של פונקציות, ומהם מקרי השימוש האופייניים שלה בתכנות?

- א. פונקציות למדא ב-Python משמשות ליצירת אובייקטי פונקציה חדשים לשימוש קבוע באפליקציות, עם אותן יכולות כמו פונקציות המוגדרות באמצעות `def`.
- ב. הן מאפשרות הגדרת פונקציות אנונימיות קטנות בשורה אחת, המשמשות בדרך כלל היכן שנדרשים אובייקטי פונקציה לתקופות קצרות, לדוגמה, עם פונקציות כמו `map()` או `filter()`.
- ג. סוג זה של פונקציה מנהל באופן אוטומטי הקצאת זיכרון ואיסוף זבל, ובכך משפר משמעותית את ביצועי האפליקציה.
- ד. פונקציות למדא משמשות בראש ובראשונה להצהרה ולניהול משתנים גלובליים בטווחי ראות מקומיים כדי לשפר מודולריות ושימוש חוזר בקוד.

**תשובה נכונה: ב**

**הסבר:**

פונקציות למדא ב-Python הן פונקציות אנונימיות, כלומר, פונקציות ללא שם. הן מוגדרות באמצעות מילת המפתח `lambda` ומשמשות ליצירת פונקציות פשוטות "על המקום" בשורה אחת.

*   אפשרות א' אינה נכונה: פונקציות למדא אינן מיועדות לשימוש קבוע ויש להן יכולות מוגבלות בהשוואה לפונקציות רגילות שנוצרו באמצעות `def`.

*   אפשרות ב' נכונה: פונקציות למדא הן פונקציות אנונימיות המוגדרות בשורה אחת, והן משמשות בדרך כלל כאשר נדרשת פונקציה פשוטה לשימוש קצר טווח (לדוגמה, בעת העברה ל-`map()`, `filter()`, `sort()`).
    
*   אפשרות ג' אינה נכונה: פונקציות למדא אינן מנהלות זיכרון באופן אוטומטי. הן פועלות בדיוק כמו פונקציות רגילות מבחינת ניהול זיכרון.

*   אפשרות ד' אינה נכונה: פונקציות למדא אינן קשורות לניהול משתנים גלובליים.

**מאפייני מפתח של פונקציות למדא:**

1.  **אנונימיות:** לפונקציות למדא אין שם, בניגוד לפונקציות רגילות המוגדרות באמצעות `def`.
2.  **קומפקטיות:** הן נכתבות בשורה אחת ויכולות להכיל ביטוי יחיד בלבד.
3.  **תחום יישום:** פונקציות למדא משמשות לעיתים קרובות כארוגמנטים לפונקציות מסדר גבוה יותר, כגון `map()`, `filter()`, `sorted()` וכו', היכן שיש צורך להעביר פונקציה פשוטה לעיבוד נתונים.
4.  **מגבלות:** פונקציות למדא אינן יכולות להכיל ביטויים מרובים או מבנים מורכבים, לדוגמה לולאות ומשפטי תנאי.

**דוגמה:**

```python
# דוגמה לשימוש ב-lambda עם map()
numbers: list[int] = [1, 2, 3, 4, 5]
squared_numbers = list(map(lambda x: x**2, numbers))
print(f"Квадраты чисел: {squared_numbers}") # פלט: ריבועי המספרים: [1, 4, 9, 16, 25]

# דוגמה לשימוש ב-lambda עם filter()
numbers: list[int] = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(f"Чётные числа: {even_numbers}") # פלט: מספרים זוגיים: [2, 4, 6]

# דוגמה לשימוש ב-lambda עם מיון רשימה
points: list[tuple[int, int]] = [(3, 1), (1, 2), (2, 3)]
sorted_points = sorted(points, key=lambda p: p[1])
print(f"Сортировка по 2ому элементу кортежа: {sorted_points}") # פלט: מיון לפי האיבר השני של ה-tuple: [(3, 1), (1, 2), (2, 3)]
```

**לסיכום:**
*   פונקציית הלמדא משמשת להעלאה בריבוע של כל איבר ברשימה numbers באמצעות `map()`.
*   פונקציית הלמדא משמשת לסינון מספרים זוגיים מהרשימה `numbers` באמצעות `filter()`.
*   פונקציית הלמדא משמשת למיון רשימת ה-tuples `points` לפי האיבר השני.

לפיכך, **אפשרות ב'** היא התשובה הנכונה.
