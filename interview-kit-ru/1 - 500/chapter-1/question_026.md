**שאלה 26.** מהי המשמעות של מילת המפתח `None` בפייתון?

- א. היא מצביעה על היעדר ערך או ערך אפסי במשתנה.
- ב. זהו טיפוס נתונים מיוחד שניתן להקצות רק למשתני מחרוזת.
- ג. הוא מייצג ערך מספרי אפס בחישובים מספריים.
- ד. הוא משמש להגדרת לולאה אינסופית בתכנות פייתון.

**תשובה נכונה: א**

**הסבר:**

מילת המפתח `None` בפייתון מציינת היעדר ערך או ערך אפסי. זוהי קבועה מיוחדת המייצגת אובייקט מטיפוס `NoneType`.

*   **אפשרות א'** נכונה: `None` משמש לייצוג היעדר ערך, בדומה ל-null בשפות אחרות.
*   **אפשרות ב'** אינה נכונה: `None` הוא טיפוס נתונים נפרד (`NoneType`) וניתן להקצותו למשתנים מכל טיפוס.
*   **אפשרות ג'** אינה נכונה: `None` אינו מייצג את הערך המספרי 0.
*   **אפשרות ד'** אינה נכונה: `None` אינו משמש להגדרת לולאות אינסופיות.

**דוגמאות:**

```python
# הקצאת None למשתנה
my_variable: str | None = None
print(f"ערך המשתנה: {my_variable}")   # פלט: ערך המשתנה: None
print(f"טיפוס המשתנה: {type(my_variable)}")  # פלט: טיפוס המשתנה: <class 'NoneType'>

# דוגמה לשימוש ב-None בפונקציה
def my_function(input_value: int | None = None) -> int | None:
    if input_value is None:
      return None
    return input_value * 2
print(f"תוצאת my_function() : {my_function()}")  # פלט: תוצאת my_function() : None
print(f"תוצאת my_function(5) : {my_function(5)}")  # פלט: תוצאת my_function(5) : 10
```

**בתוצאה:**

*   למשתנה `my_variable` ניתן להקצות `None` כערך, וזה לא יגרום לשגיאה.
*   `None` הוא טיפוס נתונים נפרד (`NoneType`).
*   הפונקציה `my_function` מטפלת במקרה שבו לא מועבר לה ערך, ומחזירה None.

לפיכך, **אפשרות א'** היא הנכונה.
