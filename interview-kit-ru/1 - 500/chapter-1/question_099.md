### `question_99.md`

**שאלה 99.** בפייתון, מה עושה הדקורטור `@classmethod` עם מתודה?

- א. הוא ממיר את המתודה למתודה סטטית, השייכת למחלקה.
- ב. הוא מגביל את המתודה כך שלא ניתן יהיה לדרוס אותה במחלקות בנות.
- ג. הוא מאפשר לקרוא למתודה דרך המחלקה עצמה, ולא רק דרך מופעי המחלקה.
- ד. הוא הופך את המתודה לפרטית, כך שלא ניתן לגשת אליה מחוץ למחלקה.

**תשובה נכונה: ג**

**הסבר:**

הדקורטור `@classmethod` בפייתון משמש להמרת מתודה רגילה של מחלקה למתודת מחלקה.

*   **אפשרות א** אינה נכונה: הוא ממיר את המתודה למתודת מחלקה, ולא למתודה סטטית.
*   **אפשרות ב** אינה נכונה: דקורטורים אינם משפיעים על היכולת לדרוס מתודה.
*   **אפשרות ג** נכונה: `@classmethod` מאפשר לקרוא למתודה ישירות דרך המחלקה, ולא רק דרך מופע שלה.
*   **אפשרות ד** אינה נכונה: הדקורטור אינו משנה את נראות (visibility) המתודה.

**מאפיינים מרכזיים של מתודות מחלקה:**

1.  **קשירה למחלקה:** מתודת מחלקה קשורה למחלקה עצמה, ולא למופעים שלה.
2.  **ארגומנט `cls`:** מתודת מחלקה מקבלת אוטומטית הפניה למחלקה (שמקובל לקרוא לה `cls`) בתור הארגומנט הראשון, זהו הבדל ממתודות רגילות (`self`).
3.  **גישה לתכונות המחלקה:** היא יכולה לגשת לתכונות המחלקה ולשנות אותן.
4.  **קריאה דרך מחלקה או מופע:** ניתן לקרוא למתודת מחלקה גם דרך המחלקה עצמה (`ClassName.method()`) וגם דרך מופע (`instance.method()`).

**שימושים טיפוסיים למתודות מחלקה:**

*   **מתודות פקטורי (Factory methods):** מתודות מחלקה משמשות לעיתים קרובות ליצירת בנאים חלופיים, כלומר ליצירת אובייקטים של המחלקה בדרכים שונות.

**דוגמה:**

```python
class MyClass:
    class_var: int = 10

    def __init__(self, value: int):
        self.value = value
    
    @classmethod
    def create_with_default_value(cls) -> "MyClass":
      """מתודת מחלקה ליצירת מופע עם ערך ברירת מחדל"""
      return cls(100)

    def instance_method(self) -> None:
        print(f"מתודת מופע. ערך המופע: {self.value} .ערך משתנה המחלקה: {MyClass.class_var}")

# קריאה למתודת המחלקה דרך המחלקה עצמה
obj1: MyClass = MyClass.create_with_default_value()
obj1.instance_method() # פלט: מתודת מופע. ערך המופע: 100 .ערך משתנה המחלקה: 10

# קריאה למתודת מופע דרך מופע (אין פה מתודת מחלקה שמשנה את משתנה המחלקה בדוגמה זו)
# יש להניח שהיתה כאן מתודת מופע בשם modify_class_variable בדוגמה המקורית,
# אך הקוד שהוצג לא כלל אותה. נשתמש במתודה הקיימת להמחשה.
obj2 = MyClass(10)
obj2.instance_method() # פלט: מתודת מופע. ערך המופע: 10 .ערך משתנה המחלקה: 10
```
**בתוצאה:**
*   המתודה `create_with_default_value` נקראת דרך המחלקה ומחזירה מופע חדש.
*   המתודה `instance_method` מדפיסה את ערכי תכונת המופע ותכונת המחלקה.
*   (הערה: הקוד המקורי הזכיר מתודת מחלקה בשם `modify_class_variable` שמשנה את משתנה המחלקה, אך היא לא הופיעה בקוד עצמו. תרגום ההסבר מותאם לקוד הקיים).

לפיכך, **אפשרות ג** היא הנכונה.
