### `question_99.md`

**שאלה 99.** בפייתון, מה עושה הדקורטור `@classmethod` למתודה?

- א. הוא ממיר את המתודה למתודה סטטית ששייכת למחלקה.
- ב. הוא מגביל את המתודה כך שלא ניתן יהיה לדרוס אותה במחלקות יורשות.
- ג. הוא מאפשר לקרוא למתודה מהמחלקה עצמה, ולא רק ממופעים של המחלקה.
- ד. הוא הופך את המתודה לפרטית, כך שלא ניתן לגשת אליה מחוץ למחלקה.

**תשובה נכונה: ג'**

**הסבר:**

הדקורטור `@classmethod` בפייתון משמש להמרה של מתודה רגילה במחלקה למתודת מחלקה.

*   **אפשרות א'** אינה נכונה: הוא ממיר את המתודה למתודת מחלקה, לא למתודה סטטית.
*   **אפשרות ב'** אינה נכונה: דקורטורים אינם משפיעים על היכולת לדרוס מתודה.
*   **אפשרות ג'** נכונה: `@classmethod` מאפשר לקרוא למתודה ישירות מהמחלקה, ולא רק דרך מופע.
*   **אפשרות ד'** אינה נכונה: הדקורטור אינו משנה את רמת הנגישות של המתודה.

**מאפיינים מרכזיים של מתודות מחלקה:**

1.  **קשירה למחלקה:** מתודת מחלקה קשורה למחלקה עצמה, ולא למופעים שלה.
2.  **ארגומנט `cls`:** מתודת מחלקה מקבלת אוטומטית הפניה למחלקה (מקובל לכנות אותה `cls`) כארגומנט הראשון, וזהו הבדל לעומת מתודות רגילות (`self`).
3.  **גישה למאפייני מחלקה:** היא יכולה לגשת למאפייני מחלקה ולשנות אותם.
4.  **קריאה דרך המחלקה או מופע:** ניתן לקרוא למתודת מחלקה הן דרך המחלקה עצמה `ClassName.method()`, והן דרך מופע `instance.method()`.

**שימושים טיפוסיים למתודות מחלקה:**

*   **מתודות מפעל (Factory methods):** מתודות מחלקה משמשות לעיתים קרובות ליצירת בנאים חלופיים, כלומר, ליצירת אובייקטים של המחלקה בדרכים שונות.

**דוגמה:**

```python
class MyClass:
    class_var: int = 10

    def __init__(self, value: int):
        self.value = value
    
    @classmethod
    def create_with_default_value(cls) -> "MyClass":
      """מתודת מחלקה ליצירת מופע עם ערך קבוע מראש"""
      return cls(100)

    def instance_method(self) -> None:
        print(f"מתודת מופע. ערך המופע: {self.value} .ערך משתנה המחלקה: {MyClass.class_var}")

# קריאה למתודת המחלקה דרך המחלקה עצמה
obj1: MyClass = MyClass.create_with_default_value()
obj1.instance_method() # פלט: מתודת מופע. ערך המופע: 100 .ערך משתנה המחלקה: 10

# קריאה למתודת המחלקה דרך מופע
obj2 = MyClass(10)
obj2.modify_class_variable(100) # קריאה למתודת מופע שיכולה לשנות את משתנה המחלקה.
obj2.instance_method() # פלט: מתודת מופע. ערך המופע: 10 .ערך משתנה המחלקה: 100
```
**בתוצאה:**
*   מתודת `create_with_default_value` נקראת דרך המחלקה ומחזירה מופע חדש.
*   מתודת `instance_method` מדפיסה את ערך המאפיין של המופע ושל המחלקה.
*   מתודת המחלקה `modify_class_variable` משנה את משתנה המחלקה.

לפיכך, **אפשרות ג'** היא התשובה הנכונה.
