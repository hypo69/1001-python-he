**שאלה 28.** בפייתון, מה עושה המתודה `append()` כאשר היא מופעלת על רשימה?

- א. היא ממזגת רשימה אחרת עם הרשימה הנוכחית במיקום שצוין.
- ב. היא מוסיפה איבר חדש לסוף הרשימה, ומגדילה את גודלה.
- ג. היא מחשבת את הסכום הכולל של כל האיברים המספריים ברשימה.
- ד. היא מסירה את האיבר האחרון מהרשימה ומחזירה אותו.

**תשובה נכונה: B**

**הסבר:**

המתודה `append()` בפייתון משמשת להוספת איבר *בודד* לסוף רשימה. פעולה זו מגדילה את גודל הרשימה באחד.

*   **אפשרות A** אינה נכונה: למיזוג רשימות משתמשים במתודה `extend()` או באופרטור `+`, ולא ב-`append()`.
*   **אפשרות B** נכונה: `append()` אכן מוסיפה איבר לסוף הרשימה, ומגדילה את גודלה.
*   **אפשרות C** אינה נכונה: `append()` אינה מבצעת פעולות חשבון כלשהן.
*   **אפשרות D** אינה נכונה: להסרת האיבר האחרון מרשימה והחזרת ערכו משתמשים במתודה `pop()`, ולא ב-`append()`.

**דוגמה:**

```python
my_list: list[int] = [1, 2, 3]
print(f"Изначальный список: {my_list}") # פלט: הרשימה המקורית: [1, 2, 3]

# מוסיפים איבר חדש לסוף הרשימה באמצעות append()
my_list.append(4)
print(f"Список после append(4): {my_list}")  # פלט: הרשימה אחרי append(4): [1, 2, 3, 4]

my_list.append([5, 6])
print(f"Список после append([5, 6]): {my_list}")  # פלט: הרשימה אחרי append([5, 6]): [1, 2, 3, 4, [5, 6]]
```

**תוצאה:**

בדוגמה ניתן לראות ש-`append()` מוסיפה איבר חדש לסוף הרשימה, וכן מוסיפה רשימה אחרת כאיבר יחיד.

לפיכך, **אפשרות B** היא התשובה הנכונה.
