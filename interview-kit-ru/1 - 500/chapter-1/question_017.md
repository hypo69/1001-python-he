**שאלה 17.** מה יהיה ההתנהגות של פונקציית Python הבאה?

```python
def check_number(n):
    if n % 2 == 0:
        return "Even"
    elif n % 3 == 0:
        return "Divisible by 3"
    else:
        return "Other"
```

- A. הפונקציה מחזירה את הערך `"Even"` רק אם `n` מתחלק ב-2 וב-3.
- B. הפונקציה מחזירה את הערך `"Divisible by 3"` עבור כל המספרים המתחלקים ב-3, ללא קשר אם הם זוגיים או לא.
- C. הפונקציה מחזירה את הערך `"Even"` עבור כל המספרים הזוגיים, ואת הערך `"Divisible by 3"` עבור מספרים שאינם זוגיים אך מתחלקים ב-3.
- D. הפונקציה אינה יכולה להחזיר את הערך `"Other"`.

**תשובה נכונה: C**

**הסבר:**

הפונקציה `check_number(n)` בודקת את המספר `n` בהתאם לתנאים מסוימים ומחזירה מחרוזת בהתאם לתוצאת הבדיקה.

*   **`if n % 2 == 0:`**: בודק האם המספר `n` זוגי (כלומר, האם הוא מתחלק ב-2 ללא שארית). אם התנאי מתקיים, הפונקציה מחזירה מיד את המחרוזת `"Even"` ומפסיקה את ביצועה.

*   **`elif n % 3 == 0:`**: אם התנאי הקודם לא התקיים (כלומר, המספר `n` אינו זוגי), נבדק האם `n` מתחלק ב-3 ללא שארית. אם תנאי זה מתקיים, הפונקציה מחזירה את המחרוזת `"Divisible by 3"` ומפסיקה את ביצועה.

*   **`else:`**: אם אף אחד מהתנאים הקודמים לא התקיים (כלומר, המספר `n` אינו זוגי ואינו כפולה של 3), הפונקציה מחזירה את המחרוזת `"Other"`.

בואו ננתח את אפשרויות התשובה:

*   **אפשרות A** אינה נכונה: הפונקציה מחזירה `"Even"` אם המספר מתחלק ב-2 בלבד, ולא אם הוא מתחלק ב-2 וב-3.
*   **אפשרות B** אינה נכונה: אם מספר הוא זוגי (מתחלק ב-2), הבלוק `if n % 2 == 0:` מתבצע ראשון והפונקציה מחזירה `"Even"`, מבלי לבדוק התחלקות ב-3.
*   **אפשרות C** נכונה: הפונקציה עובדת בדיוק כך, בודקת תחילה זוגיות ולאחר מכן התחלקות ב-3.
*   **אפשרות D** אינה נכונה: הפונקציה יכולה להחזיר `"Other"` אם המספר אינו עומד באף אחד משני התנאים הראשונים.

**דוגמאות:**

```python
def check_number(n: int) -> str:
    if n % 2 == 0:
        return "Even"
    elif n % 3 == 0:
        return "Divisible by 3"
    else:
        return "Other"

print(check_number(4))   # Вывод: Even
print(check_number(9))  # Вывод: Divisible by 3
print(check_number(7))  # Вывод: Other
print(check_number(6)) # Вывод: Even (число делится на 2, первая ветка выполняется)
```

**בסיכום:**

*   `check_number(4)` מחזיר `"Even"`, מכיוון ש-4 מתחלק ב-2.
*   `check_number(9)` מחזיר `"Divisible by 3"`, מכיוון ש-9 אינו זוגי, אך מתחלק ב-3.
*   `check_number(7)` מחזיר `"Other"`, מכיוון ש-7 אינו מתחלק ב-2 ואינו מתחלק ב-3.
*   `check_number(6)` מחזיר `"Even"`, מכיוון ש-6 מתחלק ב-2, והתנאים הבאים אינם נבדקים.

לפיכך, **אפשרות C** היא היחידה הנכונה.
