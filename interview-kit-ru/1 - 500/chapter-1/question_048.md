### `question_048.md`

**שאלה 48.** ב-Python, מהו תפקיד המשתנה `__name__`, וכיצד הוא משמש בדרך כלל בסקריפטים ומודולים?

- א. המשתנה `__name__` משמש לזיהוי שם המחלקה של אובייקט, ומאפשר התייחסות דינמית לסוג המחלקה ברחבי היישום.
- ב. זהו משתנה מובנה המכיל את שם המודול שבו הוא משמש; אם המודול מופעל כתוכנית ראשית, הוא מכיל את המחרוזת `"__main__"`.
- ג. משתנה זה פועל כמזהה עבור כתובות זיכרון, ומסייע בניהול ישיר של מיקומי אובייקטים בתוך יישומי Python.
- ד. `__name__` הוא כלי דיבוג של Python המדפיס את נתיב הביצוע של הפונקציה הנוכחית, ומסייע במעקב ובתיעוד.

**תשובה נכונה: B**

**הסבר:**

המשתנה `__name__` הוא משתנה מיוחד ומובנה ב-Python, המוגדר אוטומטית על ידי מפרש Python. הוא משמש כדי לקבוע האם מודול מופעל כסקריפט ראשי או מיובא למודול אחר.

*   **אפשרות א** אינה נכונה: `__name__` אינו קשור לשמות מחלקות או לטיפוס דינמי.
*   **אפשרות ב** נכונה: כאשר מודול מופעל כתוכנית ראשית, Python מגדיר את `__name` ל-"__main__". אם מודול מיובא לסקריפט אחר, `__name` מוגדר לשם המודול.
*   **אפשרות ג** אינה נכונה: `__name` אינו קשור לניהול כתובות זיכרון.
*   **אפשרות ד** אינה נכונה: `__name` אינו מיועד לדיבוג, אלא לקביעת הקשר ההפעלה של המודול.

**כיצד משמש `__name__`:**

הוא משמש לעתים קרובות כדי לקבוע האם הסקריפט הוא קובץ ההפעלה הראשי או מיובא כמודול. זה מאפשר להריץ קוד מסוים, כגון קריאה לפונקציה `main()`, רק אם הסקריפט הופעל ישירות.

**דוגמה:**

```python
# my_module.py
print(f"Имя модуля: {__name__}")

def my_function() -> None:
    print("Вызов my_function")

if __name__ == "__main__":
    # This code runs if the script is executed directly.
    print("Этот код выполняется, если скрипт запущен напрямую.")
    my_function()

# main.py
import my_module

print(f"Имя модуля из main.py: {__name__}")
my_module.my_function()
```

**כתוצאה מכך:**

1.  אם מפעילים את `my_module.py` ישירות:
    *   `print(f"Имя модуля: {__name__}")` ידפיס `Имя модуля: __main__`.
    *   התנאי `if __name__ == "__main__":` יתקיים, מכיוון שהמודול הופעל ישירות.
    *   יודפס `"Этот код выполняется, если скрипт запущен напрямую."` וגם `"Вызов my_function"`.
2.  אם מפעילים את `main.py`:
    *   `print(f"Имя модуля из main.py: {__name__}")` ידפיס `Имя модуля из main.py: __main__` (מכיוון ש-`main.py` הוא הסקריפט הראשי).
    *   `print(f"Имя модуля: {__name__}")` ב-`my_module.py` ידפיס `Имя модуля: my_module`, מכיוון שהמודול `my_module` יובא לתוך main.py.
    *   `my_module.my_function()` ידפיס `"Вызов my_function"`

לכן, **אפשרות B** היא התשובה הנכונה.
