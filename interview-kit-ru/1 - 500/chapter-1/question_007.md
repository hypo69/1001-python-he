**שאלה 7.** בעת יצירת פונקציה ב-Python שמחשבת את עצרת המספר באמצעות רקורסיה, איזו מבין ההגדרות הבאות לפונקציה ממומשת נכונה ומתאימה לעקרונות הרקורסיה?

- A
```python
def factorial(n):
    return n * factorial(n-1) if n > 1 else 1
```

- B
```python
def factorial(n): return factorial(n-1) * n if n == 0 else 1
```

- C
```python
def factorial(n):
    factorial(n-1) * n
```

- D
```python
def factorial(n):
    return n * factorial(n) if n > 1 else 1

```

**תשובה נכונה: A**

**הסבר:**

*   **פונקציה רקורסיבית:** פונקציה רקורסיבית היא פונקציה שקוראת לעצמה. חלק חשוב מפונקציה רקורסיבית הוא קיומם של **מקרה בסיס** (base case), שמפסיק את הרקורסיה, ו**מקרה רקורסיבי**, שקורא לפונקציה עם נתוני קלט ששונו כדי להתקרב למקרה הבסיס.

*   **עצרת (Factorial):** עצרת מספר `n` (מסומנת כ-`n!`) היא מכפלת כל המספרים הטבעיים עד `n`. לדוגמה, `5! = 5 * 4 * 3 * 2 * 1 = 120`.

*   **מימוש נכון של רקורסיה:**
    *   **מקרה בסיס:** עבור עצרת, מקרה הבסיס הוא `n = 1` (או `n <= 1`, כולל 0!). במקרה זה, העצרת שווה ל-1.
    *   **מקרה רקורסיבי:** עבור `n > 1`, העצרת מחושבת כ-`n * factorial(n-1)`, כלומר המספר הנוכחי מוכפל בעצרת של המספר הקטן ממנו באחד.
*   **בדיקת תנאי:** חשוב שהקריאה ל-`factorial(n-1)` תתבצע רק כאשר `n>1`, אחרת תהיה רקורסיה אינסופית.

**דוגמה:**

```python
def factorial(n: int) -> int:
    # אם n גדול מ-1, זהו המקרה הרקורסיבי
    if n > 1:
      return n * factorial(n-1)
    # אחרת, זהו מקרה הבסיס
    else:
      return 1


# הדפסת עצרת 5
print(f"factorial(5): {factorial(5)}") # פלט: factorial(5): 120

# הדפסת עצרת 0
print(f"factorial(0): {factorial(0)}") # פלט: factorial(0): 1

# הדפסת עצרת 1
print(f"factorial(1): {factorial(1)}") # פלט: factorial(1): 1
```

**לסיכום:**

*   אפשרות **A** `def factorial(n): return n * factorial(n-1) if n > 1 else 1` מממשת נכונה את הפונקציה הרקורסיבית לחישוב עצרת, עם מקרה בסיס נכון וקריאה רקורסיבית מתאימה.
*   לאפשרות **B** `def factorial(n): return factorial(n-1) * n if n == 0 else 1` יש מקרה בסיס שגוי, שלא יטפל בערכים חיוביים של n.
*   לאפשרות **C** `def factorial(n): factorial(n-1) * n` אין מקרה בסיס והיא אינה מחזירה ערך, מה שיוביל לרקורסיה אינסופית ולשגיאה.
*   לאפשרות **D** `def factorial(n): return n * factorial(n) if n > 1 else 1` יש רקורסיה אינסופית, מכיוון שהיא קוראת לפונקציית העצרת עם אותו מספר ולא עם `n-1`.

לכן, תשובה **A** היא הנכונה, מכיוון שהיא מכילה הגדרה נכונה של פונקציה רקורסיבית, עם מקרה בסיס ושלב רקורסיבי שמקטין את נתוני הקלט ומקרב אותם למקרה הבסיס.
