**שאלה 7.** בעת יצירת פונקציה בפייתון המחשבת עצרת (פקטוריאל) של מספר באמצעות רקורסיה, איזו מההגדרות הבאות לפונקציה ממומשת כהלכה ותואמת את עקרונות הרקורסיה?

- A
```python
def factorial(n):
    return n * factorial(n-1) if n > 1 else 1
```

- B
```python
def factorial(n): return factorial(n-1) * n if n == 0 else 1
```

- C
```python
def factorial(n):
    factorial(n-1) * n
```

- D
```python
def factorial(n):
    return n * factorial(n) if n > 1 else 1

```

**תשובה נכונה: A**

**הסבר:**

*   **פונקציה רקורסיבית:** פונקציה רקורסיבית היא פונקציה המפעילה את עצמה. חלק חשוב בפונקציה רקורסיבית הוא קיומו של **מקרה בסיס** (base case) אשר עוצר את הרקורסיה, ו**מקרה רקורסיבי** אשר מפעיל את הפונקציה עם נתוני קלט שהשתנו כדי להתקרב למקרה הבסיס.

*   **עצרת (פקטוריאל):** עצרת של מספר `n` (מסומן כ-`n!`) הוא מכפלת כל המספרים הטבעיים עד ל-`n`. לדוגמה, `5! = 5 * 4 * 3 * 2 * 1 = 120`.

*   **מימוש רקורסיה תקין:**
    *   **מקרה בסיס:** עבור עצרת, מקרה הבסיס הוא `n = 1` (או `n <= 1`, כולל 0!). במקרה זה העצרת שווה ל-1.
    *   **מקרה רקורסיבי:** עבור `n > 1`, העצרת מחושבת כ-`n * factorial(n-1)`, כלומר המספר הנוכחי מוכפל בעצרת של המספר הקטן ממנו באחד.
*   **בדיקת תנאי:** חשוב שקריאת `factorial(n-1)` תתבצע רק כאשר `n > 1`, אחרת תהיה רקורסיה אינסופית.

**דוגמה:**

```python
def factorial(n: int) -> int:
    if n > 1:
      return n * factorial(n-1)
    else:
      return 1


print(f"factorial(5): {factorial(5)}") # Вывод: factorial(5): 120

print(f"factorial(0): {factorial(0)}") # Вывод: factorial(0): 1

print(f"factorial(1): {factorial(1)}") # Вывод: factorial(1): 1
```

**לסיכום:**

*   אפשרות **A** `def factorial(n): return n * factorial(n-1) if n > 1 else 1` מממשת כהלכה פונקציה רקורסיבית לחישוב עצרת, עם מקרה בסיס תקין וקריאה רקורסיבית נכונה.
*   לאפשרות **B** `def factorial(n): return factorial(n-1) * n if n == 0 else 1` יש מקרה בסיס שגוי אשר לא יטפל בערכים חיוביים של n.
*   לאפשרות **C** `def factorial(n): factorial(n-1) * n` אין מקרה בסיס ואינה מחזירה ערך, מה שיוביל לרקורסיה אינסופית ושגיאה.
*    לאפשרות **D** `def factorial(n): return n * factorial(n) if n > 1 else 1` יש רקורסיה אינסופית, מכיוון שהיא קוראת לעצרת של אותו מספר ולא של `n-1`.

לפיכך, התשובה **A** היא הנכונה, מכיוון שהיא מכילה הגדרה תקינה של פונקציה רקורסיבית, עם מקרה בסיס וצעד רקורסיבי אשר מקטין את נתוני הקלט ומקרב אותם למקרה הבסיס.
