**שאלה 655.** מה יודפס למסך על ידי קוד הפייתון הבא?

```python
def mystery_function(x):
    if x == 0:
        return 0
    return x + mystery_function(x - 1)

print(mystery_function(10))
```

-   A. 10
-   B. 55
-   C. 100
-   D. 0
-   E. שגיאה

**תשובה נכונה: B**

**הסבר:**

קוד זה מדגים פונקציה רקורסיבית בשם `mystery_function(x)`.

1.  **הפונקציה `mystery_function(x)`:**
    *   **מקרה בסיס:** אם `x` שווה ל-0, הפונקציה מחזירה `0`.
    *   **מקרה רקורסיבי:** אם `x` אינו שווה ל-0, הפונקציה מחזירה את הסכום של `x` והתוצאה של קריאה ל-`mystery_function(x - 1)`.

2.  **קריאה לפונקציה:**
    *   `print(mystery_function(10))`: קורא לפונקציה `mystery_function` עם הארגומנט 10, ולאחר מכן מדפיס את התוצאה.

3.  **פעולת הרקורסיה:**
    *   `mystery_function(10)` תחזיר `10 + mystery_function(9)`.
    *   `mystery_function(9)` תחזיר `9 + mystery_function(8)`.
    *   ...
    *   `mystery_function(1)` תחזיר `1 + mystery_function(0)`.
    *   `mystery_function(0)` תחזיר `0`.
    *   התוצאה תהיה הסכום `10 + 9 + 8 + ... + 1 + 0`.

4.  **חישוב הסכום:**
    *   זהו סכום המספרים מ-0 עד 10, אשר מחושב לפי הנוסחה `n * (n + 1) / 2` עבור n = 10, ששווה ל-`10 * 11 / 2 = 55`.

**ניתוח האפשרויות:**
*   **A. 10:** לא נכון.
*   **B. 55:** נכון.
*   **C. 100:** לא נכון.
*   **D. 0:** לא נכון.
*   **E. שגיאה:** לא נכון. הקוד יתבצע ללא שגיאות.

**לסיכום:**
*   הפונקציה `mystery_function` מחשבת את סכום המספרים מ-0 עד `x`.
*   נעשה שימוש ברקורסיה לקריאה עוקבת לפונקציה.

לפיכך, התשובה הנכונה היא **B. 55**.
