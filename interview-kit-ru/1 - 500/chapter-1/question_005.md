### `question_005.md`

**שאלה 5.** בעת הגדרת פונקציה ב-Python, ניתן להגדיר ערכי ברירת מחדל עבור פרמטרים, מה שמספק גמישות רבה יותר בעת קריאה לפונקציה. מה תהיה התוצאה של קריאה לפונקציה הבאה:

```python
def multiply(a, b=2): 
    return a * b
```
אם קוראים לפונקציה באופן הבא:
```python
multiply(5)
```

- א. הפונקציה תחזיר 5.

- ב. הפונקציה תגרום לשגיאת `TypeError` עקב ארגומנט חסר.

- ג. הפונקציה תחזיר 10, מכיוון שיעשה שימוש בערך ברירת המחדל של `b`.

- ד. הפונקציה תחזיר `None`, מאחר שסופק רק פרמטר אחד.

**תשובה נכונה: ג**

**הסבר:**

*   **פרמטרים עם ערכי ברירת מחדל:** ב-Python ניתן להגדיר ערכי ברירת מחדל עבור פרמטרים של פונקציה. משמעות הדבר היא שאם בעת קריאה לפונקציה לא מצוין ארגומנט עבור פרמטר זה, ייעשה שימוש בערך ברירת המחדל.

*   **יישום ערך ברירת המחדל:** אם קוראים לפונקציה `multiply` עם ארגומנט אחד, למשל `multiply(5)`, הערך שהועבר, `5`, יוקצה לפרמטר `a`, ועבור הפרמטר `b` ייעשה שימוש בערך ברירת המחדל, שהוא `2`.
*  **תוצאה:** הפונקציה תחזיר את תוצאת הכפלה של הערך שהועבר ל-`a` בערך ברירת המחדל של `b`, כלומר 5 * 2 = 10.

**דוגמה:**

```python
def multiply(a: int, b: int = 2) -> int:
    return a * b

result1: int = multiply(5)
print(f"multiply(5): {result1}") # Вывод: multiply(5): 10

result2: int = multiply(5, 3)
print(f"multiply(5, 3): {result2}") # Вывод: multiply(5, 3): 15
```

**בתוצאה:**

*   בעת קריאה ל-`multiply(5)`, הערך `5` מוקצה לפרמטר `a`, והפרמטר `b` מקבל את ערך ברירת המחדל שלו `2`. הפונקציה מחזירה `5 * 2 = 10`.

*   בעת קריאה ל-`multiply(5, 3)`, הערך `5` מוקצה לפרמטר `a`, והערך `3` מוקצה לפרמטר `b`. הפונקציה מחזירה `5 * 3 = 15`.

לפיכך, תשובה **ג** נכונה, מכיוון שהיא מתארת במדויק את התוצאה של קריאה לפונקציה `multiply(5)` תוך שימוש בערך ברירת המחדל עבור הפרמטר `b`.
