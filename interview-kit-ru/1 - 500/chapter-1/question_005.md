### `question_005.md`

**שאלה 5.** בעת הגדרת פונקציה ב-Python, ניתן להגדיר ערכי ברירת מחדל לפרמטרים, 
מה שמעניק גמישות רבה יותר בעת קריאה לפונקציה. מה תהיה התוצאה של קריאה לפונקציה הבאה:

```python
def multiply(a, b=2): 
    return a * b`
```
, אם הפונקציה נקראת באופן הבא:
```python
multiply(5)
```


- א. הפונקציה תחזיר 5.

- ב. הפונקציה תגרום ל-`TypeError` עקב חוסר בארגומנט.

- ג. הפונקציה תחזיר 10, מכיוון שייעשה שימוש בערך ברירת המחדל של `b`.

- ד. הפונקציה תחזיר `None`, מאחר שסופק רק פרמטר אחד.

**תשובה נכונה: ג**

**הסבר:**

*   **פרמטרים עם ערכי ברירת מחדל:** ב-Python, ניתן להגדיר ערכי ברירת מחדל עבור פרמטרים של פונקציה. משמעות הדבר היא שאם בעת קריאה לפונקציה לא מצוין ארגומנט עבור פרמטר זה, ייעשה שימוש בערך ברירת המחדל.

*   **שימוש בערך ברירת המחדל:** כאשר קוראים לפונקציה `multiply` עם ארגומנט אחד, למשל `multiply(5)`, הערך שהועבר `5` יוקצה לפרמטר `a`, ועבור הפרמטר `b` ייעשה שימוש בערך ברירת המחדל, שהוא `2`.
*   **תוצאה:** הפונקציה תחזיר את תוצאת הכפלת הערך שהועבר ל-`a` בערך ברירת המחדל של `b`, כלומר 5 * 2 = 10.

**דוגמה:**

```python
def multiply(a: int, b: int = 2) -> int:
    return a * b

result1: int = multiply(5)
print(f"multiply(5): {result1}") # פלט: multiply(5): 10

result2: int = multiply(5, 3)
print(f"multiply(5, 3): {result2}") # פלט: multiply(5, 3): 15
```

**בסיכום:**

*   בקריאה ל-`multiply(5)`, הערך `5` מוקצה לפרמטר `a`, והפרמטר `b` מקבל את ערך ברירת המחדל שלו `2`. הפונקציה מחזירה `5 * 2 = 10`.

*   בקריאה ל-`multiply(5, 3)`, הערך `5` מוקצה לפרמטר `a`, והערך `3` מוקצה לפרמטר `b`. הפונקציה מחזירה `5 * 3 = 15`.

לפיכך, תשובה **ג** היא הנכונה, מאחר שהיא מתארת בדיוק את התוצאה של קריאה לפונקציה `multiply(5)` עם שימוש בערך ברירת המחדל עבור הפרמטר `b`.
