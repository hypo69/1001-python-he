### `question_008.md`

**שאלה 8.** איזו מהטענות הבאות לגבי רשימות ב-Python היא נכונה, במיוחד בהתייחס לגמישות סוגי האלמנטים שרשימה יכולה להכיל?

- א. רשימות ב-Python יכולות להכיל רק אלמנטים מאותו סוג נתונים, כגון כולם מספרים שלמים או כולם מחרוזות.
- ב. רשימות ב-Python יכולות להכיל אלמנטים מסוגי נתונים שונים, כגון מספרים שלמים, מחרוזות ואובייקטים, באותה רשימה.
- ג. רשימות ב-Python אינן יכולות להכיל סוגי אוספים (collections) אחרים, כגון רשימות או מילונים אחרים.
- ד. רשימות ב-Python הן בלתי ניתנות לשינוי (immutable), כלומר, לאחר יצירתן, לא ניתן לשנות את האלמנטים שלהן.

**תשובה נכונה: B**

**הסבר:**

רשימות ב-Python גמישות מאוד ביחס לסוגי הנתונים שהן יכולות להכיל. בניגוד למערכים (arrays) בשפות תכנות אחרות, רשימות Python אינן מגבילות את האחסון לסוג נתונים יחיד בלבד.

*   **אפשרות א'** אינה נכונה, מכיוון שרשימה יכולה להכיל אלמנטים מסוגים שונים.
*   **אפשרות ב'** נכונה, שכן היא משקפת נכונה את הגמישות של רשימות Python.
*   **אפשרות ג'** אינה נכונה, מכיוון שרשימות יכולות להכיל אוספים אחרים, כולל רשימות ומילונים.
*   **אפשרות ד'** אינה נכונה, מכיוון שרשימות הן *ניתנות לשינוי* (mutable), כלומר, לאחר יצירתן, ניתן לשנות את תוכנן.

**דוגמה:**

```python
# יצירת רשימה עם אלמנטים מסוגים שונים
my_list: list[str | int | list[int] | dict[str, int]] = [
    10,           # מספר שלם
    "привет",       # מחרוזת
    [1, 2, 3],     # רשימה של מספרים שלמים
    {"a": 1, "b": 2} # מילון
]
print(my_list)   # פלט: [10, 'привет', [1, 2, 3], {'a': 1, 'b': 2}]

# בדיקת יכולת השינוי (mutability)
my_list[0] = 20  # שינוי האלמנט הראשון
print(my_list)   # פלט: [20, 'привет', [1, 2, 3], {'a': 1, 'b': 2}]
my_list.append(True) # הוספת אלמנט נוסף
print(my_list) # פלט: [20, 'привет', [1, 2, 3], {'a': 1, 'b': 2}, True]
```

**לסיכום:**

בדוגמה זו יצרנו רשימה בשם `my_list` המכילה אלמנטים מסוגי נתונים שונים: מספר שלם, מחרוזת, רשימה נוספת ומילון. בנוסף, שינינו בהצלחה אלמנט ברשימה, מה שמאשש את היותה ניתנת לשינוי.

לפיכך, **אפשרות ב'** היא התשובה הנכונה היחידה.
