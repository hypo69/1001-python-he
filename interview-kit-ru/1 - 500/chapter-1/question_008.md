### `question_008.md`

**שאלה 8.** איזו מבין הטענות הבאות לגבי רשימות בפייתון היא נכונה, במיוחד כאשר מדובר על גמישות סוגי האלמנטים שרשימה יכולה להכיל?

*   א. רשימות בפייתון יכולות להכיל רק אלמנטים מאותו סוג נתונים, כגון כל המספרים השלמים או כל המחרוזות.
*   ב. רשימות בפייתון יכולות להכיל אלמנטים מסוגי נתונים שונים, כגון מספרים שלמים, מחרוזות ואובייקטים, באותה רשימה.
*   ג. רשימות בפייתון אינן יכולות להכיל סוגי אוספים (collections) אחרים, כגון רשימות אחרות או מילונים.
*   ד. רשימות בפייתון הן בלתי ניתנות לשינוי (immutable), כלומר, לאחר יצירתן, לא ניתן לשנות את האלמנטים שלהן.

**תשובה נכונה: ב**

**הסבר:**

לרשימות בפייתון יש גמישות רבה ביחס לסוגי הנתונים שהן יכולות להכיל. בניגוד למערכים בשפות תכנות מסוימות אחרות, רשימות בפייתון אינן מגבילות אתכם לאחסון סוג נתונים אחד בלבד.

*   **אפשרות א** אינה נכונה, מכיוון שרשימה יכולה להכיל אלמנטים מסוגים שונים.
*   **אפשרות ב** נכונה, שכן היא משקפת נכונה את גמישותן של רשימות בפייתון.
*   **אפשרות ג** אינה נכונה, מכיוון שרשימות יכולות להכיל אוספים אחרים, כולל רשימות ומילונים.
*   **אפשרות ד** אינה נכונה, מכיוון שרשימות הן *ניתנות לשינוי* (mutable), כלומר, לאחר יצירתן ניתן לשנות את תוכנן.

**דוגמה:**

```python
# יוצרים רשימה עם אלמנטים מסוגים שונים
my_list: list[str | int | list[int] | dict[str, int]] = [
    10,           # מספר שלם
    "привет",       # מחרוזת
    [1, 2, 3],     # רשימת מספרים שלמים
    {"a": 1, "b": 2} # מילון
]
print(my_list)   # פלט: [10, 'привет', [1, 2, 3], {'a': 1, 'b': 2}]

# בודקים את היכולת לשינוי (mutability)
my_list[0] = 20  # משנים את האלמנט הראשון
print(my_list)   # פלט: [20, 'привет', [1, 2, 3], {'a': 1, 'b': 2}]
my_list.append(True) # מוסיפים אלמנט נוסף
print(my_list) # פלט: [20, 'привет', [1, 2, 3], {'a': 1, 'b': 2}, True]
```

**תוצאה:**

בדוגמה זו יצרנו רשימה בשם `my_list` המכילה אלמנטים מסוגי נתונים שונים: מספר שלם, מחרוזת, רשימה נוספת ומילון. בנוסף, שינינו בהצלחה אלמנט ברשימה, מה שמאשר את היותה ניתנת לשינוי (mutable).

לפיכך, **אפשרות ב** היא התשובה הנכונה היחידה.
