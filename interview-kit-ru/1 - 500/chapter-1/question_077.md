**שאלה 77.** מהן השיטות העיקריות להעתקת רשימות בפייתון, ומה ההבדל בין העתקה שטחית (shallow copy) להעתקה עמוקה (deep copy)? הצג דוגמאות המדגימות את שיטות ההעתקה השונות ואת השפעתן.

-   א. בפייתון קיימת רק שיטה אחת להעתקת רשימות - שימוש במתודה `copy()`, כל שאר השיטות יוצרות הפניות לרשימה המקורית.
-   ב. קיימות שיטות להעתקה שטחית (`[:]`, `list()`, `copy()`) ולהעתקה עמוקה (`deepcopy()`). העתקה שטחית יוצרת רשימה חדשה, אך אלמנטים של רשימות מקוננות נשארים הפניות לרשימה המקורית. העתקה עמוקה יוצרת עותקים בלתי תלויים לחלוטין של רשימות, כולל כל האובייקטים המקוננים.
-   ג. קיימות שלוש שיטות להעתקת רשימות: `[:]`, `list()` ו-`copy()`, אשר מבצעות העתקה עמוקה.
-   ד. בפייתון לא ניתן להעתיק רשימות, ניתן רק לשנות את שמן.

**תשובה נכונה: ב**

**הסבר:**

בפייתון קיימות מספר שיטות להעתקת רשימות, אשר יוצרות או עותקים *שטחיים* (shallow copies) או עותקים *עמוקים* (deep copies). הבנת ההבדל ביניהם חיונית למניעת שגיאות הקשורות לשינויים בלתי רצויים בנתונים.

1.  **העתקה שטחית (Shallow Copy):**
    *   יוצרת אובייקט רשימה חדש, אך אינה מעתיקה באופן רקורסיבי את האובייקטים הכלולים ברשימה המקורית.
    *   אם אלמנטים של הרשימה המקורית הם אובייקטים הניתנים לשינוי (mutabile) בעצמם (לדוגמה, רשימות מקוננות), הרשימה שהועתקה תכיל הפניות לאובייקטים אלה, ולא עותקים שלהם.
    *   שיטות ליצירת עותק שטחי:
        *   **פרוסה (`[:]`):** `new_list = old_list[:]`
        *   **הפונקציה `list()`:** `new_list = list(old_list)`
        *   **המתודה `copy()`:** `new_list = old_list.copy()`
2.  **העתקה עמוקה (Deep Copy):**
    *   יוצרת אובייקט רשימה חדש ומעתיקה באופן רקורסיבי את כל האובייקטים הכלולים ברשימה המקורית, כולל אובייקטים מקוננים הניתנים לשינוי.
    *   שינויים ברשימה שהועתקה באופן עמוק לא ישפיעו על הרשימה המקורית ולהיפך.
    *   משתמשים בפונקציה `deepcopy()` מהמודול `copy`.
        *   `import copy; new_list = copy.deepcopy(old_list)`

**דוגמאות קוד (העתקה שטחית):**

```python
import copy

old_list = [1, 2, [3, 4]]

# העתקה שטחית (באמצעות פרוסה)
new_list1 = old_list[:]

# העתקה שטחית (באמצעות list())
new_list2 = list(old_list)

# העתקה שטחית (באמצעות .copy())
new_list3 = old_list.copy()

print(f"Original list id: {id(old_list)}")
print(f"new_list1 id: {id(new_list1)}")
print(f"new_list2 id: {id(new_list2)}")
print(f"new_list3 id: {id(new_list3)}")


old_list[0] = 100 # שינוי ברשימה הישנה
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

old_list[2][0] = 5 # שינוי ברשימה מקוננת
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

```

```
new_list2 id: 140493391091520
new_list3 id: 140493391091456
new_list1: [1, 2, [3, 4]]
new_list2: [1, 2, [3, 4]]
new_list3: [1, 2, [3, 4]]
new_list1: [1, 2, [5, 4]]
new_list2: [1, 2, [5, 4]]
new_list3: [1, 2, [5, 4]]
```

**דוגמאות קוד (העתקה עמוקה):**

```python
import copy

old_list = [1, 2, [3, 4]]

# העתקה עמוקה (באמצעות deepcopy)
new_list = copy.deepcopy(old_list)
print(f"Original list id: {id(old_list)}")
print(f"new_list id: {id(new_list)}")


old_list[0] = 100
old_list[2][0] = 5
print(f"old_list: {old_list}")
print(f"new_list: {new_list}")
```

**פלט:**

```
Original list id: 140493391086464
new_list id: 140493391085952
old_list: [100, 2, [5, 4]]
new_list: [1, 2, [3, 4]]
```
