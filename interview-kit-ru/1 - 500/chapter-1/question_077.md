### `question_077.md`

**שאלה 77.** מהן הדרכים העיקריות להעתקת רשימות בפייתון, ומהו ההבדל בין העתקה שטחית לעתקה עמוקה? צרף דוגמאות המדגימות דרכים שונות להעתקה ואת השפעתן.

-   א. בפייתון יש רק דרך אחת להעתקת רשימות - שימוש במתודה `copy()`, כל הדרכים האחרות יוצרות הפניות לרשימה המקורית.
-   ב. קיימות דרכים להעתקה שטחית (`[:]`, `list()`, `copy()`) ולהעתקה עמוקה (`deepcopy()`). העתקה שטחית יוצרת רשימה חדשה, אך האלמנטים של רשימות מקוננות נשארים הפניות לרשימה המקורית. העתקה עמוקה יוצרת עותקים בלתי תלויים לחלוטין של רשימות, כולל כל האובייקטים המקוננים.
-   ג. קיימות שלוש דרכים להעתקת רשימות: `[:]`, `list()` ו-`copy()`, המבצעות העתקה עמוקה.
-   ד. בפייתון לא ניתן להעתיק רשימות, אפשר רק לשנות את שמן.

**התשובה הנכונה: ב**

**הסבר:**

בפייתון קיימות מספר דרכים להעתקת רשימות, היוצרות או עותקים *שטחיים* או עותקים *עמוקים*. הבנת ההבדל ביניהם קריטית למניעת שגיאות הקשורות לשינויים לא רצויים בנתונים.

1.  **העתקה שטחית (Shallow Copy):**
    *   יוצרת אובייקט רשימה חדש, אך אינה מעתיקה באופן רקורסיבי את האובייקטים הכלולים ברשימה המקורית.
    *   אם אלמנטים ברשימה המקורית הם עצמם אובייקטים ניתנים לשינוי (לדוגמה, רשימות מקוננות), הרי שברשימה שהועתקה יישמרו הפניות לאובייקטים אלו, ולא עותקים שלהם.
    *   דרכים ליצירת העתקה שטחית:
        *   **פריסה (`[:]`):** `new_list = old_list[:]`
        *   **הפונקציה `list()`:** `new_list = list(old_list)`
        *   **המתודה `copy()`:** `new_list = old_list.copy()`
2.  **העתקה עמוקה (Deep Copy):**
    *   יוצרת אובייקט רשימה חדש ומעתיקה באופן רקורסיבי את כל האובייקטים הכלולים ברשימה המקורית, כולל אובייקטים מקוננים ניתנים לשינוי.
    *   שינויים ברשימה שהועתקה בעותק עמוק לא ישפיעו על הרשימה המקורית ולהיפך.
    *   נעשה שימוש בפונקציה `deepcopy()` מהמודול `copy`.
        *   `import copy; new_list = copy.deepcopy(old_list)`

**דוגמאות קוד (העתקה שטחית):**

```python
import copy

old_list = [1, 2, [3, 4]]

# Shallow copy (using slice)
new_list1 = old_list[:]

# Shallow copy (using list())
new_list2 = list(old_list)

# Shallow copy (using .copy())
new_list3 = old_list.copy()

print(f"Original list id: {id(old_list)}")
print(f"new_list1 id: {id(new_list1)}")
print(f"new_list2 id: {id(new_list2)}")
print(f"new_list3 id: {id(new_list3)}")


old_list[0] = 100 # Changing the original list
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

old_list[2][0] = 5 # Changing the nested list
print(f"new_list1: {new_list1}")
print(f"new_list2: {new_list2}")
print(f"new_list3: {new_list3}")

```

**פלט:**
```

new_list1 id: 140493391092032
new_list2 id: 140493391091520
new_list3 id: 140493391091456
new_list1: [1, 2, [3, 4]]
new_list2: [1, 2, [3, 4]]
new_list3: [1, 2, [3, 4]]
new_list1: [1, 2, [5, 4]]
new_list2: [1, 2, [5, 4]]
new_list3: [1, 2, [5, 4]]
```

**דוגמאות קוד (העתקה עמוקה):**

```python
import copy

old_list = [1, 2, [3, 4]]

# Deep copy (using deepcopy)
new_list = copy.deepcopy(old_list)
print(f"Original list id: {id(old_list)}")
print(f"new_list id: {id(new_list)}")


old_list[0] = 100
old_list[2][0] = 5
print(f"old_list: {old_list}")
print(f"new_list: {new_list}")
```

**פלט:**

```
Original list id: 140493391086464
new_list id: 140493391085952
old_list: [100, 2, [5, 4]]
new_list: [1, 2, [3, 4]]
```
