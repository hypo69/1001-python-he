**שאלה 81** כיצד ניתן לשרשר (לאחד) שתי רשימות ב-Python?

 - א. באמצעות אופרטור `*`.
 - ב. באמצעות אופרטור `+`.
 - ג. באמצעות מתודה `append()`.
 - ד. באמצעות מתודה `concat()`.

**תשובה נכונה: ב. באמצעות אופרטור `+`**

**הסבר:**

ב-Python, לאיחוד שתי רשימות לרשימה חדשה אחת משתמשים באופרטור `+`. אופרטור זה יוצר רשימה חדשה המכילה את כל האיברים מהרשימה הראשונה, ולאחריהם את כל האיברים מהרשימה השנייה.

*   **אופרטור `+`:**
    *   יוצר רשימה חדשה המהווה תוצאה של איחוד שתי הרשימות המקוריות.
    *   הרשימות המקוריות אינן משתנות.

**ניתוח האפשרויות:**
*   **באמצעות אופרטור `*`:** שגוי. אופרטור `*` משמש לשכפול רשימה מספר מוגדר של פעמים.
*   **באמצעות אופרטור `+`:** נכון. אופרטור `+` משמש לשרשור (איחוד) רשימות.
*   **באמצעות מתודה `append()`:** שגוי. המתודה `append()` מוסיפה איבר בודד לסוף רשימה קיימת, ולא מאחדת שתי רשימות.
*   **באמצעות מתודה `concat()`:** שגוי. ב-Python אין מתודה מובנית `concat()` לרשימות.

**דוגמה לשרשור רשימות עם האופרטור `+`:**

```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]

concatenated_list = list1 + list2 # נוצרת רשימה חדשה מאיחוד שתי האחרות

print(f"list1: {list1}")
print(f"list2: {list2}")
print(f"concatenated_list: {concatenated_list}")

# פלט
# list1: [1, 2, 3]
# list2: [4, 5, 6]
# concatenated_list: [1, 2, 3, 4, 5, 6]
```

**לסיכום:**
*   האופרטור `+` הוא הדרך הפשוטה והנפוצה ביותר לאיחוד רשימות ב-Python.
*   הרשימות המקוריות אינן משתנות, ונוצרת רשימה חדשה המכילה את כל האיברים משתי הרשימות המקוריות.
*   שרשור רשימות באמצעות `+` עשויה להיות פחות יעילה עבור רשימות גדולות מאוד.

לפיכך, התשובה הנכונה היא **באמצעות אופרטור `+`**.
