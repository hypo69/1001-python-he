**שאלה 75.** מהי הערכה עצלה (lazy evaluation) ב-Python, וכיצד היא פועלת? ספק דוגמה המדגימה שימוש בחישובים עצלים.

-   א. הערכה עצלה היא שיטת הערכה בה ביטויים מחושבים מיד לאחר הגדרתם.
-   ב. הערכה עצלה היא אסטרטגיית חישובים בה ביטוי מחושב רק כאשר ערכו נדרש. ב-Python ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.
-   ג. הערכה עצלה מתייחסת למצב שבו נעשה שימוש רק במשתנים שאליהם יש גישה.
-   ד. הערכה עצלה היא שיטה להאצת חישובים על ידי ביצועם במקביל.

**תשובה נכונה: ב**

**הסבר:**

הערכה עצלה (lazy evaluation) היא אסטרטגיית חישובים בה ביטוי אינו מחושב באופן מיידי, אלא נדחה עד לרגע שבו ערכו נדרש באמת. אסטרטגיה זו מאפשרת לחסוך במשאבי מחשוב, כיוון שהחישובים אינם מבוצעים כל עוד התוצאה שלהם אינה נדרשת בפועל.

*   **מושגי יסוד בהערכה עצלה:**
    *   **חישובים נדחים:** ערכי ביטויים אינם מחושבים באופן מיידי, אלא רק כאשר הם נדרשים.
    *   **חיסכון במשאבים:** מאפשר לחסוך בזיכרון ובזמן מעבד הודות לחישוב ערכים נדרשים בלבד.
    *  **סדרות אינסופיות:** מאפשר לייצג סדרות נתונים אינסופיות, מכיוון שהערכים מחושבים לפי דרישה.
    *   **שימוש בגנרטורים ואיטרטורים:** Python מאפשר לממש חישובים עצלים באמצעות גנרטורים ואיטרטורים.

*   **גנרטורים וחישובים עצלים:**
    *   גנרטורים הם פונקציות מיוחדות המשתמשות ב-`yield` להחזרת ערכים אחד בכל פעם.
    *   גנרטורים מחשבים את הערך הבא רק כאשר נקראת הפונקציה next או מתרחשת איטרציה בלולאת for, במקום לחשב את כל הערכים באופן מיידי.

**דוגמה (מתוך שאלת המקור):**

```python
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib_gen = fibonacci()
for i in range(10):
    print(next(fib_gen))
```

**תיאור הדוגמה:**

*   **`fibonacci()`:** פונקציית גנרטור שמייצרת סדרת מספרים אינסופית של פיבונאצ'י.
    *  `while True`: לולאה אינסופית
    *   `yield a`: מחזירה את מספר פיבונאצ'י הבא ומשהה את ביצוע הפונקציה.
*   `fib_gen = fibonacci()`:  יוצר אובייקט גנרטור.
*    `for i in range(10)`: לולאה שתרוץ 10 פעמים.
*   `next(fib_gen)`: קורא לגנרטור, ומאלץ אותו לחשב ולהחזיר את ערך פיבונאצ'י הבא.
*  הגנרטור מייצר ערכים לפי דרישה ורק כאשר הם נדרשים ללולאה.

**דוגמאות נוספות:**
```python
# חישוב עצל באמצעות range
numbers = range(10000000000) # יוצר אובייקט range, אך המספרים אינם מחושבים
print(numbers[0]) # החישובים אינם מתרחשים עד לרגע זה
print(numbers[1]) # range מחשב עצלה את האיברים לפי דרישה.

# חישובים עצלים באמצעות map
numbers = [1,2,3,4]
squared_numbers = map(lambda x: x**2, numbers) # map אינו מחשב את הערכים, אלא רק מחזיר איטרטור
print(list(squared_numbers)) # ממירים לרשימה כדי לחשב את כל ריבועי המספרים.
```
