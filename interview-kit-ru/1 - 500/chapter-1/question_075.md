**שאלה 75.** מהי הערכה עצלה (lazy evaluation) בפייתון, וכיצד היא פועלת? הצג דוגמה המדגימה שימוש בחישובים עצלים.

-   א. הערכה עצלה היא שיטת הערכה שבה ביטויים מחושבים מיד לאחר הגדרתם.
-   ב. הערכה עצלה היא אסטרטגיית חישוב שבה ביטוי מחושב רק כאשר הערך שלו נחוץ. בפייתון ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.
-   ג. הערכה עצלה היא כאשר נעשה שימוש רק במשתנים אליהם יש התייחסות.
-   ד. הערכה עצלה היא שיטה להאצת חישובים באמצעות ביצועם במקביל.

**תשובה נכונה: ב**

**הסבר:**

הערכה עצלה (lazy evaluation) היא אסטרטגיית חישוב שבה ביטוי אינו מחושב מיידית, אלא נדחה עד לרגע שבו ערכו אכן נחוץ. אסטרטגיה זו מאפשרת לחסוך במשאבי חישוב, מכיוון שהחישובים אינם מבוצעים עד אשר התוצאה שלהם נחוצה בפועל.

*   **מושגי יסוד של הערכה עצלה:**
    *   **חישובים נדחים:** ערכי ביטויים אינם מחושבים מיד, אלא רק כאשר הם נחוצים.
    *   **חיסכון במשאבים:** מאפשר לחסוך בזיכרון ובזמן מעבד הודות לכך שמחושבים רק הערכים הדרושים.
    *   **סדרות אינסופיות:** מאפשר לייצג סדרות נתונים אינסופיות, מכיוון שערכים מחושבים לפי דרישה.
    *   **שימוש בגנרטורים ואיטרטורים:** פייתון מאפשר לממש חישובים עצלים באמצעות גנרטורים ואיטרטורים.

*   **גנרטורים וחישובים עצלים:**
    *   גנרטורים – אלה פונקציות מיוחדות המשתמשות ב-`yield` כדי להחזיר ערכים בזה אחר זה.
    *   גנרטורים מחשבים את הערך הבא רק כאשר נקראת הפונקציה `next` או מתבצעת איטרציה בלולאת `for`, במקום לחשב את כל הערכים בבת אחת.

**דוגמה (מתוך טקסט השאלה):**

```python
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib_gen = fibonacci()
for i in range(10):
    print(next(fib_gen))
```

**תיאור הדוגמה:**

*   `fibonacci()`: פונקציית גנרטור שמייצרת סדרה אינסופית של מספרי פיבונאצ'י.
    *   `while True`: לולאה אינסופית
    *   `yield a`: מחזירה את מספר פיבונאצ'י הבא ומשהה את ביצוע הפונקציה.
*   `fib_gen = fibonacci()`: יוצר אובייקט גנרטור.
*   `for i in range(10)`: לולאה שתתבצע 10 פעמים.
*   `next(fib_gen)`: קורא לגנרטור, מה שגורם לו לחשב ולהחזיר את ערך פיבונאצ'י הבא.
*   הגנרטור מייצר ערכים לפי דרישה ורק כאשר הם נחוצים ללולאה.

**דוגמאות נוספות:**
```python
# חישוב עצל באמצעות range
numbers = range(10000000000) # יוצרים אובייקט range, אך המספרים אינם מחושבים
print(numbers[0]) # החישובים אינם מתרחשים עד לרגע זה
print(numbers[1]) # range מחשב באופן עצל את האלמנטים לפי דרישה.

# חישובים עצלים באמצעות map
numbers = [1,2,3,4]
squared_numbers = map(lambda x: x**2, numbers) # map אינו מחשב את הערכים, אלא רק מחזיר איטרטור
print(list(squared_numbers)) # ממירים לרשימה כדי לחשב את כל ריבועי המספרים.
```

**ניתוח האפשרויות:**
*   א. הערכה עצלה היא שיטת הערכה שבה ביטויים מחושבים מיד לאחר הגדרתם.: לא נכון. זוהי הפעולה ההפוכה מהערכה עצלה.
*   ב. הערכה עצלה היא אסטרטגיית חישוב שבה ביטוי מחושב רק כאשר הערך שלו נחוץ. בפייתון ניתן לממש זאת באמצעות גנרטורים ואיטרטורים.: נכון.
*   ג. הערכה עצלה היא כאשר נעשה שימוש רק במשתנים אליהם יש התייחסות.: לא נכון. זה קשור לא לחישובים עצלים, אלא לתחום ההגדרה של משתנים.
*   ד. הערכה עצלה היא שיטה להאצת חישובים באמצעות ביצועם במקביל.: לא נכון.
