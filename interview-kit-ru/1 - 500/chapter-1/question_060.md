**שאלה 60.** מהי המטרה העיקרית של האופרטור `assert` בפייתון?

*   א. לקבוע את המצב ההתחלתי של משתנים בתחילת התוכנית.
*   ב. להפסיק את ביצוע התוכנית אם לא מתקיים תנאי מסוים.
*   ג. להבטיח שתנאי בקוד נשאר אמיתי, על ידי זריקת `AssertionError` אם התנאי מתגלה כשקרי.
*   ד. להצפין נתונים רגישים ביישום כדי למנוע דליפת נתונים.

**תשובה נכונה: C**

**הסבר:**

האופרטור `assert` בפייתון הוא כלי לניפוי שגיאות (debugging) ולבדיקות (testing), המאפשר לבדוק את אמיתות של תנאי מסוים.

*   **אפשרות א** אינה נכונה: `assert` אינו משמש לקביעת המצב ההתחלתי של משתנים.
*   **אפשרות ב** אינה נכונה: `assert` אינו *רק* מפסיק את הביצוע; הוא עושה זאת אם התנאי שקרי ופולט `AssertionError`.
*   **אפשרות ג** נכונה: `assert` מבטיח שתנאי שאמור להתקיים בשלב מסוים נשאר אמיתי, וגורם לשגיאה אם לא.
*   **אפשרות ד** אינה נכונה: `assert` אינו קשור להצפנת נתונים.

**כיצד פועל האופרטור `assert`:**

1.  `assert` מקבל כקלט תנאי (ביטוי לוגי), הנבדק לאמיתות.
2.  אם התנאי אמיתי (`True`), התוכנית ממשיכה להתבצע כרגיל.
3.  אם התנאי שקרי (`False`), פייתון מייצר חריגה מסוג `AssertionError` , וביצוע התוכנית נעצר.
4.  ניתן להעביר הודעה אופציונלית, שתודפס יחד עם החריגה.

**מטרת השימוש ב-`assert`:**

*   **ניפוי שגיאות:** משמש כאמצעי לניפוי שגיאות, המאפשר לזהות במהירות שגיאות בלוגיקה של התוכנית.
*   **בדיקות:** מסייע להבטיח שהקוד פועל כהלכה, באמצעות הוספת בדיקות בנקודות מפתח.
*   **גילוי שגיאות בשלבים מוקדמים:** מאפשר לגלות בעיות הקשורות למצב נתונים שגוי או ללוגיקת פעולה, בשלבים מוקדמים של הפיתוח.

**דוגמה:**

```python
def calculate_discounted_price(price: float, discount: float) -> float:
    """Вычисляет цену со скидкой, проверяя корректность скидки.""" # Original Russian comment
    """מחשב את המחיר לאחר הנחה, בודק את תקינות ההנחה.""" # Translated Hebrew comment
    assert 0 <= discount <= 1, "Скидка должна быть между 0 и 1" # Проверка на валидность скидки # Original Russian comment
    assert 0 <= discount <= 1, "ההנחה צריכה להיות בין 0 ל-1" # בדיקה של תקינות ההנחה # Translated Hebrew comment
    return price * (1 - discount)
    
price: float = calculate_discounted_price(100, 0.2)
print(f"Цена со скидкой: {price}") # Вывод: Цена со скидкой: 80.0 # Original Russian comment
print(f"מחיר לאחר הנחה: {price}") # פלט: מחיר לאחר הנחה: 80.0 # Translated Hebrew comment

try:
   price: float = calculate_discounted_price(100, 1.2) # Вызовет AssertionError # Original Russian comment
   # This will cause an AssertionError # English comment
except AssertionError as e:
    print(e) # Вывод: Скидка должна быть между 0 и 1 # Original Russian comment
    print(e) # פלט: ההנחה צריכה להיות בין 0 ל-1 # Translated Hebrew comment
```

**כתוצאה מכך:**
* בקריאה הראשונה, כאשר ההנחה נמצאת בטווח המותר, התוכנית ממשיכה להתבצע באופן רגיל.
* בקריאה השנייה, `discount` גדול מ-1, והתנאי `0 <= discount <= 1` אינו מתקיים, מה שמוביל לזריקת `AssertionError`.

לפיכך, **אפשרות ג** היא הנכונה.
