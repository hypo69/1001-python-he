### `question_79.md`

**שאלה 79.** מה עושה האופרטור `is` בפייתון?

- א.  בודק האם שני המשתנים מצביעים על אותו אובייקט.
- ב.  משווה את הערכים של שני משתנים.
- ג.  מוודא שטיפוסי המשתנים זהים.
- ד.  ממיר את טיפוס האובייקט למספר שלם.

**תשובה נכונה: א**

**הסבר:**

בפייתון, האופרטור `is` משמש לבדיקת זהות אובייקטים (object identity), ולא לשוויון בערכם (equality). משמעות הדבר היא ש-`is` בודק האם שני משתנים מצביעים על אותו אובייקט בדיוק בזיכרון.

*   **אפשרות א** נכונה: האופרטור `is` בודק זהות אובייקטים, לא את ערכיהם.
*   **אפשרות ב** אינה נכונה: להשוואת ערכים משתמשים באופרטור `==`.
*   **אפשרות ג** אינה נכונה: `is` אינו בודק טיפוסים, אלא רק את זהות האובייקטים.
*   **אפשרות ד** אינה נכונה: `is` אינו עוסק בהמרת טיפוסים.

**כיצד פועל האופרטור `is`:**

1.  הוא משווה שני אובייקטים (משתנים) על פי ה*זהות* שלהם (כלומר, על פי המיקום שלהם בזיכרון), ולא על פי הערכים שלהם.
2.  מחזיר `True`, אם שני האופרנדים מפנים לאותו אובייקט בזיכרון.
3.  מחזיר `False` אחרת.

**הבדלים חשובים מ-`==`:**

*   `==`: בודק שוויון בערכים של אובייקטים. מחזיר `True` אם ערכי האובייקטים שווים.
*   `is`: בודק זהות אובייקטים. מחזיר `True` אם שני המשתנים מצביעים על אותו אובייקט בזיכרון.

**דוגמה:**

```python
a: list[int] = [1, 2, 3]
b: list[int] = [1, 2, 3]
c: list[int] = a

print(f"a == b: {a == b}")   # Output: a == b: True (values are equal)
print(f"a is b: {a is b}")   # Output: a is b: False (different objects)

print(f"a == c: {a == c}")  # Output: a == c: True (values are equal)
print(f"a is c: {a is c}")  # Output: a is c: True (same object)


num1: int = 1000
num2: int = 1000
print(f"num1 == num2: {num1 == num2}") # Output: num1 == num2: True
print(f"num1 is num2: {num1 is num2}") # Output: num1 is num2: False

num3: int = 100
num4: int = 100
print(f"num3 == num4: {num3 == num4}") # Output: num3 == num4: True
print(f"num3 is num4: {num3 is num4}") # Output: num3 is num4: True
```
**בתוצאה:**

*   ל-`a` ול-`b` יש ערכים זהים, אך הם אובייקטים שונים בזיכרון, ולכן `==` מחזיר `True` ו-`is` מחזיר `False`.
*   `a` ו-`c` מצביעים על אותו אובייקט, ולכן גם `==` וגם `is` מחזירים `True`.
*   פייתון, לשם אופטימיזציה של עבודה עם מספרים קטנים, יוצר אובייקט יחיד עבורם בזיכרון. לכן בדוגמה עם `num3` ו-`num4`, שני המשתנים יפנו לאותו אובייקט.

לפיכך, **אפשרות א** היא התשובה הנכונה.
