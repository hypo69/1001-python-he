### `question_79.md`

**שאלה 79.** מה מבצע האופרטור `is` ב-Python?

- א. בודק אם שני המשתנים מצביעים על אותו אובייקט.
- ב. משווה את הערכים של שני משתנים.
- ג. מוודא שסוגי המשתנים זהים.
- ד. ממיר את סוג האובייקט למספר שלם.

**תשובה נכונה: א**

**הסבר:**

ב-Python, האופרטור `is` משמש לבדיקת *זהות* אובייקטים, ולא לשוויון שלהם לפי ערך. משמעות הדבר היא ש-`is` בודק האם שני משתנים מצביעים על אותו אובייקט בזיכרון.

*   **אפשרות א'** נכונה: האופרטור `is` בודק את זהות האובייקטים, ולא את הערכים שלהם.
*   **אפשרות ב'** אינה נכונה: להשוואת ערכים משתמשים באופרטור `==`.
*   **אפשרות ג'** אינה נכונה: `is` אינו בודק סוגים, אלא רק את זהות האובייקטים.
*   **אפשרות ד'** אינה נכונה: `is` אינו עוסק בהמרת סוגים.

**כיצד פועל האופרטור `is`:**

1.  הוא משווה שני אובייקטים (משתנים) לפי *זהותם* (כלומר, לפי מיקומם בזיכרון), ולא לפי ערכיהם.
2.  מחזיר `True` אם שני האופרנדים מפנים לאותו אובייקט בזיכרון.
3.  מחזיר `False` במקרה אחר.

**הבדלים חשובים מ-`==`:**

*   `==`: בודק שוויון ערכי האובייקטים. מחזיר `True` אם ערכי האובייקטים שווים.
*   `is`: בודק את זהות האובייקטים. מחזיר `True` אם שני המשתנים מצביעים על אותו אובייקט בזיכרון.

**דוגמה:**

```python
a: list[int] = [1, 2, 3]
b: list[int] = [1, 2, 3]
c: list[int] = a

print(f"a == b: {a == b}")   # פלט: a == b: True (הערכים שווים)
print(f"a is b: {a is b}")   # פלט: a is b: False (אובייקטים שונים)

print(f"a == c: {a == c}")  # פלט: a == c: True (הערכים שווים)
print(f"a is c: {a is c}")  # פלט: a is c: True (אותו אובייקט)


num1: int = 1000
num2: int = 1000
print(f"num1 == num2: {num1 == num2}") # פלט: num1 == num2: True
print(f"num1 is num2: {num1 is num2}") # פלט: num1 is num2: False

num3: int = 100
num4: int = 100
print(f"num3 == num4: {num3 == num4}") # פלט: num3 == num4: True
print(f"num3 is num4: {num3 is num4}") # פלט: num3 is num4: True
```
**בתוצאה:**

*   ל-`a` ול-`b` יש ערכים זהים, אך הם אובייקטים שונים בזיכרון, לכן `==` מחזיר `True` ואילו `is` מחזיר `False`.
*   `a` ו-`c` מצביעים על אותו אובייקט, לכן גם `==` וגם `is` מחזירים `True`.
*   Python, מטעמי אופטימיזציה בעבודה עם מספרים קטנים, יוצר אובייקט אחד עבורם בזיכרון. לכן בדוגמה עם `num3` ו-`num4`, שני המשתנים יפנו לאותו אובייקט.

לפיכך, **אפשרות א'** היא התשובה הנכונה.
