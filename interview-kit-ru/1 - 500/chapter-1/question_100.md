### `question_100.md`

**שאלה 100.** מהי התוצאה של הפעולה הבאה המשתמשת במתודת `update()` של מילון Python?

```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}
dict1.update(dict2)
```

מה יכיל `dict1` לאחר העדכון?

- A. `{'a': 1, 'b': 2, 'c': 4}`
- B. `{'a': 1, 'b': 3, 'c': 4}`
- C. `{'b': 3, 'c': 4}`
- D. `{'a': 1, 'b': 2}`

**תשובה נכונה: B**

**הסבר:**

המתודה `update()` במילוני Python משמשת למיזוג שני מילונים.

*   **אפשרות A** אינה נכונה: היא מייצגת באופן שגוי את הערך של המפתח `"b"`; הוא צריך להיות `3`.

*   **אפשרות B** נכונה: זו הפלט הנכון. המתודה `update()` תוסיף מפתחות חסרים ותעדכן מפתחות קיימים, תוך שימוש במפתחות וערכים מהמילון המועבר כארגומנט ל-`update()`.

*   **אפשרות C** אינה נכונה: היא אינה לוקחת בחשבון את המפתח `"a"` מ-`dict1`.

*   **אפשרות D** אינה נכונה: אפשרות זו מציגה את המצב ההתחלתי של `dict1`, לפני העדכון.

**כיצד עובדת `update()`:**

1.  `update()` נקראת על המילון `dict1`.
2.  היא מקבלת מילון אחר, `dict2`, כארגומנט.
3.  היא מוסיפה את זוגות המפתח-ערך מ-`dict2` לתוך `dict1`.
4.  אם ב-`dict1` כבר קיים מפתח שנמצא גם ב-`dict2`, ערך המפתח ב-`dict1` יעודכן בערך מ-`dict2`.
5.  המתודה משנה את המילון שעליו היא הופעלה, כלומר את `dict1` במקרה זה.

**דוגמה:**

```python
dict1: dict[str, int] = {'a': 1, 'b': 2}
dict2: dict[str, int] = {'b': 3, 'c': 4}
print(f"מצב dict1 התחלתי: {dict1}") # פלט: מצב dict1 התחלתי: {'a': 1, 'b': 2}

dict1.update(dict2)
print(f"dict1 לאחר עדכון: {dict1}") # פלט: dict1 לאחר עדכון: {'a': 1, 'b': 3, 'c': 4}
```

**כתוצאה מכך:**

*   במקור, ל-`dict1` יש מפתחות `'a'` ו-`'b'` עם ערכים `1` ו-`2` בהתאמה.
*   לאחר קריאה ל-`dict1.update(dict2)`:
    *   המפתח `'a'` נשאר ללא שינוי.
    *   המפתח `'b'` ב-`dict1` מתעדכן בערך מ-`dict2` (מ-2 ל-3).
    *   ל-`dict1` מתווסף זוג מפתח-ערך חדש `'c': 4` מהמילון `dict2`.
*   המצב הסופי של `dict1` הוא `{'a': 1, 'b': 3, 'c': 4}`.

לפיכך, **אפשרות B** היא הנכונה.
