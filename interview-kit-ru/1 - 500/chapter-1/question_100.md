### `question_100.md`

**שאלה 100.** מהי התוצאה של הפעולה הבאה המשתמשת במתודת `update()` במילון Python?

```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}
dict1.update(dict2)
```

מה יכיל `dict1` לאחר העדכון?

- A. `{'a': 1, 'b': 2, 'c': 4}`
- B. `{'a': 1, 'b': 3, 'c': 4}`
- C. `{'b': 3, 'c': 4}`
- D. `{'a': 1, 'b': 2}`

**תשובה נכונה: B**

**הסבר:**

המתודה `update()` במילוני Python משמשת למיזוג שני מילונים.

*   **אפשרות A** אינה נכונה: היא מציגה באופן שגוי את ערך המפתח `"b"`; ערכו צריך להיות `3`.

*   **אפשרות B** נכונה: זוהי הפלט הנכון. המתודה `update()` תוסיף מפתחות חסרים ותעדכן מפתחות קיימים, תוך שימוש במפתחות ובערכים מהמילון המועבר למתודה `update()`.

*   **אפשרות C** אינה נכונה: היא אינה לוקחת בחשבון את המפתח `"a"` מ-`dict1`.

*   **אפשרות D** אינה נכונה: אפשרות זו מציגה את המצב ההתחלתי של `dict1`, לפני העדכון.

**כיצד פועלת `update()`:**

1.  המתודה `update()` נקראת על המילון `dict1`.
2.  היא מקבלת מילון נוסף `dict2` כארגומנט.
3.  היא מוסיפה את זוגות המפתח-ערך מ-`dict2` ל-`dict1`.
4.  אם ב-`dict1` כבר קיים מפתח שקיים גם ב-`dict2`, ערך המפתח ב-`dict1` יעודכן לערך מ-`dict2`.
5.  המתודה משנה את המילון שעליו היא הופעלה, כלומר את `dict1` במקרה זה.

**דוגמה:**

```python
dict1: dict[str, int] = {'a': 1, 'b': 2}
dict2: dict[str, int] = {'b': 3, 'c': 4}
print(f"מצב התחלתי של dict1: {dict1}") # פלט: {'a': 1, 'b': 2}

dict1.update(dict2)
print(f"dict1 לאחר העדכון: {dict1}") # פלט: {'a': 1, 'b': 3, 'c': 4}
```

**כתוצאה מכך:**

*   בתחילה, `dict1` מכיל את המפתחות `'a'` ו-`'b'` עם הערכים `1` ו-`2` בהתאמה.
*   לאחר קריאה ל-`dict1.update(dict2)`:
    *   המפתח `'a'` נשאר ללא שינוי.
    *   המפתח `'b'` ב-`dict1` מעודכן לערך מ-`dict2` (מ-2 ל-3).
    *   זוג מפתח-ערך חדש `'c': 4` מתווסף ל-`dict1` מהמילון `dict2`.
*   המצב הסופי של `dict1` שווה ל-`{'a': 1, 'b': 3, 'c': 4}`.

לפיכך, **אפשרות B** היא הנכונה.
