### `question_73.md`

**שאלה 73.** כיצד ניתן לקבל את מספר ההופעות של ערך ברשימת Python?

- א. `count()`
- ב. `length()`
- ג. `size()`
- ד. `number()`

**תשובה נכונה: א**

**הסבר:**

כדי לספור את מספר ההופעות של ערך מסוים ברשימת Python, משתמשים בשיטה `count()`.

*   **אפשרות א** נכונה: השיטה `count()` מחזירה את מספר ההופעות של הערך המצוין ברשימה.
*   **אפשרות ב** אינה נכונה: `length()` או `len()` משמשים לקבלת האורך הכולל (מספר האלמנטים) של הרשימה, ולא את מספר ההופעות של ערך ספציפי.
*   **אפשרות ג** אינה נכונה: `size()` אינה שיטה המיועדת לרשימות ב-Python.
*   **אפשרות ד** אינה נכונה: `number()` אינה שיטה המיועדת לרשימות ב-Python.

**אופן פעולת `count()`:**

1.  השיטה `count()` נקראת על הרשימה.
2.  היא מקבלת ארגומנט אחד - הערך שמספר ההופעות שלו צריך להיספר.
3.  היא מחזירה מספר שלם, המייצג את מספר הפעמים שהערך המצוין מופיע ברשימה.

**דוגמה:**

```python
my_list: list[int] = [1, 2, 2, 3, 2, 4, 5]

# סופרים את מספר ההופעות של הערך 2
count_of_2: int = my_list.count(2)
print(f"מספר ההופעות של 2 ברשימה: {count_of_2}") # פלט: מספר ההופעות של 2 ברשימה: 3

# סופרים את מספר ההופעות של הערך 6
count_of_6: int = my_list.count(6)
print(f"מספר ההופעות של 6 ברשימה: {count_of_6}") # פלט: מספר ההופעות של 6 ברשימה: 0
```

**כתוצאה מכך:**

*   `my_list.count(2)` מחזיר 3, מכיוון שהמספר 2 מופיע שלוש פעמים ברשימה.
*   `my_list.count(6)` מחזיר 0, מכיוון שהמספר 6 אינו מופיע ברשימה.

לפיכך, **אפשרות א** היא הנכונה.
