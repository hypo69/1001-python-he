### `question_050.md`

**שאלה 50.** מהן ההשלכות של שימוש באופרטור `import` בסקריפטים של Python, במיוחד בהקשר של ניהול תלויות ותכנות מודולרי?

- A. אופרטור `import` מגדיל את זמן הריצה של סקריפטים, על ידי טעינת כל מודולי הספרייה הזמינים בתחילת הריצה, ללא קשר לשאלה האם נעשה בהם שימוש בסקריפט.
- B. הוא מאפשר למתכנתים לגשת לקוד ממודולים או מספריות אחרות בתוכניות שלהם, ובכך מקדם שימוש חוזר בקוד ותכנות מודולרי.
- C. באמצעות שימוש ב-`import`, Python מקמפל כל מודול מיובא לקוד בית נפרד כדי למנוע קומפילציה מחדש בריצות עתידיות, תוך הפחתת הגמישות.
- D. האופרטור משמש להצפנה והגנה על קוד מודולים כדי למנוע גישה בלתי מורשית ושינוי בבסיס הקוד.

**תשובה נכונה: B**

**הסבר:**

האופרטור `import` ב-Python הוא בסיסי לארגון קוד למודולים ולשימוש חוזר בקוד מקבצים או ספריות אחרות.

*   **אפשרות A** אינה נכונה: האופרטור `import` אינו טוען את כל המודולים הזמינים, אלא רק את אלה שמצוינים במפורש בבקשת ה-`import`.

*   **אפשרות B** נכונה: זהו בדיוק האופרטור `import` שהופך קוד ממודולים אחרים לזמין, ומאפשר שימוש חוזר בקוד וארגון התוכנית בצורת מודולים.

*   **אפשרות C** אינה נכונה: Python אכן מקמפל מודול מיובא לקוד בית (קובץ `.pyc`), אך הדבר נעשה כדי להאיץ ייבואים עתידיים, לא כדי להפחית גמישות.

*   **אפשרות D** אינה נכונה: האופרטור `import` אינו מצפין קוד.

**הסבר מפורט:**

*   **מודולים:** ב-Python, כל קובץ עם סיומת `.py` הוא מודול. הוא יכול להכיל פונקציות, מחלקות ומשתנים שניתן להשתמש בהם בתוכניות אחרות.
*   **`import`**: מילת המפתח `import` טוענת מודול למרחב השמות הנוכחי, ומאפשרת להשתמש בקוד שלו. ניתן לייבא מודול בשלמותו (לדוגמה, `import math`) או לייבא אלמנטים ספציפיים מתוך מודול (לדוגמה, `from math import sqrt`).
*   **מטרת הייבוא:** מאפשר למפתחים לכלול פונקציונליות ממודול אחד במודול אחר.
*   **שימוש חוזר בקוד:** מתכנתים יכולים לעשות שימוש חוזר בקוד בחלקים שונים של התוכנית או בתוכניות שונות.
*   **מודולריות:** התוכנית יכולה להיות מחולקת לבלוקים לוגיים (מודולים), שכל אחד מהם מבצע פונקציה מוגדרת משלו.

**דוגמה:**

```python
# my_module.py
def my_function():
    print("Hello from my_module")

# main.py
import my_module  # ייבוא המודול my_module

my_module.my_function() # קריאה לפונקציה מהמודול המיובא

from math import sqrt # ייבוא פונקציה מהמודול math
print(f"Корень из 16: {sqrt(16)}") # שימוש בפונקציה sqrt מהמודול math
```

**בתוצאה:**

*   הסקריפט `main.py` מייבא בהצלחה את המודול `my_module` ומשתמש בפונקציה `my_function()`.
*   הסקריפט `main.py` מייבא את הפונקציה `sqrt` מהמודול `math` ומשתמש בה.

לפיכך, **אפשרות B** היא הנכונה, ומתארת במדויק את הייעוד והשימוש באופרטור `import`.
