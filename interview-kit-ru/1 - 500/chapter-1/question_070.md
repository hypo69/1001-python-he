**שאלה 070.** מהי קומפוזיציית פונקציות, וכיצד ניתן ליישם אותה בפייתון? הבא/י דוגמת קוד המדגימה קומפוזיציית פונקציות, כאשר התוצאה של פונקציה אחת מועברת כארגומנט לפונקציה אחרת.

-   א. קומפוזיציית פונקציות היא תהליך של יצירת אובייקטים ממחלקות, המשתמשים בערכים המתקבלים מאובייקטים אחרים.
-   ב. קומפוזיציית פונקציות היא תהליך שבו פונקציה קוראת לעצמה מספר פעמים.
-   ג. קומפוזיציית פונקציות היא צירוף של שתי פונקציות או יותר, שבו תוצאת הביצוע של פונקציה אחת מועברת כארגומנט לפונקציה אחרת.
-   ד. קומפוזיציית פונקציות היא דרך ליצור פונקציות שיכולות לרוץ בתהליכונים נפרדים.

**תשובה נכונה: ג**

**הסבר:**

קומפוזיציית פונקציות (function composition) היא תבנית פונקציונלית המאפשרת לחבר שתיים או יותר פונקציות באופן כזה שתוצאת הביצוע של פונקציה אחת מועברת כארגומנט לפונקציה אחרת. הדבר מאפשר ליצור פונקציות חדשות, מורכבות יותר, מפונקציות פשוטות וניתנות לשימוש חוזר.

*   **מושגי יסוד בקומפוזיציית פונקציות:**
    *   **שרשרת חישובים:** תוצאה של פונקציה אחת הופכת לערך הקלט עבור הפונקציה הבאה.
    *   **יישום סדרתי:** הפונקציות מיושמות בזו אחר זו בסדר מוגדר.
    *   **פונקציה חדשה מפשוטות:** קומפוזיציה מאפשרת ליצור פונקציות מורכבות מפונקציות פשוטות יותר.
    *   **אפשרות לשימוש חוזר:** קומפוזיציה מאפשרת לעשות שימוש חוזר בפונקציות שנוצרו בעבר.
*   **יישום קומפוזיציה בפייתון:**
    *   בפייתון, קומפוזיציה יכולה להיות מיושמת ידנית באמצעות קריאות מקוננות של פונקציות, או באמצעות האופרטור `|` (החל מפייתון 3.9), או באמצעות שימוש בפונקציות למדא.

**דוגמה (מטקסט השאלה):**

```python
def double(x):
    return 2 * x

def square(x):
    return x ** 2

def compose(f, g):
    return lambda x: f(g(x))

composed = compose(square, double)
print(composed(5))  # 100
```

**תיאור הדוגמה:**

1.  **`double(x)`:** פונקציה שמכפילה את המספר `x` ב-2.
2.  **`square(x)`:** פונקציה שמחזירה את המספר `x` בריבוע.
3.  **`compose(f, g)`**: פונקציה המיישמת קומפוזיציה.
    *   מקבלת את הפונקציות `f` ו-`g` כארגומנטים.
    *   מחזירה פונקציית למדא `lambda x: f(g(x))`, אשר מיישמת תחילה את פונקציה `g` על הארגומנט `x`, ולאחר מכן מיישמת את פונקציה `f` על התוצאה של `g(x)`.
4.  **`composed = compose(square, double)`:** נוצרת פונקציה חדשה `composed`, שהיא קומפוזיציה של `square` ו-`double`. היא שקולה ל-`lambda x : square(double(x))`.
5.  `composed(5)`: מקבלת את הארגומנט `5` ומבצעת את הפונקציות בסדר `(5*2)**2`, כלומר `double(5)` (ששווה ל-`10`) ולאחר מכן `square(10)` (ששווה ל-`100`).

**דוגמאות נוספות:**
```python

# דוגמה לקומפוזיציה באמצעות קריאות מקוננות של פונקציות
def add1(x):
   return x + 1
def multiply_by_3(x):
   return x*3

def composed_func_manual(x):
    return add1(multiply_by_3(x))

print(composed_func_manual(5)) # הדפסה: 16

# דוגמה לקומפוזיציה באמצעות שימוש ב-reduce ופונקציית למדא
from functools import reduce

def compose_with_reduce(*funcs):
    return lambda x: reduce(lambda acc, f: f(acc), funcs, x)


composed2 = compose_with_reduce(add1, multiply_by_3)
print(composed2(5)) # הדפסה: 16
```
**ניתוח האפשרויות:**
*   **א. קומפוזיציית פונקציות היא תהליך של יצירת אובייקטים ממחלקות, המשתמשים בערכים המתקבלים מאובייקטים אחרים.:** לא נכון. קומפוזיציית פונקציות אינה קשורה ליצירת אובייקטים ממחלקות.
*   **ב. קומפוזיציית פונקציות היא תהליך שבו פונקציה קוראת לעצמה מספר פעמים.:** לא נכון. זהו תיאור של רקורסיה.
*   **ג. קומפוזיציית פונקציות היא צירוף של שתי פונקציות או יותר, שבו תוצאת הביצוע של פונקציה אחת מועברת כארגומנט לפונקציה אחרת.:** נכון.
*   **ד. קומפוזיציית פונקציות היא דרך ליצור פונקציות שיכולות לרוץ בתהליכונים נפרדים.:** לא נכון.

**לסיכום:**
*   קומפוזיציית פונקציות מאפשרת ליצור פעולות מורכבות יותר מפשוטות.
*   שימוש בקומפוזיציה תורם למודולריות ושימוש חוזר בקוד.

לפיכך, התשובה הנכונה היא **ג. קומפוזיציית פונקציות היא צירוף של שתי פונקציות או יותר, שבו תוצאת הביצוע של פונקציה אחת מועברת כארגומנט לפונקציה אחרת.**
