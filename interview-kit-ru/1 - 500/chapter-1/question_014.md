**שאלה 14.** פונקציות Python מוגדרות באמצעות מילת המפתח `def`, ואחריה שם הפונקציה וסוגריים עגולים. איזו מההגדרות הבאות כוללת פרמטר ברירת מחדל, המאפשר לקרוא לפונקציה עם פחות ארגומנטים מאשר מספר הפרמטרים שהוגדרו?

```python
- A. def print_value(x="Hello"): print(x)
- B. def print_value(x, y): print(x)
- C. def print_value(x, y="Hello", z): print(x + y + z)
- D. def print_value(x): y = "Hello"; print(x + y)
```

**תשובה נכונה: A**

**הסבר:**

ב-Python ניתן להגדיר ערכים לפרמטרים כערכי ברירת מחדל בעת הגדרת הפונקציה. הדבר מאפשר לקרוא לפונקציה מבלי להעביר ערכים לכל הפרמטרים. פרמטרים בעלי ערכי ברירת מחדל חייבים להיות מוגדרים לאחר הפרמטרים החובה.

*   **אפשרות A**: `def print_value(x="Hello"): print(x)` - זוהי הגדרה תקינה של פונקציה עם פרמטר ברירת מחדל. אם בקריאה לפונקציה `print_value()` לא יועבר ארגומנט, המשתנה `x` ישתמש בערך ברירת המחדל `"Hello"`.
*   **אפשרות B**: `def print_value(x, y): print(x)` - זוהי הגדרה של פונקציה הדורשת שני פרמטרים חובה.
*   **אפשרות C**: `def print_value(x, y="Hello", z): print(x + y + z)` - הגדרה זו של פונקציה מכילה שגיאת תחביר. פרמטרים עם ערכי ברירת מחדל (`y`) חייבים להופיע אחרי כל הפרמטרים ללא ערכי ברירת מחדל (`x`, `z`).
*   **אפשרות D**: `def print_value(x): y = "Hello"; print(x + y)` - זוהי הגדרה של פונקציה עם פרמטר אחד, אך בגוף הפונקציה נוצר משתנה `y` עם ערך מסוג מחרוזת.

**דוגמה:**

```python
# דוגמה עם האפשרות הנכונה (A)
def print_value(x: str="Hello") -> None:
    print(x)

print_value()    # פלט: Hello (נעשה שימוש בערך ברירת המחדל)
print_value("World")  # פלט: World (ערך ברירת המחדל נדרס)

# דוגמה לאפשרות שגויה (C) - גורמת לשגיאה
# def print_value(x, y="Hello", z): print(x + y + z) # SyntaxError: non-default argument follows default argument
```

**לסיכום:**

*   באפשרות A, ניתן לקרוא לפונקציה `print_value()` ללא ארגומנטים, ואז ייעשה שימוש בערך ברירת המחדל `"Hello"`. אם מועבר ארגומנט, ערך ברירת המחדל נדרס.
*   אפשרות C תגרום לשגיאה `SyntaxError`, מכיוון שפרמטרים עם ערך ברירת מחדל חייבים להיות ממוקמים בסוף.

לפיכך, **אפשרות A** היא ההגדרה הנכונה היחידה של פונקציה עם פרמטר ברירת מחדל.
