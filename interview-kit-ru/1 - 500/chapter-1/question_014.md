**שאלה 14.** פונקציות ב-Python מוגדרות באמצעות מילת המפתח `def`, ואחריה שם הפונקציה וסוגריים עגולים. איזו מההגדרות הבאות כוללת פרמטר עם ערך ברירת מחדל, המאפשר קריאה לפונקציה עם פחות ארגומנטים מאשר מספר הפרמטרים שהוגדרו?

```python
- A. def print_value(x="Hello"): print(x)
- B. def print_value(x, y): print(x)
- C. def print_value(x, y="Hello", z): print(x + y + z)
- D. def print_value(x): y = "Hello"; print(x + y)
```

**תשובה נכונה: A**

**הסבר:**

ב-Python, ניתן להגדיר ערכי ברירת מחדל לפרמטרים בעת הגדרת הפונקציה. הדבר מאפשר קריאה לפונקציה מבלי להעביר ערכים עבור כל הפרמטרים. פרמטרים עם ערכי ברירת מחדל חייבים להופיע לאחר פרמטרים חובה (פרמטרים ללא ערך ברירת מחדל).

*   **אפשרות A**: `def print_value(x="Hello"): print(x)` - זוהי הגדרה נכונה של פונקציה עם פרמטר בעל ערך ברירת מחדל. אם בעת קריאה לפונקציה `print_value()` לא יועבר ארגומנט, המשתנה `x` ישתמש בערך ברירת המחדל `"Hello"`.
*   **אפשרות B**: `def print_value(x, y): print(x)` - זוהי הגדרה של פונקציה הדורשת שני פרמטרים חובה.
*   **אפשרות C**: `def print_value(x, y="Hello", z): print(x + y + z)` - הגדרת פונקציה זו מכילה שגיאת תחביר. פרמטרים עם ערכי ברירת מחדל (`y`) חייבים להופיע לאחר כל הפרמטרים ללא ערכי ברירת מחדל (`x`, `z`).
*   **אפשרות D**: `def print_value(x): y = "Hello"; print(x + y)` - זוהי הגדרה של פונקציה עם פרמטר אחד, אך בגוף הפונקציה נוצר משתנה `y` עם ערך מחרוזתי.

**דוגמה:**

```python
# דוגמה לאפשרות הנכונה (A)
def print_value(x: str="Hello") -> None:
    print(x)

print_value()    # פלט: Hello (נעשה שימוש בערך ברירת המחדל)
print_value("World")  # פלט: World (ערך ברירת המחדל נדרס)

# דוגמה לאפשרות השגויה (C) - גורמת לשגיאה
# def print_value(x, y="Hello", z): print(x + y + z) # שגיאת תחביר: פרמטר ללא ערך ברירת מחדל מופיע לאחר פרמטר עם ערך ברירת מחדל
```

**לסיכום:**

*   באפשרות A, הפונקציה `print_value()` יכולה להיקרא ללא ארגומנטים, ואז ייעשה שימוש בערך ברירת המחדל `"Hello"`. אם כן יועבר ארגומנט, ערך ברירת המחדל יידרס.
*   אפשרות C תגרום לשגיאה `SyntaxError`, מכיוון שפרמטרים עם ערך ברירת מחדל חייבים להיות ממוקמים בסוף.

לפיכך, **אפשרות A** היא ההגדרה הנכונה היחידה של פונקציה עם פרמטר בעל ערך ברירת מחדל.
