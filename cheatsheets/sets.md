# קבוצות (Sets) ב-Python

**1. מבוא: מהן קבוצות?**

במדעי המחשב ובמתמטיקה, קבוצות הן דרך לייצג אוספים של איברים ייחודיים. חשוב להבין ש:

*   **ייחודיות:** כל איבר בקבוצה חייב להיות ייחודי. חזרות אינן מותרות.
*   **חוסר סדר:** סדר האיברים בקבוצה אינו משנה.

**2. קבוצות ופירות**

נדמיין שיש לנו רק את הפירות עצמם:

*   🍎 (תפוח)
*   🍐 (אגס)
*   🍉 (מלון)
*   🧺 (סלסלה)

חשוב ש:

1.  **באוסף *אין* פירות זהים:** אם באוסף יש תפוח, אז לא יהיה שם תפוח נוסף זהה. כל פרי ייחודי באוסף שלו.
2.  **סדר הפירות אינו חשוב:** אם באוסף יש תפוח ואגס, זה זהה לאילו היו שם אגס ותפוח. הסדר אינו משנה.

לדוגמה, `{🍎, 🍐, 🍉}` — זוהי קבוצה שבה יש תפוח, אגס ומלון.

**3. מדוע קבוצות דורשות ייחודיות של איברים?**

*   קבוצות מיועדות למעקב אחר *נוכחות* איברים, לא אחר *כמותם*.
*   ייחודיות האיברים מפשטת ביצוע פעולות על קבוצות.
*   קבוצות משמשות למניעת יתירות נתונים.

**4. פעולות על קבוצות (אוספי פירות):**

1.  **איחוד (Union): "אוספים את כל הפירות לאוסף אחד"**
    *   מאחד שני אוספי פירות, אוסף את כל הפירות לאוסף חדש. אם בשני האוספים יש פירות זהים, באוסף החדש יהיה רק פרי אחד כזה.
    *   אם באוסף A יש {🍎, 🍐}, ובאוסף B יש {🍐, 🍉}, אז באוסף A ∪ B יהיה {🍎, 🍐, 🍉}.

2.  **חיתוך (Intersection): "מוצאים פירות משותפים"**
    *   מחפשים *רק* את הפירות שישנם גם באוסף A וגם באוסף B.
    *   אם באוסף A יש {🍎, 🍐}, ובאוסף B יש {🍐, 🍉}, אז באוסף A ∩ B יהיה רק {🍐}.

3.  **הפרש (Difference): "פירות שיש באוסף אחד, אבל אין באחר"**
    *   מחפשים *רק* את הפירות שישנם באוסף A, אבל אין אותם באוסף B.
    *   אם באוסף A יש {🍎, 🍐}, ובאוסף B יש {🍐, 🍉}, אז באוסף A - B יהיה {🍎}, ובאוסף B - A יהיה {🍉}.

4.  **הפרש סימטרי (Symmetric Difference): "פירות שישנם רק באחד מהאוספים"**
    *   מחפשים *את כל* הפירות שישנם או באוסף A או באוסף B, אבל לא בשניהם בו-זמנית.
    *   אם באוסף A יש {🍎, 🍐}, ובאוסף B יש {🍐, 🍉}, אז באוסף A ^ B יהיה {🍎, 🍉}.

5.  **תת-קבוצה (Subset): "האם כל הפירות מאוסף אחד נמצאים באחר?"**
    *   בודקים האם אוסף A הוא תת-קבוצה של אוסף B. פירוש הדבר שכל הפירות מאוסף A נמצאים גם באוסף B.
    *   **דוגמה:** אם באוסף A יש {🍎, 🍐}, ובאוסף B יש {🍎, 🍐, 🍉}, אז A <= B.

6.  **על-קבוצה (Superset): "האם אוסף אחד מכיל את כל הפירות מאחר?"**
    *   בודקים האם אוסף A הוא על-קבוצה של אוסף B. פירוש הדבר שכל הפירות מאוסף B נמצאים גם באוסף A.
    *   אם באוסף A יש {🍎, 🍐, 🍉}, ובאוסף B יש {🍎, 🍐}, אז A >= B.

```python
from typing import Set

def create_fruit_set(fruit_string: str) -> Set[str]:
    """
    יוצר קבוצת פירות ממחרוזת.

    Args:
        fruit_string: מחרוזת עם פירות (🍎, 🍐, 🍉, 🧺).

    Returns:
        קבוצת פירות ייחודיים.
    """
    # בודקים שהמחרוזת מכילה רק את סמלי היוניקוד המותרים
    if not all(fruit in ["🍎", "🍐", "🍉", "🧺"] for fruit in fruit_string):
        raise ValueError("המחרוזת יכולה להכיל רק את הסמלים 🍎, 🍐, 🍉, 🧺")
    # משתמשים ב-set() ליצירת הקבוצה
    return set(fruit_string)

def display_set(fruit_set: Set[str]) -> str:
  """
  ממיר קבוצת פירות למחרוזת לצורך הצגה.

    Args:
        fruit_set: קבוצת פירות.

    Returns:
        מחרוזת להצגה.
  """
  return "{" + ", ".join(fruit_set) + "}"


# יוצרים קבוצות פירות
fruits_set_A = create_fruit_set("🍎🍐")  # קבוצה A: {🍎, 🍐}
fruits_set_B = create_fruit_set("🍐🍉")  # קבוצה B: {🍐, 🍉}
fruits_set_C = create_fruit_set("🍎🍐🍉")  # קבוצה C: {🍎, 🍐, 🍉}
fruits_set_D = create_fruit_set("🧺")  # קבוצה D: {🧺}

# מציגים את הקבוצות
print(f"קבוצה A: {display_set(fruits_set_A)}")
print(f"קבוצה B: {display_set(fruits_set_B)}")
print(f"קבוצה C: {display_set(fruits_set_C)}")
print(f"קבוצה D: {display_set(fruits_set_D)}")

# איחוד קבוצות (Union)
union_result = fruits_set_A | fruits_set_B
print(f"A ∪ B: {display_set(union_result)}")  # תוצאה: {🍎, 🍐, 🍉}

# חיתוך קבוצות (Intersection)
intersection_result = fruits_set_A & fruits_set_B
print(f"A ∩ B: {display_set(intersection_result)}")  # תוצאה: {🍐}

# הפרש קבוצות (Difference)
difference_result_AB = fruits_set_A - fruits_set_B
print(f"A - B: {display_set(difference_result_AB)}")  # תוצאה: {🍎}
difference_result_BA = fruits_set_B - fruits_set_A
print(f"B - A: {display_set(difference_result_BA)}")  # תוצאה: {🍉}

# הפרש סימטרי של קבוצות (Symmetric Difference)
symmetric_difference_result = fruits_set_A ^ fruits_set_B
print(f"A ^ B: {display_set(symmetric_difference_result)}")  # תוצאה: {🍎, 🍉}

# תת-קבוצה (Subset)
subset_result1 = fruits_set_A <= fruits_set_C
print(f"A <= C: {subset_result1}")  # תוצאה: True (A תת-קבוצה של C)
subset_result2 = fruits_set_A <= fruits_set_B
print(f"A <= B: {subset_result2}") # תוצאה: False (A אינה תת-קבוצה של B)

# על-קבוצה (Superset)
superset_result1 = fruits_set_C >= fruits_set_A
print(f"C >= A: {superset_result1}")  # תוצאה: True (C על-קבוצה של A)
superset_result2 = fruits_set_B >= fruits_set_A
print(f"B >= A: {superset_result2}")  # תוצאה: False (B אינה על-קבוצה של A)

# בדיקה נוכחות איבר
print(f"🍎 in A: {'🍎' in fruits_set_A}")  # תוצאה: True
print(f"🍉 in A: {'🍉' in fruits_set_A}")  # תוצאה: False
```

*   **`create_fruit_set(fruit_string)`:** פונקציה זו יוצרת קבוצה ממחרוזת עם פירות.
    *   `set(fruit_string)` ממיר את המחרוזת לקבוצה, מסיר כפילויות והופך את סדר האיברים ללא משמעותי.
    *   אנו בודקים שהמחרוזת מורכבת רק מסמלי יוניקוד מותרים.
*   **`display_set(fruit_set)`:** פונקציה זו משמשת להמרת קבוצה למחרוזת קריאה לצורך פלט.
*   **דוגמאות:** אנו יוצרים מספר קבוצות ומיישמים עליהן פעולות שונות. תוצאות כל פעולה מודפסות למסך.
