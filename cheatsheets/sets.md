# קבוצות (מאגדים) בפייתון

**1. מבוא: מה הן קבוצות?**

במדעי המחשב ובמתמטיקה, קבוצות הן דרך לייצג מאגדים של איברים ייחודיים. חשוב להבין ש:

*   **ייחודיות:** כל איבר בקבוצה חייב להיות ייחודי. אין לאפשר כפילויות.
*   **אי-סדר:** סדר האיברים בקבוצה אינו משנה.

**2. קבוצות ופירות**

נדמיין שיש לנו רק את הפירות עצמם:

*   🍎 (תפוח)
*   🍐 (אגס)
*   🍉 (אבטיח)
*   🧺 (סל)

חשוב ש:

1.  **במאגד *אין* פירות זהים:** אם במאגד יש תפוח, לא יהיה שם תפוח זהה נוסף. כל פרי ייחודי במאגד שלו.
2.  **סדר הפירות אינו חשוב:** אם במאגד יש תפוח ואגס, זה זהה כאילו היו שם אגס ותפוח. הסדר אינו משנה.

לדוגמה, `{🍎, 🍐, 🍉}` – זוהי קבוצה המכילה תפוח, אגס ואבטיח.

**3. מדוע קבוצות דורשות ייחודיות של איברים?**

*   קבוצות נועדו למעקב אחר *קיום* איברים, ולא אחר *כמותם*.
*   ייחודיות האיברים מפשטת ביצוע פעולות על קבוצות.
*   קבוצות משמשות להסרת יתירות נתונים.

**4. פעולות על קבוצות (מאגדי פירות):**

1.  **איחוד (Union): "אוספים את כל הפירות למאגד אחד"**
    *   מאחד שני מאגדי פירות על ידי איסוף כל הפירות למאגד חדש. אם בשני המאגדים יש פירות זהים, במאגד החדש יהיה רק פרי אחד כזה.
    *   אם במאגד A יש {🍎, 🍐}, ובמאגד B יש {🍐, 🍉}, אז במאגד A ∪ B יהיה {🍎, 🍐, 🍉}.

2.  **חיתוך (Intersection): "מוצאים את הפירות המשותפים"**
    *   מחפש *רק* את הפירות הקיימים גם במאגד A וגם במאגד B.
    *   אם במאגד A יש {🍎, 🍐}, ובמאגד B יש {🍐, 🍉}, אז במאגד A ∩ B יהיה רק {🍐}.

3.  **הפרש (Difference): "פירות הקיימים במאגד אחד אך לא בשני"**
    *   מחפש *רק* את הפירות הקיימים במאגד A אך אינם קיימים במאגד B.
    *   אם במאגד A יש {🍎, 🍐}, ובמאגד B יש {🍐, 🍉}, אז במאגד A - B יהיה {🍎}, ובמאגד B - A יהיה {🍉}.

4.  **הפרש סימטרי (Symmetric Difference): "פירות הקיימים רק באחד מהמאגדים"**
    *   מחפש *את כל* הפירות הקיימים או במאגד A או במאגד B, אך לא בשניהם בו-זמנית.
    *   אם במאגד A יש {🍎, 🍐}, ובמאגד B יש {🍐, 🍉}, אז במאגד A ^ B יהיה {🍎, 🍉}.

5.  **תת-קבוצה (Subset): "האם כל הפירות ממאגד אחד קיימים בשני?"**
    *   בודק האם מאגד A הוא תת-קבוצה של מאגד B. המשמעות היא שכל הפירות ממאגד A קיימים גם במאגד B.
    *   **דוגמה:** אם במאגד A יש {🍎, 🍐}, ובמאגד B יש {🍎, 🍐, 🍉}, אז A <= B.

6.  **על-קבוצה (Superset): "האם מאגד אחד מכיל את כל הפירות מהשני?"**
    *   בודק האם מאגד A הוא על-קבוצה של מאגד B. המשמעות היא שכל הפירות ממאגד B קיימים גם במאגד A.
    *   אם במאגד A יש {🍎, 🍐, 🍉}, ובמאגד B יש {🍎, 🍐}, אז A >= B.

```python
from typing import Set

def create_fruit_set(fruit_string: str) -> Set[str]:
    """
    יוצר קבוצת פירות ממחרוזת.

    Args:
        fruit_string: מחרוזת המכילה פירות (🍎, 🍐, 🍉, 🧺).

    Returns:
        קבוצה של פירות ייחודיים.
    """
    # בדיקה שהמחרוזת מכילה רק תווים מותרים
    if not all(fruit in ["🍎", "🍐", "🍉", "🧺"] for fruit in fruit_string):
        raise ValueError("המחרוזת יכולה להכיל רק את הסימנים 🍎, 🍐, 🍉, 🧺")
    # משתמשים ב-set() ליצירת הקבוצה
    return set(fruit_string)

def display_set(fruit_set: Set[str]) -> str:
  """
  ממיר קבוצת פירות למחרוזת לצורך הצגה.

    Args:
        fruit_set: קבוצת הפירות.

    Returns:
        מחרוזת לצורך הצגה.
  """
  return "{" + ", ".join(fruit_set) + "}"


# יוצרים קבוצות פירות
fruits_set_A = create_fruit_set("🍎🍐")  # קבוצה A: {🍎, 🍐}
fruits_set_B = create_fruit_set("🍐🍉")  # קבוצה B: {🍐, 🍉}
fruits_set_C = create_fruit_set("🍎🍐🍉")  # קבוצה C: {🍎, 🍐, 🍉}
fruits_set_D = create_fruit_set("🧺")  # קבוצה D: {🧺}

# מציגים את הקבוצות
print(f"קבוצה A: {display_set(fruits_set_A)}")
print(f"קבוצה B: {display_set(fruits_set_B)}")
print(f"קבוצה C: {display_set(fruits_set_C)}")
print(f"קבוצה D: {display_set(fruits_set_D)}")

# איחוד קבוצות (Union)
union_result = fruits_set_A | fruits_set_B
print(f"A ∪ B: {display_set(union_result)}")  # תוצאה: {🍎, 🍐, 🍉}

# חיתוך קבוצות (Intersection)
intersection_result = fruits_set_A & fruits_set_B
print(f"A ∩ B: {display_set(intersection_result)}")  # תוצאה: {🍐}

# הפרש קבוצות (Difference)
difference_result_AB = fruits_set_A - fruits_set_B
print(f"A - B: {display_set(difference_result_AB)}")  # תוצאה: {🍎}
difference_result_BA = fruits_set_B - fruits_set_A
print(f"B - A: {display_set(difference_result_BA)}")  # תוצאה: {🍉}

# הפרש סימטרי קבוצות (Symmetric Difference)
symmetric_difference_result = fruits_set_A ^ fruits_set_B
print(f"A ^ B: {display_set(symmetric_difference_result)}")  # תוצאה: {🍎, 🍉}

# תת-קבוצה (Subset)
subset_result1 = fruits_set_A <= fruits_set_C
print(f"A <= C: {subset_result1}")  # תוצאה: True (A תת-קבוצה של C)
subset_result2 = fruits_set_A <= fruits_set_B
print(f"A <= B: {subset_result2}") # תוצאה: False (A אינה תת-קבוצה של B)

# על-קבוצה (Superset)
superset_result1 = fruits_set_C >= fruits_set_A
print(f"C >= A: {superset_result1}")  # תוצאה: True (C על-קבוצה של A)
superset_result2 = fruits_set_B >= fruits_set_A
print(f"B >= A: {superset_result2}")  # תוצאה: False (B אינה על-קבוצה של A)

# בדיקה האם איבר קיים
print(f"🍎 in A: {'🍎' in fruits_set_A}")  # תוצאה: True
print(f"🍉 in A: {'🍉' in fruits_set_A}")  # תוצאה: False
```

*   **`create_fruit_set(fruit_string)`:** פונקציה זו יוצרת קבוצה ממחרוזת פירות.
    *   `set(fruit_string)` הופכת את המחרוזת לקבוצה, מסירה כפילויות והופכת את סדר האיברים לבלתי חשוב.
    *   אנו בודקים שהמחרוזת מורכבת רק מסימני היוניקוד המורשים.
*   **`display_set(fruit_set)`:** פונקציה זו משמשת להמרת קבוצה למחרוזת קריאה לצורך הצגה.
*   **דוגמאות:** אנו יוצרים מספר קבוצות ומיישמים עליהן פעולות שונות. תוצאות כל פעולה מודפסות על המסך.
