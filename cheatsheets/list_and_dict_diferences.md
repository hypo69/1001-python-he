**דוגמאות לשימוש ברשימות ומילונים לצורך ייצוג נתוני מוצרים**

אפרוט מוצר אבסטרקטי לפי מאפיינים ואדגים כיצד להשתמש ברשימה לייצוג מוצרים בקטגוריה.

*   **מילון (`dict`)** – זוהי דרך אידיאלית לייצוג *מאפיינים* של מוצר יחיד, בה קיימים זוגות של 'מפתח-ערך'.
*   **רשימה (`list`)** – זוהי דרך מצוינת לייצוג *אוסף* של מוצרים מאותו סוג, כאשר כל מוצר יכול להיות מיוצג על ידי מילון.

**1. מילון (`dict`) לצורך ייצוג מאפייני מוצר**

מילון לצורך ייצוג מאפיינים של מוצר יחיד. במקרה זה, מפתחות המילון יהיו שמות המאפיינים, והערכים יהיו הערכים המתאימים להם.

```python
product = {
    "id": "12345",
    "name": "Смартфон SuperPhone X",
    "brand": "SuperTech",
    "price": 999.99,
    "color": "Space Gray",
    "screen_size": 6.7,
    "storage": "256 GB",
    "camera": "108 MP",
    "os": "Android 13",
    "available": True
}

print("--- Характеристики товара ---")
for key, value in product.items():
    print(f"{key}: {value}")

print("\n--- Доступ к конкретным характеристикам ---")
print(f"Название: {product['name']}")
print(f"Цена: ${product['price']}")
print(f"Доступен: {'да' if product['available'] else 'нет'}")

```

**הסבר הקוד:**

*   המילון `product`, בו המפתחות הם שמות מאפייני המוצר (id, name, brand, price וכו'), והערכים הם הערכים המתאימים להם.
*   השימוש ב-`product.items()` עבור איטרציה על כל זוגות 'מפתח-ערך' והדפסתם למסך.

**2. רשימה (`list`) לצורך ייצוג מוצרים בקטגוריה**

רשימה לצורך ייצוג רשימת מוצרים בקטגוריה מסוימת.
במקרה זה, כל איבר ברשימה ייצג מוצר נפרד, אשר בתורו יכול להיות מיוצג על ידי מילון.

```python
products = [
    {
        "id": "12345",
        "name": "Смартфон SuperPhone X",
        "brand": "SuperTech",
        "price": 999.99,
        "available": True
    },
    {
        "id": "67890",
        "name": "Планшет TabPro S",
        "brand": "TechGenius",
        "price": 799.99,
        "available": False
    },
    {
        "id": "13579",
        "name": "Наушники SoundBeats Pro",
        "brand": "AudioMax",
        "price": 199.99,
        "available": True
    }
]
print("\n--- Список товаров в категории ---")

for product in products:
   print(f"{product['name']} ({product['brand']}), Price: ${product['price']}, Available: {'Yes' if product['available'] else 'No'}")

print("\n--- Доступ к первому товару ---")

first_product = products[0]
print(f"ID: {first_product['id']}, Name: {first_product['name']}")

print("\n--- Вывод только имен товаров ---")
for product in products:
  print(f"Name: {product['name']}")
```

**הסבר הקוד:**

*   הרשימה `products`, בה כל איבר הוא מילון המייצג מוצר נפרד.
*   ביצעתי איטרציה על הרשימה והדפסתי למסך מידע אודות כל מוצר, תוך שימוש במפתחות המילון לגישה למאפייני המוצר.
*   הדגמתי כיצד ניתן לגשת למוצר הראשון ברשימה, באמצעות שימוש באינדקס `0`.
*   ביצעתי איטרציה על הרשימה שוב, והדגמתי כיצד ניתן להדפיס רק את שמות המוצרים.
