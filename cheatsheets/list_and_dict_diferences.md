**הנחיה מערכתית למודל שפה: תרגום מסמכים מרוסית**

עליך לתרגם מסמך מרוסית לשפות שונות בהתאם להקשר התוכן:

1.  **טקסט שאינו קוד** – כל טקסט מחוץ לחלקי קוד יש לתרגם לעברית אקדמית, ברורה ורהוטה. יש לשמור על רמת ניסוח גבוהה, מתאימה למסמכים טכניים או אקדמיים.

2.  **קוד תוכנה** – אין לשנות את הקוד עצמו בשום אופן. אין להוסיף, למחוק או לערוך שורות קוד.

3.  **הערות בתוך הקוד** (כולל תגובות עם `#`, מסמכי docstring, והערות בתוך שורות קוד) – יש לתרגם רק את ההערות וה-docstrings לעברית אקדמית. עם זאת, במקרה שהוראות ניתנות באנגלית בקוד – יש להשאיר אותן באנגלית תקנית, ולהחליף את ההערות הרוסיות באנגלית טכנית ברורה.

4.  **אין לתרגם מילים או שמות קוד, משתנים, שמות קבצים או פונקציות**.

5.  **יש לשמר את המבנה המקורי של המסמך**, כולל רווחים, מבנה קבצים, פסקאות, ותיחום של קטעי קוד.

6.  **יש להקפיד על עקביות בתרגום מונחים טכניים** – מונח ברוסית שמופיע כמה פעמים צריך להיות מתורגם באותה הצורה בכל המקומות, בהתאם להקשר.

---

דוגמאות לשימוש ברשימות ומילונים לייצוג נתוני מוצרים. אני אפרט מוצר מופשט לפי מאפיינים ואראה כיצד להשתמש ברשימה לייצוג מוצרים בקטגוריה.

*   **מילון (`dict`)** – זוהי דרך אידיאלית לייצוג *מאפיינים* של מוצר יחיד, שבה יש זוגות "מפתח-ערך".
*   **רשימה (`list`)** – זוהי דרך מצוינת לייצוג *קבוצה* של מוצרים מאותו סוג, שבה כל מוצר יכול להיות מיוצג על ידי מילון.

**1. מילון (`dict`) לייצוג מאפייני מוצר**

מילון לייצוג מאפיינים של מוצר יחיד. במקרה זה, המפתחות של המילון יהיו שמות המאפיינים, והערכים – ערכיהם המתאימים.

```python
product = {
    "id": "12345",
    "name": "Смартфон SuperPhone X",
    "brand": "SuperTech",
    "price": 999.99,
    "color": "Space Gray",
    "screen_size": 6.7,
    "storage": "256 GB",
    "camera": "108 MP",
    "os": "Android 13",
    "available": True
}

print("--- מאפייני מוצר ---")
for key, value in product.items():
    print(f"{key}: {value}")

print("\n--- גישה למאפיינים ספציפיים ---")
print(f"שם: {product['name']}")
print(f"מחיר: ${product['price']}")
print(f"זמין: {'כן' if product['available'] else 'לא'}")

```

**הסבר קוד:**

*   המילון `product`, שבו המפתחות הם שמות מאפייני המוצר (id, name, brand, price וכו'), והערכים – ערכיהם המתאימים.
*   `product.items()` לאיטרציה על כל זוגות "מפתח-ערך" ולהצגתם על המסך.

**2. רשימה (`list`) לייצוג מוצרים בקטגוריה**

רשימה לייצוג רשימת מוצרים בקטגוריה מסוימת.
במקרה זה, כל איבר ברשימה ייצג מוצר נפרד, אשר, בתורו, יכול להיות מיוצג על ידי מילון.

```python
products = [
    {
        "id": "12345",
        "name": "Смартфон SuperPhone X",
        "brand": "SuperTech",
        "price": 999.99,
        "available": True
    },
    {
        "id": "67890",
        "name": "Планшет TabPro S",
        "brand": "TechGenius",
        "price": 799.99,
        "available": False
    },
    {
        "id": "13579",
        "name": "Наушники SoundBeats Pro",
        "brand": "AudioMax",
        "price": 199.99,
        "available": True
    }
]
print("\n--- רשימת מוצרים בקטגוריה ---")

for product in products:
   print(f"{product['name']} ({product['brand']}), Price: ${product['price']}, Available: {'Yes' if product['available'] else 'No'}")

print("\n--- גישה למוצר הראשון ---")

first_product = products[0]
print(f"ID: {first_product['id']}, Name: {first_product['name']}")

print("\n--- הצגת שמות המוצרים בלבד ---")
for product in products:
  print(f"Name: {product['name']}")
```

**הסבר קוד:**

*   הרשימה `products`, שבה כל איבר - זהו מילון המייצג מוצר נפרד.
*   ביצעתי איטרציה על הרשימה והצגתי על המסך מידע על כל מוצר, תוך שימוש במפתחות המילון לגישה למאפייני המוצר.
*   הצגתי כיצד ניתן לגשת למוצר הראשון ברשימה, באמצעות האינדקס `0`.
*   ביצעתי איטרציה על הרשימה פעם נוספת, והצגתי כיצד ניתן להציג רק את שמות המוצרים.
