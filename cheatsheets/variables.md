**משתנים בפייתון: מה הם ומדוע הם נחוצים**

משתנים הם מכלים בעלי שם לאחסון נתונים בזיכרון המחשב. הם מאפשרים גישה לנתונים באמצעות שם, במקום להשתמש בהם ישירות.

דוגמה:
```python
x = 10
y = 'Привет, мир!'
```
כאן `x` ו-`y` הם משתנים. `x` מאחסן את המספר 10, ו-`y` מאחסן את המחרוזת 'Привет, мир!'.

### **כיצד פועלים משתנים בפייתון?**
1.  **טיפוסיות דינמית (Dynamic Typing)**:
    בפייתון אין צורך לציין את טיפוס המשתנה בעת יצירתו – זה נעשה אוטומטית. לדוגמה:
    ```python
    a = 42      # מספר שלם (int)
    b = 3.14    # מספר נקודה צפה (float)
    c = 'Текст' # מחרוזת (str)
    ```

2.  **מודל אחסון נתונים מבוסס-הפניות (Reference Model)**:
    משתנים בפייתון הם הפניות לאובייקטים בזיכרון. לדוגמה:
    ```python
    x = 5
    y = x  # כעת גם y מפנה לאובייקט 5
    x = 10 # כעת x מפנה לאובייקט אחר 10, ו-y עדיין מפנה ל-5
    ```

---

### **כללי מתן שמות למשתנים**
1.  **כללים מחייבים**:
    -   שם המשתנה יכול לכלול אותיות, ספרות וסימן `_`, אך אינו יכול להתחיל בספרה.
        ✅ דוגמאות: `my_var`, `_data`, `var123`
        ❌ שגוי: `123var`, `my-var`
    -   שם המשתנה רגיש לאותיות גדולות/קטנות.
        דוגמה: `age` ו-`Age` הם משתנים שונים.

2.  **המלצות לשמות בעלי משמעות**:
    -   השתמש בשמות המשקפים את מהות הנתונים.
        ❌ גרוע: `a = 100`, `b = 'Имя'`
        ✅ טוב: `salary = 100`, `username = 'Имя'`
    -   עבור שמות מרובי מילים השתמש בסגנון **snake_case**:
        דוגמה: `user_age`, `total_cost`.

3.  **מילים שמורות (Reserved Words)**:
    אסור להשתמש במילות מפתח של פייתון (לדוגמה, `if`, `for`, `while`) כשמות משתנים.

    כדי לראות רשימת מילות מפתח, הרץ:
    ```python
    import keyword
    print(keyword.kwlist)
    ```

---

### **מאפיינים של אחסון טיפוסי נתונים**
1.  **טיפוסי נתונים בפייתון**:
    -   **מספרים**: `int`, `float`, `complex`
    -   **מחרוזות**: `str`
    -   **רשימות**: `list`
    -   **טאפלים**: `tuple`
    -   **סטים (קבוצות)**: `set`
    -   **מילונים**: `dict`

2.  **טיפוסים ניתנים לשינוי (Mutable) ובלתי ניתנים לשינוי (Immutable)**:
    -   **ניתנים לשינוי**: `list`, `dict`, `set`.
        דוגמה:
        ```python
        lst = [1, 2, 3]
        lst.append(4)  # הרשימה השתנתה
        ```
    -   **בלתי ניתנים לשינוי**: `int`, `float`, `str`, `tuple`.
        דוגמה:
        ```python
        name = 'Alice'
        name[0] = 'B'  # שגיאה, מחרוזות אינן ניתנות לשינוי
        ```

3.  **פונקציה `type` לבדיקת טיפוס**:
    ```python
    x = 10
    print(type(x))  # <class 'int'>
    ```

### **עצות למתחילים**
1.  השתמש בשמות משתנים בעלי משמעות כדי שהקוד שלך יהיה מובן.
2.  זכור שפייתון אינה דורשת הצהרת טיפוס למשתנה, אך היה קשוב כדי לא להתבלבל עם טיפוסי הנתונים.
3.  למד פונקציות מובנות לעבודה עם משתנים, כגון `type()`, `id()` ומודולים, כגון `sys`, כדי להבין טוב יותר כיצד פייתון מנהלת זיכרון.
