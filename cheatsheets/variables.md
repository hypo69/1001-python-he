**משתנים בפייתון: מה הם ולמה הם נחוצים**

משתנים הם מכלים (קונטיינרים) בעלי שם המשמשים לאחסון נתונים בזיכרון המחשב. הם מאפשרים לגשת לנתונים לפי שם, במקום להשתמש בהם ישירות.

דוגמה:
```python
x = 10
y = 'Привет, мир!'
```
כאן `x` ו-`y` הם משתנים. `x` מאחסן את המספר 10, ו-`y` מאחסן את המחרוזת 'Привет, мир!'.

### **כיצד פועלים משתנים בפייתון?**
1.  **טיפוסיות דינמית**:
    בפייתון אין צורך לציין את טיפוס המשתנה בעת יצירתו – הדבר מתבצע באופן אוטומטי. לדוגמה:
    ```python
    a = 42      # מספר שלם (int)
    b = 3.14    # מספר עשרוני (float)
    c = 'Текст' # מחרוזת (str)
    ```

2.  **מודל הפניה לאחסון נתונים**:
    משתנים בפייתון הם הפניות (references) לאובייקטים בזיכרון. לדוגמה:
    ```python
    x = 5
    y = x  # y מפנה כעת גם הוא לאובייקט 5
    x = 10 # x מפנה כעת לאובייקט אחר 10, בעוד ש-y עדיין מפנה ל-5
    ```

---

### **כללים למתן שמות למשתנים**
1.  **כללים מחייבים**:
    *   שם משתנה יכול להכיל אותיות, ספרות ותו `_`, אך אינו יכול להתחיל בספרה.
        ✅ דוגמאות: `my_var`, `_data`, `var123`
        ❌ שגוי: `123var`, `my-var`
    *   שם המשתנה רגיש לאותיות גדולות וקטנות (case-sensitive).
        דוגמה: `age` ו-`Age` הם משתנים שונים.

2.  **המלצות לשמות משמעותיים**:
    *   השתמש בשמות המשקפים את מהות הנתונים.
        ❌ גרוע: `a = 100`, `b = 'Имя'`
        ✅ טוב: `salary = 100`, `username = 'Имя'`
    *   לשמות מרובי מילים, השתמש בסגנון **snake_case**:
        דוגמה: `user_age`, `total_cost`.

3.  **מילים שמורות**:
    אסור להשתמש במילות מפתח של פייתון (לדוגמה, `if`, `for`, `while`) כשמות משתנים.

    כדי לראות רשימת מילות מפתח, הרץ את:
    ```python
    import keyword
    print(keyword.kwlist)
    ```

---

### **מאפיינים של אחסון טיפוסי נתונים**
1.  **טיפוסי נתונים בפייתון**:
    *   **מספרים**: `int`, `float`, `complex`
    *   **מחרוזות**: `str`
    *   **רשימות**: `list`
    *   **Tuple-ים**: `tuple`
    *   **קבוצות**: `set`
    *   **מילונים**: `dict`

2.  **טיפוסים ניתנים לשינוי וטיפוסים בלתי ניתנים לשינוי**:
    *   **ניתנים לשינוי (Mutable)**: `list`, `dict`, `set`.
        דוגמה:
        ```python
        lst = [1, 2, 3]
        lst.append(4)  # הרשימה שונתה
        ```
    *   **בלתי ניתנים לשינוי (Immutable)**: `int`, `float`, `str`, `tuple`.
        דוגמה:
        ```python
        name = 'Alice'
        name[0] = 'B'  # שגיאה, מחרוזות הן בלתי ניתנות לשינוי
        ```

3.  **פונקציית `type` לבדיקת טיפוס**:
    ```python
    x = 10
    print(type(x))  # <class 'int'>
    ```

### **עצות למתחילים**
1.  השתמש בשמות משתנים משמעותיים כדי שקוד שלך יהיה קריא ומובן.
2.  זכור שפייתון אינה דורשת הצהרה על טיפוס משתנה, אך היה זהיר כדי לא להתבלבל עם טיפוסי הנתונים.
3.  למד פונקציות מובנות לעבודה עם משתנים, כגון `type()`, `id()`, ומודולים, כגון `sys`, כדי להבין טוב יותר כיצד פייתון מנהלת זיכרון.
