### **זמן פולינומי**

**זמן פולינומי** הוא מונח המשמש בתורת הסיבוכיות החישובית לתיאור זמן הריצה של אלגוריתם, אשר גדל כפולינום של גודל הקלט. אם זמן הריצה של אלגוריתם ניתן לביטוי כ-\(O(n^k)\), כאשר \(n\) הוא גודל הקלט ו-\(k\) הוא קבוע, אזי אלגוריתם כזה רץ בזמן פולינומי.

#### **דוגמאות:**
1.  **מיון רשימה**: אלגוריתמים כמו מיון מיזוג או מיון מהיר רצים בזמן \(O(n \log n)\), וזהו זמן פולינומי.
2.  **מציאת מסלול קצר ביותר בגרף**: אלגוריתם דייקסטרה רץ בזמן \(O(n^2)\) או \(O(n \log n)\) בהתאם למימוש, וגם זהו זמן פולינומי.

#### **מאפיינים:**
*   אלגוריתמים הרצים בזמן פולינומי נחשבים **יעילים** ו**ישימים בפועל**.
*   בעיות שניתן לפתור בזמן פולינומי שייכות ל**מחלקה P**.

---

### **זמן אקספוננציאלי**

**זמן אקספוננציאלי** הוא זמן ריצה של אלגוריתם, אשר גדל באופן אקספוננציאלי ביחס לגודל הקלט. אם זמן הריצה ניתן לביטוי כ-\(O(k^n)\), כאשר \(n\) הוא גודל הקלט ו-\(k\) הוא קבוע, אזי אלגוריתם כזה רץ בזמן אקספוננציאלי.

#### **דוגמאות:**
1.  **בעיית הסוכן הנודד**: פתרון באמצעות מעבר על כל המסלולים האפשריים דורש זמן \(O(n!)\), שהוא גרוע יותר מזמן אקספוננציאלי.
2.  **מעבר על כל תת-הקבוצות**: אלגוריתם הבודק את כל תת-הקבוצות האפשריות של קבוצה בגודל \(n\) רץ בזמן \(O(2^n)\).

#### **מאפיינים:**
*   אלגוריתמים הרצים בזמן אקספוננציאלי נחשבים **בלתי יעילים** עבור נתונים גדולים, שכן זמן הריצה הופך להיות גדול באופן בלתי ישים אפילו עבור \(n\) קטן יחסית.
*   בעיות שניתן לפתור רק בזמן אקספוננציאלי שייכות לעתים קרובות ל**מחלקות NP-קשות** או **NP-שלמות**.

---

### **השוואה בין זמן פולינומי לזמן אקספוננציאלי**

| **מאפיין**              | **זמן פולינומי**                  | **זמן אקספוננציאלי**                  |
| :----------------------- | :-------------------------------- | :------------------------------------ |
| **גידול זמן הריצה**      | איטי (לדוגמה, \(n^2\), \(n^3\))   | מהיר (לדוגמה, \(2^n\), \(3^n\))      |
| **דוגמאות לבעיות**       | מיון, מציאת מסלול קצר ביותר       | בעיית הסוכן הנודד, מעבר על תת-קבוצות |
| **ישימות בפועל**         | יעיל עבור נתונים גדולים           | אינו ישים עבור נתונים גדולים         |
| **מחלقت סיבוכיות**       | P                                 | NP-קשות, NP-שלמות                    |

---

### **מדוע זה חשוב?**

1.  **זמן פולינומי**:
    *   אלגוריתמים הרצים בזמן פולינומי נחשבים **ישימים בפועל**, שכן הם יכולים לעבד כמויות גדולות של נתונים בזמן סביר.
    *   בעיות מ**מחלקה P** (הניתנות לפתרון בזמן פולינומי) מהוות בסיס ליישומים רבים במדעי המחשב, כגון עיבוד נתונים, רשתות, קריפטוגרפיה ובינה מלאכותית.

2.  **זמן אקספוננציאלי**:
    *   אלגוריתמים הרצים בזמן אקספוננציאלי הופכים להיות **בלתי ישימים** גם עבור נתונים קטנים יחסית. לדוגמה, עבור \(n = 100\), \(2^n\) כבר עולה על מספר האטומים ביקום הנצפה.
    *   בעיות שניתן לפתור רק בזמן אקספוננציאלי, דורשות לעתים קרובות שימוש ב**שיטות קירוב**, **היוריסטיקות** או **חישובים מקבילים**.

### **דוגמה להבנה**

דמיינו שיש לכם בעיה, ואתם רוצים לפתור אותה עבור \(n = 10\) ו-\(n = 100\):

*   **זמן פולינומי (\(n^2\))**:
    *   עבור \(n = 10\): \(10^2 = 100\) פעולות.
    *   עבור \(n = 100\): \(100^2 = 10\,000\) פעולות.

*   **זמן אקספוננציאלי (\(2^n\))**:
    *   עבור \(n = 10\): \(2^{10} = 1\,024\) פעולות.
    *   עבור \(n = 100\): \(2^{100} \approx 1.26 \times 10^{30}\) פעולות.

כפי שניתן לראות, עבור \(n = 100\) האלגוריתם הפולינומי יבצע 10,000 פעולות, וזה בהחלט אפשרי/ריאלי, בעוד שהאלגוריתם האקספוננציאלי ידרוש \(1.26 \times 10^{30}\) פעולות, וזה כמעט בלתי אפשרי.

לבניית גרפים המדגימים את ההבדל בין זמן פולינומי לזמן אקספוננציאלי, ניתן להשתמש בפונקציות מתמטיות שונות. הנה דוגמאות לפונקציות שניתן להשתמש בהן להדמיה:

### **פונקציות פולינומיות**
1.  **פונקציה לינארית**:
    \( f(n) = n \)
    דוגמה: זמן ריצה של אלגוריתם המעבד כל איבר פעם אחת.

2.  **פונקציה ריבועית**:
    \( f(n) = n^2 \)
    דוגמה: זמן ריצה של אלגוריתם עם לולאות מקוננות, למשל, מיון בועות.

3.  **פונקציה מעוקבת (קובית)**:
    \( f(n) = n^3 \)
    דוגמה: זמן ריצה של אלגוריתם המעבד נתונים תלת-ממדיים.

4.  **פונקציה לוגריתמית**:
    \( f(n) = \log n \)
    דוגמה: זמן ריצה של חיפוש בינארי.

5.  **פונקציה לינארית-לוגריתמית**:
    \( f(n) = n \log n \)
    דוגמה: זמן ריצה של מיון מהיר או מיון מיזוג.

---

### **פונקציות אקספוננציאליות**
1.  **פונקציה אקספוננציאלית**:
    \( f(n) = 2^n \)
    דוגמה: זמן ריצה של אלגוריתם העובר על כל תת-הקבוצות של קבוצה.

2.  **פונקציית עצרת**:
    \( f(n) = n! \)
    דוגמה: זמן ריצה של אלגוריתם העובר על כל התמורות (לדוגמה, בעיית הסוכן הנודד).

3.  **פונקציה אקספוננציאלית עם בסיס אחר**:
    \( f(n) = 3^n \)
    דוגמה: זמן ריצה של אלגוריתם הבוחן את כל הקומבינציות האפשריות.

### **דוגמה לקוד לבניית גרפים (Python, Matplotlib)**

```python
import matplotlib.pyplot as plt
import numpy as np

# Диапазон значений n
n = np.linspace(1, 20, 100)

# Полиномиальные функции
linear = n
quadratic = n**2
cubic = n**3
logarithmic = np.log(n)
nlogn = n * np.log(n)

# Экспоненциальные функции
exponential = 2**n
factorial = [np.math.factorial(int(i)) for i in n]  # Факториал определён только для целых чисел

# Построение графиков
plt.figure(figsize=(10, 6))

# Полиномиальные функции
plt.plot(n, linear, label='Linear: $f(n) = n$')
plt.plot(n, quadratic, label='Quadratic: $f(n) = n^2$')
plt.plot(n, cubic, label='Cubic: $f(n) = n^3$')
plt.plot(n, logarithmic, label='Logarithmic: $f(n) = \log n$')
plt.plot(n, nlogn, label='Linearithmic: $f(n) = n \log n$')

# Экспоненциальные функции
plt.plot(n, exponential, label='Exponential: $f(n) = 2^n$')
plt.plot(n, factorial, label='Factorial: $f(n) = n!$')

# Настройка графика
plt.yscale('log')  # Логарифмическая шкала для удобства
plt.xlabel('Input size (n)')
plt.ylabel('Time complexity')
plt.title('Comparison of Polynomial and Exponential Time Complexity')
plt.legend()
plt.grid(True)
plt.show()
```
