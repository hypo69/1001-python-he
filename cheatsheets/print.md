# פונקציה print()

פונקציה `print()` ב-Python משמשת להצגת מידע בקונסולה. זוהי אחת הפונקציות הפשוטות והנפוצות ביותר, במיוחד לצורך דיבוג והצגת נתונים. נסקור את ההיבטים העיקריים של פעולתה.

## תחביר
```python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

### פרמטרים:
1.  **`*objects`**:
    *   רשימת האובייקטים שיש להציג. ניתן להעביר ערך אחד או יותר, מופרדים בפסיק.
    *   דוגמה:
        ```python
        print('Привет', 'мир')
        ```
        תוצאה: `Привет мир`

2.  **`sep`** (ברירת מחדל `' '`):
    *   מפריד בין אובייקטים, אם הועברו מספר אובייקטים.
    *   דוגמה:
        ```python
        print('Привет', 'мир', sep=' - ')
        ```
        תוצאה: `Привет - мир`

3.  **`end`** (ברירת מחדל `'\n'`):
    *   מציין מה יתווסף בסוף השורה. ברירת המחדל היא מעבר שורה.
    *   דוגמה:
        ```python
        print('Привет', end='!')
        ```
        תוצאה: `Привет!`

4.  **`file`**:
    *   הזרם אליו יופנה הפלט (ברירת מחדל `sys.stdout` — פלט סטנדרטי).
    *   דוגמה: פלט לקובץ.
        ```python
        with open('output.txt', 'w') as f:
            print('Привет, файл!', file=f)
        ```

5.  **`flush`** (ברירת מחדל `False`):
    *   אם מוגדר ל-`True`, מרוקן את חוצץ הפלט באופן מיידי.

## דוגמאות שימוש

### מחרוזת פשוטה
```python
print('Hello, world!')
```
תוצאה: `Hello, world!`

### הצגת מספר ערכים
```python
name = 'Анна'
age = 25
print('Имя:', name, ', Возраст:', age)
```
תוצאה: `Имя: Анна , Возраст: 25`

### הגדרת מפריד
```python
print(1, 2, 3, sep=' -> ')
```
תוצאה: `1 -> 2 -> 3`

### הגדרת סיום שורה
```python
for i in range(3):
    print(i, end=' ')
```
תוצאה: `0 1 2`

### שימוש ב-f-strings עם הצגת שם משתנה
החל מ-Python 3.8, ניתן להשתמש ב-f-strings להצגת ערכי משתנים יחד עם שמותיהם בפורמט `שם=ערך`. זה נוח לצורך דיבוג, כיוון שמאפשר לראות מיד איזה משתנה ואיזה ערך מוצגים.
```python
name = 'Иван'
age = 30
print(f'{name=}, {age=}')
```
תוצאה: `name='Иван', age=30`

## טיפים שימושיים למתחילים

1.  **דיבוג קוד**:
    השתמש ב-`print()` לבדיקת ערכי משתנים:
    ```python
    x = 10
    y = 20
    print('Сумма:', x + y)
    ```

2.  **פלט מעוצב**:
    להצגת מחרוזות עם הצבת ערכים עדיף להשתמש בעיצוב:
    ```python
    name = 'Иван'
    age = 30
    print(f'Меня зовут {name}, мне {age} лет.')
    ```

3.  **רישום יומן (Logging)**:
    בפרויקטים גדולים יותר, לניהול הפלט עדיף להשתמש במודול `logging`, אך בשלבים הראשוניים `print()` עוזר להציג נתונים במהירות.
