# פונקציה `print()`

פונקציית `print()` ב-Python משמשת לפלט מידע לקונסולה. זוהי אחת הפונקציות הפשוטות והנפוצות ביותר, במיוחד עבור ניפוי שגיאות (debugging) והצגת נתונים. נבחן את ההיבטים העיקריים של פעולתה.

## תחביר
```python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

### פרמטרים:
1.  **`*objects`**:
    *   רשימת אובייקטים שיש להדפיס. ניתן להעביר ערך אחד או יותר, מופרדים בפסיק.
    *   דוגמה:
        ```python
        print('Привет', 'мир')
        ```
        תוצאה: `Привет мир`

2.  **`sep`** (ברירת מחדל `' '`):
    *   המפריד בין האובייקטים, אם הועברו מספר אובייקטים.
    *   דוגמה:
        ```python
        print('Привет', 'мир', sep=' - ')
        ```
        תוצאה: `Привет - мир`

3.  **`end`** (ברירת מחדל `'\n'`):
    *   מציין מה יתווסף בסוף השורה. ברירת המחדל היא מעבר שורה.
    *   דוגמה:
        ```python
        print('Привет', end='!')
        ```
        תוצאה: `Привет!`

4.  **`file`**:
    *   הזרם אליו יופנה הפלט (ברירת מחדל `sys.stdout` – פלט סטנדרטי).
    *   דוגמה: פלט לקובץ.
        ```python
        with open('output.txt', 'w') as f:
            print('Привет, файл!', file=f)
        ```

5.  **`flush`** (ברירת מחדל `False`):
    *   אם מוגדר ל-`True`, מרוקן בכוח את חוצץ הפלט.

## דוגמאות שימוש

### שורה פשוטה
```python
print('Hello, world!')
```
תוצאה: `Hello, world!`

### פלט של מספר ערכים
```python
name = 'Анна'
age = 25
print('Имя:', name, ', Возраст:', age)
```
תוצאה: `Имя: Анна , Возраст: 25`

### התאמת המפריד
```python
print(1, 2, 3, sep=' -> ')
```
תוצאה: `1 -> 2 -> 3`

### התאמת סוף השורה
```python
for i in range(3):
    print(i, end=' ')
```
תוצאה: `0 1 2`

### שימוש ב-f-strings עם פלט של שם המשתנה
החל מ-Python 3.8, ניתן להשתמש ב-f-strings כדי להדפיס ערכי משתנים יחד עם שמם בפורמט `שם=ערך`. זה נוח לניפוי שגיאות, כיוון שמאפשר לראות מיד איזה משתנה ואיזה ערך מודפס.
```python
name = 'Иван'
age = 30
print(f'{name=}, {age=}')
```
תוצאה: `name='Иван', age=30`

## טיפים שימושיים למתחילים

1.  **ניפוי שגיאות קוד**:
    השתמש ב-`print()` לבדיקת ערכי משתנים:
    ```python
    x = 10
    y = 20
    print('Сумма:', x + y)
    ```

2.  **פלט מעוצב**:
    להדפסת מחרוזות עם החלפת ערכים עדיף להשתמש בעיצוב מחרוזות:
    ```python
    name = 'Иван'
    age = 30
    print(f'Меня зовут {name}, мне {age} лет.')
    ```

3.  **רישום (Logging)**:
    בפרויקטים גדולים עדיף להשתמש במודול `logging` לניהול הפלט, אך בשלב התחלתי `print()` עוזר להדפיס נתונים במהירות.
