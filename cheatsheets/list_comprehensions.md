**מהי הכללת רשימה?**

בצורתה הפשוטה ביותר, הכללת רשימה (list comprehension) היא דרך קומפקטית ליצור רשימה חדשה המבוססת על רשימה קיימת (או כל אובייקט איטרבילי). במקום להשתמש בלולאת `for` מסורתית ולהוסיף אלמנטים לרשימה בזה אחר זה, ניתן לעשות זאת בשורת קוד אחת.

*   **קומפקטיות:** הן מאפשרות לכתוב פחות קוד.
*   **קריאות:** כאשר מתרגלים לתחביר שלהן, הן נתפסות כקלות יותר להבנה מאשר לולאות `for` מקבילות.
*   **ביצועים:** במקרים מסוימים, הכללות רשימה יכולות להיות מהירות יותר מלולאות `for`.

להלן התחביר הבסיסי של הכללת רשימה:

```python
new_list = [expression for item in iterable]
```

*   **`expression`**: ביטוי שמגדיר כיצד יחושב כל אלמנט ברשימה החדשה. זה יכול להיות כל דבר: ערך פשוט, פעולה מתמטית, קריאה לפונקציה.
*   **`item`**: משתנה שמקבל את הערך של כל אלמנט מתוך ה-`iterable` בתורו.
*   **`iterable`**: אובייקט איטרבילי, לדוגמה רשימה, טאפל, מחרוזת או תוצאה של הפעלת `range()`.

**דוגמה:**

נניח שיש לך רשימה של מספרים, ואתה רוצה ליצור רשימה חדשה המכילה את הריבועים של מספרים אלה.

**ללא הכללת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
squares = []
for number in numbers:
    squares.append(number**2)
print(squares)  # Вывод: [1, 4, 9, 16, 25]
```

**עם הכללת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
squares = [number**2 for number in numbers]
print(squares) # Вывод: [1, 4, 9, 16, 25]
```

הכללת הרשימה הופכת את הקוד לקצר וברור יותר!

**הוספת תנאי**

הכללות רשימה גם מאפשרות להוסיף תנאים לבחירת האלמנטים שיכללו ברשימה החדשה.

```python
new_list = [expression for item in iterable if condition]
```

* **`condition`**: תנאי שחייב להתקיים כדי שהאלמנט יכלל ברשימה החדשה.

**דוגמה:**

ניצור רשימה של ריבועים רק של מספרים זוגיים מהרשימה המקורית שלנו `numbers`.

**ללא הכללת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
even_squares = []
for number in numbers:
    if number % 2 == 0:
        even_squares.append(number**2)
print(even_squares) # Вывод: [4, 16]
```

**עם הכללת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
even_squares = [number**2 for number in numbers if number % 2 == 0]
print(even_squares) # Вывод: [4, 16]
```

הוספתי את `if number % 2 == 0` להכללת הרשימה כדי לסנן רק את המספרים הזוגיים.

**הכללות רשימה מרובות שורות**

כאשר הכללות רשימה הופכות מורכבות יותר, ניתן לפצל אותן למספר שורות לקריאות טובה יותר.

**דוגמה:**

```python
numbers = [1, 2, 3, 4, 5]
squared_odds = [
    number**2
    for number in numbers
    if number % 2 != 0
]
print(squared_odds) # Вывод: [1, 9, 25]
```
זה הופך את הקוד למובן יותר, במיוחד כשיש תנאים ארוכים או ביטויים מורכבים.

**תרגיל 1: המרת טמפרטורות**

יש לך רשימה של טמפרטורות במעלות צלזיוס, ואתה צריך להמיר אותן למעלות פרנהייט. הנוסחה להמרה היא: `F = (C * 9/5) + 32`.

להלן רשימת הטמפרטורות שלך בצלזיוס:

```python
celsius_temperatures = [0, 10, 20, 30, 40]
```

**משימה:**

1.  באמצעות הכללת רשימה, צור רשימה חדשה בשם `fahrenheit_temperatures` המכילה את הטמפרטורות מ-`celsius_temperatures` שהומרו למעלות פרנהייט.
2.  הדפס את הרשימה שהתקבלה `fahrenheit_temperatures` למסך.

**רמז:**

*   השתמש בנוסחה `(celsius * 9/5) + 32` כביטוי בהכללת הרשימה.
*   חשוב מהו ה-`iterable` ומהו ה-`item` במקרה זה.

נסה לכתוב את הקוד.

הפתרון שלי:
```python
# Исходный список температур в Цельсиях
celsius_temperatures = [0, 10, 20, 30, 40]

# Используем списковое включение для преобразования в Фаренгейты
# משתמשים בהכללת רשימה להמרה לפרנהייט
fahrenheit_temperatures = [
    (celsius * 9/5) + 32  # Выражение: формула преобразования
    # ביטוי: נוסחת ההמרה
    for celsius in celsius_temperatures # Перебор элементов списка
    # מעבר על איברי הרשימה
]

# Выводим результат
# מדפיסים את התוצאה
print(fahrenheit_temperatures) # Вывод: [32.0, 50.0, 68.0, 86.0, 104.0]
```

**ניתוח קוד:**

1.  **`celsius_temperatures = [0, 10, 20, 30, 40]`**: זוהי הרשימה המקורית עם הטמפרטורות במעלות צלזיוס.
2.  **`fahrenheit_temperatures = [...]`**: כאן אנו יוצרים רשימה חדשה באמצעות הכללת רשימה.
3.  **`(celsius * 9/5) + 32`**: זהו הביטוי שמבוצע עבור כל אלמנט. הוא ממיר את הטמפרטורה מצלזיוס לפרנהייט.
4.  **`for celsius in celsius_temperatures`**: זהו חלק מהכללת הרשימה שמבצע איטרציה על כל אלמנט ברשימה `celsius_temperatures`, ומקצה את ערכו למשתנה `celsius` בכל איטרציה.
5.  **`print(fahrenheit_temperatures)`**: מדפיס את רשימת הטמפרטורות שהתקבלה בפרנהייט.

**חשוב לציין:**

*   במקרה זה, `celsius_temperatures` הוא ה-`iterable`, ו-`celsius` הוא ה-`item`.
*   אני משתמש בעיצוב מרובה שורות של הכללת הרשימה כדי לשפר את קריאות הקוד.
*   בתוך הביטוי אני מיישם ישירות את נוסחת ההמרה, תוך שימוש בערך הנוכחי של `celsius`.

**תרגיל 2: סינון והמרת מחרוזות**

יש לך רשימה של מילים, ואתה צריך לבצע את הפעולות הבאות:

1.  לסנן מילים שמתחילות באות "a" (רישיות או קטנות אינן משנות).
2.  להמיר את כל המילים שנותרו לאותיות רישיות.

להלן רשימת המילים שלך:

```python
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]
```

**משימה:**

1.  באמצעות הכללת רשימה, צור רשימה חדשה בשם `transformed_words` המכילה רק את המילים מרשימת `words` שאינן מתחילות באות "a" (או "A"), ובה בעת מומרות לאותיות רישיות.
2.  הדפס את הרשימה שהתקבלה `transformed_words` למסך.

**רמזים:**

*   השתמש בשיטת המחרוזת `startswith()` כדי לבדוק אם מילה מתחילה באות מסוימת. אל תשכח את הרישיות! המר את הכל לאותיות קטנות.
*   השתמש בשיטת המחרוזת `upper()` כדי להמיר מחרוזת לאותיות רישיות.
*   חשוב על סדר הפעולות. קודם מסננים, אחר כך ממירים.

הפתרון שלי:

```python
# Исходный список слов
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]

# Используем списковое включение для фильтрации и преобразования
# משתמשים בהכללת רשימה לסינון והמרה
transformed_words = [
    word.upper() # Преобразуем в верхний регистр
    # ממירים לאותיות רישיות
    for word in words # Перебираем слова в списке
    # עוברים על המילים ברשימה
    if not word.lower().startswith("a") # Фильтруем слова, начинающиеся не с 'a'
    # מסננים מילים שאינן מתחילות ב-'a'
]

# Выводим результат
# מדפיסים את התוצאה
print(transformed_words) # Вывод: ['BANANA', 'KIWI', 'ORANGE']
```

**ניתוח קוד:**

1.  **`words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]`**: זוהי הרשימה המקורית של מילים.
2.  **`transformed_words = [...]`**: יוצרים רשימה חדשה באמצעות הכללת רשימה.
3.  **`word.upper()`**: זהו הביטוי שמבוצע עבור כל מילה שעברה את הסינון. הוא ממיר את המילה לאותיות רישיות.
4.  **`for word in words`**: מבצע איטרציה על כל המילים ברשימה `words`.
5.  **`if not word.lower().startswith("a")`**: זהו תנאי הסינון.
    *   `word.lower()`: תחילה אנו ממירים את המילה לאותיות קטנות, כדי שההשוואה תהיה בלתי תלויה ברישיות.
    *   `startswith("a")`: לאחר מכן אנו בודקים אם המילה מתחילה באות "a".
    *   `not`: הופכים את התוצאה, כדי להשאיר רק את המילים ש *אינן* מתחילות באות "a".
6.  **`print(transformed_words)`**: מדפיס את רשימת המילים שהומרו שהתקבלה.

**נקודות מפתח:**

*   שיטות המחרוזות `lower()`, `upper()` ו-`startswith()` משמשות בביטוי ובתנאי של הכללת הרשימה.
*   תנאי הסינון `if not word.lower().startswith("a")` מבטיח שברשימה החדשה יכללו רק מילים שאינן מתחילות ב-"a" (ללא קשר לרישיות).
*   הסינון מיושם תחילה, ורק לאחר מכן ההמרה לאותיות רישיות.
*   שוב נעשה שימוש בעיצוב מרובה שורות לשם קריאות.

**תרגיל 3: יצירת מילון מרשימת טאפלים**

יש לך רשימה של טאפלים, כאשר כל טאפל מכיל שני אלמנטים: שם וגיל של אדם.

```python
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]
```

**משימה:**

1. באמצעות הכללת רשימה, צור מילון בשם `people_dict`, שבו המפתחות יהיו שמות האנשים, והערכים - גילם.
2. הדפס את המילון `people_dict` למסך.

**רמזים:**

*   הכללות רשימה יוצרות רשימות, אך ניתן להשתמש בהן ליצירת רשימה של טאפלים, שניתן לאחר מכן להעביר לבנאי `dict()`.
*   חשוב כיצד לחלץ את השם והגיל מכל טאפל.
*   זכור שמילון הוא מבנה נתונים שמאחסן זוגות של מפתח-ערך.

**הסבר קצר:**

בפייתון ניתן ליצור מילון מרשימה של טאפלים, כאשר כל טאפל מורכב משני אלמנטים (מפתח, ערך).
לשם כך אני משתמש בבנאי `dict()`. לדוגמה:
```python
my_list = [("a", 1), ("b", 2)]
my_dict = dict(my_list)
print(my_dict) # Вывод {'a': 1, 'b': 2}
```
המשימה שלך היא ליצור באמצעות הכללת רשימה את רשימת הטאפלים הזו, שתשמש לאחר מכן ליצירת המילון.

הפתרון שלי:

```python
# Исходный список кортежей с данными о людях
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]

# Я использую списковое включение для создания списка кортежей, который потом преобразую в словарь
# אני משתמש בהכללת רשימה ליצירת רשימת טאפלים, שאותה אני ממיר לאחר מכן למילון
people_dict = dict(
    [
      (name, age) # Создаем кортеж (имя, возраст)
      # יוצרים טאפל (שם, גיל)
      for name, age in people_data # Перебираем кортежи в people_data
      # עוברים על הטאפלים ב-people_data
    ]
)

# Выводим полученный словарь
# מדפיסים את המילון שהתקבל
print(people_dict) # Вывод: {'Alice': 30, 'Bob': 25, 'Charlie': 35, 'David': 28}
```

**ניתוח קוד:**

1.  **`people_data = [...]`**: זוהי הרשימה המקורית של טאפלים, כאשר כל טאפל מכיל שם וגיל של אדם.
2.  **`people_dict = dict(...)`**: אנו משתמשים בבנאי `dict()` כדי ליצור מילון מרשימה של טאפלים.
3.  **`[ (name, age) for name, age in people_data ]`**: זוהי הכללת רשימה שיוצרת רשימה של טאפלים.
    *   `(name, age)`: זהו הביטוי שיוצר טאפל משני אלמנטים: `name` ו-`age`.
    *   `for name, age in people_data`: חלק זה מבצע איטרציה על כל הטאפלים מרשימת `people_data`. בכל איטרציה, הטאפל נפרק לשני משתנים: `name` ו-`age`.
4. **`print(people_dict)`**: מדפיס את המילון שנוצר.

**נקודות מפתח:**

*   הכללת רשימה יוצרת רשימה של טאפלים.
*   הבנאי `dict()` ממיר רשימה של טאפלים למילון, תוך שימוש באלמנט הראשון של הטאפל כמפתח, והשני כערך.
*   פריקת הטאפלים בלולאת `for name, age in people_data` הופכת את הקוד לקריא יותר.
*   הכללת הרשימה בדוגמה זו משמשת להכנת הנתונים עבור המילון.
