**מהי הבנת רשימה?**

בצורתה הפשוטה ביותר, הבנת רשימה היא דרך קומפקטית ליצור רשימה חדשה המבוססת על רשימה קיימת (או על כל אובייקט הניתן לאיטרציה). במקום להשתמש בלולאת `for` מסורתית ולהוסיף איברים לרשימה בזה אחר זה, ניתן לעשות זאת בשורת קוד אחת.

*   **קומפקטיות:** הן מאפשרות לכתוב פחות קוד.
*   **קריאות:** לאחר שמתרגלים לתחביר שלהן, מוצאים שהן קלות יותר להבנה מלולאות `for` מקבילות.
*   **ביצועים:** במקרים מסוימים, הבנות רשימה יכולות להיות מהירות יותר מלולאות `for`.

הנה התחביר הבסיסי של הבנת רשימה:

```python
new_list = [expression for item in iterable]
```

*   **`expression`**: הביטוי שמגדיר כיצד יחושב כל איבר ברשימה החדשה. זה יכול להיות כל דבר: ערך פשוט, פעולה מתמטית, קריאה לפונקציה.
*   **`item`**: משתנה שמקבל את הערך של כל איבר מתוך ה-`iterable` בתורו.
*   **`iterable`**: אובייקט הניתן לאיטרציה, לדוגמה, רשימה, tuple, מחרוזת או תוצאה של הפונקציה `range()`.

**דוגמה:**

נניח שיש לך רשימה של מספרים, ואתה רוצה ליצור רשימה חדשה המכילה את הריבועים של מספרים אלה.

**ללא הבנת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
squares = []
for number in numbers:
    squares.append(number**2)
print(squares)  # פלט: [1, 4, 9, 16, 25]
```

**עם הבנת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
squares = [number**2 for number in numbers]
print(squares) # פלט: [1, 4, 9, 16, 25]
```

הבנת רשימה הופכת את הקוד לקצר וברור יותר!

**הוספת תנאי**

הבנות רשימה גם מאפשרות להוסיף תנאים לבחירת האיברים שייכללו ברשימה החדשה.

```python
new_list = [expression for item in iterable if condition]
```

* **`condition`**: התנאי שחייב להתקיים כדי לכלול את האיבר ברשימה החדשה.

**דוגמה:**

ניצור רשימה של ריבועים של מספרים זוגיים בלבד מתוך הרשימה המקורית שלנו `numbers`.

**ללא הבנת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
even_squares = []
for number in numbers:
    if number % 2 == 0:
        even_squares.append(number**2)
print(even_squares) # פלט: [4, 16]
```

**עם הבנת רשימה:**

```python
numbers = [1, 2, 3, 4, 5]
even_squares = [number**2 for number in numbers if number % 2 == 0]
print(even_squares) # פלט: [4, 16]
```

הוספתי את התנאי `if number % 2 == 0` להבנת הרשימה כדי לסנן רק את המספרים הזוגיים.

**הבנות רשימה רב-שורתיות**

כאשר הבנות רשימה הופכות למורכבות יותר, ניתן לפצל אותן על פני מספר שורות לקריאות טובה יותר.

**דוגמה:**

```python
numbers = [1, 2, 3, 4, 5]
squared_odds = [
    number**2
    for number in numbers
    if number % 2 != 0
]
print(squared_odds) # פלט: [1, 9, 25]
```
זה הופך את הקוד לקריא יותר, במיוחד כאשר יש תנאים ארוכים או ביטויים מורכבים.

**תרגיל 1: המרת טמפרטורות**

יש לך רשימה של טמפרטורות במעלות צלזיוס, ואתה צריך להמיר אותן למעלות פרנהייט. הנוסחה להמרה היא: `F = (C * 9/5) + 32`.

הנה רשימת הטמפרטורות שלך בצלזיוס:

```python
celsius_temperatures = [0, 10, 20, 30, 40]
```

**משימה:**

1.  השתמש בהבנת רשימה כדי ליצור רשימה חדשה `fahrenheit_temperatures`, המכילה את הטמפרטורות מ-`celsius_temperatures`, שהומרו למעלות פרנהייט.
2.  הדפס את הרשימה המתקבלת `fahrenheit_temperatures` למסך.

**רמז:**

*   השתמש בנוסחה `(celsius * 9/5) + 32` כביטוי בהבנת הרשימה.
*   חשוב מהו ה-`iterable` ומהו ה-`item` במקרה זה.

נסה לכתוב את הקוד.

הפתרון שלי:
```python
# רשימת הטמפרטורות המקורית בצלזיוס
celsius_temperatures = [0, 10, 20, 30, 40]

# משתמשים בהבנת רשימה להמרה לפרנהייט
fahrenheit_temperatures = [
    (celsius * 9/5) + 32  # ביטוי: נוסחת ההמרה
    for celsius in celsius_temperatures # מעבר על איברי הרשימה
]

# מדפיסים את התוצאה
print(fahrenheit_temperatures) # פלט: [32.0, 50.0, 68.0, 86.0, 104.0]
```

**פירוט הקוד:**

1.  **`celsius_temperatures = [0, 10, 20, 30, 40]`**: זוהי הרשימה המקורית עם טמפרטורות במעלות צלזיוס.
2.  **`fahrenheit_temperatures = [...]`**: כאן אנו יוצרים רשימה חדשה באמצעות הבנת רשימה.
3.  **`(celsius * 9/5) + 32`**: זהו הביטוי שמבוצע עבור כל איבר. הוא ממיר את הטמפרטורה מצלזיוס לפרנהייט.
4.  **`for celsius in celsius_temperatures`**: זהו חלק מהבנת הרשימה שעובר על כל איבר ברשימה `celsius_temperatures`, ומקצה את ערכו למשתנה `celsius` בכל איטרציה.
5.  **`print(fahrenheit_temperatures)`**: מדפיס את רשימת הטמפרטורות בפרנהייט שהתקבלה.

**חשוב לציין:**

*   במקרה זה, `celsius_temperatures` הוא ה-`iterable`, ו-`celsius` הוא ה-`item`.
*   אני משתמש בעיצוב רב-שורתי של הבנת הרשימה כדי לשפר את קריאות הקוד.
*   בתוך הביטוי, אני מיישם ישירות את נוסחת ההמרה, תוך שימוש בערך הנוכחי של `celsius`.

**תרגיל 2: סינון והמרת מחרוזות**

יש לך רשימה של מילים, ואתה צריך לבצע את הפעולות הבאות:

1.  לסנן את המילים שמתחילות באות "a" (רישיות/קטנות אינן משנה).
2.  להמיר את כל המילים הנותרות לאותיות רישיות.

הנה רשימת המילים שלך:

```python
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]
```

**משימה:**

1.  השתמש בהבנת רשימה כדי ליצור רשימה חדשה `transformed_words`, המכילה רק את המילים מתוך הרשימה `words` שאינן מתחילות באות "a" (או "A"), ובנוסף הומרו לאותיות רישיות.
2.  הדפס את הרשימה המתקבלת `transformed_words` למסך.

**רמזים:**

*   השתמש במתודת המחרוזת `startswith()` כדי לבדוק האם מילה מתחילה באות מסוימת. אל תשכח את הרישיות! המר הכל לאותיות קטנות.
*   השתמש במתודת המחרוזת `upper()` כדי להמיר את המחרוזת לאותיות רישיות.
*   חשוב על סדר הפעולות. קודם מסננים, אחר כך ממירים.

הפתרון שלי:

```python
# רשימת המילים המקורית
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]

# משתמשים בהבנת רשימה לסינון והמרה
transformed_words = [
    word.upper() # ממירים לאותיות רישיות
    for word in words # עוברים על המילים ברשימה
    if not word.lower().startswith("a") # מסננים מילים שאינן מתחילות ב-'a'
]

# מדפיסים את התוצאה
print(transformed_words) # פלט: ['BANANA', 'KIWI', 'ORANGE']
```

**פירוט הקוד:**

1.  **`words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]`**: זוהי רשימת המילים המקורית.
2.  **`transformed_words = [...]`**: אנו יוצרים רשימה חדשה באמצעות הבנת רשימה.
3.  **`word.upper()`**: זהו הביטוי שמבוצע עבור כל מילה שעברה את הסינון. הוא ממיר את המילה לאותיות רישיות.
4.  **`for word in words`**: עוברים על כל המילים ברשימה `words`.
5.  **`if not word.lower().startswith("a")`**: זהו תנאי הסינון.
    *   `word.lower()`: תחילה אנו ממירים את המילה לאותיות קטנות, כדי שההשוואה לא תהיה תלוית רישיות.
    *   `startswith("a")`: לאחר מכן אנו בודקים האם המילה מתחילה באות "a".
    *   `not`: הופכים את התוצאה, כדי להשאיר רק את המילים ש*אינן* מתחילות באות "a".
6.  **`print(transformed_words)`**: מדפיס את רשימת המילים המומרות שהתקבלה.

**נקודות מפתח:**

*   מתודות המחרוזת `lower()`, `upper()` ו-`startswith()` בביטוי ובתנאי של הבנת הרשימה.
*   תנאי הסינון `if not word.lower().startswith("a")` מבטיח שרק מילים שאינן מתחילות ב-"a" (ללא תלות ברישיות) יכללו ברשימה החדשה.
*   קודם מיושם הסינון, ורק אחר כך ההמרה לאותיות רישיות.
*   שוב, אני משתמש בעיצוב רב-שורתי לקריאות.

**תרגיל 3: יצירת מילון מרשימת tuples**

יש לך רשימה של tuples, כאשר כל tuple מכיל שני איברים: שם ואת גיל האדם.

```python
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]
```

**משימה:**

1.  השתמש בהבנת רשימה כדי ליצור מילון `people_dict`, כאשר המפתחות יהיו שמות האנשים, והערכים – הגילאים שלהם.
2.  הדפס את המילון המתקבל `people_dict` למסך.

**רמזים:**

*   הבנות רשימה יוצרות רשימות, אך ניתן להשתמש בהן כדי ליצור רשימות של tuples, אותן ניתן להעביר לאחר מכן לקונסטרוקטור `dict()`.
*   חשוב כיצד לחלץ את השם והגיל מכל tuple.
*   זכור שמילון הוא מבנה נתונים השומר זוגות של מפתח-ערך.

**הסבר קצר:**

בפייתון ניתן ליצור מילון מרשימת tuples, כאשר כל tuple מורכב משני איברים (מפתח, ערך).
לשם כך, אני משתמש בקונסטרוקטור `dict()`. לדוגמה:
```python
my_list = [("a", 1), ("b", 2)]
my_dict = dict(my_list)
print(my_dict) # פלט: {'a': 1, 'b': 2}
```
המשימה שלך היא ליצור רשימה זו של tuples באמצעות הבנת רשימה, שתשמש לאחר מכן ליצירת המילון.

הפתרון שלי:

```python
# רשימת ה-tuples המקורית עם נתוני האנשים
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]

# אני משתמש בהבנת רשימה כדי ליצור רשימה של tuples, שאותה ממיר לאחר מכן למילון
people_dict = dict(
    [
      (name, age) # יוצרים tuple (שם, גיל)
      for name, age in people_data # עוברים על ה-tuples ב-people_data
    ]
)

# מדפיסים את המילון שהתקבל
print(people_dict) # פלט: {'Alice': 30, 'Bob': 25, 'Charlie': 35, 'David': 28}
```

**פירוט הקוד:**

1.  **`people_data = [...]`**: זוהי רשימת ה-tuples המקורית, כאשר כל tuple מכיל שם וגיל של אדם.
2.  **`people_dict = dict(...)`**: אנו משתמשים בקונסטרוקטור `dict()` כדי ליצור מילון מרשימת tuples.
3.  **`[ (name, age) for name, age in people_data ]`**: זוהי הבנת רשימה שיוצרת רשימה של tuples.
    *   `(name, age)`: זהו הביטוי שיוצר tuple משני איברים: `name` ו-`age`.
    *   `for name, age in people_data`: זהו החלק שעובר על כל ה-tuples ברשימה `people_data`. בכל איטרציה, ה-tuple נפתח לשני משתנים: `name` ו-`age`.
4.  **`print(people_dict)`**: מדפיס את המילון שנוצר.

**נקודות מפתח:**

*   הבנת רשימה יוצרת רשימה של tuples.
*   הקונסטרוקטור `dict()` ממיר רשימה של tuples למילון, תוך שימוש באיבר הראשון של ה-tuple כמפתח, והשני כערך.
*   פתיחת ה-tuples בלולאה `for name, age in people_data` הופכת את הקוד לקריא יותר.
*   הבנת רשימה בדוגמה זו משמשת להכנת הנתונים עבור המילון.
