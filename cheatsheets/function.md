## הכרזת פונקציה

פונקציה מוכרזת באמצעות מילת המפתח `def`, אחריה מופיע שם הפונקציה, רשימת פרמטרים בסוגריים עגולים ונקודתיים. גוף הפונקציה נכתב בהזחה.

```python
def название_функции(параметры):
    # פעולות
    return результат
```

### דוגמה:
```python
def сложение(a: int, b: int) -> int:
    """מחזירה את סכום שני מספרים."""
    return a + b
```

כאן:
-   `a: int` ו-`b: int` — פרמטרים של הפונקציה עם אנוטציות סוגים.
-   `-> int` — אנוטציה של סוג הערך המוחזר.

## פרמטרים של פונקציה

פונקציות יכולות לקבל פרמטרים, המייצגים נתוני קלט. הם מצוינים בסוגריים לאחר שם הפונקציה.

דוגמה עם פרמטר יחיד:
```python
def приветствие(имя: str) -> str:
    """מברכת את המשתמש בשמו."""
    return f"Привет, {имя}!"
```

### סוגי פרמטרים:
1.  **פרמטרים חובה** — חייבים להימסר בעת קריאה לפונקציה.
2.  **פרמטרים אופציונליים** — יכולים לקבל ערכי ברירת מחדל.
    ```python
    def приветствие(имя: str, возраст: int = 18) -> str:
        return f"Привет, {имя}! Тебе {возраст} лет."
    ```

## החזרת ערך

פונקציה יכולה להחזיר ערך באמצעות מילת המפתח `return`. אם `return` אינה בשימוש, הפונקציה כברירת מחדל מחזירה `None`.

דוגמה:
```python
def умножение(a: int, b: int) -> int:
    """מחזירה את מכפלת שני מספרים."""
    return a * b
```

## משתנים מקומיים וגלובליים

-   **משתנה מקומי** — הוא משתנה שקיים רק בתוך פונקציה. הוא נוצר ומושמד בכל קריאה לפונקציה.
-   **משתנה גלובלי** — הוא משתנה שזמין בכל הקוד, כולל פונקציות.

דוגמת שימוש במשתנה גלובלי:
```python
x = 10  # משתנה גלובלי

def показать_x() -> int:
    return x  # פנייה למשתנה גלובלי
```

אם יש צורך לשנות משתנה גלובלי מתוך פונקציה, יש להשתמש במילת המפתח `global`:
```python
x = 10  # משתנה גלובלי

def изменить_x() -> None:
    global x
    x = 20
```

## פונקציות מקוננות

ב-Python פונקציות יכולות להיות מקוננות, כלומר פונקציה אחת יכולה להיות מוגדרת בתוך אחרת. פונקציה מקוננת יכולה לפנות למשתנים של הפונקציה החיצונית.

דוגמה:
```python
def внешний(a: int, b: int) -> int:
    """פונקציה המשתמשת בפונקציה מקוננת לחישוב ההפרש."""
    
    def вложенный(x: int, y: int) -> int:
        """פונקציה מקוננת, המחזירה את ההפרש."""
        return x - y
    
    return вложенный(a, b)
```

## רקורסיה

רקורסיה היא כאשר פונקציה קוראת לעצמה. זה שימושי למשימות שניתן לפרק למשימות אנלוגיות קטנות יותר (למשל, עצרת).

דוגמת רקורסיה:
```python
def факториал(n: int) -> int:
    """מחשבת עצרת של מספר באמצעות רקורסיה."""
    if n == 0:
        return 1  # מקרה בסיס
    return n * факториал(n - 1)  # קריאה רקורסיבית
```

## טיפול בחריגות באמצעות `try` ו-`except`

Python מספקת מנגנון לטיפול בשגיאות באמצעות בלוקים `try` ו-`except`. קוד שעלול לגרום לשגיאה מוכנס לבלוק `try`, והשגיאות מטופלות בבלוק `except`.

דוגמת טיפול בשגיאות:
```python
def деление(a: int, b: int) -> float:
    """מחלק מספר אחד במספר אחר, תוך טיפול בשגיאות אפשריות."""
    try:
        result = a / b
    except ZeroDivisionError:
        return "Ошибка: деление на ноль"
    except Exception as e:
        return f"Произошла ошибка: {e}"
    return result
```

כאן:
-   בלוק `try` מנסה לבצע את פעולת החלוקה.
-   בלוק `except ZeroDivisionError` לוכד את שגיאת חלוקה באפס.
-   בלוק `except Exception as e` לוכד חריגות אחרות ומציג הודעת שגיאה.

## דוגמת שימוש בפונקציות

```python
# חיבור שני מספרים
print(сложение(5, 3))  # 8

# פונקציה מקוננת
print(внешний(10, 4))  # 6

# רקורסיה לחישוב עצרת
print(факториал(5))  # 120

# טיפול בחריגות בעת חלוקה
print(деление(10, 2))  # 5.0
print(деление(10, 0))  # Ошибка: деление на ноль
```
