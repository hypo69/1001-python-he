# Python. шпаргалки: ספריית `schedule`

ספריית schedule בפייתון מיועדת לתזמון פשוט ונוח של ביצוע משימות באופן קבוע, המאפשרת להגדיר באיזו תדירות יש לבצע פונקציה מסוימת (לדוגמה, כל שעה, יום, שבוע).

```bash
pip install schedule
```

### שימוש
כדי לתזמן משימה, יש להגדיר פונקציה ולהשתמש ב-`schedule.every()` כדי להקצות מרווח ביצוע.

#### דוגמה: הפעלת משימה כל 10 שניות
```python
import schedule
import time

def job():
    print("Задача выполнена!") # המשימה בוצעה!

# Запланировать выполнение задачи каждые 10 секунд
# תזמון ביצוע המשימה כל 10 שניות
schedule.every(10).seconds.do(job)

while True:
    schedule.run_pending()
    time.sleep(1)  # Предотвращает чрезмерное использование процессора
                   # מונע שימוש מוגזם במעבד
```

סקריפט זה מבצע את הפונקציה `job()` כל 10 שניות. לולאת `while` בודקת כל הזמן אם יש משימות מתוזמנות ומבצעת אותן בזמן הנכון.

### הפעלת משימות בזמן ספציפי
ניתן לתזמן ביצוע משימות בזמן מוגדר מראש, תוך שימוש בשיטה `.at()` ופרמטרים אחרים של מרווח:

```python
# Запустить задачу каждый день в 9:00 утра
# הפעלת משימה כל יום בשעה 9:00 בבוקר
schedule.every().day.at("09:00").do(job)

# Запустить задачу каждый понедельник в 12:30 дня
# הפעלת משימה בכל יום שני בשעה 12:30 בצהריים
schedule.every().monday.at("12:30").do(job)
```

### ניהול משימות מרובות
ניתן לתזמן מספר משימות, על ידי הגדרת פונקציות נפרדות והקצאת מרווחי זמן שונים עבורן.

```python
def task1():
    print("Задача 1 выполнена!") # משימה 1 בוצעה!

def task2():
    print("Задача 2 выполнена!") # משימה 2 בוצעה!

schedule.every(5).seconds.do(task1)
schedule.every().hour.do(task2)

while True:
    schedule.run_pending()
    time.sleep(1)
```

### הסרת משימות מתוזמנות
כדי להסיר משימה, יש להשתמש בשיטה `schedule.clear()`. ניתן להסיר את כל המשימות או לציין משימה ספציפית באמצעות תגיות.

```python
# Добавление задач с тегами
# הוספת משימות עם תגיות
schedule.every(5).seconds.do(task1).tag("fast_task")
schedule.every().hour.do(task2).tag("hourly_task")

# Удалить конкретную задачу по тегу
# הסרת משימה ספציפית לפי תגית
schedule.clear("fast_task")

# Удалить все запланированные задачи
# הסרת כל המשימות המתוזמנות
schedule.clear()
```

### הפעלת המתזמן ברקע
עבור יישומי ממשק גרפי או ביצוע רקע מתמשך, ניתן להפעיל את המתזמן בתהליכון (thread) נפרד.

```python
import threading

def run_scheduler():
    while True:
        schedule.run_pending()
        time.sleep(1)

scheduler_thread = threading.Thread(target=run_scheduler, daemon=True)
scheduler_thread.start()
```
