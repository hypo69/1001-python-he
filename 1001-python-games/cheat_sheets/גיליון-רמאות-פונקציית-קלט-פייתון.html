# הפונקציה <span dir="ltr"><code>input()code>span>

הפונקציה <span dir="ltr"><code>input()code>span> בפייתון משמשת לקבלת נתונים מהמשתמש באמצעות קלט טקסט. היא משהה את ביצוע התוכנית עד שהמשתמש יזין נתונים וילחץ Enter. לאחר מכן היא מחזירה את הערך שהוזן כמחרוזת.

תחביר


input([prompt])

- <span dir="ltr"><code>promptcode>span> (אופציונלי): מחרוזת המוצגת למשתמש לפני הקלט. זו יכולה להיות הודעה עם הוראות, לדוגמה: <span dir="ltr"><code>"הכנס את שמך: "code>span>.

דוגמה לשימוש


# בקשת שם מהמשתמש
name = input('הכנס את שמך: ')
print(f'שלום, {name}!')

**תוצאת ביצוע:**


הכנס את שמך: אלכס
שלום, אלכס!

מאפיינים

  • **מחזירה מחרוזת**

    • כל הנתונים שהוזנו באמצעות <span dir="ltr"><code>input()code>span> מפורשים כמחרוזות. אם נדרש מספר, יש להמיר אותו:

    ```python

    age = int(input('הכנס את גילך: '))

    print(f'גילך: {age}')

    ```

  • **טיפול בשגיאות**

    • כדי למנוע שגיאות במהלך ההמרה (לדוגמה, אם המשתמש הזין טקסט במקום מספר), ניתן להשתמש בבלוק <span dir="ltr"><code>try-exceptcode>span>:

    ```python

    try:

    number = int(input('הכנס מספר: '))

    print(f'הכנסת את המספר {number}')

    except ValueError:

    print('שגיאה: יש להזין מספר.')

    ```

  • **שימוש בלולאות**

    • לעתים קרובות <span dir="ltr"><code>input()code>span> משמש בלולאות לבקשת נתונים חוזרת:

    ```python

    while True:

    text = input('הכנס משהו (או "יציאה" לסיום): ')

    if text.lower() == 'יציאה':

    print('סיום התוכנית.')

    break

    print(f'הכנסת: {text}')

    ```

    טיפים למתחילים

    - ודא שסוג הנתונים תואם את הציפיות שלך (לדוגמה, המר קלט למספר אם נדרש).

    - תמיד בדוק נתונים כדי למנוע שגיאות קלט.

    - השתמש בהודעות <span dir="ltr"><code>promptcode>span> ברורות ותמציתיות כדי שהמשתמש יבין מה נדרש ממנו.
