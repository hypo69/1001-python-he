**מדריך פקודות PowerShell**

**1. יסודות ניווט ועבודה עם קבצים וספריות**

  • **<span dir="ltr"><code>Get-ChildItemcode>span> (או <span dir="ltr"><code>gcicode>span>, <span dir="ltr"><code>lscode>span>, <span dir="ltr"><code>dircode>span>)**: מקבל רשימה של קבצים ותתי-ספריות במיקום שצוין.

    • * **תחביר**: <span dir="ltr"><code>Get-ChildItem [נתיב] [פרמטרים]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-Pathcode>span>: מציין את הנתיב לספרייה.

    * <span dir="ltr"><code>-Includecode>span>: מסנן לפי שם קובץ (עם תווים כלליים <span dir="ltr"><code>*code>span> ו-<span dir="ltr"><code>?code>span>).

    * <span dir="ltr"><code>-Excludecode>span>: מוציא קבצים לפי שם.

    * <span dir="ltr"><code>-Recursecode>span>: מציג קבצים ותיקיות בכל תתי-הספריות.

    * <span dir="ltr"><code>-Forcecode>span>: הצג קבצים מוסתרים.

    * <span dir="ltr"><code>-Filecode>span>: הצג קבצים בלבד.

    * <span dir="ltr"><code>-Directorycode>span>: הצג תיקיות בלבד.

    * **דוגמאות:**

    * <span dir="ltr"><code>Get-ChildItemcode>span>: רשימת קבצים ותיקיות בספרייה הנוכחית.

    * <span dir="ltr"><code>Get-ChildItem -Path C:\Users\User\Documentscode>span>: רשימת קבצים ותיקיות ב-<span dir="ltr"><code>C:\Users\User\Documentscode>span>.

    * <span dir="ltr"><code>Get-ChildItem -Include *.txtcode>span>: רשימת קבצי טקסט בלבד בספרייה הנוכחית.

    * <span dir="ltr"><code>Get-ChildItem -Path C:\ -Recurse -Directorycode>span>: הצג את כל הספריות בכונן C.

    * <span dir="ltr"><code>Get-ChildItem -Forcecode>span>: הצג את כל הקבצים, כולל מוסתרים.

  • **<span dir="ltr"><code>Set-Locationcode>span> (או <span dir="ltr"><code>slcode>span>, <span dir="ltr"><code>cdcode>span>)**: משנה את הספרייה הנוכחית.

    • * **תחביר**: <span dir="ltr"><code>Set-Location [נתיב]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>Set-Location C:\Windowscode>span>: עבור לספריית <span dir="ltr"><code>C:\Windowscode>span>.

    * <span dir="ltr"><code>Set-Location ..code>span>: עבור לספריית האב.

    * <span dir="ltr"><code>Set-Location /code>span> - עבור לשורש הכונן.

  • **<span dir="ltr"><code>New-Itemcode>span>**: יוצר קובץ או ספרייה חדשים.

    • * **תחביר**: <span dir="ltr"><code>New-Item -Path [נתיב] -ItemType [סוג] -Name [שם]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-ItemTypecode>span>: <span dir="ltr"><code>filecode>span> או <span dir="ltr"><code>directorycode>span>.

    * <span dir="ltr"><code>-Namecode>span>: שם הפריט החדש.

    * <span dir="ltr"><code>-Valuecode>span>: תוכן הקובץ.

    * **דוגמאות:**

    * <span dir="ltr"><code>New-Item -ItemType directory -Name NewFoldercode>span>: צור תיקייה <span dir="ltr"><code>NewFoldercode>span>.

    * <span dir="ltr"><code>New-Item -ItemType file -Name myfile.txtcode>span>: צור קובץ ריק <span dir="ltr"><code>myfile.txtcode>span>.

    * <span dir="ltr"><code>New-Item -ItemType file -Name myfile.txt -Value "Hello, world!"code>span>: צור קובץ <span dir="ltr"><code>myfile.txtcode>span> עם תוכן.

  • **<span dir="ltr"><code>Remove-Itemcode>span> (או <span dir="ltr"><code>rmcode>span>, <span dir="ltr"><code>delcode>span>, <span dir="ltr"><code>erasecode>span>)**: מוחק קבצים וספריות.

    • * **תחביר:** <span dir="ltr"><code>Remove-Item [נתיב] [פרמטרים]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-Recursecode>span>: מחק את כל תתי-הספריות.

    * <span dir="ltr"><code>-Forcecode>span>: מחיקה כפויה (כולל קבצים "לקריאה בלבד" וספריות).

    * <span dir="ltr"><code>-Confirmcode>span> - בקש אישור לכל מחיקה.

    * **דוגמאות:**

    * <span dir="ltr"><code>Remove-Item myfile.txtcode>span>: מחק את הקובץ <span dir="ltr"><code>myfile.txtcode>span>.

    * <span dir="ltr"><code>Remove-Item -Path C:\Temp -Recurse -Forcecode>span>: מחק את תיקיית <span dir="ltr"><code>C:\Tempcode>span> עם כל התיקיות והקבצים המקוננים.

  • **<span dir="ltr"><code>Copy-Itemcode>span>**: מעתיק קבצים וספריות.

    • * **תחביר**: <span dir="ltr"><code>Copy-Item [נתיב_מקור] [נתיב_יעד] [פרמטרים]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-Recursecode>span>: העתק את כל תתי-הספריות.

    * <span dir="ltr"><code>-Forcecode>span>: דרוס קבצים קיימים ללא בקשה.

    * **דוגמאות:**

    * <span dir="ltr"><code>Copy-Item -Path myfile.txt -Destination mycopy.txtcode>span>: העתק את הקובץ <span dir="ltr"><code>myfile.txtcode>span> ל-<span dir="ltr"><code>mycopy.txtcode>span>.

    * <span dir="ltr"><code>Copy-Item -Path C:\Source -Destination D:\Backup -Recursecode>span>: העתק את תיקיית <span dir="ltr"><code>C:\Sourcecode>span> לכל תתי-הספריות בתיקיית <span dir="ltr"><code>D:\Backupcode>span>.

  • **<span dir="ltr"><code>Move-Itemcode>span>**: מעביר קבצים וספריות.

    • * **תחביר**: <span dir="ltr"><code>Move-Item [נתיב_מקור] [נתיב_יעד] [פרמטרים]code>span>

    * <span dir="ltr"><code>-Forcecode>span> - העבר בכוח ודרוס.

    * **דוגמאות:**

    * <span dir="ltr"><code>Move-Item -Path myfile.txt -Destination D:\Documentscode>span>: העבר את הקובץ <span dir="ltr"><code>myfile.txtcode>span> לתיקיית <span dir="ltr"><code>D:\Documentscode>span>.

    * <span dir="ltr"><code>Move-Item -Path "C:\MyFolder" -Destination "D:\" -Forcecode>span>: העבר את תיקיית C:\MyFolder ל-D:\ בכוח, גם אם קיימת שם תיקייה באותו שם.

  • **<span dir="ltr"><code>Rename-Itemcode>span>**: משנה שם של קובץ או ספרייה.

    • * **תחביר**: <span dir="ltr"><code>Rename-Item -Path [נתיב] -NewName [שם_חדש]code>span>

    * **דוגמה:**

    * <span dir="ltr"><code>Rename-Item -Path myfile.txt -NewName newfile.txtcode>span>: שנה את שם הקובץ <span dir="ltr"><code>myfile.txtcode>span> ל-<span dir="ltr"><code>newfile.txtcode>span>.

  • **<span dir="ltr"><code>Get-Contentcode>span> (או <span dir="ltr"><code>gccode>span>)**: מציג או מקבל את תוכן הקובץ.

    • * **תחביר**: <span dir="ltr"><code>Get-Content [נתיב]code>span>

    * **דוגמה:**

    * <span dir="ltr"><code>Get-Content myfile.txtcode>span>: הצג את תוכן הקובץ <span dir="ltr"><code>myfile.txtcode>span>.

  • **<span dir="ltr"><code>Set-Contentcode>span>**: מחליף או יוצר את תוכן הקובץ.

    • * **תחביר:** <span dir="ltr"><code>Set-Content [נתיב] [פרמטרים]code>span>

    * <span dir="ltr"><code>-valuecode>span> - טקסט לכתיבה.

    * **דוגמה:** <span dir="ltr"><code>Set-Content myfile.txt "טקסט חדש"code>span> - החלף את תוכן הקובץ <span dir="ltr"><code>myfile.txtcode>span>.

  • **<span dir="ltr"><code>Add-Contentcode>span>**: מוסיף תוכן לסוף הקובץ.

    • * **תחביר:** <span dir="ltr"><code>Add-Content [נתיב] [פרמטרים]code>span>

    * <span dir="ltr"><code>-valuecode>span> - טקסט להוספה.

    * **דוגמה:** <span dir="ltr"><code>Add-Content myfile.txt "טקסט נוסף"code>span> - הוסף טקסט לסוף <span dir="ltr"><code>myfile.txtcode>span>.

    **2. ניהול תהליכים:**

  • **<span dir="ltr"><code>Get-Processcode>span> (או <span dir="ltr"><code>gpscode>span>)**: מקבל רשימה של תהליכים פועלים.

    • * **תחביר**: <span dir="ltr"><code>Get-Process [פרמטרים]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-Namecode>span>: סנן לפי שם תהליך.

    * <span dir="ltr"><code>-Idcode>span>: סנן לפי מזהה תהליך.

    * <span dir="ltr"><code>-IncludeUserNamecode>span>: הצג את המשתמש שהפעיל את התהליך.

    * **דוגמאות:**

    * <span dir="ltr"><code>Get-Processcode>span>: רשימת כל התהליכים הפועלים.

    * <span dir="ltr"><code>Get-Process -Name notepadcode>span>: רשימת תהליכים בשם <span dir="ltr"><code>notepadcode>span>.

    * <span dir="ltr"><code>Get-Process -IncludeUserNamecode>span>: רשימת כל התהליכים הפועלים עם משתמשים.

  • **<span dir="ltr"><code>Stop-Processcode>span>**: מסיים תהליך.

    • * **תחביר**: <span dir="ltr"><code>Stop-Process [פרמטרים]code>span>

    * <span dir="ltr"><code>-Idcode>span> - ציין מזהה תהליך.

    * <span dir="ltr"><code>-Namecode>span> - ציין שם תהליך.

    * <span dir="ltr"><code>-Forcecode>span> - סיים תהליך בכוח.

    * **דוגמאות:**

    * <span dir="ltr"><code>Stop-Process -Name notepadcode>span>: סיים את כל תהליכי <span dir="ltr"><code>notepadcode>span>.

    * <span dir="ltr"><code>Stop-Process -Id 1234code>span> : סיים תהליך עם מזהה 1234.

    * <span dir="ltr"><code>Stop-Process -Name chrome -Forcecode>span> : סיים בכוח את כל תהליכי <span dir="ltr"><code>chromecode>span>.

    **3. ניהול שירותים:**

  • **<span dir="ltr"><code>Get-Servicecode>span>**: מקבל רשימה של שירותים.

    • * **תחביר**: <span dir="ltr"><code>Get-Service [פרמטרים]code>span>

    * **פרמטרים עיקריים:**

    * <span dir="ltr"><code>-Namecode>span>: הצג רק שירותים עם השם שצוין.

    * <span dir="ltr"><code>-DisplayNamecode>span>: הצג רק שירותים עם השם המוצג שצוין.

    * <span dir="ltr"><code>-Statuscode>span>: סנן לפי סטטוס (Running, Stopped).

    * **דוגמאות:**

    * <span dir="ltr"><code>Get-Servicecode>span>: רשימת כל השירותים.

    * <span dir="ltr"><code>Get-Service -Name Spoolercode>span>: הצג את שירות Spooler.

    * <span dir="ltr"><code>Get-Service -Status Runningcode>span>: הצג שירותים פועלים.

  • **<span dir="ltr"><code>Start-Servicecode>span>**: מפעיל שירות.

    • * **תחביר**: <span dir="ltr"><code>Start-Service [שם_שירות]code>span>

    * **דוגמה:** <span dir="ltr"><code>Start-Service Spoolercode>span> - הפעל את שירות Spooler.

  • **<span dir="ltr"><code>Stop-Servicecode>span>**: עוצר שירות.

    • * **תחביר**: <span dir="ltr"><code>Stop-Service [שם_שירות]code>span>

    * <span dir="ltr"><code>-Forcecode>span> - עצור שירות בכוח.

    * **דוגמה:** <span dir="ltr"><code>Stop-Service Spoolercode>span>: עצור את שירות Spooler.

    * <span dir="ltr"><code>Stop-Service Spooler -Forcecode>span> - עצור את שירות Spooler בכוח.

  • **<span dir="ltr"><code>Restart-Servicecode>span>**: מפעיל מחדש שירות.

    • * **תחביר:** <span dir="ltr"><code>Restart-Service [שם_שירות]code>span>

    * **דוגמה:** <span dir="ltr"><code>Restart-Service Spoolercode>span> - הפעל מחדש את שירות Spooler.

    **4. עבודה עם רשת**

  • **<span dir="ltr"><code>Test-NetConnectioncode>span>**: בודק חיבור רשת.

    * **דוגמאות:**

    * <span dir="ltr"><code>Test-NetConnection google.com -Port 80code>span>: בדוק חיבור ל-google.com בפורט 80.

  • **<span dir="ltr"><code>Get-NetIPConfigurationcode>span>**: מקבל תצורת רשת.

    • * **תחביר**: <span dir="ltr"><code>Get-NetIPConfigurationcode>span>

    * **דוגמה:**

    * <span dir="ltr"><code>Get-NetIPConfigurationcode>span>: הצג תצורת רשת.

  • **<span dir="ltr"><code>Resolve-DnsNamecode>span>**: שולח שאילתת מידע DNS.

    • * **תחביר**: <span dir="ltr"><code>Resolve-DnsName [שם_מארח]code>span>

    * **דוגמה:** <span dir="ltr"><code>Resolve-DnsName google.comcode>span>: שלח שאילתת מידע DNS עבור <span dir="ltr"><code>google.comcode>span>.

    **5. עבודה עם הרישום**

  • **<span dir="ltr"><code>Get-ItemPropertycode>span>**: מקבל ערך מאפיין מהרישום.

    • * **תחביר**: <span dir="ltr"><code>Get-ItemProperty -Path [נתיב_ברישום]code>span>

    * **דוגמה:** <span dir="ltr"><code>Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion"code>span>

  • **<span dir="ltr"><code>Set-ItemPropertycode>span>**: מגדיר ערך מאפיין ברישום.

    • * **תחביר**: <span dir="ltr"><code>Set-ItemProperty -Path [נתיב_ברישום] -Name [שם_מאפיין] -Value [ערך]code>span>

    * **דוגמה:** <span dir="ltr"><code>Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name Wallpaper -Value "C:\image.jpg"code>span>

  • **<span dir="ltr"><code>Get-Variablecode>span>**: מציג משתנים נוכחיים.

    • * **תחביר**: <span dir="ltr"><code>Get-Variablecode>span>

  • **<span dir="ltr"><code>Get-Aliascode>span>**: מציג כינויים לפקודות.

    **הערות:**

  • פקודות <span dir="ltr"><code>PowerShellcode>span> (cmdlets) בדרך כלל בעלות צורה של <span dir="ltr"><code>פועל-שם עצםcode>span> (לדוגמה, <span dir="ltr"><code>Get-Processcode>span>, <span dir="ltr"><code>Set-Locationcode>span>).
  • <span dir="ltr"><code>PowerShellcode>span> אינו רגיש לרישיות, כך שתוכל לכתוב פקודות כ-<span dir="ltr"><code>Get-ChildItemcode>span> או <span dir="ltr"><code>get-childitemcode>span>.
  • <span dir="ltr"><code>PowerShellcode>span> עובד עם אובייקטים, כך שתוכל להשתמש באופרטור <span dir="ltr"><code>|code>span> כדי להעביר את הפלט של פקודה אחת לקלט של אחרת (לדוגמה, <span dir="ltr"><code>Get-Process | Sort-Object -Property CPUcode>span>).
  • פקודות רבות תומכות בשימוש בתווים כלליים (*) לעבודה עם מספר קבצים (לדוגמה <span dir="ltr"><code>Get-ChildItem *.txtcode>span>).
  • עבור עבודה עם פקודות מסוימות נדרשות הרשאות מנהל מערכת.
