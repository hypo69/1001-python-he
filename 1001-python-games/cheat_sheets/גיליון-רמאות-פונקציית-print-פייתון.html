# הפונקציה <span dir="ltr"><code>print()code>span>

הפונקציה <span dir="ltr"><code>print()code>span> בפייתון משמשת להדפסת מידע לקונסולה. זוהי אחת הפונקציות הפשוטות והנפוצות ביותר, במיוחד לצורך דיבוג והצגת נתונים. נבחן את ההיבטים העיקריים של פעולתה.

תחביר


print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)

פרמטרים:

  • **<span dir="ltr"><code>*objectscode>span>**:

    • - רשימת אובייקטים להדפסה. ניתן להעביר ערך אחד או יותר, מופרדים בפסיקים.

    - דוגמה:

    ```python

    print('שלום', 'עולם')

    ```

    תוצאה: <span dir="ltr"><code>שלום עולםcode>span>

  • **<span dir="ltr"><code>sepcode>span>** (ברירת מחדל <span dir="ltr"><code>' 'code>span>):

    • - מפריד בין אובייקטים, אם הועברו מספר אובייקטים.

    - דוגמה:

    ```python

    print('שלום', 'עולם', sep=' - ')

    ```

    תוצאה: <span dir="ltr"><code>שלום - עולםcode>span>

  • **<span dir="ltr"><code>endcode>span>** (ברירת מחדל <span dir="ltr"><code>'\n'code>span>):

    • - מציין מה יתווסף בסוף השורה. ברירת המחדל היא מעבר שורה.

    - דוגמה:

    ```python

    print('שלום', end='!')

    ```

    תוצאה: <span dir="ltr"><code>שלום!code>span>

  • **<span dir="ltr"><code>filecode>span>**:

    • - זרם שאליו יופנה הפלט (ברירת מחדל <span dir="ltr"><code>sys.stdoutcode>span> — פלט סטנדרטי).

    - דוגמה: פלט לקובץ.

    ```python

    with open('output.txt', 'w') as f:

    print('שלום, קובץ!', file=f)

    ```

  • **<span dir="ltr"><code>flushcode>span>** (ברירת מחדל <span dir="ltr"><code>Falsecode>span>):

    • - אם מוגדר ל-<span dir="ltr"><code>Truecode>span>, מאלץ את מאגר הפלט להתרוקן.

    ---

    דוגמאות שימוש

    מחרוזת פשוטה

    
print('Hello, world!')

    תוצאה: <span dir="ltr"><code>Hello, world!code>span>

    הדפסת מספר ערכים

    
name = 'אנה'
age = 25
print('שם:', name, ', גיל:', age)

    תוצאה: <span dir="ltr"><code>שם: אנה , גיל: 25code>span>

    הגדרת מפריד

    
print(1, 2, 3, sep=' -> ')

    תוצאה: <span dir="ltr"><code>1 -> 2 -> 3code>span>

    הגדרת סיום שורה

    
for i in range(3):
    print(i, end=' ')

    תוצאה: <span dir="ltr"><code>0 1 2code>span>

    שימוש ב-f-strings עם הדפסת שם משתנה

    החל מפייתון 3.8, ניתן להשתמש ב-f-strings להדפסת ערכי משתנים עם שמותיהם בפורמט <span dir="ltr"><code>שם=ערךcode>span>. זה נוח לדיבוג, מכיוון שזה מאפשר לראות מיד איזה משתנה ואיזה ערך מודפס.

    
name = 'איתן'
age = 30
print(f'{name=}, {age=}')

    תוצאה: <span dir="ltr"><code>name='איתן', age=30code>span>

    ---

    טיפים שימושיים למתחילים

  • **דיבוג קוד**:

    • השתמש ב-<span dir="ltr"><code>print()code>span> כדי לבדוק ערכי משתנים:

    ```python

    x = 10

    y = 20

    print('סכום:', x + y)

    ```

  • **פלט מעוצב**:

    • להדפסת מחרוזות עם החלפת ערכים עדיף להשתמש בעיצוב:

    ```python

    name = 'איתן'

    age = 30

    print(f'שמי {name}, אני בן {age} שנים.')

    ```

  • **רישום (Logging)**:

    • בפרויקטים גדולים עדיף להשתמש במודול <span dir="ltr"><code>loggingcode>span> לניהול פלט, אך בשלב ההתחלתי <span dir="ltr"><code>print()code>span> עוזר להדפיס נתונים במהירות.
