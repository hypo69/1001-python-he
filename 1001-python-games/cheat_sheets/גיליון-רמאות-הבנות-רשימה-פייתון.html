**מהן הבנות רשימה?**

בצורתן הפשוטה ביותר, הבנת רשימה היא דרך קומפקטית ליצור רשימה חדשה על בסיס רשימה קיימת (או כל אובייקט ניתן לאיטרציה). במקום להשתמש בלולאת <span dir="ltr"><code>forcode>span> מסורתית ולהוסיף אלמנטים לרשימה אחד אחד, ניתן לעשות זאת בשורת קוד אחת.

  • **תמציתיות:** הן מאפשרות לכתוב פחות קוד.
  • **קריאות:** כאשר תתרגל לתחביר שלהן, תמצא אותן קלות יותר להבנה מאשר לולאות <span dir="ltr"><code>forcode>span> מקבילות.
  • **ביצועים:** במקרים מסוימים, הבנות רשימה יכולות להיות מהירות יותר מלולאות <span dir="ltr"><code>forcode>span>.

    • הנה התחביר הבסיסי של הבנת רשימה:

    
new_list = [expression for item in iterable]
  • **<span dir="ltr"><code>expressioncode>span>**: ביטוי המגדיר כיצד יחושב כל אלמנט ברשימה החדשה. זה יכול להיות כל דבר: ערך פשוט, פעולה מתמטית, קריאה לפונקציה.
  • **<span dir="ltr"><code>itemcode>span>**: משתנה המקבל את הערך של כל אלמנט מתוך <span dir="ltr"><code>iterablecode>span> בתורו.
  • **<span dir="ltr"><code>iterablecode>span>**: אובייקט ניתן לאיטרציה, לדוגמה, רשימה, טאפל, מחרוזת או תוצאת הפעולה <span dir="ltr"><code>range()code>span>.

    • **דוגמה:**

    נניח שיש לך רשימת מספרים, ואתה רוצה ליצור רשימה חדשה המכילה את ריבועי המספרים הללו.

    **ללא הבנת רשימה:**

    
numbers = [1, 2, 3, 4, 5]
squares = []
for number in numbers:
    squares.append(number**2)
print(squares)  # פלט: [1, 4, 9, 16, 25]

    **עם הבנת רשימה:**

    
numbers = [1, 2, 3, 4, 5]
squares = [number**2 for number in numbers]
print(squares) # פלט: [1, 4, 9, 16, 25]

    הבנת רשימה הופכת את הקוד לקצר וברור יותר!

    **הוספת תנאי**

    הבנות רשימה מאפשרות לך גם להוסיף תנאים לבחירת אלמנטים שיכללו ברשימה החדשה.

    
new_list = [expression for item in iterable if condition]
  • **<span dir="ltr"><code>conditioncode>span>**: תנאי שחייב להתקיים כדי שאלמנט ייכלל ברשימה החדשה.

    • **דוגמה:**

    ניצור רשימת ריבועים של מספרים זוגיים בלבד מתוך רשימת <span dir="ltr"><code>numberscode>span> המקורית שלנו.

    **ללא הבנת רשימה:**

    
numbers = [1, 2, 3, 4, 5]
even_squares = []
for number in numbers:
    if number % 2 == 0:
        even_squares.append(number**2)
print(even_squares) # פלט: [4, 16]

    **עם הבנת רשימה:**

    
numbers = [1, 2, 3, 4, 5]
even_squares = [number**2 for number in numbers if number % 2 == 0]
print(even_squares) # פלט: [4, 16]

    הוספתי <span dir="ltr"><code>if number % 2 == 0code>span> להבנת הרשימה כדי לסנן רק מספרים זוגיים.

    **הבנות רשימה מרובות שורות**

    כאשר הבנות רשימה הופכות מורכבות יותר, ניתן לפצל אותן למספר שורות לקריאות טובה יותר.

    **דוגמה:**

    
numbers = [1, 2, 3, 4, 5]
squared_odds = [
    number**2
    for number in numbers
    if number % 2 != 0
]
print(squared_odds) # פלט: [1, 9, 25]

    זה הופך את הקוד למובן יותר, במיוחד כאשר יש לך תנאים ארוכים או ביטויים מורכבים.

    **תרגיל 1: המרת טמפרטורות**

    יש לך רשימת טמפרטורות בצלזיוס, ואתה צריך להמיר אותן לפרנהייט. הנוסחה להמרה היא: <span dir="ltr"><code>F = (C * 9/5) + 32code>span>.

    הנה רשימת הטמפרטורות שלך בצלזיוס:

    
celsius_temperatures = [0, 10, 20, 30, 40]

    **משימה:**

  • באמצעות הבנת רשימה, צור רשימה חדשה <span dir="ltr"><code>fahrenheit_temperaturescode>span> המכילה טמפרטורות מ-<span dir="ltr"><code>celsius_temperaturescode>span>, שהומרו לפרנהייט.
  • הדפס את רשימת <span dir="ltr"><code>fahrenheit_temperaturescode>span> שהתקבלה למסך.

    • **רמז:**

  • השתמש בנוסחה <span dir="ltr"><code>(celsius * 9/5) + 32code>span> כביטוי בהבנת הרשימה.
  • חשוב מהו <span dir="ltr"><code>iterablecode>span> ומהו <span dir="ltr"><code>itemcode>span> במקרה זה.

    • נסה לכתוב את הקוד.

    הפתרון שלי:

    
# רשימת הטמפרטורות המקורית בצלזיוס
celsius_temperatures = [0, 10, 20, 30, 40]

# נשתמש בהבנת רשימה להמרה לפרנהייט
fahrenheit_temperatures = [
    (celsius * 9/5) + 32  # ביטוי: נוסחת המרה
    for celsius in celsius_temperatures # איטרציה על אלמנטים ברשימה
]

# הדפס את התוצאה
print(fahrenheit_temperatures) # פלט: [32.0, 50.0, 68.0, 86.0, 104.0]

    **פירוק קוד:**

  • **<span dir="ltr"><code>celsius_temperatures = [0, 10, 20, 30, 40]code>span>**: זוהי רשימת הטמפרטורות המקורית בצלזיוס.
  • **<span dir="ltr"><code>fahrenheit_temperatures = [...]code>span>**: כאן אנו יוצרים רשימה חדשה באמצעות הבנת רשימה.
  • **<span dir="ltr"><code>(celsius * 9/5) + 32code>span>**: זהו הביטוי המבוצע עבור כל אלמנט. הוא ממיר את הטמפרטורה מצלזיוס לפרנהייט.
  • **<span dir="ltr"><code>for celsius in celsius_temperaturescode>span>**: חלק זה של הבנת הרשימה עובר על כל אלמנט ברשימת <span dir="ltr"><code>celsius_temperaturescode>span>, ומקצה את ערכו למשתנה <span dir="ltr"><code>celsiuscode>span> בכל איטרציה.
  • **<span dir="ltr"><code>print(fahrenheit_temperatures)code>span>**: מדפיס את רשימת הטמפרטורות שהתקבלה בפרנהייט.

    • **הערות חשובות:**

  • במקרה זה, <span dir="ltr"><code>celsius_temperaturescode>span> הוא ה-<span dir="ltr"><code>iterablecode>span>, ו-<span dir="ltr"><code>celsiuscode>span> הוא ה-<span dir="ltr"><code>itemcode>span>.
  • אני משתמש בעיצוב מרובה שורות עבור הבנת הרשימה כדי לשפר את קריאות הקוד.
  • בתוך הביטוי, אני מיישם ישירות את נוסחת ההמרה באמצעות הערך הנוכחי של <span dir="ltr"><code>celsiuscode>span>.

    • **תרגיל 2: סינון והמרת מחרוזות**

    יש לך רשימת מילים, ואתה צריך לעשות את הדברים הבאים:

  • לסנן מילים שמתחילות באות "a" (לא תלוי רישיות).
  • להמיר את כל המילים הנותרות לאותיות גדולות.

    • הנה רשימת המילים שלך:

    
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]

    **משימה:**

  • באמצעות הבנת רשימה, צור רשימה חדשה <span dir="ltr"><code>transformed_wordscode>span> המכילה רק את המילים מרשימת <span dir="ltr"><code>wordscode>span> שאינן מתחילות באות "a" (או "A"), והן מומרות לאותיות גדולות.
  • הדפס את רשימת <span dir="ltr"><code>transformed_wordscode>span> שהתקבלה למסך.

    • **רמזים:**

  • השתמש בשיטת המחרוזת <span dir="ltr"><code>startswith()code>span> כדי לבדוק אם מילה מתחילה באות מסוימת. אל תשכח את הרישיות! המר הכל לאותיות קטנות.
  • השתמש בשיטת המחרוזת <span dir="ltr"><code>upper()code>span> כדי להמיר מחרוזת לאותיות גדולות.
  • חשוב על סדר הפעולות. תחילה סנן, אחר כך המר.

    • הפתרון שלי:

    
# רשימת המילים המקורית
words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]

# נשתמש בהבנת רשימה לסינון והמרה
transformed_words = [
    word.upper() # המר לאותיות גדולות
    for word in words # עבר על מילים ברשימה
    if not word.lower().startswith("a") # סנן מילים שאינן מתחילות ב-'a'
]

# הדפס את התוצאה
print(transformed_words) # פלט: ['BANANA', 'KIWI', 'ORANGE']

    **פירוק קוד:**

  • **<span dir="ltr"><code>words = ["apple", "banana", "apricot", "kiwi", "Avocado", "orange"]code>span>**: זוהי רשימת המילים המקורית.
  • **<span dir="ltr"><code>transformed_words = [...]code>span>**: יוצרים רשימה חדשה באמצעות הבנת רשימה.
  • **<span dir="ltr"><code>word.upper()code>span>**: זהו הביטוי המבוצע עבור כל מילה מסוננת. הוא ממיר את המילה לאותיות גדולות.
  • **<span dir="ltr"><code>for word in wordscode>span>**: עובר על כל המילים ברשימת <span dir="ltr"><code>wordscode>span>.
  • **<span dir="ltr"><code>if not word.lower().startswith("a")code>span>**: זהו תנאי הסינון.

    • * <span dir="ltr"><code>word.lower()code>span>: תחילה אנו ממירים את המילה לאותיות קטנות כדי שההשוואה לא תהיה תלוית רישיות.

    * <span dir="ltr"><code>startswith("a")code>span>: לאחר מכן אנו בודקים אם המילה מתחילה באות "a".

    * <span dir="ltr"><code>notcode>span>: הופכים את התוצאה כדי לשמור רק את המילים ש*אינן* מתחילות באות "a".

  • **<span dir="ltr"><code>print(transformed_words)code>span>**: מדפיס את רשימת המילים שהומרו.

    • **נקודות מפתח:**

  • שיטות מחרוזת <span dir="ltr"><code>lower()code>span>, <span dir="ltr"><code>upper()code>span>, ו-<span dir="ltr"><code>startswith()code>span> בביטוי ובתנאי של הבנת הרשימה.
  • תנאי הסינון <span dir="ltr"><code>if not word.lower().startswith("a")code>span> מבטיח שרק מילים שאינן מתחילות ב-"a" (ללא תלות ברישיות) ייכללו ברשימה החדשה.
  • הסינון מיושם תחילה, ולאחר מכן ההמרה לאותיות גדולות.
  • שוב, אני משתמש בעיצוב מרובה שורות לקריאות.

    • **תרגיל 3: יצירת מילון מרשימת טאפלים**

    יש לך רשימת טאפלים, כאשר כל טאפל מכיל שני אלמנטים: שם וגיל של אדם.

    
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]

    **משימה:**

  • באמצעות הבנת רשימה, צור מילון <span dir="ltr"><code>people_dictcode>span>, שבו המפתחות יהיו שמות האנשים, והערכים — גילם.
  • הדפס את המילון <span dir="ltr"><code>people_dictcode>span> למסך.

    • **רמזים:**

  • הבנות רשימה יוצרות רשימות, אך ניתן להשתמש בהן ליצירת רשימות של טאפלים, שניתן לאחר מכן להעביר לבנאי <span dir="ltr"><code><span dir="ltr">dict()span>code>span>.
  • חשוב כיצד לחלץ את השם והגיל מכל טאפל.
  • זכור שמילון הוא מבנה נתונים המאחסן זוגות מפתח-ערך.

    • **הסבר קצר:**

    בפייתון ניתן ליצור מילון מרשימת טאפלים, כאשר כל טאפל מורכב משני אלמנטים (מפתח, ערך).

    לשם כך, אני משתמש בבנאי <span dir="ltr"><code><span dir="ltr">dict()span>code>span>. לדוגמה:

    
my_list = [("a", 1), ("b", 2)]
my_dict = dict(my_list)
print(my_dict) # פלט {'a': 1, 'b': 2}

    המשימה שלך היא להשתמש בהבנת רשימה כדי ליצור רשימת טאפלים זו, אשר תשמש לאחר מכן ליצירת המילון.

    הפתרון שלי:

    
# רשימת טאפלים מקורית עם נתוני אנשים
people_data = [
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35),
    ("David", 28),
]

# אני משתמש בהבנת רשימה כדי ליצור רשימת טאפלים, שאותה אני ממיר לאחר מכן למילון
people_dict = dict(
    [
      (name, age) # צור טאפל (שם, גיל)
      for name, age in people_data # עבר על טאפלים ב-people_data
    ]
)

# הדפס את המילון שהתקבל
print(people_dict) # פלט: {'Alice': 30, 'Bob': 25, 'Charlie': 35, 'David': 28}

    **פירוק קוד:**

  • **<span dir="ltr"><code>people_data = [...]code>span>**: זוהי רשימת הטאפלים המקורית, כאשר כל טאפל מכיל שם וגיל של אדם.
  • **<span dir="ltr"><code>people_dict = dict(...)code>span>**: אנו משתמשים בבנאי <span dir="ltr"><code><span dir="ltr">dict()span>code>span> כדי ליצור מילון מרשימת טאפלים.
  • **<span dir="ltr"><code>[ (name, age) for name, age in people_data ]code>span>**: זוהי הבנת רשימה היוצרת רשימת טאפלים.

    • * <span dir="ltr"><code>(name, age)code>span>: ביטוי זה יוצר טאפל משני אלמנטים: <span dir="ltr"><code>namecode>span> ו-<span dir="ltr"><code>agecode>span>.

    * <span dir="ltr"><code>for name, age in people_datacode>span>: חלק זה עובר על כל הטאפלים ברשימת <span dir="ltr"><code>people_datacode>span>. בכל איטרציה, הטאפל נפרק לשני משתנים: <span dir="ltr"><code>namecode>span> ו-<span dir="ltr"><code>agecode>span>.

  • **<span dir="ltr"><code>print(people_dict)code>span>**: מדפיס את המילון שנוצר.

    • **נקודות מפתח:**

  • הבנת רשימה יוצרת רשימת טאפלים.
  • הבנאי <span dir="ltr"><code><span dir="ltr">dict()span>code>span> ממיר רשימת טאפלים למילון, תוך שימוש באלמנט הראשון של הטאפל כמפתח, ובשני כערך.
  • פירוק טאפלים בלולאה <span dir="ltr"><code>for name, age in people_datacode>span> הופך את הקוד לקריא יותר.
  • הבנת רשימה בדוגמה זו משמשת להכנת נתונים למילון.
