# אופרטורים וסמלים בפייתון

| סמל | שימוש בפייתון | שם באנגלית | תיאור מפורט יותר |

|--------|------------------------|---------------------|--------------------------|

| <span dir="ltr"><code>+code>span> | חיבור אריתמטי, שרשור מחרוזות | Plus | אופרטור חיבור. משמש הן למספרים והן לחיבור מחרוזות. |

| <span dir="ltr"><code>-code>span> | חיסור אריתמטי, מינוס אונרי | Minus | אופרטור חיסור. מינוס אונרי משנה את סימן המספר. |

| <span dir="ltr"><code>*code>span> | כפל אריתמטי, פריקת טאפלים | Multiply, asterisk | אופרטור כפל. משמש גם לפריקת אלמנטים מתוך אובייקטים ניתנים לאיטרציה. |

| <span dir="ltr"><code>/code>span> | חלוקה אריתמטית | Divide by, slash | אופרטור חלוקה עם נקודה צפה. |

| <span dir="ltr"><code>//code>span> | חלוקה שלמה | Floor division | אופרטור חלוקה שלמה. מחזיר את החלק השלם של החלוקה. |

| <span dir="ltr"><code>%code>span> | שארית חלוקה | Modulus, percent | אופרטור שארית חלוקה. |

| <span dir="ltr"><code>**code>span> | העלאה בחזקה | Exponentiation | אופרטור העלאה בחזקה. |

| <span dir="ltr"><code>=code>span> | אופרטור השמה | Equals | מקצה ערך למשתנה. |

| <span dir="ltr"><code>==code>span> | אופרטור השוואה לשוויון | Equal to | בודק אם שני ערכים שווים. |

| <span dir="ltr"><code>!=code>span> | אופרטור השוואה לאי-שוויון | Not equal to | בודק אם שני ערכים אינם שווים. |

| <span dir="ltr"><code><code>span> | אופרטור השוואה "קטן מ-" | Less than | בודק אם הערך הראשון קטן מהשני. |

| <span dir="ltr"><code>>code>span> | אופרטור השוואה "גדול מ-" | Greater than | בודק אם הערך הראשון גדול מהשני. |

| <span dir="ltr"><code><=code>span> | אופרטור השוואה "קטן מ- או שווה ל-" | Less than or equal to | בודק אם הערך הראשון קטן מהשני או שווה לו. |

| <span dir="ltr"><code>>=code>span> | אופרטור השוואה "גדול מ- או שווה ל-" | Greater than or equal to | בודק אם הערך הראשון גדול מהשני או שווה לו. |

| <span dir="ltr"><code>( )code>span> | סוגריים לקיבוץ ביטויים, קריאה לפונקציות | Parentheses | משמשים לשינוי סדר ביצוע פעולות, וכן לקריאה לפונקציות. |

| <span dir="ltr"><code>[ ]code>span> | אינדוקס, חיתוכים, יצירת רשימות | Square brackets | משמשים לגישה לאלמנטים של רשימות, טאפלים ומחרוזות, וכן ליצירת רשימות. |

| <span dir="ltr"><code>{ }code>span> | יצירת קבוצות, מילונים | Curly braces | משמשים ליצירת קבוצות ומילונים. |

| <span dir="ltr"><code>:code>span> | אופרטור חיתוך, יצירת מילונים | Colon | משמש ליצירת חיתוכים ובתחביר מילונים. |

| <span dir="ltr"><code>,code>span> | מפריד אלמנטים ברשימות, טאפלים, קבוצות | Comma | מפריד אלמנטים ברשימות, טאפלים וקבוצות. |

| <span dir="ltr"><code>.code>span> | גישה לתכונות, קריאה לשיטות | Dot | משמש לגישה לתכונות אובייקטים וקריאה לשיטות. |

| <span dir="ltr"><code>;code>span> | מפריד הוראות (בשימוש נדיר) | Semicolon | בפייתון בדרך כלל אינו נדרש, אך יכול לשמש להפרדת הוראות באותה שורה. |

| <span dir="ltr"><code>#code>span> | הערה | Hash, pound, number sign | משמש להוספת הערות לקוד. |

| <span dir="ltr"><code>@code>span> | דקורטור | At sign | משמש ליצירת דקורטורים. |

| <span dir="ltr"><code>&code>span> | AND ביטבי | And | אופרטור AND ביטבי. |

| <span dir="ltr"><code>\|code>span> | OR ביטבי | Or | אופרטור OR ביטבי. |

| <span dir="ltr"><code>^code>span> | XOR ביטבי | XOR | אופרטור XOR ביטבי. |

| <span dir="ltr"><code>~code>span> | השלמה ביטבית | Bitwise NOT | אופרטור השלמה ביטבית. |
