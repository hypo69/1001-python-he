**1. ניהול קבצים וספריות:**

  • **<span dir="ltr"><code>attribcode>span>**: מציג או משנה תכונות של קובץ או ספרייה (מוסתר, לקריאה בלבד, ארכיון וכו').

    • * **תחביר:** <span dir="ltr"><code>attrib [+r|-r] [+a|-a] [+s|-s] [+h|-h] [כונן:][נתיב]שם_קובץcode>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>+rcode>span>: הגדר תכונה "לקריאה בלבד"

    * <span dir="ltr"><code>-rcode>span>: הסר תכונה "לקריאה בלבד"

    * <span dir="ltr"><code>+acode>span>: הגדר תכונה "ארכיון"

    * <span dir="ltr"><code>-acode>span>: הסר תכונה "ארכיון"

    * <span dir="ltr"><code>+scode>span>: הגדר תכונה "מערכת"

    * <span dir="ltr"><code>-scode>span>: הסר תכונה "מערכת"

    * <span dir="ltr"><code>+hcode>span>: הגדר תכונה "מוסתר"

    * <span dir="ltr"><code>-hcode>span>: הסר תכונה "מוסתר"

    * **דוגמאות:**

    * <span dir="ltr"><code>attrib +h myfile.txtcode>span>: הפוך את הקובץ <span dir="ltr"><code>myfile.txtcode>span> למוסתר.

    * <span dir="ltr"><code>attrib -r myfile.txtcode>span>: הסר את התכונה "לקריאה בלבד" מהקובץ <span dir="ltr"><code>myfile.txtcode>span>.

    * <span dir="ltr"><code>attrib *.*code>span>: הצג תכונות של כל הקבצים בספרייה הנוכחית.

  • **<span dir="ltr"><code>cdcode>span> או <span dir="ltr"><code>chdircode>span>**: משנה את הספרייה הנוכחית.

    • * **תחביר:** <span dir="ltr"><code>cd [נתיב]code>span> או <span dir="ltr"><code>chdir [נתיב]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>..code>span>: עבור לספריית האב.

    * <span dir="ltr"><code>\code>span>: עבור לשורש הכונן.

    * **דוגמאות:**

    * <span dir="ltr"><code>cd Documentscode>span>: עבור לתיקיית <span dir="ltr"><code>Documentscode>span> בספרייה הנוכחית.

    * <span dir="ltr"><code>cd C:\Users\User\Downloadscode>span>: עבור לתיקיית <span dir="ltr"><code>Downloadscode>span> בכונן C.

    * <span dir="ltr"><code>cd ..code>span>: עבור לספריית האב.

    * <span dir="ltr"><code>cd \code>span>: עבור לשורש הכונן הנוכחי.

  • **<span dir="ltr"><code>copycode>span>**: מעתיק קובץ אחד או יותר.

    • * **תחביר:** <span dir="ltr"><code>copy [קובץ_מקור] [קובץ_יעד]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/acode>span> - העתק קובץ ASCII

    * <span dir="ltr"><code>/bcode>span> - העתק קובץ בינארי

    * <span dir="ltr"><code>/vcode>span> - ודא שהקבצים הועתקו כהלכה

    * **דוגמאות:**

    * <span dir="ltr"><code>copy myfile.txt mycopy.txtcode>span>: העתק את הקובץ <span dir="ltr"><code>myfile.txtcode>span> ל-<span dir="ltr"><code>mycopy.txtcode>span>.

    * <span dir="ltr"><code>copy *.txt C:\Backupcode>span>: העתק את כל הקבצים עם סיומת <span dir="ltr"><code>.txtcode>span> לתיקיית <span dir="ltr"><code>C:\Backupcode>span>.

  • **<span dir="ltr"><code>delcode>span> או <span dir="ltr"><code>erasecode>span>**: מוחק קובץ אחד או יותר.

    • * **תחביר:** <span dir="ltr"><code>del [שם_קובץ]code>span> או <span dir="ltr"><code>erase [שם_קובץ]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/pcode>span>: בקש אישור לפני מחיקת כל קובץ.

    * <span dir="ltr"><code>/fcode>span>: מחק קבצים "לקריאה בלבד".

    * <span dir="ltr"><code>/scode>span>: מחק קבצים מתתי-ספריות.

    * <span dir="ltr"><code>/qcode>span> - מחק קבצים ללא בקשה.

    * **דוגמאות:**

    * <span dir="ltr"><code>del myfile.txtcode>span>: מחק את הקובץ <span dir="ltr"><code>myfile.txtcode>span>.

    * <span dir="ltr"><code>del *.tmpcode>span>: מחק את כל הקבצים עם סיומת <span dir="ltr"><code>.tmpcode>span>.

    * <span dir="ltr"><code>del /f /s *.logcode>span> - מחק את כל הקבצים עם סיומת .log בספרייה הנוכחית ובכל תתי-הספריות, ללא אישור.

  • **<span dir="ltr"><code>dircode>span>**: מציג רשימה של קבצים וספריות בספרייה הנוכחית.

    • * **תחביר:** <span dir="ltr"><code>dir [נתיב] [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/acode>span>: הצג את כל הקבצים (כולל מוסתרים).

    * <span dir="ltr"><code>/wcode>span>: הצג רשימה בפורמט רחב.

    * <span dir="ltr"><code>/pcode>span>: הצג רשימה עמוד אחר עמוד.

    * <span dir="ltr"><code>/scode>span>: הצג קבצים גם מתתי-ספריות.

    * <span dir="ltr"><code>/bcode>span> - הצג רק את שם הקובץ.

    * **דוגמאות:**

    * <span dir="ltr"><code>dircode>span>: הצג רשימה של קבצים וספריות בספרייה הנוכחית.

    * <span dir="ltr"><code>dir C:\Users\User\Documents /acode>span>: הצג את כל הקבצים בספריית <span dir="ltr"><code>Documentscode>span>.

    * <span dir="ltr"><code>dir /wcode>span>: הצג רשימה של קבצים וספריות בספרייה הנוכחית בפורמט רחב.

    * <span dir="ltr"><code>dir /b /s *.txtcode>span>: הצג את כל קבצי *.txt עם נתיב מלא.

  • **<span dir="ltr"><code>mkdircode>span> או <span dir="ltr"><code>mdcode>span>**: יוצר ספרייה חדשה.

    • * **תחביר:** <span dir="ltr"><code>mkdir [נתיב]code>span> או <span dir="ltr"><code>md [נתיב]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>mkdir NewFoldercode>span>: צור תיקייה חדשה <span dir="ltr"><code>NewFoldercode>span> בספרייה הנוכחית.

    * <span dir="ltr"><code>mkdir C:\Backupcode>span>: צור תיקייה חדשה <span dir="ltr"><code>Backupcode>span> בכונן <span dir="ltr"><code>Ccode>span>.

  • **<span dir="ltr"><code>movecode>span>**: מעביר קובץ אחד או יותר או ספריות.

    • * **תחביר:** <span dir="ltr"><code>move [קובץ_מקור] [קובץ_יעד]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>move myfile.txt C:\Documentscode>span>: העבר את הקובץ <span dir="ltr"><code>myfile.txtcode>span> לתיקיית <span dir="ltr"><code>C:\Documentscode>span>.

    * <span dir="ltr"><code>move dir1 dir2code>span> - העבר את תיקיית <span dir="ltr"><code>dir1code>span> לתיקיית <span dir="ltr"><code>dir2code>span>.

  • **<span dir="ltr"><code>rdcode>span> או <span dir="ltr"><code>rmdircode>span>**: מוחק ספרייה.

    • * **תחביר:** <span dir="ltr"><code>rd [נתיב]code>span> או <span dir="ltr"><code>rmdir [נתיב]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/scode>span>: מחק ספרייה עם כל תתי-הספריות שלה.

    * <span dir="ltr"><code>/qcode>span> - מחק ספרייה ללא בקשה.

    * **דוגמאות:**

    * <span dir="ltr"><code>rd myfoldercode>span>: מחק תיקייה ריקה <span dir="ltr"><code>myfoldercode>span>.

    * <span dir="ltr"><code>rd /s /q myfoldercode>span> - מחק את תיקיית <span dir="ltr"><code>myfoldercode>span> עם כל תוכנה ללא בקשה.

  • **<span dir="ltr"><code>rencode>span> או <span dir="ltr"><code>renamecode>span>**: משנה שם של קובץ או ספרייה.

    • * **תחביר:** <span dir="ltr"><code>ren [שם_ישן] [שם_חדש]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>ren myfile.txt newfile.txtcode>span>: שנה את שם הקובץ <span dir="ltr"><code>myfile.txtcode>span> ל-<span dir="ltr"><code>newfile.txtcode>span>.

  • **<span dir="ltr"><code>typecode>span>**: מציג את תוכן קובץ טקסט.

    • * **תחביר:** <span dir="ltr"><code>type [שם_קובץ]code>span>

    * **דוגמה:** <span dir="ltr"><code>type myfile.txtcode>span>: הצג את תוכן הקובץ <span dir="ltr"><code>myfile.txtcode>span>.

  • **<span dir="ltr"><code>xcopycode>span>**: מעתיק קבצים וספריות (כולל תתי-ספריות).

    • * **תחביר:** <span dir="ltr"><code>xcopy [מקור] [יעד] [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/scode>span>: העתק ספריות ותתי-ספריות (למעט ריקות).

    * <span dir="ltr"><code>/ecode>span>: העתק ספריות ותתי-ספריות (כולל ריקות).

    * <span dir="ltr"><code>/icode>span>: אם היעד אינו קיים, צור אותו.

    * <span dir="ltr"><code>/ycode>span>: דכא בקשת אישור להחלפה.

    * <span dir="ltr"><code>/dcode>span> - העתק רק קבצים חדשים.

    * **דוגמאות:**

    * <span dir="ltr"><code>xcopy C:\Source D:\Destination /scode>span>: העתק את ספריית <span dir="ltr"><code>C:\Sourcecode>span> לכל תתי-הספריות בתיקיית <span dir="ltr"><code>D:\Destinationcode>span>.

    * <span dir="ltr"><code>xcopy C:\Source D:\Destination /e /ycode>span>: העתק את ספריית <span dir="ltr"><code>C:\Sourcecode>span> לכל תתי-הספריות בתיקיית <span dir="ltr"><code>D:\Destinationcode>span>, כולל ריקות וללא בקשת אישור להחלפה.

    **2. פעולות דיסק:**

  • **<span dir="ltr"><code>diskpartcode>span>**: מפעיל את כלי ניהול הדיסקים (מחיצות, כרכים).

    • * **תחביר:** <span dir="ltr"><code>diskpartcode>span>

    * **הערות:**

    * <span dir="ltr"><code>diskpartcode>span> הוא כלי אינטראקטיבי שיש לו סט פקודות משלו.

    * נדרשות הרשאות מנהל מערכת כדי להשתמש בו.

    * תוכל ללמוד עוד על השימוש ב-<span dir="ltr"><code>diskpartcode>span> באמצעות הפקודה <span dir="ltr"><code>helpcode>span> בתוך כלי זה.

  • **<span dir="ltr"><code>formatcode>span>**: מפרמט דיסק.

    • * **תחביר:** <span dir="ltr"><code>format [כונן:] [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/qcode>span> - פרמוט מהיר

    * <span dir="ltr"><code>/fs:file-systemcode>span> - בחר סוג מערכת קבצים (לדוגמה, NTFS, FAT32)

    * **דוגמאות:**

    * <span dir="ltr"><code>format D: /qcode>span>: פרמוט מהיר של כונן D:.

    * <span dir="ltr"><code>format E: /fs:NTFScode>span>: פרמוט כונן E: למערכת קבצים NTFS.

    * **אזהרה:** פרמוט כונן מוחק את כל הנתונים עליו!

  • **<span dir="ltr"><code>labelcode>span>**: מגדיר או משנה את תווית הדיסק.

    • * **תחביר:** <span dir="ltr"><code>label [כונן:][תווית]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>label D: NewLabelcode>span> - הגדר תווית בכונן D כ-NewLabel.

    * <span dir="ltr"><code>label D:code>span> - נקה תווית מכונן D.

    **3. ניהול מערכת:**

  • **<span dir="ltr"><code>clscode>span>**: מנקה את מסך שורת הפקודה.

    • * **תחביר:** <span dir="ltr"><code>clscode>span>

  • **<span dir="ltr"><code>datecode>span>**: מציג או משנה את התאריך הנוכחי.

    • * **תחביר:** <span dir="ltr"><code>date [תאריך_חדש]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>datecode>span>: הצג את התאריך הנוכחי.

    * <span dir="ltr"><code>date 12-25-2024code>span>: שנה את התאריך ל-25 בדצמבר 2024.

  • **<span dir="ltr"><code>exitcode>span>**: יוצא משורת הפקודה.

    • * **תחביר:** <span dir="ltr"><code>exitcode>span>

  • **<span dir="ltr"><code>shutdowncode>span>**: מכבה או מפעיל מחדש את המחשב.

    • * **תחביר:** <span dir="ltr"><code>shutdown [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/scode>span>: כבה את המחשב.

    * <span dir="ltr"><code>/rcode>span>: הפעל מחדש את המחשב.

    * <span dir="ltr"><code>/acode>span>: בטל כיבוי או הפעלה מחדש.

    * **דוגמאות:**

    * <span dir="ltr"><code>tasklistcode>span>: הצג רשימה של תהליכים פועלים.

    * <span dir="ltr"><code>tasklist /vcode>span>: הצג רשימה של תהליכים פועלים עם מידע מפורט.

  • **<span dir="ltr"><code>taskkillcode>span>**: מסיים תהליך.

    • * **תחביר:** <span dir="ltr"><code>taskkill [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/pid pidcode>span> - סיים תהליך לפי מזהה.

  • **<span dir="ltr"><code>timecode>span>**: מציג או משנה את השעה הנוכחית.

  • **<span dir="ltr"><code>vercode>span>**: מציג את גרסת מערכת ההפעלה.

    • * **תחביר:** <span dir="ltr"><code>vercode>span>

  • **<span dir="ltr"><code>systeminfocode>span>**: מציג מידע מפורט על המערכת.

    • * **תחביר:** <span dir="ltr"><code>systeminfocode>span>

    **4. רשת:**

  • **<span dir="ltr"><code>ipconfigcode>span>**: מציג תצורת רשת.

    • * **תחביר:** <span dir="ltr"><code>ipconfig [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/allcode>span>: הצג מידע מלא על מתאמי רשת.

    * <span dir="ltr"><code>/releasecode>span>: שחרר כתובת IP.

    * <span dir="ltr"><code>/renewcode>span>: בקש כתובת IP חדשה.

    * **דוגמאות:**

    * <span dir="ltr"><code>ipconfigcode>span>: הצג תצורת רשת בסיסית.

    * <span dir="ltr"><code>ipconfig /allcode>span>: הצג מידע מלא על כל מתאמי הרשת.

    * <span dir="ltr"><code>ipconfig /releasecode>span> - שחרר כתובת IP.

    * <span dir="ltr"><code>ipconfig /renewcode>span> - בקש כתובת IP חדשה.

  • **<span dir="ltr"><code>pingcode>span>**: שולח בקשת הד למחשב אחר ברשת.

    • * **תחביר:** <span dir="ltr"><code>ping [שם_מארח_או_כתובת_IP] [אפשרויות]code>span>

    * <span dir="ltr"><code>-n xxxcode>span> - מספר בקשות.

    * <span dir="ltr"><code>-tcode>span> - פינג ללא הגבלה.

    * **דוגמאות:**

    * <span dir="ltr"><code>ping google.comcode>span>: פינג לשרת <span dir="ltr"><code>google.comcode>span>.

    * <span dir="ltr"><code>ping 192.168.1.100code>span>: פינג למחשב עם כתובת IP <span dir="ltr"><code>192.168.1.100code>span>.

    * <span dir="ltr"><code>ping google.com -n 10code>span> - בצע 10 פינגים.

  • **<span dir="ltr"><code>tracertcode>span>**: מציג את נתיב החבילות למחשב אחר ברשת.

    • * **תחביר:** <span dir="ltr"><code>tracert [שם_מארח_או_כתובת_IP]code>span>

    * **דוגמה:** <span dir="ltr"><code>tracert google.comcode>span>: הצג את נתיב החבילות ל-<span dir="ltr"><code>google.comcode>span>.

  • **<span dir="ltr"><code>netstatcode>span>**: מציג חיבורי רשת.

    • * **תחביר:** <span dir="ltr"><code>netstat [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>-acode>span>: מציג את כל החיבורים.

    * <span dir="ltr"><code>-bcode>span>: מציג יישומים שיצרו חיבורים.

    * **דוגמאות:**

    * <span dir="ltr"><code>netstatcode>span> - הצג את כל החיבורים הפעילים.

    * <span dir="ltr"><code>netstat -a -bcode>span> - הצג את כל החיבורים והיישומים שפתחו אותם.

  • **<span dir="ltr"><code>nslookupcode>span>**: שולח שאילתת מידע DNS.

    • * **תחביר:** <span dir="ltr"><code>nslookup [שם_מארח]code>span>

    * **דוגמה:** <span dir="ltr"><code>nslookup google.comcode>span>: שלח שאילתת מידע DNS עבור <span dir="ltr"><code>google.comcode>span>.

    **5. פקודות אחרות:**

  • **<span dir="ltr"><code>assoccode>span>**: מציג או משנה שיוכי קבצים.

    • * **תחביר:** <span dir="ltr"><code>assoc [.סיומת]=[סוג_קובץ]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>assoc .txtcode>span>: הצג את סוג הקובץ עבור סיומת <span dir="ltr"><code>.txtcode>span>.

    * <span dir="ltr"><code>assoc .txt=txtfilecode>span>: הגדר את סוג הקובץ עבור סיומת <span dir="ltr"><code>.txtcode>span> כ-<span dir="ltr"><code>txtfilecode>span>.

  • **<span dir="ltr"><code>callcode>span>**: קורא קובץ אצווה אחד מתוך אחר.

    • * **תחביר:** <span dir="ltr"><code>call [שם_קובץ_אצווה]code>span>

    * **דוגמה:** <span dir="ltr"><code>call mybatch.batcode>span>: קרא לקובץ האצווה <span dir="ltr"><code>mybatch.batcode>span>.

  • **<span dir="ltr"><code>chcpcode>span>**: משנה את דף הקוד של הקונסולה.

    • * **תחביר:** <span dir="ltr"><code>chcp [מספר_דף_קוד]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>chcp 1251code>span>: הגדר את דף הקוד לעברית.

    * <span dir="ltr"><code>chcpcode>span>: הצג את דף הקוד הנוכחי.

  • **<span dir="ltr"><code>choicecode>span>**: מאפשר למשתמש לבחור מתוך רשימת אפשרויות.

    • * **תחביר:** <span dir="ltr"><code>choice [/c [אפשרויות]] [/n] [/t [שניות]] [/d [אפשרות]] [טקסט]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/ccode>span>: מציין אפשרויות זמינות.

    * <span dir="ltr"><code>/ncode>span>: מסתיר את רשימת האפשרויות הזמינות.

    * <span dir="ltr"><code>/tcode>span>: מציין זמן קצוב (בשניות).

    * <span dir="ltr"><code>/dcode>span>: מציין אפשרות ברירת מחדל.

    * **דוגמה:** <span dir="ltr"><code>choice /c yn /t 10 /d n "האם אתה באמת רוצה לצאת?"code>span>

  • **<span dir="ltr"><code>findcode>span>**: מחפש מחרוזת טקסט בקובץ.

    • * **תחביר:** <span dir="ltr"><code>find "מחרוזת" [שם_קובץ]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>find "Hello" myfile.txtcode>span> - מחפש את המחרוזת "Hello" בקובץ myfile.txt.

  • **<span dir="ltr"><code>findstrcode>span>**: מחפש מחרוזות טקסט בקבצים (חזק יותר מ-<span dir="ltr"><code>findcode>span>).

    • * **תחביר:** <span dir="ltr"><code>findstr [אפשרויות] "מחרוזת" [שם_קובץ]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/icode>span> - חיפוש ללא תלות ברישיות.

    * <span dir="ltr"><code>/ncode>span> - הצג מספרי שורות.

    * <span dir="ltr"><code>/scode>span> - חפש בכל תתי-הספריות.

    * **דוגמאות:**

    * <span dir="ltr"><code>findstr /i "error" myfile.logcode>span>: מחפש את המחרוזת "error" (ללא תלות ברישיות) ב-<span dir="ltr"><code>myfile.logcode>span>.

    * <span dir="ltr"><code>findstr /n "error" *.logcode>span> - מחפש את המחרוזת "error" בכל קבצי .log, ומציג מספרי שורות.

  • **<span dir="ltr"><code>gpupdatecode>span>**: מעדכן מדיניות קבוצתית.

    • * **תחביר:** <span dir="ltr"><code>gpupdate [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/forcecode>span> - עדכן בכוח.

    * **דוגמה:** <span dir="ltr"><code>gpupdate /forcecode>span> - עדכן בכוח מדיניות קבוצתית.

  • **<span dir="ltr"><code>helpcode>span> או <span dir="ltr"><code>/?code>span>**: מציג עזרה עבור פקודה.

    • * **תחביר:** <span dir="ltr"><code>help [שם_פקודה]code>span> או <span dir="ltr"><code>[שם_פקודה] /?code>span>

    * **דוגמה:** <span dir="ltr"><code>help dircode>span> או <span dir="ltr"><code>dir /?code>span>: הצג עזרה עבור הפקודה <span dir="ltr"><code>dircode>span>.

  • **<span dir="ltr"><code>pausecode>span>**: משהה ביצוע קובץ אצווה וממתין ללחיצת מקש.

    • * **תחביר:** <span dir="ltr"><code>pausecode>span>

  • **<span dir="ltr"><code>pathcode>span>**: מציג או משנה משתני סביבה PATH.

    • * **תחביר**: <span dir="ltr"><code>path [נתיב]code>span>

    * **דוגמאות**:

    * <span dir="ltr"><code>pathcode>span>: הצג את הערך הנוכחי של משתנה PATH.

    * <span dir="ltr"><code>path C:\bincode>span> - הוסף את תיקיית C:\bin למשתנה PATH.

  • **<span dir="ltr"><code>promptcode>span>**: מגדיר את מחרוזת ההנחיה של שורת הפקודה.

    • * **תחביר:** <span dir="ltr"><code>prompt [טקסט]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>prompt $p$gcode>span>: הגדר את מחרוזת ההנחיה לנתיב הנוכחי.

    * <span dir="ltr"><code>prompt MyPrompt>code>span>: הגדר את מחרוזת ההנחיה ל-<span dir="ltr"><code>MyPrompt>code>span>.

  • **<span dir="ltr"><code>setcode>span>**: מציג, מגדיר או מוחק משתני סביבה.

    • * **תחביר:** <span dir="ltr"><code>set [שם=[ערך]]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>setcode>span>: הצג את כל משתני הסביבה.

    * <span dir="ltr"><code>set myvar=myvaluecode>span>: הגדר את משתנה הסביבה <span dir="ltr"><code>myvarcode>span> ל-<span dir="ltr"><code>myvaluecode>span>.

    * <span dir="ltr"><code>set myvar=code>span>: מחק את משתנה הסביבה <span dir="ltr"><code>myvarcode>span>.

  • **<span dir="ltr"><code>startcode>span>**: מפעיל תוכנית או פותח קובץ.

    • * **תחביר:** <span dir="ltr"><code>start [שם_תוכנית_או_קובץ] [אפשרויות]code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>start notepad.execode>span>: הפעל את פנקס הרשימות.

    * <span dir="ltr"><code>start myfile.txtcode>span>: פתח את הקובץ <span dir="ltr"><code>myfile.txtcode>span> באמצעות התוכנית המוגדרת כברירת מחדל.

    * <span dir="ltr"><code>start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" "https://www.google.com"code>span> - פתח אתר אינטרנט בכרום.

  • **<span dir="ltr"><code>treecode>span>**: מציג את מבנה הספריות הגרפי.

    • * **תחביר:** <span dir="ltr"><code>tree [כונן:][נתיב] [אפשרויות]code>span>

    * **אפשרויות:**

    * <span dir="ltr"><code>/fcode>span>: הצג גם שמות קבצים.

    * <span dir="ltr"><code>/acode>span> - השתמש בתווי ASCII.

    * **דוגמאות:**

    * <span dir="ltr"><code>treecode>span> - הצג את מבנה הספרייה הנוכחית.

    * <span dir="ltr"><code>tree /fcode>span> - הצג את מבנה הספרייה הנוכחית עם קבצים.

  • **<span dir="ltr"><code>wherecode>span>**: מוצא את מיקום הקובץ.

    • * **תחביר:** <span dir="ltr"><code>where [שם_קובץ]code>span>

    * **דוגמה:** <span dir="ltr"><code>where notepad.execode>span>

    **6. פקודות קובץ אצווה:**

  • **<span dir="ltr"><code>echocode>span>**: מציג טקסט על המסך.

    • * **תחביר:** <span dir="ltr"><code>echo [טקסט]code>span>

    * <span dir="ltr"><code>echo oncode>span> - הפעל הצגת פקודות בעת ביצוע קובץ אצווה.

    * <span dir="ltr"><code>echo offcode>span> - בטל הצגת פקודות בעת ביצוע קובץ אצווה.

    * **דוגמאות:**

    * <span dir="ltr"><code>echo Hello, world!code>span>: הצג על המסך "Hello, world!".

    * <span dir="ltr"><code>echo offcode>span> - בטל פלט פקודות.

    * <span dir="ltr"><code>echo oncode>span> - הפעל פלט פקודות.

  • **<span dir="ltr"><code>remcode>span>**: מוסיף הערה לקובץ אצווה.

    • * **תחביר:** <span dir="ltr"><code>rem [הערה]code>span>

    * **דוגמה:** <span dir="ltr"><code>rem This is a commentcode>span>

  • **<span dir="ltr"><code>gotocode>span>**: קופץ לתווית.

    • * **תחביר:** <span dir="ltr"><code>goto [תווית]code>span>

    * **דוגמה:**

    ```batch

    :start

    echo Hello

    goto end

    echo This will not be displayed

    :end

    echo Goodbye

    ```

  • **<span dir="ltr"><code>ifcode>span>**: מבצע לוגיקה מותנית.

    • * **תחביר:** <span dir="ltr"><code>if [not] condition (פקודה1) else (פקודה2)code>span>

    * **דוגמאות:**

    * <span dir="ltr"><code>if exist file.txt echo file existscode>span> - בדיקה אם קובץ קיים.

    * <span dir="ltr"><code>if %var%== "text" echo variable has valuecode>span> - בדיקת ערך משתנה.

  • **<span dir="ltr"><code>forcode>span>**: מבצע פעולה מחזורית.

    • * **תחביר:** <span dir="ltr"><code>for %%משתנה in (קבוצה) do [פקודה]code>span>

    * **דוגמה:**

    ```batch

    for %%a in (*.txt) do (

    echo %%a

    type %%a

    echo -------------

    )

    ```

    * הצג את שם כל קובץ <span dir="ltr"><code>.txtcode>span> ותוכנו.

    **אזהרה:**

  • היזהר בעת שימוש בפקודות שיכולות לשנות את המערכת, במיוחד בעת שימוש בהרשאות מנהל מערכת.
  • לפני ביצוע הפקודה <span dir="ltr"><code>formatcode>span>, ודא שבחרת את הכונן הנכון.
