**משתנים בפייתון: מהם ולמה הם נחוצים**

משתנים — הם מיכלים בעלי שם לאחסון נתונים בזיכרון המחשב. הם מאפשרים לגשת לנתונים לפי שם, במקום להשתמש בהם ישירות.

דוגמה:


x = 10
y = 'שלום, עולם!'

כאן <span dir="ltr"><code>xcode>span> ו-<span dir="ltr"><code>ycode>span> — משתנים. <span dir="ltr"><code>xcode>span> מאחסן את המספר 10, ו-<span dir="ltr"><code>ycode>span> — את המחרוזת 'שלום, עולם!'.

**כיצד פועלים משתנים בפייתון?**

  • **טיפוס דינמי**:

    • בפייתון אין צורך לציין את טיפוס המשתנה בעת יצירתו — זה נעשה אוטומטית. לדוגמה:

    ```python

    a = 42 # מספר שלם (int)

    b = 3.14 # מספר נקודה צפה (float)

    c = 'טקסט' # מחרוזת (str)

    ```

  • **מודל אחסון נתונים מבוסס הפניה**:

    • משתנים בפייתון — הם הפניות לאובייקטים בזיכרון. לדוגמה:

    ```python

    x = 5

    y = x # y כעת גם מצביע על אובייקט 5

    x = 10 # x כעת מצביע על אובייקט אחר 10, ו-y עדיין מצביע על 5

    ```

    ---

    **כללי מתן שמות למשתנים**

  • **כללים מחייבים**:

    • - שם משתנה יכול להכיל אותיות, ספרות וסימן <span dir="ltr"><code>_code>span>, אך אינו יכול להתחיל בספרה.

    ✅ דוגמאות: <span dir="ltr"><code>my_varcode>span>, <span dir="ltr"><code>_datacode>span>, <span dir="ltr"><code>var123code>span>

    ❌ לא נכון: <span dir="ltr"><code>123varcode>span>, <span dir="ltr"><code>my-varcode>span>

    - שם משתנה רגיש לאותיות רישיות/קטנות.

    דוגמה: <span dir="ltr"><code>agecode>span> ו-<span dir="ltr"><code>Agecode>span> — הם משתנים שונים.

  • **המלצות לשמות משמעותיים**:

    • - השתמש בשמות המשקפים את מהות הנתונים.

    ❌ גרוע: <span dir="ltr"><code>a = 100code>span>, <span dir="ltr"><code>b = 'שם'code>span>

    ✅ טוב: <span dir="ltr"><code>salary = 100code>span>, <span dir="ltr"><code>username = 'שם'code>span>

    - לשמות מרובי מילים, השתמש בסגנון **snake_case**:

    דוגמה: <span dir="ltr"><code>user_agecode>span>, <span dir="ltr"><code>total_costcode>span>.

  • **מילים שמורות**:

    • אסור להשתמש במילות מפתח של פייתון (לדוגמה, <span dir="ltr"><code>ifcode>span>, <span dir="ltr"><code>forcode>span>, <span dir="ltr"><code>whilecode>span>) כשמות משתנים.

    כדי לראות רשימת מילות מפתח, הרץ:

    ```python

    import keyword

    print(keyword.kwlist)

    ```

    ---

    **מאפייני אחסון טיפוסי נתונים**

  • **טיפוסי נתונים בפייתון**:

    • - **מספרים**: <span dir="ltr"><code>intcode>span>, <span dir="ltr"><code>floatcode>span>, <span dir="ltr"><code>complexcode>span>

    - **מחרוזות**: <span dir="ltr"><code>strcode>span>

    - **רשימות**: <span dir="ltr"><code>listcode>span>

    - **טאפלים**: <span dir="ltr"><code>tuplecode>span>

    - **קבוצות**: <span dir="ltr"><code>setcode>span>

    - **מילונים**: <span dir="ltr"><code>dictcode>span>

  • **טיפוסים ניתנים לשינוי ובלתי ניתנים לשינוי**:

    • - **ניתנים לשינוי**: <span dir="ltr"><code>listcode>span>, <span dir="ltr"><code>dictcode>span>, <span dir="ltr"><code>setcode>span>.

    דוגמה:

    ```python

    lst = [1, 2, 3]

    lst.append(4) # הרשימה השתנתה

    ```

    - **בלתי ניתנים לשינוי**: <span dir="ltr"><code>intcode>span>, <span dir="ltr"><code>floatcode>span>, <span dir="ltr"><code>strcode>span>, <span dir="ltr"><code>tuplecode>span>.

    דוגמה:

    ```python

    name = 'אליס'

    name[0] = 'B' # שגיאה, מחרוזות בלתי ניתנות לשינוי

    ```

  • **פונקציית <span dir="ltr"><code>typecode>span> לבדיקת טיפוס**:

    • ```python

    x = 10

    print(type(x)) # <class 'int'>

    ```

    ---

    **טיפים למתחילים**

  • השתמש בשמות משתנים משמעותיים כדי שהקוד שלך יהיה מובן.
  • זכור שפייתון אינה דורשת הצהרת טיפוס משתנה, אך היזהר לא להתבלבל עם טיפוסי נתונים.
  • למד פונקציות מובנות לעבודה עם משתנים, כגון <span dir="ltr"><code>type()code>span>, <span dir="ltr"><code>id()code>span> ומודולים, כגון <span dir="ltr"><code>syscode>span>, כדי להבין טוב יותר כיצד פייתון מנהלת זיכרון.
